Exporting Manufacturing Assemblies

Manufacturing Assembly structures can be exported as internal products into a target product document.

When exporting a Manufacturing Assembly structure, an option is also provided to export into a hierarchy of external products. This results in a set of product documents, with each one corresponding to a Manufacturing Assembly in the selected hierarchy. The output product documents can then be saved via File > Save in order to save the changes.

  1. Create a Manufacturing Assembly hierarchy.

  2. Right-click the parent Manufacturing Assembly and select Save as CATProduct from the contextual menu.

  3. The Save in a New CATProduct dialog box is displayed.

    Select Export each manufacturing assembly to an external file.

  4. Click OK.
    The selected Manufacturing Assembly and its child Manufacturing Assemblies are exported as external CATProduct documents for further use.

  5. Select File > Save Management to see the new CATProducts that have been created in the session.

    These exported CATProduct documents can be saved via File > Save.

Notes

The Export each manufacturing assembly to an external file option is available only for Manufacturing Assemblies. Manufacturing Kits are not supported.
