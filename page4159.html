Selecting the Parameters for Simulation Reporting

This procedure describes how to set up simulation reporting mechanisms.

Previously, in Production System Analysis, once the report is setup (Report Setup Command) and the simulation is run, two XML’s are generated for each run. One is the ActivityStatistics XML and the other is the SystemStatistics XML. Both these XML’s are generated in the Result Output Directory (Set in the Tools | Option) in a folder containing name of the document and under different run folders. Each run has a different XML.

This enhancement will now remove the creation of multiple SystemStatistics XML files, and provide a single SystemStatistics XML file which will contain all the simulation run statistics.

During a simulation run, details of the simulation are collected and stored in an XML which conforms to a particular schema. This Statistics XML data is then analyzed and then a resulting report is generated by PSY.
XML Schema - An XML Schema is a description of a type of XML document, typically expressed in terms of constraints on the structure and content of documents of that type, above and beyond the basic syntax constraints imposed by XML itself. An XML schema provides a view of the document type at a relatively high level of abstraction.

When completed, this file must be saved.

  1. Click Report Setup

    The Statistics dialog box appears.

  2. Select one or more resources on the geometry or from the PPR tree. Please note that only Programmable Resources, such as devices, can be selected.

    The names of the resources appear under the Selected Resources column.
    To remove any resources from the list, select the resources in the dialog box and then select the Remove Resource button. 

  3. In the Selected Resources column, select a resource.

    A list of parameters available to that resource appears in the Available Parameter area.

  4. Click one or more parameters from the Available Parameter list, and then select the >> button.

    The parameters appear in the Tracked Parameters area of the Statistics dialog box.
    The parameters in the Selected Parameters column are only associated with the selected resource.  If another resource is selected, only the parameters associated with that resource (which may be none) appear in the Selected Parameters column. 
      Parameters can be shifted between the lists using the arrow buttons.

  5. Select the OK button.

  6. Click Multiple Resource Simulation and run the simulation with one or more robots.
