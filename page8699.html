Customizing

  Before you start your first working session, you can customize the way you work to suit your habits.
This type of customization is stored in permanent setting files: these settings will not be lost if you end your session.

  1. Select the Tools  >  Options command.
 

The Options dialog box opens.
 

  1. Select the Shape category in the left-hand box.
 

  1. Select the ICEM Shape Design AeroExpert sub-category.
 

One tab is displayed:
 
 

  • The General tab lets you set the general options.

  • The Approximation tab lets you set the approximation options.

  • The Associativity tab lets you set the associativity options.

  • The Manipulators tab lets you set the control point options.

  • The Tolerance tab lets you set tolerances.
