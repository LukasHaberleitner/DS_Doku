A way to create an Addendum Surface is:

  • Analyze the shell configuration
  • Manually create Stable Split Curves, several Surface Extensions or geometries to create the Punch.

An alternative to this time-consuming process is:

  • Analyze the boundary and create several children boundaries, with a type, ready to be used in further commands.
    See Creating a Boundary Analysis Manager.
  • Extrapolate the analyzed boundary in Surface Extension, where the auto-pilot mode controlled by the minimum G1 and G2 length makes the extrapolation easy.
    See Extending Surfaces.
  • Connect two Surfaces Extensions by selecting them in Connect Surface.
    See Creating a Connect Surface.

Surface Extension Manager gather the above capabilities to provide a simple and effective way to compute extensions.
See Creating a Surface Extension Manager.
