Assigning Elements

Abaqus for CATIA V5 provides a library of elements that can be assigned to the parts in your model. When you create a Nonlinear Structural Analysis case, an Explicit Dynamics case, or a Thermal Analysis case, Abaqus for CATIA V5 assigns a default element type to each mesh part in your model. When you are working in the Nonlinear Structural Analysis workbench, you can modify the default element assignment that is applied to all parts. In addition, you can modify the element assignment to a selected part or to a selected region within a part. You cannot change the element assignments from their default values in the Thermal Analysis workbench.
