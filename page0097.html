Creating Line Analysis Connections

Line analysis connections are used for projecting a line onto parallel faces of separate parts in the assembly. You can use a line analysis connection as support for a seam welding connection property.

This task shows you how to create a line analysis connection between two parts in the assembly.

  1. Click the Line Analysis Connection icon .

    The Line Analysis Connection dialog box appears, and a Line Analysis Connection object appears in the specification tree under the Analysis Connections objects set.

  2. You can change the identifier of the line analysis connection by editing the Name field.

  3. Select the first part.

  4. Select the second part.

  5. Select the line that will be used as the welding line. You can select only a single line.

  6. Click OK in the Line Analysis Connection dialog box.

    A symbol representing the line analysis connection appears between the two parts.
