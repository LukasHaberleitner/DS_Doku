Setting Default Translator to Kawasaki AS

This section describes how to set the default upload and download translator to Kawasaki AS.

  1. Select Tools > Options > Resource Detailing > Offline Programming tab.

  2. Under Downloader Settings, choose Kawasaki AS to be the Selected Downloader.


     

  3. Under Uploader Settings, choose Kawasaki AS to be the Selected Uploader.

  4. Click OK and exit the options dialog box.
