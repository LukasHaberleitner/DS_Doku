Handling Move Activities

The procedures below describe how to create, copy, and edit move activities.

 Each move activity:

  • defines the object or objects to be moved, 
  • defines the track along which the object or objects are moved.

Any given product may have been created with a track.  If so, that track can be imported and used to create a move activity.  In addition, a move activity created with one part can be copied, disassociated from that part, and associated with a different part.
