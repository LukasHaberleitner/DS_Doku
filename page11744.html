Defining Home Positions for a V5 Mechanism

NC Machine Tool Builder allows the user to define one or more standard configurations (or home positions) for a mechanism. Each home position is characterized by a unique name and a set of command values. Home positions are typically used to define the desired states of a mechanism; for example, the open and closed configurations of a gripper. This procedure describes several methods that you can use to define home positions for a mechanism.
Open the V4Axis_Machine.CATProduct file from the  online/mtbug_C2/samples/V5/4AxisMachine directory.  In addition:
  • add the new machine
  • add the machine kinematics definition
  • create mount parts and points
  • create travel limits

  1. Click Home Positions . 

    The Home Position Viewer dialog box is displayed.

  2. Click New button.

    The Home Position Editor dialog box is displayed with the standard jog controls in the upper section and a text entry field in the lower section. The jog controls are used to specify the desired set of command values for the home position, while the text entry field is used to specify its name.

  3. For the first home position, leave the position values as you find them, and enter the string Zero_Home in the Home Position area's Name field

  4. Click OK.

    Zero_Home has been added to the list of home positions for the mechanism.

  5. Define a second home position named Home.  Use the slider bar to set the value of Axis X to a new home position.

  6. Click OK.

  7. In the Home Position Viewer, double-click on each home position.

    This has the effect of immediately moving the mechanism to the specified coordinates.

  8. In the Home Position Viewer, select Home and click the Modify button.

  9. Move the mouse into the geometry area near Axis Y; use the manipulator to change the value of Axis Y.

  10. Click OK.

  11. Click Close on the Home Position Viewer.
For additional information on entering command values, see Jogging a Mechanism.
