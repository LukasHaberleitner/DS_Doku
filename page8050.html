Interoperability with the Part Design Workbench

  You can work in the FreeStyle Shaper/FreeStyle Optimizer workbench only, creating forms and modifying them.
However in an industrial environment, it is always useful to have integrated design tools at one's disposal. The integration between the FreeStyle Shaper workbench and the Part Design workbench is described in the chapter, based on a step-by-step scenario composed of several tasks illustrating how to work with both surface and body type elements, that is elements created using the Part Design workbench.
