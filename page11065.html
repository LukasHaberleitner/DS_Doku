Solving a Rule Base

This task explains how to solve a rule base. 
The rule base must have been created or imported from an external file, and its parameters set. 

To solve a rule base:
  • Click the icon in the Knowledge Expert Workbench. This icon is activated when the document or one of the rule base relations has been modified. A to-be-solved rule base is displayed in the specification tree with a solve icon. Click the Solve icon in the workbench to solve the rule base.
  • Right-click the rule base and select RuleBase Object >  Manual Complete Solve. This command solves the rule base no matter what its status is (to-be-solved or not).
  • Right-click the rule base and select RuleBase Object > Manual Optimized Solve to perform a solve if changes have been made since the last solve operation was performed.
