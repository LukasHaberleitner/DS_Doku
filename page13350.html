Checking Feature Fold Visibility

This task shows how to check the fold/unfold visibility of a Sheet Metal Part. Indeed in some cases, you can notice that a few features are no longer visible once the part is unfolded.
This capability gives you the list of these elements.

Open the TangentWall1.CATPart document from the samples directory.

  1. Click the Check Feature icon .
 

The list of the features that will not be visible is you unfold the part, using the Unfold icon is displayed in the dialog box.

  1. Click OK to exit the list.
