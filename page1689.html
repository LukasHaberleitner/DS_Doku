KWScript Behavior: Data Sheet  

Accessing this behavior This behavior can be accessed by clicking the KWScript Behavior icon () in the Reuse toolbar.
Describing this behavior This behavior enables you to specify how to execute a Generative Script. For more information about Generative Scripts, see Product Knowledge Template User's Guide.
Using this Behavior Interface

Name: Name attributed to the behavior.
Comment: Comment attributed to the behavior.
File path: Enter the name of the CATGScript to open.

Note that the KWScript capabilities require the Product Knowledge Template license.
