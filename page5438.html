Merging Domains in the Virtual Stacking

This task shows you how to merge sequences or plies in the virtual stacking.
Either work on the part where you have defined cells or open the VSMergeDomains01.CATPart document.

  1. Click Merge domains in the Tools Palette.
    The Merge domains dialog box appears.

  2. Click Run Search.
    The dialog box is populated with the list of plies or sequences that can receive cells.

  3. Select a ply or a sequence.
    The potential providers are displayed in the dialog box and the candidate cell is highlighted in the authoring window

    If you select the potential provider in the dialog box, you can see the cells it contains.
    Merge domains gives a result if virtual plies (not only virtual sequences) are symmetrical.

  4. Click Apply to validate the choice for this ply.
    The dialog box is updated.

  5. Repeat for other potential receivers.

  6. Click OK when you are satisfied to validate and exit the Merge domains dialog box.
