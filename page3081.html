Selects normal or(and) parallel faces
 
Selects faces that are:
  • normal to a main axis,
  • parallel or perpendicular to a reference face.

The Define Normal Faces dialog box is displayed:

The By Axis tab lets you select all the flat faces normal to a main axis of the current axis system.

  • Select the Reference Body,
  • Choose an axis and click OK.

The By Face tab lets you select flat faces with reference to the face you have picked.

  • Select the Reference Body and a Reference Face,
  • Choose Perpendicular and/or Parallel and click OK.
  • The Reference Body is populated by default if you select a face before entering the dialog box.
