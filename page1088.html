DPM Machining Process Planner

This page deals with DPM Machining Process Planner customizing options.

The Commands tab has options that enable you to:

  • define machining operation parameters at the time of creation

  • use Knowledgeware checks when in the Sequence Assistant command

  • use a line balancing Gantt chart that displays a target cycle time line.

The In-Process Model tab provides options to:

  • set the color for the current machined face and previously machined faces of the IPM

  • generate the IPM from tools, geometry, or from either

  • automate the cruise/zoom capability used when inspecting geometry changes.
