Activities    

The Activities tab contains two categories of options:

  • Macroactivities and operations
  • Items
  • Update Status

Macroactivities and operations

These options are used when generating the activity list.

Generate Activities based on EBOM

When design changes have occurred, you can update the activities using DPM Assembly Process Planner (the default) or, if the option is enabled, you can update the activities from the EBOM using the Update Activities and IPMs command. The changes taken into account are determined by what you select under Update Status.

Smart naming of macroactivities (label contains output product name)

If this option is selected, the macroactivity (process) label contains the name of the associated output product.

By default, this check box is cleared.

Smart naming of operations (label contains output product name)

If this option is selected, the operations or activity label contains the name of the associated output product.

By default, this check box is cleared.

Generate macroactivities and operations for hidden products

If this option is selected, macroactivities and operations are generated for products in the No Show space. This option applies when building the activity list only.

By default, this check box is cleared.

Macroactivities and operations can be created without geometry

If this option is selected, you can create macroactivities and operations without having to associate a product, plate or shape from the PPR product list. This option applies when manually creating macroactivities and operations only.

By default, this check box is cleared.

Items

These options are used when building the activity list  as well as when manually creating macroactivities and items.

Generate weld items

If this option is selected, weld items are automatically generated as appropriate under the joining activity.

By default, this check box is cleared.

Generate attachment and alignment line items

If this option is selected, attachment and alignment line items are automatically generated as appropriate.

By default, this check box is cleared.

Generate butt alignment items

If this option is selected, butt alignment items are automatically generated as appropriate.

By default, this check box is cleared.

Generate inverse bending curve items

If this option is selected, inverse bending curve items are automatically generated as appropriate under the profile bending activity.

By default, this check box is cleared.

Generate roll line items

If this option is selected, roll line items are automatically generated as appropriate under the plate forming activity.

By default, this check box is cleared.

Generate edge preparation items

If this option is selected, edge preparation items are automatically generated as appropriate under the joining activity.

By default, this check box is cleared.

Update Status

When you select Generate Activities based on the EBOM, the updates that are taken into account are determined by what is selected in this area. You can choose to have only what is new, what is deleted, what is modified or all the options.
