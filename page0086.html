Installation Requirements

Before installing Abaqus for CATIA V5, verify that your system meets the following requirements:

Supported platforms

The definitive list of supported platforms and other system requirements is given in the Dassault Systï¿½mes Program Directory. The Program Directory contains additional documentation, similar to release notes, available at http://media.3ds.com/support/progdir. A support contract and private login account are required for access (see https://iam.3ds.com/self_service/login/service/websupport/).

The list of supported platforms for CATIA V5 and for Abaqus solvers reside on separate pages of the Program Directory:

  • For CATIA V5, choose Version 5 as the product Line and the V5 release Level, then choose V5 Client Side in the left-hand pane.

  • For Abaqus, choose SIMULIA as the product Line and the Established Products release Level, then choose Abaqus under Prerequisites in the left-hand pane.

Choose a platform configuration that satisfies the requirements on both pages.

CATIA V5 requirements

For information about which Abaqus versions are supported by Abaqus for CATIA V5 and Abaqus for CATIA V5 Prerequisite, refer to the Abaqus for CATIA V5 Compatibility Matrix on 3ds.com.

Running Abaqus for CATIA V5 requires either a CATIA GPS license or an Abaqus for CATIA Complete (ACC) license; the latter license provides access to most of the functionality granted by the CATIA licenses for GPS and GAS. If you are using Abaqus for CATIA V5 with a license for GPS, other CATIA V5 licenses are necessary to allow access to some functionality. Some examples of additional license requirements are described in Table B–1.

Table B–1 Licenses used for some Abaqus for CATIA V5 functions.

If you are using Abaqus for CATIA V5 with a license for AFC Complete, you do not need any other CATIA V5 simulation licenses. The AFC Complete license provides access to all Abaqus for CATIA V5 functionality.

Some Abaqus for CATIA V5 functionality requires support from specific types of geometry. You need the appropriate CATIA V5 licenses to create this geometry, as described in Table B–2.

Licensing

Abaqus for CATIA V5 and Abaqus for CATIA V5 Prerequisite use either the FLEXnet network license manager or the Dassault Systï¿½mes License Server (DSLS). A license server and a valid license file are required. For FLEXnet you can use an existing SIMULIA FLEXnet license server, or you can install the license server. For DSLS you can use the DSLS license server that serves the existing V5 installation. For more information on licensing, see Abaqus Licensing. You can obtain a license file through your local office representative.

Abaqus analysis products

Abaqus/Standard or Abaqus/Explicit is required to obtain simulation results but can be installed on machines separate from the client machine. For information about which releases of Abaqus can be used with Abaqus for CATIA V5, refer to the Abaqus for CATIA V5 Compatibility Matrix on 3ds.com.

Abaqus for CATIA V5 writes output database files in the format of the current version of Abaqus. If you import or attach an Abaqus output database file created using earlier versions of Abaqus, Abaqus for CATIA V5 upgrades it to the current format. The upgraded output database is saved with the original file name, and a copy of the original version is saved as filename_old.odb.

Administrator privileges

Administrator privileges are required to install the licensing for Abaqus for CATIA V5.

Microsoft Excel (for Integrated X–Y Plotting)

Abaqus for CATIA V5 requires Microsoft Excel to create and display integrated X–Y plots of history data or energy variables. For more information about the types of X–Y plots that require Microsoft Excel, see Exporting and Plotting History Data.
