Creating a Positioned Sketch

This task shows you how to position a sketch by selecting a surface from an existing part. To do so this scenario is divided into five parts:
  • Positioning the Sketch
  • Creating the sketch
  • Constraining the Sketch
  • Generating a Pad
  • Creating a Pocket

Positioning the Sketch
If it is not displayed, open the Getting_Started1.CATPart document.
 
  Note that the part appears in the Part Design workbench.

  1. Select Positioned Sketch in the Sketcher toolbar.
    The Sketch Positioning dialog box is displayed.

  2. Keep the Positioned option selected.

  3. Select Plane.2 either from the specification tree or the geometry area.

  4. Specify the reference plane for the sketch as shown here.
    The Reference field now indicates the reference plane and the Type fields of the Origin and Orientation areas are now activated.

  5. Select Curve intersection in the Type field of the Origin area.

  6. Select the part cylindrical surface to make its axis intersect with the absolute axis origin.

  7. Select Parallel to line in the Type field of the Orientation area.

  8. Select the Output as shown here.

  1. Click OK.

Creating the Sketch

You get automatically in the Sketcher workbench and you are ready to sketch a profile.
 

  1. Click Axis in the Profile toolbar.

  2. Create an axis as shown here.
 

  1. Create a circle.

  2. Click Profile in the Profile toolbar.

  3. Start drawing the profile in order to get a sketch like the one shown here.
 

To create the arc of circle using Profile , simply keep holding the left mouse button between the arc of circle first and endpoints.
 

Constraining the Sketch

  1. Multi-select the arc of circle center and the circle.

  2. Click Constraint Defined in Dialog Box in the Constraint toolbar.
    The Constraint Definition dialog box is displayed.

  3. Select Concentricity in the dialog box.

  4. Click OK.
    The constraint has been applied as shown above.

Generating the Pad

  1. Click Exit Workbench in the Workbench toolbar.
    You are now back in the Part Design workbench and the sketch is displayed.

  1. Click Pad .
    The Pad Definition dialog box is displayed.

  1. Enter 10 in the Length field.

  2. Right-click in the Selection field and click Go to profile definition contextual command.
    The Profile Definition dialog box is displayed.

  1. Select Edge.3 from the geometry area.
 
 

The Profile Definition dialog box is updated with the selected sketch.

  1. Click OK in the Profile Definition dialog box.

A warning dialog box is displayed to inform you that the profile is not valid.

  1. Click OK in the Warning dialog box.

  2. Click the Reverse Side command to change its direction and solve the problem.
    The profile is displayed as shown here. The normal is now well positioned.
 

  1. Click OK in the Pad Definition dialog box to validate the pad creation.
 

Creating a Pocket

  1. Click Pocket .
    The Pocket Definition dialog box is displayed.

  1. Right-click the Selection field.

  2. Click Go to profile definition.
 
 

The Profile Definition dialog box is displayed.

  1. Select the circle from the geometry area.

  2. Click the arrow to have it displayed upwards.
    The Profile Definition dialog box is updated with the selected element.

  1. Click OK  in the Profile Definition dialog box.
    The Pocket Definition dialog box is displayed back.

  1. Keep the Up to Last option selected.

  2. Click OK  in the Pocket Definition dialog box to validate the creation.
    The pocket is created as shown here.
  • The pad has been created and its lines have been automatically trimmed so that it is tangent with the first part.
  • To have more details on the Sketch positioning option see Creating a Positioned Sketch.
