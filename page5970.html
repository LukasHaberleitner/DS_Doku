Duplicating  Generative Geometry

This task will show you how to duplicate a sub-element of a generative view in order to modify a sub-part of a generated view independently from the part linked to the drawing.
Open the GenDrafting_part.CATDrawing document.

  1. Double-click the view you want to modify to activate it.

    The view is automatically highlighted.

  2. Select the elements you want to modify and right-click them.

  3. Select Duplicate Geometry.

    The selected geometry is duplicated on the view at the same position. It is highlighted after creation. Click on the drawing to de-select it.
  • The resulting geometry is created udner the generative one. To access it, you can hide the corresponding generative geometry.
  • The resulting geometry is a 2D geometry and has standard 2D geometry behavior and not specific behavior of the original geometry.
  • If you want to copy/paste the generative geometry, it is recommended to use the Duplicate Geometry command.
