Inserting a Functional Set

Functional Sets are used to organize the Functional Features and Generative Shape Design elements inside a Solid Functional Set. The application creates Functional Sets under a Solid Functional Set or another Functional Set.

This task shows you how to insert a Functional Set into a Solid Functional Set.
 
Open a CATPart document containing a Solid Functional Set.

  1. Select the Solid Functional Set in which you want to insert the new Functional Set.

  2. Select Insert  > Functional Set. 

    A functional set is created in the selected solid functional set. It is defined as the current node, meaning that it determines the location of the features you are going to create. It can contain any Functional Features, Sketches, and GSD features.
Functional Sets can contain:
  • Any functional features
  • Sketches
  • Generative Shape Design features
  • Geometrical Sets
Functional Sets cannot contain any Ordered Geometrical Sets.
You can create nested Functional Sets under a Functional Set.
