Sending Annotated Views

 

You can send annotated views to other users.

Note: The user sending the annotated view has to be a publisher in the meeting.

  1. Click the Annotated Views icon . The Annotated Views  dialog box appears.

     

     

  2. Select an annotated view from the list.

  3. Click OK  or Apply  button to share the annotated view.
    Note: Only one Annotated view can be selected at a time.
