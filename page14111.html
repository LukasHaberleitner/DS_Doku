Creating a Fillet Based Projected Curve with an Angle

This task shows you how to create a fillet based projected curve with an angle. The curve is the result of an intersection between a tangential sweep based on a circular sweep and a bottom plane or bottom shell.
Before you begin, see information in Creating an Addendum Design.
Open AddendumSurface01.CATPart from the samples directory.

  1. Click Fillet Based Projected Curve With Angle   in the Addendum Design toolbar.


  2. Select the direction to define the angle of the projected curve.

  3. Select a Variable Offset Curve, lying on a support, used to compute the fillet.
    The fillet is a sweep.

  4. Select a Bottom Plane or Shell to compute the intersection.

  5. Select a join of surface extensions, or several.

  6. Enter the radius of the fillet.

  7. Enter the Angle of Projection.
    It impacts the length of the projected curve.

  8. Modify the length of Sweep Until Bottom.
    • Increase the length if the sweep is too short to reach the bottom.
    • Decrease the length if the sweep is too long and results in twists.

  9. Modify the Fillet Angle to avoid multi-domains while computing the curve.

A Fillet Based Projected Curve With Angle node is created in the specification tree. It contains the parameters used for computation.
