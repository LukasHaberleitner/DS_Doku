Production System Settings  

This page deals with the Production System Settings options for Resource Detailing products.

  • Save Statistics
  • Resource State Priorities
  • Random Seeds
  • Distribution Mode
     

Save Statistics

Statistics are generated for each simulation run. At the start of a new run, the statistics of the previous run are over-written by the new one. In order store the statistics of a run, select Store statistics for simulation run check box before running the simulation. Otherwise, the run statistics of the previous run are overwritten.

The initial settings is that the check box is selected.

Resource State Properties

State Color

Resource state color can be set using this color combo.

State Priorities

At any instant a resource can be in different states but the statistics on the resource are collected on the basis of the priorities set for the various states. Priorities of states are relative.

The initial settings of both options appear above.

Random Seeds

Number of Streams

Number of streams can be set within this field; the default value is 2000.

Set Button

When you alter the entry in the Number of Streams field, and click the Set button, the Stream Number, Seed, and Antithetic options appear.

Stream Number

This option is used to select a particular stream. After selection, the Seed and Antithetic value is automatically set to that of the selected stream number.

Seed

This option is used to change the value of the seed for the stream number selected in previous option. The value of the random seed can be any random positive integer

Antithetic

By default, the value for Antithetic is set to False. If you set Antithetic to True, the inverse of the current stream is used. Using Antithetics for random numbers is useful in variance reduction.

The initial settings of all options appear above.

Negative distribution sample mode

Select a Negative distribution sample mode.  The options listed are Negative, Zero, or Resample.

The initial setting is Negative.
