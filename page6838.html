Replacing Cable References

This task shows you how to replace a cable reference of a routed cable.

1.

 With your document open, click the Replace Cable Reference button and select the cable. The Catalog Browser displays.
2. Double-click the catalog containing the cable reference. A list of cable references from the selected catalog displays.

Double-click the cable reference you want to use to replace the existing cable reference. A message displays indicating that the existing cable reference is to be replaced.

Click OK to replace the cable reference. The replaced cable reference displays in the specifications tree.
