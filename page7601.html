Parabolic Pentahedron

A parabolic pentahedron is a fifteen-node solid element.

Type

solid element

Physical property

solid

Mesh connectivity

WE15

Number of nodes

15

Degrees of freedom
(per node)

3 (translations)

Type of behavior

elastic

This element has six Gauss points:

P1 (0,1667 ; 0,1667 ; -0,577)             P2 (0,6667 ; 0,1667 ; -0,577)

P3 (0,1667 ; 0,6667 ; -0,577)             P4 (0,1667 ; 0,1667 ; 0,577)

P5 (0,6667 ; 0,1667 ; 0,577)               P6 (0,1667 ; 0,6667 ; 0,577)

There are only three translations.
