Rule Based Surface Mesher

 

The Rule Based Meshing (RBM) product must be installed.
Creating and Modifying Rules: Create and modify a mesh file rule that defines a rule based surface mesh part.
Creating a Rule Based Surface Mesh Part: Mesh a surface using rules and parameters that are defined in a mesh file rule.
  Using the Rule Based Surface Mesh Editor: Modify a rule based surface mesh part using the Rule Based Surface Mesher editor.
