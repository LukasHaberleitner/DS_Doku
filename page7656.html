Recording a Simulation Automatically

This task shows you how to record a simulation automatically using the Automatic insert functionality, the Clash Detection (Stop) in Mouse moving mode.
You can manipulate the shuttle with the mouse or the space ball/space mouse
Open the TANK_SHUTTLE.CATProduct document.

In Tools > Options > Digital Mockup > DMU Fitting > DMU Manipulation, select the following options:

  • In the Automatic Insert Configuration area, select While mouse moving.
  • In the Manipulation Mode for Shuttle Simulation, select the Automatic insert check box.

For more information, see DMU Manipulation.

  1. Double-click the Shuttle.1 (defined on the tank) in the specification tree:

    The Edit Shuttle dialog box appears.

  2. Select the Angle check box, and set the angle to 45 deg

  3. Click OK.

    The shuttle motion is defined with respect to the rotation angle defined.

  4. Select Shuttle.1 in the specification tree.

  5. Select Insert > Simulation.

    The Edit Simulation dialog box appears.

  6. Verify that the Automatic insert check box is selected and, on the DMU Check toolbar, Clash Detection (Stop) is activated.

  7. Manipulate the shuttle using the LMB; as you do so, the simulation is recorded. Move the shuttle as often as necessary; the shots are recorded automatically

    The simulation stops when the shuttle is in collision, the product is highlighted. This position is not recorded, the stop on collision lets you validate the simulation.
    This is what you obtain:

  8. Click OK.
The recording precision depends on the manipulation speed. It means the precision is approximate as the recording limit is based on the upper distance selected.
