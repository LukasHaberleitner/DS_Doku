Filtering Mesh Parts

This task shows you how to use the Mesh Part Selector button in the Advanced Meshing Tools workbench.

  1. Click the Mesh Part Selector button .

    The Mesh Part Selector dialog box appears.

    • Mesh Parts: lets you select the desired mesh parts.
    • Remove: lets you remove a previously selected mesh part.
    • Remove all: lets you remove all the selected mesh parts.

  2. Select the desired Mesh Part.

  3. Click OK in the Mesh Part Selector dialog box.
