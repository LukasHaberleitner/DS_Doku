Creating a 2D Component Reference

This task will show you how to create a detail sheet and then position a 2D component on this sheet. This 2D component will then be instantiated on a design drawing sheet.

Differentiating the design sheet from the detail sheet allows assigning a structure to the document. This means separating the drawing elements from the re-usable components.

  1. Click New Detail Sheet in the Drawing toolbar (Sheets sub-toolbar).

    The newly created detail sheet automatically appears with a view at its bottom left corner.

  2. Click New View in the Drawing toolbar. A new view is created on the sheet.

  3. Create a 2D component inside this new view. For example, create two circles on the detail sheet.
Note that you can customize both the design and the detail sheet background colors. For more information, see  Infrastructure User's Guide.
