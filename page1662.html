Logical Combination: Data Sheet

Accessing this behavior This behavior can be accessed by clicking the Logical Combination icon () in the Structure toolbar.
Describing this behavior This behavior is a combination of various behaviors enabling you to work with the aggregation concept.  
More information  The logical combination is designed to execute the behaviors that are ready for execution at a given time.  When executing a logical combination of behaviors, all you can know is if the combination can operate the behaviors it contains. The order in which the behaviors will be launched is not guaranteed. One behavior can be launched while another one is operating.
For the behaviors to be launched in a defined order in a logical combination, select Focus in the behaviors properties tab.
The behaviors belonging to a Logical Combination might be launched simultaneously.
If the behaviors are independent, and one of the behaviors is not solved, the combination will be shown as solved.
