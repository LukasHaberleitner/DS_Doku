Generating Layout Views from Functional Tolerancing and Annotation Using the Batch Monitor

This task shows you how to generate several 2D layouts from the Functional Tolerancing and Annotation using the batch utility.
  • Make sure you start the scenarios below from the 2D Layout for 3D Design window.

  1. Select Tools > Utility.. > 3DTolerancingToLayoutBatch.
    The 3DTolerancingToLayoutBatch dialog box is displayed, offering the following layout creation options:

  2. Under Selected Operation, select any one of the following 3D Tolerancing context:

    • Annotation Set
    • Captures
    • Views

  3. Under Document Selection, select any one of the following from the Document Location list:

    • File
    • ENOVIA V5 VPM: Search a document on the ENOVIA VPM server.

  4. Select File and click Browse :
    1. Under Input Options, click any one of the following options and select an input file:
      • Member: To select a single part file.
      • All: To select all the parts in the selected folder.
      • String: To enter a string.
    2. Select the required check boxes to select an input file.
    3. Under Rename Criteria and Output Options, enter the required parameters.

  5. Under Output, click Browse and select an output path.

  6. Select the Run Local option.

  7. Click Run.
    The batch process is run and an output report is generated.
