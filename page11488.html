What's New?

Enhanced Functionality

Design Change
When analyzing the geometry specified on a Machining Operation, you can now use new commands to:
  • reframe the main viewer on the selected geometry
  • center the specification tree on the selected geometry.
These commands are available in the Geometry Analyzer dialog box.
Resources
When you modify the total length value (ST) of a tool assembly, you can now select the Assembly total length linked to tool gage check box to keep the height value of the last stage of the tool holder during modification.
