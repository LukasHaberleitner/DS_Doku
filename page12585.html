About Version Management

 

The upgrade process ensures that part features such as the graphic properties and the visualization are applied as per the latest algorithm (version).

 

To upgrade the version of all the parameters, right-click the 3D Shape and select 3D Shape object > Upgrade.

The upgrade process ensures upward compatibility for all the part features which are impacted by the latest changes in the algorithm.

  • Upgrade operation may result in slight changes in the part features impacted by the latest version.
  • The upgrade process only applies to the part and not to its children.
