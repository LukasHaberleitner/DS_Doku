SmarTeam Global Refresh

 

The SmarTeam Global Refresh command allows you to update a complete CATIA product, applying the latest available revisions to all of the product’s children.
  Available from the SmarTeam right-click menu, the Global Refresh functionality launches the following actions automatically:
  • Performs a Check Out operation on the selected product and forces a sub-assembly to be checked out if its children are not the latest
  • Opens the selected product in a CATIA session to calculate all new CGRs
  • Performs a Check In operation to validate all the changes made in the database.

To use Global Refresh, do the following:

  1. Start CATIA and then connect to SmarTeam.
  2. Disable the option “Display Check-Out/New-Release dialog box on first modification” in order to avoid Check Out on the fly panels.
  3. Select a CATIA Product in SmarTeam. Note that selection of multiple objects is not supported.
  4. From the right-click menu, select CATIA and then select the Global Refresh command. A new version of the CATProduct is created and linked to the latest version of all of the Product’s children.

 

Notes:

  • Following standard SmarTeam – CATIA integration behavior, the Global Refresh operation creates a new version each time it is performed after a modification is made.
  • The Global Refresh operation works only if all children of the selected CATIA product are in “Check In” or “Release” status; i.e., if one or more of the child objects is in Checked out or in New Release state, the command cannot be performed.
