Creating a Local Corner Relief

This task explains how to define a corner relief locally on a set of supports.
Open the NEWCornerRelief01.CATPart document.
 

Creating a circular or square Corner Relief

 

 

 

 

 

 

 

 

 

  1. Click Corner Relief .
    The Corner Relief Definition dialog box is displayed.

  2. Select the type of corner relief you wish to create : circular or square (creating a local corner relief with a User Profile is explained in the second task of this topic).

  3. Select the supports on which you want the corner relief to be created (here we chose Cylindrical Bend 1 and Cylindrical Bend 2).

     
    • You can create several corner relieves in the process, by selecting manually several bending faces.

    • You can select all the bending faces with the Select all command. This command is available with a right-click in the support field.

    The Select All command only selects conical, cylindrical and planar bending faces.
       
    By default the corner relief center is the barycenter of the supports but you can also define it, either by selecting an already exixting point or by creating it.
    To create this point :
    1. Select the corner relief on which you want to define the center with a left click

    2. Right-click the line and select Create center.

    3. Create the point.

    4. Click OK.

  4. Enter the radius or length of the corner relief.

  5. Click OK to create the corner relief.
     
 

Creating a Local Corner Relief with a User Profile
Open the NEWCornerRelief01.CATPart document again.

The part needs to be unfolded prior to creating the corner relief.

  1. Click Corner Relief .

    The Corner Relief Definition dialog box is displayed.

  2. Select the User Profile type.

  3. Select the supports on which you want the corner relief to be created (here we chose Cylindrical Bend 1 and Cylindrical Bend 2).

  4. Select an already existing sketch or click Sketcher and select the xy plane.

  5. Click Profile and draw the sketch to define your corner relief.

  6. Click Exit workbench .

  7. Click OK to create the Corner Relief.

     
    With a user profile type, you can only create one corner relief at a time.
