About the Virtual Stacking Management

When you click Virtual Stacking, you enter a sub-workbench dedicated to the virtual stacking management.
  Virtual Stacking aims at defining the complete stacking, i.e. stacking sequence and virtual sequences/plies shapes by cells coverage.
It is made of:
  • The table, i.e. a 2D viewer that can be paned and zoomed.

    Inside this table:
    • Cells have a colored background corresponding to the ply orientation color.
    • Selections are associative with the 3D viewer (cells coverage, real plies contours if they have been created).
  • A set of tools gathered in the Tools Palette to perform various operations on virtual plies.
  • Some are basic operations:
    • Select
    • Repeat
    • Merge domains
    • Create a new Stacking Area
    • Check Ply Rules
    • Single Row edition
    • Multiple Row edition
    • Cell valuation
    • Apply Main Stacking Sequence Laminate
    • Move Row
    • Swap Rows
    • Duplicate Row
    • Insert Blank Row (sequence or ply)
    • Cut Row
    • Copy Row
    • Paste Row
    • Select Row from Cell
  • Some are advanced ones:
    • Recomputation from Grid
    • Maximize Symmetry
  • A set of options to change the table display (view, entity, orientation, filter), make a virtual plies review,
    manage virtual plies symmetry and columns order, undo/redo actions, display information, import/export date,
    highlight real plies contours, merge virtual plies.
  • Contextual menus.
