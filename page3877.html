Creating a Front View 

This task shows you how to create a front view.
See Using a View/Annotation Plane for more information.
See also Creating a Section View/Annotation Plane, Creating a Section Cut View/Annotation Plane.

Open the Common_Tolerancing_Annotations_01.CATPart document.

  1. Click View From Reference in the Views/Annotation Planes toolbar.

  2. Select the face as shown.

You have to select a planar element only to perform this command.
 

The View Creation dialog box appears. For more information on View Creation dialog box, see More about View Creation Dialog Box.

  1. Select Front View  as view type in the View Creation dialog box.

    Important: Only selection of planar element is compatible/authorized with this command. You must select a planer element (like plane, planer surface, axis system, or sketch) to complete the operation.

  2. Click OK in the View Creation dialog box.
    Clicking anywhere in the geometry area is equivalent to clicking OK in the dialog box associated with this command.

The front view is created.
Front views are represented by a blue reference axis, its normal axis is red until you create an annotation, and are identified as Front View.1 in the specification tree.

  1. Right-click the annotation plane in the geometry or in the specification tree and select the Invert Normal contextual menu.

The front view normal is reversed.
When the view annotation plane is created on an axis system, the Invert Normal command is not available because the view annotation plane by default takes the axis system orientation.
