Creating 3D Spline

This task shows you how to create a 3D Spline feature.

  1. Click Create a 3D Spline in the Selection Tools  toolbar.

  2. Select a visualization plane.

  3. Select check boxes to perform the corresponding actions.
    • Edit Mode, switches to edition, provided at least two points have been created.
      Note that selecting a 3D Spline before launching the command starts the edit mode.
    • Close feature
    • Point on support, creates points on geometries
    • Update when dragging points
    • Project spline, projects the spline on a support, provided all the points of the spline are on the same support.
    • Extrapolate spline, extrapolates the spline to intersect the sides of the support surface.
      The spline is smoothed for a better quality.
      The spline must be projected on the support surface (Volumes or solids are not supported).

 

  1. Alternatively, from the work area:
    • Creates a projected point on the feature under the cursor
    • Creates a projected point on the feature under the cursor, that has the same coordinates as the snapped point.
    • Creates a projected point on the feature under the cursor that has coordinates on the edge.
    • Creates a point in the work area. The grid is an information of the location of the point.
    • Adds points on the polyline.
    • Snaps the point on the polyline.

  2. In edition mode, use the contextual menu to:
    • Set a tangency
    • Invert a tangency direction
    • Remove or edit a point.

A Spline feature is created in the specification tree. It contains the spline elements.
