Drawing Production

You use Generative View Styles (GVS) to produce structural drawings. These view styles let you customize the appearance of drawings, using a set of parameters defined in an XML file.

Note: If you are a DS Passport customer, you can read the QA00000006067 article from the Knowledge Base for more information about view generation of the moved or multiple instantiated CATPart or CATProduct containing one or more structural objects.
