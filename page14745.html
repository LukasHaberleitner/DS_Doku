Adding Resolved Sections to Profile Families

This task explains how to add a directory containing resolved sections for a Profile family in a catalog.
1. Open the catalog and select the Profile family.
2. Click the Create/Modify Catalog button . The Create/Modify Catalog dialog box displays.
3. Select StrFunctionalDesign or Structure Design from the Application drop-down list.

For Parts Directory, open the directory where the resolved sections are stored.

4. Under Options, select Add All Components to the Selected Family.

Click OK.

The resolved sections for the selected Parts Directory display on the Reference tab.
