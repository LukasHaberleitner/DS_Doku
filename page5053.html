Batch Extraction of Workshop Documents

All workshop documents for a given process can be extracted at once with a single command, without any additional user interaction required. All extracted documents are saved at the location defined in Project Resource Management (PRM). Upon completion, a trace panel displays the results of the extraction process.

Before extracting a batch of workshop documents, you must have a process for which activities have been created with Build Activity List .

Extracting a Batch of Workshop Documents
 

  1. Select Batch Extraction of Workshop Documents .
    A dialog box displays the number of operations that workshop documents will be created for, and prompts for confirmation to proceed with the operation:

  2. Click Yes to begin extracting documents. A dialog box displays the progress of the operation, including which document is currently being processed and the estimated time remaining until complete:

  3. After all of the documents have been extracted, an information panel displays traces that were generated for each operation:

Extracted File Layout

Files generated by Batch Extraction of Workshop Documents are organized in sub-folders at the location specified by DpmStrWDSaveLocation in Project Resource Management. By default, this location is C:\temp.

The following folders are created:

  • PlateFabricationMacroActivity
  • ProfileFabricationMacroActivity
  • AssemblyMacroActivity

Under each of these sub-folders, additional folders are created based on each interim product. For example, if a profile fabrication sketch of BottShell_TStf_009.11 has been extracted, a folder named BottShell_TStf_009.11 (name of design interim product) will automatically be created under the ProfileFabricationMacroActivity folder and it will look like this.

C:\temp\ProfileFabricationMacroActivity\BottShell_TStf_009.11\Part1.9_ProfileIMCOperation.38_Profile fabrication sketch.CATDrawing

The same is also true for assembly level workshop documents. For example, a PinJig XML file for DECK_1 will be saved in the following location:

C:\Temp\AssemblyMacroActivity\DECK_1\Pinjigworkshopdocument.XML
