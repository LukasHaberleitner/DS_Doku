PspBuildPart (Object)

Represents Interface to create and modify a part.
Role: To build and define a part.
Method Index
ChangePartType
ChangePartTypeWithFeat Returns the part after changing the part type of an existing part.
ListPartParametricAttributes Returns the part parametric attribute names.
NewPart
