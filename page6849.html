Linking 2D Object with 3D Document

This task shows you how to link a schematic object with a physical device. It allows you to define the physical part type of a component. Associated commands allows you to select the physical part name and query the physical properties of a component. The following procedures were adopted in R11 and differ from those in previous releases.
