Running Commands

This task explains how to run commands from the power input box.

  1. To run a command from the power input box, enter a command like this:

    c:command_name

    where "command_name" is the name of the command as it appears in the menus.

    For example, enter the following command:

    c:New...

    or:

    c:New

    to run File > New.

  2. Press Enter to run the command.
    Note that if the command you want to run is not available in the current workbench, a warning message is displayed.

When pointing at icons, the syntax for running the associated command is displayed to the left of the power input box, to remind you that you can also run the command from the power input box. For example, when you point at New in the Standard toolbar, the following message is displayed:

c:New
