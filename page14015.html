Removing Profile Cards Display

  The main function of Profile Cards being to describe the general properties of documents, most of the time end users do not need to display them when saving their documents for the first time. To make end users gain time, system administrators therefore can now prevent the application from displaying profile cards that appear during first save operations.

To ensure that profile cards are not displayed during save operations, proceed as follows:

  1. Launch SmarTeam System Configurator.
    To know how to use this tool, see the SmarTeam documentation.

  2. Search for CATIA application.

  3. Set CATIA.Refresh_Screen to FALSE.

  4. Click the Save Changes button to confirm the changes.
