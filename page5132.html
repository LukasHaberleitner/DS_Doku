Editing PP Words

Reading Manufacturing data from a V4 model and importing APT source generates hundreds of PP (Post Processor) instructions and appends lots of PP words to machining operations. This task explains how to solve this problem by:
  • Merging Several PP instructions,
  • Merging Several PP Instructions In a Manufacturing Program
  • Editing PP Words inside a Tool Path.

Please note that:

  • After a merge, each line of a multi-line PP instruction will be merged into the new PP instruction as a separate line.
  • You can merge PP Instructions even if some of them contain more than one line of PP Words.
  • PP instructions with more than 100 lines can be processed.
    Depending on the number of lines and on the line length,  scrolling bars will appear in the dialog box.

Merging Several PP Instructions

  1. Select the PP instructions and choose Merge PP Instructions in their contextual menu:

    You may select only PP Instructions (the command will not work if you have selected an item that is not a PP Instruction).
    The PP Instructions selected do not need to be sequential.

  2. The selected PP instructions are merged into the first PP instruction.

Merging Several PP Instructions In a Manufacturing Program

  1. Select the Manufacturing Program and choose Merge PP Instructions in its contextual menu:

  2. The PP Instructions are merged:

Edit PP Words inside a Tool Path

  1. Select the tool path and click Edit Tool Path in the Tool Path Management toolbar.

  2. The Tool Path Editor dialog box is displayed:

    If you have selected only one object, the title Tool Path Editor is replaced with the name of the operation.

  3. Click PP Word Modification .
    The tool path is displayed with yellow dots and positions of PP words are displayed with white circles:

    Move your cursor on a white circle. The marker is highlighted and its syntax is displayed:

    The PP Word Modification dialog box is displayed:

    The buttons let you:

    • select the previous PP Word,
    • select the next PP Word,
    • insert a new PP Word in the tool path
      (select a point in the tool patch, click . The operating mode is then the same as for editing),
    • delete a PP Word (select the corresponding white circle and click ),
    • edit a PP Word.

  4. Click Edit .
    The Current PP Word dialog box is displayed:

    Click to access the PP table, when it exists.
    The PP Words Selection Assistant is displayed:

    Select the type of Major Words you require from the drop down list:

    Select a Major or a Minor word. It is displayed in the Current Selection field. Click Apply.
    The PP Words are displayed in the Current PP Word  dialog box.
    Click Close to revert to the Current PP Word  dialog box.
    Click OK to validate and revert to the PP Word Modification dialog box.

  5. Click OK to validate and exit the PP Word Modification dialog box.
