Uploading or Importing Fanuc RJ TPE Robot Programs

This procedure describes how to upload or import a Fanuc RJ TPE robot program.
Open the FanucRJUpload.CATProcess file.
Uploading reads a TPE (robot language) file and creates the corresponding activities in the PPR tree. The following steps take place:
  • The TPE file is read and an XML file is generated. The commands in the TPE file are mapped to elements in the XML file. The generated XML file is independent of the robot language.
  • The XML file is read and the required activities, actions, sequences or other items are generated as specified.

  1. Click Import Robot Program  in the Offline Programming toolbar.

  2. Select robot R2000iA_if-200F.1 in the PPR tree.
    A File Selection dialog box appears.

  3. In the File Selection dialog box, navigate to the tpetask3 program file and select Fanuc.jar.
    The Uploading Method dialog box opens.
     

  4. Select Robot Base Coordinates and click OK.

    An uploading progress bar displays the status of upload.

    Finally the Upload Information dialog box opens. This has information about the status of upload, unparsed statements, logical errors etc.

    Note: Daihen will be replaced by Fanuc RJ TPE, P001.ASC by tpetask3.ls and P002.ASC by tpetask2.ls in the above dialog box.

  5. Similarly, upload the task FanucRJ_BodyPickandPlace by selecting the resource R2000iA_if-165F.1.
