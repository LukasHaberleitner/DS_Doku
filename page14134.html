Creating a Detail to Result Link

This task shows you how to export the process concept geometry to a target part.
Open the part from Creating a Detail to Detail Link

  1. Click Link Detail to Result  in the Structure Definition toolbar.

  2. Select the target result part.
    It contains the dynamic copy of the process, including the detailed geometries from the detailed parts.

  3. Select the reference method plan
    It is used to replicate the process in the result part.

  4. Select all the concept to detail links used to design the detailed geometries in the detailed part.
    They are used to retrieve the detailed geometries and integrate them in the dynamic process copy.

  5. Optional: Select a symmetry plane.
    This selection is useful to design one method plan for a left part and create a fully symmetric method plan for the right part.
    The dynamic copy is the complete symmetry of the selected reference method plan.

A link feature is created in the target result part.

  • Its name is Link to<Reference Feature Name>
  • It is created under a geometrical set, with the same name as the reference method plan.

The link feature replicates the reference method plan in the result part.

  • It creates a copy with link of each axis and geometry used in the reference process.
  • It creates a new Process Body that includes a replica of each step and tool.
  • It also creates a replica of some required features:
    • Part Positioning feature
    • Scrap Partition feature
    • Blank Partition feature.

Those features guaranty that the XML export from the reference process part and the result part are the same (as long as no detailed geometry is designed). The process replica is thus the same as the one from the process part.

On top of this replica, the link analyzes each concept to detail link to retrieve the detailed geometries. Each detailed geometry found in the Detailed Geometry set of the tools set and custom geometries link is copied and associated with the corresponding step, tool, and custom geometry in the process replica. This way, the result part process is automatically filled with the detailed geometries when they are designed in the detailed parts.
