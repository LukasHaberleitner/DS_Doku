Renaming a View

This task will show you how to rename an existing view.
Open the GenDrafting_part_Rename_View.CATDrawing document.

  1. Right-click the frame of the view to be modified. A contextual menu is displayed.

  2. Select Properties. The Properties dialog box is displayed.

  3. Enter a new name in the View Name area. For example, enter B as identifier in the ID field.

  4. Click OK to validate and exit Properties dialog box.

    The view is renamed.
Note that a new identifier (B-B) is also assigned to the callout in the mother view.
