DMU Fastening Review in a Nutshell
The DMU Fastener Review application is a scalable product dedicated to the review of Automotive Body in White fasteners, created and managed by the Automotive Body in White Fastening P3 application.

The DMU Fastener Review application enables the visualization, edition, and checking of fasteners, but no modification can be performed. In addition, reports can be generated from the application in order to list data information in a neutral ASCII text format or HTML format.

If the ABF license is not available, the FAR license allows Automotive Body in White Fastening data to be displayed and edited but not modified.

The BiW flat reporting function can be run interactively with the FAR license only. This function cannot be run using the ABFBatchExReport utility with the same FAR license. In order to run the ABFBatchExReport utility an ABF license is required.

As a scalable product, the DMU Fastener Review application can be used in cooperation with other products such as DMU Review products.
