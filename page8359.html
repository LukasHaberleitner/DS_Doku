Using Predefined Postures  

This task describes how to display and use predefined manikin postures.
There are three predefined postures in the anthropometry module: Stand, Reach, and Span. These postures are used to display the associated variables. Each selection cancels the previous one.

The manikin's default posture is standing.
To display a predefined posture:
 

  1. Select the manikin.

  2. In the Anthropometry Editor toolbar, use the pull-down Postures toolbar to select a posture.

  3. You will see a different set of variables depending on the selected posture.
