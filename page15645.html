Saving a Rule Base in ENOVIA V5 VPM

This task explains how to save a rule base in ENOVIA V5 VPM. In this scenario, you:
  • Create a Product in ENOVIA V5 VPM and sends the Product to CATIA.
  • add components to the Product to create a saw
  • create Expert Checks.
  • save the Product in ENOVIA.
Make sure you have:
  •  selected ENOVIAV5 in the View > Toolbars  menu.
  • enabled ENOVIA in the Document Environments field (Tools>Options>General>Document.)
To perform this scenario, you will need the following files:
  • Bearing.CATPart
  • Case.CATPart
  • Cog_wheel.CATPart
  • Ring.CATPart
  • Slide.CATPart

  1. Create a Product in ENOVIA V5 VPM (in this scenario the product is named Expert). Note that you can close ENOVIA V5 VPM after creating the Product.

Send the Product to CATIA

  1. To send the product to CATIA:

    • In CATIA, click the Connect user to ENOVIA V5 icon ( ) in the ENOVIA V5 toolbar. The ENOVIA Logon dialog box is displayed:
    • Enter your user name, and your password.
    • Select a "role" and a workbook and click OK.
    • Click the Search ENOVIA icon (). The VPM Search dialog box is displayed.
    • In the Objects scrolling list, select the Product option.
    • Enter the Owner's ID.
    • Click OK when done. The Product ID is displayed in the Result window.
    • Double-click the product. The VPM Navigator view is displayed.
    • Right-click the root product (Expert) and select Open.... The Open Modes dialog box is displayed. Leave the default settings and click OK. The CATIA view is displayed.

Add Components to the Product

  1. Click the root product, select Insert>Existing Component... and select the following documents:

    • Ring.CATPart
    • Cog_wheel.CATPart
    • Bearing.CATPart
    • Slide.CATPart
    • Case.CATPart

Add Expert Checks to the Product

  1. From the Start>Knowledgeware menu, access the Knowledge Expert workbench.

  2. Click the Expert Check icon (). Enter the name of the check (Hole_Check) in the Check Editor and click OK.

  3. Enter the following check, and click OK when done.

    H:Hole
    (H.Diameter == 11.0 mm) AND (H.Activity == TRUE)

  4. Click the Expert Check icon (). Enter the name of the check (Fillet_Check) in the Check Editor and click OK.

  5. Enter the following check and click OK when done.

    H:ConstantEdgeFillet
    H.Radius >= 2mm

  6. Click the Expert Check icon (). Enter the name of the check (Pocket_Check) in the Check Editor and click OK.

  7. Enter the following check, and click OK when done.

    P:Pocket
    P.FirstLength >= 20mm

  8. Right-click the default technological package and select Properties. Enter a new name: Expert_TP. Click OK when done.

  9. Right-click the Rulebase and select RuleBase object>Manual Complete Solve.

Save the Product in ENOVIA

  1. Click the Save in ENOVIA V5 icon (). The Save in ENOVIA dialog box is displayed. Click OK. The Product and the technological package containing the Rule Base are saved in ENOVIA.
