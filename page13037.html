Creating Weld Information on Faces

This task explains how to create weld information on the free faces of the structural objects.
In the shipbuilding industries, different design units are combined to create a huge structure. In such case, it becomes difficult to predict how the structural objects will be welded with another structural object. This command creates weld information on the faces of the structural objects which can be used during manufacturing processes. The weld information is set on the lateral faces of the plate and the end faces of the profiles in Structure Detailed Design and is transferred to the corresponding free lateral faces of the plate and end faces of the shapes in Structure Design respectively.
  • Set the StrWeldPreparationSample  file in the Project Resource Management for Structure Detailed Design workbench. The sample file is located at ..\..\startup\EquipmentAndSystems\Structure\StructuralCatalogs\Weld. This resource is available only in Structure Detailed Design.
  • Open a structure containing plates, transverse bulkheads, seamed plates, stiffeners.

  1. Click Start > Equipment & Systems > Structure Discipline > Structure Detail Object Design.
    You are switched to Structure Detail Object Design workbench.

  2. Click Weld Information on Face  in Detailing Tools toolbar.
    The Weld  Information on Face dialog box appears.

  3. Select the elements for which you have to generate the weld information on faces.
    You can select the elements from different design units.
    The name of the selected element is displayed in the Selected Elements box. If there are more than one element, the number of selected elements appear.
    The free lateral faces of the plates and free faces of the profiles are displayed in the dialog box.


    The Operated Element column lists the selected elements. The Face Name list displays the faces of the selected element. The Weld Status lists the status if the weld information is created or not.
    If the status is set as OK, the weld information exist on the face. If it is Unset, the information needs to be created.

  4. Select one or more rows in the dialog box and click Auto Weld.
    The weld status is changed to OK. The weld information is created. The weld information is generated from the text file assigned in Project Resource Management. The thickness or section of the operated element is evaluated and the weld information is defined accordingly.

    If the thickness or section of the operated element is not listed in the text file, the following error message appears in the Weld Status column when you click Auto Weld.
    Value not found: Check PRM definition.
    Check the text file assigned in StrWeldPreparationSample resource to refer thickness or section of the operated elements for which welds can be created.

  5. To view the weld information for face, right-click in any row and select Define Weld.
    The Weld Face Parameters dialog box appears displaying the information related to welded elements.

    You can edit the margin and edge preparation value. These values are predefined in the text file assigned in the StrWeldPreparationSample  resource in the Project Resource Management.

  6. Click OK to close the Weld Face Parameters dialog box.

  7. If you want to delete weld information for any faces, select the required rows and click Delete Weld.
    The weld face parameters are deleted.

  8. Click OK in the Weld Information on Face dialog box.
    The weld information is stored on the free faces of the plates and profiles. A feature, WeldSet, is created in the specification tree. It carries the information related to weld faces. If the selected elements belong to different design unit, then a weld set is created for each design unit.

    • For profiles, the weld information is stored on the free end faces only if the end cuts applied at the extremity are Short Point, Long Point, Weld and Metal to Metal.
    • During the piece part generation, ensure that you select Add Weld Information to Coping check box in the Options dialog box to transfer the weld information for each structural objects to the corresponding object in Structure Design workbench. For More information, refer to Structure Design User's Guide: Piece Parts: Generating Piece Parts.
    • You can place the annotations (added material) and a weld symbol on the weld feature (for plate, stiffener and beam) in the 2D view. For more information, see Graphic Replacement of Plates and Annotation Text Style.
