Flipping Direction (Normal) on Multiple Features

This procedure describes how to flip direction (sometimes called "normal") on multiple features simultaneously.
This procedure assumes that you have created features.

  1. Click Flip Feature Normal.

    The arrows for normal appear in green on the geometry; the Tools Palette and the Flip Feature Normal dialog box appear.
    All the arrows represent features visible on the geometry from the current view; more features may be selected than are visible.

  2. Use one or more of the tools in the Tool Palette to select the features.

  3. Select the Finish button.

    The arrows showing normal turn purple; the Flip Measurement Direction check box becomes available.

  4. Select the Flip  Normal check box, and click OK.

    The selected features' normal arrows turn black.
Respective measurement path, construct features and alignment will also get updated.
