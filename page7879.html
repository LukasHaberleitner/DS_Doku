Connection Summary

This task shows you how to visualize all analyses or mesh parts connected to a particular selection and the connections (using a hide / show system).

The connection summary command lists all meshes connected to a selection.
For example: if you select an empty connection mesh (this means that this connection does not connect anything) then the list in the connection summary will be empty.

A reporting tool is also available in an assembled analysis context or in a basic analysis on part.

You can use the connection summary functionality with all connection types.

The connection summary functionality is not available in the Surface Mesher and the Advanced Surface Mesher workshops.

Open the sample32.CATAnalysis document from the samples directory.
In this particular example, you will only use nodes to nodes connections.

  1. Click Connection Summary in the Mesh Analysis Tools toolbar.

    The Connection Summary dialog box appears.

    • Summary by: lets you choose if you want to work in Analysis context or in a Mesh Part context.

    • Selection: lets you select an analysis or a mesh part (connection or not) either in the 3D viewer or in the specification tree.

      If you select the Selection field, the selection is highlighted both in the specification tree and the 3D visualization.

    • List:

      • Analysis connected to the selection / Mesh part connected to the selection: lets you see which analysis (or mesh part) is connected to the selected analysis (or mesh part).

        If you select a mesh part while you selected Analysis as Summary by option, the system will automatically select the analysis associated to this mesh part.

      • Selection connected by: lets you see which connection is used to connect the selected analysis (or mesh part) and the analysis (or mesh part) connected to the selection.

    • More Information: gives you global and specific information on the connection mesh part.

      This button is available when only one line of the list is selected.

    • Show All Meshes of the List: lets you display all the mesh parts contained in the list.

    • Show All Meshes: lets you display all the mesh parts (contained or not in the list).

    • Save All in HTML File: lets you generate and save an HTML file containing information on all the entities of the list.

    • Close: if you click this button a warning message appears. You can decide to keep the current 3D visualization or you can decide to retrieve the initial 3D visualization (state of the document before launching this command).

      Any modification of the document (such as entering in a sub analysis) before launching the Connection Summary command will invalidate the document.

  2. Select Analysis as Summary by option.

  3. Select Analysis Manager.1 in the specification tree (under ProductPointPoint).

    A list is displayed in the Connection Summary dialog box.

    Note that the 3D visualization has changed: the geometry is hidden and only the mesh parts of the selected analysis and the connection mesh parts are still displayed:

    You can select several lines in the displayed list.

  4. Select the third line of the list in the Connection Summary dialog box.

    The 3D visualization is modified: only the entities of the third line are displayed.

  5. Click More Information.

    The Connection Summary dialog box appears.

    This dialog box gives you global and specific information on the connection mesh part.

    If you select the Connection mesh part and the Connected mesh part fields, the selection is highlighted both in the specification tree and the 3D visualization.

  6. Click Close.

  7. Right-click the third line of the list.

    You access the following contextual menu.

    • Node Selection: lets you select a node and gives you all the mesh parts or analyses connected to this node (even if they are hidden).
      Multi selection of lines is not available.

      The Node Selection dialog box appears.

      • Select a node in the 3D visualization.

        The Node Selection dialog box is updated:
      • Click Close.

    • Reframe On Connected Analysis: lets you reframe on the connected analysis both in the 3D visualization and in the specification tree.
      Multi selection of lines is not available.

    • Reframe On Connection: lets you reframe on the connection both in the 3D visualization and in the specification tree.
      Multi selection of lines is not available.

    • Center Graph On Connected Analysis: lets you see the connected analysis in the specification tree (if needed the specification tree is expanded and the connection is highlighted).
      Multi selection of lines is not available.

    • Center Graph On Connection: lets you see the connection in the specification tree (if needed the specification tree is expanded and the connection is highlighted).
      Multi selection of lines is not available.

    • Hide Connected Analysis: lets you hide the connected analysis in the 3D visualization.
      Multi selection of lines is available.

    • Hide Connection: lets you hide the connection in the 3D visualization.
      Multi selection of lines is available.

    • Show Connected Analysis: lets you show the connected analysis in the 3D visualization.
      Multi selection of lines is available.

    • Show Connection: lets you show the connection in the 3D visualization.
      Multi selection of lines is available.

    • Save Selection in HTML File: lets you generate and save an HTML file containing information on the entities of the selected line.
      Multi selection of lines is available.

  8. Click Show All Meshes of the List.

    You retrieve the 3D visualization of all the mesh parts contained in the displayed list.

  9. Click Show All Meshes.

    The 3D visualization changes again: all the mesh parts (contained or not in the list) are displayed.

  10. Select Mesh Part as Summary by option.

  11. Select a mesh part.

  12. Select a line in the displayed list.

  13. Click Save All in HTML File.

  14. Choose the directory and the title of the report file.

  15. Click OK.

    The .html report is automatically displayed.

  16. Click Close in the Connection Summary dialog box.

    A warning message appears to inform you that you can keep the current 3D visualization (Yes button) or you can decide to retrieve the initial 3D visualization (No button).

  17. Click No.
