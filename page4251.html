Activating a Mode

This procedure describes how to set the CMM to manual mode or to computer numerical control (CNC) mode.  

Typically, activities such as defining and measuring features and defining and activating alignments are done in manual mode.  CNC mode is turned on to inspect features.

Setting Manual Mode
 

  1. Select the process. 

  2. Click Activate Manual Mode from the Inspection Activities toolbar.   

    The Activate Manual Mode activity appears on the PPR tree.

Setting CNC mode
 

  1. Select the process. 

  2. Click Activate CNC Mode command from the Inspection Activities toolbar.   

    The Activate CNC Mode activity appears on the PPR tree.
