Export

Please refer to Infrastructure user's guide to know more about the Compatibility Export options.

3D annotation
You can select this option to save the 3D annotations in a 3D XML file.
By default, this option is selected for Dynamic Tessellation and Static Tessellation.
This option is inactive in case of XML Tessellation.
