Creating Polygons

This task shows you how to create a polygon. You can create a polygon with number of sides ranging from three to 24. The default number of sides is six. A polygon can be drawn by selecting a center of either incircle or circumcircle of the polygon.

  1. Click Polygon in the Profiles toolbar (Predefined Profile sub-toolbar).



    The Sketch tools toolbar now displays the values of coordinates of the polygon center and the type of the construction circle for sketching the polygon.

  2. Select the type of construction circle for sketching the polygon.

    • InCircle: The point lies on mid-point of one of the edges of the polygon. This is the default selection.
    • Circumcircle: The point lies on one of the vertices of the polygon.

  3. In the Sketch tools toolbar, enter the required values.

    1. Enter the horizontal or vertical coordinates of the center of the polygon.
    2. Enter the radius and the angle of the polygon at which the point lies with respect to positive horizontal axis.
    3. To create a hexagon, keep the Number of Sides locked at the default value of six and proceed to Step 4.
    4. To create a polygon with number of sides more or less than six, click As default number of sides  and, in the Number of sides box, enter the required number of sides.
      By default, As default number of sides    is locked.

  4. The polygon is created.

    Point on Circle: In Circle with six sides (default selection)  Point on Circle: Circum Circle with six sides 
       
  • You can also define the coordinates of the center of polygon and the point on the circle by clicking on the required points in the work area.
  • The number of sides can be directly varied by moving the mouse in clockwise and anti-clockwise direction to increase and decrease the number respectively.
  • The type of construction circle and number of sides are saved as a user preference.
