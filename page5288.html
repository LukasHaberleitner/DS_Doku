Customizing

 

Before you start generating a Ply Book you can customize the way it will be generated.

  1. Select the Tools > Options command. The Options dialog box displays.

  2. Choose the Mechanical Design category in the  left-hand box, then Composites Design.


    The Composites Design tab deals with Composites Design general options.
    The Export Ply Data tab deals with export of ply data to DXF, IGES 2D or OGES 3D.
    The Laser Projection tab that defines export and preview options for laser projections.

  1. Set options in these tabs according to your needs.

  2. Click OK when done.
