Before You Begin

You can modify the properties of a group of views of different kinds (interactive, generative or detail views), either on a single sheet or on different sheets. These modifications will be applied to the properties values common to the selected views. In other words, when you edit the properties values of different views, only the properties values of the views with the same parameters will be modifiable. Still, mixed properties values (that is, values which are not common to the selected views) will be modifiable.

Note that you can select the views from different sheets.
Open the GenDrafting_part_Move_View02.CATDrawing document.

  1. Multi-select views from the specification tree (you can use the Shift or the Ctrl key). In this particular case, select all the views.

  2. Select Edit > Properties from the menu bar.

    The Properties dialog box is displayed.

  3. In the Visualization and Behavior area, clear the Display View Frame check box.

  4. Click Apply to validate.

    The view frame is removed.

  5. Multi-select views from the specification tree. In this particular case, select the top, bottom, left and right views.

  6. In the Dress-up area, select the Hidden Lines check box.

  7. Click OK to validate and exit the Properties dialog box.

    Hidden lines are displayed.
  • If the view name is not common, no value is displayed in the View name and Name Editor With Formula fields.
  • If the view scale is not common, no value is displayed in the Scale field.
  • In the case of mixed values, the modifiable fields appear white and empty.
  • If one view is locked, mixed properties values will also be locked.
If you modify one view (view A), prior to selecting another view (view B), click the Apply switch button if you do not want to lose the modifications first applied to view A.
