General

The General tab lets you define the default end cut settings for the beam, stiffener and stiffener on free edge.

The Default End Cut Settings area displays a list of structural objects on which you can define the default end cut settings.
  • Beam
  • Stiffener
  • Stiffener On Free Edge.


For these structural objects, you can define the end cut type such as Short Point, Long Point, Weld or Metal To Metal.
By default, the end cut type is set as Short Point.
