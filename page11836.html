Component Manipulation Restrictions

  The following information concerns all workbenches that you may use while in a Mold Tooling work session.

There is a certain number of restrictions concerning the manipulation of components and of drilled CATParts.
Certain actions should be completely avoided. Failure to do so can produce serious performance problems and inconsistent links.

These actions are:

  • Copy/Paste (whether accessed via the contextual menu or the toolbar menu),
  • Copy/Paste special (whether accessed via the contextual menu or the toolbar menu),
  • Cut/Paste (whether accessed via the contextual menu or the toolbar menu),
  • Drag/Drop,
  • Delete (whether accessed via the contextual menu, the toolbar menu or the Delete key on your keyboard).

 

Instead of the above functions, you should used the:

  • Delete Component,
  • Add New Instance,
  • Isolate Component,
  • and Replace Reference menu items

in the contextual pop-up menu that is specific to the component that you are manipulating (also in the contextual menu in the menu bar).
Likewise, Replace Component... and Replace component in Session... should not be used (whether accessed via the contextual menu or the toolbar menu).
They too can produce serious performance problems and inconsistent links.
You cannot insert components into a flexible assembly.
