Drawing Management

The Generative Drafting workbench lets you manage CATDrawing documents.

 

Create a new drawing: create a CATDrawing document.
Open a drawing: open a CATDrawing document.
Edit drawing links: edit the links which exist from a CATDrawing document to an existing CATPart document, CATProduct document, sheet metal part or a .model V4/V5 document.
Update drawings via the batch monitor: update a list of CATDrawing documents using the batch monitor.
Updating Drawings using PLMBatchDrawingUpdate Via the Batch Monitor: update a list of CATDrawing documents using the PLM batch monitor.
Save as: export a CATDrawing to External Formats
Importing from files: import other formats into a CATDrawing
