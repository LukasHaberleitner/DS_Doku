Cache Activation

For more information, refer to Cache Settings.

Work with the cache system

Activating the cache allows you to work in visualization mode, rather than in design mode and may prove useful when working with Electrical Wire Routing.

In visualization mode, a representation of the geometry only is available and the corresponding cgr file is inserted from the cache system.
Using a cache system considerably reduces the time required to load your data.

By default, this option is cleared.

When the routing is completed, come back to the design mode .
To do so:
  • Turn the cache activation mode off
  • Select Edit > Representations >  Design Mode

  • If need be, update the geometry using Update .
