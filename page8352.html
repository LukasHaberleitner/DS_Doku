Editing the Anthropometric Variables

This task describes how to display and modify all the anthropometric variables.
The manikin should already be displayed with its anthropometry.

  1. Select the Display Variables from the Anthropometry Editor toolbar.

  2. The Variable Edition dialog box is displayed.

  3. Change the Variable from Automatic to Manual.

  4. Change the Value to the required percentage.

  5. The selected variable will appear in the 3D viewer and its color will change.
