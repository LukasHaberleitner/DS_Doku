Copying and Pasting Tracks
Right-click the track to be copied. Select Copy, then right-click the tracks item in the specification tree, and select Paste . Double-click the pasted track to change the object selection.
Analyzing in Track Context
Redefine a track, click Activate Analysis button in the Track dialog box, select analyses from the Analysis list, edit analyses, and review analysis data.
Associating Sections with Tracks
Create one or more sections of the data; create a track; attach the sections to the track.
Exporting and Importing Tracks in Neutral Format
Select the track and then Tools > Simulation > Track File Export or Tools > Simulation > Track File Import.  Enter a name and select .xml format from the Save as Type list.
Editing Time Line in Tracks
Select the track to be edited, click More, and either drag and drop the segment or enter a precise value in the shot time field.
