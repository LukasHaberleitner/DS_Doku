Optimal CATIA PLM Usability for Functional Tolerancing and Annotation

When working with ENOVIA V5, the safe save mode ensures that you only create data in CATIA V5 that can be correctly saved in ENOVIA V5.
  ENOVIA V5 offers two different storage modes: Workpackage (Document kept - Publications Exposed) and Explode (Document not kept). Functional Tolerancing & Annotation has been configured to work in the Workpackage mode only.

Functional Tolerancing and Annotation Commands in Enovia V5
  All the Functional Tolerancing and Annotation commands are available with the Workpackage mode in ENOVIA V5 LCA and VPM.
