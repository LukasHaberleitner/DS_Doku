Uploading or Importing Panasonic CSR Robot Programs

This procedure describes how to upload or import a Panasonic CSR robot program.
Open the PAN_Upload.CATProcess file.
Uploading reads a Panasonic CSR robot program file and creates the corresponding activities in the PPR tree. The following steps take place:
  • The Panasonic CSR robot program is read and an XML file is generated. The commands in the Panasonic file are mapped to elements in the XML file. The generated XML file is independent of the robot language.
  • The XML file is read and the required activities, actions, sequences or other items are generated as specified.

Uploading Panasonic CSR program
 

  1. Click Import Robot Program  in the Offline Programming toolbar.

  2. Select robot TA-1400.1 in the PPR tree.
    A File Selection dialog box appears.

  3. In the File Selection dialog box, navigate to the desired program file, WELD_L86_101.csr, included in the archive and select Panasonic.jar as upload translator.
    The Uploading Method dialog box opens.
     

  4. Select Robot Base Coordinates and click OK.

  5. Click Robot Task Simulation  in the Simulation toolbar, and select robot TA-1400.1.

    The Select dialog box appears.

  6. Choose the robot task as shown above and click OK.
    The Process Simulation toolbar appears.

  7. Click Run in the Process Simulation toolbar to simulate the process.
