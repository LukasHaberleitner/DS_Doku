An Overview of Abaqus for CATIA V5

Abaqus for CATIA V5 is a modeling and graphical user interface extension to CATIA V5 that provides access to advanced simulation capabilities within the CATIA V5 environment. Abaqus for CATIA V5 consists of two workbenches that are added to the CATIA V5 environment: the Nonlinear Structural Analysis workbench and the Thermal Analysis workbench. In these workbenches you can reuse familiar CATIA features, such as contact connections, and enhance them with general analysis capabilities, such as highly accurate interface friction. In Abaqus for CATIA V5 you create a .CATAnalysis document from which nonlinear simulations can be performed. The Abaqus solver performs the analysis and generates an output database. Finally, you can view the results in Abaqus for CATIA V5. Refer to Installation Requirements for a list of the CATIA and Abaqus products that are required to perform these nonlinear simulations.

Abaqus for CATIA V5 makes extensive use of the CAD-CAE associativity concept. Associativity means that any part modifications you make outside the Abaqus for CATIA V5 workbenches are reflected automatically when you perform tasks within the Nonlinear Structural Analysis and Thermal Analysis workbenches. In particular, any parametric changes to the parts are reflected, so there is no need to update the part specifications within the workbenches.
