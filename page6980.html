Placing and Modifying Equipment

This section discusses placing and modifying equipment.

Efficient Equipment Placement

Equipment and parts placement should take 1.5 to 3 seconds. Placement performance depends on the way reference parts and catalogs are managed. Some recommendations for more efficient placement are listed below:

  • Work with fully resolved parts. Remove all data from the part that is not needed, such as parametric definitions and geometry specifications.

  • Use smaller sub-catalogs. A sub-catalog should be used for each family (e.g. gate valve, ball valve, elbow, etc.)

  • Make sure that in a catalog of resolved parts, the value of the keyword ReferenceStatus is set to FullyResolved.

  • When creating a catalog using the Create/Modify Catalog command, make sure that the option Reference document preview is selected in the Preview section. This will reduce the number of image files that are associated to the catalog.

  • Clear (un-select) the following option Check for sufficient space during parts placement on runs. To do so, click Tools - Options in the menu bar, select Equipment & Systems and the Design Criteria tab.
