Slicing a Mesh into Several Sections

This task shows how to slice a mesh by several other meshes.

The sectioning meshes have to split the initial mesh in two parts, i.e. for each section the intersection line must be closed.
Open the MultiSection01.CATPart document.
 

  1. Click Multi-section Slice .
    The Multi-section Slice dialog box is displayed:

  2. Select Main.1 as the mesh To Slice.
    Select Section.1, Section.2 and Section.3 as the slicing Sections.
    Multi-selection is available.
    is available to hide or show the selection.

  3. Click Apply. The slices are computed and displayed in different colors.

  4. Click OK to validate and exit the dialog box.
    A single MultiSlice feature is created in the specification tree.

  5. If necessary, right-click the MultiSlice feature and select Properties in the contextual menu,
    then go to the Multi-Slice tab of the Properties dialog box to edit the name or color of the cells of the MultiSlice:
