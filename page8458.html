Route a Run Within a Pathway

This task shows you how to route a run within a pathway.
1. With your pathway document open, click the Route Thru a Pathway button.
2. From the Section dialog box, select the type of run, the set point and other options.
3. Click on the pathway in which you want to route your run. This displays set points on the pathway. Select a position for the run by clicking on one of the points. For example, if you select Top Center the run will align to the top center of the pathway. You can click Apply in the Run dialog box to see how the run looks and to try different positions. Click OK when you are finished.
