Connecting to the SmarTeam Database  

Your first task consists in connecting to the SmarTeam Database.

  1. Launch a CATIA session.

  2. Ensure that the settings dealing with linked document localization strategies as well as the display of the SmarTeam File Open User Interface are set to define the application's behavior we want.

  3. From the SmarTeam toolbar, click Connect .

    or select SmarTeam > Connect.

    The SmarTeam User Login window appears:

  4. For the purpose of our scenario, enter admin as the user name, without any password.

  5. Click OK.
    Now you are connected, go on to the next task to access the part to be edited.
