Routing Wire

This task explains how to route wire in a wiring diagram.

In order to complete this procedure you must have created a wiring diagram as described in Creating a Wire Level View of a Cable Diagram.

  1. Open a document containing a wire level view of a cable diagram:

  2. In your wire level view document, select the cable in the specifications tree associated with the wire that you want to route.

  3. Click the Route Wire button.

    The Route Line dialog box displays:


     

  4. Select the route mode you want to use:
     

    • Horizontal/Vertical : You can only route in horizontal or vertical segments.
    • Horizontal/Vertical/45 degrees : You can route in segments that are horizontal, vertical or at a 45-degree angle.
    • Point-to-point : You can route in any direction.
    • Single step : The wire is routed automatically.

  5. Select the wire you want to route from the list of wires.

    Depending upon which route mode you are using, the wire is routed in one step or you are prompted to select the start and end points of your route. The following diagram shows all three wires routed:



    The ellipse represents the cable containing the three wires.
