Using Quick View

In order to allow users to view specific items in their geometric data, Inspection offers a set of quick view commands that turn on or off specific features.  Each command serves as a toggle: if the feature currently is not shown in the geometry, clicking on the command will make it visible and vice versa. 

When defining a group point feature, irrespective of the current view setting, the view mode is temporarily changed to Feature Normal View. The view is restored to its original value once the group point feature has been defined. This is done so that the user can also see if the normal of the generated point is proper, and can flip it using the Flip Normal option, if required.
