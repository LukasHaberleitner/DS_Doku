Re-establishing the Link with a Process Library File

It may happen on occasion that someone (either intentionally or unintentionally) moves a Process Library file from the location where it was originally saved. When this happens, opening any CATProcess document which uses that Process Library will display a notification message like the following:

Note that if the Process Library file has been physically deleted from the disk (rather than just moved to another location as described above), the same message will pop-up for the CATProcess document but there will be no way to reroute the link to another existing library -- the Open desk will not help in this context.

Finding a Library File that has been moved
Click on the Desk button in the dialog window, then select the "Find" menu item:

Using this method, the link to the proper Process Library file can be re-established. The CATProcess document must be saved (to save the new link(s) inside the Process document) and subsequently reopened to see all Activities in the PPR Tree.
