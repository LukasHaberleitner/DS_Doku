Sweep Roughing the Part

This task will show you how to rough machine the whole part using a sweep roughing operation.

  1. Click Manufacturing view  .

  2. Select Whole from the list of areas.

  1. Click Sweep Roughing .

  2. Go to the tool tab and select tool T2 End Mill D20.

  1. Click Tool Path Replay to compute the operation.
    The toolpath is displayed on the part.
    Click OK to close the small dialog box that is displayed (bottom right).

  2. Click OK to close the operation dialog box.

  3. Now you are going to machine the Top surface.
