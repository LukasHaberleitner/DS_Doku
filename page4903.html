Using Simulation Trace

This procedure describes how to use the simulation trace command and its associated options.
To follow this procedure, open SimTrace.CATProcess.

  1. Click Simulation Trace .

  2. Select the object for which you wish to create the trace.

    The compass attaches itself to the base location of the object.
    The Simulation Trace dialog box appears:
    By default, the Delete previous trace when simulation starts again is enabled as a result of the option being selected on the Simulation Trace tab on the Tools > Options > DELMIA Infrastructure page.  How you set the option on this dialog box will override the setting in Tools > Options.

  3. Click Add a trace button.

    The trace appears on the List of Simulation trace(s):

  4. Click Process Simulation .

    The Simulation Trace dialog box disappears

  5. Click Run to see the simulation.

    The trace appears on the geometry. Note that the track image (which is created by the Move Activity command) is the black line; the simulation trace (which runs from the base location) is in red.  The color of the line is controlled by the Simulation Trace tab on the Tools > Options > DELMIA Infrastructure page. This line appears in red, for better contrast; the default value is white.

  6. Run the mouse over the trace.

    Data about the trace appears.  The default data appears below: the length from the starting point, the time elapsed in the simulation when the simulation is at that point and the name of the point. The other data you may select is controlled by the Simulation Trace tab on the Tools > Options > DELMIA Infrastructure page.
    If you select the trace with your right mouse button, a context menu appears, which allows you to perform many of the functions available on the Simulation Trace dialog box.
     

  7. Click Simulation Trace again, and when the dialog box opens, select the previous trace.

    The Simulation Trace options appear highlighted.

  8. Click Hide/Show Trace , then click it again.

    The trace disappears then reappears on the geometry.

  9. Click Export Trace .

    A Save As dialog box appears.

  10. Navigate to the desired directory and save the trace as TraceCam1.1.xls.

  11. Click Delete Trace .

    The trace disappears on the geometry and from the dialog box.

  12. Click Import Trace.

    A File Selection dialog box appears.

  13. Navigate to the directory in which you stored TraceCam1.1.xls and select the Open button.

  14. Select the Add a trace button.

    The trace reappears in the Simulation Trace dialog box.

  15. Deselect Activate Trace.

  16. Run Process Simulation to verify that no trace appears.

  17. Click Simulation Trace

  18. Select the Cam.

  19. Reselect Activate Trace option, then select the Offset button.

    The Offset dialog box appears.

  20. Use the buttons on Mode to select the offset from the base location.

  21. Select the location so that the compass snaps to it.

  22. Select the Add a trace button, and then run Process Simulation to observe the new trace.
