Managing Hyperlinks Associated with an Activity  

This procedure describes how to add to a hyperlink to an activity. In addition, you can remove a hyperlink or edit one.
 

Add Hyperlink  
 

  1. On the PPR tree, select the activity to which you are adding the hyperlink.

  2. Click Add Hyperlink .

    The Manage Hyperlink dialog box appears:

  3. Click the Browse button to select the hyperlink, or paste or enter a hyperlink; then click the OK button.
 

Edit Hyperlink

 

  1. Click the activity that has a hyperlink associated with it.

  2. Click Add Hyperlink .

    The Manage Hyperlink dialog box appears:

  3. Click the hyperlink listed in the Link to File or URL area.

    The Goto, Remove, and Edit buttons become available;

  4. Click the Edit button; the URL moves up to the URL field in the Definition area.

  5. Modify the URL.  Use the Browse button if necessary.

  6. Click OK.
 

Remove Hyperlink

 

  1. Click the activity that has a hyperlink associated with it.

  2. Click Add Hyperlink .

    The Manage Hyperlink dialog box appears:

  3. Click the hyperlink listed in the Link to File or URL area.

    The Goto, Remove, and Edit buttons become available;

  4. Click the Remove button; the URL is erased.

  5. Click OK.
