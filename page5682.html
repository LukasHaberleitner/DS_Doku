Resetting Component Position

 

This task will show you how to reset and store the position of all or selected elements.
 
Open the Move.CATProduct document.

  1. Move some of the elements using the 3D Compass (see Moving Objects Using the 3D Compass in the Infrastructure User Guide).

  1. To reset the position of all elements that have been moved, click the Reset Position icon .

    All elements are reset to their initial position.

  1. To reset the position of selected elements only, first select the elements, then click the Reset Position icon .

    The selected elements are reset to their initial position.

  2. To set the new position of the product as a initial position, select the elements, then click Store Current Position  icon .
    The new position is stored. After this if you change the position and use Reset Component Position command, then the product position will be reset to stored position and not the initial position.
In the above, the term initial position signifies the following:
  • for an existing document - the position of a product when the document was opened
  • for an inserted document - the position of a product when the document was inserted
  • for a new document - the position of a product when the document was created
