Action Script Structure

An action script is written in text format and is organized in blocks consisting of related sets of statements. 
A block consists in an instruction designed to create an object followed by a set of statements surrounded by braces ({ }).  Statement blocks can be nested and the most enclosing one within a script corresponds to the document creation.  

A document is made up of a hierarchy containing objects, their properties and the features they own. An action script reflects this object hierarchy.
