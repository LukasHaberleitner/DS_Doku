Creating Annotations 

  About Welding Symbols: lists the available welding symbols for AWS, ISO and JIS standards.
Create Weld Features: click this icon, select a geometrical element which represents the weld.
  Create a Text: click the icon, select a face and enter your text in the dialog box.
  Create a Flag Note: click the icon, select the object you want to represent the hyperlink, enter a name for the hyperlink and the path to the destination file.
