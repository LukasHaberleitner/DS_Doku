Annotation Set Structure in Specification Tree

The Annotation Set is organized using sub-node by type elements in the specification tree and regrouping elements by their type: view, text, etc.
  Available sub-node by type elements in alphabetical order are:
  • Captures
  • Datums
  • Deviations
  • Dimensions
  • Distances
  • Geometrical Tolerances
  • Notes
  • Reference Frames
  • Restricted Areas
  • Surface Texture
  • Views
  • Welds
  These sub-node by type elements appear in the specification tree when:
  • creating a type of element never created before.
  • opening an existing document by parsing by types its elements.

These sub-node by type elements are temporary and exist only during the document session.
As a consequence, they are never saved in the document, in the same way the Hide/Show status on these elements is available during the document session only.
