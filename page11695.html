Using Simulation Commands

The files in the list below provide supplementary information about using simulation commands within NC Machine Tool Simulation.  Performing cycle time validations in Machining Tool Path Simulation is done in the same manner as in other simulation products in the V5 family. In addition, when using Machine Simulation, you should be familiar with the options available with the Process Simulation command, most of which are available with Machine Simulation.  For a description of the differences between the two commands, see Simulating NC Tool Paths.
