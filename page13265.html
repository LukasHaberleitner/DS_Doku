System Creation

This task explains creation of a structure functional system and is explained in more detail in Creating a System. This system will be the repository for structure functional design objects.

 

 

 1. Open the Structure Functional Object Design workbench, and click the Structure System button .

The Structure System dialog box displays.

.

2.

 Click OK to create the system. In the image below, the system entry in the specifications tree has been expanded to show all its contents.
