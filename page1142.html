Customizing

Before starting your first working session, you can set options needed to run the product or customize the way you work to suit your habits. This type of customization is stored in permanent settings files: these settings will not be lost if you end your session.

  1. From the menu bar, select Tools > Options. The Options dialog box appears.

  2. On the navigation bar on left of the dialog box, expand the Digital Process for Manufacturing tree and select the DPM - Work Instructions workbench.

    The tabs for the workbench appear:

  3. These tabs allow you to define the following options:
    • General
      Select the General tab to specify a style sheet to be used for Work Instruction formatting.
    • Plug-Map View Settings
    • Select Plug-Map View Settings choose which logical data and termination list items to be included in the data report tables.
