Slider Property

Slider property is a physical property assigned to a connection between two parts.

The relative translation of the secondary node with respect to the main nodes set is orthogonal to the direction joining the secondary node to the set of main nodes.

The input and output characteristics are:

  • Input:
    • Direction (optional)
    • Local Axis (optional)
  • Output:
    • Point force vector

The output characteristics can be expressed at the given positions in the element and in different axis system:
