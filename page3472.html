Transferring Files

Files can be transferred between users during the conference. This enables you to perform conferencing on VPM data, as one member of a conference can transfer the necessary PSN file that will enable another member of the conference to visualize the same ENOVIAVPM data. 

Note: Both users must have access to a VPM database.
 

In the Send area of the File Transfer page, the Sender:

  1. Clicks on the To selection button and chooses the intended recipient from the proposed list of conference members.

  2. Enters the file name in the File text-entry field or clicks the Browse button to select the file by browsing his directories.

  3. Presses the Send button to send the file to the intended recipients.

 
 

 

In the Receive area of the File Transfer page, each Recipient:

  1. In order to specify the location at which he wishes to place the sent file, he enters the directory name in the Directory: text-entry field or clicks the Browse button to select the directory by browsing his directories.

  2. Presses the Accept button to accept the file at the specified location.
