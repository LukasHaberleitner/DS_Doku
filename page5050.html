Extracting an Assembly Drawing

This task shows you how to extract an assembly drawing. An assembly drawing is a workshop document that is associated to an assembly operation, and shows an isometric view of the interim products to be assembled at the stage of the selected operation.

  1. Select Workshop Documents Extraction .
    The Workshop Documents Extraction dialog box appears.

  2. Select an Assembly operation in the PPR tree.

  3. Click Launch. A progress dialog is displayed.

  4. Open the assembly drawing at the saved location using File > Open.

    For DPM - Structure Lofting, the default location for saved workshop documents is D:\DELMIA\PPRClient\data\product_cadpath\. Therefore, the assembly drawing above will be located at D:\DELMIA\PPRClient\data\product_cadpath\AssemblyMacroActivity\TBhd_1132_1.1\TBhd_1132_1.1 JoiningOperation.1_Assembly Drawing.CATDrawing.

    For Structure Manufacturing Preparation, the default location is C:\temp, and the assembly drawing above will be located at C:\temp\AssemblyMacroActivity\TBhd_1132_1.1\TBhd_1132_1.1 JoiningOperation.1_Assembly Drawing.CATDrawing.

    Please refer to Managing Project Resources to change the default location for saved workshop documents.
