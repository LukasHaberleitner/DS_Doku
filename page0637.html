Managing Products in an Assembly 

This task consists in managing products in an assembly.
Open an assembly.

  1. Select Tools > Product Management...

 

The Product Management dialog box appears.
 
 

The following is displayed for each components contained in the assembly:

  • Part Number: the part number.

  • Document: the document source file.

  • Status: the status of the component.

  • Representation: the associated representation.
 

You can modify the part number in the New part number field and replace the associated representation in the New representation field of the selected product.

The ... button opens the Replace Representation dialog box where you can choose a new representation for a component.
 

  1. Click OK to validate.
