Saving an Enhanced Scene in ENOVIAVPM

Enhanced Scenes can be saved in ENOVIAVPM, but only in the context of a DMU Review. See Saving DMU Applicative Data in ENOVIAVPM in the DMU Navigator User's Guide.
 
The Enhanced Scenes saved in ENOVIAVPM cannot be used in a drafting scenario: it is not possible to create a drawing with a view from this scene.
