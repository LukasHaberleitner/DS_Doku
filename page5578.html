MULTICAx SolidEdge (SE) Plug-in in a Nutshell

The MULTICAx SolidEdge (SE) Plug-in allows users to import:
  • SolidEdge native parts. This import converts  SolidEdge part geometry into the CATIA Graphical Representation (CGR) or CATPart formats, which renders it readable by V5. Users can perform this import interactively or from the command line.
  • VDA-FS (*.vda): parts only
  • Parasolid formats (*.x_t and *.x_b): parts only.
