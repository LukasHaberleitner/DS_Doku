Creating Components

This section explains how to create components. It includes the following tasks:
 

Build graphic in a detail sheet
Create new component
Create connector on component
Define pins on component
Manage potential connection on terminal board
Define a component group
Associate component reference to graphic detail
Create cable
Store in catalog.

Prior to creating your first component, make sure the following options are properly set up:
the diagrams option and the working units and grid activity.

NOTE: Using the CATDUAV5 tool can result in some reference components being deleted from a catalog or document. See Storing Objects in a Catalog for more information.
