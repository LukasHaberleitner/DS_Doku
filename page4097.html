Modifying a Package

Licensing

  Package Definition is only available with a Vehicle Occupant Accommodation product license.
This task describes how to modify a package.
A package is a set of motor vehicle dimensions which is to be used as an input by occupant posture prediction algorithms. The package parameters are defined in the Society of Automotive Engineers's (SAE) standard J1100 of September 2005 (see: www.sae.org).

  1. Double-click on a package (in the 3D geometry window or in the PPR tree) or right-click the package and select Definition.
    The Package Definition dialog box appears.

     
    Each tab in the Package Definition dialog box shows dimensions of the different seat rows : Driver, Second row and Third row.
     
    All items in the Combo Box of the Driver tab shows a different category of motor vehicle dimensions: the Seat, the Steering Wheel, the Accelerator Pedal and the Foot Rest the dimensions.

    Select the Show illustrating figure to see information on the dimensions of the current section
     

  2. Enter your specifications in the dialog box or modify the dimensions using green arrow manipulators.

  3. To associate a geometrical object with a package component, select the appropriate Geometry editor, then select the object in the 3D view. Note that you can associate SgRP, AHP, PRP and Rotation Axis Position with points and the Steering wheel with a circle.

  4. After entering your specifications, click OK.
