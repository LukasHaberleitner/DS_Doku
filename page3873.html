Numerical Properties

This reference describes the Numerical properties concerning:
  • Coordinate Dimensions.
 
  This page deals with the properties concerning:
  • The Format Options.
 

Format Options
 
 

Description
Defines the numerical display format applied to the dimension. To get the detailed description about the format selected from the drop down list, click the information point.
By default, NUM.DIMM.
 

Display
Defines the display factor depending on the selected format.
By default, 1 factor is selected.
 

Format
Decimal format in only available.
By default, Decimal is selected.

Precision
Defines the precision value.
By default, 0.01 is selected.
