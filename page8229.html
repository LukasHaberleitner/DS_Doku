The Manikin Contextual Window

Right-clicking on the Manikin in the PPR tree text brings up the contextual window.

The selections are:

  • Center graph - centers the information
  • Reframe On - centers the data in the 3D geometry window
  • Hide/Show - Hides/Shows the selection
  • Properties - Displays the properties
  • Open Sub-Tree - Opens the sub tree
  • Cut- cuts objects
  • Copy - copy objects
  • Paste - paste objects
  • Paste Special - special properties for paste
  • Delete - Deletes a object
