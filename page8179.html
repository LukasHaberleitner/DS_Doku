Carry Analysis

This task describes how to execute a carry analysis using the Carry Analysis dialog box. 
For more information on Carry analysis, see the Analysis Introduction in the Getting Started section.

  1. Select the Carry Analysis from the Ergonomic Tools toolbar. The Carry Analysis dialog box appears after the Manikin is selected.

  2. The Carry Analysis dialog box contains only the Snook & Ciriello guideline. This dialog box has the following fields:
     

    Guideline

     

    		 This list presents the guidelines available to perform a carry analysis. Only one guideline is available.

     
    Specifications
     
    • 1 carry every:
      Use this field to determine the carrying frequency. Select the arrows to increase or decrease the value indicated in the text field or directly enter a new value using the keyboard.
    •  Distance of carry:
      This specification is used to determine the distance the object will be carried. Select the arrows to increase or decrease the value indicated in the text field or directly enter a new value using the keyboard.
    • Population sample:
      Three population percentiles are provided: 90th, 75th, and 50th. These percentiles represent the percentage of the population able to perform the task safely. The selected percentile takes the manikin gender into account.

       
    Score

    Immediately after the Specification fields are completed, the results are displayed in the Score zone.

    • Maximum Acceptable Weight:
      The maximum acceptable weight is defined as the weight that the selected population can carry with reasonable safety.
       
    Warning

    Immediately after the Specification fields are completed, the results are displayed in the Warning zone.
