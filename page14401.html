Hangers

The following computed attributes and methods are in the HangerLayout Knowledgeware dictionary. Not all methods take input parameters, and for those that do, the parameters are described in the Description column.

Whether you can see the value returned by the method depends upon how you are using it in Knowledgeware. For example, if you write a Knowledgeware rule or check causing a component to display differently based on the outcome of that rule or check, you will not see the calculated value. You will only see the result of the calculated value.

You can see the value returned by a computed attribute in:

  • A report you generate through the Tools - Report Definition dialog.
  • The text displayed in your diagram, after you define a text template using the Insert Link Template dialog.
  • The Edit - Properties dialog, providing the administrator configured them to display as described in Changing the Display Order of Properties.

 

Object Type Attribute/Method Name Description
HgrCablePenetration Forward Frame The closest forward transversal (cross planes) frame name on which the Hanger is placed.
HgrCablePenetration OffsetForward Frame Offset of the Hanger from the forward frame.
HgrCablePenetration ShipSide Indicates on which side the hanger lies (port/starboard of the center line).
2-Port, 1=Starboard, 0-Center.
HgrCablePenetration OffsetCenterLine Offset measured from the center of the Hanger to the center line of the ship.
HgrRack Forward Frame The closest forward transversal (cross planes) frame name on which the Hanger is placed.
HgrRack OffsetForward Frame Offset of the Hanger from the forward frame.
HgrRack ShipSide Indicates on which side the hanger lies (port/starboard of the center line).
2-Port, 1=Starboard, 0-Center.
HgrRack OffsetCenterLine Offset of the Hanger from the forward frame.
HgrSupportPart AttachConnectivityCheck() Checks if the hanger is connected to a structure at the attach connector.
HgrSupportPart ListOfCables Contains a list of all the cables routed through a specific hanger.
HgrSupportPart PercentageFill Cable fill percentage of a hanger.
HgrSupportPart SupportConnectivityCheck() Checks if the hanger is supporting an object (pipe, HVAC, etc.) at the support connector location.
HgrSupportPart SupportedWeight Total weight the hanger is supporting. This depends on the hanger and the cable weight.
