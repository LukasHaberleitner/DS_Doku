Adding a Main Stacking Sequence

This task shows you how to import and export a laminate table to define a Main Stacking Sequence.

See More about Materials and Laminates for more information about the format.
Available in Composites Engineering Design (CPE), Composites Design for Manufacturing (CPM) and Composites Braiding (CPB).
Open an empty CATPart in either Composites Design or Grid Composites Design.

Import/Export a Laminate Table

You can import/export a laminate table.

  1. Click Composites Parameters  in the Parameters toolbar.
    Go to the Main Stacking Sequence tab.

  2. Enter the path of the spreadsheet to import and click Import.
    • The dialog box is populated with the Main Stacking Sequence information and that of the stacking sequences using it.
      See About Main Stacking Sequence for more information.
    • A Main Stacking Sequence node is created under Composites Parameters.
    • Imported laminates are displayed in the Laminates tab and marked SS-MSS.

  3. Enter the path of the file to create by export and click Export.
    Note that export files always include the color index line.
 

Add or Edit a Main Stacking Sequence

You can add or edit a Main Stacking Sequence.
You cannot:

  • Remove a layer from a Main Stacking Sequence.
  • Change the layer material or orientation in a Main Stacking Sequence.
  • Modify the layers order in a Main Stacking Sequence.

  1. Click Add MSS Laminate.
    Enter required information as explained in Adding Laminates.
    • The laminate name must be unique.
    • All laminate color indices must be unique.
    • You cannot switch a laminate to a stacking sequence.

    The laminate is added.

  2. Select a laminate and click Edit MSS Laminate.

    Select a line and click Add/Remove Orientation.
    The laminate orientations are updated accordingly.
    • You can modify the orientations of an existing laminate, or specify the orientations of a laminate you have added as explained above.
    • When adding or editing a laminate, you can only add or remove orientations without adding a layer to the Main Stacking Sequence.

  3. Click Add Layer.

    Specify the layer attributes.
    • You can add a layer at any position in the Main Stacking Sequence, above the selected line.
      If you have not selected a line, the layer is added at the end of the table.
    • If defined for one line, the layer must be defined for all and be unique.
    • If defined for one line, the layer level must be defined for all and be a numerical value (0 by default).
    • Material ID and orientations must correspond to existing ones in the model.
    • You can create a layer without associated laminates, provided the Main Stacking Sequence contains at least one layer with at least one associated laminate.

  4. Select a line a clik Edit Layer.

    Proceed as above.

    • You can modify the sequence name, the level and the laminates, not the material nor the orientation.

  5. If no Main Stacking Sequence exists yet
    • Click Add Layer.
      A layer is created without laminate.
      A Main Stacking Sequence is created.
    • Click Add MSS Laminate to associate a main Stacking Sequence laminate to the created layer.
    • Click Add/Remove Orientation to complete the laminate definition.
      A first laminate is added to the layer.
    • Add laminates as required.

  6. Click Remove MSS (if at least one Main Stacking Sequence wih one layer exists).
    The Main Stacking Sequence is removed.

If plies had been generated before the modifications of the Main Stacking Sequence, update is done as follows after the validation of the Composites Parameters.

  • The grid definition is automatically updated.
  • Edit the virtual stacking and select Recomputation from Grid in the Tools Palette. Validate the modifications.
  • An update of the plies group is proposed.
    • Plies generation is started.
    • Validate the plies generation.
