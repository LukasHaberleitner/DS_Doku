ISO Symbols Mapping Rule

This task explains the isogen symbols mapping design rule.

The isogen symbols mapping design rule included with this application is used when you are creating a 2-D isometric drawing from a 3-D document. The process uses third party software called Alias I-Run. You use this table to enter the different values for the attribute PipingSymbol in this application and their equivalent names in Alias I-Run. For instance, the symbol B_TEE_WM is called TE in Alias I-Run.

This is a general design rule and the associated table is located in the directory ...\intel_a\Startup\EquipmentAndSystems\XXXXX\DesignRules,where XXXXX is the application name, such as Piping.

The column PipingSymbol lists the Piping Design piping symbol values while the column ISOSymbol lists the equivalent Alias I-Run symbol values.

Note: The design rule text file located in the above directory contains an additional Comments column, which lists the Piping Symbols that are OK and the ones that are Not Supported.
