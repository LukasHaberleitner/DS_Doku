CatMergeMode (Enumeration)

enum CatMergeMode {
  catMergeOff,
  catMergeOn
}
Merge mode.

Sweep elements can be limited by existing material. This enum lists appropriate values to set this property.
Values:
catMergeOff
The shape is not limited by existing material.
catMergeOn
The shape is limited by existing material
