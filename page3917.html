Freestyle Commands in Other Workbenches

You can customize another workbench by using Freestyle commands and toolbars.

The Tools > Customize... function allows you to use Freestyle commands in other workbenches.

For FreeStyle dashboard-based commands, added in a dedicated workbench:

  • Tools Dashboard default status options are applied, except for Keep Original option which is active.
  • To dispay and manage Tools Dashboard options you have to add either:
    • the Generic Tools toolbar of FreeStyle in the dedicated workbench to access the Visualize New Dashboard dialog box option which display the Tools Dashboard dialog box.
    • the New Dashboard of FreeStyle in the dedicated workbench which display the Tools Dashboard dialog box.
