Inserting a New Component

This task shows you how to insert a component into an existing assembly.

This command lets you perform the following:

  • create an instance from the reference component
  • use a context-specific representation inside it (see Managing Representations).
Open the ManagingComponents01.CATProduct document.
In the specification tree, select ManagingComponents01 and click the New Component icon .
The structure of your assembly now includes Product1(Product1.1).
If you want you can define the default part number of the component to be imported. To see how this is done refer to Customizing Product Structure Settings.
