ISO Endstyle Mapping

This task explains the isogen endstyle mapping design rule.

The isogen endstyle mapping design rule included with this application is used when you are creating a 2-D isometric drawing from a 3-D document. The process uses third party software called Alias I-Run. You use this table to enter the different values for the attribute endstyle in this application, and their equivalent names in Alias I-Run. For instance, the endstyle BUTT WELD is called BW in Alias I-Run.

This is a general design rule and the associated table is located in the directory ...\intel_a\Startup\EquipmentAndSystems\XXXXX\DesignRules,where XXXXX is the application name, such as Piping.
