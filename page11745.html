Defining Machine Axis Names  

NC Machine Tool Builder allows the user to define machine axis names.  This is especially useful in defining rotary axis names.
Open the V4Axis_Machine.CATProduct file from the  online/mtbug_C2/samples/V5/4AxisMachine directory.  In addition:
  • add the new machine
  • add the machine kinematics definition
  • create mount parts and points
  • create travel limits

  1. Click Axis Names .

    The Jog dialog box is displayed.

  2. Use the Jog dialog box or manipulators on the geometry to set the machine in the desired position.

  3. In the Applicative frame, select a new axis name.

  4. Click the Close button.

    You cannot select a name that is already used. 
    These names are not associated to joint creation since the axis names are related to moving joints only.
    The name, which is set on the machine, appears in the the Machine Editor dialog box, in the case of rotary axes.
For additional information, see Jogging a Mechanism.
