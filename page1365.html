Exiting Version 5

This task explains hows you how to end a Version 5 session.
 

  1. Select Start > Exit.

If new documents are still open is your session, the Exit dialog box is displayed to let you:
  • save these documents before exiting the session. In that case, click Yes and use the Save As dialog box that opens to save your documents. Refer to Saving Documents for the First Time or Under Another Name for more information
  • exit the session without saving your documents. In that case, click No
  • cancel the exit. In that case, click Cancel.

If existing documents have been modified during the session but are still not saved, the Exit dialog box lets you:

  • exit the session without saving your modified documents. In that case, click Yes
  • save the modifications made to your documents. In that case, click No then select File > Save Management to save your documents. Refer to Managing Document Save for more information on this command.
