Accessing Non-geometric VPM and LCA Data

Non-geometric data (such as spreadsheets and text documents) referenced by products and resources stored in ENOVIA VPM and LCA can be accessed via Manufacturing Hub commands.
  • These commands are activated only after a project is loaded from the Manufacturing Hub.
  • A connection to VPM or LCA from the DPM session must already exist prior to using these commands.

Opening non-geometric data

  1. Select Open non-geometric ENOVIA VPM V4 document to access data stored in VPM, or Open non-geometric ENOVIA VPM V5 document to access data stored in LCA.

  2. Select the product or resource in the V5 document that links to the non-geometric data.

  3. A dialog box lists all non-geometric data linked to the selected product or resource. Select a non-geometric document of interest and click OK.

  4. The selected document is retrieved from VPM or LCA and opened with the default application associated with the selected document type.
