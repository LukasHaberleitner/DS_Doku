Target Tag Reconciliation

If you select the Disable Deletion of Used Tags check box, then users cannot delete a tag that is referenced by an existing robot motion.

If you select the Delete Target Tag with Robot Motion Deletion check box, then the effect of deleting a robot motion is that you delete the tag that is the motion's target, as long as the target tag is not also referenced by another robot motion.

The initial setting is that both options are disabled.
