Replacing the Detailed Process Part

This task shows you how to manage the update after parts have been replaced by geometrically similar parts.

  1. Make sure the part contains a Concept to Detail Link.

  2. Click Detailed Process Part Replace   in the Design Change toolbar.

  3. Expand the Synchronize dialog box.

    All the links between parts are listed, with their status.

  4. Select the synchronization options.
    • ClickSynchronize All Elements
      Or select elements and click Synchronize Selected Elements.
    • Select Reframe On Selection or Center graph on Selection.
      When you click an elements, the corresponding action is performed.
    • Select KO elements and click Delete,
      Or click Delete All KO.
      The selected broken links, or all broken links are deleted.

  5. Expand the Update Part dialog box.

  6. Select one processed element from the list.
    • Processed Elements lists the features updated within Detailed Process Part Replace.
      The list is lost when you exit.
    • Parent Features shows the parent feature of the selected feature.
      Use this list to navigate through the data model.

    A reframe is done on the selected feature. Only this feature and its parent of higher dimensions are shown.
    When you select a parent feature, a reframe is done, but the list is not updated unless you double-click the parent feature.

  7. Double-click a processed element to edit it.

  8. Specify the settings:
    • Enter the Geometric Tolerance (Same as in Geometrical Link).
    • Select the required check boxes.
      • Automatic Orientation Check. It works as follows:
        • For Points on Curve:
          Sometimes, the reference point of a point on curve switches sides, and is moved to the other end of the support curve. Detailed Process Part Replace updates the point on curve using its ration, then 1.0-ratio. It keeps the point at the shortest distance from the previous point position.
        • For Line Point/Direction, Plane Offset, Parallel Curve, Extrude, Offset, Translate, Rotate features:
          Sometimes, the direction used for those features is reversed (Plane going to the other side of its reference, or extrude going from 20mm to -20mm for example). Detailed Process Part Replace updates those features with their previous length, angle, or characteristic value, then with minus the same value. It keeps the result with the center of gravity at the shortest distance from the previous result.
        • For Split, Trim, Connect Curve, Spline, Boundary features:
          Sometimes, those feature switch sides (inverted tangent directions for splines or connect curve, other side of split, for example). Detailed Process Part Replace computes each possible side, and keeps the one which contains the same number of domains and with the center of gravity at the shortest distance from the previous geometry.
        • For Extract features:
          Sometimes, the target edge of face of an extract is split in several elements after a design change. To manage this behavior, Detailed Process Part Replace automatically replaces the extract by a multiple extracts of the corresponding new elements.
        • For Multiple Extract, Extrapolate features:
          Sometimes, the target edge or face of those features is split in several elements after a design change. To manage this behavior, Detailed Process Part Replace automatically adds the necessary elements in the feature definition.
      • Display Previous Geometry. Displays the geometry of the selected features as it was before update, or before an update error. It is displayed in purple, and is not pickable. It is especially helpful to find a feature the orientation of which has changed.
      • Display Process Sub Geometries
      • Reframe On Selection
      • Center Graph on Selection

  9. Select an action:
    • Process Part
    • Process Selected Feature
    • Edit Selected Feature
    • Delete Selected Feature.

  10. Expand the Check Detailed Design dialog box.
    • Check the links.
    • Apply the detailed geometry to the tools as required.
