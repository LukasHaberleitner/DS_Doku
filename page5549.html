Importing Part Files Interactively

This task shows you how to import JT files into a product structure document.
For more information about import characteristics and format extensions, please refer to the Requirements chapter.

  1. Open a empty or existing product structure document.

  2. On the product structure document, use the right mouse button to select the product under which the JT data are to be imported.
    Select Components -> Existing Component.  

  3. In the File Selection dialog box, select JT files (*.jt) from the format list.

  4. Navigate to the directory containing the file and select it.
    The imported data appear under the selected product, as a CATProduct + CATPart or CGR.

An alternative method to performing steps 2 and 3 is to select the product with the left mouse button, then, from the menu bar, select Insert->Existing Component.
