Anthropometric Copy/Paste Functions

 This task describes how to copy the anthropometry from one manikin and paste it on another manikin.
After this procedure is complete, the second manikin's anthropometry is set like the first, and the manikin is automatically updated.

  1. From the Specification tree, select the anthropometry to copy.

  2. Select the Edit  > Copy command or click the right mouse button to activate the contextual menu. Select Copy.

  3. Select the second manikin's anthropometry.

  4. Select the Edit  > Paste command (or Paste from the contextual menu).
