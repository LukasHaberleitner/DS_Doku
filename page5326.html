Checking Contours

This task shows you how to check and validate contours for plies.
Available in Composites Engineering Design (CPE), Composites Design for Manufacturing (CPM) and Composites Braiding (CPB).
Open the CheckContour1.CATPart document.

  1. Click Check Contours in the Analysis toolbar.
    The Check Contours dialog box is displayed:

  2. Select the stacking.
    • You can select a stacking, plies groups, sequences or individual plies.
    • Multi-selection of entities is available.
    • In the multi-selection dialog box, click to select entities using the Stacking Management.
    The dialog box is automatically updated:

    The command indicates valid contours, self-intersecting contours, or contours that are not closed.
    If a flattening exists, this column is populated as well.

  3. Place the cursor over a line you want to visualize, right-click it and select the action you require:

  4. Click Close to exit the dialog box.
