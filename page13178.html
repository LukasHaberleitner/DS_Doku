Sweeping and Filleting

 

This task shows how to create swept surfaces and fillets on both sides of the part.

You will use the profile element on the side of the part for this. In this task you will also create a symmetrical profile element on the opposite side of the part.
 

  1. Click Sweep .

    The Swept Surface Definition dialog box appears.

  1. Select the Explicit type.

  2. Select the profile element (Corner.1).

  3. Select the guide curve (Guide.1).

  4. Select the central curve (Spline.1) as the spine.

     

  5. Click OK to create the swept surface.

  6. Click Symmetry .

    The Symmetry Definition dialog box appears.

  7. Select the profile element to be transformed by symmetry.

  8. Select the YZ plane as reference element.

  9. Click OK to create the symmetrical profile element.

  10. Click Sweep again.

  11. Select the profile (Symmetry.3) and the guide curve (Guide.2).

  12. Select the central curve (Spline.1) as the spine.

  13. Click OK to create the swept surface.

  14. To create a fillet between the side portion and the central part click Shape Fillet .

  15. Select the side sweep element and the central portion of the part, then enter a fillet radius of 1mm (make sure the arrows are pointing up).

  16. Click Preview to preview the fillet, then OK to create it.

  17. Repeat the filleting operation between the other sweep element and the central portion of the part, and entering a fillet radius of 1mm (make sure the arrows are pointing up).

  18. Click OK to create the fillet.
