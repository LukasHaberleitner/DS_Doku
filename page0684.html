Setting Up Parameters

Default Standard File
The Default Standard File path corresponds to the path of the file where the default standards are stored.
You can change the path by clicking Browse icon.

Automatic Standard Import
You can automatically import the standard by selecting Automatic Standard Import.
If this option is selected, any BiW feature creation (BiWJoint, BiW JointBody, BiW Fastener) automatically imports the standard.
By default, this option is selected.

Check the Standard File Consistency at Import Time
You can also select Check the Standard File Consistency at Import Time to check the consistency of the standard values when importing the standard file (whose path is mentioned above)
If the standard file is not valid, it cannot be imported and an error message is issued.
By default, this option is not selected.
 

Defining the Export Directory
The Export default directory is the directory where the export file is stored. It is possible to use environment variables in the Export directory. For more information, refer to Cache Management (section Cache Location) in the CATIA Infrastructure User's Guide.
You can change the path by clicking Browse icon.
 

Defining the Report Directory
The Report default directory is the directory where the report file is stored. It is possible to use environment variables in the Report directory. For more information, refer to Cache Management (section Cache Location) in the CATIA Infrastructure User's Guide.
You can change the path by clicking the Browse icon.
 

Setting Up Extract Parameters

Fixed Drawing Spot Size
You can define a fixed size for the 2D drawing spots: select Fixed Drawing Spot Size and define a diameter value.
As a consequence, all spots in the 2D document will have this diameter, whatever their size in the 3D document.
By default, this option is not selected.

Angular Tolerance
You can define a user Angular Tolerance.
By default, this option is set to 90deg.

Include View Reverse Direction
You can select Include View Reverse Direction to visualize 2D reverse spots.
By default, this option is not selected.
For further information, refer to the Drafting Integration chapter.
 

Allow creation Enovia unsaved data
You can allow the creation of unsaved data from Enovia by selecting Allow.
If you do not select this option, commands will be grayed out in Enovia V5.
By default, this option is not selected.
For further information, refer to the Optimal CATIA PLM Usability for Automotive BiW Fastening chapter.
 

Lock Fastener Identifier
If you select Lock, then renaming of Fastener identifier is not possible.
The Renames BiW Fasteners icon in Automotive Biw Fasteners toolbar and Renames BiW Fasteners command in the Tools menu are disabled.
By default, this option is not selected.
  • The Lock option can either be set by the administrator or by user.

    • If the Administrator activates the Lock option, then you will not be able to rename the fastener ID. The access rights to the Lock option for the user can also be disabled by administrator by setting a lock on this option. A closed lock is displayed near the Lock option indicating that you cannot change this setting.

      In this case, a red lock is displayed, near the Lock option.

    • If you activate the Lock option, the identifier of the fastener cannot be renamed. 
      However, it can be disabled by the administrator.
The following settings are applicable to Automotive BiW Fastening Parameters while creating joint elements.  
 

Standard Settings
If you select Lock Standard settings, then changing standard settings is not possible.
By default, this option is not selected.
 
 

Naming Rule
If you select Lock Naming rule numbering, then modification of naming rule parameters is not possible.
If you select Lock Auto naming and name uniqueness, then auto naming and checking name uniqueness is not possible.
 
 

Design Check Rules
If you select Lock Thickness count, then modification of thickness count option, as well as changing the values of thickness count parameters, is not possible.
By default, this option is not selected.
 
If you select Lock Fastener location on surface, then modification of fastener location on surface option is not possible.
By default, this option is not selected.
 
If you select Lock Fastener normal vector with respect to closest surface, then modification of fastener normal vector with respect to closest surface option is not possible.
By default, this option is not selected.
 
If you select Lock Joined thickness crossing, then modification of joined thickness crossing option is not possible.
By default, this option is not selected.
 
If you select Lock Joined thickness normals alignment, then modification of joined thickness normals alignment option, as well as changing the value of tolerance parameter is not possible.
By default, this option is not selected.
 
If you select Lock Maximum/Minimun thickness ratio, then modification of maximum/minimum thickness ratio option, as well as changing the value of maximum thickness ratio, is not possible.
By default, this option is not selected.
 
If you select Lock Thickness gap, then modification of thickness gap option, as well as changing the values of minimum and maximum thickness gap, is not possible.
By default, this option is not selected.
 
If you select Lock Minimum spot spacing, then modification of minimum spot spacing option as well as changing its value is not possible.
By default, this option is not selected.
 
If you select Lock Spot center distance to flange edge, then modification of spot center distance to flange edge option, as well as changing the value of minimum spot center distance to flange edge, is not possible.
By default, this option is not selected.
 
If you select Lock Spot border distance to flange edge, then modification of spot border distance to flange edge option, as well as changing the value of minimum spot center distance to flange edge, is not possible.
By default, this option is not selected.
 
If you select Lock Diameter overhang distance, then modification of diameter overhang distance option, as well as changing the value of maximum overhang distance is not possible.
By default, this option is not selected.
 

Joint body creation option
If you select Turn on joint body creation option, then it creates a joint body after clicking OK in the BiW Joint Definition dialog box.
By default, this option is selected.
