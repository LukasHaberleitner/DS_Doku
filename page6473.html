Routing Wires from External Data

This task shows you how to route wires.
The system has been selected in the previous task.

  1. Select Automatic Wire Routing .

  2. The Wire Routing dialog box opens with connectivity and attribute information:

  3. Select wires and click the right arrow .

    The selected wires shift to the right column: they will be routed.

    Using the double right-arrow select all the wires and send them in the right column.

  4. Click Route.

    The Automatic Wire Routing Report dialog box is displayed:

    The specification tree is updated, showing the wires routed:

  5. Close the report window.

Place the pointer over a wire in the specification tree to highlight the bundle segments containing the selected wire in the geometry area.
