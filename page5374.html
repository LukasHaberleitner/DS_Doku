Producing Drawings with Generative View Styles

This task shows you how to produce drawings using generative view styles (GVS):
  • Producing a Drawing with a GVS File,
  • Modifying the Generative View Style after the View is Generated,
Generative view styles make it possible to customize the appearance of drawings via a set of parameters defined in an XML file.
Some generative view style parameters are specifically available to produce customized drawings from parts designed in the Composites Design workbench. It is the administrator's job to provide suitable styles.
Save the following xml files: Copy CompositesGVSContourOnly.xml, CompositesGVSExplodedOnly.xml and CompositesGVSFlattenOnly.xml from install_root/resources/standard/generativeparameters/SAMPLES or from the documentation samples directory to install_root/resources/standard/generativeparameters.

Open the GVS1.CATPart document.

Before starting, make sure the Prevent generative view style creation option is unselected
in Tools  > Options  > Mechanical Design > Drafting > Administration tab,
prior to generating a view in a .CATDrawing document. This activates generative view style functionalities.

Producing a Drawing with a GVS File

  1. Select Start > Mechanical Design > Drafting.
    The New Drawing Creation dialog box appears. The empty sheet layout is pre-selected.

    Click OK.
    You switch to the Drafting workbench and an empty drafting sheet is created.
    For ease of use, tile the windows horizontally using the Window > Tile Horizontally menu item.

  2. Click Front View in the Views toolbar from the Generative Drafting workbench.
    The Generative View Style toolbar is automatically displayed.

  3. Return to the 3D document.
    Select the entity you want to be represented in the drawing.
    It can be the stacking, a plies group, a sequence or a ply.
    In our example, we have selected Plies Group.1

  4. Then select a plane in the specification tree.
    In our example, we have selected xy plane.
    A preview of the view is displayed in the drawing.

  5. Click in the drawing to validate the view creation.
    The front view is created, displaying the plies of Plies group.1 as defined in the chosen generative view style.

  6. Click Offset section cut in the Sections toolbar.
    The Generative View Style toolbar is automatically displayed.

  7. Select the CompositesGVSExplodedOnly style in the drop down list.

  8. Sketch the cutting profile of the offset section cut, then click in the drawing to create the section.

  9. Position the section cut in the drawing sheet, then click in the drawing to create the offset section cut.
    The offset section cut is created, displaying the exploded plies only as defined in the chosen generative view style.

  10. Return to the 3D document.
    Select Start > Mechanical Design > Drafting to create a new drawing sheet.
    Click OK.

  11. Click Front View in the Views toolbar from the Generative Drafting workbench.

  12. Select the CompositesGVSFlattenOnly style in the drop down list.

  13. Return to the 3D document.
    Select a plane in the specification tree.
    In our example, we have selected the yz plane.

  14. Click in the drawing to validate the view creation.
    The front view is created, displaying the flatten plies as defined in the chosen generative view style.
 

Modifying the GVS after the View is Generated
 

  1. Right-click the Front-view node and select Properties.
    The Properties dialog box is displayed.

  2. In the Generative View Style list, select CompositesGVSExplodedOnly.

    Click OK.

  3. Click Update to update the view.
    The exploded plies are displayed in the drawing.
You can also modify the Generative View Style using the Set View Style command.
To do so, right-click the Front-view node and select Front View Object>Generative View Style>Set View Style.
Then select the appropriate style in the Choose a style dialog box.
Set the View Generation Mode to Approximate in Tools > Options > Mechanical Design > Drafting > View tab before generating a drawing with the CompositesGVSExplodedOnly style on a part using the tessellated constant offset or the draped tessellated surface ply representation.
