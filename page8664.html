How to Open Product Structure Documents Attached at Multiple Levels

This task shows you how how to open Product Structure Documents at multiple levels. This section describes the following:
  • Creating a Product Structure with Documents at Multiple Levels
  • Opening a Document at an Intermediate Level
  • Opening a Document at the Bottom Level
  • Opening Documents at Multiple Levels

Creating a Product Structure with Documents at Multiple Levels
 

  1. Create a PRC "PRC_Multiple", insert one instance "Assembly_Multiple" under it and send the structure to CV5.

  2. Insert two parts "Part1_Multiple" and Part2_Multiple" under it and Save the structure back to EV5. Refresh in EV5 product editor. The following structure will appear.

 

  1. In Document Management create one document with the name "Doc_Multiple" with catpart under it.

  2. Select "Assembly_Multiple" and click RMB, from contextual menu select Insert > Documents.

  3. Click on Existing Tab. Select Environment "DOCDIR" and Type "Document Revision".

  4. Search and select for Doc_Multiple from the result window. Select "Define as geometry" checkbox.

  1. Click OK. The following screen appears.

  1. Lock the document "Doc_Multiple". The Structure will be as given below.

 

Opening a Document at an Intermediate Level
 

  1. Select Doc_Multiple and send it to CV5.

  1. The Doc is loaded as seen below.

  1. Close all in CV5.

Opening a Document at the Bottom Level

  1. Select Part1_Multiple and Part2_Multiple and send it to CV5 as assembly.

  

  1. Only Part1_Multiple and Part2_Multiple will get loaded in CV5.

 

Opening Documents at Multiple Levels
 

When the document attached at an intermediate level is a CATPart document, it is impossible (see below). When the document attached at an intermediate level is a CATIA V4 model document, it is possible with the following restriction: the intermediate level document must be opened first, then the bottom level documents.

  1. Now select Doc_Multiple and send it to CV5 in the already opened session.

  1. Nothing will get loaded. You will get an error message.

 

  1. Do multiple selection of all the docs and send it to CV5.First select Doc_Multiple.

 

  1. Only Part1_Multiple and Part2_Multiple will get loaded. Doc_Multiple will not get loaded. You will get an error message same as Step2.

  2. Do multiple selection of all the docs and send it to CV5. First select the parts and not the Doc_Multiple (Intermediate Doc).

  3. Only Doc_Multiple will get loaded.Part1_Multiple and Part2_Multiple will not get loaded. You will get the following error message.
