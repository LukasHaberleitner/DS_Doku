Linear Quadrangle

Linear Quadrangle is a four-nodes surface element based on the Reissner/Mindlin theory.
 

Type

surface element

Physical property

shell
membrane
shear panel

Mesh connectivity

QD4

Number of nodes

4

Degrees of freedom
(per node)

6 (3 translations and 3 rotations)

Type of behavior

elastic
 

This element has four gauss points:

P1 (-1/ ; -1/)               P2 (1/ ; -1/)

P3 (1/ ; 1/)                  P4 (-1/ ; 1/)
