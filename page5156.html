Managing Tool Paths

  This is where you can find the functions you need to edit tool paths for all operations. Please note:
  • Tool path editor supports multi-selection, it works on several operations or several manufacturing programs.
    For instance, select all machining operations under a tool change and then click Edit Tool Path .
  • Before using any of the functions below, you must have computed a tool path. 
  • You have the choice to display the tool center or the tool tip.
    See the General parameters in the Customizing section of this guide.
  • Cancel is available for:
    • Editing a point,
    • Editing an area,
    • Connection tool paths,
    • Modifying approaches and retracts in tool paths,
    • Modifications done with the dialog box,
  • Local undo via Ctrl-Z is always available.
