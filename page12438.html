Bottom Plane Definition Using a Point

This section describes how to define a bottom plane using a point for Pocketing, Profile Contouring, Groove Milling, Trochoid Milling, and Facing operations. When you do not have a floor to define for a profile, you can select a point as the bottom. The axis of the bottom is taken from the tool axis. Any change in the tool axis effectively change the direction of bottom (always keeping it perpendicular to the tool axis and passing through the selected point).

Supported Operations and Modes

The following operations and modes are supported:

  • Pocketing for both Hard bottom and Soft bottom
  • Profile Contouring in Between Two Planes mode, for both Hard bottom, and Soft bottom
  • Groove Milling for both Hard bottom and Soft bottom
  • Trochoid Milling
  • Facing

Point Selection

For these mentioned operations, if you tries to select the bottom plane then you can select any point also apart from planar face or surface. While searching for selection you can get the following message in the message bar Select a point or a planar face or surface to define the bottom.

If you select any point then that point is taken as the relimiting element. A virtual plane is created at that point and normal of the virtual bottom plane is in the direction of the tool axis. A manipulator is shown at the selected point to show the virtual bottom plane. This manipulator is part of the operation only and it is visible only when the operation is edited.

If you changes the tool axis from Strategy tab then the direction of manipulator (showing virtual bottom plane) changes so that it always remains perpendicular to the tool axis. This manipulator helps to identify that the correct tool direction is selected. If there is no tool axis specified (0,0,1) is taken as the default tool axis.

If you directly selects a planar face or surface then the functionality works as it was working before this enhancement and there will be no change.

Prismatic Machining Areas and Prismatic Rework Areas

Facility of selecting the bottom plane using a point is extended to Prismatic Machining Area and Prismatic Rework Area features also. If you creates Prismatic Machining Area or Prismatic Rework Area feature separately using Insert > Machining features > Milling Features, you can select bottom plane as a point.

As there is no tool axis defined for these features, (0,0,1) is taken as the default tool axis. Once this feature is selected in any Pocketing or Profile contouring operation, the direction of the virtual bottom plane is guided by the operation tool axis and it always remain perpendicular to the tool axis. Any change in the tool axis is reflected in the bottom plane direction. Once any Prismatic Machining Area or Prismatic Rework Area is selected in the operation the bottom plane direction is over ridden by the tool axis direction of the operation. From then onwards, direction of the bottom plane in the features remain as that direction only, no matter feature is part of the operation or not. But this is applicable only if bottom is selected as point.

Data Persistence and Plane Manipulator

The point selected by this functionality is persistent and saving the operation saves this point in the operation. When you reopens the operation, the manipulator (showing virtual bottom plane) is displayed at the selected point.

Limitations

With the selection of point the for bottom plane, the following operations does not work:

  • Pocketing: Island detection and contour detection does not work.
  • Profile Contouring: For Between Two Planes mode contour detection does not work.
  • Groove Milling: Contour detection does not work.
  • Facing: Contour detection does not work.
  • Prismatic Machining Area and Prismatic Rework Area: Island detection and contour detection does not work.
