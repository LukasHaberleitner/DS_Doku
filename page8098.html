Customizing Settings for Freestyle

  Before you start your first working session, you can customize the way you work to suit your habits.
This type of customization is stored in permanent setting files: these settings will not be lost if you end your session.

  1. Select the Tools > Options command.
    The Options dialog box opens.

  2. Select the Shape category in the left-hand box.

  3. Select the FreeStyle sub-category.
    Two tabs are displayed:
    • The General tab lets you set the general options.
    • The Manipulators tab lets you set the control point options.

  4. Select the General category in the left-hand box.

  5. Select the Display sub-category.
    Various tabs are displayed, the Performance tab also interferes with Freestyle Shaper & Annotation.
    • The Performance tab lets you set the performance options.

  6. Select the Parameters and Measure sub-category.
    Various tabs are displayed, the Scale tab also interferes with Freestyle Shaper & Annotation.
    • The Scale tab lets you set the required scale.
