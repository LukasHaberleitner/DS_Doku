Extracting or Cutting a Subdivision Surface

This task shows you how to extract or cut a subdivision surface by selecting faces or edges to create a new subdivision surface.

Create a 3D shape containing a subdivision surface.
 

  1. Select the subdivision surface.

  2. Click Extract/Cut in the Styling Surfaces toolbar.
    You can also access this command by pressing ALT+SHIFT+J.

    A Tools Palette is displayed:

    Option Description Shortcuts
    Face Selection Select faces to extract as new surfaces. ALT+SHIFT+F
    Edge Selection Select edges to define limits for creating new surfaces. ALT+SHIFT+E
    Linear Propagation Propagates the selection along the line.  
    Loop Propagation Propagates the selection in a loop.  
    Fill Propagation Propagates the selection by filling in all directions (faces only).  
    Selection Inversion Inverts the selection (when extracting a face only). ALT+SHIFT+I
    Apply (Enter) Validates the command.  

  3. Click Face Selection and select one or more faces.

    A solution is previewed:

    Click Apply (Enter) to validate. The selected faces are kept

  4. To invert the selection, click Selection Inversion .
     The complementary selection is computed and previewed.


    Click Apply (Enter) to validate. The selected faces are kept.
