CatDraftMultiselectionMode (Enumeration)

enum CatDraftMultiselectionMode {
  catNoneDraftMultiselectionMode,
  catDraftMultiselectionByNeutralMode
}
Multi Selection Mode.
The elements to be drafted can be selected explicitly or can implicitly selected as neighbors of the neutral face
Values:
catNoneDraftMultiselectionMode
The elements to be drafted can be selected explicitly
catMultiselectionByNeutralMode
The elements to be drafted are the neighbors of the neutral face
