Using This Guide

This guide is intended for the user who needs to become quickly familiar with the Electrical Cableway Routing product. The user should be familiar with basic Version 5 concepts such as document windows, standard and view toolbars. Prior to reading this book, we recommend that you read the Infrastructure User's Guide.

The Systems Space Reservation product is a prerequisite.
For more information about it, refer to Systems Space Reservation User's Guide.

The sections of the book describe in detail the procedures for using all of the features of the Electrical Cableway Routing product.
