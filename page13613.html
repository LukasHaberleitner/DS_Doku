Space Analysis and 3DXML

3D XML (EXtensible Markup Language) is a universal, lightweight, XML-based format that enables users to capture and share live, accurate 3D data quickly and easily.
3D XML compresses highly complex data, provides rapid file transmission and shorter load times while maintaining the exact geometry of the files exchanged.

Space Analysis supports the 3D XML format version 4. This enhancement enables upward compatibility with CATIA products for specific operations:

  • Open and/or insert a 3D XML document containing sections, measurements made with Measure Between or Measure Item items.

  • Save as in 3D XML format a CATProduct containing sections, measurements made with Measure Between or Measure Item items.
