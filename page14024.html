Enabling the Design Release Operation from SmarTeam

The SmarTeam – CATIA Integration can be customized to enable a design Release operation from SmarTeam.

.

Perform the customization by performing the following procedure:

  1. Install SmarTeam – Regulatory Compliance.

  2. Using the Data Model Designer (DMD), add the Electronic Signature and Job Server mechanisms to the database.

  3. Using Script Maintenance, add the function CreateJobBasedOnFileType to the Release operation After event.
    Script name: JobServer.ebs.

  4. Click Tools.

  5. In the Admin Settings profile card, add the following values:

    • In the Section field, enter Job Server

    • In the Subject field, enter CATIA

    • In the First Value field, enter INTERNAL

    • In the Second Value field, enter LEAVE.

    • In the Fourth Value field, enter the path of the Log folder.

      All other fields remain empty.
