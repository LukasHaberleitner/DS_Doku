Creating a Drawbead

This task shows you how to create a Drawbead from the Press Axis System, a Guide, a Support Surface, and a Profile.
Open DrawbeadP1.CATPart from the samples directory.

  1. Click Create a Drawbead   in the Addendum Design toolbar.

  2. Select the Press Axis System.
    The Z-Axis of the Press Axis System becomes the direction of the drawbead.

  3. Select a Guide and a Support Surface.
    The selected guide is the neutral fiber of the drawbead. It lies on the support surface.
    Tangency constraints are created between the selected support surface and the drawbead.

  4. Select a type of profile from the list.
    A representation of the selected profile type, with its parameters, is displayed in the dialog box.

  5. Modify the parameters as required.

  6. Click Apply.
    A preview of the drawbead is displayed in the graphic area.
    • Profile features are created along the guide.
    • The surfaces associated to the profiles features are generated, one surface per profile feature, according to the selected profile type.

  7. Use the contextual menu of a profile feature to delete it.

  8. Place the pointer over the guide and click to add a profile feature.

  9. Select a profile feature and drag it along the guide to move it.

  10. Click OK.
    A Drawbead feature is created under Die Face Design Set.

    It contains
    • The global parameters of the drawbead
    • A Profile Set that contains the Drawbead Profile features with their own parameters
    • A Multisection Surface Set.

    Note that the parameters of all the Drawbead Profile features are linked to the parameters of the Drawbead feature:
    If you modify a parameter of the Drawbead feature (global modification), all the corresponding parameters of the Drawbead Profile features are modified.
    You cannot modify a parameter of a given Drawbead Profile feature.

  11. To make local changes to parameters of a given Drawbead Profile feature
    • Under that Drawbead Profile, right-click Parameters (to modify all the parameters) or only the required parameter.
    • In the contextual menu, select xxxObject > Formula > Deactivate.
      That parameter or that set of parameters is isolated.
      Edit them locally, with their own values.
    • To recreate the link with the parameters of the Drawbead feature, select xxxObject > Formula > Activate
      The local parameter values are replaced by the global values.
