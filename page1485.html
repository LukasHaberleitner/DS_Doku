Reframing On an Object

 

This task explains how to center a selected object in a V5 window.

  1. Select an object, either in the geometry area or in the specification tree.

  2. Right-click then select Reframe On.

    You focus in on the selected object which is centered in the application window.
    Before
     
    After

In Drafting, the Reframe On command works differently than in other workbenches. Reframe is actually performed on the view, in order to show not only the element which is being reframed on, but also the particular view which contains this element.
