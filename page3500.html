CATCompCore

Description

Describes the Composites feature that you create when you click in the Composites workbench.

Inheritance path

Standard -> Feature -> Composites -> CATCompCore.

Attributes

Axis

Defines the rosette of the core feature.

Direction

Defines the direction of the core feature.

Material

Defines the material of the core feature.

  
 

CATCompCutPiece

Description

Describes the Composites feature that you create when you click in the Composites workbench.

Inheritance path

Standard -> Feature -> Composites -> CATCompCutPiece.

Attributes

Axis

Defines the rosette of the cut-piece.

Direction

Defines the direction of the cut-piece.

FatherPly

Defines the father ply of the cut-piece.

Material

Defines the material of the cut-piece.
