Validating a Motion  

Validating Positions Automatically (without defining a clash specification)
Select a simulation and click the arrow within Clash Detection from the DMU Check toolbar. Undock the toolbar if necessary. Activate Clash detection (On or Stop modes), then replay your simulation.
Performing an Experiment and Generating a Report
Define measures, interferences, and distances specifications. Select a sequence (a simulation object), click Edit and Perform Experiment from the DMU Simulation toolbar.

Licensing
  The following functionalities are available only with a DMU Space Analysis license.
Detecting Clashes
Select two items, then click Clash . Add the clash specification into an existing simulation. Replay the simulation.
Measuring Distances
Select two items then click Distance and Band Analysis . Add the distance specification into an existing simulation. Now as you replay the simulation, distance is measured between the components selected.
