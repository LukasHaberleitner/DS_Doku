Exporting an Assembly and a Drawing From ENOVIAVPM

 

This task explains how to export an assembly and drawing from ENOVIAVPM .

  1. Extract the Drawing and the assembly from ENOVIAVPM .

    • load the Assembly structure
    • load the Assembly drawing through a PSN opened from the part on which the CATDrawing relies (not from VPMAccess)
    • make sure that no update is required

  2. Use the Edit > Links > Replace command to replace the links of the product in the CATIA session. The links between the drawing and the products are solved. When performing an "Edit > Links > Replace" operation on a VPM assembly structure, the PSN window of this structure must be active.

  3. Save the product in file mode.

    • from the File menu, access the Save Management command.
    • save the Product
    • in the Save Management dialog box, check the Propagate directory option to save the documents in the structure
    • check the Enables independent saves option and check the Reset option to reset the automatic saving of the drawing. The drawing is still not saved and is not up-to-date

  4. Update the drawing.

  5. From the File menu, access the Save As command to save the drawing in File mode.

  6. Close CATIA.

  7. Launch a new CATIA session and re-open the File product structure. Your files are now ready to be externalized.

  8. From the File menu, access the Save Management command. Open the assembly drawing.

  9. In the Edit > Links window, use the Replace button to replace the links of the product in the CATIA session. The links between the drawing and the products are solved.

  10. Update the drawing and save it. (If you need to make sure that everything ran fine, close CATIA, re-launch it, and reload the assembly and the drawing in a new CATIA session to validate the export.)
