General

This page deals with the General options for Manufacturing System Definition. These options apply to both the System Editor and the System Gantt. The calculation of the utilization of logical resource is started when the System Editor and System Gantt are first opened. The colors are updated if relevant changes occur for the utilization.

Resource Utilization

Consider Product Flows
A product flow expresses the concept that one resource may start its work after another resource. When a product flow is defined for activities, selecting the Consider Product Flows check box shows the timing considerations required in both the Gantt chart and in the output of the Process Verification Across Resources command.

By default, the check box is not selected.

Calculate and Display Utilization of Logical Resources
The utilization of logical resources (like stations and workplaces) is calculated in percentages based on the takt time, which is a measure of how long a resource is busy. For example, if the takt time is 60 seconds and the range goes from 50% to 75% then the corresponding time range is 30 (60 x 50%) to 45 seconds (60 x 75%). Any resource utilized from 30 to 45 seconds (or 50% to 75%) is marked in the defined color for this range.

If the calculation of utilization is unchecked, then the System Editor uses the color defined for the node background in Tools > Options > Digital Process for Manufacturing > PERT Chart.

For calculating the utilization of a logical resource in either the System Gantt or System Editor, three types of resources are available:

Resource with links to a TSA, physical resource, ownsbehavior = FALSE, Hide in System Editor = TRUE
If a resource has at least one link to a TSA of another resource (Process uses Resource link) no utilization is calculated for this resource. If the resource is a physical resource (ownsbehavior = FALSE on plantype or CAtProduct, CATPart, etc assigned), no utilization can be calculated. If Hide in System Editor = TRUE is set for the plantype of the resource, then no utilization can be calculated. The calculation of the utilization stops at this level and no further utilization for the children of this resource can be calculated. This type of resource is not shown in the System Gantt.
 
Resource with behavior and TSAs
For a resource with non-empty behavior, utilization is calculated with the following formula:
Utilization [%] = (Time in which the resource is busy/cycle time of resource) *100

Time in which the resource is busy is the duration of time from TSAs with respect to the defined control flow, parallel activities and times where the resource is not busy.

Cycle time of resource is the cycle time as shown in the image, persisted in the attribute cycletime of the resource itself or its father, grandfather etc. If no valid cycle time could be retrieved, utilization is 0. The utilization of the resource is not persisted, it is available in the session.

Resource without a behavior or with an empty behavior
If a resource has no behavior or has an empty behavior, the utilization is calculated from the utilization of the children with the following formula:
Utilization [%] = (Sum of utilization of children/number of children)

By default, the check box is selected.

Utilization Criteria

Takt time (sec)
Setting the takt time determines the remainder of the utilization criteria because the remaining calculations are based on the takt time. Four ranges are offered; you can opt to use 2 or 3. For each range, a color and one or two percentage values are defined. deselect the checkbox if you do not want one or both of the midranges. If the range is deselected, the corresponding fields for color, percentage value(s), and absolute value(s) are no longer accessible. The selection for the color field is hidden and the other fields disappear.

When the percentage value is changed, the corresponding absolute time value also get updated. The ranges cannot overlap, and an error message appears in case of misleading inputs, such as holes between the ranges. Misleading inputs are checked when the percentage value is changed or a range is selected or deselected.

If utilization cannot be calculated for a resource, then the System Editor uses the color defined for the node background in Tools > Options > Digital Process for Manufacturing > PERT Chart. In the System Gantt, the default color for resources (light blue) is used.

The default color values and ranges are shown; by default, all four ranges are available.
