Managing Knowledgeware Expressions

This task explains how to create or edit expressions used in reports.

Expressions are user-defined computer attributes that are added to report definitions.
You use the Knowledgeware Expression dialog box to gather the information needed to create an expression.
1. From the menu bar, select Tools - Report - Report Expressions.

The Knowledgeware Expression dialog box displays.

2. To create a Knowledgeware expression:

Select New from the Expression Name drop-down list.

The Reporter Expression Editor dialog box displays.
3. Create the expression according to Knowledgeware format guidelines.

For Signature of the Edited Expression:

  • Include only one input of type Product, Part or Feature.
  • Include only one output of type String, Real or Integer.

Click OK when you are done.

NOTE: Make sure you follow Knowledgeware format guidelines, If not, a message informs you that you cannot save the expression.
4. Click Save As on the Knowledgeware Expression dialog box to save the new Knowledgeware expression.

The name you enter to save the file also serves as the expression name.

You must save expressions in a PRM-defined directory. If you attempt to save an expression in a non-PRM directory, a message alerts you of the error.

You can use these expressions when defining a report format on the Report Definition dialog box. Expressions selected from the Expression drop-down list are added to a report definition template.

Use expressions as you use attributes. All fields, such as Sort, Group By and Sum, apply to expressions the same way they apply to attributes.

See Defining the Report Format for more information.
5. To edit a Knowledgeware expression:

Use the Knowledgeware Expression dialog box to select an expression from the Expression Name drop-down list.

Click the Edit button.

The Reporter Expression Editor displays.

Make the necessary edits on the Reporter Expression Editor dialog box. Click OK when you are done.

NOTE: Make sure you follow Knowledgeware format guidelines, If not, a message informs you that you cannot save the expression.
6. Click Save As on the Knowledgeware Expression dialog box to save the edited Knowledgeware expression.
