Opening a Template

PPR Manufacturing Hub Navigator allows you to open and edit templates that are stored in the Manufacturing Hub.

This scenario assumes that you have:

  • Access to the Manufacturing Hub
  • One or more templates saved in the Manufacturing Hub

Opening a Template

  1. Select File > Open Template from Manufacturing Hub, or select Open Template from Manufacturing Hub in the Manufacturing Hub toolbar.

  2. Log into the Manufacturing Hub when prompted:

  3. Select a template to open.

    Once logged in, a dialog box is presented which lists all of the templates currently stored in the Manufacturing Hub database:

    Any of the listed templates may be selected by clicking on the name of the project to highlight it, then selecting OK to load the selected template.

  4. The selected template is opened in PPR Manufacturing Hub Navigator:

    5. All of the capabilities of PPR Manufacturing Hub Navigator can be used with templates in the same manner as projects, including template structure navigation and using the PPR Editor commands to edit the template.
