Creating a Second Multi-Sections Surface

This task shows how to create the second multi-sections surface at the bottom of the part.

In the illustrations below, the first multi-sections surface and both sweeps have been hidden.

  1. Click Multi-Sections Surface .

    The Multi-Sections Surface Definition dialog box appears.

  2. Select the horizontal edges on the pads as sections for the multi-sections surface.

    Make sure arrows point the same way.

  3. Click OK to create the surface.

    The specification tree is updated to show the created surface.
