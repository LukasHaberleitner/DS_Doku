Installation Overview

A complete installation of Abaqus for CATIA V5 requires the components described below to be installed, in the order listed:

  1. Install the general availability release of CATIA V5 according to the installation procedure provided with the media.

  2. Install the CATIA V5 documentation.

    The Abaqus for CATIA V5 User's Guide is delivered as part of the CATIA V5 documentation.

  3. Choose the type of license server you want to use for your Abaqus for CATIA V5 installation and perform the steps specified:

    • If you plan to use FLEXnet licensing for your AFC installation, install the SIMULIA FLEXnet license server. Follow the instructions to install FLEXnet licensing from the Established Products GA assembly media. If you already have a SIMULIA FLEXnet license server for Abaqus or another SIMULIA product, you may reuse it for Abaqus for CATIA V5 licensing.

    • If you plan to use the Dassault Systï¿½mes License Server (DSLS), complete the remainder of the AFC installation without specifying a license server. After the installation is complete, set up DSLS as a Network License Client; for more information about setting up Network License Clients, refer to the “Common Installation and Administration Tools” section of the Installation & Deployment Guide in the V5 documentation collection.

      After you set up DSLS as a Network License Client, you can change the default value for the path of the client configuration file (but not the path of the license files) by setting the environment variable DSLS_CONFIG to the full pathname of the file. For example, on Windows:

      set DSLS_CONFIG=C:\SpecialProject\DSLicSrv.txt

      For more information about this environment variable, refer to the “File Locations, Settings and Registry Entries” section of the Dassault Systemes License User's Guide in the V5 documentation collection.

  4. For FLEXnet licensing, configure the licensing for Abaqus for CATIA V5 and, if needed, Abaqus for CATIA V5 Prerequisite.

    Detailed installation instructions for FLEXnet license configurations are provided in Configuring Licensing for Abaqus for CATIA V5 Prerequisite (ACP) and Abaqus for CATIA V5.

If you use AFC Complete to access Abaqus for CATIA V5 and you use FLEXnet licensing, you must set two environment variables before starting Abaqus for CATIA V5:

  • Set ABAQUSLM_LICENSE_FILE = @server, where server is the name of the license server where the AFC Complete license is installed.

  • Set AFC_LIC_SERVER_USE_ENV = true

These environment variables are not needed if you use a license for the CATIA Generative Part Structural Analysis (GPS) simulation product to access Abaqus for CATIA V5.
