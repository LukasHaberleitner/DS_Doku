Adding Geometry

This task shows you how to add geometry to the user surface of an annotation.
See 3D Annotations and Annotation Planes concept.
Open the Annotations_Part_02.CATPart document:
  • Improve the highlight of the related geometry, see Highlighting of the Related Geometry for 3D Annotation.

  1. Right-click the annotation as shown on the part and select the Associated Geometry > Geometry Connection Management from the contextual menu. 

    The Connection Management dialog box appears.
    In Graph of Geometrical Links, the feature linked to the selected annotation is displayed. If the linked feature is in Hide state, the last child feature in the specification tree in Show state and of the highest priority level is displayed.

  2. Check that Unique option is activated in Scope. 

  1. Right-click User Surface.13 in the Graph of Geometrical Links as shown and select the Add Geometry from the contextual menu. 

    You can also select Add multiple Geometries  to add several geomtries at a time.

     
      A new geometric component is added to the user surface User Surface.13 feature and prompt you to select the new geometrical element to be linked to the annotations.

  1. Select the surface as shown on the part.

    The Connection Management dialog box displays the new added geometry to the structure: Geometric Component.n1

          This task shows you how to add geometry to the user surface of an annotation.
          See 3D Annotations and Annotation Planes concept.
          Open the Annotations_Part_02.CATPart document:
          • Improve the highlight of the related geometry, see Highlighting of the Related Geometry for 3D Annotation.

              A new geometric component is added to the user surface User Surface.13 feature and prompt you to select the new geometrical element to be linked to the annotations.

          1. Select the surface as shown on the part.

            The Connection Management dialog box displays the new added geometry to the structure: Geometric Component.n1

          2. Click the Check Validity command to check the new geometry component validity relative to the selected annotation.

          3. Click OK.

          4. Right-click the annotation again and select the Geometry Connection Management contextual menu.

          5. Right-click the User Surface, and select Add multiple Geometries.

          6. Click   in the Tools Palette. All the geometrical components are automatically added under the User Surface node. The option in Tools Palette saves your time. You need not manually add the edges every time.

          1. Right-click Geometric Component.13 in the Graph of Geometrical Links as shown and select the Delete contextual menu.

          2. Click the Check Validity

            The Connection Management dialog box displays the updated structure.

          3. Click OK.

            Only one geometric component is now linked to the annotation.

          In Graph of Geometrical Links, the feature linked to the selected annotation is displayed. If the linked feature is in Hide state, the last child feature in the specification tree in Show state and of the highest priority level is displayed.

        2. Check that Unique option is activated in Scope Range. 

        1. Right-click Group of Surfaces.8 in the Graph of Geometrical Links as shown and select the Add Component contextual menu. 

          You can also select Add multiple Components  to add several components at a time. 
            A new component is added to the group of surfaces Group of Surfaces.8 feature and prompt you to select the new geometrical element to be linked to the annotations.

        1. Select the hole surface as shown on the part.

          The Connection Management dialog box displays the new connected geometry: Geometric Component.n1.

        2. Click the Check Validity command to check the new geometry component validity relative to the selected annotation.

        3. Select Group of Surfaces.8 in the Graph of Geometrical Links as shown and select Current option in Geometry Highlight.

          This option shows you the current group of surfaces of the selected annotation after the modification.

        1. Select now Initial option in Geometry Highlight.

          This option shows you the initial group of surfaces of the selected annotation before the modification.
          Options in Geometry Highlight are always applied to the selection in the Graph of Geometrical Links.

        1. Click OK.

          The four geometric components are now linked to the annotation.

        2. Right-click the annotation again and select the Geometry Connection Management contextual menu.

        1. Right-click User Surface.n2 in the Graph of Geometrical Links as shown and select the Rename contextual menu. 

          The Rename dialog box appears.

        2. Enter the new name: Hole Surface and Click OK.

          The feature has been renamed.
          You can remove a user surface from the group of surfaces using the Delete command from the contextual menu. 
          But as a group of surfaces is made of two user surfaces at least, it cannot be reduced to one user surface, thus the Delete command is disabled when a group of surfaces contains two user surfaces.

        3. Click OK.
