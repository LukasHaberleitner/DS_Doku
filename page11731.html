Creating a V5 Mechanism  

This procedure shows you how to create a new V5 mechanism. This command may be used to create mechanism corresponding to auxiliary NC resources (e.g., tool changer).

  1. From the NC Machine Tool Builder, click New Mechanism .

    A mechanism is added to the Applications leaf of the specification tree.

  2. Double-click Mechanism.1 leaf on the specification tree.

    The Kinematic Simulation dialog box appears.

  3. Enter a specific name for the mechanism.
