Annotating

These tasks are documented in the DMU Navigator User's Guide where more information on annotating can be found.

Add 3D annotations: Click the 3D Annotation icon, then click where you want to place the text, enter the text in the Annotation Text dialog box and click OK.

Create annotated views: Click the Create an Annotated View icon, then annotate the active view using commands in the DMU 2D Marker toolbar.

Manage annotated views: Click the Manage Annotated Views icon, then double-click the desired 2D view in the dialog box to recover it.
