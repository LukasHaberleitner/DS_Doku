Searching Fillets

This task shows you search fillets on a surface, and enable the detection of variable fillets on top of constant fillets, and of non-symmetrical G2 fillets.
Open CircleAnalysisP1.CATPart from the samples directory.

  1. Click Fillet Search     in the Modify Fillet Radius toolbar.
    If the check box Display detailed text in dialog in Tools/Options/Mechanical Design/Die Face Design is cleared,
    the dialog box looks like this:
     
    If the check box Display detailed text in dialog in Tools/Options/Mechanical Design/Die Face Design is selected,
    the dialog box looks like this:

  2. Select the Stamping Direction.
    The Z-Axis of the stamping direction is used to analyze the convexity of the fillets.

  3. Select the shape to analyze.
    Multi-selection is supported.

  4. Select a Convexity Analysis option.
    • Keep All
    • Concave Only
    • Convex Only.

  5. Click Apply.

    Fillet Search
    • Computes the fillets detected on the surface.
    • Displays the results as a list, with, for each fillet,
      • Its name
      • The values of the minimum and maximum radius.

  6. To sort the results in Geometrical Sets, using ranges of radius, open the Radius Ranges dialog box.
    • Click Create Range Set.
      A Geometrical Set named Fillet Search -xxmm to yymm is created.
    • Optional: Select a line and edit the minimum and maximum radius of the Geometrical Set.
    • Repeat as necessary.

  7. Back in the main dialog box, click OK.
    Fillets are created
    • A Geometrical Set is created in the specification tree.
    • It contains a Fillet Search feature, and the fillets sorted by type.
