Locating Related Products and Activities

The Center Graph command in the Lofting Tools toolbar allows you to:

  • Select a product and see the macro activity that creates it highlighted and centered in the PPR tree, or
  • Select an operation in the PPR tree and see the product that it produces centered and highlighted in the PPR tree
You must have already run Build Activity List that automatically generated all the operations for shell plate and shape fabrication.

Selecting a Product to Locate an Activity

  1. Select Center Graph in the Lofting Tools toolbar.

  2. Select a product in the either the 3D viewer or the PPR tree.

  3. The macro activity that creates the selected product is highlighted and centered in the PPR tree.
 

Selecting an Operation/Activity to Locate a Product

  1. Select Center Graph in the Lofting Tools toolbar.

  2. Select an operation or macro-activity in the PPR tree:

  3. The interim part/product the selected product is highlighted and centered in the PPR tree.
