Placing Annotation Breaks

This task explains how to place an annotation break along a route or on a component.
To place an annotation break along a route in your 2-D document:
1. Click the Place Annotation Break button .

The Catalog Browser displays.
2. Select Annotation Breaks on the Catalog Browser, and select a specific annotation break symbol.

Place the annotation break symbol on the route.
  You can annotate both sides of a break.

After placing an annotation break symbol on the route, click the Place Annotation Break button, and select the same break symbol from the Catalog Browser.

Select the break symbol in your document, and both sides of the break are annotated.
3. You can move an annotation break symbol along a route, using its slide manipulator ().

Select the annotation break symbol and the slide manipulator displays.

Drag the slide manipulator along the route to move the annotation break symbol.
  When you place an annotation break along a route, you break the route into two routes, as with the Break Route command. 

When deleting an annotation break, a message box displays, requiring you to determine whether to connect the two routes(similar to using the Close Route command,) or keep them as separate routes.

4. When you place an annotation break on a segment that can be pushed, and you select the break symbol or the segment, a push segment manipulator displays.

You can push the route segment as explained in Pushing Route Segments.
  You can replace an annotation break symbol with another symbol.

Click the Place Annotation Break button and select another break symbol from the Catalog Browser.

Select the break symbol in your document, and your new selection replaces the previous break symbol.

5. You can place an annotation break symbol on a component.

Click the Place Annotation Break button and select the component. The annotation break snaps to the closest connector.
