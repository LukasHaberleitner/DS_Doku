Splitting Compartments

This task shows you how to split a larger compartment by a functional plate, and create new compartments.
1.  With your document open, select the Split Compartment button .

The Compartment Split dialog box displays.
2. Select the compartment you want to split.
3. Select the functional plate used to split the compartment.

The functional plate must completely traverse the compartment. Otherwise, a message informs you that the selected functional plate cannot split the compartment correctly.
4. Click Preview to view the results, then click OK.
  The compartment is split by the functional plate (as shown below), and new compartments are created.
