Features    

The Features tab contains the following categories of options:

  • Attachment Lines
  • Alignment Lines
  • Added Material
  • Fit-Up
  • Edge Preparation Type
  • Opening Preparation
  • Plate Forming Operation

Attachment Lines

Smart naming

If this option is selected, the attachment line label contains the name of the associated product.

By default, this check box is selected.

Alignment Lines

Offset from attachment line extremities

This option specifies the default distance from the end of the attachment line at which alignment marks are located.

By default, this option is set to 150 mm.

Default extremity

This option specifies where you want the alignment mark applied. This option can take four values: None, Start extremity only, End extremity only, Both extremities.

By default, this option is set to Start extremity only.

Added Material

Margin to be added

This option specifies the default amount when adding material to plates in a joining operation.

By default, this option is set to 4 mm.

Fit-Up

Margin to be reduced by

This option specifies the default amount that is removed from fit-up products.

By default, this option is set to 2 mm.

Edge Preparation Type

Edge Prep Type

This option specifies the default type when preparing edges of plates and shapes.

By default, this option is set to Square.

Nesting package

This option specifies the nesting package used for bevel extraction.

By default, this option is set to ALMA.

Opening Preparation

Opening Prep Type

This option specifies the default type when preparing openings.

By default, this option is set to None.

 

Plate Forming Operation

Number of roll lines

This option determines the default number of roll lines when detailing plate forming operations.

By default, this option is set to 3.
