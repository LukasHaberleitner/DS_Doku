Geometry  

 

This page deals with the following options in the Geometry tab:

  • Geometry
  • Colors
 

Geometry  
 

Create associative use-edges

Select this option if you want to create use-edges, which are associated to the geometrical inputs.

By default, this check box is not selected.

The administrator can lock the Create associative use-edges option.
 
     
 

Colors  
 

Protected elements

Specifies the color used to display the associative use-edges.

By default, yellow color is selected.
