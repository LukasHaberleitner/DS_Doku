Command Description

Hide Allows to hide a set of elements according to their type, or whether they are currently selected or not
In Work Object See Scanning a Part and Defining In Work Objects
Parameterization Analysis... See Analyzing Using Parameterization
Customize... Allows customizing the workbench
Visualization Filters... Allows layer filters management
Options... Allows customizing settings
Standards... See Managing Standards in the Interactive Drafting documentation
