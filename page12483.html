Changing Activity Properties

This procedure explains how to change the properties of an activity.  It has two parts:  
 
  • The first part explains how to change the properties.
  • The second part provides information about each of the properties that may be changed.
     

Change Properties
 

  1. On the PPR tree, right-click on the activity whose properties need to be changed and select Properties.
    The Properties dialog box appears.

  2. Modify the parameters available for modification, and click OK.

 

Properties Tabs
The four tabs described in the table below can change or control an activity's properties: 
 
  • General
  • Specific
  • Items
  • Resources
  • Precedence
     

The General tab
Use this tab to:
 
  • Set the following parameters:
    • Specified cycle time
    • Specified beginning date
    • Icon path
  • Add a description
  • Change the activity's label, which appears on the PPR tree and on the PERT and Gantt charts.

The Precedence Tab

The Precedence tab is used to define rules which guarantee that activities occur in a specified order. Information about managing precedence constraints is provided in the section titled Precedence Constraints Management.

The Specific Tab
The Specific tab is used to add specific parameters to an activity in the context of the specific process of which it is a part.  For instance, in the example below, Transfer Time is being set at a value of two:

To create a new parameter:

  1. Type its name in the Attribute Name box.

  2. Select a type from the Attribute Type box.

  3. Click the Add button.

  4. Click the Apply button.

To remove an existing parameter:

  1. Select the parameter to be removed.

  2. Click the Remove button.

  3. Click the Apply button.

Note:

These parameters are only a subset of all the parameters than can be added using the Knowledgeware methods.

The Items and Resources Tabs
