Positioning Link Management after View Creation

You can align or orient a view from reference according to its parent (reference) view after the view creation.

This task shows you how to:

  • Create a Positioning Link
  • Modify an Existing Positioning Link
  • Replace the Reference View in a Positioning Link
  • Delete a Positioning Link
Open or create a 2D layout representation containing geometry.

Create a Positioning Link

You can create a positioning link for a view from reference.

  1. Right-click the view frame and select View Positioning > Create Positioning Link.

  2. In the layout, select the reference view with which the positioning link is to be created.

    The positioning link between the view from reference and the parent (reference) view is created only:
    • If a folding line exists.
    • If both the views are non-parallel.

Modify an Existing Positioning Link

You can modify an existing positioning link.

  1. Right-click the view frame and select View Positioning > Positioning Link on Reference View (XXX).

  2. Select the required option from the following:
    • For alignment:
      • No alignment
      • Perpendicular to folding direction
      • Along folding direction
    • For orientation:
      • No orientation
      • Orientation

If the Orientation option is selected, the Angle box in the View tab of the Properties dialog box is unavailable for modification.

Replace the Reference View in a Positioning Link

You can replace the reference view in a positioning link.

  1. Right-click the view frame and select View Positioning > Replace Reference View in Positioning Link.

  2. Select the required reference view.

Delete a Positioning Link

You can delete a positioning link.

Right-click the view frame and select View Positioning > Delete Positioning Link.
