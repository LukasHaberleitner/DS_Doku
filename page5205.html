Cache Mode

This section discusses ways in which you can design efficiently in cache mode, as well as tips about using it.

Because cache mode loads only the minimum amount of data required to view your document, certain functions work differently when you try to modify or create objects in cache mode. You may, for example, be required to select certain objects twice before the item is recognized as a valid select.  You may also have to move your mouse pointer over an object to confirm that it is a valid object before you can select it.
