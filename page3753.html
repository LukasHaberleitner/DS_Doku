Creating Points Using Intersection  

This task shows you how to create one or more points by intersecting curve type elements.
Open the Intersection_Point.CATPart document.
   

  1. Multi-select the elements to be used for intersecting.

 

 

  1. Click Intersection Point from the Profile toolbar (Point sub-toolbar).

  1. Select one curve type element with which the elements first selected will intersect and on which intersection points will be created.

 
The intersecting points automatically appear on the curve type element last selected.

 

 
The constraints appear on the condition that Geometrical Constraint is active in the Sketch tools toolbar.
