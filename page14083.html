Creating a Cam Direction

This task shows you how to analyze geometrical elements to create and optimize a cam direction.
Open the StampingDirection01.CATPart from the samples directory.

  1. Click Create Cam Direction   in the Process Part Separation toolbar.
    If the check box Display detailed text in dialog in Tools/Options/Mechanical Design/Die Face Design is cleared, the dialog box looks like this:


    If the check box Display detailed text in dialog in Tools/Options/Mechanical Design/Die Face Design is selected, the dialog box looks like this:


    When required, click Open to access the dialog boxes.

  2. Click Symmetry Constraint to access and define symmetry constraints for the cam direction.
    • Select Enable Symmetry.
    • Select a symmetry plane.
    • Select a reference axis.
    • Clear Enable Symmetry to remove the symmetry constraints.

  3. Select the axis system used as base axis.
    The convert to cam algorithm uses the axis system to compute the cam rotation axis.
    Only extrinsic rotations along Z and Y Axis of the press direction are applied to the edited cam or stamping direction.

  4. Select the elements to analyze.
    The selection determines the proposed analyses:
    • Draft and Depth analyses when only surfaces are selected.
    • Shear angle analysis when only wires are selected.
    • Draft, Depth, Shear angle, Normals, 2D/3D Cut when both surfaces and wires are selected.

  5. Manage the analyses:
    • Select an analysis from the list.
      If only wires are selected, a support surface may be required to perform some analyses.
      Some surfaces are automatically retrieved.
    • Click Detailed Analysis.
    • Follow the messages or labels to check and correct errors.
    • When necessary, edit the Options to modify the analysis behavior.

  6. Go to the Optimization section to optimize the cam direction with respect to the displayed analyses.
    • Cone Variation Angle: Defines an angle around the current position to compute the best position.
    • Center: Centers the axis origin on the analyzed elements. If surfaces and wires are selected, only curves are taken into account.
    • Mean Normal: Computes the mean normal of the elements to align the Z-Axis on this normal.
    • Convert to Cam: Converts the current position of the compass into a cam compatible solution.

  7. Click Automated Optimization to center the axis on the analyzed elements, find the best solution within a given cone variation angle and convert the computed best solution to a cam compatible solution.

  8. Go to Extract Geometry to convert the results of the displayed analysis to dead geometry.
    • Extract Areas: Results are stored as dead geometries and 3D labels.
    • Extract Direction: Extracts the Z-Axis on each analyzed point.
    • Extract Geometries As: Extracts geometries as selected.
    • Extract Values As: Extracts values as selected.
    • Extract Points Number Using: Extracts the points number with the selected method.

  9. Check the Cam Status.
