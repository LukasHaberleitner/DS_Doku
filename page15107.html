Working with a Drawing and a Product  

 

This task describes the methodology for working with a Drawing and a Product.
 

Reconciling an Assembly Drawing (Overwrite or Reload Mode)

  1. Open the file-based drawing and the assembly:

    • load the assembly structure

    • load the CATDrawing

    • the Update remains disabled

  2. Reconcile the Drawing and the assembly structure:

    • map the drawing and the assembly structure on the database.

    • apply Overwrite/Reload rule on the Drawing and Parts

  3. The Drawing is now updated.

  4. In the Product window, select File > Save, to save the Product in the database.

  5. Select Edit > Links to ensure the rerouting of the reloaded references.
    Each time an Edit > Links > Replace operation is performed on an assembly structure, the assembly structure is already available in CATIA V5.  

  6. Click Update to update the drawing.

  7. Save the drawing in the database.

Updating an Assembly Drawing and the Parts in the Database

  1. Apply the Overwrite rule to the Assembly drawing.

  2. Apply the Reload rule to the Part.

  3. Click Apply, to implement the reconciliation.

  4. Perform assembly drawing links resolution by using Edit > Links > Replace.

  5. Click Update to update the drawing.

  6. Save the documents in the database.

Updating an Assembly Drawing and the Parts Modified in the Database

  1. Apply the Overwrite rule to the Drawing.

  2. Apply the Overwrite rule to the Assembly.

  3. Apply the Reload rule to the Part.

  4. Click Apply, to implement the reconciliation.

  5. Perform assembly drawing links resolution by using Edit > Links > Replace.

  6. Click Update to update the drawing.

  7. Save the documents in the database.

  4. Click Apply, to implement the reconciliation.
