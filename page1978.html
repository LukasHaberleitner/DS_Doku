The GetWorkbench method applied to a ProductDocument object with the "SPAWorkbench" parameter value allows to retrieve a SPAWorkbench object. This objects allows to retrieve the differents collections of SpaceAnalysis objects but those methods are now deprecated. The object itslef is kept for possible future use.

The Inertia object associated to a product is retrieved using the GetTechnologicalObject method directly on the Product object with the "Inertia" parameter value for argument. 

The Sections collection, Clashes collection, ClashResults collection and Distances collection of a product are retrieved the same way with the corresponding "Sections", "Clashes", "ClashResults" and "Distances" values.
