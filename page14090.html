Computing a Global Unfold

This task shows you how to compute the unfolded contour of a surface on a plane to evaluate the size of the blank of a part to produce.
Open Unfold01.CATPart from the samples directory.
A SIMULIA installation is required to access the whole functionality.
SIMULIA installation must be R2021x FD06 or later, with QUF specific license.

  1. Click Global Unfold in the Process Part Separation  toolbar.
    If the check box Display detailed text in dialog in Tools/Options/Mechanical Design/Die Face Design is cleared,
    the dialog box looks like this:

    If the check box Display detailed text in dialog in Tools/Options/Mechanical Design/Die Face Design is selected, the dialog box looks like this:

  2. Select the surface to unfold.

  3. Specify the plane that supports the unfold.
    You can select either a plane, or an axis system, from which the XY plane is taken as support plane.

  4. Select an edge to freeze.
    Select an edge or a wireframe element of the surface, as long as possible, and in an area that is not deformed by stamping.
    The edge to freeze becomes the reference to position the unfold in the support plane. It is also used as fixed points for the one-step computation process.

  5. Select edges to transfer to their unfolded position.
    Edges are computed and created on the corresponding unfolded position on the support plane.

  6. Specify the computation mode.
    • Geometric computation
    • Material-based computation. It is a one-step computation process.
      Note that the computation requires that:
      • A material is attached to the surface or its parent.
        An already attached material is automatically selected when you select the surface to unfold.
      • The material has at least elastic properties.

  7. For a material-based computation, enter
    • The material name.
      Click the Material icon to display all availabe materials.
      Materials are defined using the SIMULIA format, in a Material Catalog file specified under Realistic Trim Line in the Tools > Options.
    • The material thickness
    • The thickness orientation in regard with the surface. It defines where the matter is found from the surface to unfold. It can be Side 1, Side 2 or Middle Surface. The direction of the matter is illustrated by an arrow.

    Click the orientation arrow to invert it.

  8. Under Advanced Options, select a Mesh quality.
    • Default
    • Coarse. Reduces the computation time, but provides approximate results.

  9. Under Realistic Unfold Options,
    • Select the Enable non converged results check box.
      It is selected by default.
      The result is kept if the convergence of the simulation reaches the 32nd iteration.
    • Enter a Convergence tolerance to control the display of the result areas where the convergence is not achieved, based on their last displacement.
      • Areas above the Convergence tolerance  are displayed in red.
      • Areas below the Convergence tolerance  are displayed in green.
    • Select the Improve trim line fidelity check box.
      It is selected by default.
      It activates the computation of a second type of result.
      Use the list to display the required type of result (No update required).

  10. Click Edit or Create Material Partition  to modify the information as explained in Defining Material Partitions.

The unfolded contour of the surface to unfold is displayed on the support plane, positioned from the frozen edge.
Transferred edges are aggregated under Global Unfold in the specification tree.
