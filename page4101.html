Using VOA Reporting

Licensing

  New VOA Report  is only available with a Vehicle Occupant Accommodation product license.
This task describes how to create a VOA Report.

See also: XML Settings for VOA
A VOA Report  is a report in HTML format containing the dimension values for the package, and the parameters and the status for the Posture Prediction.

  1. From the Vehicle Occupant Accommodation toolbar, click New VOA Report .
    The Reporting dialog box appears.

  2. Select the the package(s) and Posture Prediction(s) you want to have in the Report.

  3. Select VOA.xsl from the Available XSL Files list.

  4. Click OK.
    The VOA report is created in the directory you specified in "Saved Report File Path" with the name you specified at the "Report Name" field.

  5. Open the Report File from Your Windows Explorer.
    The VOA report looks like the following.
