Importing the IDF File

This task shows how to import the IDF data previously saved to a new document.
You can also import IDF data from an ECAD system.

  1. Select File > Open in the menu bar.

    The File Selection dialog box opens.

  2. Select .idf in the Files of type field.

  3. Select the DemoExport.idf file.

  4. Press Open.

    The Circuit Board Design Import window pops up.
    The DemoExport.idf file is already selected.

  5. Press OK.
    The Status bar shows the progression.
