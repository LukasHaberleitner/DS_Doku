Moving an Area
This task shows you how to move an area.
1. Make the parent element of the area active.

See Making an Element Active for instructions.
2. Select an element in the area.
The area axis is displayed.
3. Place the cursor over the axis and drag it to the new location.
  • Select one of the axis arrows to move along a plane.
  • Select the center of the axis to move in any direction.
