Creating a Network

This task shows you how to create a network.   
A network is a mechanism for organizing and grouping logical sets and routables. You can create a network (such as a Walkway Net), create a logical set under it, and then create pathways under them.  A network will display in the specifications tree only as an organizational element. The pathway will also display in your viewer.
1. Click on the Reservations Network button. The Reservations Network Creation dialog box is displayed.
2. Click on the down arrow and make a selection in the pull-down menu to designate your network as being of a certain type, i.e. walkway.
3. Click OK.
