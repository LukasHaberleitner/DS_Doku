Access from the Start Menu

This task describes how to access the Human Measurements Editor from the Start menu.
Your manikins must already be created via the Human Builder workbench.

  1. Select the manikin in the PPR tree.

     

  2. Select Start > Ergonomics Design & Analysis > Human Measurements Editor.

        
     

  3. The anthropometry is displayed for the selected manikin. You can modify any anthropometric value by double-clicking the corresponding arrow.
