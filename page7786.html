Using the Display Only Parents Option to Retrieve a Creation Context

The Display Only Parents button allows you to edit the functional specifications (mainly local modifiers such as drafts, chamfers and fillets features) using brep features by displaying the minimum creation context of these features. Pushing this button deactivates the feature and all the features which are not parents from it.

 
 
 

This enables you to select or unselect the brep features used by the feature.

 
Open the DisplayOnlyParents.CATPart.
The aim of the task is to change Spline 5 into a simple Line.

 

 

  1. Double-click Spline.5 in the specification tree to enter the Sketcher and delete it.

  2. Click Profile  and draw a line to join Point.5 and Point.6.

  3. Click Exit Workbench.

    The following dialog box is displayed.

  4. Edit EdgeFillet.7.

    After a warning message, the Edge Fillet Definition dialog box is displayed.

  5. Click Display Only Parents button.

    EdgeFillet.7, EdgeFillet.9 and Draft.3 are deactivated. Consequently, you are now able to select the missing edge.

  6. Select the two yellow lines. They turn red.

  7. Click OK in the Edge Fillet Definition dialog box.

    The EdgeFillet.7 line has disappeared from the update diagnosis dialog box.

  8. Edit EdgeFillet.9.

    After a warning message, the Edge Fillet Definition dialog box is displayed.

  9. Click OK.

  10. Edit Draft.3.

    After a warning message, the Draft Reflect Line Definition dialog box is displayed.

  11. Click Display Only Parents button. 

  12. Select EdgeFillet.9.

  13. Click OK.
