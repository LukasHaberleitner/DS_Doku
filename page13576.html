Toolbars

 

This section describes the various icons available in the 3 Axis Surface Machining workbench.

The toolbars

  • Machining Operations,

  • Tool Path Editor,

  • Probing Operations,

  • Machining Feature,

  • Geometry Management,

  • Minimum Tool Length,

are described below. When a toolbar contains icons belonging to another machining workbench, only the icon belonging to 3 Axis Surface Machining is described.

Information is also available by clicking on the links below:

  • Manufacturing Program Toolbar

  • Auxiliary Operations Toolbar

  • NC Output Management Toolbar

  • Machining Process

  • Manufacturing Program Optimization

  • Edge Selection Toolbar

  • Face Selection Toolbar
