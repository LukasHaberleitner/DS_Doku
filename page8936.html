Manipulating an Object

This task shows you how to select and manipulate the complete object.

  1. In the Modification toolbar, click Modification .

  2. Select the object to manipulate.

  3. In the Tools Palette, click Object Selection .
    Alternatively, you can press Alt + Ctrl + O.
    The Object Selection command is available when you select below commands in the Tools Palette:
    • Translation / Rotation / Affinity
    • Local Normals
    • Translation
    • Rotation
    • Affinity

  4. Manipulate the object and apply the modifications.
