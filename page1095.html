Stickers

 
  This tab deals with the following categories of options:
  • Default Image
  • Real Time Representation.

Default Image
 
This area lets you replace the default sticker image with your own image. 

Simply enter the path to the user-defined image or click the button to open the File Selection dialog box which lets you browse your folders to the desired location.
 

Real Time Representation
 
If this option is selected, it means that when a sticker is applied, its representation is displayed in the geometry area:

  However, note that when you select the sticker in the specification tree, the sticker representation is displayed even if the Activate real time representation check box is cleared.
  By default, this option is activated.
