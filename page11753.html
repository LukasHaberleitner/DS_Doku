Jogging with Home Positions

This procedure describes how to move a mechanism to one of its home positions.
Open the V4Axis_Machine.CATProduct file from the  online/mtbug_C2/samples/V5/4AxisMachine directory.  In addition:
  • add the new machine
  • add the machine kinematics definition.
  • create mount parts and points
  • create travel limits
  • create home positions

  1. Click Jog Mechanism .

    The Jog dialog box appears.

  2. In the Predefined Position area, select Zero_Home from the Home list.

    Observe how the mechanism moves to the specified configuration.

  3. In the Predefined Position area, select Home from the Home list.
