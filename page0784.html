Templates    

The Templates tab contains the following categories of options:

Plate Forming Templates

Minimum height

This option specifies the minimum height for plate forming templates.

By default, this option is set to 250 mm.

Profile Bending Operation

Templates

This option specifies whether or not templates are to be generated.

By default, this option is enabled.

Maximum length

This option determines whether to generate more than one template. If the length along the neutral axis is greater than the maximum length entered, more than one template is generated.

By default, this option is set to 6000mm.
