Defining the Working Zone

This task shows you how to define an area on a mesh before performing a subdivision, an extrusion, an association, etc. Selected vertices, edges or faces are displayed in the 3D area. Therefore only these elements can be modified. The other elements are protected and cannot be modified.

Open the Chair2.CATPart document.

  1. Click Working Zone Definition .

  2. Click the surface.
    A Tools Palette is displayed:
    The following icons let you:
    • select elements
    • swap the selection

  3. Select one or more elements using the Ctrl or Shift key.

  4. Click Selection Inversion to swap the selection of the selected elements.
    All protected elements become unprotected and vice-versa.

  5. Click Modification .
    Previously selected elements are displayed and protected ones are hidden so that they cannot be selected.
    You can now work within a limited zone.

    If you created several working zones, make sure to define the working zone or zones that you want to work on as current. Any new working zone becomes current by default.
    • To define a working zone as current, right-click it in the tree and select Set Working Zone as current.
    • To define two or more working zones as current, right-click them in the tree and select Add Working Zone as current. You can select multiple working zones and right-click one of them to define them all as current.
    • You can right-click a working zone and select Add elements to feature selection. If you then edit the subdivision surface, the elements of the working zone are included to the current selection.
    • Hover over an existing working zone element in the tree to highlight its included elements in the 3D area.
    • The Tools Palette of the Modification command appears.
    • Working Zone Activation/Deactivation is active by default; you can only modify the elements from the current working zone.

    Click Working Zone Activation/Deactivation to deactive the working zone and work on all of the elements of the subdivision surface.
    Do not click All Elements Selection to include all of the elements to a working zone. Deactive the working zone instead.

    • In the 3D area, only the elements included in the working zone are visible and can be modified.

  6. Deform the surface using the Transformation icons.

  7. Click Working Zone Activation/Deactivation in the Tools Palette to reset the protection of selected elements.
    All cells are displayed and can be modified.
    The Working Zone Activation/Deactivation icon only appears in the Tools Palette of the current command when the Working Zone Definition icon has been selected first. It is active by default. Click it to deactivate the working zone.

  • A protected face is a face with at least one protected vertex.
  • A protected edge is an edge with at least one protected vertex.
The working zone is only active during the session for each subdivision surface. Therefore, when you close a model and reopen it, all cells are visible again.
