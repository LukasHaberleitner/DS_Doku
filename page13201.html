Customizing Report Layouts

To customize the layout of a reconciliation report, you need to define an xsl document. This task shows you how to do so. To know how to generate a report, see Generating a Reconciliation Report.

  1. Select Tools > Options.
    The Options dialog box is displayed.

  2. Select Infrastructure > Product Structure in the left column of the dialog box.

  3. Click the Reconciliation tab.

  4. In the Output directory field of the Report Generation area, enter the directory path to define the directory where the html file will be generated.

  5. Enter the XSL file name in the Input XSL field.

  6. Click OK when done.
