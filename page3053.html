Orienting Faces

This task shows you how to select the orientation of faces by colors and reorient them.

More Reference Information is available in the Face Orientation Parameters chapter.

Use this functionality before computing a main pulling direction.
The model with faces to re-orient should be geometrically and topologically correct

  1. Open file FirstMoldedPart.CATPart in the samples directory. 

  2. Click Face Orientation in the Pulling Direction toolbar, select all the faces and change the Reference Direction to 0,1,0 (as in the picture below).

  3. Click Apply. The dialog box tells you that there are 113 faces in the model and that 77 of them need to be reoriented.

  4. Click to see the list of surfaces that need to be inverted and to modify the selection (if required).
    Click Close to return to the Surface Orientation dialog box.

  5. Click OK to invert the surfaces that need to be and to close the dialog box.
