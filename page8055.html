Interoperability with the Wireframe Workbench

You can work in the FreeStyle Shaper/FreeStyle Optimizer workbench only, creating forms and modifying them.
However in an industrial environment, it is always useful to have integrated design tools at one's disposal. The integration between the FreeStyle Shaper workbench and the Wireframe workbench is described in the chapter, based on four tasks illustrating how to create wireframe geometry using the Wireframe workbench.
