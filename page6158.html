Customizing

Before you start your first working session, you can customize the way you work to suit your habits. This type of customization is stored in permanent setting files: these settings will not be lost if you end your session.

  1. Select the Tools > Options command. The Options dialog box is displayed.

  2. Choose the Shape category in the  left-hand box, then Digitized Shape Editor.

    You will find the Digitized Shape Editor options in the following tabs:

    • General, that defines the Command Behavior and Warn Start.
    • Display Modes that defines display modes for clouds of points.

  3. Set options in these tabs according to your needs.

  4. Click OK when done.
