Defining Built-in Slots

This task explains you how to define a built-in end slot. This is usually done by an administrator.
For defining the built-in slot, you have to:
  • Create a Slot Reference
  • Add Slots to a Catalog

These steps apply to Ship Structure Detail Design application.

A slot is an opening created for the free passage of the profile. The built-in slot is created using a 3D Opening. The input to this opening is a SuperCurve. This SuperCurve is modeled using a set of parameters. The administrator defines these parameters and assigns appropriate name. The advantage of using a built-in slot is that the data size of the model is smaller. For more information about slot parameters, refer to About Slots.

Create a Slot Reference

  1. Click Start > Mechanical > Part Design.
    The New Part dialog box appears.

  2. In the Enter part name box, enter the slot name and click OK.
    The nomenclature for slot is <SlotType>_<ProfileType>. Here we named it as RECT_Tee.

    The RECT_Tee.CATPart is created.

  3. Under the geometrical set, define the parameters and formulas.

    1. Click Tools > Formulas to define the parameters and formulas.
    2. Create parameters.
      For more information about creating parameters, refer to CATIA Infrastructure User's Guide: Using Knowledgeware Capabilities: Parameters.
    3. Differentiate them using prefix Priv_ and Pub_ respectively.
      The parameters are created in the tree below the Parameters node.
      Only public parameters are visible while instantiating a slot.
    4. Create formulas using the private and public parameters.
      The formulas are defined to relate private parameters with public parameter values. These formulas are defined by the administrator.
      For more information about creating parameters, refer to CATIA Infrastructure User's Guide: Using Knowledgeware Capabilities: Relations: Formulas.

    The parameters and formulas are defined.

  4. Create a design table:

    1. Create an excel file or modify the excel file sample.
      Here we referred RECT_Tee.xls.
      This sample file is provided with the application. It is located in the following folder:
      .../OS/startup/EquipmentAndSystems/Structure/DetailingFeatures.
    2. In the excel file, enter the part number in the PartNumber column.
      The part number should be named as <SlotType>_<ProfileType>_<SectionName>.
    3. Create a design table which defines values for all types of sections for a given family using the excel file.
      Here we created the design table for family Tee.
      For more information about design table, refer to CATIA Infrastructure User's Guide: Using Knowledgeware Capabilities: Relations: Design Tables.
    4. Associate the design table with the respective part with the help of part number.
      The design table is added under the Relations node in the specification tree.
      The public parameters are listed in the design table.

  5. Create a power copy:

    1. Click Insert > Knowledge Templates > Power Copy.
      The Powercopy Definition dialog box appears.
    2. Select the geometrical set in the specification tree.
    3. Select the Parameters tab.
    4. Select the parameter which you want to publish.
    5. Select the Published Name check box and edit the parameter name by adding Pub_ as prefix.
    6. Click OK.

    The power copy is created.

  6. Click File > Save to save the slot reference.
    The slot reference is created and saved.

Add Slots to a Catalog
 A sample catalog, StructureDetailingFeatures.catalog is provided. This catalog, managed as a project resource at the structure discipline level, is located in the following folder:
.../OS/startup/EquipmentAndSystems/Structure/DetailingFeatures
You need to update the catalog as required, and store new features.
You also need to modify Chapter and Family information. For more information, refer to Using the Structure Detailing Features Catalog.

  1. Click Start > Infrastructure > Catalog Editor.
    You entered the Catalog Editor workbench.

  2. Open the existing catalog: StructureDetailingFeatures.catalog.

    The catalog editor opens showing the contents of the sample catalog.
    Every slot type should be associated with each profile type. There exist 12 combination:
    • RECT_Tee, RECT_Bulb, RECT_Angle
    • BOOT_Tee, BOOT_Bulb, BOOT_Angle
    • BULB_Tee, BULB_Bulb, BULB_Angle
    • FLAR_Tee, FLAR_Bulb, FLAR_Angle

  3. Activate the Tee node.

  4. Click Add Generative Part  in the Data toolbar.
    The Part Family Definition dialog box appears.

  5. Click Select Document.
    Select the required CATPart. The path of the selected part is displayed in the File Name box.

  6. Click OK.
    In the Reference tab, for every configuration of the design table, a part family configuration is created.
    In the Keyword tab, all the public parameters along with their values are listed.

  7. Click File > Save to save the part in the catalog.
