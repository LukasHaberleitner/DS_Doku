Placing End Cuts on Multiple Profiles

This task explains how to place end cuts on multiple profiles, such as stiffeners on a plate, stiffeners on a free edge, and beams.

The procedure, described below, applies to both the Structure Detail Design and Structure Functional Design applications.

You can place an end cut by using the End Cut command.
1. Open your 3D session containing the profiles requiring the end cut. In the example given below, end cuts need to be placed on two stiffeners on a plate, a beam, and a stiffener on free edge.
  2. Click the End Cut button . The End Cut dialog box displays.
  3. To select the two stiffeners on the plate, select the plate in your 3D session.

The two stiffeners are highlighted and their names (Profile), Extremities, and End Cuts are displayed in the End Cut dialog box.

Select the beam and the stiffener on free edge in the 3D session. Both these profiles are highlighted and their details are displayed in the End Cut dialog box.

You can also pre-select all the profiles and then click the End Cut button.

  3. Select the extremities, where you want to place the end cut. You can select the extremities in one of the following ways:

In the End Cut dialog box:

  • Press Ctrl and select each extremity manually.
  • Sort the extremities by clicking Extremity and then using the Shift key select the extremities.

In the 3D session:

  • Click to select each extremity manually. It gets highlighted in the End Cut dialog box.
  • Click close to one extremity. All the extremities in its close proximity get selected. These appear highlighted in the End Cut dialog box.
  4. Right-click one of the selected extremities in the End Cut dialog box or in the 3D session. Select Define End Cut.
  5. Select the end cut you want to place. For more information about types of end cuts that can be placed, see Placing End Cuts.

The name of the selected end cut appears in the End Cut dialog box. In the example given below, Web-Snipe-Radius-R1 end cut is placed on all the selected profiles.
