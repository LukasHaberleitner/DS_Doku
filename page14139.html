Performing a Bulk Upgrade

This task shows you how to quickly upgrade several features.
Activate a process part created with the wizard.

  1. Click Bulk Upgrade  in the Design Change toolbar.

  2. Select one processed element from the list.
    • Processed Elements lists the features updated within Bulk Upgrade.
      The list is lost when you exit.
    • Parent Features shows the parent feature of the selected feature.
      Use this list to navigate through the data model.

    A reframe is done on the selected feature. Only this feature and its parent of higher dimensions are shown.
    When you select a parent feature, a reframe is done, but the list is not updated unless you double-click the parent feature.

  3. Double-click a processed element to edit it.

  4. Select Display Previous Geometry.
    The geometry of the selected features is displayed as it was before update, or before an update error. It is displayed in purple, and is not pickable. It is especially helpful to find a feature the orientation of which has changed.

  5. Select an action:
    • Edit Selected Feature
    • Upgrade Selected Feature
    • Upgrade Part.
