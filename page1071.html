Part Design

The Part Design tab provides following check boxes for thread/tap and technological results.

  • Thread /Tap:
    • Display symbolic representation:

      shows a texture representing the thread/tap on the face of the part.
      By default, this check box is cleared.
    • Main diameter used as reference for standard management: Enables the main diameter mode in the part.
      By default, this check box is selected.
  • Technological Results:
    • Display Technological Result data as Knowledge Parameters: Retrieves the technological result data as knowledge parameters.
      By default, this check box is cleared.
  • Insertion Solid Feature:
    • Insertion in PartBody if surface tool in work object:

      If an element in a geometrical set is in work object then on loading a solid feature, a new warning message appears and then the feature is inserted in the first valid parent body of the current geometrical set. Select this check box to insert the feature in the Part Body.
