Customizing

  Before you start your first working session, you can customize the way you work to suit your habits.
This type of customization is stored in permanent setting files: these settings will not be lost if you end your session.

  1. Select the Tools  >  Options command.
    The Options dialog box opens.

  2. Select the Digital Mockup category in the left-hand box.

  3. Select the DMU Tolerancing Review sub-category.
    One tab is displayed:
    • The DMU Tolerancing tab lets you set the digital mockup tolerancing options.
    • The General tab lets you set general settings to be used in the DMU Tolerancing Review workbench.
    • The Visualization tab lets you customize the visualization of cutting plane and display the background and for highlighting the sub-elements.
    • The Geometry tab lets you set options that will be used when creating 2D geometry.
    • The Dimension tab lets you customize options that will be used when creating or re-positioning dimensions.

  4. Select the Infrastructure category in the left-hand box.

  5. Select the Product Structure sub-category.
    Various tabs are displayed, the Cache Management and Cgr Management tabs also interfere with DMU Tolerancing Review.

    • Cache Management

    • Cgr Management

  6. Select the General category in the left-hand box.

  7. Select the Display sub-category.
    Various tabs are displayed, the Navigation tab also interfere with Functional Tolerancing & Annotation.
