About DMU Data Flow Processes
Sharing a Mock-up with another Site: Select View  >  Commands List in the menu bar, select  DMUCacheExport  from the proposed list and click OK.
Studying a Variant: Select View  >  Commands List in the menu bar, select  SaveAsFrozen  from the proposed list and click OK.
Preparing a Design Review: Select View  >  Commands List in the menu bar, select  SaveAsFrozen  from the proposed list and click OK.
Archiving a CATProduct and related documents: Select View  >  Commands List in the menu bar, select  SaveAsFrozen  from the proposed list and click OK.
Sharing a Stand-alone Light Copy of a Mock-up: Select View  >  Commands List in the menu bar, select  SaveAsFrozen  from the proposed list and click OK.
