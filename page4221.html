Altering Thickness on Multiple Features

This procedure describes how to change the thickness on multiple features simultaneously.
This procedure assumes that you have created features.

  1. Click Change Thickness/CORTOL.

    The arrows for normal appear in green on the geometry; the Tools Palette and the Change Thickness/CORTOL dialog box appear.
    All the arrows represent features visible on the geometry from the current view; more features may be selected than are visible.

  2. Use one or more of the tools in the Tool Palette to select the features.

    The arrows showing normal turn purple.

  3. Select the Finish button.

  4. Select the Thickness check box and select a thickness.

  5. (Optional) Select the CORTOL check box, and select the values.

  6. Click OK.

    A Thickness Modification confirmation appears.

  7. Click Yes.
Respective measurement path, construct features and alignment will also get updated.
