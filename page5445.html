Applying a Laminate from a Main Stacking Sequence

This task shows you how to apply a Main Stacking Sequence to a cell of a virtual stacking.

  1. Select a virtual stacking column (cell or stacking area) in the table, or a cell in the graphic area.

  2. Click Apply Main Stacking Sequence Laminate in the Tools Palette.

  3. Select the laminate defined from the Main Stacking Sequence.
    The selected column is updated.
    If virtual sequences need to be added, they are added at the right position.
    The laminate is not stored in the virtual stacking.
