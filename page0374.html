Advanced Operations

The tasks in this section show you how to create advanced operations in your manufacturing program.

Advanced Finishing
Select the Advanced Finishing icon then select the geometry to be machined. Specify the tool to be used.

Specify machining parameters, feeds and speeds, and NC macros as needed.

The Advanced Finishing Machining user task illustrates some of this operation's capabilities.

Stiffener
Select the Stiffener icon then select the geometry to be machined (inclined or horizontal planes). Specify the tool to be used.
The Stiffener user task illustrates some of this operation's capabilities.
