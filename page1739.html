Get Item Behavior: Data Sheet Behavior   

Accessing this behavior This behavior can be accessed by clicking the Get item Behavior icon () in the List Management toolbar.
Describing this behavior This behavior enables you to retrieve one object from the list.
Using this Behavior Interface

Name:  Name attributed to the behavior.
Comment: Comment attributed to the behavior.
List List: Select the list from which the item will be retrieved.
  At position Enter the position of the item to be retrieved.
