Having Access to V4 Model Comment

This task shows you how to access the V4 Model Comment from CATIA Version 5.
Open the document MODEL_COMMENT.model.

From CATIA Version 5, you can have access to V4 Model Comment. For this, you need to follow these steps:

  • Select File > Open the Model of your choice.
  • Select one or more geometrical elements in the geometry area.
  • Select Edit > Properties.
  • Choose the Comment Tab, and you can visualize the Comment of the V4 Model.
For more information about V4 Model Transfer, refer to Transfer of a V4 Model Comment into a V5 CATProduct or a CATPart.
