The GetWorkbench method applied to a ProductDocument object with the "NavigatorWorkbench" parameter value allows to retrieve a NavigatorWorkbench object featuring a View method. This method allows to apply the viewpoint of an annotated view.

The GetWorkbench method applied to a ProductDocument object with the "SceneWorkbench" parameter value allows to retrieve a SceneWorkbench object featuring an aggregated Scenes collection. 

The GetTechnologicalObject method applied to a ProductDocument object with the "DMUReviews" parameter value allows to retrieve a DMUReviews collection.

Other DMU Navigator objects are retrieved using the GetTechnologicalObject method on the Product object with the following arguments:
