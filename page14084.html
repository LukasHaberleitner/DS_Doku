Defining Material Partitions

This task shows you how to define a material partion, by cutting a surface to create sections, and defining material properties for each section.

  1. Click Material Partition  in the Process Part Separation toolbar.


  2. Select the surface where the partition is created.
    Multi-selection is supported.
    A contextual menu is available.
    One Material Section is created and displayed.

  3. Select the cutting elements that split the surface.
    Multi-selection is supported.
    A contextual menu is available.
    More Material Sections are created.

  4. Select the side to apply the material.
    • Side 1
    • Side 2
    • Middle surface

    The side is indicated by arrows in the graphic area..

  5. In the table
    • Select one material section.
    • Click Apply material and select a material.
      The Material column is updated.
    • Enter the Thickness and the K-Factor of the material.
      The table is updated.

A Material  Partition feature is created under Partition Set. It contains the Material Sections.
Each Material Section contains its material properties parameters.
