Using Knowledgeware Rules

The procedures that follow describe how to use Knowledgeware Rules with inspection.  Knowledgeware rules are used to determine feature paths for measurements.  Whenever features are modified, the feature paths are modified according to the rules.

A default set of Knowledgeware rules is launched with Inspect.  The path to the default rules can be modified using Tools>Options>Inspect>General.

You can get further information about Knowledgeware by reading the Knowledgeware Advisor's User Guide and the Knowledgeware Expert User Guide.
