Manikin Reports in the PPR Tree

If you do not have an active product with a manikin, follow these steps to create a manikin.

  1. The Reports are stored in Reports in the Profile of the Manikin. Double-clicking on the Report 1 (1Analysis) opens the Report Definition dialog box.


     

  2. Double-click on the Reports node and the Report Definition Dialog appears showing the Selected analysis.

  3. Using this dialog box, the report analysis can be changed.

  4. Right-clicking on the Report1  in the PPR tree, the contextual window appears.

     

  5. The selections for the window are:
     

    • Open in New Window
    • Definition
    • Update
    • View
    • Define Input Parameters
    • Deactivate
