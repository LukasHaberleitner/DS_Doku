Using the Stacking Management in Selection Mode

This task shows you how to use the Stacking Management as a selection tool in other commands.
Impacted commands are:
  • Butt Splice Zone
  • Check Contour
  • Composites Corner
  • New Stacking Area
  • Flattening
  • Limit Contour
  • Material Excess
  • Multiple Core Samples
  • No Splice Zone
  • Numerical Analysis
  • Ply Data Export
  • Edit Producibility Table
  • Skin Swapping
  • Splice Plies
  • Unfold Composites Entity
Open a document with a stacking.
 
In this mode:
  • In the Tools Palette, only Select Row from Cell is available.
  • Display > Columns, Filters > Rows, Check Contours > 3D Contours Full Check, Review and Entity Preview are available.
  • Previously selected entities are highlighted in the table.
  • Review is possible on each entity level.

  1. In the dialog box of the command, click Multi-Selection of entities .

  2. In the dialog box that opens, click Stacking Management .

  3. Select the entities:
    • You can select several entities in one entity level.
    • You can select entities on each entity level.
    • You can use the filters to select entities.
    • Selected entities evolve with the entity level.
    • Review is possible on each entity level.
    • Previously selected entities are highlighted in the 2D Viewer.
    • Use Ctrl or Shift to select the required rows.
    • Or right-click a cell in the 2D Viewer and select Select/Deselect Row from Cell in the contextual menu to select it.
    • Or select a cell and click Select Row from Cell from the Tools Palette or use the space bar of the keyboard.
    • Click OK to validate and exit the dialog box.

    All the entities selected in the same level are stored. This selection is propagated to the original dialog box.

  4. If previously selected entities exist, use Ctrl to add or remove entities to the selection.
