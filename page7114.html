Export Data from Extrema

This task shows you how to transfer extrema (coordinates, type, value) into a .txt, .xls, or .xlsx file.

Only available with the ELFINI Structural Analysis (EST) product.

Open the sample35.CATAnalysis document from the samples directory.

  • Compute the solution.
  • Generate one image.
    In this particular example, click Von Mises Stress .
  • Create extrema.

  1. Right-click Extrema in the specification tree and select Export Data .

    The Export Data dialog box appears.

    • Output directory: lets you select the directory to which the data will be exported using the Choose Directory dialog box.
    • File name: lets you choose the desired file name.
    • File type: lets you choose the type of the file in which the data will be exported. This file type can be either .txt, .xls or .xlsx.

  2. Select the desired Output directory.

  3. Modify the file name if needed.

  4. In the File type list, select Text File [*.txt].

  5. Click OK.

    You can find below an extract of the exported data file:

    In the exported data table, you can find 3 types of data:

    • The coordinates of the extremum: x (mm), y (mm), z (mm) corresponding to the value's position (node, center of element...).
    • The type of extremum:Global_Maximum, Global_Minimum, Local_Maximum, Local_Minimum.
    • The value of the extremum.
