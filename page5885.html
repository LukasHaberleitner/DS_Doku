Simulation Toolbars

The Simulation toolbar has three sub-toolbars: Run Simulation, Compile Processes, and Update Cycle Times. The Process Simulation and Simulation Control Tools pop-up toolbars appear when you select the Process Simulation command. The Replay pop-up toolbar appears when you select the Replay command (from the Simulation toolbar) or Instant Replay command (from the Simulation Control Tools sub-toolbar). The Automatic Sync pop-up toolbar appears when you select the Automatic Synchronization command.

The hyperlinks below link to the procedures that describe how to use the commands in the Simulation toolbar and its associated pop-up and sub-toolbars.
