Defining a Single Insert Connector

This task explains how to add electrical behavior to a standard part to make it a single insert electrical connector.
The document is already open from the previous task.

  1. Click Define Connector .
    You are prompted to select the part.

  2. Select the part either in the specification tree or in the geometry area.

    The Connector Definition dialog box opens:
    Note: Keep Single Insert Connector as the Type.

  3. Click OK to validate.

  4. Save the document with another name: PartStd2.CATPart for example.

    This document now contains an electrical connector that you will insert later in the assembly.
