Using the Virtual Stacking Contextual Menus

This task shows you how to start commands from contextual menus instead of from the Tools Palette.
Either work on the part where you have defined cells or open the VirtualStackingGA01.CATPart document.

  1. Right-click a cell of the table of the 2D viewer.

    • A contextual menu is displayed:
      • Cell Valuation
      • Move row
      • Swap rows
      • Select Row from Cell

  1. Right-click a row of the table of the 2D viewer.

    • A contextual menu is displayed:
      • Single row edition
      • Multiple row edition
      • Move row
      • Swap rows
      • Duplicate row
      • Insert blank row
      • Cut Row
      • Copy Row
      • Paste Row

  2. Select the menu item your need.
