Compare Tool 

This tab appears only if you have a DELMIA license.

This page describes the options that determine the behavior of the Compare Processes command or tool. This command can be used on flat files or on files that originate in the Manufacturing Hub, although not all options are available for flat files.

The options include:

  • Compare Conditions for a Project
  • Compare Conditions for Processes
  • Compare Conditions for Products
  • Compare Conditions for Resources
  • Compare Conditions for Fasteners
  • 3D Coordinates for Comparison
  • Save to Manufacturing Hub

Compare Conditions for the Project

 

Use Compare Conditions of the Project
When you select this check box, the Compare Processes command uses the compare conditions defined for the project within DELMIA Process Engineer (DPE). If this option is not selected, the Compare Processes command uses the compare conditions as described below.

The default value is that the check box is not selected.

Compare Conditions for Processes

 

Use V5 Default Compare Conditions
If you do not have a project loaded from the Manufacturing Hub, this option is selected by default and cannot be changed. If you have a project loaded from the Manufacturing Hub, you can either leave this option selected or clear the check box.

The default value is that the check box is selected.

Check Also Comparability of Selected Nodes
In addition to comparing two processes, you can select one or more nodes within a PPR tree, and the Compare Processes command provides comparison data on the selected nodes.  You can select this check box whether or not you are using a project from the Manufacturing Hub or a V5 file, and regardless of the default compare conditions you have selected.

The default value is that the check box is not selected.

Use Default Compare Conditions (OEM Object UUID)
You can select this check box only if you are using a project from the Manufacturing Hub.  Under First Selected Object and Second Selected Object, each list contains all of the attributes that are common for all plantypes defined in the project. Using these lists, one, two, or three pairs of attributes may be used as compare conditions for processes.

The default value is that the check box is not selected.

Use 3D Coordinates for Comparison

 

Tolerance Value X, Y, Z
If you select the check box, you activate a position comparison for the fasteners. For the comparison, enter the tolerance values for each of the coordinates (x, y, z).

The default value is that the check box is not selected.

Save to Manufacturing Hub Project

The Save to Manufacturing Hub project button is only enabled for those Manufacturing Hub users that are members of the administrative group. This button saves the compare conditions defined in this dialog box back to the Manufacturing Hub project. Once saved to the project, the compare conditions defined here become the default for all users of the project. Each user may, however, redefine their own compare conditions via this dialog box.

The default value is that the options are not saved to the Manufacturing Hub.
