Visualizing the Related Surfaces 

This task shows you how to show and notice the surfaces which are referenced.

  1. Select the Related Surfaces option:
    All the surfaces, which were referenced or toleranced using the CATIA V4 FD&T function, are highlighted.

  2. Un-select the 3D Visualization option:

    The toleranced surfaces highlight has been turned off.

  3. Re-select the 3D Visualization option:

    The toleranced surfaces are highlighted again.
