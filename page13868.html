Getting Started

The following short tutorial provides an introduction to the CATIA Version 5 System Space Reservation product, It is intended to give you a feel for the product's capabilities in a few step-by-step scenarios, which are listed below.

Enter the Workbench
Create an Equipment Reservation
Set Correct Working Units and Grid
Changing the Current Axis
Saving Documents
Updating Documents

  These tasks can be completed in about 15 minutes.
