XML Settings for VOA

Licensing

This describes the functionalities for XML Settings for VOA.

Functionality:

  • Capability to automatically generate a report displaying the actual state of settings
  • Capability to automatically find differences between two different set of settings
  • Capability to compare settings configuration between two sites
  • To track evolutions between Releases
  • Capability to import/Export a settings configuration

  1. An XML setting file is in the Tools/Options/Human Builder/Vehicle Occupant Accommodation.
