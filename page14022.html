Enabling Multi-Sheet CATDrawing Documents Preview

You can now customize the CGM (Computer Graphics Metafile) format used by the SmarTeam viewer to view multi-sheet drawings.

This task shows you how to enable the preview of the different sheets of a CATDrawing document in the SmarTeam viewer.
This capability requires that SmarTeam Release 17 is installed.

  1. In CATIA select Tools > Options.
    The Options dialog box is displayed.

  2. Choose the General  > Compatibility category in the left-hand box.

  3. Click the Graphic Formats tab.

  4. Keep all default options.

  5. Select Save multisheet document in a single vectorial file.

  6. Click OK to validate.
    The next time you save drawings in SmarTeam, the customized CGM format is taken into account.

    Once in the viewer, the new behavior is as follows:

    • The default sheet displayed is the last saved sheet.

    • Contextual commands (Sheets  > Sheet.x) let you access the other sheets.

    • The Hide or Delete additional commands allow you to hide a sheet.
