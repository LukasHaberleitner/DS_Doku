Using This Guide

This book describes how to use the Version 5 Systems Routing product. Before you read it, you should be familiar with basic Version 5 concepts such as document windows, standard tool bars, and view tool bars.

To get the most out of this guide, you should start with the tutorial in the Getting Started section. It demonstrates how to create some basic elements of a plant in a hierarchical structure.

The remaining sections of the book describe in detail the procedures for using all of the features of the Systems Routing. The procedures are divided into user tasks and customization sections.
