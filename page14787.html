Insulation Thickness Rule

This task explains the insulation thickness design rule.

The insulation thickness design rule is used by the parts placement function. the thickness of the insulation is determined based on the information in the table.

This is a specification-related design rule and the associated table is located in the directory ...\intel_a\Startup\EquipmentAndSystems\XXX\InsulationSpecification.

1. The image below shows the insulation thickness design table.

The input information is nominal size and minimum and maximum temperature, which is a range. Based on these inputs the insulation thickness is determined. The temperature range is optional and the insulation thickness can be determined based on nominal size alone.
