Using the Controls Manipulation Mode  

This task describes how to use the controls manipulation mode.
Note that you can simulate controls manipulation only on positioned manikins.

  1. From the Vehicle Occupant Accommodation toolbar, click Controls Manipulation Mode .

  2. Select a manikin's arm.
    The compass moves to the steering wheel.

  3. Rotate the compass around its Z axis.
    The hands move in a circular motion.


  4. Select the right leg.
    The compass moves to the accelerator pedal rotation axis position.

  5. Rotating the compass around its Z axis
    The right toes move in a circular motion around the pedal's rotation axis.

  6. Click Controls Manipulation Mode again to quit this mode.
