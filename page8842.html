Modifying and Deforming Surfaces

This task shows you how to deform surfaces by manipulating the mesh. Three types of elements can be selected: vertices, edges, and faces.
There are several ways to deform a surface. You can either:
 
  • Select the element if the icon is active.
  • Select the element first then click the icon if it is not active.
  • Double-click the element in the specification tree or in the 3D geometry.

  1. Click Modification to display the Tools Palette toolbar.

    You can also access this command by pressing SPACE.

  • move along mesh lines

  • modify the weight of vertices or edges
 
  •  pick elements
 
  •  Automatic Merge: You can activate this option to merge vertices automatically. Click Edition to access the merging distance parameters. You can specify a value in any unit (mm, inch, etc.): it is converted automatically to mm. Select Relative Mode to define a ratio for the length of the merged edge. During the manipulation, a merge icon appears in the 3D area when a merge is possible.
