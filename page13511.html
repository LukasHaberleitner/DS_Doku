Creating a 4-Axis Curve Sweeping Operation

This task shows how to insert a 4-Axis Curve Sweeping operation in the program.
This type of Multi-Axis Machining is suitable for styling parts.

To create the operation you must define:

  • the geometry to be machined
  • the tool that will be used
  • the parameters of the machining strategy with the tool axis guided in Tangent Axis mode
  • the feedrates and spindle speeds
  • the macros (transition paths) .

See 4-Axis Curve Sweeping Parameters for more information on the parameters.
Open the 4AxisSweeping.CATPart document, then select Machining > Surface Machining from the Start menu.
Make the Manufacturing Program current in the specification tree.  

  1. Click 4-Axis Curve Sweeping .
    The 4-Axis Curve Sweeping dialog box appears directly at the Geometry tab page .
    This page includes a sensitive icon to help you specify the geometry to be machined.

  2. Click the red area representing the part to machine.
    The dialog box disappears.
    Select the part as shown below.

    Double-click anywhere in the 3D viewer to validate the selection and revert to the dialog box.
    The red area has turned green.

  3. Go to the Machining Strategy tab.
    Click the line in the sensitive icon and select the curve as shown below.

  4. Click Tool Path Replay.
    The tool path is displayed:

  5. Click OK to validate.
