A Warning Message When Server is Lost

Interoperability is performed between ENOVIA V5 VPM and external, file based applications.

 

The following applications are supported:

  • CATIA V424 Refresh 1,

  • CATIA V5,

  • ENOVIA V5 VPM DMU Navigator,

  • other applications supported in browsing mode.

 

Several levels are linked to these applications:

  • For individual documents:

    • Browsing: enables to extract a file from ENOVIA repository and send it to a viewer.

    • Editing: is available to applications supporting CATIA V4 and CATIA V5. Integration is then  performed in the application File / Save.

  • For assemblies:

    • DMU: enables the documents to be opened in context with the ability to modify assembly structure information.
Interoperability between ENOVIA and CATIA or using VPM Navigator is properly supported only between a client & server machine running in the same time-zone.
If a Backbone client does not answer after a lapse of time, the transaction is aborted.
