Using Swept Volume

Defining a Swept Volume
You generated a replay object. Click Swept Volume, then select the required settings in the dialog box displayed. Click Preview, then Save and save as a format type of your choice (cgr, wrl, .model)
Filtering Swept Volume Positions
Click Swept Volume, then select the required settings in the dialog box displayed, select the Filter Positions check box and enter a Filtering precision. Click Preview and Save as a .cgr file.
Defining a Swept Volume from any Moving Reference
Click Swept Volume, then select the required settings in the dialog box displayed, fill in the Reference field. Click Preview and Save as a .cgr file.
