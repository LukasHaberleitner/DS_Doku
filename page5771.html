Customizing DMU Navigator

This task explains how to customize your Digital Mockup Navigator settings.
This type of customization is stored in permanent setting files; these settings will not be lost if you end your session.
 

  1. Select the Tools  >  Options menu.
    The Options dialog box is displayed.

  2. Choose the Digital Mockup category in the left-hand box.
    The options for Digital Mockup settings appear, organized in tab pages.

  3. Click the General  tab.

    The General tab enables you to customize General DMU settings.
     

  4. Click the DMU Navigator tab.

    The DMU Navigator tab enables you to customize DMU Navigator settings such as hyperlink representation, marker default properties, scene default properties, spatial query parameters, etc.
     

  5. Click the DMU Marker tab.

    The DMU Marker tab enables you to customize DMU Navigator settings such as hyperlink representation, marker default properties, scene default properties, spatial query parameters, etc.
     

  6. Click the Multiprocess Settings tab.

    The Multiprocess Settings tab enables you to customize multi-process settings. These multi-processes concern the CATDMUUtility processes that are created as child processes for the tessellation of documents when performing a File  >  Open of a CATProduct or when running the CATDMUUtility batch process using the -mp option.
     

  7. Click the Overlay Settings tab.

    The Overlay Settings tab enables you to define an overlay that will warn partners that some parts have a certain level of confidentiality. 
     

  8. Click the 2D Workshop tab.

    The DMU 2D Workshop tab enables you to customize the background color of your 2D Workshop session.
     

  9. Four tabs in the Infrastructure  >  Product Structure category enable you to customize product structure-related settings.

     

     

    • Cache Management
    • Cgr Management
    • Product Visualization
    • Product Structure
       

  10. Two tabs in the Infrastructure  >  Display category enable you to customize display-related settings.

     

     

    • Visualization
    • Navigation
       

  11. Two tabs in the Infrastructure  >  Compatibility category enable you to customize compatibility-related settings.

     

     

    • External Formats
    • 3D XML
       

  12. Three tabs in the Infrastructure  >  Collaboration Infrastructure category enable you to customize collaboration-related settings.

     

     

    • Network
    • Identification
    • Connectivity
       

  13. Set options in these tabs according to your needs.

  14. Click OK in the Options dialog box to confirm the chosen options.
