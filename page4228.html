Creating a Default Knowledgeware Rule

This procedure describes how to use Knowledgeware Rules with inspection products.  Knowledgeware rules are used to determine measurement paths for inspection.

  1. From the menu, select Start>Inspection Offline to verify that all Inspection DLLs are available, then exit from Inspect.

  2. Go to Tools>Options>Infrastructure>Part Infrastructure>Display, and select the Parameters and Relations options.

    For more information, see Part Infrastructure for Knowledgeware Options in the V5 Infrastructure User's Guide.

  3. From the menu, select Start > Knowledgeware > Knowledgeware Expert.

  4. Click Expert Rule .

    The Rule Editor dialog box appears.

  5. Modify the name of the rule and select the OK button.

    The Rule Editor dialog box for the rule you created appears.

  6. Select the Push button.

    The Browser window appears.

  7. In the Dictionary column, select DNBInsFeatPackage to see parameters, such as DNBInsCircle, which are exposed by Inspect.

    (If the parameters are not immediately available, click on the DNBInsCircle.)

  8. Use the Rule Editor dialog box to enter the rules (see below). 

    You can double click on the parameters within the Browser dialog box, and the text will appear where the cursor was last in the Inspect Rules dialog box.

  9. Select the OK button on the Rule Editor dialog box when done.

  10. Create as many rules as required.

  11. Select the Close button on the Browser dialog box.

  12. Save the product file as InspectRules.CATProduct.
