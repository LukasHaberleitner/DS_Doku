Digital Process for Manufacturing

Before starting your first working session, you can set options needed to run the product or customize the way you work to suit your habits. This type of customization is stored in permanent settings files: these settings will not be lost if you end your session.

Before using Digital Process for Manufacturing for the first time, the Libraries option must be set to indicate the location where process libraries are stored, as well as which ones to use for the next new process document to be created.  

  1. From the menu bar, select Tools > Options. The Options dialog box appears.

  2. In the navigation panel on left side of the dialog box, select the Digital Process for Manufacturing workbench.

    The tabs for the selected workbench appear:

  3. These tabs allow you to define the following options:
    • Tree options, which includes the following options:
      • Hierarchy tree
      • Collapse/Expand Inactivation
      • Process Tree Synchronization
      • 3D Viewer Type in Process Documents
      • ProductList/ResourceList Display Order
    • Libraries options, including:
      • List of default libraries
      • Generate Process Identifier
    • Compare Tool options, including:
      • Compare Conditions for a Project
      • Compare Conditions for Processes
      • Compare Conditions for Products
      • Compare Conditions for Resources
      • Compare Conditions for Fasteners
      • 3D Coordinates for Comparison
      • Save to Manufacturing Hub
    • GANTT Chart display options, including:
      • Display Options
      • Display activity relations
    • Manufacturing Hub options, including:
      • Save Options
      • Load Options
      • Options offered by the following buttons:
        • Advanced Options
        • Load Relation Options
    • MBOM Viewer Display options, including:
      • MBOM Viewer Options
      • PPR Tree Options
      • Show Fasteners
    • Part Consumption options, including:
      • Part Relations
      • Highlight Unassigned Parts
      • Show Effectivity on Relation
      • Attributes on Relations
      • Modification Table
    • PERT Chart options, including:
      • Color
      • Type of Link
      • Inside a PERT Node...
      • Grid
      • Open PERT in ...
      • Description
    • PPR Navigator options:
      • Lifecycle Options
    • Product Flow Editor options:
      • Product Flow Viewer Setting
    • State Management options
    • Verification options, including:

      • PPR Tree Synchronization

      • Engineering Requirements

      • Output Relation

      • Navigation

  4. Click OK when done.
