Editing and Deleting End Cuts

This task explains how to edit and delete end cuts on profiles, such as stiffeners, stiffeners on free edge, and beams. Steps that are specific to stiffeners are indicated.

You can edit and delete end cuts by:

  • Using the Stiffener Dialog Box
  • Using the End Cut Command

To learn how to place end cuts, see Placing End Cuts.
 

Using the Stiffener Dialog Box

This method is used for editing and deleting end cuts on a stiffener only.
1. With your 3-D session open, determine the end cuts you will be editing or deleting.

 

2.

 To edit an end cut:

Double-click the stiffener with the end cut to be edited.

The Stiffener dialog box displays. See Using the Stiffener Dialog Box.

Right-click the End Cut to be edited, and select Define End Cut.

The End Cut dialog box displays.

  3. You can change the Type and End Cut, and modify Parameter values, as necessary. However, if you select a contextual end cut to be placed on a stiffener with no limits, an error message is displayed. A contextual end cut can be placed only on a stiffener with limits.

Click OK to return to the Stiffener dialog box, then click OK to close the Stiffener dialog box.
  4. To delete an end cut:

Double-click the stiffener with the end cut to be deleted.

The Stiffener dialog box displays. 

Right-click the End Cut to be deleted, and select Delete End Cut.

The end cut is deleted. In the image below, the Web Snipe Radius  end cut is deleted and a Short Point is displayed in the End Cut column. A Short Point is created when a stiffener limit is defined. In the image below, the stiffener End is limited by a shell.

Click OK to close the Stiffener dialog box.
 

Using the End Cut command
1. To edit the end cut:

Click the End Cut button . The End Cut dialog box displays.
  2. In your 3-D session, select the stiffener with the end cut. The Profile name, Extremities, and End Cut are displayed. In the image below, a stiffener with Web Snipe Radius R1 end cut is selected.

You can also select more than one profile. You can also pre-select the profile and then click the End Cut button.
  3. Right-click the end cut to be edited and select Define End Cut. You can change the end cut by selecting one of the given Cutback end cuts: Short Point, Long Point, Metal to Metal, and Weld.

If the stiffener does not have limits, then on selecting a cutback end cut, an error message is displayed in the End Cut dialog box. Right-click and select Refresh End Cut Column to start again. You can define the limit by selecting More. For more information, see Placing End Cuts.
  4. For more end cut options, select More. The End Cut dialog box displays.

You can change the Type and End Cut. For built-in and UDF end cuts (Standard and Contextual), you can also change the parameter values, as required.

Standard UDF end cut

For UDF contextual end cuts, you can also change the shape required as input by selecting a different shape in your 3-D session. The selected shape is displayed as Object in the Inputs tab.

For Cutback end cuts, such as Short Point, you can also change the limit of the stiffener.

Cutback end cut

     

Click the Inputs tab. The current limit is displayed as Object. Select a different limit in your 3-D session. The new limit is displayed in the Inputs tab and the end cut is now placed at this limit. In the image below, the short point limit is changed from LONG -5 to LONG -17.

       

  5. Click OK to close the End Cut dialog box.
  6. To delete an end cut:

Click the End Cut button , and then the stiffener.
  7. In the End Cut dialog box, right-click the end cut and select Delete End Cut.

The end cut is deleted and a Short Point end cut is placed on the stiffener. The Short Point indicates that a limit has been defined on the stiffener.
