Capturing the Display Area

You can define a capture point of view for a view/annotation plane and display it using the Display View command.
Open an existing 3D shape containing view/annotation planes.

  1. Right-click a view/annotation plane in the specification tree and select viewname > Display Area > Capture.
    The view/annotation plane is displayed parallel to the screen, oriented according to its normal.

  2. Right-click a view/annotation plane in the tree and select viewname > Display Area > Remove.
    The Display Area has been removed.

  3. Right-click the view in the specification tree and select Display View.
    The defined or captured display area is used to display the view, and the display area is automatically reframed to the captured area.
