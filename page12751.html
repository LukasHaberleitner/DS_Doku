Keeping Edges

You may need to explicitly indicate edges that should not be filleted, if a radius is too large for example. 
In this case you cannot select boundary edges to be kept, but only internal edges, i.e. edges limiting two faces.

Open the EdgeFillet2.CATPart document.
 

  1. Proceed as above, but once you have selected the edge to be filleted,
    Click More >> to expand the dialog box,
    then go to Edge(s) to keep and select the edge you wish to keep.


    This edge is displayed in pink, meaning that it will not be affected by the fillet operation.
If you have difficulties selecting the edge, use the up/down arrows to display the
preselection Navigator.
 

  1. Then, click OK to create the fillet surface.
