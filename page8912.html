Using Cyclic Toolbar and Menu Toolbar  

This task shows you how to use the cyclic toolbar and the menu toolbar.

A sphere has been created.
The Cyclic Menu is selected by default in Tools > Options > Shape > Imagine & Shape > General tab, Command Options Menu Mode area.

  1. Click Level Text Help in General Options toolbar. The icon state changes and different levels of help are displayed in the same icon cyclically. The current active level icon is always seen.

  2. Go to Tools > Options > Shape > Imagine & Shape > General tab, Command Options Menu Mode area and select the Toolbar Menu option.

  3. Click OK.
    The new option is applied only after you restart the CATIA session. A message indicating this is displayed.

  4. Click Level Text Help . You can see that the different levels of help are available in a expandable toolbar and the active level is indicated by an underline. Click the required icon in the expandable toolbar.

    Depending on the need you can select the required display mode: Cyclic Menu or Toolbar Menu.
  • These options are available for the following commands:
    • Level Text Help
    • Show Coord
    • Number of Sections
    • View and Selection mode
  • In the Toolbar Menu mode, if you restart CATIA, the expandable toolbar has the first icon as defined in the menu for the toolbar. The active level is indicated by an underline and may not necessarily be the first icon.
