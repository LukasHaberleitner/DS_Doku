Hybrid Design

Version 5 Release 14 introduces major enhancements to help you design parts. The underlying idea is to provide you with capabilities developed for gathering bi-dimensional and tri-dimensional elements within the same work environment. This is what is referred to as Hybrid Design.

Working in a hybrid design environment concretely means that you can create wireframe and surface features within the same body. This capability aims at having a quick understanding of parts creation.

This new way of designing therefore induces interfaces homogeneity between Part Design and Generative Shape Design, which makes the use of these products much easier.
