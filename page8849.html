Creating Elements Using the Compass

You can create wireframe elements, i.e. points, lines, or planes that are positioned by the compass.
Create a 3D shape containing a subdivision surface.

  1. From the Modification toolbar, click Modification .
    The Tools Palette appears.

  2. Click Robot Definition .

    You can also access this command by pressing Ctrl+Shift+C.

    The Tools Palette is modified.

  3. Click Wireframe Creation .

    You can also access this command by pressing Ctrl+Shift+Y.

  4. Move the compass to the required position.
    The position of the compass determines where the elements are to be created.

  5. Move the cursor to highlight a part of the compass:

    • If the cursor is close to the center of the compass, you can create a point.
    • If the cursor is close to an axis of the compass, you can create a line.
    • If the cursor is close to a plane of the compass, you can create a plane.

  6. Click to create the element.
    The element is added to the specification tree.
