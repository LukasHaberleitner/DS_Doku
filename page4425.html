Changing Manufacturing Context Appearance

The look of a Manufacturing Context can be defined by the Edit Visual Parameters of Manufacturing Context command. This command determines the color, transparency, and visibility of the products and resources that compose a Manufacturing Context.

Defining visual parameters for a Manufacturing Context
 

  1. Click Edit Visual Parameters of Manufacturing Context command in the Manufacturing Hub Commands toolbar.

    Note: If this command is executed before any Manufacturing Context has been computed, the following warning will be displayed to prompt the user to first compute a Manufacturing Context.

  2. The Define Visu Parameters dialog box is displayed:

    Set Color:
    This combo box defines the color to apply to the products and resources of the Manufacturing Context. 

    Transparency:
    This slider defines the transparency of the products and resources of the Manufacturing Context.

    Hide:
    When enabled, this check box hides the products and resources of the Manufacturing Context.

  3. Click OK button to apply these visualization parameters to the loaded Manufacturing context.
