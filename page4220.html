Defining a MGD

This procedure describes how define the Material Growth Direction (MGD) for a part.  This procedure applies to features that have positive value for thickness.  Once the MGD for a part has been defined, the MGD for that part is set when features are defined.

  1. Select MGD.

    The Material Growth Direction dialog box appears.

  2. Press the MGD button.

  3. Select a point on the part.

    • To see the part in its own window, select the Show Preview Window check box. If the part or the point on the part is not what you want, click the preview window's Reject button; otherwise, click Accept.
    • To hide all parts that already have MGD defined, select the Hide all parts having MGD defined check box.

  4. Click on the part.

    A red arrow appears showing the direction of material growth.

  5. Select the line with the part name on the dialog box.

    All of the buttons become available.

  6. In the Thickness box, type the value or use the arrows to change the value.

  7. Click the Flip MGD button, if appropriate.

    The red arrow showing MGD will flip directions.

  8. Click OK.
Once you have performed this procedure, all the features on the part selected will, by default, have the correct MGD and thickness.
You can use the MGD command and this dialog box to alter a previously set thickness or to remove a thickness from a part.  To remove a previously determined thickness, select the appropriate line in the Material Growth Direction dialog box and click on the Remove button.
