Copying / Pasting an ENOVIA LCA Product on a File Product

This task shows you when it is possible or not to copy / paste an ENOVIA LCA Product on another file or ENOVIA product.

  1. Create an exploded Product Structure in LCA:

  2. Load the Product Structure to CATIA V5:

  3. Create a new CATIA Product:

  4. Copy the first instance levels of the PRC:

  5. Paste them to the new CATIA Product:
It is not allowed to paste an ENOVIA LCA Product (or PLM) on a file Product:
  Note: If the PDM properties of the new Product is changed to Structure exposed, the Pasting is possible.
To sum up:

In a file Product:

  • It is possible to copy / paste a file Product,
  • It is not possible to copy / paste an ENOVIA LCA Product.

In an ENOVIA Product:

  • It is possible to copy / paste a file Product,
  • It is possible to copy / paste an ENOVIA LCA Product.
   
For more information about Cutting / Copying / Pasting objects in the Product Structure and Paste Special, please refer to "Reusing your Product Structure", in the Product Structure User's Guide.
