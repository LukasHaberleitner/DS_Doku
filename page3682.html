Annotation Parameters

The annotation parameters are located in the Annotation node of the Standard Definition editor. They deal with the position of text leaders and annotation blanking parameters. 

Note: The parameters which allow you to customize annotation leader symbols are located in the Dimension node of the standard editor. The parameters located in the Dress-Up node let you customize the appearance of dress-up elements, such as markup arrows and threads.

 
These parameters depend on a given parent standard.
