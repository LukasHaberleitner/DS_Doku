Browsing GVS Parameters

 This task shows you how to browse GVS parameters, which are stored in an XML file. You must be an administrator to change parameter values.
The FabSketchDocument_PlateStyle.xml file will be browsed.

  1. Select the Standards... command from the Tools  menu.

    The Standard Definition dialog box appears.

  2. Select the generativeparameters category.

  3. From the file list, select the FabSketchDocument_PlateStyle.xml file to display the default properties defined by your administrator.

  4. Click the plus sign in the left-hand side to expand the XML tree structure and browse all properties.

  5. Select the desired property to display the corresponding value on the right-hand side of the dialog box.

  6. Click OK to close when done.
