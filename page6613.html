Administration

This page deals with the administration option allowing you to use the Generative View Style with Electrical Harness Flattening:

Generative View Style

Prevent generative view style creation

Click to clear this option to be able to select a generative view style in the View creation commands.
The Generative View Style toolbar is displayed.
For more information, refer to the customizing section about generative view style.

By default, this option is selected.
