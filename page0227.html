Advanced Selection Options for Supports

You can right-click in the Support field to reveal options that can streamline the selection of components for use as a support for an Abaqus for CATIA V5 feature. The support management options include the following:

Remove All Supports

The Remove All Supports method clears all components you have selected for the support in the currently selected dialog box.

Extend Selected Face

The Extend Selected Face method enables you to select multiple faces as the support for a surface-based feature. For each face you select, Abaqus for CATIA V5 selects all faces in the model that are within a user-specified angle from the selected face and connected to it. You can also refine your selection from the Extend Selected Face options by selecting Adjacent faces only, Meshed surfaces only, or both of these options.

The Extend Selected Face option is available from the Support field for definition of the following boundary condition and load features:

  • Displacement BC

  • Clamp BC

  • Velocity BC

  • Acceleration BC

  • Submodel BC

  • Temperature BC

  • Pressure Load

  • Distributed Load

  • Load Density

  • Surface Heat Flux

  • Film Condition

For rigid couplings and smooth couplings, you can use the Extend Selected Face method to define the faces in the primary Support field only. The Extend Selected Face method is compatible with surface groups specified in Abaqus for CATIA V5 or from other CATIA analysis cases.
