Display Properties

This reference describes the Display properties concerning:
  • Text.
  • Flagnote.
  • Surface Texture.
  • Note Object Attribute.
 
  This page deals with the properties concerning:
  • The 3D Display.
 

3D Display
 
 

Parallel To Screen
Defines whether the annotation is displayed in a plane parallel to the screen monitor or not.
By default, this option is not selected.
The Parallel to Screen option is not available for Weld annotation.
 

Zoomable
Defines whether the annotation keeps its aspect ratio independently of the document zoom or not. This property is only available when the Parallel To Screen property is selected.
By default, this option is not selected.
