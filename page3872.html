Manufacturing Properties

  • Surface Texture

Engineering Specifications


For the annotations under annotation set with engineering properties, you can specify the engineering specifications based on which the manufacturing requirements are generated.
 
 

To Manufacture 

For annotations that belong to engineering specifications, the check box defines whether an annotation is consumable or not, i.e. whether an engineering intent to be consumed while manufacturing or not.
When this check box is selected, the requirements specifications are generated for that annotation.

By default, this check box is clear.
 

Manufacturing Specifications
  For annotations that belong to engineering specifications, the check box defines whether an annotation is consumable or not, i.e. whether an engineering intent to be consumed while manufacturing or not.
When this check box is selected, the requirement specification are generated for that annotation feature.  
 

Manufactured

For annotations that belong to manufacturing specifications, the check box defines whether an annotation is consumable or not, i.e. whether an engineering intent to be consumed while manufacturing or not.
When this check box is selected, the feature is considered as manufactured and, thus, the requirement specifications are not generated for that annotation.

  By default, this check box is clear.
