Integration with the Drafting Workbench

  Export a drawing view to a layout: create, from a selected view in a drawing, a layout view containing annotations, dress-up elements and geometry.
  Create drawings and drawing views from a layout: create a drawing from a layout, and create a drawing view from a layout design view or isometric view.
  Importing DXF to 2D Layout for 3D Design: create a DXF/DWG file, and import it to 2D Layout.
  Migrating Draw Data from .Model Files to a Layout: create a .model file, and migrate it to 2D Layout.
