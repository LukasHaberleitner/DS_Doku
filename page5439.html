Creating a New Stacking Area in the Virtual Stacking

This task shows you how to create new stacking areas in the virtual stacking.
Either work on the part where you have defined cells or open the VirtualStackingGA01.CATPart document.

  1. Make sure the View mode is set to Stacking areas.

  2. Click New Stacking Area in the Tools Palette.

  3. In the dialog box that opens:

    • In the Initialize from list, select the stacking area that will be used to initialize the new one.
      The Stacking sequence list is updated accordingly.
    • In the authoring window, select the cells that will be part of the stacking area.
      Multi-selection of entities is available.
    • Right-click the stacking sequence and decide to add or remove a ply.
    • Click OK when you are done.
      • The virtual stacking is updated with a new stacking area (new plies are listed).
      • If a sequence contains more than one ply, the name will be SA.i where is is the number of plies.
