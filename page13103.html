Creating a Rolling Offset

This task shows you how to create a rolling curve at a required offset distance from a reference curve.

Open the RollingOffset1.CATPart document.

  1. Click Rolling Offset in the Wireframe toolbar (Curves Offset sub-toolbar).

    The Rolling Offset Definition dialog box appears.

  2. In the Curves box, select the reference curve to be offset.

    You can select several curves to be offset by clicking the icon.

  3. In the Support box, select the support element.

    Here, we selected the xy plane. If the reference curves lie on a plane, the Support box is automatically
    filled.

  4. In the Offset box, type the value or use the arrow to change the value.
    Here we entered 5mm as the offset value.

  5. Click Preview to see the offset curve.

  6. Click OK.

    The curve (identified as Rolling offset.xxx) is added to the specification tree.
