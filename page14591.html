Copying Attributes    

This task explains how to copy selected attributes from one object to another.
The two objects must have common attributes. The objects must have the same domain, same attribute name, same usage, same data type, and the attributes of the target object must be writable.
1. Click the Copy Attributes button and select the object to copy the attributes to (the target).
2. Click the object from which you are copying from, and the Copy Common Attributes dialog box displays.
3. Select the attributes under the Attribute Name column that you want to copy. Select as many as appropriate, and click OK.

The dialog box closes and the attributes are copied to the first object you selected.
