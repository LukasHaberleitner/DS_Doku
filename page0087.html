Abaqus Licensing

Abaqus for CATIA V5 and Abaqus for CATIA V5 Prerequisite run Abaqus jobs by using a license server, which is a process running on a single computer (license server host) on a network. You can use either the FLEXnet license manager or the Dassault Systï¿½mes License Server (DSLS). Network licensing allows a limited number of Abaqus for CATIA V5 and Abaqus for CATIA V5 Prerequisite sessions to run simultaneously on any supported computer on a network. The license file is a text file that stores licensing data.

Note:  Abaqus for CATIA V5 licensing consists of an equal number of tokens for the Nonlinear Structural Analysis workbench and the Thermal Analysis workbench. One token is checked out when you enter either workbench, which is equivalent to one license of Abaqus for CATIA V5. If you have installed Abaqus for CATIA V5 Prerequisite, one token is checked out when you start a CATIA V5 session, which is equivalent to one license of Abaqus for CATIA V5 Prerequisite.
