Getting Started

This tutorial will guide you step-by-step through your first Advanced Meshing Tools session, allowing you to get acquainted with the product. You just need to follow the instructions as you progress.

Before starting this tutorial, you should be familiar with the basic commands common to all workbenches.

The main tasks proposed in this section are:

Entering the Advanced Meshing Tools Workbench
Defining the Surface Mesh Parameters
Setting Constraints and Nodes
Launching the Mesh Operation
Analyzing Element Quality
Mesh Editing
Re-meshing a Domain

 

Altogether, this scenario should take about 15 minutes to complete.
