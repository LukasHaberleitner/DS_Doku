Tolerancing Schema Catalog 

This section gives more information on creation of tolerancing schema catalog. The following topics are described:

  • User Features Tolerances Administration Toolbar

  • Upgrade of Reference User Features
  • Create Tolerancing Schema
  • Create Catalog of Tolerancing Schema
  • Path to Tolerancing Schema Catalog

The described procedures can be performed only by Administrator.

To automatically tolerance user features see Tolerancing User Features Automatically.
 

User Feature Tolerances Administration Toolbar  
 

The User Feature Tolerances Administration toolbar is available in Administrator mode. The following commands are available:

  • User Feature Faces Identifiers Creation
  • User Feature Tolerancing Schema Creation
 

Upgrade of Reference User Features

 

The automatic tolerance creation of user features requires a complete mapping between the faces, edges and vertexes of the reference tolerancing schema and the user feature or technological result of user feature that is to be toleranced.

  1. Select the reference user features and click User Feature Face Identifiers Creation or vice-versa.
    The Upgrade report message is displayed confirming the upgrade and asking you to edit the user feature to update the internal components.

  2. To edit the user feature, right-click it and select XXX.object > Definition or double-click it in the specification tree.
    The Userfeature Definition dialog box is displayed.

  3. Click Change/Update Components.
    The UDF Edition warning message is displayed asking you if you want to proceed.

  4. Click Yes in the message box.

  5. Click OK in the Userfeature Definition dialog box.

  6. Save the changes.
 

Create Tolerancing Schema

 

The tolerancing schema is defined from an existing annotation set in the part.

Select the annotation set and click User Feature Tolerancing Schema Creation or vice-versa to convert it to tolerancing schema. This command is not available if the annotation set is already converted to tolerancing schema.

The following rules have to be checked for successful creation of tolerancing schema:

  • The annotation set belongs to a part and is not an assembly annotation set or a result annotation set.
  • All the features are valid.
  • There is no automatic selection default annotation in the annotation set.
  • There is no capture that manages the display of geometrical sets and/or bodies.
  • There is no or only one Datum Reference Frame fully defined on geometrical elements that are not part of the user feature.
  • The Datum Reference Frame is not defined using a mix of geometrical elements from the user feature and external geometry of the user feature.
  • The annotation set features that are linked to the geometry (except the datum features of the external Datum Reference Frame) are applied only to the geometrical elements of the user feature and/or automatic FTA construction geometry features.

On running this command one of the following messages is displayed depending on the schema creation status:

  • If the schema has been successfully created:
    Schema successfully created.
  • If already created schema has been modified and needs a update:
    Schema already created, is updated.
  • If already created schema has no change:
    Schema already created, remains unchanged.
  • If schema cannot be created as it does not follow the above listed rules. The failed rule is displayed as the message. For example:
    Some features are not linked to the user feature. User feature not completely upgraded.

When the schema is successfully created or updated as in the first two cases, click OK to add this schema in the catalog. The catalog is created if it did not exist. If the catalog is already created then it is updated.
 

Create Catalog of Tolerancing Schema

 

To allow multiple users to use this tolerancing schema, it has to be stored in a catalog.

For more information on catalog creation, see the Component Catalog Editor User's Guide.
 

Path to Tolerancing Schema Catalog

  The path to the tolerancing schema catalog is given in Tools > Options > Mechanical Design > Functional Tolerancing & Annotation > Tolerancing tab, User Features Tolerancing Schemas Catalog.
