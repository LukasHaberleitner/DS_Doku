Creating a Drawing Document from a Template

This task shows you how to create a drawing document from a template.

  1. Run a query to access your CATPart document.

  2. Select SmarTeam > New From...
    The New From...Documents dialog box is displayed.

  3. In the displayed tree, double-click on the folder of interest or select the + sign to open the folder.

  4. Select an existing drawing.
    A new drawing is created starting from the selected template.
    Each drawing document is represented in the database as a tree. The root object of the tree represents the drawing itself then each sheet of the drawing is represented as a child of the drawing.

    If you want to verify the document before opening it, select the Viewer tab to preview it.

    It is now possible to see the different sheets of a CATDrawing document in the SmarTeam viewer. For more information, see Enabling Multi-Sheet CATDrawing Documents Preview.

  5. Click OK.

  6. Insert in your drawing the projection views of the CATPart document and insert the dimensions.

    For more information about projections and dimensions, see the CATIA - Generative Drafting User's Guide.

No Template Project is Defined

In case no templates project is defined, selecting File > New From... launches the standard SmarTeam open dialog allowing you to select any document from the database.
