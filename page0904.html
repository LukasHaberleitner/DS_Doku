Display

This page deals with Display options in the following tabs:
  • the Tree Appearance tab lets you set options for displaying the specification tree
  • the Tree Manipulation tab lets you set options for manipulating the specification tree
  • the Navigation tab lets you customize navigation options
  • the Performance tab lets you customize performance settings
  • the Visualization tab lets you customize visualization settings
  • the Thickness & Font tab lets you set thickness properties
  • the Layer Filter tab lets you set options for layer filters
  • the Linetype tab lets you set linetype properties.
