Customizing

Before you start your first working session, you can set options needed to run the product or customize the way you work to suit your habits.

This type of customization is stored in permanent settings files: these settings will not be lost if you end your session.

  1. From the menu bar, select Tools>Options.

    The Options dialog box appears.

  2. On the navigation bar on left of the dialog box, find the Machining Simulation category.

    If the category is not expanded (i.e., if there is a plus sign next to it), expand it by clicking on the plus sign.

  3. Click on the Machining Path Simulation>NC Machine Simulation workbench.

    The tabs for the workbench appear.
    These tabs enable you to define the following options:
    • General, lets you define the path to any D5 VNC data.
    • Simulation, which includes the following options:

      • Tool Path Display

      • Tool Path Modification

      • Integrated Material Removal

  4. On the navigation bar, select the General category and the Compatibility sub-category.

    The tabs appear.
    Select the DELMIA D5 tab (you may have to use navigation arrows to see the tab), which enables you to set the configuration files.

  5. On the navigation bar, select the Machining category.

    The tabs appear.
    Select the Output tab, which enables you to set the post-processing options.

  6. Set options in these tabs according to your needs.

  7. Click OK.
