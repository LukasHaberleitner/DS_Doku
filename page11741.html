Defining Machine Attributes

Once the machine kinematics are defined, the following attributes on the machine also must be defined for the machine definition to be complete and for the machine to be usable by other workbenches:

  • Workpiece and tool mount part and point
  • Travel limits on each of the machine joints
  • Home Position
  • Tool Change Position
  • Define Axis Names
  • Speed and Acceleration limits.

The procedures that follow describe how to do these tasks.
