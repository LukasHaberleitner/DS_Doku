Creating Surface Texture Symbols 

This task shows you how to create a surface texture symbol annotation.
Open the Annotations_Part_04.CATPart document.

  1. Select the Front View.1 annotation plane.

  2. Click the Surface Texture  in Annotations toolbar.

  1. Select the surface as shown on the part.

    This scenario illustrates the creation of a surface texture symbol by selecting geometry,
    but you can also select any Part Design or Generative Shape Design feature in the specification tree.
    In this case, the created annotation will not be attached to the selected feature,
    but to its geometrical elements at the highest level.

  1. The Surface Texture Symbol dialog box appears.

    The new surface texture symbol allows you to respect the following standard parameters:
    • ISO
    • ASME
    • ANSI
    • JIS

    You can modify these the standard parameters using:
    Tools > Standards... command, then selecting the Drafting category.
    The old surface texture symbols created in previous document are migrated at edition.
    If you slide a new surface texture symbol outside its reference, an extension line is added,
    its overrun is fixed to 2mm and cannot be modified.

    The Tools Palette is displayed according to the type of face selected depending on the canonicity. In this scenario the Tools Palette is not used. For more information, refer to Propagating Geometry Selection for Feature Creation.

  2. Select Ra symbol and enter a value: 0.8

  1. Click OK.

    The surface texture symbol is created without a leader.
    To add one, see Adding Leaders and Using Breakpoints.
    • While creating or modifying the surface texture symbol, to reverse the direction of the symbol, click the red manipulator .
    • The manipulator is not available when there is a leader added to the surface texture annotation.

About Surface Texture

  • By default, there is a 1 millimeter space between the geometry and the extension line. There is also a 1 millimeter space between the end of the extension line and the surface texture. These values cannot be customized.
  • The default parameters of the surface texture are 1 for thickness and solid for line type. These values cannot be customized.
    For more information, see Surface Texture Parameters and Styles.
  • You can modify the surface texture at any time. To do so, double-click the surface texture and enter the required modifications in the Surface Texture dialog box.
  • If you have selected the Use style values to create new objects option in Tools> Options> Mechanical Design> Drafting> Administration tab, the Surface Texture dialog box is pre-filled with custom style values (as defined in the Standards Editor). In this case, Properties toolbars and the Tools Palette are disabled during the creation of the surface texture.
    However, if you have not selected this option, the Surface Texture dialog box is pre-filled with the last entered values (if any). In this case, the Properties toolbar and the Tools Palette are active during the creation of the surface texture.
  • If you have selected the Use style values to create new objects option, you can reset the current style values in the Surface Texture Editor dialog box at any time using the Reset button.

Position and Orientation 

  • By default, the orientation of surface texture is determined according to the orientation of the line it is associated with.
    To modify the orientation, right-click the symbol and select Properties. In the Text tab, under Orientation, select the Invert check box.
  • When this is not already the case, you can link the surface texture position and the orientation to another element.
