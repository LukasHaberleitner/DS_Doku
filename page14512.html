Connecting Routables

This task shows you how to connect a routable, such as run or path reservation. A run is used as an example here. NOTE: It is better to use the Connect Parts command if it is available in the application you are using.
You can use this function to connect two runs, or connect a run to a part. To connect to a hole connector use the Connect Parts command.

1.

 Click the Connect Run button (or right click on a run, see note below) and then select one of the elements you want to connect. The element you select first will be the follower element and will be the one that moves to join the leader element. You must click on one of the green connection indicators on the follower run, as shown in the illustration below.

NOTE: In many applications the Connect Run button is not available in the toolbar. In these applications you should right click on the run and select from the drop down menu, as shown below. The run you right click on will be the follower run, as explained above.
2. Select the leader element. The first run will move to connect to the leader run and the connection symbol will display. When two runs are connected at the ends of a segment a star connection is established.

When the connection is in the middle of one run, as shown in the step below, a leader-follower connection is established.
The follower run will connect to the leader at the green connection indicator that you select. However, on the leader run you can select any location and the follower run will connect to that location. If you select a location in the middle of the leader run then you may have to move the two near to each other to create the connection. You can also connect a run to a part using the method described above. If the part does not have a connector then you must create one.
