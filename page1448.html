Printing a Document Quickly

This task explains how to print a document quickly to the default printer using default print settings.
You can only print a document if a default printer has been set up. On Windows, you print using the default printer (declared by your Windows system administrator). However, on UNIX, you can only print after you have set up a printer, as explained in Setting Up Your Printers on UNIX and Windows.

  • When your document is open, click Quick Print in the Standard toolbar.

The current document is sent to your default printer using the following print settings: Rotation: 0 and Display for the Print Area.
