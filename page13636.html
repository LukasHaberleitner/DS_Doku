Managing Sections

From Release 13 users will not be able to directly place parametric sections. This was done to introduce consistency between file-based and database (Enovia/SmarTeam) behavior. Sections must be resolved before being placed. Users may continue to create parametric parts. However, parts must be resolved before being added to the catalog. A tool is being provided in Release 13 to convert parametric parts to resolved (included in this section). Also, administrators need to note that resolved sections are stored in the resource 'Structure Specifications Catalog'.
