Saving and Using Assembly Templates in ENOVIA V5 VPM  

This task is designed to show you how to save assembly templates in ENOVIA V5 VPM and how to re-use them in CATIA. In the scenario described below, you instantiate ribs into a plane wing using the document template feature.
The scenario is divided into the following steps:
  • You open the Wing_Transfo.CATProduct file (This file already contains 4 instantiated clamping pins (see picture opposite).)  and save it in ENOVIA V5 VPM.
  • You create the document template in the Rib_Fastened.CATProduct file and save the file in ENOVIA V5 VPM. Then you create a catalog (Rib_Template.catalog)  referencing the document template and save the catalog in ENOVIA LCA.
  • You save the Left_Wing.CATProduct file in ENOVIA V5 VPM and reload it in CATIA. Then you load the Rib_Template.catalog in CATIA and instantiate the document template into the wing.
 
Before you start, make sure you have checked the Keep link with selected object option in the Part Infrastructure>General tab.

Save the Wing_Transfo.CATProduct File in ENOVIA V5 VPM

  1. In CATIA, open the Wing_Transfo.CATProduct file. The following image is displayed:

  1. Save your files in ENOVIA V5 VPM.

    • In CATIA, click the Connect to ENOVIA V5 VPM icon () to connect your ENOVIA V5 VPM database.
    • Click the Set PDM Properties icon.
      • In the Set PDM Properties window, select Enovia5 from the Destination PDM scrolling list.
      • Select the Publications Exposed storage mode. Click OK when done.
    • In CATIA, click the Save data in ENOVIA V5 VPM Server... icon (). The Save in ENOVIA V5 dialog box is displayed.
    • Click OK. Your data are saved in the ENOVIA V5 VPM database. Close the file in CATIA.

  1. In ENOVIA V5 VPM, click the ENOVIA Home icon (). Expand the Content Management node, right-click the Documents folder and select the Activate command.

  1. The Content Management Startup Selection dialog box is displayed. Click the Search Documents and click OK to launch the search.

  2. In the opening Search window,

    • select Document Revision from the Search for: scrolling list.
    • select the Creator field, enter the Creator's name and click Search.

  3. The Wing_Transfo.CATProduct file is displayed. Right-click it and select the Send To>CATIA V5 command. The file opens in CATIA. Close your file.

Create the Document Template in the Rib_Fastened.CATProduct File

  1. Open the Rib_Fastened.CATProduct file.

  2. From the Start>Knowledgeware menu, access the Product Knowledge Template workbench.

  3. Click the Create a Document Template icon (). The Document Template Definition window is displayed.

  4. In the Document Template Definition window, click the Inputs tab to select the inputs. In the geometry, expand the External References node located below the Rib node. Select all external references and assign them a role (see table below.)

    Name Role
    Line.1 Web
    Rear Rear
    Front Front
    Upper Offset Upper_New
    Lower Offset Lower_New

  5. Click the Published Parameters tab to publish parameters.

    • Click . The Select parameters to insert window is displayed.

    • Use the arrow key to select the following parameters:

      • Geometrical Set.1\Offset.3\Offset
      • Geometrical Set.4\Parallel.1\Offset1
      • Geometrical Set.5\Parallel.3\Offset1

       
    • Click OK twice. The Document template is added to the KnowledgeTemplates node.

  6. Click the Save data in ENOVIA V5 VPM Server... icon () to save your document in ENOVIA V5 VPM. Do not close your file in CATIA.

Create and Save the Catalog Containing the Document Template

  1. From the Start>Infrastructure menu, access the Catalog Editor workbench.

  2. Click the Add Family icon (). In the Component Family Definition dialog box, enter Rib and click OK when done.

  3. Double-click the Rib family and click the Add Component icon (). Click Select external feature and select the document template in the Rib_Fastened.CATProduct file.

  4. Click the Set PDM Properties icon. In the Set PDM Properties window, select Enovia5 from the Destination PDM scrolling list, and select the Publications Exposed storage mode. Click OK when done.

  5. Click the Save data in ENOVIA V5 VPM Server... icon (). The Save in ENOVIA V5 dialog box is displayed.

  6. Click OK. Your catalog is saved in the ENOVIA LCA database. Close the files in CATIA.

Save the Left_Wing.CATProduct File in ENOVIA V5 VPM

  1. In CATIA, open the Left_wing.CATProduct file.

  2. Save your files in ENOVIA V5 VPM.

    • Click the Set PDM Properties icon. In the Set PDM Properties window, select Enovia5 from the Destination PDM scrolling list, and select the Publications Exposed storage mode. Click OK when done.
    • In CATIA, click the Save data in ENOVIA V5 VPM Server... icon (). The Save in ENOVIA V5 dialog box is displayed. Click OK.
    • Your data are saved in the ENOVIA V5 VPM database.

Instantiate the Document Template into the Left_Wing.CATProduct File

  1. In ENOVIA V5 VPM, right-click the saved product and select the Send To>CATIA V5 command. The document opens in CATIA.

  1. In ENOVIA V5 VPM, click the ENOVIA Home icon (). Expand the Content Management node, right-click the Documents folder, and select the Activate command.

  2. The Content Management Startup Selection dialog box is displayed. Click the Search Documents and click OK to launch the search.

  3. In the Search window,

    • select Document Revision from the Search for: scrolling list.
    • select the Creator field, enter the creator's name and click Search.

  1. From the list, right-click the Rib_Template.catalog file, and select the Send To>CATIA V5 Catalog Browser command.

  2. The catalog browser is displayed in CATIA. Double-click Rib, and DocumentTemplate.1.

  3. Expand the Left_Wing/Left_Ref Publications node in the specification tree, and instantiate the rib template contained in the catalog using the wing published features. Repeat this step 4 times.

  4. In CATIA, click the Save data in ENOVIA V5 VPM Server... icon (). Your data are saved in ENOVIA V5 VPM.
