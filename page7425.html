Displaying Annotation in the Normal View

This task shows you how to display an annotation plane in the normal view. The normal view means a view with a plane parallel to the screen.
The operating mode described here applies to annotation plane, annotation and any planar element too.
Open the Annotations_Part_05.CATPart document.

  1. Select the Front View.2 annotation plane.

  2. Select the Normal View icon:

    The annotation plane is displayed in the normal view.
