Composites Design Parameters

Composites Design parameters are located in the Composites Design > Ply or Composites Design > Core node of the generative view style XML file (available via Tools > Standard > generativeparameters category > DefaultGenerative Style.xml file).
   
  These parameters specify whether the elements should be projected in the view. They are detailed in the table below.
   
 

Ply
 
Parameters

Value

 Description
Contour yes/no Determines whether the 3D contours of the plies should be extracted or not.
Exploded surface yes/no Determines whether the exploded surfaces of the plies should be extracted or not.
Flatten yes/no Determines whether the flatten shapes of the plies should be extracted or not.
Highlight shell yes/no Manages the generation of a shaded ply/cut piece.
Seed point yes/no Manages the generation of the projection of the ply/cut piece seed point (both location or indication types).
Seed curve yes/no Manages the generation of the projection of ply/cut piece seed curve (available with partner application).
Rosette yes/no Manages the generation of the ply rosette.
Name yes/no Manages the generation of the ply name.
Contour thickness 1-8 Manages the 2D representation of ply/cut piece contour.
Other geometry yes/no Manages the extraction of  the geometry aggregated by Ply/Cut Piece/Core features.
Overwrite color inheritance yes/no Manages color inheritance for ply, cut pieces or core features dress-up.
