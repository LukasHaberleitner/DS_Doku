SurfacicFlangeSurf-Fillet 

Description

Describes a surfacic flange.

Attributes

BendRadius
Defines the radius.

FD_BTLB
Defines the Bend Tangent Line from the Base feature side in the folded view

FP_BTLB
Defines the Bend Tangent Line from the Base feature side in the flattened view

FD_BTLF
Defines the Bend Tangent Line from the Feature side in the folded view

FP_BTLF
Defines the Bend Tangent Line from the Feature side in the flattened view

FD_IML
Defines the Inner Mold Line in the folded view

FP_IML
Defines the Inner Mold Line in the flattened view

FD_OML
Defines the Outer Mold Line in the folded view

FP_OML
Defines the Outer Mold Line in the flattened view

FP_2OML
Defines the Secondary Outer Mold Line in the flattened view

Inheritance path

 Standard - Feature  >  Standard - Visualizable -> MechanicalModeler - GeometricFeature  >  MechanicalModeler  >  MechanicalFeature

Joggle  

Description

Describes a feature which causes the flange to be locally deformed.

Attributes

Clearance
Defines the cross section radius of the bead.

Depth
Defines the joggle depth.

EndRadius
Defines the joggle end radius.

Runnout
Defines the joggle runout.

StartRadius
Defines the joggle start radius.

Inheritance path

 Standard - Feature  >  Standard - Visualizable -> MechanicalModeler - GeometricFeature  >  MechanicalModeler  >  MechanicalFeature

Web  

Description

Describes the web, which is the main constituent of a part.

Inheritance path

 Standard - Feature  >  Standard - Visualizable -> MechanicalModeler - GeometricFeature  >  MechanicalModeler  >  MechanicalFeature
