From a Previously Created Part

This task shows how to create a board from an existing part.
Open the Scenario1.CATProduct document.
It contains a part with two pads.

  1. Click Create Board .
    The PartBody contains two pads, so you have to select the pad which will represent the board.

  2. Select Pad.1.
    You can add holes, pockets and constraint areas, as well as technological properties
