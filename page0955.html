Approximation

This page deals with the options allowing you to define default parameters for approximation options available on the Approximation tab of particular curve and surface commands.
Note: On the commands' approximation tabs, these default parameters are set with the Defaults option for selected approximation options only.

Curve
  • G0 Tolerance
  • Order type and value in U direction
  • Segmentation type and value in U direction

Surface
  • G0 and G1 Tolerance
  • Order type and value in U and V direction
  • Segmentation type and value in U and V direction
