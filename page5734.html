Converting a Simulation into a Sequence

This task shows how to convert an existing simulation into a sequence.

In our example, there is one existing simulation to be converted. The result expected is a sequence containing a simulation action.
Open the CONVERT_SIMULATION.CATProduct document. A camera and 3 viewpoints are already defined.

  1. Select  Simulation.1 either in the specification tree or in the geometry area.

 

  1. Select Tools > Simulation > Convert Simulation...
    The conversion is automatically launched.

The convert simulation command is very useful to reuse old simulations.
