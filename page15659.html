Working with Assembly Templates in ENOVIA V5 VPM Using VPM Navigator

To define assembly templates, external documents can be selected either in a CATIA session or by making a query in VPM Navigator.

To instantiate an assembly template, you must have created a catalog and stored it in ENOVIA. Then, perform a query and send the catalog to the catalog browser.

For more information about the VPM Navigator, see VPM Navigator User's Guide.
