Compiling a Video Simulation      

This procedure describes how to compile a video simulation. 

There are two commands that compile replays:

  • Compile Process Video
  • Compile Resource Video 

This procedure uses Compile Process Video as an example for describing the basic functions of each. The Compile Resource Video  command records simulation behaviors. For a more complete discussion of the resource simulation, see About Resource Simulation.

The capture phase takes into account the step-size of the simulation so that it can be replayed or played back at real-time, i.e. the replay time of the video will be equal to the cycle time of the entire process that was simulated.

The options for this simulation include moving formats VFW Codec and as well as still image snapshots (JPEG format) of each step in the simulation.

  1. After you have run the simulation and verified that it is one you want to compile, click the Compile Process Video . 

    The Compile Simulation dialog box appears.

  2. Select the format.

    The options are:
    • VFW Codec (default)
    • Still images (JPEG)

  3. (Optional) Select a compression format by clicking the Setup button.

    The Choose Compressor dialog box appears.

    1. In the Compressor box, select a compressor method from the list.

    2. (Optional) Use the Configure button to select the options for the selected format.

    3. (Optional) Modify the Key Frame and Data Rate options.

    4. Select the OK button.

  4. In the box next to the File name... button, enter the file name for the video simulation.

  5. Click the File name... button.

    The Save As dialog box appears so that you can navigate to the directory in which you wish to save the video simulation file.

  6. (Optional) Select the Retain Initial State check box.

    See About Retain Initial State for information.

  7. (Optional) Select the Apply Viewpoint check box.

    The Save As dialog box appears so that you can navigate to the directory in which you wish to save the video simulation file.

  8. Click OK to start the compile.

    You can cancel the compilation at any time by using the Cancel button. Canceling the compilation process midway still results in generation of the video file, up to the point of cancellation.
      If you attempt to compile a process that has only non-simulation activities, a message appears indicating that there are no simulation activities in the process. The command ends without allowing the user to create a video because there is nothing to be captured.
