Product Flow Editor 

This tab appears only if you have a DELMIA license.

This page describes the option that determines the behavior of the Product Flow Viewer. 

Product Flow Viewer Setting

User-defined activities created with Process Engineer appear in the Product Flow Viewer in DPM, and attributes for these activities are stored in an XML file. The XML for Statusbar field indicates the location of this XML file.

Enter the path and filename for the XML file, or use the Browse button to locate the file graphically via a file selection dialog.
