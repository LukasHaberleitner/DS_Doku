Saving as

 


 		 This task shows how to export an object (not necessarily a mesh) to files (other than STL) to be used by 3D printers.

While STL format, 3MF and AMF formats all export the mesh geometry, AMF and 3MF formats can export color and other properties as well.

Open the SurfTessellation1.CATPart model from the samples directory.
 

  1. Open the CATPart containing the elements to export.

  2. Select File > Save as.

  3. From the Save as type list, select
    • amf (*.amf)
      Creates an ASCII AMF file that supports
      • Color on triangles (not on vertices)
      • Texture attributes, whatever their creation mode (material, image, mapping texture) but not stickers.
        Textures are exported on a part or a product, respecting the WYSIWYG convention.
    • 3mf (*.3mf)
      Creates a zip file containing the xml definition as mesh of the exported model (vertices - triangles)
     

  4. In the same dialog box, enter the name and path of the export file.

  5. Click Save.
