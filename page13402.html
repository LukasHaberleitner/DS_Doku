Creating Walls

 

This section explains and illustrates different methods to create walls.

 Create a wall from a sketch: use Sketcher to define the profile, and set the material side.

Create a wall tangent to another one: select a profile coincident with an existing wall, and select the wall to which it should be tangent
Create a wall from an edge: select a wall edge, set the height, limits, angle, then the sketch and material sides.
