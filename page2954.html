Customizing

 

Before you start generating a Ply Book you can customize the way it will be generated.

  1. Select the Tools > Options command. The Options dialog box displays.

  2. Choose the Mechanical Design category in the  left-hand box, then Composites Design.

  1. There is only one single tab: Composites Design

  2. Set options in this tab according to your needs.

  3. Click OK when done.
