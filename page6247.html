Creating a Mesh on Contour Lines

 

This task shows you how to mesh points with respect of contour lines.
A contour line is a scan of points with the same altitude.
Open the ContourLinesMesh01.CATPart model the from the samples directory.

  1. Click Mesh on Contour Lines .
    The dialog box is displayed:

  2. Select ContourLines as the scan containing the contour lines.
    Note that:
    • The scan must contain only contour lines.
    • If you select a Contour Map, the contour lines it contains (in fact scans) are extracted and used as input.

  3. Click Apply. The mesh is displayed:

    It passes by all points and respects the contour lines stream imposed by the scan.

    The resulting mesh is such that no triangle is created with its three vertices belonging to the same contour line.

  4. Click OK to validate and exit the dialog box.
    The Mesh on Contour Lines.1 is created. It is a datum.
