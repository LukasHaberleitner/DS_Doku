Creating a Datum Reference Frame  

This task shows you how to create a datum reference frame on body in white surfaces.

  1. Click the Add button in the Semantic Tolerancing Advisor dialog box .

    The Datum and datum systems  dialog box appears.

  2. Select the each datum and specify its box as is. 

  3. Click Primary.
    The DRF Box1 Edition dialog box appears.

    •  Datum Features & Modifiers frame:
      • Datum Feature drop-down list: displays available datums. The suffixes [CF] and (P) are available for datums with contact features and projected extension respectively.
      • Datum Modifiers: a set of datum modifiers are available for selection. Modifiers are available for selection based on the feature. The placement of the pointer decides the
        For a single datum:
         
        • Maximum material condition and least material condition cannot be used together.
        • Contacting feature cannot be used with maximum material condition, minimum material condition, and projected extension modifiers.
        • Orientation constraint and point situation feature cannot be used together (also applicable to a common datum).
        • The Point Situation Feature dialog box appears when you insert a [PT] modifier just after a datum feature that is established from a pattern of features. It allows you to define or edit the point situation feature that corresponds to a [PT] modifier applied to a datum feature.
        • The Straight Line Situation Feature dialog box appears when you insert an [SL] modifier just after a datum feature that is established from a pattern of features. It allows you to define or edit the straight line situation feature that corresponds to a [SL] modifier applied to a datum feature.
        • The Plane Situation Feature dialog box appears when you insert a [PL] modifier just after a datum feature that is established from a pattern of features. It allows you to define or edit the plane situation feature that corresponds to a [PL] modifier applied to a datum feature.
    • Custom drop-down list: allows you to customize the Degrees of Freedom that are constrained by the set of datum features of the considered box (according the box order of precedence).
      Only available when:
      • The Axis System box of the Datum Reference Frame dialog box is valuated.
      • For any box, one Datum Feature is defined at least in the or Datum Features & Modifiers frame.
      • For Box2, a customization is defined in Box1.
      • For Box3, a customization is defined in Box2.

  4. Click OK in the DRF Box1 Edition dialog box.

  5. Click OK in the Datum and datum systems dialog box.
    The Semantic Tolerancing Advisor dialog box is updated. If a datum reference frame with the same definition already exists, a warning message appears.

    The Semantic Tolerancing Advisor dialog box is updated.
    The datum reference frame is created in the specification tree.

    Following operations are not supported:
    • Pasting a semantic datum or semantic datum target feature created using ISO 5459:2011 in a level which does not support the same.
    • Instantiating a PowerCopy that contains semantic datum target feature or geometric tolerance with DRF feature created using ISO 5459:2011 in a level which does not support the same.
    • Applying a tolerance schema that contains either a datum target semantic feature, a semantic datum feature, or a DRF feature created using ISO 5459:2011 in a level which does not support the same.
    • Importing a tolerancing set that contains either a datum target semantic feature, a semantic datum feature, or a DRF feature created using ISO 5459:2011 in a level which does not support the same.
    Note: When the standard of a feature is changed from ISO to ASME using the feature properties, the DRF panel is updated. The modifiers are also updated accordingly.

  6. Do not close the Semantic Tolerancing Advisor dialog box to perform the next task.
