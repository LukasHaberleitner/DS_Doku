What's New?

Enhanced Functionality

Use the volumetric filter on a subassembly's data only
In previous releases, when you used the volumetric filter, all items that met the volumetric criteria were loaded. Now you can specify a subassembly so that only data from that subassembly that meet the criteria is loaded.
When a DPM session is linked to an ENOVIA server, you can load data from DPE into the same DPM session
Previously, a batch offered a similar functionality. Now a VB script is offered.
Progress bar provided for loading user context
