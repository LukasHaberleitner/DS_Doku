Viewer Mapping

   This tab lets you define the lists used in the Tools Viewer.
  • Functions lists the displayed names of Mold Tooling Design functions.
  • Elements lists the displayed types of Mold Tooling Design Components, Plates and CoolantChannel.
  • Associations lists the elements associated to a function with the following syntax:
    FunctionName#ElementType.

To add Functions or Elements (Those are taken into account only by the Tool Viewer, not by other commands):

  • Type the name of the function or element in the blank field at the bottom of the corresponding column.
  • Hit Enter.
  • The function or element is added to the list.

  • You can create a sub-type that is not pre-defined in Mold Tooling Design, and create a specific icon that will be seen in the element types list.
    This icon is a bitmap file, that must have the same properties as the other icons displayed in the element types list:
    • The dimension must be 16x16 pixels.
    • The extension must be .bmp.
    • The name must be I_XXX, where XXX is the sub-type name.
    • You must save the bitmap file in a specific directory, and add the path of this directory to the variable CATGraphicPath.
    • If the bitmap file is not found, the default UserComponent icon is used.

To remove Functions or Elements:

  • Select its name in the list,
  • Click to edit the name and erase it.
  • Note that no check is performed on the Associations list. You have to update it yourself.

To associate/dissociate Functions and Elements:

  • Select a function, select an element and click Associate
  • Select an association and click Dissociate to dissociate a function and an element.
