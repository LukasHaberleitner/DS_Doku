Creating a Balloon

This task will show you how to create a balloon on a right projection view.
Open the GenDrafting_part.CATPart document.

  1. Click Balloon and select the attachment point of the balloon leader.

  2. Select the balloon anchor point.

  3. The Balloon Creation dialog box appears: enter the required text.

  4. Click OK.
Note that you can create associative balloons on views generated from a product.
