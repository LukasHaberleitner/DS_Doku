Creating an Addendum Surface

This task shows you how to create one addendum surface, that is a multi-sections surface, to extend the stamped part.
Open AddendumSurface01.CATPart from the samples directory.

  1. Click Addendum Surface in the Addendum Design toolbar.

Define the Mandatory Inputs

  1. Select
    • A press axis.
    • A surface.
    • An open boundary.

    Two axis profiles are created, at the start and at the end of the boundary.

  2. Click Apply to create the addendum surface.

    Profiles and coupling guides are listed in the tables below.

    Click Apply after any modification. The tables are updated accordingly.

  3. Optional: To create an addendum surface that is tangent to one or two existing addendum surfaces,
    select the first and possibly a second addendum neighbor.
    When you select an addendum neighbor to specify a tangency, Addendum Surface uses the profile of the addendum neighbor to generate the multi-sections surface.
    You can no longer modify the profile of the addendum surface you are creating.
    If an addendum neighbor is deleted, or if the tangency with the addendum surface you are creating is lost, the tangency is removed from the multi-sections surface.

 

Define the Profiles

The created profiles are listed in the table under Profiles.

Each row corresponds to a profile, with the following information:

  • Name
  • Type
  • Position of the boundary
  • Sketch
  • List of the parameters.

  1. Select one or several profiles.

    From the contextual menu, you can
    • Remove profiles
    • Reset the rotations
    • Edit the sketch.

  2. Double-click a constraint to edit its value.

  3. Under Type, double-click the name of the profile template.

    In the dialog box that appears, select another template to modify the type of the current profile.
    The name of the template contains the type:

    • Line Profile
    • Top Fillet Profile
    • Boss Profile
    • Adaptive Profile

  4. Under Position, double-click the number of the position.
    In the dialog box that appears, enter another position number.

    The profiles are reordered accordingly.

  5. Under Input Profile, double-click the sketch.
    In the dialog box that appears, edit the sketch or create one.

  6. Under Parameters, double-click the parameter.
    In the dialog box that appears, edit the parameters.

  7. In the 3D area, you can
    • Rotate the profile along the Y-axis, or the Z-axis, without breaking the tangency.
      Note that the Z-axis used is the normal to the part at the origin of the axis profile.
    • Drag profiles to reorder them along the boundary.
    • Select the points of the coupling guide, independently from the order.
      The next Apply reoders points and profiles.

 

Define the Coupling Guides

The created splines used as coupling guides to create the multi-sections surface (addendum surface) are listed in the table under Coupling guides.

Each row corresponds to a guide (spline) with the following information:

  • Name
  • Coupling points, that are the list of points used by the spline.

  1. Right-click one row and use the contextual menu to delete or edit the spline.

  2. Click Add coupling guide.
    A row is added to the table.

  3. Select points on the profile.
    When you select a point, Addendum Surface tries to create points on every profile, using a distance or a ratio.
    The spline crosses every profile to ensure a correct update of the multi-sections surface.

  4. In the 3D area, use the handles to manipulate the points.
    Use the shift key for multi-selection.

  5. Click OK.

An Addendum Surface node is created in the specification tree. It contains:

  • Two visualizations nodes.
    • Profile Set with the Axis Profiles. Each Axis Profile contains a point, a sketch and the parameters associated with the sketch.
    • Coupling Guide Set with the guiding splines.
  • The addendum surface created as a Multi-sections Surface, with the related parameters.

Tips:

  • Double-click the multi-sections surface to edit it in Generative Shape Design, wit the following retrieved inputs:
    • The profiles (outputs of the sketch aggregated under the Axis Profiles.
    • The wire (input boundary of the addendum surface).
    • The spline used as the coupling guide.
  • Double-click Addendum Surface, or an Axis Profile to edit them in Addendum Surface.

 

Modify or Create a Profile Template

The default profile templates are available under win_b64\sta rtup\Tooling\TDF.
You can add a profile or modify an existing one. Some rules apply.

  1. Create or open a profile.
    Below is an example of a Boss Profile.

  2. Position the sketch.
    • Select the YZ plane of the Axis Profile as the support plane of the sketch.
    • Select the origin point of the Axis Profile as the origin of the sketch.

  3. Represent the press direction in the template.
    Project the Z-Axis of the press direction in the sketch to constraint the last line of the sketch with an angle.

  4. Create constraints parameters in the part, and link them to the constraints in the sketch.

  5. Inside the sketch, verify that the profile starts in (0,0), and is tangent to the H direction.

  6. Define the profile as an output of the sketch.
    Note that for an Adaptive Profile, multiple outputs exist, requiring that reactions are embedded in the sketch. The reactions activate or deactivate the sketch depending on specific rules, to have only one output profile active.
