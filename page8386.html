Editing/Creating Preferred Angles

This procedure describes how to Edit the Preferred Angles.

The Degrees of Freedom has been incorporated with the Preferred angles. Selecting the DOF has become easier, as this selection has been incorporated with each specific function.

By dividing the total range of motion of the manikin into a certain number of ranges, it is possible to compute a global and a local score that quantifies the current posture. The Edit Preferred Angle enables you to define these ranges on individual DOFs.

Selecting another segment while the command is running opens the Preferred angles dialog box for the new segment and closes it for the previous one.

  1. Click Angular Limitations and a segment. The limits arrows are displayed. These arrows are set by default at the mean values of movement limits. The colored region represents the total range of motion for that DOF.

     

  2. Double-click in the colored region that represents the total range of motion. The Preferred angles dialog box for that segment is displayed.



  3. Or with the right mouse button, select anywhere in the white range of motion region to display the contextual data.

  4. Select a option to activate it.

    • Edit...
      Displays the Preferred Angles dialog box, makes the perimeter zone of the preferred angle being edited become red. You can modify the values of the arrows with the mouse or with the Preferred Angles dialog box.
    • Add
      Displays the Preferred Angles dialog box and splits the selected range in half. You can modify the values of the arrows with the mouse or with the Preferred Angles dialog box.

     
    • Remove
      Removes the selected preferred angle.
    • DOF1-3
      Shows the current Degree of Freedom.

      DOF 1


      DOF 2


      DOF 3

     

  5. The Preferred Angles dialog box can be used to modify preferred angles.

     

  6. This dialog box contains the following elements:

    Body Element: The name of the active body element
    Degree Of Freedom: The name of the active degree of freedom
    Mode: Default
    Name: An editor to rename the preferred angle
       To add or modify a memo
    Color: A combo to select the preferred angle's color
    Score: A number must be entered, 1 to 100 for the preferred angles to show on the surface. 
    Note: the maximum number will be used in the All but Maximum Scores, therefore depending on how many angles the highest score for the angle will be represented in viewing the color of the customized Segment on the manikin. e.g., if you have 4 customized angles and score's 1,2,3, and 4, when using this setting you will not see the customized color on the manikin for score 4.
      A spinner to change the score of the preferred angle
    Limits: Spinners to change the values of the limits
      Sliders to modify the values of the limits
      Navigation buttons to select a preferred angle in the list (next, previous, first, last)

  7. Close to exit from the Preferred Angles dialog box.
