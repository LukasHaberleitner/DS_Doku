Scaling a Dimension

This task will show you how to:
  • scale a dimension in a view.
  • scale a dimension in a 2D component.
  • scale a dimension in a 2D component instance.
  • scale a dimension in a 2D component from a catalog.
Open the GenDrafting_front_view.CATDrawing document.

Scale a Dimension in a View
 

  1. Create a diameter and a coordinate dimension, as well as a datum feature and a datum target.

  2. Multi-select the dimensions and annotations, then set the font size to 7,5mm.

  3. Right-click the frame of the front view and select Properties.

  4. In the Scale and Orientation area, enter 1:2 as the new scale in the Scale field.

  5. Click OK.

    The view is updated but the dimensions and annotations are not scaled.

  6. Multi-select the dimensions and annotations again.

  7. Right-click and select Properties.

  8. In the Text tab, select the Apply Scale check box.

  9. Click OK.

    The dimensions and the annotations are scaled.
 

Scale a Dimension in a 2D Component
 

  1. Create a 2D component in a new detail sheet.

  2. Create a diameter and a coordinate dimensions, as well as a datum feature and a datum target.

  3. Multi-select the dimensions and annotations, then set the font size to 5mm.

  4. Right-click the frame of the view in the detail sheet and select Properties.

  5. In the Scale and Orientation area, enter 1:2 as the new scale in the Scale field.

  6. Click OK.

    The view is updated but the dimensions and annotations are not scaled.

  7. Multi-select the dimensions and annotations again.

  8. Right-click and select Properties.

  9. In the Text tab, select the Apply Scale check box

  10. Click OK.

    The dimensions and the annotations are scaled.

     
 

Scale a Dimension in a 2D Component Instance
 

  1. Double-click the front view.

  2. Click Instantiate 2D Component .

  3. In the detail sheet Sheet.2 Detail, click on the component.

  4. Position the component in the front view.

  5. Back in the detail sheet, multi-select the dimensions and annotations.

  6. Right-click and select Properties.

  7. In the Text tab, select the Apply Scale check box.

  8. Click OK.

    The dimensions and annotations are not scaled anymore.

  9. Go back to the front view.

    The dimensions and annotations in the front view are scaled, but those in the 2D component instance are not.

  10. Back in the detail sheet, multi-select the dimensions and annotations.

  11. Right-click and select Properties.

  12. In the Text tab, select the Apply Scale check box.

  13. Click OK.

    The dimensions and annotations in the front view and those in the 2D component instance are scaled.
 

Scale a Dimension in a 2D Component from a Catalog
 

  1. In the detail sheet, make sure that Apply Scale is checked in the Properties dialog box for all dimensions and annotations.

  2. Select File > Save as.

  3. In the Save as type field, select catalog and click OK.

  4. Click New and select Drawing.

  5. Click OK to create a new drawing, then create a new view.

  6. Click Catalog browser in the Catalog toolbar .

  7. In the Catalog dialog box, double-click Sheet.2 (detail).

  8. Then, double-click 2D Component.1. A preview of the component is displayed.

  9. Drag the component to position it in the view.

    Then click once to end the positioning.

  10. Right-click the frame of the front view and select Properties.

  11. In the Scale and Orientation area, enter 1:2 as the new scale in the Scale field.

  12. Click OK.

    The dimensions and annotations in the front view are scaled.
  • If you have saved a 2D component with the Apply Scale check box cleared, the dimensions and annotations of the imported 2D component will not be scaled when modifying the view's scale.
  • Note that the behavior of dimensions and annotations when modifying the scale is homogeneous.
  • However, if you use pre-R14 data, the original behavior of dimensions and annotations when modifying the scale is preserved.
