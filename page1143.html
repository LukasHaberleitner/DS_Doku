General    

This page describes how to set the General options of DPM Work Instructions.

Work Instructions Customization

Apply Style Sheet to WIs

This textbox specifies the location of an XSL stylesheet that determines the formatting of Work Instructions. By default this file is WorkInstructions.xsl, however the path to a custom XSL file can be specified in its place.

3D Rendering Elements

This portion of the General Tab allows you to determine what Work Instruction Elements to have available in 3D rendering.

Image Capture Settings

This portion of the General Tab allows you to specify features of image captures.
