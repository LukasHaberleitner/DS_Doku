Detecting Clashes

This task shows you how to detect contacts and clashes between all the components in your document.

  1. Click Clash in the DMU Space Analysis toolbar. The Check Clash dialog box appears.

    Contact + Clash checks whether two products occupy the same space zone as well as whether they are in contact. Between all components is the default value for the second Type drop-down list.

  2. Click Apply to run the analysis:

    The Check Clash dialog box expands to show the global results. 21 interferences have been detected. The first conflict is selected by default: a contact.

  3. Select the first clash conflict in the list: the penetration depth is given.

    A Preview window also appears showing the products in the selected conflict.

    The clash is identified by red intersection curves, the value of the penetration depth is given and the direction of extraction indicated.

  4. Click OK when done.
