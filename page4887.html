Seeing Distance Analysis Outside of Simulations

Licensing
  Distance and Band Analysis may not be available on every configuration.
This procedure explains how to observe distance analysis without running a simulation.  When you move an object using the manipulation handle or the compass, you may change distance information that you cannot see in a simulation; using the Interactive Analysis command enables you to see the distance data.
To follow this procedure, use Insert Product to insert Cam_Assembly1.CATProduct under the product node of the PPR tree.

  1. Click Distance and Band Analysis .

    The Edit Distance and Band Analysis dialog box appears.

  2. For Type, select Along Y; from the list underneath, select Between two selections.

  3. Click within the Selection: 1 box, then select the Cam Gear.

  4. Click within the Selection: 2 box, then select the Cam Shaft.

  5. Click OK.

  6. Repeat steps 1 - 5, to create the analysis Distance.2, with following exceptions:

    For Type, select Band analysis, then Between two selections
    For Selection: 2, select Cam.

  7. Click Analysis Configuration .

  8. Use the arrow buttons to move Distance.1 and Distance.2 from the Available to the Selected pane.

  9. On the geometry, move Cam Gear along the y axis, away from the Cam.

  10. Click Interactive Analysis .

    The Interactive Analysis dialog box appears.
    By default, View By State is set to All Analysis.  The other options are Active or Inactive. Also by default, the Automatic update when activated check box is selected.

  11. For Distance.2, click on Active, in the State column.

    The state changes to Inactive.
    To change simultaneously all the values in the State column, click State once to make all the states Active; twice to make all states Inactive.

  12. Move the Cam Gear again along the Y-axis. 

    Note that the value for Distance.2 does not change, although the value for Distance.1 does.
