Activating and Deactivating Annotation Set Update

   
   

 		 You can activate or deactivate the Annotation Set’s update using the shortcut menu command. This improves the performance of geometry modifications.

When the FTA annotation set update is deactivated:

 

  • The FTA features of the annotation set are no more updated when performing:

    ï¿½        Update All of the 3D part or product which contains the FTA annotation set

    ï¿½         Update All of the 3D Shape Representation

    ï¿½         Local contextual Update of the 3D Shape Representation

    ï¿½         Local contextual Update of the FTA annotation set

    ï¿½         Automatic update on geometry modifications (sketch, Pad, pocket etc.

     

When the FTA annotation set update is reactivated:

ï¿½         The annotation set becomes not up to date.

ï¿½         The not up to date mask applies even if the annotation set is in switched off state.

 

  • The FTA features of the annotation set are updated when performing:

    ï¿½          Update All of the 3D part or product which contains the FTA annotation set

    ï¿½         Update All of the 3D Shape Representation

    ï¿½         Local contextual Update of the 3D Shape Representation

    ï¿½         Local contextual Update of the FTA annotation set

    ï¿½         Automatic update on geometry modifications (sketch, Pad, pocket etc.)
