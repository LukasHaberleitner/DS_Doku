Using Selection Tools

This task presents available selection tools.
 

  1. Click one icon in the Selection Tools toolbar.
    Polygon Face Selection: Selects faces by drawing a polygonal trap.
    Polygon Visible Face Selection: Selects faces that are visible from the current point of view by drawing a polygonal trap.
    Tangency Face Selection: Selects faces by tangency propagation from the last selected face.
    Color Selection: Selects faces that are of the same color as, and connected to the last selected face.
    Belt Selection: Selects faces by propagation on a belt.
    Select the first two faces to start the propagation.
    Belt Selection Up to: Selects faces by propagation on a belt, up to a given face.
    Select the first two faces to start the propagation, and a third to stop it.
    Up To Boundary Face Selection: See Selecting Faces up to Boundary
    Chaining Edges: See Chaining Edges
    Convert to 3D Spline: See Converting a Wireframe to a 3D Spline
    Create a 3D Spline: See Creating a 3D Spline
    Create a 3D Polyline: See Creating a 3D Polyline
    Split Stable Curve: See Specifying a Functional Area by a Stable Split Curve
