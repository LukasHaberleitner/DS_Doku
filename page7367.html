Finding And Replacing Text

This task explains first how to locate a string of characters and then how to replace it. This helps you save time in finding your desired annotations in a large Annotation Set.

Find Command:

This command enables the user to enter a textual string and the command will search the corresponding string in the list of selected annotations, or all annotations of selected View(s).
If annotations are not selected, the command searches the string in all annotations present in the active Annotation Set.
If the command finds the input string in supported annotation types, it highlights the located textual string instance.
If the located string is replaceable, it also allows the user to launch Replace command

Strings can be found in the following elements:

  • Text
  • Flag note
  • Coordinate dimension
  • Datum feature (semantic or non-semantic)
  • Datum target (semantic or non-semantic)
  • Dimension (Semantic or non-semantic)
  • Weld

Replace Command:

This command enables the user to find and replace a particular textual string with intended string, in identified and supported annotations,
It is possible to replace the textual string one by one or for all occurrences in one step.

Strings can be replaced in the following documents:

  • Text
  • Flag Note
  • Non-semantic Datum Feature
  • Non-semantic Datum Target
  • Non-semantic Dimension (except the value)
  • Weld

 
1. Select Edit > Find or Replace from the menu bar.
To directly access the Replace dialog box, press Ctrl+H. 

2. In the Find what box, enter a text to search.

3. Click Find Next.
The following message appears in the dialog box: Searching All Annotations. If you previously selected a given number of sheets or elements in the document, the message will be Searching All Current Elements.

The first instance is found and highlighted.

 


4. If needed, click Find Next to search for other instances.
The searched string is highlighted with the color specified for selection through Tools > Options > Display > Visualization > Colors > Selected elements.

  

5. To replace the searched text string with another, enter a replacement text and click Replace.

To replace all instances of the text, click Replace All.

You can also match case, find whole words only, or re-frame the window.

6. Select Close.

  
 

You can replace the normal text string from any linked text.

You cannot replace the following:
  •  An attribute link (red color)
  • A string in a locked annotation
  • A semantic annotation
  • A coordination dimension
  • A non-semantic dimension value
