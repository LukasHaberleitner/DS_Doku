Creating an Offset Connection Between Segments

This task shows you how to create an offset connection between two segments. Creating this connection makes a leader-follower relationship between the two segments and maintains a fixed distance between them.
If you create the connection only between two segments, the two will maintain the offset if you move one. But other segments of the follower routable may change in length to allow the offset to be maintained between the two segments that have a connection. If you do not want this to happen you can create a connection between the other segments too.

You will get two behaviors, depending on whether you check a design option. Both are explained below. To select or clear the option go to: Tools - Options - Equipment & Systems - Design Criteria and check or clear the option Match run extremity points during offset connection creation. The following document will be used to show the procedure.

1. With the option cleared:

Click the Create an offset segment connection button . The Run dialog box displays.

2. Select the segment you want to be the follower. The first segment you select becomes the follower, while the second becomes the leader.
3. Select the second segment. The following happens:
  • The follower segment becomes parallel to the leader segment.
  • The compass displays.
  • You can see a connector line between the two segments when you click the Preview button.

4. If you want to change the offset distance between the two segments then enter it in the Offset field in the Run box.  You can also select the set points between which you want the connection. For both round and rectangular routables you can choose to have the offset connection between the:
  • Outside edge to outside edge
  • Centerline to centerline
  • Outside edge to centerline
5. To create a connection between other segments of the same two routables select other segments and follow the steps shown above.
6. Click Preview to preview the connection, or OK to create the connection.
7. To modify the connection, select the follower run, click the Create offset segment connection button, select the follower segment and enter your changes.
8. To delete offset connections select the follower routable, right click,  then click on the line corresponding to the routable and click Delete offset connections. All connections between the two routables will be deleted.

9. You will get the following behavior when you select the option Match run extremity points during offset connection creation (Tools - Options - Equipment & Systems - Design Criteria) :

After you do Steps 1 and 2 as described above, the following happens:

  • The follower segment becomes parallel to the leader segment.
  • The follower segment changes in length to match the leader segment in length.
  • The compass displays.
  • You can see a connector line between the two segments when you click the Preview button.

The difference in behavior with the option checked is that the two segments become equal in length because the extremity points are matched. You can now follow the procedures from Step 4 onward.
