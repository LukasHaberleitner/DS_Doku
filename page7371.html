Managing Power Copies

Create Power Copy: select the Insert > Advanced Replication Tools > Powercopy Creation command,
select the elements making up the Power Copy from the specification tree,
define a name for the Power Copy and its reference elements then choose an icon for identifying it.
Instantiate Power Copy: select the Insert > Instantiate From Document... command,
select the document or catalog from Catalog Browser, containing the Power Copy ,
complete the Inputs within the dialog box selecting adequate elements in the geometric area.
Save Power Copy into a Catalog: select the Power Copy from the specification tree,
select the Insert > Advanced Replication Tools > Save In Catalog... command,
enter the catalog name and click Open.
