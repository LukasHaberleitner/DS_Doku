Power Machining: Center

This task shows you how to machine the center of a part.

Open the MultiPockets1.CATPart from the samples directory.

  1. Create the rough stock:

    • select the Creates rough stock icon,
    • pick the part,
    • select PartBody as the Destination.

  2. In the Multi-Pockets Operations toolbar, pick the Power Machining icon

  3. Select Machining Program.1 in the specification tree.

    The Power machining dialog box opens.

  4. Select the Rough stock and the part to machine.

  5. Go to the Machining tab and make sure the Machining strategy is set to Center(1) only.

  6. Click Tool path replay .

  7. Now we are going to optimize the tool path.
    In the Geometry tab, place the cursor on the machining direction arrow and select Optimize from its contextual menu.

  8. Push the Tool Path Replay button.

    You can see that the tool path direction has been adapted to the geometry to machine, i.e. defined by the shape of each pocket and set along the main direction (X or Y).

  9. In the Machining tab, in the Center parameters, go to the HSM tab and make sure the HSM option is selected.

  10. Push the Tool Path Replay button. You can note that the corners are rounded.
