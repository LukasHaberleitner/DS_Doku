Replay the Tool Path

This task shows you how to replay the tool path of the pocketing operation.
1. Select the pocketing operation in the tree then select Replay Tool Path .

The tool path is computed and the Replay dialog box appears.
2. Choose a Point by Point replay of the tool path.

You can set the number of computed points to be replayed at each step of the verification by means of the spinner.
3. Click the button to position the tool at the start point of the operation. 
4. Click the button to start  the replay and continue to click that button to move the tool along the computed trajectory.
5. Click OK to quit the replay mode.
