Managing the Compass for Curves

This task shows you how to use and manage the compass.

A curve has been created.
  This kind of selection enables a filter mode to select only specified elements.
It is available through the Tools Palette, which appears when modifying a curve.
 
  1. Click Modification from the Modification toolbar.

    A Tools Palette appears with the following commands:
    Option Description
    Robot Definition Modifies the origin and orientation of the compass, as well as exit the compass definition and return to the previous function
     Current Axis  Defines the position of the compass along the axis system that is defined as the current axis
     Automatic Robot Defines the position of the compass on the selected elements. 
    Reset Robot Reset the compass orientation to the model axis
  2. Select the curve to define the compass origin.
The compass is locked by a single click on a vertex and is not modified when you add elements to the selection (the origin is not modified, but its orientation is). To free its position, click Reset Compass .
