Changing Frame Sizes for Fixed Frames

You can have multiple fixed size frames for annotations in the same drawing or product.
You can change the size of the frames and manage several fixed sized frames using standard files. Changing the size of the frame in the standard file changes it in the entire drawing  or product.

This task is for the role of an owner or an administrator. Select Tools > Standards to open the Standard Definition dialog box.

  1. In the Category list, select drafting and in the File list, select the required standard .xml .

  2. Under the standards, select Frame.

  3.  In the List Of Frames:
    1. Select the required type of the frame and click .
      The same frame is duplicated in the list. The new frame is automatically named in a sequential order. You can now edit this frame by changing its defining parameters and its name.

      You can add as many frames as required.
      To change the frame in the work area, you have the following methods:
       
      • In the Text Properties toolbar, select the appropriate frame in the with the Frame box. The appropriate suffix that is added automatically for multiple sizes of a frame.
      • Right-click the frame and select Properties. In the Text tab, select the required frame in the Frame section.
    2. Select the frame to be removed and click Remove.
      You have to retain at least one frame of each type. Removing an already referenced frame from the standard, switches the frame to the first defined frame.

      The selected frame is removed.

    The frame sizes are changed. You can now use these frames in the drawing or product.
