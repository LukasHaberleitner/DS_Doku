Getting Started

Before getting into the detailed instructions for using Structure Design, here is a short step-by-step tutorial that is intended to give you a feel of what you can accomplish with the product and will show you how to use some of the key functionalities.
You will need a Version 5 session and should be familiar with basic concepts such as document windows, standard and view toolbars.
You should be able to complete this tutorial in about 20 minutes.
