Administration Task: Setting Standard Parameters  

Sheet Metal Design parameters are located in the Standard  > Sheet Metal Design > Unfolded Curves node of the standard XML file (available via Tools  >  Standards  >  General category  >  SheetMetalDesign.xml file):
 


Sheet Metal Design parameters apply to unfolded curves types. These parameters enable you to customize subtypes.
Note: You must also define these parameters in the DefaultGenerativeStyle.xml file. The subtypes defined must have the same values in both files. For more information, see Generative Drafting User's Guide: Administration Tasks: Setting Workbench-Specific Generative View Style Parameters: Sheet Metal Parameters.
 

Parameter

 Value

Description
Subtype_1 > Name  String Determines the name of the subtype.
