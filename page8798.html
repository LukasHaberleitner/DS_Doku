Creating a Flange Surface

The task shows you how to create a flange surface.

  1. Click on the Isometric View icon in the Quick view toolbar.

  2. Click the Flange icon in the Surface Creation toolbar.

    The Flange dialog box is displayed.
 

  1. Select Alignment: Shell normal.

  2. Select one edge of the created surface as Guide.
    The Surface is automatically selected.

  3. Click OK. The flange surface (identified as Flange.xxx) is created and added to the specification tree.
 

  1. In the specification tree, double-click on Length, change in the dialog box the length value to 50 and press OK.
