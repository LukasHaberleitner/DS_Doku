What's New?

Enhanced Functionality

Handling Non-Intersection Input Situations for Corner Fillets (V5-6R2024SP2)
The Corner Fillet command using the Blend type no longer fails if the two horizontal input surfaces (surfaces 1 and 3) do not intersect. The command is more robust for updates. The scope and usability of ready-to-use templates is improved.
Displaying Analysis Results (V5-6R2024SP1)
You can now display the individual results of the Connect Checker Analysis on each comb's spike by hovering over the comb. With MinInfo and MaxInfo, you can display the local and global minimum and maximum values. A contextual menu is available to create points at these positions.
See Checking Connections Between Surfaces.
Keeping Input Selection in Cutting Plane Analysis
You can now select the new Keep input element selection after Ok is pressed option available in Tools  > Options > Shape > ICEM Shape Design > General > Cutting Plane Analysis Option. It keeps the input elements of the Cutting Plane Analysis selected after the validation so that you can reuse them to create a new analysis.
If you select the new option, you can assess the geometry with different parameters on the same set of input elements. This reduces mouse clicks and mouse miles required to reselect the input for a new analysis.
