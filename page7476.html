Editing Annotations in a Cache System

This task shows you how to edit annotations in a cache system.

Select the option Work with the cache system, in Tools > Options > Infrastructure > Product Structure, Cache Management tab, Cache Activation area.

Open the SupportAndSpheresVisulizationAnnotation.CATProduct document or use the file saved in the previous task of creating annotations in a cache system.
  1. Double-click the annotation to edit it.
    You can see that the annotation is displayed in red color and cannot be edited. This is because the related geometry is in     Visualization mode, which is seen in the specification tree.
  2. To edit the annotation, right-click the root Product node (in this case SupportAndSpheres) and select Representations > Design Mode.
    You can see that the Product is now switched to the Design Mode displaying the annotation in default color.
    You can click on individual Product or Part nodes in the specification to switch them to Design Mode. Clicking on the root Product node converts all the sub-level Products and Parts to Design Mode.
  3. You can now edit the annotation.
    The Text Editor dialog box is displayed. Edit the text to Annotation in Design mode in the Text Editor.

  4. Click OK to validate the text edition.
