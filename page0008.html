BfmMngr

 

Description

Describes the Automotive BiW Fastening Parameters feature you create when you click Automotive Biw Fastening Parameters in the Automotive BiW Fastening workbench.

Inheritance path

Standard  >  Feature  >  Automotive BiW Fastening  >  BfmMngr.

 
 

BfmJoint  

Description

Describes the BiW Joint feature you create when you click BiW Joint in the Automotive BiW Fastening workbench. 

Inheritance path

Standard  >  Feature  >  Automotive BiW Fastening  >  BfmJoint

Attribute

BfmFELS
Defines the Forecast Elements Count, which are contained in this BiW Joint.

 
 

BfmJointBody  

Description

Describes the BiW Joint Body feature that you create when you click BiW JointBody in the Automotive BiW Fastening workbench. 

Inheritance path

Standard  >  Feature  >  Automotive BiW Fastening  >  BfmJointBody

Attributes

BfmFELS
Defines the Forecast Fasteners Count which are contained in this BiW Joint Body.

BfmSTY
Defines the Stacking Type with which the Contact Zones referenced by this BiW Joint Body are assembled (Lap Joint, Hemmed Joint, Unspecified).

BfmNTH 
Defines the Number of Linked Zones which are contained in this BiW Joint Body.

 
 

BfmJointElement 

Description

Describes all Fastener features. For instance, the BiW Welding Spot Point that you create when you click BiW Welding SpotPoint in the Automotive BiW Fastening workbench, is a feature of BfmJointElement type.

Inheritance path

Standard  >  Feature  >  Automotive BiW Fastening  >  BfmJointElement.

Attributes

BfmFIN
Defines the Finish Attribute of this Joint Element (A, B, C, D or other).

BfmJID
Defines the Joint Name which contains this Joint Element.

BfmJBID
Defines the Joint Body Name which contains this Joint Element.

BfmMID
Defines the Manufacturing Code of this Joint Element.

BfmNTH 
Defines the Number of Linked Zones of this Joint Element.

BfmPCATS
Defines the Process Category of this Joint Element (Welding, Adhesive, Sealant, BiW Mechanical, Unspecified or other).

BfmPTYPS
Defines the Process Type of this Joint Element (for instance, 21,14... or other).

BfmREG
Defines the Regulation Attribute of this Joint Element (A, B, C, D or other) .

BfmROB
Defines the Robustness Attribute of this Joint Element (A, B, C or other).

BfmSDF
Defines the Shape Definition of this Joint Element (3D point, hemisphere, etc.).

BfmSTY
Defines the Stacking Type of the Joint Body which contains this Joint Element.

  
 

BfmSpotPoint

Description

Describes the BiW SpotPoint feature of BfmSpotPoint type you create when you click, for instance BiW Welding SpotPoint (a Welding Spot Point is created) or the BiW Adhesive SpotPoint icon (Adhesive Spot Point) in the Automotive BiW Fastening workbench. 

Inheritance path

Standard  >  Feature  >  Automotive BiW Fastening  >  BfmSpotPoint.

Attributes

Same as BfmJointElement, as well as:

BfmDIA
Defines the diameter of this Spot Point (this attribute type is length and its unit is mm).

BfmLEN
Defines the shank length value of the Spot Point (with shank definition).

BfmMAT 
Defines the material of this Spot Point (ADHA, ADHB or other).

  
 

BfmSpotProjection 

Description

Describes the BiW Welding Spot Projection feature when you click the BiW Welding SpotProjection icon in the Automotive BiW Fastening workbench.

Inheritance path

Standard  >  Feature  >  Automotive BiW Fastening  >  BfmSpotProjection.

Attributes

Same as BfmJointElement, as well as:

BfmTC1
Defines the Instance Name of the projection component of this Spot Projection.

BfmTPN1
Defines the Part Number of the projection component of this Spot Projection.

BfmTZ1
Defines the Projection Zone's Name of this Spot Projection.

  
 

BfmCurveBead

Description

Describes all Fastener features. For instance, the BiW Welding Curve Bead that you create when you click the BiW Welding CurveBead icon in the Automotive BiW Fastening workbench, is a feature of BfmJointCurveBead type.

Inheritance path

Standard  >  Feature  >  Automotive BiW Fastening  >  BfmCurveBead.

Attributes

Same as BfmJointElement, as well as:

BfmBAS
Defines the Base Value of this Curve Bead (specify the unit: mm for instance).

BfmDISM
Defines the Discretization Method of this Curve Bead (SAG, STEP, Unspecified).

BfmDISV
Defines the Discretization Parameter of this Curve Bead (specify the unit: mm for instance).

BfmHEIGHT
Define the height value of this Curve Bead.

BfmMAT
Defines the material name of this Curve Bead.
