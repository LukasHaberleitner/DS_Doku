Defining Default Values for Generative Shape Design Elements

This task shows you how to save default values for Generative Shape Design features in the standards.
You must be logged as an administrator.

  1. Select Tools > Standards to access the interactive standard editor.

  2. From the Category list, select 3DModellingDefaultValues.
    Once a category is selected, the corresponding .xml file names are displayed in the File list.

  3. In the File list, select 3DModellingDefaultValues.xml.

  4. Expand the 3DModellingDefaultValues node and keep on expanding sub-nodes to display the complete XML tree structure.

  5. Select the required element in the tree.
    Its current default value is displayed in the right pane.

  6. Specify the required default value.

  7. Click OK to save the standard.
    If you want to save it as a new one, you can click Save as New button at the end of the editor tree.
    The following default values can be set in the standard:

    • Extrude: Limit 1, Limit 2
    • Cylinder: Radius, Length 1, Length 2
    • Revolve: Angle 1, Angle 2
    • Sphere: Part/Whole sphere mod
    • Circle: Part/Whole circle mode
    • Offset: Smoothing modes, regularization, maximum deviation
    • Sweep: Fill twisted areas, Compute C0 vertices as twisted areas, Canonical shape detection for non-canonic result, Canonical shape detection
    • Extrapolate: Maximal deviation imposed
    • Healing: Merging distance

For more information about standards and their customization, see Infrastructure User's Guide: Customizing Standards.
