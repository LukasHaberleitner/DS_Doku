Version Requirements

DPM-Process Engineer Compatibility

For proper operation, the release version and service pack level of the DPM-based application used (Process and Resource Definition, Machining, Body-in-White, Assembly) must identically match the release version and service pack level of DELMIA Process Engineer used. This means, for example, that DPM Process and Resource Definition Release 10 Service Pack 2 is supported only when used with Process Engineer Release 10 Service Pack 2. 

Other combinations are not guaranteed to provide the functionality described in this user guide, and are not supported by DELMIA.
