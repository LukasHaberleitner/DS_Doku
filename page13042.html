Edit/Query Object Properties

This section explains ways of querying and modifying the properties of objects. Edit/Multi-edit Objects explains ways of editing structure objects - including their geometry. Edit-Properties is a function common to several workbenches and allows you to edit more general properties. It also allows you to change the values of properties added using the Feature Dictionary. Structure samples provided with this application do not have such properties, therefore you will not see a tab unique to Structure Functional Design until you add properties using the Feature Dictionary.
