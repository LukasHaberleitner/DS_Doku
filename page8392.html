Selecting or Editing the DOF (Degrees of Freedom)  

This procedure describes how to select or edit the manikin's DOF (degree of freedom).

The DOF icon represents the movement being edited. For example, the DOF can be flexion/extension, abduction/adduction, rotation, etc.

When required, this function is in the dialog box, as shown in the example.

  1. Click Angular Limitations .

  2. Select the Left Arm.  The 3D geometry window displays the limitations.


  3. Right-click in the limitations pie, and the contextual menu appears. Select the corresponding DOF.
