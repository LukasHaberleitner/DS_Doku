Managing the Robot

This task shows you how to use and manage the robot.

A sphere with two cuts has been created.
  This kind of selection enables a filter mode to select only specified elements.
It is available through the Tools Palette, which appears when creating or modifying a surface for instance.
  The robot is defined depending on the selected elements : generally the robot plane (x and y axis) is computed using the mean plane computation of the selected elements.
 
  1. Click Modification from the Modification toolbar.

    A Tools Palette appears with the following commands:
    Option Description
    Robot Definition Modifies the origin and orientation of the robot, as well as exit the robot definition and return to the previous function
    Reset Robot Reset the robot orientation to the model axis
     Current Axis  Defines the position of the robot along the axis system that is defined as the current axis
     Screen Plane Defines the position of the robot in the screen plane. The X-axis is horizontal, the Y-axis is vertical, and the Z-axis is orthogonal to the screen plane.
     Automatic Robot  Defines the position of the robot on the selected elements.
  2. Successively select three vertices to define the following elements:
    • Robot origin
    • Z axis
    • X and Y axes (drag the arc of circle to move the robot)
  3. Click the center of the robot to activate the Robot Definition mode and do either of the following:
    • Press Ctrl at once to reset the robot.
    • Drag the center of the robot to place it onto an element (face, edge, or vertex) of the mesh.
This capability is available with the following commands: Translation, Rotation, Affinity, and Alignment.
The robot is locked by a single click on a vertex and is not modified when you add elements to the selection (the origin is not modified, but its orientation is). To free its position, click Reset Robot .
