Modifying the Submodel

You can make the following modifications to the submodel before you analyze it:

  • You can use the Nonlinear Structural Analysis workbench to change the analysis procedure. The submodel can use either a general static step, a heat transfer step, or a linear perturbation step.

  • You must remove any loads or boundary conditions that you applied to regions of the global model that you removed subsequently.

  • If a boundary condition is applied outside the region to which you applied the submodel boundary condition, you must ensure that it corresponds to the loading of the global model.

  • Similarly, if a load is applied to the submodel, you must ensure that it corresponds to the loading of the global model.

  • In most cases you will apply a more refined mesh to the submodel. You can change the element type assigned to the submodel; however, you cannot change the dimensionality.
