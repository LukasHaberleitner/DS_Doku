Tolerancing 

This page deals with the options concerning:
  • The Tolerancing Standard
  • The User Features Tolerancing Schemas Catalog
  • The Semantic Control
  • The Note Object Attribute

Tolerancing Standard

Default standard at creation
Seven conventional standards are available:
  • ANSI: American National Standards Institute
  • ASME: American Society for Mechanical Engineers
  • ASME 3D: American Society for Mechanical Engineers
  • ISO: International Organization for Standardization
  • ISO 3D: International Organization for Standardization
  • ISO 3D DS: International Organization for Standardization
  • JIS: Japanese Industrial Standard
  • JIS 3D: Japanese Industrial Standard
  • JIS 3D DS: Japanese Industrial Standard
By default, the ISO standard is selected.

User Features Tolerancing Schemas Catalog

Defines the file name and path to the tolerancing schema catalog that is to be used for the automatic tolerancing of user feature or technological result of user feature.

Semantic Control
Defines the semantic control options:

Always try to create semantic tolerances and dimensions

Tries to set as semantic an annotation during its creation. Applicable on annotation which can be semantic, keep in mind that the annotation display could be modified and/or moved according to the standard semantic.
By default, this option is selected.

Always try to create semantic general tolerances on dimensions

Tries to set as semantic a general tolerances dimension during its creation. Applicable on dimension which can be semantic, available only if Always try to create semantic tolerances and dimensions is selected.
By default, this option is not selected.

Non-semantic tolerance creation allowed
Defines whether non-semantic tolerances creation is allowed.
By default, this option is selected.

Non-semantic dimension creation allowed
Defines whether non-semantic dimensions creation is allowed.
By default, this option is selected.

Note Object Attribute
Defines the Note Object Attribute options:

Allow Note Object Attribute creation
Defines whether note object attribute can be created by user. Enable or disable the icon and menu item commands.
By default, this option is not selected.

Isolate the 2D component instance/reference
Creates NOAs with no link to the dittos or details.
This check box is available only if the Allow Note Object Attribute creation check box is selected.
By default, this option is not selected.
