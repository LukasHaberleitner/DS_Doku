Clearance + Contact + Clash
 
  • If the total sag (sag1 + sag2) is less than the shortest distance (d), and the shortest distance (d) is less than the specified clearance distance (D), a clearance is detected:
    sag1 + sag2 < d < D

Penetration Depth versus Authorized Penetration
 
  • The penetration depth (an interference result) is not to be confused with authorized penetration (an input).
    • A clash is detected between products A and B.
    • V is the penetration vector and visualizes the penetration depth.
    • Note that V is greater than the authorized penetration in our illustration.
    • If the authorized penetration is greater than distance d, no clash is detected.
 
 

Sag
  The sag corresponds to the fixed sag value for calculating tessellation on objects (3D fixed accuracy) set in the Performances tab of Tools > Options > General > Display. By default, this value is set to 0.2 mm. The sag value set in this tab is offset from the skin inwards (blue contour) on both selection 1 and selection 2.
