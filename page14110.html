Creating a Variable Offset Curve

This task shows you how to create a variable parallel curve on a support.
Before you begin, see information in Creating an Addendum Design.
Open AddendumSurface01.CATPart from the samples directory.

  1. Click Variable Offset Curve  in the Addendum Design toolbar.

  2. Select the Stamped Part that is the support of the curve.

  3. Select a Boundary Partition with defined sections.

  4. Select a join of surface extensions or several.

  5. Modify the Clearance Ratio to reduce the curvature of the variable offset curve. 

  6. Modify the Smoothing Deviation value to smooth the result, thus improving the quality of the prehem surfaces.

A Output Tooling node is created in the specification tree. It contains the Variable Offset Curves with their Clearance Ratio.
