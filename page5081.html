Annotation Text Style

When you generate graphic replacement for structure objects, you can select an annotation text style.
AnnotationTextStyle
Draw
Draws an annotation text style. Yes or No.
 
ATS_AutoPosition
Causes the inclusion of a leader. Yes or No.
  
ATS_CharacterOffset
Non-zero values cause inclusion of a leader. Default is 0.
 
ATS_Attribute
Name of the attribute whose value is drawn. Can have more than a single value. Separate the attribute names with commas.
You can view the annotation of weld information for plate and stiffener. The administrator must add the additional input to the ATS attribute. For plate, it is sfm_plate_weld_addedmaterial  and for stiffener, it is sfm_stiff_weld_addedmaterial.
  • You can change the position of annotations and it is retained when you update the drafting view.
  • The deleted annotations are not created when you update the view update. However, the default annotations are recreated during drafting view update, if all the annotations related to specific object are deleted.
