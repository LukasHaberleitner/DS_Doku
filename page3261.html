Migrating Draw Data from .Model Files to a Layout  

This task shows you how to migrate drawing data from .model files to a layout.

Open pm6-7b-1e_gun_body_draw.model.

  1. Open the .model file.

  2. Select the *DRAFT tab.

  3. Copy the *DRAFT, view or detail view from the specification tree. In this case copy SECTION A/A.

  4. Paste the selected element in the layout.
    The view is pasted in Sheet.1.
  • The *DRAFT is converted to a 2D Layout sheet.
  • The view is converted to a 2D Layout view. A auxiliary view is created with the same plane as in V4.
  • The detail is converted to a 2D Layout detail sheet.
  • The converted 2D Layout detail sheet does not display the sheet format in it.
  • For more details on conversion of V4 elements to V5, refer to V4 Integration User's Guide.
