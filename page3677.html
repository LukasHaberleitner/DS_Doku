Dress-Up parameters  

This section deals with dress-up parameters. These let you define the appearance of dress-up elements, such as markup arrows and threads. 
 

Defining Dress-Up Parameters

Dress-up parameters are located in the Dress-Up node of the Standard Definition editor, available via Tools > Standards:

  • Thread
  • Symbols
