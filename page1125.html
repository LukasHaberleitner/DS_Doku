Penetration

 

The Penetration Management Functions using the following catalogs:
 

  • Penetration Openings Catalog (Openings.catalog)

  • Penetration Shapes Catalog (PenShapes.catalog)

  • Penetration Cutout Rules (PenetrationCutoutRules.CATProduct)
     

The three categories of options has a following sub-options in common :
 

  • File: Lets you to enter the path giving the default location in which the catalogs are stored.

  • ENOVIA V5 VPM:   Lets you specify the path in VPM giving the default location in which the catalogs are stored.
