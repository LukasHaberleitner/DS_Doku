Setting Default Collision Checks  

NC Machine Tool Simulation provides users with two options for creating collision checks:
  • setting default collision checks, which enables users to select quickly the most common objects likely to be involved in collision checks
  • setting specific collision checks, which enable users to specify with precision different aspects of the collision check.

This procedure describes how to perform default collision checking using Options for Default Clashes and the Create Default Clashes .  These commands provide Machine Path Simulation users with shortcuts to setting up the kinds of clash checking most used in Machining Path Simulation.

Using Options for Default Clashes and the Create Default Clashes  creates interference objects automatically. The checks are done among tools, machine head and fixtures, the design part, and stock. The interference objects are selected and the Analysis level and Visualization mode are both set to Highlight. The Monitor is set Off.
Open the Process1.CATProcess sample file, assign a machine to it, and set the automatic workpiece mount.

Setting the Default Clash Options
 

  1. In the Simulation Analysis Tools toolbar, click Options for Default Clashes .

    The Default Clash Options dialog box appears.

  2. Select the desired interference checks from the Resources and Products list and/or from the Global Options.

    Interference objects between stock and design are not added (even when selected), as this would lead to detection of collisions as the tool moves over the stock.

    But, incase integrated material removal is ON, by default, collisions between "shank/holder versus stock" and "cutting part of tool versus stock" are detected if the tool moves in rapid. To enable the analysis of such clashes, the interference object between Stock and Used tools is created during analysis.

    Run the Collision Check

  3. In the Simulation Analysis Tools toolbar, click Create Default Clashes .

    The clash objects are created automatically relative to the options selected.
    Once you click Create Default Clashes, Analysis Mode is turned On and the icon appears highlighted.
    Although you do not need to select Analysis Configuration , you can observe the configuration and alter the Analysis level, Visualization mode, or Monitor status if you do so.
     

  4. Run the simulation to view the collisions.

Design, stock and fixtures must be defined as separate CATParts, since objects are checked for collision at this level.
