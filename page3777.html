Instantitating Fix Together Items  

This task will show you how to instantiate a Fix Together item one or more times.

This task shows you how to:
  • Instantiate from an Existing Fix Together Item
  • Instantiate from a List of Points
  • Instantiate from a List of Points (Drafting and 2D Layout for 3D Design)

Create a Fix Together item.
 

Instantiate from an Existing Fix Together Item
  You can instantiate a Fix Together instance from an already existing Fix Together item.

  1. Right-click the FixTogether.x item from the specification tree or right-click the Fix Together item from the graphics area.
    A contextual menu appears.

  2. Select FixTogether.x object > Instantiate Fix Together.
    A fix together instance is created and follows the pointer. The SmartPick assistant is available, to help positioning precisely the fix together instance.

  1. Select a point or click in the empty space to position the instance.
    The Fix Together instance is created. 

    Simultaneously, the Sketch tools toolbar appears.

 

  1. Select one of the following commands:
    • Explode: instantiate and explode a fix together. The fix together instance is not linked to its reference, so behaves as an independent geometry.
    • Fixed Together: instantiate and apply a fixed together constraint to the geometry elements. This option is active, by default.
    • Change Fix Together Origin : define another origin of the Fix Together.
    • Change Fix Together Position : define another position of the Fix Together to the selected point.
    • Change Fix Together Angle : rotate Fix Together instance around its origin.
    • Flip Fix Together Horizontally : flip the Fix Together instance horizontally.
    • Flip Fix Together Vertically : flip the Fix Together instance vertically.
    • Angle: change the angle of Fix Together instance by typing the desired value.

 

  1. Click to validate and finalize the position of the Fix Together instance.
 

Instantiate from a List of Points
 

You can create numerous fix together instances using a list of points as fix together origin points from another fix together item.

You can create the fix together instances either by using trap selection of points or by selecting each point where the instances are to be created.

This scenario is applicable only for Sketcher.
Create points.
 

  1. Create a group of 2D geometry and fix these elements together.
    The Fix Together constraint is created and displayed in a clip symbol.

  2. Create fix together instances by selecting multiple points.
    • Select all the points (you can trap them using the pointer).
    • Select Insert > Constraint > Constraint Creation > Instantiate Fix Together.
    • Select the fix together item from the specification tree or from the graphics area.
      The fix together instances appear and simultaneously the Sketch tools to let you define the transformations common for all the instances that are being created.
    • Optional: Select the required option in the Sketch tools.
      The selected transformation is applied to all the instances.

  3. Create fix together instances by selecting individual points.
    • Select Insert > Constraint > Constraint Creation > Instantiate Fix Together.
    • Select the fix together item from the specification tree or from the graphics area.
      The fix together instance appears.
    • Select a point or click in the empty space to position the fix together instance.
      The first fix together instance is created and simultaneously the Sketch tools appears to let you define the transformations common for all the instances that are being created.
    • Optional: Select the required option in the Sketch tools.
    • Select the point where you want to create another fix together instance.
      The instance is created at the selected point. Likewise, you can select the points and create the required instances.

  4. Click in an empty space to end the creation.
 

Instantiate from a List of Points (Drafting and 2D Layout for 3D Design)
Create points.
 

  1. Create a group of 2D geometry and fix these elements together.
    The Fix Together constraint is created and displayed in a clip symbol.

  2. Right-click the fix together instance and select Instantiate Fix Together.
    A fix together instance is created and follows the pointer.

  3. Select all the points (you can trap them using the pointer).
    The fix together instances appear and simultaneously the Tools Palette to let you define the transformations common for all the instances that are being created.

  4. Optional: Select the required option in the Tools Palette.
    The selected transformation is applied to all the instances.

  5. Click in an empty space to end the creation.
