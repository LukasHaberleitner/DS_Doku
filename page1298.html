Working with SDAM and FCS Integration


Starting V5-6R2022, the legacy CORBA architecture that uses Orbix implementation and the vault can be replaced with Shared Data Access Manager (SDAM) architecture (HTTP based) for the client server communication. File Collaboration Server can be used instead of the ENOVIA V5 vault for document related transactions. For more information, see SDAM and FCS Integration.
