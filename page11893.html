Using Geometric Analysis

  • Measure Between

This procedure describes how to geometric or measure analysis. With this functionality, you can geometrically analyze the results of material removal after video simulation. This includes:

  • linear distance measure
  • angular measurement between edges and surfaces
  • center and radius measurement of arcs (3-point method)
    To use Arc through 3 Points and receive an accurate result, set the Tool and faceting option to Smaller in the Performance section of the Tools > Options > Machining > Photo/Video tab. A simpler alternative is to use the Pick Point Analysis in Video Mode and Video Measure to get accurate center and radius information for circular entities. Pick Point Analysis provides accurate results regardless of the Tools > Options settings.

  • center-to-center distance
Complete the procedures as described in Accessing NC Manufacturing Verification and Comparing the Machined Stock Part and the Design Part, including:
  • Accessing NC Manufacturing Verification via Replay Tool Path
  • Clicking Full Video   within the Replay dialog box
  • Selecting Setup Editor from the Window menu.
    The Replay dialog box remains visible.

    A Measure window opens, containing the geometry and the Measure dialog box.
