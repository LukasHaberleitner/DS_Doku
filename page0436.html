Analysis Assembly

You can use the Nonlinear Structural Analysis or Thermal Analysis workbench to analyze an assembly composed of other analysis documents. Using this “analysis assembly” approach allows analysts to reuse information from the analysis documents (.CATAnalysis files) for individual parts to create the analysis document for the assembly. An analysis assembly uses the mesh and properties associated with the parts instead of creating a new mesh and properties in the analysis document for the assembly. Loads, boundary conditions, element assignments, and interactions must still be defined in the analysis document for the assembly.

This chapter discusses the creation of multiple analysis documents and their combination into an assembly. The analysis assembly process in the Nonlinear Structural Analysis and Thermal Analysis workbenches is similar to the capabilities available in the CATIA Generative Assembly Structural Analysis (GAS) product. It provides the flexibility for groups to work independently on model components and analyses, while reducing the effort required to assemble them to run as a single analysis. See About Analysis Assembly in the CATIA V5 Generative Assembly Structural Analysis User's Guide for more information on analysis assembly in the GAS workbench.
