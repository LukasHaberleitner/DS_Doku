Keeping the Curve on a Plane

This task explains how to keep a curve into a plane, either at creation or edition.
A curve already exists.

  1. Click Modification .

  2. Click the curve.

    The Planarity icon is activated to let you impose the curve to remain in its plane.

  • The Planarity icon is automatically active when creating a curve created by a sketch in the view plane.

  • It can be activated on an existing planar curve, at edition, so that the curve remains planar.
  • If you deselect it and you modify the curve so that it is not planar anymore, this icon is hidden.
