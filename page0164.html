Fields

Predefined fields are time-dependent, non-solution-dependent fields that exist over the spatial domain of the model. Abaqus for CATIA V5 supports the definition of the following fields for structural analysis:

Creating Initial Temperatures: Applies an initial temperature to a geometry selection.

Creating Temperature Histories: Applies a temperature history to a geometry selection.

Creating Initial Velocities: Applies an initial velocity to a geometry selection in an Explicit Dynamics case.

Creating Mass Scaling: Artificially increases the mass of a body in an Explicit Dynamics case to improve the analysis speed.

You apply environmental actions, such as fields, to supports (geometrical features) on your model: points/vertices, curves/edges, surfaces/faces, or volumes/parts. You can either select the support and then set the field specifications or set the field specifications and then select the support.
