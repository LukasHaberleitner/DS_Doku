Configuring V5 to Load D5 Machines

This procedure describes how to configure the V5 to load the sample D5 devices created for this documentation.

  1. Select Tools > Options > General > Compatibility > DELMIA D5 from the menu bar.

    For more information about setting the options in this tab, see DELMIA D5.

  2. In the Root Libraries  field, click on the Browse button and navigate to the startup/Manufacturing/Samples/NCMachineToollib directory. 

  3. Click OK to close the directory browser.

  4. Click OK to close the Tools > Options dialog box.
To load a D5 machine, see Loading a D5 Machine.
