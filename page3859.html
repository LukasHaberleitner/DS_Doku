View Properties

This reference describes the View properties concerning:
  • Front View.
  • Section View.
  • Section Cut View.
 
  This page deals with the properties concerning:
  • The Name.
  • The Orientation and Scale.
  • The Attributes.
 

Name
 
 

Name
  Defines the annotation view name.
  By default, the default creation name.
 

ID
  Not relevant.
 

Suffix
  Not relevant.
 

Orientation and Scale
 
 

Angle
  Not relevant.
 

Scale
  Not relevant.
 

Ratio
  Defines the view ratio which is applied to all its annotations.
  • This ratio corresponds to a same scale factor applied to all texts,
    symbols and graphic representations contained in annotations regardless of their own properties.
  • The reverse of this ratio is applied when generating the related 2D drafting view, except when the view is contained in an offset or aligned section view.
    In these cases, the reverse ratio of the first annotation view is applied to all the views of the offset or aligned section view.
  By default, 1:1.
 

Attributes
 
 

Display View Frame
  Defines whether the annotation view frame is displayed or not.
  By default, this option is selected.
 

Lock View
  Not relevant.
