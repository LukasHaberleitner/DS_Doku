Parameters

Click the Formula icon to create parameters.
Click the Add Set of Parameters icon to create sets of parameters. These sets of parameters are all grouped below the Parameters node.
Click the Parameters Explorer icon to add new parameters to a feature.
Click the Add parameters on geometry icon to add new parameters to a face, a vertex, or an edge.
