Deleting Images from the Album

This task explains how to delete images from the album.

  1. Select Tools > Image > Album to display the album contents:

  2. Select the images to be deleted from the album. In our example, we want to delete "Capture_000".

  3. Right-click then select Erase to delete the selected image.

    A warning window opens to prompt you to confirm the removal. Click Yes to validate or No to cancel the removal and go back to the Album.

    If you click Yes, the selected image is removed and the image list is updated.

    In our example, "Capture_000" has been deleted and therefore, it is no longer displayed in the list:
