Inserting a local modifier in a chain of local modifiers

This task is divided in two subtasks. The first one allows you to understand what happens if the local modifier is not inserted in the chain of local modifiers. The second one shows you how to insert it.

Open the Feature_Modifier02.CATPart document.

  1. Click Draft Reflect Line .
    The Draft Reflect Line Definition dialog box is displayed.

  2. Select Added Prism.2 (make sure you did not selected a patterned one) and enter 25 deg in the Angle field.

  3. Click OK.
    Note that the Draft is only created on the selected Prism.

  4. Right-click Circular Pattern.1 in the specification tree and select Parents/Children.
    Note that Circular Pattern.1 is located between Added Prism.2 and Draft.1.

    Double-clicking the pattern feature shows that the added prism feature was used as the feature to pattern.

  Open the Feature_Modifier02.CATPart document again.

  1. Right-click Circular Pattern.1 on the Specification Tree and select Circular Pattern.1 object > Deactivate.

     

  2. Click Draft Reflect Line .
    The Draft Reflect Line Definition dialog box is displayed.

  3. Select Added Prism.2 and enter 25 deg in the Angle field.

  4. Click OK.

  5. Right-click Circular Pattern.1 on the Specification Tree and select Circular Pattern.1 object > Activate.
    Note that the Draft has been created on every Prism (even the patterned ones).

     

    Right-click Draft.1 in the specification tree and select Parents/Children.
    Note that Draft.1 is now located between Added Prism.2 and Circular Pattern.1.

    Double-clicking the pattern feature shows that the draft feature was used as the feature to pattern.

    The pattern was modified due to the insertion of the local modifier into the chain of local modifiers.



    To insert a draft reflect local modifier between added prism and pattern, it is necessary to deactivate the pattern then create the draft local modifier then reactivate the pattern to get the required effect. This is a good method when inserting a local modifier in a chain of local modifiers.
