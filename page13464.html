Aggregating Bodies

 

This task shows you, when creating a bend, to aggregate, under the current body containing a wall, the created bend as well as the body containing the second wall.
Open the NEWBendExtremities03.CATPart document.
Make sure that Wall.2 (in Body.1) is defined as current in the specification tree.
This capability is not available with the Bend From Flat command.

  1. Select Bend .

    The Bend Definition dialog box opens.

  2. Select Wall.2 and Wall.3 in the specification tree.

    The Bend Definition dialog box is updated.

  3. Click Preview to visualize the bend.

  4. Click OK in the Bend Definition dialog box.

     

    The Bend (identified as Cylindrical Bend.x) is created and added to the specification tree.

    You can see that it appears under the Body.1 node and aggregates Body.2 containing Wall.3.
     
    You can perform the same operation by defining PartBody as the current object and selecting Wall.1 and Wall.2 as inputs. PartBody contains the cylindrical bend, that aggregates Body.1 (itself containing Wall.2).
