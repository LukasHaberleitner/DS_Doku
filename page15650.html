Sending ENOVIA Clashes to CATIA V5

The following task shows you how to send clashes from ENOVIA LCA to CATIA V5.

Have a CATIA and ENOVIA session running and connected

 

  1. Make sure, the connection is established between between CATIA V5 and ENOVIA LCA (Connect to Enovia LCA   )

  2. Perform a clash query in ENOVIA either:

    • Clicking Clash Manager in the vertical toolbar or

    • Using the ENOVIA Search.

  3. Select the desired clashes in the result query window

  4. Right-click and select the Send To > Product Editor + CATIA V5 from the contextual menu.

    Important:
    When working with DMU Space Engineering Assistant, you can also use the Send To > Space Engineering Assistant provided your CATIA client is connected to  SPE server
