Dimensioning and Tolerancing Pattern Holes 

This task shows you how to dimension and create a geometrical tolerance on a pattern of holes using Tolerancing Advisor.

  1. Select one of the holes.

    The Semantic Tolerancing Advisor dialog box is updated.
  2. Select All same diameter parallel cylinder option.
    The four holes at the four corners are selected.
    The Semantic Tolerancing Advisor dialog box is also updated.
  3. Click the Pattern Diameter icon (Pattern): .
    The Limit of Size Definition dialog box appears.
  4. Select Numerical values.
  5. Click OK.
  6. Click Label A|B|C in the Semantic Tolerancing Advisor dialog box.
    The     Semantic Tolerancing Advisor dialog box is updated.
  7. Click the Position-with-DRF Specification icon: .
    The     Geometrical Tolerance dialog box appears.
  8. Change the Numerical value to 0.2mm and select Maximum Material Condition (MMC) modifier for both toleranced elements and datum elements.
  9. Click OK.
    The position tolerance is selected.
  10. Click Framed (Basic) Dimensions in Annotations toolbar.
    The     Framed (Basic) Dimensions Management dialog box appears.
  11. Select the rectangular coordinates option in the Framed (Basic) Dimensions Management dialog box, then select Automatic Creation.
    The construction geometry and framed dimensions are created. They appear both in the specification tree and the geometry.
  12. Click OK in the Framed (Basic) Dimensions Management dialog box.
Note that the Tolerancing Advisor can be kept running as it allows launching any other command.
