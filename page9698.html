HTSEndEffector (Enumeration)

enum HTSEndEffector {
  EE_LEFTHAND,
  EE_RIGHTHAND,
  EE_LEFTLEG,
  EE_RIGHTLEG,
  EE_LINEOFSIGHT,
  EE_BODY,
  EE_NECK
}
End Effector Options used in specifying end-effector in setting Part-Relations for
Values:
EE_LEFTHAND
worker's left-hand
EE_RIGHTHAND
worker's right-hand
EE_LEFTLEG
worker's left-leg
EE_RIGHTLEG
worker's right-leg
EE_LINEOFSIGHT
worker's line-of-sight
EE_BODY
worker's body(H-Point)
EE_NECK
worker's neck
