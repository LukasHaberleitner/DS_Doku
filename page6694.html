Dump Parameter Settings

The customization enables you to generate a report displaying the actual state of settings of all tab pages. 

  1. Select Tools > Options.
    The Options dialog box appears.

  2. Click to create the dumping settings to a .vbs script macro file.
    The Dump of Parameters dialog box appears.

  3. Select the appropriate options.

  4. Browse the output directory.

  5. Click Yes to dump the settings.
    A dump file is created with an extension .catvbs.

  6. Click to reset the default parameter values.
    The Reset dialog box appears.

  7. Select the options for which you have to reset the default values.

  8. Click Yes.

    The default value parameters are restored.

For more information about the customizing settings, refer to Infrastructure User's Guide: Customizing: About Settings.
