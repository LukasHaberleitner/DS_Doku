Defining a Circle Feature

This procedure describes how to define a circle feature. 
This procedure assumes that the activatedprobe.CATProcess file (created in earlier procedures) is open.

  1. Select the parent process of the circle definition activity. 

  2. On the toolbar, select the black arrow at the bottom of Define Generic:

    A new toolbar appears beneath Define Generic.
    You may not see Define Generic on the Inspection Activities toolbar; the toolbar shows either Define Generic (the initial default) or the last define feature command used.  If, for instance, you last defined a line, you see Define Line instead of the Define Generic.

    If you see the Define Circle on the Inspection Activities toolbar, you can simply click on the center of the command and not on the black triangle.  The Define Circle is activated without recourse to the Define Feature Activities sub-toolbar.

  3. Click Define Circle in the Define Feature Activities sub-toolbar.

    The Define Circle dialog box appears.

  4. Select the circle on the 3D geometry.

    The circle turns orange when selected; a green plane with a line sticking out of it shows the direction of normal.
    The parameters in the Define Circle dialog box also change to reflect the selected circle. 

  5. Click OK.

    The activity is added to the PPR tree.
    The defined circle appears highlighted in green on the geometry.
Once you have defined a feature, you are ready to create a path for measuring the feature.
