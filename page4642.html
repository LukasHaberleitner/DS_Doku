Viewing Information Boxes during Simulations

This procedure describes how to display the names of currently simulated activities while Process Simulation is active.
This procedure assumes that you have a process with activities loaded.

  1. Select the process on the PPR tree.

    If you select an activity instead of a process, only the activity is simulated, but not the whole process.

    If you select nothing, the entire process is selected by default.

  2. Click Process Simulation on the Simulation toolbar.

    The icon turns orange. The Process Simulation and Simulation Control Tools pop-up toolbars appear.

  3. While Process Simulation is active, click Simulated Activities On/Off  on the Simulation Control Tools toolbar:

    The icon turns orange (because it is a toggle on/off command) and Simulated Activities dialog box appears; it provides the name of the activity as it occurs during the simulation.
    If no activity is simulated at a certain moment, the dialog is empty. As soon as the simulation starts, the dialog box is updated with the name of the currently simulated activity.

    If multiple activities are simulated in parallel, all the names are displayed.

    If the currently simulated activity belongs to an active task for a certain resource, the name of the activity having the resource assigned is displayed on one line and the name of the simulated operation is displayed on the next line, indented.

  4. Click Simulated Activities On/Off to remove the view of the information box, or click on the X (close) button on the Simulated Activities dialog box.

  5. End the simulation using one of the methods below:

    • Press the X (close) button in the upper right hand corner of the Process Simulation pop-up toolbar OR
    • Click Process Simulation (the command changes from orange to its original colors).
If you end the simulation without closing the Simulated Activities dialog box, it closes automatically.

By default, the Simulated Activities dialog box is closed when you activate Process Simulation ; you must use Simulated Activities On/Off to open it.
