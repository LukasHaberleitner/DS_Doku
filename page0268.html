Accessing CATIA V5 Functionality from Abaqus for CATIA V5

Some of the functionality of Abaqus for CATIA V5 depends on CATIA V5 commands, for example, the commands that create properties and groups and the commands that mesh your geometry. Access to all of the relevant CATIA functionality—with the exception of pressure data mapping, which requires the pressure data set from a GPS pressure load—is available with a license of the Abaqus for CATIA V5 Complete (ACC) product.

Alternatively, you can obtain licenses for individual CATIA products depending on the functionality that you wish to access. If you do not have a license for Abaqus for CATIA V5 Complete, you must have a license of the CATIA GPS simulation product to use Abaqus for CATIA V5. In addition to the license for GPS, other CATIA V5 licenses are necessary to allow access to some functionality, as described in Table 1–1.

Table 1–1 Licenses used for some functions.

CATIA V5 Simulation License Functionality
GASAnalysis of an assembly of parts, including connections and connection properties
FMSShell meshes and mesh constraints for spot welding and other connection properties
FMDTetrahedron filler, extruded, and sweep meshes
ESTLocal shell and beam properties and grouping tools
CPDDefinition of composite properties
