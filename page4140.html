Creating Device Profiles

These procedures explain how to create and edit device profiles.  All robots have profiles that make up part of their controller data.  You can create new profiles for four aspects of the robot profile: tools, motion, accuracy, and object frame. In addition, you can define an auxiliary device for a robot (e.g., a rail).

In addition to the procedures, this section contains a reference subsection, About Robot Controller Profiles, which gives a detailed description of the variables assigned during profile creation.

To perform this procedure, you must have a robot loaded in the PPR tree (a .CATProcess document). As an example, you may wish to use BodySide_Task.CATProcess.

Depending on the product you are using, you may not have access to all of the profiles here.
