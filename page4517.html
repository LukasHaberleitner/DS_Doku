Downloading or Creating Panasonic CSR Robot Programs

This procedure describes how to download or create a Panasonic CSR robot program.
Open the PAN_Download.CATProcess file.
Downloading generates a Panasonic CSR robot program file from task lists associated with the robot in the PPR tree.

The following steps take place:

  • An XML file is generated from the activities defined in the PPR tree. This involves a mapping between the activities in the PPR tree and the elements in the XML file. The XML file generated is independent of the robot language (Panasonic CSR, in this case).
  • A style sheet transformation file (XSLT) is written for the Panasonic translator. This file defines the mapping between the activities in XML file and the Panasonic commands.
  • The XSLT is then run through the XML file to generate the required Panasonic CSR program.

Downloading Panasonic CSR program

  1. Click Create Robot Program  in the Offline Programming toolbar.

  2. Select RobotTask.1 of TA-1400.1 in the PPR tree.
    The Downloading Options dialog box opens.

  3. Choose the settings as shown above, and click OK.
    The Download Editor window finally appears.

     


     

  4. Save the downloaded program.
