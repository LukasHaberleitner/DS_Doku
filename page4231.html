About the Catalog Browser

This section provides general information about the catalog browser in addition to what is provided in Loading and Positioning Resources.
  V5 provides a library of data about inspection equipment.  This data can be re-used.  In order to access these libraries, the catalog browser must be configured with a directory path that indicates the location where the library catalogs have been installed.
Each resource loaded from the catalog browser is treated as indivisible.  Although the parts that make up the resource appear on the PPR tree, selecting any part of a resource means that the entire resource has been selected.  Resources loaded from other sources may not be treated as indivisible.
Products and resources appear in product structure documents.  The files containing resources have a .CATProduct extension.  Users may create resource parts or load V5-created resources from the catalog browser.  
In addition to using the catalog, you can load product or resource documents (.CATProduct files) into your inspect .CATProcess document using the Insert Product and Insert Resource commands.  The use of these commands is documented in Inserting Products and Resources in the Process and Resource Definition User's Guide.
Note that once an item has been loaded, the other parts can be positioned relative to it before you click on a point in the 3D geometry.  The positioning can be done using the white plane with a vector that appears when you move the mouse around before clicking to confirm that the resource should be loaded.
