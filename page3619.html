Adding Global Attributes

This task will show you how to add global attributes in a text or a table cell.
The global attributes are displayed as a string that is replaced by the actual value of that global attribute while printing or exporting the drawing.
Open a 3D part containing a hole and a drawing representation containing a view.

  1. Click Text or Text With Leader from the Annotation toolbar.

  2. Click in the sheet to create the annotation.

  3. Right-click in the drawing and select Insert Global Attribute.

  4. In the Insert Global Attribute dialog box, select a placeholder display from the following:
    • *: Stores the attribute in the form of an asterisk.
    • $$Attribute: Stores the attribute in the form of $$ followed by the name for that particular global attribute.

  5. Select an attribute from the following and click Apply:
    • User
    • Time
    • Date
    • Day
    • Month
    • Year
    Except the User attribute, the values of all the attributes are according to the local settings of the computer.

  6. To add several attributes, select the required attributes and click Apply.
