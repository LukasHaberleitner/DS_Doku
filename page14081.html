Using the Process Generator Wizard

This task shows you how to easily go through the whole Die Face Design process.
Open the StampingDirection01.CATPart from the samples directory.

Initialize the Process

  1. Click Process Generator   Wizard    in the Method Plan toolbar.

  2. Define the Number of Operations.

  3. Define the Number of Blanks.

  4. Select a Standard File from the list.
    See About Die Face Design Standards for more information.
    • If a Process Method Plan is created, the standard file is read to initialize the parameters.
    • If no Process Method Plan exists, but an analysis is created, the standard file is read to initialize the analysis parameters.
    • If a Process Method Plan is created from the read file, and an analysis is created afterwards, relations are created depending on the type of the elements to analyze (Geometry resulting from a cut tool, geometry resulting form a curve partition, …).
    • Global parameters are stored under the Process Method Plan (and created if they do not yet exist).

  5. Select the Check Process Result check box if necessary.
    The result of the process is compared with the expected result, with possible warnings.
    A check is done to ensure all trim areas are used.
    Missing or different areas are displayed in the graphic area.
 

Define the Part

  1. Go to the Method Plan tab.

    Use the contextual menus under the items to create missing geometries.

  2. Select selection options as required (Update on selection, Reframe on selection, Center Planning Tree).

  3. In the Part Definition tab select a Process Method Plan.

  4. If needed, click to open the Method Plan dialog box.

  5. Press Define Surface Positions.
    See Defining the Positioning for more information.

  6. Press Define Die Face Sheet Metal Parameters
     
    You can
    • Define a Material Thickness and a Material Name for each input part.
    • Specify which side of the part the input surface represents (Top Surface, Middle Surface or Bottom Surface)
    • Select the Thickness Direction of the part.
      By default, the thickness is measured along the press axis.
    A warning is displayed until metal properties are fully defined in thee Die Face Sheet Metal Parameterss.
    The current release does not support several materials in one single part.
    The thickness, material name and material direction parameters are used to compute the unfolded surface in the stamp partition process.

  7. If required, press Define Blank Properties.
    See Defining the Blank Material Properties for more information.
    Material and Thickness are stored under the model and not yet used in the concept process.

  8. Select the curves to define the flanges.
    They are the curves that split the result of the positioning to define folded areas.
    If necessary, select the Automatic Flange Symmetry check box.

  9. Press Manage Intermediate Geometries.
    See Creating a Stamp Partition for more information.

  10. Select elements to cut the curves.
    See Creating a Trim Partition for more information.

  11. If necessary, select the Automatic Trim Symmetry check box.

  12. Press Manage Cut Tolerance/Geometries and enter required values.

  13. Repeat for each blank.
 

Assign Geometries to Operations

  1. Go to the Operation Planning tab.

    Use the Not Planned Only filter to only see not planned elements.
    The tree displays:
    • The blank geometry.
    • A Not Planned node with the list of geometries from the partitions that are not associated with a process tool.
    • A node for each process step with
      • A node per die side
      • A node per tool with
        • The associated direction
        • The associated geometry
        • Possible sub-tools.

  2. Verify, edit and add elements as required. For example:
    • Drag elements in a node as required.
    • Drop a geometry from Not Planned to OP.
      The element is now Planned.
    • Click Create OP to create one.
    • Select an OP. Validate it, verify and edit its properties, or define its press position.
    • Select an OP and click Create Tool.
      When planning an element in a step, a tool is created by default.
      • For a stamped surface, a punch tool is automatically created in the lower die of the press.
        If not already present, a blank holder is created in the lower die and a die tool is created in the upper die.
      • If the selected element contains one punch tool only, with a custom geometry punch contour, Edit Punch Contour is proposed.
      • When you select a blank holder tool, Create Blank Holder is proposed to create the geometry from
        • The corresponding punch contour.
          Note that multiple punches in the same process step are not supported.
        • Offset parameters, defined by the milling and drawbead offset parameters of the blank holder tool.
        • The specified blank.
        • Note that the distance between the blank contour and the punch contour must be sufficient to allow an offset of the punch contour, using the milling and drawbead offset, to still be contained in the blank geometry.
        • For other elements, a tool is created in the target OP.
    • Select a tool and edit and validate it.
    • Select several tools and click Group Tools.
      All selected geometries are grouped into a single tool. The direction of this tool is the direction of the tool of the first selected geometry.
    • Select a tool, then Create Cam.
    • Select the Cam Swivel option.
      When selected, the tool direction is unlocked for step swiveling.
      The tool direction is moved when a swiveling is done (Blank or part swiveling).
      When cleared, the tool direction is not impacted by the swiveling.
    • Select the direction to optimize.
    • Select an analysis to display the current results.
    • Select the element to analyze.
    • Press Optimize (or Edit Robot Coordinates for a manual optimization).

    The cam is created under Grouped Tools.

 

Manage Scraps

  1. Go to the Scraps tab.

  2. Press Automatic Planning of Scraps, if required.

  3. Select addendum surfaces to compute scraps element during the process definition.
    A blank selected under Part Definition is automatically selected as addendum surface.
    Use the contextual menu to create surfaces.

  4. Press Manual Planning of Elements.

  5. Select custom elements to add and join to the result of the process step.
    Select the first and the last stop that define the range within which the custom elements are joined.
 

Assign Detailed Geometry

  1. Go to the Concept to Detail tab.

    The top table displays operations and their associated detailed geometry, if any.
    The detailed geometry of an operation corresponds to the resulting stage of the stamped surface at the end of the operation.
    The bottom table displays the tools found inside the selected step and their associated geometry.
    If selected from an outside part, the selection of elements is always forced to no link mode .

  2. Select an operation in the top table, then the detailed geometry to associate.
    The selected geometry is displayed in the process view, under the Inputs node as Step Part Final Geometry.

  3. Select a tool in the bottom table, then the detailed geometry to associate.
    The selected geometry is displayed in the process view, under the Results node of the tool, as Tool Geometry.

  4. Alternatively, select the part that contains the process proxy related to the selected step.
    The step geometry and the related tool geometries are automatically retrieved from the Result Detailed Geometry geometrical set of the proxy.
 

Configure Analyses

  1. Go to the Analysis Configuration tab.

  2. Press an analysis and enter the required values in the dialog box that opens.

  3. Press Check All Analyses.

  4. Select a method plan or an operation.

  5. Press Select Analyses and select check boxes.

  6. Review the analyses status in the displayed table.

  7. Optional: Open the Detailed Analysis dialog box and enter parameters values.

  8. Select Extract Geometry options to convert the results of the displayed analysis to dead geometry.
    • Extract Areas: Results are stored as dead geometries and 3D labels.
    • Extract Direction: Extracts the Z-Axis on each analyzed point.
    • Extract Geometries As: Extracts geometries as selected.
    • Extract Values As: Extracts values as selected.
    • Extract Points Number Using: Extracts the points number with the selected method.

  9. Press Select Analyses and Launch Export
    The analysis results are displayed in the table, ready for review.
   
 

 Following elements are created in the specification tree:

  • A DFD Process Body that contains:
    • A Process Method Plan.
    • Operations.
  • A Method Plan  that contains:
    • Press Direction
    • Part Positioning
    • Blank Partition
    • Stamp Partition  
      • Planned Elements
      • Not Planned Elements
    • Trim Partition
      • Planned Elements
      • Not Planned elements
    • Scrap Partition
  • A Stamping Direction Set that contains:
    • Join Stamping Direction
    • Stamping Direction Analysis Sets
 

Edit mode is available.
