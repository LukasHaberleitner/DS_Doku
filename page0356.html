Thermal Analysis Workbench Description

This section contains an index to the icons and menus that are specific to the Thermal Analysis workbench. The menu items are located along the top border of your analysis document. The icons are grouped in toolbars, which are located along the right side of the window. The following toolbars are available:

  • Analysis Control Toolbar

  • Analysis Steps Toolbar

  • Assembly Properties Toolbar

  • Model Manager Toolbar

  • Nonlinear and Thermal Properties Toolbar

  • Postprocessing Toolbar

  • Prescribed Conditions Toolbar

The final section, Specification Tree, discusses the structure of the Thermal Analysis specification tree.
