Create a Plant Site

This task provides a basic description of how to create a plant site. More information on creating areas is provided in the section Using Areas.

1. Click the Area button . The Area Creation dialog box displays.

2. Click the Rectangular-Location button , then click the drop down arrow in the Type field and select Site as the area Type.
3. To define the size of the site, key in these values:
  • X Length 1000 ft
  • Y Length 1000 ft
  • Height 100 ft
4. Click at any location. The plant site is created.
5. To display the entire site in your window, click the Fit All In button .
