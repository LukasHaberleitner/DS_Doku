Refreshing your Display in CATIA

The Refresh command updates the product structure and file desk icons with regard to SmarTeam information.

This is particularly useful for concurrent engineering. For instance, if you are using checked-in parts within their assembly, you can use Refresh to check whether its loaded parts correspond to the latest revision. Another user may have performed a check-out, modifications and a check-in while the first user is still using the old file revision.

  1. Select SmarTeam > Refresh.
    The display of icons representing documents in the specification tree is updated. For more about the different icons, see CATIA Tree Icons.
