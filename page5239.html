Defining Properties for a Part

This task shows you how to add properties to a part.

1.

 With the part displayed, click on the Build Part button to display the Create Part dialog box.
2. Select the part to activate the buttons on the Create Part dialog box, and then click the Define Properties button . The Properties dialog box displays.

3. Click the Compartment and Access tab to display properties and enter the values you want.
4. Click Apply or OK.
