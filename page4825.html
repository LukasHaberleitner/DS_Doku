Designing in Context

This task explains how to create a sheet metal part in an assembly context.

For the Sheet Metal Design workbench, open the Scenario2.CATProduct document.
For the Generative Sheet Metal Design workbench, open the NEWScenario2.CATProduct document.
   
This scenario, which is valid for both the Sheet Metal Design workbench and the Generative Sheet Metal Design workbench, is illustrated using screen captures from the Sheet Metal Design workbench. Results will slightly differ in the Generative Sheet Metal Design workbench, for which Automatic bends are not available.
   

Open the part in the Assembly Design workbench (Start  >  Mechanical Design  >  Assembly Design).
The document contains two parts.

 

  1. Right-click Product1 in the specification tree and select Components  >  New Part...

Provided the Manual Input option is checked in Tools  >  Options  >  Infrastructure  >  Product Structure, Product Structure tab, the Part Number dialog box is displayed:

  1. Enter Part3 in the New part Number field and click OK.
    A New Part dialog box proposes two locations to define the origin point.
    For more information, refer to Inserting a New Part, in the Product Structure User's Guide.

 

  1. Click No to locate the part origin according to the Product1 origin point.

    Make sure you are in Design Mode:
    • Select Product1
    • Choose Edit  >  Representations  > Design Mode

  2. Expand the tree and activate the Part3 Part body by double-clicking.

  3. Switch to the Sheet Metal Design workbench or to the Generative Sheet Metal Design workbench.

  4. Click Sheet metal parameters to create the sheet metal characteristics for the part:

    • 1mm for the Thickness

    • 3mm for the Bend radius

    • Linear for the Bend extremities

    and click OK.

  5. Choose the Tools  >  Options  >  Infrastructure  >  Part Infrastructure, General tab and check the Keep link with selected object option, then click OK.

  6. Click Sketcher and select the zx plane.

  7. Click Profile .

  8. Sketch the profile and set the constraints as shown below:

    • 5mm between the sheet metal vertical walls and each pad
    • 0mm between the sheet metal horizontal walls and each pad top
    • 0mm between the last point of the sheet metal sketch and the right pad side.

  9. Click Exit to return to the 3D world.

  10. Click Extrusion .

  11. Select the sheet metal profile.

    The Extrusion Definition dialog box appears.
     
     

  12. Enter 70mm for Length1 then click OK.

    The Material Side should be set to the outside.

  13. Perform this step only if you are using the Sheet Metal Design workbench: click Automatic Bends .

    The bends are created.
    The new features are shown in the specification tree:
    • Extrusion.1 with five walls
    • Automatic Bends.1 with four bends (for the Sheet Metal Design workbench only).
