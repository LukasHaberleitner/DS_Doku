Showing Profile Cards

This task shows you how to access SmarTeam profile cards for the documents of your choice.

In CATIA, open any product to reconcile and display the Reconcile Window.

 

  1. In CATIA View right-click the object which profile card you want to see.

  2. Select Show Profile Card in the contextual menu.
    The profile card corresponding to the document is displayed.

  • Show Profile Card  is available only if the document has been mapped onto a SmarTeam object.

  • You cannot use this capability after multi-selecting objects.
