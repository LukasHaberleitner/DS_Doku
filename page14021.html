Enabling 3DXML Format for CATProduct Documents

This task shows you how to enable 3DXML (EXtensible Markup Language) to preview CATProduct documents.

  1. From SmarTeam, select Tools > Applications Setup.
    The Application Tools dialog box is displayed.

  2. Select CATIA Product.

  3. Click Tools.
    The CATIA Product - Application Tools dialog box is displayed.

  4. Click the Embedded Viewer tab.

  5. Click Modify.
    The Update dialog box is displayed.

  6. Enter ".CATProduct.3dxml" in the Viewer file extension field:

  7. Click OK to validate.

    The next time you save products in SmarTeam, the 3DXML file is generated.

Pre-Version 5 Release 17 Files

It is not possible to convert the files stored in SmarTeam with versions before V5R17 into 3D XML files.
