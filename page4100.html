Using the Catalog Browser

This procedure explains how to use Catalog Browser  to apply the attributes of a reference package to other packages.
For more information about catalogs, see the Component Catalog Editor User Guide.
The project must have a Package.

  1. From the Vehicle Occupant Accommodation toolbar, click Catalog Browser  .
    The catalog browser appears with the previous catalog.

  2. Open a catalog containing references to packages by clicking the Browse another catalog button.

  3. Select a family containing a reference to a package.

  4. In the 3D viewer, select the package created at step 1.

  5. Double click on the package reference in the catalog browser.

  6. The attributes of the reference package are pasted on the new package.
