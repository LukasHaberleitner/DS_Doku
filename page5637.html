MULTICAx ID Plug-in  in a Nutshell

The MULTICAx ID Plug-in allows users to import IDI data generated from IDEAS native data (part and assembly models). This import converts IDI part geometry into the CATIA Graphical Representation (CGR) or CATPart formats, which renders it readable by V5.  The MULTICAx ID Plug-in imports IDI assembly files into a V5 product structure document. Users can perform this import interactively or from the command line.
