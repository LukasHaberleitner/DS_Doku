Using a Reaction with a Document Template: Instantiation Event  

This task explains how to use the Instantiation event associated with the reaction feature.  In this scenario, you are going to instantiate a document template containing a keypad into a .CATProduct file already containing a mobile phone support.

  1. Open the KwrTowHook.CATProduct file. Note that the document already contains a document template.

  1. From the Start > Knowledgeware menu, access the Knowledge Advisor workbench and click the Reaction icon (). The Reaction dialog box opens. 

  • In the Source type field, select Selection and select Document Template.1 below the Knowledge Templates node.
  • From the Available events list, select Instantiation.
  • In the Action field, select Knowledgeware action and enter the following message: Message("Hook instantiated").
    Click OK when done. The Reaction feature is added to the Relations node.

  1. Save your file and close it.

  2. Open the KwrDestinationProduct.CATProduct file.

  3. From the Start > Knowledgeware menu, access the Product Knowledge Template workbench.

  4. Click the Instantiate From Document... icon.

  5. In the File Selection dialog box, select the KwrTowHook.CATPart file that you have saved and click Open. The Insert Object dialog box is displayed.

  6. Make the appropriate selection in the geometry and click OK when done. The document template is instantiated and the reaction is launched.
For more information about the Reaction feature window, see Using the Reaction Feature Window.
