Loading Work Packages from the Light Network

This task explains how to load work packages from the light network to modify data.
Use the Electrical Cable Database Cable Selection dialog box to load cable data from the database, and to route cables.
  The Electrical Cable Database Cable Selection dialog box displays a list of cables, and the light network displays in your document.
  1. To load work packages from the light network:

Select one or more cables from the list of cables displayed on the Electrical Cable Database Cable Selection dialog box.

Click the Load Enovia Document for Network Object button , and the Load Document from Enovia dialog box displays.

Work packages for the selected cables display under the Document Name column. The following types of documents are listed:

  • The schematic document to which the cable belongs.
  • The 3-D FromEquipment and ToEquipment work packages, if any.
  • The From/To cableway work packages, if any.
  • The cableway work packages that the cable route goes through.
    You can also select nodes or segments in the document on the light network, and work packages for the selected objects display on the Load Document from Enovia dialog box.
To work with route segments, you need to set Options from the menu bar as follows:

Select Tools - Options - Equipment and Systems - Electrical Cabling Discipline - Electrical Cableway Routing.

Under Manage Cables from Database Display and Selection Options, select Segment.
  2. Click OK to load all the documents listed in the dialog box. The work packages display in your document and in the specifications tree.

You can remove documents from the list in the dialog box so they will not load. Select the documents you want to remove and click the Remove Work Package from List button.
