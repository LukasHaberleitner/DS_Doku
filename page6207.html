Generating Meshes

This task shows you how to generate a polygonal mesh from curves and selected triangles.
Open the Vase2.CATPart document.

  1. Click Generate Mesh .

    The Generate Mesh dialog box opens.

  2. Using the Ctrl key, select the curves and triangles (if necessary).

  3. Define the tolerances:

    • Point Tolerance: Tolerance for the fitting points

    • Curve Tolerance: Tolerance for the fitting curves

    • Tessellation: Resolution of the generated mesh

  4. Click Apply to preview the polygonal mesh.

  5. Click OK to generate it.

  6. Perform the operation again with the other set of curves.

    The generated mesh passes through the selected curves and vertices of the selected triangles.
    The mesh (identified as Mesh Generation.xxx) is added to the specification tree.
