Defining a Command

You can either define a command after joint creation or during joint creation.
In our example, you define a command after the joint creation.

  1. In the specification tree, double-click Revolute.3.

    The Joint Edition dialog box appears.

  2. Select the Angle driven check box.
    The Angle driven option lets you assign an angle type command to the revolute joint.

  3. Click Ok.

    The command is identified in the specification tree.
