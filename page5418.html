Creating the Solid

This task shows you how to create the solid in two steps:
  • Creating the Solid with Junction Lines (recommended),
  • Creating the Solid with Ramp Contours (optional),
 

Creating the Solid with Junction Lines  
Open the GridSolidGA03.CATPart document.












































































  1. Click Solid From Iso-Thickness Areas in the Top Surface & Solid toolbar.
    The Solid from Iso-thickness Areas dialog box is displayed.
    Select Solid. The name of the solid is proposed. You can edit it.

  2. Select a group of iso-thickness areas.

  3. Select the Junction lines option and select those junction lines in the 3D viewer
    or from the specification tree.
    Multi-selection is available.

  4. If variable cores created as described in Creating a Variable Core exist, select the Elevate using variable cores check box and define a transfer mode.
    The resulting solid or top surface will be elevated as explained in Taking Variable Cores into Account in Solids or Top Sufaces.

  5. Click Apply. The solid is computed.

  6. Click OK to validate and exit the dialog box.
    The solid is created:

When inaccuracies in the input Junction Lines result in computation failure, the precise locations of all the problems are shown in the work area. The location can be a point or a line.

Warnings are displayed when gaps or sharp edges are detected.

  5. Click OK to validate and exit the dialog box.

  6. Repeat those steps to define all the ramp contours as shown below:

  7. Click Solid From Iso-Thickness Areas areas .
    The Solid from Iso-thickness Areas is displayed.
    Select Solid. The name of the solid is proposed. You can edit it.

  8. Select a group of iso-thickness areas.

  9. Select the Standard contours option and select the contours you have just created.
    Click Apply. The solid is computed and displayed. Click OK to validate and exit the dialog box.

Creating a Top Surface as OML

  1. Select Top Surface and select required inputs.

  2. Select the Invert Draping Direction check box.
    The computed top surface represents the Outer Mold Line.
    This check box is used mainly for plies generated with Main Stacking Sequence.
