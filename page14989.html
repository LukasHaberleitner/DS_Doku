Creating Two Swept Surfaces

This task shows how to create two swept surfaces between opposite edges of the two pads.

  1. Click Sweep .

    The Swept Surface Definition dialog box appears.

  2. Select With two guide curves as the sub-type.

  3. Select the vertical edge of Pad 2 as the Profile.

  4. Select the bottom line as Guide Curve 1.

  5. Select the inclined line as Guide Curve 2.

  6. Select the bottom vertex of the profile as the Anchor point 1.

    Anchor point 2 is automatically computed.

  7. Click OK to create the swept surface.

  8. Repeat these steps on the other side to create a second swept surface.
