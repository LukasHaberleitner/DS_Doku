MedianFeature (Object)

Interface for accessing Median Feature modifier on a Semantic GDT (ISO Standard only).
Property Index
Modifier Retrieves Median Feature modifier display.

Properties


Property Modifier() As (Read Only)
Retrieves Median Feature modifier display.
Empty string is returned if modifier not applied; this could suggest that either:
_ Median Feature modifier display has been formally switched off,
_ Median Feature is not displayed if Dimension of Visualization is existing for given GDT.
