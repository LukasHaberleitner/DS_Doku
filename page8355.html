Changing the Gender of an Manikin   

This task describes how to change the gender of a manikin. In this example, the manikin is replaced by a new manikin of opposite sex but with the same percentile.

  1. Select the Display Variables from the Anthropometry Editor toolbar. The manikin to be changed is a man and in the 50th percentile.

     
     

  2. Set the Gender to Woman. The result is a female manikin (50th percentile).

     
     

  3. You can also modify manikin gender by selecting the Male or Female in the Anthropometry Editor toolbar.
