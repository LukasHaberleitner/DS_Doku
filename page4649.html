Using Instant Replay

This procedure describes how to use Instant Replay within Process Simulation .
This procedure assumes that you have a process with activities loaded.

  1. Select the process on the PPR tree.

    • If you select an activity instead of a process, only the activity will be simulated, but not the whole process.
    • If you select nothing, the entire process is selected by default.
    • You can also select an activity that exists under a resource task (e.g., a robot task or a human task).

  2. Click Process Simulation on the Simulation toolbar.

    The icon turns orange.  The Process Simulation and Simulation Control Tools pop-up toolbars appear.

  3. Click Instant Replay On/Off from the Simulation Control Tools pop-up toolbar.

    Instant Replay On/Off is highlighted in orange, and a buffer field appears.

  4. Enter a value in the buffer to specify the number of seconds of the simulation you wish to replay.

    If you have altered the default time units in Tools > Options > General > Parameters and Measures > Units, the number entered in the buffer reflects the time unit you selected, not seconds.
      If you leave the default value for the buffer (i.e., zero), the buffer size is unlimited.

  5. (Optional) On the Simulation Control Tools pop-up toolbar, select any other simulation controls you wish to use before starting Instant Replay.

    Automatic Real-time Simulation is disabled during Instant Replay.

  6. Click Run on the Process Simulation pop-up toolbar, then Replay on the Simulation toolbar. 

    The Replay pop-up toolbar replaces the Process Simulation and Simulation Control Tools pop-up toolbars.

  7. Use the commands on the Replay pop-up toolbar to replay the simulation as much as you like. 

    You can run analysis during Replay .

  8. Exit Replay by:

    • Pressing the X (close) button in the upper right hand corner of the Replay pop-up toolbar OR
    • Clicking Replay (the command changes from orange to its original colors).
    Returning to Process Simulation keeps the recorded history buffer intact, and it continues recording.

  9. End the simulation using one of the methods below:

    • Pressing the X (close) button in the upper right hand corner of the Process Simulation pop-up toolbar OR
    • Clicking Process Simulation (the command changes from orange to its original colors).

Using Analysis with Process Simulation and Replay
 

Switching from Process Simulation to Instant Replay

While switching to Replay mode, if you have verbose mode active for any analysis objects, the messages that appear on the Analysis Information dialog box remain on display.

In the Replay mode, if the time displayed in the analysis message belongs to the replay buffer, you can double-click on the messages and launch the appropriate interference or distance commands to see the results of the analysis check. If the time displayed in the analysis message does not correspond with the current simulation history interval, double-clicking on the message has no effect.

Switching from Replay to Process Simulation

The contents of any Analysis Information dialog box are kept as they were after playing Instant Replay . You can double-click on any of these messages to check the analysis results for the specified simulation time.
