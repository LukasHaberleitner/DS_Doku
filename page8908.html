Moving Surfaces and Curves Using the 3D Compass

You can manipulate (move and rotate) the subdivision surfaces and styling curves using compass.

As manipulate commands work exactly as in the basic interface command, this section of the documentation actually points to the Infrastructure User's Guide. As such, the information detailed in this section is presented in a Infrastructure context. You should note that the Infrastructure User's Guide contains images that correspond to the basic interface commands and therefore illustrate geometry in an environment that is different from the Imagine and Shape environment (symbols, background color, for example).

 

  Manipulate Objects Using the Mouse and Compass: Drag and drop the compass onto the object to be manipulated and use mouse and compass for manipulation.
Manipulate Objects Using the Edit... Command: Object can be manipulated using the Edit... contextual menu command.
