Exporting To XML

This task shows you how to export the ply data in XML format.

See Using Non Crimp Fabrics for specific information.
Available in Composites Engineering Design (CPE) and Composites Design for Manufacturing (CPM).
Open the Export1.CATPart document.

  1. Click XML Export in the Data Export toolbar.
    The XML Export dialog box is displayed:

  2. Select the entity to export.
    It can be the complete stacking, a ply group, a sequence, a ply or a cut-piece.

  3. Use ... to browse your directories and enter the path and name of the XML file to generate.

  4. Select which geometry should be included:
    • 3D contour or
    • Flatten contour.

  5. If necessary, modify the Tessellation Parameters used to compute the tessellation points
    of the contours exported to the XML file.

  6. Click OK to export the data to the XML file and exit the dialog box.
    A progress bar is displayed. The export can be cancelled at any time.
