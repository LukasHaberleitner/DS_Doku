Adding Sections to A Closed Primitive

This task shows you how to add sections to a circular closed primitive so that the result is close to a real circle.
Let's take an example with a sphere.
A .CATPart document must be open.

  1. Click Number of Sections .
    The icon is modified to reflect the modification of the sections number:

  2. Click Sphere .

    It is created with more sections:
       
    Cylinder with more sections: Torus with more sections:
  • This capability has no effect on the box and the pyramid.
  • The visualization of the number of section icons can either be on the same icon cyclically or in an expandable separate toolbar. This depends on the option chosen in Tools > Options > Shape > Imagine & Shape > General tab, Command Options Menu Mode area.
