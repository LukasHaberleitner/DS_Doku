Finding Owning Selection Sets

 

This task shows you how to identify the selection set an element belongs to.
Open the document SelectionSets.CATProduct.

  1. Select Edit > Find Owning Selection Sets... to open the Selection Sets Edition dialog box:

  2. Select the desired element. In this scenario, we select the line.

    The names of the owning sets are displayed in the Selection Sets Selection dialog box:
    For detailed information on this dialog box, refer to Selecting Using Selection Sets.
