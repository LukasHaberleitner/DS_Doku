Improving the Performance of Move Activity Creation

 To improve performance while adding a move activity, users can use one of three "master" commands:
  • Automatic synchronization (preferred)
  • Process verification
  • Process simulation

While any of these master commands are running, users can add move activities; while Auto Synchronization is running, users can also modify or delete activities.  The effect of adding move activities while a master command is running is that the world data model is automatically updated as the activity is added. Similarly, when you use Auto Synchronization to modify or delete an activity, the world data model automatically reflects the modification.

Master commands such as the Auto Synchronization command can be thought of as a different kind of process selection. While in this command, any activity selected will lead to a 3D world synchronization (i.e., the state of the products/resources is updated to the end of the selected activity). Also, this command provides the formal notion of the "current" activity is set to the selected activity. This notion is very useful when authoring the process, since all activity creation commands ask for the starting point of insertion in the process. In a nutshell, the user will have to run one of the "master" commands (see above), synchronize the world to the desired activity in the process (setting the "current" activity), then launch a activity creation command and return to the "master" command and restart the cycle.
To create move activities with a master command, the following procedure applies:

  1. Click the master command (e.g., Auto Synchronization).

  2. Click Move Activity

  3. Create the activity. 
  To modify a move activity while Auto Synchronization is turned on:

  1. Click Modify Activity On/Off from the Automatic Sync toolbar.

  2. Select the activity to modify.

    The Player, Manipulation, and Recorder toolbars appear as well as the Track dialog box.
    You can modify the track definition, but not the shuttle definition. 

  3. Modify the track, record the new or modified shots, and select OK on the Track dialog box.
  To delete a move activity while Auto Synchronization is turned on:

  1. Click Delete Activity On/Off from the Automatic Sync toolbar.

  2. Select the activity to delete.

    A confirmation box appears, requesting verification that the deletion is desired.

  3. Click the Yes button.

    The move activity is deleted.
