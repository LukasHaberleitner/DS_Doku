Compute Maximum Turning Envelope

This task shows you how to compute the Turning profile of a part.
1. Select File > Open then select the part to machine.
  2. Select Machining > Lathe Machining from the Start menu.

The Lathe Machining workbench appears. The part is displayed in the Setup Editor window along with the manufacturing specification tree.

  3. Select Insert > Machining Features > Maximum Turning Envelope to compute the maximum turning envelope.
  4. In the Maximum Turning Envelope dialog box:
  1. Click the Mechanical Part button, then select the part in the Product List where you want to have the feature created.
  2. Click the Turning Part button, then select the body you want to work on, which is also set as design part for simulation in Part Operation.
  3. Click the Turning axis button, and select the desired turning axis from the part selected in step 4.b. Turning axis should be similar to the axis of the machine defined on the Part Operation.
    The turning axis may be initialized by default with the 'main spindle' spindle axis. So this step becomes optional.
  5. Click the Compute button to get the result.
 

Computation of Maximum Turning Envelope
The Compute button enables computing the max revolution envelope solid and sketch.
The resulting solid is created and additionally a sketch is created, to enable 2D wire frame selection (for machining operations)
All the created elements have the same characteristics (color, and so on). No specific offset is applied to some particular elements.

Manufacturing View
A Maximum Turning Envelope machinable feature is created in the Manufacturing View. You can see the Maximum Turning Envelope feature by sorting the manufacturing view by machining features.

The Maximum Turning envelope feature has the solid and the section of this solid in a sketch.
Using the Manufacturing View it is possible to edit the feature and modify its input and then the feature can be recomputed.

You can use the Manufacturing View for different actions on the feature which are described below:

  • Definition (Edit)
    You can edit the created Maximum Turning Envelope feature. It is possible to modify the tolerance and compute again.
  • Delete
    This functionality can be accessed by click on the mouse button 3.
    This command deletes the Maximum turning envelop feature (a warning is issued to ask for confirmation of deletion)
    After the deletion of Maximum Turning Envelope feature, you can manage existing operations that may be linked with the deleted results. Note that the geometry will have Not found status.
  • Show results
    The resulting Solid and sketch could be managed through Hide/Show capability
    Show Solid: the resulting solid of the feature is shown
    Show Sketch: the resulting sketch of the feature is shown.
  • Update
    The update functionality has a similar behavior as anaxial or milling feature.
    If the feature is not up to date, then an Update symbol will appear on the feature.
    You must update the feature by editing the feature (definition or double-click on the feature) and then the feature must be recomputed (using the Compute button) so that its geometry is updated.
