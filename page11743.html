Defining Travel Limits  

This task describes how to define travel limits for commands within a mechanism. 
Open the V4Axis_Machine.CATProduct file from the  online/mtbug_C2/samples/V5/4AxisMachine directory.  In addition:
  • add the new machine
  • add the machine kinematics definition
  • create mount parts and points

  1. Click Travel Limits .

    The Modify Command Limits dialog box below appears.

  2. Alter travel limits and the Caution Zone by using the spinners.

  3. Click OK.
