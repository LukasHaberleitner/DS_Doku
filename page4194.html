Using Operator Assistance

This procedure describes how to measure either a circle or slot feature using the Operator Assistance option. Operator assistance is usually used to prevent the possibility of crashing the CMM when it tries to measure a hole or slot that is fairly small in comparison to the stylus tip diameter, especially in a process which is not very precise, such as sheet metal or plastic parts

  1. Select the parent process or preceding activity for the measure circle activity. 

  2. On the Inspection Activities toolbar, select the Measure Circle or Measure Slot icon.

  3. Select the defined circle or slot you wish to measure.

  4. Either the Define Circle or Define Slot dialog box appears.

  5. Click on the More button.

    The Secure Box and Operator Assistance options appear.

  6. Enter the information as needed and check the Operator Assistance box; then click on the OK button.
The consequence of checking this box is to create additional DMIS statements.  At at runtime on the CMM, these statements have the following effects:

  1. Put the CMM in manual mode (joystick drive)

  2. Prompt the user to drive the probe tip into the CENTER of the hole or slot that is near the probe

  3. Require the user to then hit "continue" key on the OIT (Operator Interface Terminal) of the CMM

  4. CMM will then switch back to CNC mode and begin measuring the hole or slot using the current center of the tip as a "nominal" center of the feature.
