Creating a Local Offset on a Surface

This task shows you how to create a local offset on a surface, from a wireframe element representing the boundary of the offset.
Open SurfaceOffset01.CATPart from the samples directory  and create a sketch on a surface.

  1. Click Surface Offset in the Addendum Design toolbar.

  2. Select the surface on which you want to create the offset.

  3. Select a profile, that is the wireframe that forms the boundary of the offset.
    Be aware that open profiles may lead to unexpected results.

  4. Select the offset direction
    This is not required if the wireframe is planar.
    In this example, the wireframe is in blue, the direction is the Z-Axis.

  5. Enter the Depth value that defines the offset.

  6. Enter a Draft angle, if any.

  7. Specify whether fillets are required.
    If need be, enter the fillet value.

    • Standard Fillet:

  8. If required, select Reverse Side to create the offset on the other side of the surface.

An Offset Surface feature is created.
It contains the parameters of the offset.
