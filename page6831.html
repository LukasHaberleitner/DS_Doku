Defining Pins on a Component

This task shows you how to add pins to a component.
This is the list of the components on which you can create pins:
Schematic Equipment
Schematic Socket
Schematic Plug
Schematic Terminal Board
A component is defined in a detail sheet:
for example a plug:
The plug is represented with a wide pin (Id Number-1).

  1. Click the Add Pins button .

    You are prompted to select a reference component.

  2. Select the plug in the specification tree, under Reference Components, or in the detail sheet where it has been defined.

    The Add Pins dialog box opens:

  3. Enter a value in the Pins to add field.

  4. Click OK .

    The specification tree is updated: three pins have been added to the plug.
