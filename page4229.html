Editing a Default Knowledgeware Rule

This procedure describes how to edit a default Knowledgeware rule.

  1. From the menu, select Start>Inspection Offline to verify that all Inspection DLLs are available, then exit from Inspect.

  2. From the menu, select Start>Knowledgeware>Knowledgeware Expert.

  3. From the menu, select File>Open and open the InspectRules.CATProduct file.

  4. Double click on any rule in the specification tree.

    The Rule Editor dialog box appears.

  5. Edit the rule.

  6. Select OK on the dialog box.

  7. Save the InspectRules.CATProduct file.
