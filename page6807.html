Creating a Wire Level View of a Cable Diagram

This task explains how to create a wire level view of a cable diagram. You create this wire level view by importing a cable diagram into a new document, using the Import from Cable Diagram command.
You must create the cable diagram with schematic components (Place Components) and routed cables (Routing a Cable) with defined wire connectivity (Managing Wire Extremities) for those cables.
1. Open a document containing schematic components and routed cables with defined wire connectivity.
2. Open a new document.
3. Click the Import from Cable Diagram button .

The Import from Cable Diagram dialog box displays and prompts you to select a cable instance.
4. Select a cable from the cable diagram document. The Import from Cable Diagram dialog box displays the name of the cable you selected.

Click OK.
5. An image of the components from the cable diagram is imported into the new document. (The routed cable does not display in the new wire level view document.)

You can use the Route Wire command on the Schematic Wiring toolbar (now activated) to route wire as described in Routing Wire.
