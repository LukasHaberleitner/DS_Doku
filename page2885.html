Creating a Panel

This task explains how to create a panel and design its geometry.
A panel is an optional file used for manufacturing information exchange (IDF 3.0 only).
Open a new CATPart document.

  1. Click Create a panel .
    It creates the panel behavior.

  2. Click Sketch .

  3. Select the xy plane.
    The Sketcher workbench is launched.

  4. Click Profile .

  5. Sketch the contour of the panel.

    The profile looks like this:

  6. Quit the Sketch using Exit workbench  :
    You are back in Circuit Board Design workbench.

  7. Click Pad .

    The Pad Definition dialog box appears.

  8. Enter 2mm in Length.

  9. Click OK.

    The pad turns "sea green", this means the panel representation has been created.
    The panel has been created from the active part: Part1.

By default, CATIA recognizes the first pad created as a panel.
For more information about the Sketcher, refer to the CATIA V5 - Sketcher User's Guide.
