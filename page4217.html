Defining a Secure Box

This procedure describes how to define a secure box around some portion of the product or product used for Inspection.  The box can be created with an offset. The purpose of the secure box is to insure that when measurement paths are created, they do not collide with the product nor do they come closer to the part than the specified offset.

This procedure has two parts:

  • Defining the Secure Box
  • Associating a Plane of the Box to a Measurement Path

You can also associate the plane of a secure box to a measurement path at the time you are creating a measurement path.
To create a secure box, you must have part data loaded.

Creating a Secure Box
 

  1. On the Advanced toolbar, click Secure Box .

  2. Select the product.

    The Offset Dialog box appears.

  3. Select the amount of offset you want in the box and click the OK button.

    A red box surrounds the product
    The Secure Box Manipulation dialog box appears.
    As you move the cursor over the outlines of the box, you can see thin arrows showing the directions in which you can manipulate the edges of the box.

  4. When you have manipulated the box correctly, click the Yes button on the Secure Box Manipulation dialog box.

    The box turns blue.

Associating a Secure Box Plane with a Path
 

  1. To associate one of the box's planes with a specific measurement path, select the box with the right mouse button.

  2. From the contextual menu, select Application Object>Definition.

    The planes of the box appear labeled on the geometry.

  3. Select a plane with the right mouse button.

  4. From the contextual menu, select Plane Object>AssociatePlaneToPath.

    The Associate Plane to Path dialog box appears.

  5. Select the path and click OK.
You can deactivate the association later by selecting the plane.
  Once you have created a secure box, you can select planes associated with the box for the approach and retract planes used in creating the measure path for the feature.  The relevant portion of the measurement feature dialog box appears below; it can be viewed by selecting the More button.
