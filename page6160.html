Getting Started

The following tutorial aims at giving you a feel of what you can do with the Digitized Shape Editor. It provides a step-by-step scenario showing you how to use key capabilities.

The main tasks proposed in the chapter are:

Starting the Digitized Shape Editor Workbench
Importing a File
Meshing the Cloud of Points
Checking the Quality of the Mesh
Creating Curves

All together this scenario should take 15 minutes  to complete.
