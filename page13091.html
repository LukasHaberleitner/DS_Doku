Creating a Blend Corner

This command is only available with the Automotive Body in White Templates product.

This task shows you how to:
  • Create a Blend Corner Using the Point Creation Mode
  • Create a Blend Corner Using the Setback Creation Mode

 

Create a Blend Corner Using the Point Creation Mode
Open the BlendCorner.CATPart document.

  1. Click Blend Corner   in the BiW Templates toolbar.

    The Blend Corner Definition dialog box appears.

  2. From the Creation Mode list, select Point.

  3. Select two or more faces.

    • All the selected faces must belong to the same global support.
    • The Global support box is automatically filled by the surface belonging to the selected faces.

  4. Click Preview.

  5. From the Sections list, you can:

    • remove the selected section.
    • replace the selected section with another section.
    • reverse the orientation of the selected section.
    • Modify T1 and T2 to change the tension values of end points of the section.

       Changing the tension value of an end point of section influences the path of connecting curve from that point.
       

  6. Move the manipulator arrows to edit the point on the surface.

  7. Click Invert sew result to inverse the orientation of the blend corner.

  8. Click OK to create the blend corner.
    The element (identified as Blend Corner.xxx) is added to the specification tree.
 

Create a Blend Corner Using the Setback Creation Mode
Open the BlendCorner.CATPart document.

  1. Click Blend Corner   in the BiW Templates toolbar.

  2. From the Creation Mode list, select Setback.

  3. Select two or more faces.


    The selected fillet faces appears in the Sections list with default setback values.

    • All the selected faces must belong to the same global support.
    • The Global support box is automatically filled by the surface belonging to the selected faces.

  4. From the Sections list, you can:

    • remove the selected section.
    • replace the selected section with another section.
    • reverse the orientation of the selected section.
    • change the setback values for individual faces using the Setback box. You can click Apply to all if you want to apply the same setback value to all faces. You can also use the manipulators in the 3D to change the setback values.
    • Modify T1 and T2 to change the tension values of end points of the section.

       Changing the tension value of an end point of section influences the path of connecting curve from that point.

  5. Click Invert sew result to inverse the orientation of the blend corner.

  6. Click OK to create the blend corner.


    The element (identified as Blend Corner.xxx) is added to the specification tree with setback parameters under it.
