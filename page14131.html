Generating the Structure

This task shows you how to automatically create a detailed design structure from a concept part.
Open ProcessPositioning_EndP1.CATPart from the samples directory and insert it in a product.


  1. Click Structure Generator in the Structure Definition toolbar.

  2. Select the Method Plan feature that defines the concept process.

  3. Select the Target Root Product where the structure is created.

 The structure is created under the selected root product. It contains:

  • Blank Detailed Part
    • Concept to Detail Link to the Process Method Plan.
  • OP Detailed Part (for each OP)
    • Concept to Detail Link to the corresponding OP
    • Detail to Detail Link from all the previous OP to this one.
  • Process Result Part
    • Detail to Result Part Link to the Process Method Plan and all the created detailed parts.
