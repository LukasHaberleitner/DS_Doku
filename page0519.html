Specification Tree

The specification tree contains the following items:

Analysis Manager

All the objects making up an analysis document.

Links Manager

All the links to other documents managed from the .CATAnalysis document, such as part or results data.

Finite Element Model

The model representation used for performing a computer-aided engineering analysis, consisting of the system representation and various environment action representations.

Nodes and Elements

The nodes and elements that can be created are OCTREE (GPS) type.

Properties

Any specification applied to define the physical properties (geometry) of the model:

  • Material and thickness (surface)

  • Connections

  • Imported composite properties

Nonlinear and Thermal Properties

Any nonlinear or thermal specifications applied to define the physical properties (geometry) of the model:

  • Amplitudes

  • Nondefault behavior of contacting surfaces

  • Shell property enhancements

  • Springs

Nonlinear Structural or Thermal Case

A procedure for the computation of the system response to applied loads under given restraints.

Element Assignment

The elements assigned to each mesh part.

Simulation History

The sequence of analysis steps that define the environmental actions on the system.

Initialization

A special Initialization step is created at the beginning of the Simulation History.

Connections

Enhancements to connections.

Fields

Initial temperature fields.

General Static Step or Frequency Step

Field Output Requests

Generates field output that is saved to the output database file, typically for large regions or for the entire model.

History Output Requests

Generates history output that is saved to the output database file, typically more often than field output or for a smaller region of interest.

Loads

Generates distributed force systems equivalent to given static resultants or force densities of given intensity.

Boundary Conditions

Generates mechanical degree of freedom restraints on a geometry selection.

Fields

Generates initial temperature or temperature history fields.

Jobs

Analysis job definitions.

Display Groups

Analysis display group definitions.

Analysis Case Solution

Results data image requests.

Solution Sensors

Local or global sensors added to record or parameterize field or history output results, respectively, within a step.
