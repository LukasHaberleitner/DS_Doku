Automatically Updating Flatten Links

 

In combination with the Synchronize command, these functionalities let you generate a new flattened view from an existing flattened model, thereby maximizing the re-use of an existing design and reducing lead times:
  • Extract a flattened view from your 3D harness document
  • Copy (renaming) the flattened view
  • Remove the flatten links between the copy and the original 3D harness document
  • Create new flatten links between the copy and a new 3D harness document
  • Synchronize the copy and the new 3D harness document to update the flattened view.
