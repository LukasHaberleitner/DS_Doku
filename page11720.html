Accessing NC Machine Tool Builder

This procedure describes how to access the NC Machine Tool Builder  workbench and how to load a machine already created in V5.
Ascertain where, on your installation of V5, the  online/mtbug_C2/samples/V5 directory is; there is sample data provided there.

  1. Open a V5 session.

  2. Select Start > Machining Simulation > NC Machine Tool Builder.

     

  3. The NC Machine Tool Builder workbench opens.

  4. From the main menu, select File > Open.

  5. Select the 3_Axis_Machine.CATProduct document from the online/mtbug_C2/samples/V5/3AxisMachine directory; click Open to open the selected file.

    A specification tree and the geometrical data appear as shown below. The machine tool is ready to be modified.
To try assembling a machine yourself from existing V5 parts, see Assembling V5 Components to Build a Machine.
