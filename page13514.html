Managing Inner Islands

Provided the Guiding strategy is set to Parallel contour and the Stepover mode is set to Constant 3D, Contour-driven operations let you manage inner islands:
  • Set Islands Contours,
  • Select Faces and Detect Islands Contours.

Guides and islands contours must be closed.
Open file Basic3.CATPart, then select Machining > Surface Machining in the Start menu.

Set Islands Contours

Below is the result without selecting the island contour:

  1. Click Contour-driven .
    The Contour-driven dialog box is displayed.
    The dialog box opens at the geometry tab page .
    Click the red area in the sensitive icon and
    select the whole part.

  2. Go to the Strategy tab. Set Guiding strategy to Parallel contour. 
    In the Radial tab, set Stepover to Constant 3D.

  3. In the sensitive icon, click Guide 1. The Edge Selection Toolbar is displayed.
    Select the guide. Click OK in the Edge Selection toolbar to validate the selection.

  4. In the sensitive icon, click Island. The Edge Selection Toolbar is displayed.
    Select the island contour. Click OK in the Edge Selection toolbar to validate the selection.

  5. Click Tool Path Replay to compute the tool path for the operation.
    A progress indicator is displayed.
    You can cancel the tool path computation at any moment before 100% completion.
  • Condition of stop on islands is the same as for the guiding element.
  • Offset on islands has the same value as for the guiding element.
 

Select Faces and Detect Islands Contours

  1. Click Contour-driven .
    The Contour-driven dialog box is displayed.
    The dialog box opens at the geometry tab page .
    Click the red area in the sensitive icon and
    select the whole part.

  2. Go to the Strategy tab. Set Guiding strategy to Parallel contour. 
    In the Radial tab, set Stepover to Constant 3D.

  3. In the sensitive icon, right-click Guide 1 and choose Select faces.
    The label turns to Boundary Faces.
    Click Boundary Faces. The Face Selection Toolbar id displayed.
    Select the faces and click OK in the Face Selection toolbar to validate the selection.

    Inner boundaries of the selected faces are detected as islands contours.

  4. Click Tool Path Replay to compute the tool path for the operation.
    A progress indicator is displayed.
    You can cancel the tool path computation at any moment before 100% completion.
    You can see the islands have been detected automatically.
