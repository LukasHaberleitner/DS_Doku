Calculating Weight and Center of Gravity of a Spool

This task shows you how to calculate the weight and center of gravity(COG) of a spool using reports.
Select Tools > Options > General > Parameters and Measure > Knowledge Environment tab and select the Load extended language libraries option.

1.

 Open a spool to calculate weight and COG.
2. Define a report template to calculate weight and COG by selecting all the required attributes:
  • Select Tools > Report > Define.
    The Report Definition dialog box appears.
  • In the Report Title box, type the name of the report template.
  • In the Dictionary box, select PipingLayout.
  • In the Type box, select Piping Spool.
  • In the Attribute box, select the required properties one by one and click Add.
  • Click SaveAs to save the report template.

For more information, refer to Defining the Report Format.
3. Select the spool.
4. Generate the report:
  • Select Tools > Report > Generate.
    The Generate a report dialog box appears.
  • In the Report Name box, select the previously defined report template.
  • In the Report on the following objects area, select Currently Selected Objects option.
  • Click OK.

For more information, refer to Generating a Report.

The report is generated listing the attribute values in the Weight and COG dialog box.
5. Check the weight and COG values:
  • Select the spool.
  • Select Edit > Properties.
    The Properties dialog box appears.
  • Select Piping tab.
