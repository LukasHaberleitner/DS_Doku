Thickness and Font

 
This tab deals with the following categories of options:
  • thickness
  • font options

Thickness
 
The Thickness area lets you select a thickness type from the 55 values contained in the list.

If necessary, you can modify the size in pixels or the size in millimeters by selecting the value to be modified then clicking it once and entering your own value.
"Size in pixel" reflects the result displayed on screen and "Size in mm" corresponds to the printed version.

  • You cannot assign a size greater than 16 pixels.

  • Changes do not affect the current document if its standard already contains a thickness definition.
 

Font Option
 

Use the system TrueType fonts in CATIA

Enables you to use TrueType fonts provided by Windows and thus, add them to the font chooser. 

This default list of fonts can be customized. To do so, refer to About Fonts in this guide.

By default, this option is activated.
