Administration Tasks  

A new directories structure has been introduced which enables you to manage data associated to your workbenches.
However the methodology provided until now to create the expert user's working environment and to deploy workbenches remains available.

This page presents administration tasks specific to a product

The information provided in these topics are designed to help the administrator:

  • Create a working environment,

  • Collect the information from the experts, 

  • Manage the data, 

  • And deploy the end-user workbenches.

 

If you need more information about the administration, see the CATIA Administration and Installation User's Guide.
