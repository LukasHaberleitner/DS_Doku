Multi-Axis Spiral Milling

This task shows how to insert a Multi-Axis Spiral Milling operation in the program.
This type of Multi-Axis Machining is suitable to machine pockets or to engrave complex surfaces

To create the operation you must define:

  • the geometry to be machined
  • the tool that will be used
  • the parameters of the machining strategy .
  • the feedrates and spindle speeds
  • the macros (transition paths) .

See Multi-Axis Spiral Milling Parameters for more information on the parameters.
Open the SpiralMilling01.CATPart document, then select Machining > Advanced Machining from the Start menu.
Make the Manufacturing Program current in the specification tree.  

  1. Click Multi-Axis Spiral Milling  .
    The Multi-Axis Spiral Milling dialog box appears directly at the Geometry tab page .
    This page includes a sensitive icon to help you specify the geometry to be machined.

  2. Click the red area representing the part to machine.
    The dialog box disappears.
    Select the part as shown below.


    Double-click anywhere in the 3D viewer to validate the selection and revert to the dialog box.
    The red area has turned green.

  3. Click the area representing the guide faces.
    The dialog box disappears.
    Select the guide faces as shown below.

    Double-click anywhere in the 3D viewer to validate the selection and revert to the dialog box.
    The guide face area has turned green.

  4. Go to the Machining Strategy tab, then the HSM tab and select the High Speed Milling check box.

  5. Click Tool Path Replay.
    The tool path is displayed:

  6. Click OK to validate.
