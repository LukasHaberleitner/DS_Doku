Pattern Usage During Import

This task shows you that the rectangular pattern used for mounting a component, is recognized during import.
This feature is available on both the board and manufacturing panel.

Before starting, set the Tools > Options > Infrastructure > Part Infrastructure as shown below:


Also, in the Tools > Options > Equipment & Systems > Circuit Board Design > Components tab, check the With Constraints check box.

  1. Open the Pattern.idf document from the samples directory.
    The Circuit Board Design dialog box opens.

  2. Click to activate the Catalog light.

  3. Browse to select the Pattern.catalog.
    The Catalog light becomes green.

  4. Click OK to validate.
    A message informs you that the catalog is unchanged (it is in read only mode).
    The import result looks like this:
The idf file contains the board, the component and the holes. During the import, the holes coordinates are analyzed and recognized as defining a pattern. If you move or delete the component, the pattern will be updated (see the next task).
