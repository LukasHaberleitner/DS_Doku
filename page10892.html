Browsing the Kinematics Mechanism

This task shows you how to browse the properties of the selected kinematics mechanism.
Insert  the KIN_EX17* .model files from the samples folder.

They are to be found in online\kinug_*X2\samples

If you work with the Cache System, make sure you are in Design mode (select Edit > Representations > Design Mode.). for more detailed information, refer to the DMU Navigator user's Guide - Task: Viewing the Cache Content.

  1. Select KIN_EX17_00_F1_ACTIVE, and expand the tree.

  2. In the specification tree, right-click the kinematics mechanism or select the Edit > Properties...

  3. In the first case, select Properties from the contextual menu displayed.
    The Properties dialog box appears.

  4. Click OK.

  5. Click Mechanism Analysis .
    The general properties of the kinematics mechanism are displayed.

  6. (Optional) Select another mechanism using the Mechanism name drop-down list.

  7. Select the Show joints option, this is what you obtain:

  8. Click to access a graphic representation of the laws associated to each command.
    It is represented by a colored curve. When you pass the cursor along the curve, information about the law is displayed in the status bar.

    For more information, see Analyzing A Mechanism and Simulating with Laws.
