Shuttle

A shuttle is an element representing the object to be fitted or unfitted from an assembly. The shuttle can be moved about to simulate the fitting or unfitting of the object within the assembly. 

The files listed below provide information about how to use shuttles.

Defining Shuttles
Select part, click Shuttle, use the Angle option, then click OK.
About Shuttles
Gives background information about shuttles.
Editing a Shuttle
In the specification tree, either:
  • Double-click the shuttle you defined to modify it
  • Right-click the shuttle and select Shuttle.1 object > Definition .

Edit the fields required in the Edit Shuttle dialog box and select the OK button.

Snapping Objects
Create a shuttle to be snapped onto an object and click Define Snapping Axis . Customize Define Snapping Axis settings if necessary (Tools > Options > DMU Fitting > DMU Manipulation).  Record your simulation with Automatic Insert option.
