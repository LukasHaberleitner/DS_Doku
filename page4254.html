Adding a Via or Goto Point  

This procedure describes how to add a via or goto point to a process.

A via point is typically added to keep the CMM probe from clashing with the part as it measures the part.

  1. Select the parent process or preceding activity for the via point activity. 

  2. From the Inspection Activities toolbar, click Goto Point.

    The Define GOTO Point dialog box appears.
    You can either set the via point using the spinners or you can click on the 3D geometry.  The via point is marked with a blue "x."

  3. Select the OK button. 

    The via or Goto point activity appears on the PPR tree.
    The point appears as part of the measure path.
