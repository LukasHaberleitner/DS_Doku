Split

   This tab lets you decide of the type of split to apply.
  • If you clear the Split with surface reduction check box, only standard splits will be applied to the components, whatever their type.
  • If you select the Split with surface reduction check box, this split will be applied as described in the chapter Split with Surface Reduction.
    • when this check box is selected, the Coefficient of reduction field becomes available. It applies to the radius of the circular contour used to reduce the splitting surface. Its default value is 1.2. Its minimum value is 1.01 and its maximum value is 2.
  • By default, the check box is selected.
