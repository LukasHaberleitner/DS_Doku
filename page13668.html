Replacing User Defined Features

This task explains how you can replace the user defined features of the contextual endcuts and slots on shapes.
  • Open a product that contains shapes with at least a slot or an endcut.
  • Ensure that the StrUDFReplacementSample.txt is available in the project resource management (PRM). See List of PRM Resources.

  1. Click Replace UDF   in Physical Plates and Shapes toolbar.
    The Replace UDF dialog box appears.

  2. Select the required shapes from the specification tree or from the graphics area.
    The selected shapes, its feature type, and existing user defined features are displayed in the respective columns.

    Select the root product to select all the shapes that have end cuts and slots.

  3. Select one or more rows in the dialog box and click Compute.
    The system searches the existing user defined feature's name in the StrUDFReplacementSample.txt resource. If it is available, the catalog is searched for the corresponding new user defined feature. The system also checks if the new feature is properly defined.     The replacement user defined features need to be organized under a separate chapter named Replacement_UDF.

    Depending on the computation, the Replacing UDF, Status, and Message columns are updated.

    • If the computation is successful, the name of replacing user defined feature is displayed.
    • If the computation fails, the message states the reason of failure.
    • The Replace button becomes available only if the computation is successful for at least one selected shape.

  4. Select the required row and click Replace.

    If you do not select any rows, all the shapes with successful computation are considered for replacing the user defined features.

    The Status and Message columns are updated.

  5. Click OK  to close the dialog box.

    The user defined features are successfully replaced.
