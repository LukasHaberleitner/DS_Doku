The taper will then be created on the area of the ply staggering,
will be associated to the zone with the lowest thickness and
will ensure the connectivity between the zones.

You can then use the Solid From Zones command to generate the top surface from zones.
This functionality is accessible through the For Solid From Zones Creation Only check box.

Please note that:
  • Those zones created from plies do not correspond to a preliminary design of the Composites part,
    even if they use a similar data structure.
  • They are only created to generate a top surface,
  • They are created in a dedicated zones group, marked (For Solid From Zones Creation Only),
  • They can only be used in the Solid From Zones and the Connection Generator commands,
  • They will be ignored by other commands.

Available in Composites Engineering Design (CPE).
Open the ZonesGroup1.CATPart document.

  1. Click Zones Group in the Preliminary Design toolbar.
    The Zones Group Definition dialog box is displayed.

    A name is proposed by default for the zones group that you can modify.

  2. Select the surface on which the zones will be created.
    A contextual menu is available to create one.
    The draping direction is displayed in the 3D geometry.
    Click Reverse Direction to invert its direction.

  3. Select a Rosette from the list.
    See About Rosette and Rosette Transfer Type for more information.

  4. If necessary, select the For Solid From Zones Creation Only check box.

  5. Under 3D text labels, decide if you want to display label in the 3D window,
    and click ... to select the labels to display.

  6. Click OK to create the zones group.
    The feature (identified as Zones Group.xxx or Zones Group.xxx(For Solid From Zones Creation Only))
    is added to the specification tree, under the Composites PD node.
    This node will contain the structure for all the defined zones.

  7. Perform this scenario as many times as you need to create zones groups.
    In our scenario, we created two zones groups.
