Creating a User-defined Mold Base 

This task shows you how to define the plates for your own mold base.

  1. Click Create a new mold .

  2. By default, the following dialog box is displayed:

    and the mold is pre-visualized:
    This first dialog box is used to define a mold base.
    • In the Plates column you can choose to include any proposed part in your mold base by checking or unchecking the corresponding plate. (by default, the Upper bar and the Stripper are not active, only the CavityPlate and the CorePlate are compulsory)
    • You can enter the thickness of a plate using its spinner.
    • In the Dimensions area, you can define the overall dimensions of the mold base as well as the overhangs for clamping and setting plates.
    • You can also define the overlap value between the core, cavity  and stripper plates. 
    • You can use the core support plate to simulate a sprue stripper plate.

  3. Define the upper bar width, riser  width and ejector width with their spinners.

  4. Use the Enable in the Preview area to display the mold base or not.

  5. Click OK to create the mold.
  • InjectionSide, EjectionSide and EjectorSystem are defined as products for the MoldBase. This way, they can be edited separately.
  • You can modify the formula that defines the default value by:
    • Double-clicking Mold.1 to make it current.
    • Editing the mold via the contextual menu
      (Select Mold.1 in the specification tree then Mold.1 object > Edit Mold in the contextual menu),

  • You can also remove the formula by using Delete in its contextual menu.
