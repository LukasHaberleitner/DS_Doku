Using the Quick Constraint 

This task consists of using this command to create two constraints. The Quick Constraint command creates the first possible constraint as specified in the priority list.
When you select a cylinder/cone or a sphere using the Quick Constraint command, elements taken into selection are respectively the cylinder/cone axis and the sphere center only, independently of the constraint creation ordering list.
In order to select another element of a cylinder/cone/sphere you can:
  • Specify the element to be selected using the User Selection Filter tools, see Selecting Using a Filter.
  • Use the pre-selection navigator (or the keyboard arrows) for selecting the desired element, see Selecting Using the Preselection Navigator in the Infrastructure documentation.
  • Publish the desired element before and select this publication, see Publishing in Assembly Design.
Open an assembly.

Make sure the list specifying the order of constraint creation is composed as follows in the Quick Constraint options:

  • Surface contact
  • Coincidence
  • Offset
  • Angle
  • Parallelism
  • Perpendicularity

  1. Double-click the Quick Constraint

  1. Select an axis.
 

  1. Select the axis of a component.
 

If the application cannot set a surface contact due to the type of selected elements, it creates the second optional constraint mentioned in the list, that is a coincidence constraint.
 

  1. Select the required faces.
 
 

The first constraint in the list can now be set. A surface contact constraint is created.
 
Graphic symbols used for constraints can be customized. For more information, refer to Customizing Constraint Appearance.
