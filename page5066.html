Creating an Advanced Front View

In this task, you will learn about:
  • Creating an Advanced Front View without GVS
  • Creating an Advanced Front View with GVS
 

Creating an Advanced Front View without GVS

  1. Open the CATPart document. Define a new drawing sheet.

  2. Click Advanced Front View in the Views toolbar.

    3. The View Parameters dialog box is displayed.

  3. Modify the View name and Scale value, as required.

  4. Select a plane of projection or planar surface in the 3D part to define the reference plane.

  5. Click in the drawing document to generate the advanced front view.
  Similarly, you can create a front view by using Front View  in the Views toolbar. In this case, the View Parameters dialog box is not displayed.
 

Creating an Advanced Front View with GVS
Generative View Style (GVS) drawings are driven by XML templates and control the appearance and behavior of a view.

To access this mode, you need to go to Tools > Options > Drafting > Administration tab. Clear the Prevent generative view style usage check box.

  1. Open the CATPart document. Define a new drawing sheet.

  2. Click Advanced Front View   in the Views toolbar.

    3. The View Parameters dialog box is displayed.

     

  3. Modify the View name and Scale value, as required.

  4. Select the GVS file you want to use and click OK.

  5. Select a plane of projection or planar surface in the 3D part to define the reference plane.

  6. Click in the drawing document to generate the advanced front view.

    7. The GVS view automatically generates many different 2D text and symbols, such as bulkheads, stiffeners, and opening symbols.

  Similarly, you can create a front view with GVS by using Front View  in the Views toolbar. In this case, the Generative view style dialog box is displayed, where you can select the required GVS file.
