Erasing Faces and Edges

This task shows you how to erase faces and edges. Closed surfaces become open surfaces.
Open the Erasing1.CATPart document.

  1. Click Erasing in the Styling Surfaces toolbar.

    You can also access this command by pressing ALT+SHIFT+E.

  2. Select the face to erase.

    A Tools Palette is displayed:
    The following icons let you:
    • erase a face.
    • erase an edge.
    •  apply the command. This command is active after the first selection. Clicking this command validates the selection and further executes the command.

Erasing a Face

  1. Click Face Erasing (it is selected by default).

  2. Click the face or faces to erase.

  3. Press Ctrl to select multiple elements to erase.

  4. Apply the command by clicking Apply (Enter) or validating in geometry.
    Release the mouse when you are satisfied with the new shape of the surface.

    The same face cannot be cut twice by the same cutting command.
    In the Modification toolbar, click Multi-Selection to validate the current operation and select the next face to erase without exiting the Face Erasing command.
    You can also press Ctrl +Space to validate the current operation and select the next surface to cut.

 
 

Erasing an Edge
 

  1. Click Edge Erasing .

  2. Click the edge to erase.

    The erasing propagates to the connected edges:

  3. Apply the command by clicking Apply (Enter) or validating in geometry.
    Release the mouse when you are satisfied with the new shape of the surface.

    In the Modification toolbar, click Multi-Selection to validate the current operation and select the next face to erase without exiting the Edge Erasing command.
    You can also press Ctrl +Space to validate the current operation and select the next surface to erase.
Edges created using the Face Cutting command can be erased.
  • Multiple edges can be erased at the same time. Select multiple edges using Ctrl or Shift keys and validate by clicking Apply  . The help text describes the edge selection modes.
    • Press Ctrl to:

      • Add an unselected element to the selection.

      • Remove a selected element from the selection.

    • Press Shift to select using a trap:
 

  1. Press Shift.

  2. Draw the trap.

  3. Release Shift.

  • Only closed propagation or propagations up to a free edge are available.
 

Applying the Erase Command

Click Apply (Enter) to validate every face you erase or use Ctrl key to multi-select the surfaces and then click Apply (Enter) to erase them.

  • You can also validate by reselecting the same element again. This information is given in the help text when in detailed mode along with icon in geometry and the pointer changes to . In case a element can be validated but is not active the icon is seen and the pointer does not change.

  • During the command execution the selected elements are previewed on the surface.

  • The command is executed, only after the selection is validated using Apply (Enter) command.

  • Apply (Enter) is automatic applied before quitting the command.

  • When the modification is not possible, the Apply (Enter) icon is not active (grayed out).

  • You can undo or redo the changes only after the completion of command; after the Apply (Enter) command. Intermediate selection states cannot be managed by undo/redo.
