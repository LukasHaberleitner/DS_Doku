Using Applicative Parameter Profiles

An applicative parameter profile can be created for any robot reference. The following steps describe this process.
  • A feature dictionary must already be defined.

  • The project resource management file (Profiles.xml) must specify the feature dictionary to use for the current project.

  1. In the Device Attributes toolbar, click Applicative Parameter Profiles .

    The Create Parameter Profile dialog box appears.

  2. Click the Browse button to display the Class Browser dialog box, which contains hierarchical lists of all available profile types in the associated feature dictionary and the default feature dictionary. Any of the leaf node profile types may be selected for use.

  3. Select a profile type and click OK. An instance of the selected applicative profile is created and displayed in the PPR tree.
