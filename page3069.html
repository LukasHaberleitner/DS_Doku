Importing the Molded Part

This task shows you how to enter the Core & Cavity Design workbench and how to import a part to use there.
Ensure that you have a product (and not a part) in the viewer and select this product before entering Core & Cavity Design workbench.

  1. Start the application with Start > Mechanical Design > Core & Cavity Design.

  2. Click Import Model and import GettingStarted.CATPart from the samples directory.

  3. Click OK.
