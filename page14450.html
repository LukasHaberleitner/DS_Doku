Using the Line ID Menubar

This task shows you how to use the Line ID menubar.  
The Line ID menubar offers four fields you can select, as explained below. It is located beneath the application's menubar.

It is used for routing purposes, and becomes available after you click the Route a Run button and before you start routing. It is grayed out at other times. It is also applicable when placing a part in free space.

It affects your routing and part placement, and the rules are explained below.
1. Click the Route a Run button and select the Line ID (also referred to as logical line) from the drop down list in the first field.
  • If you do not make any other changes any run you route will inherit its size and specification from the Line ID.
  • If you change a value in the Size or Spec field then the run will get its size, spec or both from the value you selected in the menubar.
2. If a standard has been defined for the Line ID then it will display in the Standard field. If no standard has been defined for the Line ID then the standard that displays will be based on an Options setting. This setting is found under Tools > Options > Equipment and System > Standards.
  • Any run you route will inherit its envelope diameter from the Dimension Rule defined in the selected standard.
  • Even if a standard has been defined for the Line ID, or selected in the Options setting, you can override it by clicking the down arrow in the Standard field and selecting another standard. You obviously need to have other available standards.
3. If you place a part in space then the part will inherit data from the Line ID menubar. You cannot change any of the values in the Size, Spec or Standard fields - they will display as defined for the Line ID. You can select a different Line ID. See Select/Filter Line IDs to learn how to select a different Line ID.
