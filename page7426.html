Setting Annotation Parallel to the Screen

This task shows you how to set an annotation text parallel to the screen.
The operating mode described here applies to text, flagnote and note object attribute too.
Open the Annotations_Part_04.CATPart document:
  • Improve the highlight of the related geometry, see Highlighting of the Related Geometry for 3D Annotation.

  1. Right-click the annotation text and select Properties from the contextual menu, and click the Display tab.

  2. Check Parallel to Screen option.

  3. Click the OK.

    The text is now parallel to the screen.

  4. Move and rotate the part.

    The text is always parallel to the screen and its size is constant for any point of view or zoom value.
