Fixing Components Together 

This task consists of fixing two components together.
  • The Fix Together command attaches selected elements together. You can select as many components as you wish, but they must belong to the active component.
  • This command is also available from the Insert menu and works both in Design and Visualization mode.
  • You can select a set of attached components to apply the Fix Together command between this set and other components.
  • You can set constraints between components belonging to a set of components fixed together.
  • When you set a constraint between a component and a set of attached components, the whole set is affected by the constraint.
  • You can deactivate or activate a set of attached components by using the Deactivate/Activate contextual command available in the specification tree. Red parentheses preceding the graphic symbol indicate deactivated sets.
  • Setting a Fix Together between two parts already constrained or creating a constraint between two parts fixed together does not over constrain the assembly. The Update command first solves constraints inside the Fix Together (i.e. moves the parts in the Fix Together) and only then solves constraints outside the Fix Together.
Open an assembly.

  1. Click Fix Together in the Constraints toolbar.
 

  1. Select the first component you want to fix (CRIC_FRAME).
 

  1. Select the second component (CRIC_BRANCH_3).
 
 

The Fix Together dialog box appears, displaying the list of selected components.
 
 

To remove a component from the list, just click it.
 

  1. In the Name box, enter a new name for the group of components you want to create: FT1.
 

  1. Click OK.
 

The specification tree displays this operation.
 

The components are attached to each other, moving one of them:

  • Using the compass combined with the Shift key or using the option With respect to constraints in the Manipulate dialog box, moves the other one too.

  • For the other kind of moves the application displays the Move Warning dialog box to remind you that you are moving components fixed together:

    • The Move Warning dialog box is displayed or not according to the selected options in Tools > Options, select Mechanical Design > Assembly Design, General tab, Move components involved in a Fix Together: Always, Never, Ask each time.

    • According to the selected options in the Move Warning dialog box, the selected option in the Move components involved in a Fix Together is affected.

    • When you select the Extend selection with all involved component option, these components are moved together as long as the option are selected.

    • When you select the Do not show this warning next time option, the dialog box are not be displayed and:

      • If the Extend selection with all involved component option is checked, the Move components involved in a Fix Together option is set to Always.

      • If the Extend selection with all involved component option is unchecked, the Move components involved in a Fix Together option is set to Never.

      • To display again this dialog box, check Ask each time option in Move components involved in a Fix Together.
