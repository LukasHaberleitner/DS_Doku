Routing a Run at an Offset

This task shows you how to route a run at an offset.
You can route a run at an offset from the center as well as the bottom of the route.
See Routing a Run for basic information.
1. Click Offset to Surface in the Run dialog box.
2. In the Offset box, enter the offset distance.

  3. Apply the offset in one of the following ways:
   
  • Click Bottom Offset to apply the offset from the bottom of the route. This command is usually used when routing is above a surface at a defined distance.
   
  • Click Center Offset to apply the offset from the center of the route.
4. Click Section and enter data about the type and size of run.
  5. Create the run on the reference surface.
  6. Click OK.
  • If you press CTRL while using the Offset to Surface command, then the offset value entered is ignored. In this case, the alignment of the run depends on the options Center Offset or Bottom Offset.
  • If you press SHIFT while using the Offset to Surface command, then the offset direction is reversed.
