Creating a Pivot Flange

This task shows you how to compute the compensation of springback effect on a flange.
Open PivotFlange01.CATPart from the samples directory.

  1. Click Pivot Flange  in the Compensation toolbar.

  2. Select the Flange Surface and the Pivot Curve.

  3. Manage the number and position of the profiles.
    • Click a point on the pivot curve to add a profile.
      The number of profiles is updated.
    • Alternatively, keep the proposed number of profiles, or edit it, and click Apply.
      The profiles are dispatched accordingly along the pivot curve.
    • Drag a profile origin to move the profile.
    • Delete a rotation from the contextual menu in the list.


    Profiles are computed in the quantity specified in the dialog box.
    A plane is computed on each profile, perpendicular to the pivot curve.
    An intersection curve between the plane and the flange surface is computed.
    A rotation feature is created for each curve.
    The number and positions of the profiles are updated accordingly.
    The corresponding rotations, identified as Rotate.x, are listed in the dialog box, with their type.

  4. Select a Profile Type to define how the rotation is computed.
    • Angle: The rotation is specified by an angle
    • Distance: the rotation is specified by a distance at the tip of the profile.

  5. Edit the Clearance Length.
    The initial intersection curve is split by a point on the curve.
    Clearance length is the distance between the point and the origin of the curve.
    It is applied to the selected rotations.

  6. Select a Constraint Attenuation (Low, Medium, Strong, Very Strong).

  7. Click OK.
    A Pivot Flanges are created.
