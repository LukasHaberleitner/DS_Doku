Customizing for Analysis

This page deals with the Analysis and Simulation options in the following tabs:

  • The General tab lets you set the general options.
  • The Graphic tab lets you set the graphic options.
  • The Post Processing tab lets you set the post processing options.
  • The Quality tab lets you set the quality criteria.
  • The External Storage tab lets you set the external storage options.
  • The Reporting tab lets you set the reporting options.
