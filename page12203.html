Starting from a Script Skeleton 

This task explains how to write a script by using the Script Editor. This editor provides you with a way to create a skeleton  which reflects the structure of a .CATPart  or of a .CATProduct document. 

 

  1. In the Product Knowledge Template workbench, click the icon. The Script Editor is displayed.

  1. In the Script Editor, select File > New > CATPart document. The dialog box is displayed:

  1. Fill in the fields and click OK. A basic script is created.

  2. Save your script and/or generate the related document.
To enrich this script, see Using the Scripting Language.
