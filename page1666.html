On Cancel Behavior: Data Sheet

Accessing this behavior This behavior can be accessed by clicking the On Cancel Behavior icon () in the Event Based Behaviors toolbar.
Describing this behavior This behavior is designed to track the failure of another behavior known as the Father Object. When the failure of the behavior is detected, the On Cancel behavior executes.
Using this Behavior Interface

 

Name: Name attributed to the behavior.
Comment:  Comment attributed to the behavior.
Track Father Object to Survey

 

 In the behaviors tree, click the father object.
Children Objects This field displays the behaviors located below the main behavior.
Click Add to add children objects.
Child Object to Survey

Update

 Use the arrows to select the behavior whose failure will be tracked and that will activate the On Cancel behavior.

This object available by default enables you to subscribe to the update of a V5 feature or to the operate of a behavior.
Add button Use this button to add a behavior that will be part of the structure.
