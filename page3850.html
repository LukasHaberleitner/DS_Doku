Highlighting of the Related Geometry for 3D Annotation

This page deals with the options concerning:
  • The Selection.

Selection

Please refer to Infrastructure user's guide to know more about the Display Selection options.

Highlight faces and edges
You can select this option to improve the highlight of the related geometry when selected an annotation.
By default, this option is not selected.

Limit display of manipulators
Sets a limit on the number of elements selected in multi-selection mode (using the selection trap) on which manipulators can be displayed.
By default, value is 50 elements.
