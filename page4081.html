Distance Tab

 This page describes the Distance functionality of the Vision dialog box.

The Distance section assigns the following parameters:

  • Ponctum proximum
  • Focus distance
  • Ponctum remotum

These parameters correspond to the depth of the minimal and maximal fields as well as the distance from the focal point.

The diagram below illustrates the dimensions used by the Distance parameters.

Ponctum proximum

This parameter defines the minimum accommodation distance or depth of vision. This value corresponds to the nearest point that can be seen clearly. The default value is 5 centimeters.

Focus distance

The focus or focal distance corresponds to the length of the active line of sight. The default value is 30 centimeters.

Ponctum remotum

This parameter defines the maximum accommodation distance of the vision. It corresponds to the crystalline relax position for objects located from a distance of five meters to infinite. The default value for this parameter is set to infinite (the Infinite button is enabled).
