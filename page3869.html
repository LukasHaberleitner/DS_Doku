Dimension Texts Properties

This reference describes the Dimension Texts properties concerning:
  • Dimension.
 
  This page deals with the properties concerning:
  • The Prefix - Suffix.
  • The Associated Texts.
  • The Dimension score options.
  • The Dimension frame options.
 

Prefix - Suffix
 
Defines a text or a symbol, from the dropdown list, as prefix or suffix for the dimension text.
By default, the dimension prefix and suffix from the standard.

Associated Texts
 
Defines a text before, after, above or below the dimension text main value and dual value.
Dual value associated texts are only available with the Show dual value property selected.

The Before and After texts should not be modified by users for semantic dimensions supporting ISO 14405 2016 or ASME Y14.5 2018 standard. They are removed after the next update of dimension.

For the old data not supporting ISO 14405 2016 or ASME Y14.5 2018 standard, you can insert optional text before or after the repeatitive feature identifier while defining the semantic patten dimension.

However, the semantic pattern dimension becomes invalid if:

  •  The repetitive feature identifier does not correspond to the number of pattern elements the dimension is linked with, or the repetitive feature identifier is missing for the dimension or there are several repetitive feature identifiers.
  • The repetitive feature identifier and the optional before text of the pattern dimension are not separated by an empty character (“ ”).
By default, the associated texts from the standard.
 

Dimension score options
 
  Defines the score application for the main value and dual value:
  • No score.
  • Value.
  • All.
  Dual value score property is only available with the Show dual value property selected.
 

Dimension frame options
 
  Defines what the frame does include for the dimension.
 

Element
  Defines what the frame does include for the dimension elements:
  • Value
  • Value + Tolerance
  • Value + Tolerance + Texts
By default, Value + Tolerance + Texts
 

Main
  Defines what the frame does include for the dimension group:
  • Main.
  • Dual.
  • Main and dual.
  • Both groups.
By default, Main
