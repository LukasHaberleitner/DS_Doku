Simulating in Batch Mode

You can run the simulations in batch mode and verify the simulation results later to save yourself time visualizing the simulation and waiting for the simulation results.
Note: You must define all interference objects and then save CATProcess before launching batch mode command.

  1. Click Generate Simulation Results in Batch Mode .

    The Generate Simulation Results in Batch Mode  dialog box appears.
    By default, the name of the current .CATProcess appears in the Input CATProcess field.  Click the [...] button to select a different file.

  2. Select the programs you want to run.

  3. Select an output file destination.

  4. Click Execute.
The output CATProduct contains all the information of all simulation results. If you choose to store the CATProcess, all the simulation results can be retrieved from the stored CATProcess.
Retrieving the simulation results:

You can retrieve the simulation results using the Load Simulation results command. This command can be accessed from the contextual menu of the last activity of the program on which the batch simulation has been run.

Once the Load Simulation results command has been accessed, all the simulation results are loaded back in the session, including:

  • Fault list: Collisions and Travel Limit Violation information.
  • The Machined stock will be displayed in the viewer. You can analyze the stock using the Stock Analysis    command.
