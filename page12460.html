DPM Programming Interfaces

DPM provides an API (Application Programming Interface) which allows users to author VBScript macros that interact with Process and Resource Definition to read and write process data, and save that data in the any of the supported formats. This functionality can considerably reduce the time the time required to implement a process plan that would otherwise be tedious when implemented in the conventional manner.

All available scripting objects and their interfaces are documented in the Programming Interfaces guide, accessible from the DELMIA Documentation home page.
