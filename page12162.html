Using This Guide

This book describes how to use the Piping Design product. Before you read it, you should be familiar with basic Version 5 concepts such as document windows, standard tool bars, and view tool bars.

To get the most out of this guide, you should start with the tutorial in the Getting Started section.

The remaining sections of the book describe in detail the procedures for using all of the features of the Piping Design product. The procedures are divided into user tasks and customization sections.
