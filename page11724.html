Creating a V5 Machine  

This section describes how to create the actual machine entity on which the kinematics and machine attributes are defined.
If the V5 components organized in Assembling V5 Components to Build a Machine is already open, use that data.  Otherwise, open the V4Axis_Machine.CATProduct file from the  online/mtbug_C2/samples/V5/4Axis Machine directory.

From the NC Machine Tool Builder, click New Machine .

In the Product tree, one node, Applications, has a sub-node, Mechanisms, added.

The Mechanisms node has a leaf, Machine.1.
The New Machine command is the only means of creating a V5 machine when you have inserted existing V5 components. This command cannot be used when data (device, machine, or workcell) is imported from D5.
Activating the New Machine command, will no longer create the NC Controller node in the Product tree.
