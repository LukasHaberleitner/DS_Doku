Setting Default Translator to Panasonic CSR

This section describes how to set the default upload and download translator to Panasonic CSR.

  1. Select Tools > Options > Resource Detailing > Offline Programming tab.

  2. Under Downloader Settings, choose Panasonic to be the Selected Downloader.


     

  3. Under Uploader Settings, choose Panasonic to be the Selected Uploader.

  4. Click OK and exit the options dialog box.
