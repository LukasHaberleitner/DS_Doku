Authoring Vs. Browsing 2D Layout for 3D Design Data

This topic provides information about the authoring vs. browsing capabilities offered by 2D Layout for 3D Design.
 

Authoring 2D Layout for 3D Design Data
  With an appropriate license, 2D Layout for 3D Design offers access to all the layout authoring capabilities such as the creation and modification of layout, views, view filters, 2D geometry, dimensions, annotations, dress-up, 3D outputs, use-edges, constraints, 2D components, etc.

For more information about all these capabilities, refer to the 2D Layout for 3D Design User's Guide.
 

Browsing 2D Layout for 3D Design Data  
  With a Mechanical license, 2D Layout for 3D Design offers access to basic layout browsing capabilities such as visualizing, measuring (non-persistent), printing a layout, updating a layout, generating layout views in a drawing representation, and using product data filtering function. This enables suppliers and partners to understand, check and share the design intent of the 3D representation.

For more information, refer to Commands Available in 2D Layout Browsing Configuration.
