Updating the Standard in an Existing Document   

This task shows you how to update the standard for tolerancing in your existing CATPart or CATProduct.
You must choose a standard before creating the first annotation in a document. See also Standards.

  1. Edit the Annotation Set.X from the specification tree. Right-click it to display the contextual menu and select Properties. (or use Edit > Properties command). 
    The Properties dialog box is displayed.

  2. Click the Standard tab.

  3. Select the same Standard from the drop-down list to update the existing standard being used.

    No information neither warning message is displayed to inform the user that the standard has been updated.

  4. Click OK to validate and close the dialog box.
The standard updated in a CATPart document with Functional Tolerancing and Annotation will be reused when opening the same document in 2D Layout for 3D Design.
