Display Lists

This command allows you to create, edit and store a graphical storage list containing selected geometry within a part model.

The Display List Default
The display list Default is automatically created when opening the dialog box Display Lists. This display list has the following properties:
  • It includes all geometry elements existing in the CATPart and adopts their current 'Hide/Show' status when opening the dialog box.
    The display list Default allows you to return at any time to the original display status of the CATPart.
  • The display list can only be deleted, if no further user-defined display list exists. The complete feature is deleted when closing the dialog box and re-created with the next opening of the dialog box.
  • The 'Hide/Show' status of the elements can be modified via the option Properties in the contextual command and saved with the icon Save active Display List.
    The options Move --> and Hide are not available.
  • New created elements are automatically added to the display list default in the current 'Hide/Show' status and, if existing, to the active user-defined display list. In other user-defined display lists, new created elements are automatically set to 'Hide'.
 

The toolbar Display Lists
 

 
  • Display Lists: The dialog box Display Lists is displayed.
  • Save active Display List: The current 'Hide/Show' status of the elements is saved in the active display list.
 
  • Choose Display List: In this pull-down menu, all defined display lists are displayed and can be selected for quick and efficient graphic display changes.

The dialog box Display Lists
Open the DisplaySets.CATPart document.

  1. Click the Display Lists icon .

    The 'Display Lists' dialog box is displayed.

 

Create a new display list and copy elements
 

  1. Click the icon New empty Display List to create a new display list List.1.
 

  1. Select List.1 in the list Defined Display Lists and click again onto the list name until the list entry List.1 changes to a text field.
 

  1. Change the name 'List.1' to 'Rear Fender' and return to the display list Default.
In the list Target Display Lists the new display list Rear Fender is now selected.
The option Copy --> is activated by default.
The selection field Elements is now active.
 

  1. Select the geometrical set Rear Fender and click Apply in the dialog box Display Lists.
 

  1. Activate the new display list Rear Fender by double click onto the name.
    In the new display list, the geometrical set Rear Fender has been set to 'Show' and all other elements to 'Hide'.
 

  1. Save the new list Rear Fender with Save active Display List .

You can define the following options:
  • New empty Display List: Creates a new display list with all elements set to 'Hide'. Via the selection field Elements you can select elements which shall be set to 'Show' in the new list.
  • New Display List with all visible elements: In the new created display list, the 'Hide/Show' status of the elements is adopted from the active display list.
  • Temporary Display List: As with New empty Display List a new display list will be created. The temporary list has the name of the active list plus a hyphen as suffix.

    temporary display list from Rear Door

    For more precise processing, individual elements can be copied, i. e. set to 'Show' status, to the temporary display list.
    As any other user-defined list, temporary lists can be saved and renamed. However, if a temporary list is not explicitly saved, it will be deleted when activating another display list and it cannot be restored.

    Note: If elements are moved to the temporary list instead of being copied, their 'Hide/Show' status will not be restored in the original list after deletion of the temporary list.
  • Save active Display List: Saves the current 'Hide/Show' status of the elements in the active display list.
  • Delete Display List: Selects one or several user-defined display lists to be deleted.
    The following conditions must be fulfilled in order to be able to delete a display list:
    • The display list to be deleted must be inactive.
    • The display list 'Default' can only be deleted if no further user-defined display list exists.
    • For the geometry assigned to the display list must not be saved any analyses. Otherwise, the display list cannot be deleted and a warning appears indicating the names of the dependent analyses.
  • Defined Display Lists: You can activate a display list by double click in the list of defined display lists. The active display list is marked with *.
    The display lists listed here can be sorted ascending and descending by status (active/inactive) or name.
    In addition, the user-defined display lists can be renamed.
  • Target Display Lists: In this list, a display list can be selected as target for a Copy or Move operation.
  • Copy -->: Transfers the 'Hide/Show' status of the elements selected in the current display list to the target list.
  • Deep Copy -->:  The Show/No Show mode of copied sets (and its parents) is modified in a way that the copied elements (independent from the display mode of their parents) are set to the desired mode.
    If, for example, an element from a hidden geometrical set is copied to a target display list, in the hidden geometrical set only the copied element will be set to no Show mode when activating this display list.
  • Move -->: Sets the selected elements to 'Hide' in the current display list. In the target list the 'Show/Hide' status is taken from the current list.
  • Hide: Sets the elements selected in the current display list to 'Hide'.
  • Elements: Selects elements for Copy -->, Move --> or Hide. You can select individual elements as well as geometrical sets.
  • Change Display List immediately: Activates automatically the target list after a Copy/Move operation.
