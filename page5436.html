Using Miscellaneous Virtual Stacking Tools

This task shows you how to use the tools provided in the Virtual Stacking Tools Palette to select, cut or copy rows in the Virtual Stacking 2D viewer.
Either work on the part where you have defined cells or open the VirtualStackingGA01.CATPart document.

  1. When you have launched a command from the Tools Palette, click Select to exit that command:

    When you start Virtual Stacking, Select in the Tools Palette is active.
    Nothing happens if you click it.
    When you start a command, Select is de-activated and ready to be activated again.
    When you click Select, the current command is ended.

  2. Click Repeat to start a command in repeat mode.
    Repeat is available for:
    • Cell Valuation,
    • Move Row, Swap Rows,
    • Duplicate Row,
    • Cut Row,
    • Paste Row.

    Start one of the above commands.

    • The other commands are greyed.
    • The current command is marked Repeated in the Running Commands dialog box.

  3. Click Move Row in the Tools Palette.
    Move Row is also available from the cell and the row contextual menu.
    • In the table, select the row to move.
    • In the table, select the target row.
      The row to move is placed before the target row.
    • Move Row works for non-symmetric layers or for the upper symmetric layers only

  4. Click Swap Rows in the Tools Palette.
    • In the table, select one row, then another.
    • The two rows are swapped.
    • Swap Rows works for non-symmetrical layers or for the upper symmetrical layers only.

    • Swap Rows is also available from the cell and the row contextual menu.

  5. Click Duplicate Row in the Tools Palette and select a row (sequence or ply) in the table.
    The selected row is duplicated after the selected row.
    • For a ply (Ply.x), its name is Copy of Ply.x
    • For a sequence (Seq.x), its name is Copy of Seq.x
    • Lock symmetry is taken into account.

  6. Click Insert Blank Row in the Tools Palette and a row (sequence or ply) in the table.
    • A blank row is inserted above the one you have selected.
      You must select at least one cell in the graphic area. This cell will be covered by the new entity.
    • For a sequence, the name is New Seq(xx+1) .
    • For a ply, the name of the ply is New(Ply(xx+1),
      the name of the sequence is that of the sequence selected,
      the orientation name is that of the row selected and the field is greyed.
    • The Single row edition dialog box opens to let you edit this row.
    • Insert Blank Row is also available from the row contextual menu.

  7. Click Cut Row in the Tools Palette and select one or several rows in the table.
    The selected rows are deleted.

  8. Click Copy Row in the Tools Palette and select a row.
    • Click Paste Row in the Tools Palette.
    • Paste Row can be used with Copy Row or Cut Row.
    • The selected row is pasted in the table, before the target row, with the name Copy of xx.
    • Copy Row and Paste Row are also available from the row contextual menu.

  9. Click Select Row from Cell in the Tools Palette and pick a cell in the table.
    The whole row is selected.
    Select Row from Cell is also available from the cell contextual menu.
