Above Geometry and 3D Annotation

This task describes the Geometry and 3D Annotation. The relation between 3D Annotation and Define in Work Object has been discussed.
For 3D annotations which refer geometry, the resolution context is always applied on the final geometry result, and each time that this one is modified.

The change of Define in Work Object is not considered as a modification of the geometry, particularly when define in work a previous feature from specification tree.
