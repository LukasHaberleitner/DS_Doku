Multi-Axis Tube Machining: Around Guide

This task shows how to insert a Multi-Axis Tube Machining operation in the program.
This type of Multi-Axis Machining is suitable for parts presenting an obvious central axis.

To create the operation you must define:

  • the geometry to be machined
  • the tool that will be used
  • the parameters of the machining strategy with the tool axis guided in Tangent Axis mode
  • the feedrates and spindle speeds
  • the macros (transition paths) .

See Multi-Axis Tube Machining Parameters for more information on the parameters.
Open the Geometrytube1.CATPart document, then select Machining> Advanced Machining from the Start menu.
Make the Manufacturing Program current in the specification tree.  

  1. Click Multi-Axis Tube Machining .
    The Multi-Axis Tube Machining dialog box appears directly at the Geometry tab page .
    This page includes a sensitive icon to help you specify the geometry to be machined.

  2. Click the red area representing the part to machine.
    The dialog box disappears.
    Select the part as shown below.

    Double-click anywhere in the 3D viewer to validate the selection and revert to the dialog box.
    The red area has turned green.

  3. Right-click the area representing the check.
    The dialog box disappears.
    Select the check as shown below.

    Double-click anywhere in the 3D viewer to validate the selection and revert to the dialog box.
    The check area has turned green.

  4. Go to the Machining Strategy tab.
    Make sure the Guiding strategy is set to Around guide.
    Click the line in the sensitive icon and select the curve as shown below.

    Set the Side to machine to Inside.
