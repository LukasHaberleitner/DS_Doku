Catalog

This task explains how to insert a component into a document from a catalog.

The integration with the Catalog workbench lets you have access to the advanced catalog capabilities such as the component drag and drop.
For more information about Catalogs, refer to CATIA - Infrastructure User's Guide.

  1. Open the Import.catalog file and choose the Preview tab.

  2. Open a new CATProduct document.

  3. Tile the windows horizontally. It looks like this:

  4. Select the components you want to insert into the CATProduct.

  5. Drag and drop it onto the Product.
    An alternative is to copy then paste it (Ctrl-c then Ctrl-v)

The components are copied in the CATProduct document
