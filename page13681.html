More about Manage Piece Parts

There are a few things you need to know while working with Manage Piece Parts command.

There are different filters to display the source features according to the execution time, modified features, and so on. The various filters are:

  • All source features: lists all the source features.
  • Multiple execution: lists the source features that exist in more than one execution log.
  • Modified features: lists the modified source features.
    With the modified feature filter, you have other filters available:
    • Out-of-date: lists the out dated source features. A source feature is considered modified when its modification date is later than the execution date. The latest piece part generation execution date is be used to determine whether the source feature has been modified.
    • New features: lists the source features that is new and does not exist in any execution log.
    • Deleted features: lists the source features that exist in an execution log, but no longer exist in the source part objects.
    • Area: lists the out-of-date source features that have different area than the corresponding piece part objects.
    • Length: lists out-of-date source features that have different length than the corresponding piece part objects.
    • Center of Gravity: lists out-of-date source features that have different center of gravity than the corresponding piece part.
      The openings of a plate are not taken into account for COG calculations.
    • Geometry: lists the out-of-date source features that have different area, length, or center of gravity than the corresponding piece part objects.
    • Reviewed features: lists the modified source features that you have marked as reviewed. Such features are not be displayed in any of the modified sub-filters because they are not considered as modified.
    The Tolerance box is available with Area, Length and COG filters.
