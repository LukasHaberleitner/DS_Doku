FastenerSet (Object)

Interface DELMIAFasteners DELMIAFasteners represent collection of fasteners of both types point and curve.
Method Index
GetFastener Get a specified DELMIAFastener from the collection of Fasteners.

Methods


Func GetFastener( index) As
Get a specified DELMIAFastener from the collection of Fasteners. The specification is through the Index or the location of fastener in the list. Index starts at 1.
Parameters:
index
Integer for the index in the list. It is an input parameter.
opiFastener
Returned instance of DELMIAFastener. It contains the return DELMIAFastener at the specifed index. It is an output parameter.
