What's New?

Enhanced Functionality

Allowing Extrapolation for 3D Elements (V5-6R2024SP2)
You can extrapolate the 3D elements when intersecting 3D elements with the sketch plane, projecting 3D elements onto the sketch plane, and offsetting 3D elements.
The Allow extrapolation option is available in the following commands:
  • Project 3D Elements
  • Intersect 3D Elements
  • Offset
                  
See Projecting 3D Elements onto the Sketch Plane, Intersecting 3D Elements onto the Sketch Plane, and Offsetting Elements
Creating 3D Planes in Sketches Using Construction Lines (V5-6R2024GA)
If you create a 3D plane on a sketch with a closed profile and an existing result (for example used to create a pad), you can now duplicate and replace the selected geometries by construction geometries.
It is possible to create 3D planes without impacting the sketch and its result or generating open profile errors.
See Creating 3D Planes.
