Upgrading pre-V5-6R2013 Composites Models

This task shows you how to upgrade pre-V5-6R2013 models so that the new management of laminates as extended laminates is taken into account.
For more information, see More about Materials and Laminates.

  1. You are requested to upgrade your model to V5-6R2013 standards.

  2. If needed, select the check box in Save old model.

  3. Click OK.
    An Upgrade Report is displayed. Click Show all messages to see the details of the migration.
    The specification tree is updated: A Laminates node is created under Composites Parameters.
