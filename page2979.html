Saving a V5 Drafting Document as a CADAM Drafting Model

This task explains how to save a V5 Drafting document as a CADAM Drafting model..
1. Select the File->SaveAs... command.

2. Define CADAM as the file type by defining Save as type: cddcdd.
3. In the SaveAs dialog box, navigate to the appropriate folder.
4. Click the Save button to save the document as a CADAM model.
