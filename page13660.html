Converting Profiles

This task shows you how to convert the profiles and shapes into sub-assemblies like two plates, a plate and a flat bar, two flat bars, single plate or bent plate.
While designing a ship, certain beams and stiffeners have a particular combination of section shape, material and grade which are not easily available. In such cases, two plates or flat bars are welded to create the structural assemblies.
Open a structure containing shapes of various sections.

  1. Click Convert Profile in Physical Plates and Shapes toolbar.
    The Convert Profile dialog box appears.

  2. Select the shapes from the specification tree or from the graphics area.
    The selected shapes are displayed in the SR1 Profile column. The sections of these shapes are displayed in the Section column.

  3. Select one or more rows in the dialog box.

  4. Right-click in the Conversion Mode column and select any one of the structural assembly type in which the selected shape can be converted.
    The different structural assembly types are:

    • Plate/Plate: converts both, web and flange, to two individual plates.
    • Plate/FlatBar: converts web to plate and flange to flat bar.
    • FlatBar/FlatBar: converts both, web and flange, to flat bar.
    • Single Plate: converts flat bar to a plate. This mode is available only for flat bars.
    • Bent Plate: converts a profile with L section to a single plate. The default value for bending radius is used. The bending value can be later modified in the specification tree.
    • Only the conversion modes which are valid for the selected shape are available in the contextual menu.
    • If you select more than one profile of different sections in list, you can still define a common conversion mode from contextual menu for all of them but only those conversion modes are available which are valid for all selected sections.
    • Only certain sections are supported for conversion:
      • Tee: valid conversion modes are Plate/Plate, Plate/FlatBar & FlatBar/FlatBar.
      • L-Shape: valid conversion modes are Plate/Plate, Plate/FlatBar, FlatBar/FlatBar & Bent Plate.
      • Flat Bar: valid conversion mode is Single Plate.

  5. Click OK in the Convert Profile dialog box.
    The Conversion Result dialog box appears displaying the conversion status of shapes.

  6. Click OK.
    The shapes are converted. The newly created structural parts are added in the specification tree and the converted profiles are renamed.

    • The newly created structural parts can be edited in SR1.
    • Depending on the conversion mode, the name for new structural parts will have the name of original profile name with a suffix (Web or Flange)_(Plate or FlatBar).
    • After conversion, the original profiles or shapes get deleted.
    • The end cut information on the profiles are lost.
