Applying Constraints

You can quickly define constraints for sketch elements. To do so, you have two possibilities.

This task shows you how to:

  • Use the Sketch Tools Toolbar
  • Use the Constraints Toolbar

Using the Sketch Tools Toolbar

The Geometrical Constraint and Dimensional  Constraints options are activated by default:

  •  Geometrical Constraint : Creates detected geometrical constraints. This option is only applicable when creating a geometry or while dragging the geometry onto other geometry.
  •  Dimensional Constraint : Creates dimensional constraints. This option is only applicable while entering values in the fields of the Sketch Tools Toolbar.
  •  Automatic Dimensional Constraint : Creates dimensional constraints automatically. This option is not activated by default, and is applicable only when the Dimensional Constraint option is activated.

    • The Shift key lets you temporarily deactivate the detection of geometrical constraints when moving the mouse (SmartPick) even though the Geometrical Constraint option is activated.
    • The Ctrl key lets you temporarily lock the mouse (SmartPick) along the detected geometrical constraint. This restricts mouse movement to this constraint.
    • The Shift + Ctrl key lets you detect the geometrical constraints, but prevents its creation even though the Geometrical Constraint option is activated.
      A small symbol in shape of a lightning bolt  appears above the detected constraints in this case. For more information, see SmartPicking.

Create a rectangle with a corner.

Geometrical Constraints

  1. Keep only Geometrical Constraint activated.

  2. Click Circle from the Profile toolbar.

  3. Select the corner center to define the one of the circle.

  4. Drag the cursor to define the circle dimensions.

  5. Click in the geometry when you are satisfied with the dimensions.

The circle is created and the geometrical constraints have been detected. These constraints can be visualized either in the geometry area or in the specification tree.
 

Optionally, select the circle and move it to see the tangency constraint is well applied.

  1. Right-click the construction line and select Delete.
    The construction line has been deleted as long as its associated geometry.

Dimensional Constraints

  1. Keep only Dimensional Constraint activated.

  2. Click Circle in the Profiles toolbar.

  3. Select the corner center to define the one of the circle.
    Note that this symbol is displayed by the SmartPick as a coincidence between two points is detected.
 

The Sketch tools toolbar expands.

  1. Enter a radius value, for instance 20mm.

  1. Press Enter.

The dimensional constraint is displayed in the geometry.

If you want to keep the dimensional constraints displayed, Dimensional Constraint must be activated.
 

Use the Constraints Toolbar

  1. Keep your sketch open.

  1. Select the rectangle top line.

  1. Click Constraint in the Constraints toolbar.

  1. Click in the geometry to create the constraint.

  1. Select the rectangle left hand side line.

  1. Click Constraint from the Constraints toolbar.
    The geometry color is swapped to the purple color which means that it is over constraint.
 
You can use the Sketch analysis to have a diagnosis the over-constraint elements. This option will be seen later on in this tutorial in the Analyzing Sketches section which comes next.

  1. Click in the geometry to create the constraint.

  1. Right-click the Radius constraint and select Delete.
    The constraint is deleted and the geometry is no longer over-constrained.
  • You can also set constraints using Constraints Defined in Dialog Box available in the Constraint toolbar. You just need to select the constraints you wish to apply.
  • You can refer to Selecting Using a Filter and Selecting Using Other Selection... Command in the
    Infrastructure User's Guide to help you select the elements that you want to constrain.
 

Adding Equivalent Dimension
 

You can add new dimension constraints to the existing equivalent dimensions group of same type with minimum steps.

  1. Open a sketch containing at least one equivalent dimension group and some other constraints.

  1. Select a constraint from the existing equivalent dimensions group.

  2. Press Ctrl and select one or more normal constraints of the same type.

  3. Right-click the sketch and from the contextual menu, select Selected Objects and then select Add Equivalent Dimension.
    • If few of the selected constraints are with a formula, a warning appears.
    • Click Yes to add the selected constraints in the equivalent dimensions group ignoring the constraints with formula.
    • Click No to cancel the selection of constraints.
    The selected constraint is added to the existing equivalent dimensions group and its value is changed to the value of existing equivalent dimension.
