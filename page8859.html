Modifying and Deforming Curves

This task shows you how to manipulate and deform curves.
 

There are several ways to deform a curve by clicking Modification .
You can either:

  • select the element if the icon is active,
  • select the element first then launch the icon if it is not active,
  • double-click the element in the specification tree or in the 3D geometry.
  A Tools Palette is displayed:
 

  The following icons let you:
 
  • refine a curve
 
  • manipulate a curve
 
  • smooth a curve
 
  • define a fillet
  • erase a part of the curve
  •  create a curve lying on an existing surface or refine the same. This option is activated only after a surface is created.
 
  • keep the curve on a plane
  • switch the view to the initial plane.
 
  • define the curve characteristics
 
  • define the working area
 
  • display the transformation icons
