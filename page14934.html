About Weld

The Weld Design application lets you create:
  • A weld using two creation modes, see Weld Creation Mode.
  • A joint which is a set of two or more components which geometry is used for creating welds, see Creating Joints.
  • A joint body which is a set of welds, see Creating Joint Bodies.
  • A simple or double (symmetrical) weld.
The welding of different geometrical elements of the same part is possible only if:
  • Its shapes are in separated bodies.
  • Its shape is weld prepared.
