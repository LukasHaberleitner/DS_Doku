Locating Reference/Resulting Views

This task shows how to navigate easily between reference/resulting views in the case of section views, section cuts, auxiliary views or detail views:
  • You can locate the reference (i.e. parent) view of a resulting view.
  • You can locate the resulting (i.e. child) view of a reference view.
Open the GenDrafting_view_navigation.CATDrawing document. 

Locating a Reference View
 

  1. Right-click the detail view.

  2. Select Locate Reference View. The application window is reframed on the view where the detail view callout is defined.
 

Locating a Resulting View
 

  1. Right-click the detail view callout.

  2. Select Locate Resulting View. The application window is reframed on the detail view.
