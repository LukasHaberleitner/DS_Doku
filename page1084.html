WorkPackage Management

This task shows you how to customize Product Structure settings, for workpackage management:
  • Creation
  • Reference PSN Files
  • Mapping

Creation

Create an intermediate node: Select this check box to create an intermediate node for synchronization.

Reference PSN Files

Lets you browse the PSN (Product Structure Navigation) files and manage their opening options.

PSN Opening: Lets you specify the PSN file opening options.
The upper box lets you select the instances and the lower box lets you choose between static or dynamic opening.

Mapping

There are 2 different mappings, one for product mode and one for visu mode:

Product Mode

In this mode, the references are loaded for mapping.

  • Version
    • Standard Property: Lets you choose a standard property from the list to retrieve a version
    •  Added Property: Lets you specify another property manually
  • Revision
    • Standard Property: Lets you choose a standard property from the list to retrieve a revision
    • Added Property: Lets you specify another property manually

 

Visu Mode

In this mode, only instances are loaded and the documents are replaced by cgrs, less heavy files. In this mode, you can also choose the synchronization attributes:

Synchronization Attributes

  • Part Name
  • Version
  • Revision
  • Add Info
