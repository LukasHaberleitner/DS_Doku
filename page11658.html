Generating Simulation Results Interactively

This procedure describes how to to access results of simulation without replaying the simulation graphically.
This command can be launched at the Manufacturing Program or Machining Operation level.

  1. Click Generate Simulation Results Interactively .

  2. Select any manufacturing program or machining operation.

  3. A progress bar appears indicating the progress.

  4. Once the generation is complete, a message will be given to the user indicating the number of faults detected.

  5. The Fault List command can be subsequently accessed to analyze faults.

    Simulation results generated include the list of Collisions and Travel Limit violations. If the option to compute material removal during simulation has been turned on, then the final machined stock is also displayed. This can be analyzed using the Stock Analysis command.
