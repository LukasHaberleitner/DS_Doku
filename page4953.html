Capturing Section Results

This task illustrates how to export sectioning results to a CGM document using the Capture command.
For more information on this command, see the Infrastructure User's Guide.

 

  1. Generate your section.

  2. In the active viewer, select Tools > Image > Capture.

  3. Click Vector Mode in the Capture toolbar.

  4. Click Save As  . Different CGM types are proposed in the Save As dialog box

The CGM file can then be opened (File > Open) in the DMU Mockup 2D workbench and used for comparison purposes. For more information on 2D comparison functionalities, see the DMU Navigator's User Guide.
