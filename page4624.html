Reframing an Activity in the PPR Tree and PERT Viewer

 

This procedure illustrates how to easily locate a specific activity using the Reframe Activity in PPR/PERT Viewer command to center and highlight it within the PPR tree or PERT viewer.

  1. Search for a given activity Edit > Search.

  2. Click the Select button to select the highlighted activity:

  3. Click Reframe Activity to center the selected activity on the PPR tree and PERT chart.
