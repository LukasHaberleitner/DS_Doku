Customizing

Before you start your first working session, you can customize the way you work to suit your habits.
This type of customization is stored in permanent setting files: these settings are not lost if you end your session.

  1. Select the Tools  > Options command.
    The Options dialog box displays.

  2. On the navigation bar on left of the dialog box, select the Ergonomics and Design node, and then Human Builder.
    The tabs for the workbench appear.

    These tabs allow you to define the following options:

    • General options

    • Display options

    • Vehicle Occupant Accommodation options (license required).

  3. Set options in these tabs according to your needs.

  4. Click OK when done.
