Defining Auxiliary Devices  

This procedure explains how to add auxiliary devices to a device, such as a rail or track. 
To perform this procedure, you must have a robot and a device (such as a rail) loaded in the PPR tree (a .CATProcess document).
 

  1. From the Robot Controller toolbar, select Define Auxiliary device.

    The Define Auxiliary Devices dialog box appears.

  2. Select the robot from the PPR tree.

  3. Select the auxiliary device on the PPR tree.

  4. From the list in the Available types field, select the type of auxiliary device.

  5. Click the Apply button.  

    The device's data appears in the list:

  6. Repeat above steps for any additional devices; then click Close.
