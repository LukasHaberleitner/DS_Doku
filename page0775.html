General

 
  This page deals with options concerning:
  • Command Behavior,
  • Warm Start.

This page is found in Digitized Shape Editor, Quick Surface Reconstruction, Shape Sculptor and STL Rapid Prototyping.
 

Command Behavior

Select this option to activate undo/redo for the following commands:

  • Filter
  • Remove,
  • Mesh Cleaner,
  • Fill Holes,
  • Interactive Triangle Creation.
  By default, this option is not selected.
 

Warm Start

When a cloud of point contains more points than the value defined, the warm start is disabled.
  By default, this value is set to 1000000.
