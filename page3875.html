View/Annotation Planes

  Use a View/Annotation Plane: view/annotation plane description.
Create Principal Views: click this icon and select required views from the Principal View dialog box.
Create a Front View: click this icon and select Projection View  as view type in the View Creation dialog box.

Create a Section View/Annotation Plane: click this icon and select Section View as view type in the View Creation dialog box.
Create a Section Cut View/Annotation Plane: click this icon and select Section Cut as view type in the View Creation dialog box.
Create an Axonometric View: click this icon and select Axonometric View from the Tools Palette toolbar.
Create an Offset Section View/Section Cut: click this icon and define or select a cutting profile.
Create an Aligned Section View/Section Cut: click this icon and define or select a cutting profile.
Clip Annotations Plane: click this icon, the part is clipped by the annotation plane.
Activate a View/Annotation Plane: double-click the desired view/annotation plane.
Edit View/Annotation Plane Properties: right-click the view/annotation plane, and select the Properties contextual command.
  Manage View Associativity: right-click the view/annotation plane, and select the Change View Support contextual command.
  Capture Display Area: right-click the view/annotation plane and select XXX object > Display Area > Capture.
  Display View: right-click the view/annotation plane and select Display View.
