Moving the Middle Points of a Route

This task explains how to move the middle points of a route using node manipulators.
1. Select a route. Node manipulators (blue squares) display at the middle points (nodes) of the route.
2.

Select a manipulator, and move the node as required. As you move the manipulator, a dashed triangle shows how the route is being reshaped.

When you reach the desired position, the manipulator moves to this location and the route is reshaped.
When a node segment is connected to equipment or to another route, movement of the node manipulator is restricted.
In the examples above, the bottom node segment is connected to equipment, so movement of the node manipulator is restricted. You can only move it horizontally.

The top node segment is connected to another route, so movement of the node manipulator is restricted. You can only move it horizontally, and only up until the other route segment.
In the examples below, the node segments are not connected to equipment or to other routes.
Because this node segment is not connected to equipment, you can move the manipulator in any direction.

 

Because this node segment is not connected to another route, you can move the manipulator with no restrictions.
