Duplicating Harnesses

This task shows you how to duplicate harnesses stored as workpackages (Document kept - Publications Exposed) in ENOVIA.

This command will prove useful for versioning and configuration purposes as well as when re-using existing harnesses.
This command is only available if the document you are editing comes from ENOVIA.

Open a document stored in ENOVIA.

  1. Activate the Target product in which duplicated harness is required (Root Product contains harness to be duplicated, Target Product is where duplicate harness is required).

  2. Click Duplicate Harness .

  3. Select the geometrical bundle or the electrical bundle you want to duplicate from the RootProduct.
You can select:
  • A geometrical bundle previously stored as a workpackage.
  • An electrical bundle, only if the common parent of the electrical bundle and its associated geometrical bundle was stored as a workpackage.

Valid Selections:

  • One Electrical Bundle
  • One or more geometrical Bundle(s)
  • Selection of Multiple geometrical Bundles
    • Press Ctrl Key
    • Select all but one geometrical bundles to be duplicated
    • Release Ctrl key (to indicate the last selection)
    • Select remaining/ last geometrical bundles

If your selection does not meet the appropriate requirement, then an error message is displayed.
If the source product is not saved, a dialog will be displayed to warn you about possible loss of data. Click OK to proceed to continue to next step. Click Cancel to exit the command
 

  1. In the dialog box that appears, enter the suffix to add to new reference harness components.

  2. Enter alphanumeric suffix to be added to duplicated data.

  3. Click OK.

The harness will be duplicated in target product and the editor with source product will be closed.
  • The suffix is added to geometrical bundles, electrical bundles, multi-branchable documents and protective coverings, but not to devices and supports.
  • Duplication of harness in NOT allowed inside a product in same editor. An error message will be shown if source and target products are in same editor.
  • If you duplicated a geometrical bundle, a new geometrical bundle having the same content is generated.
  • If you duplicated an electrical bundle, a new document containing an electrical bundle and the associated geometrical bundle is generated.

Notes:

  • All instances of devices, supports, and wires from catalogs are re-instantiated in the new document.
  • All instances of multi-branchable documents, adaptive supports, and protective coverings generate new references that are instantiated in the new document. All relations (constraints, contextual design and electrical connections) are kept.
  • New geometrical bundle and electrical bundle references are generated.
