Opening the Material Library Workbench

This tasks explains how to load the Material Workbench and open a CATMaterial document.

Select Start > Infrastructure > Material Library.

The Material Library workbench is loaded, and a CATMaterial document is created.

  • By default a new family and a new material are in the document, ready to be renamed and edited.
  • The background color of the Material Library Editor is white and cannot be changed using settings.
