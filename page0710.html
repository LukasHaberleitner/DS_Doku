Components  

This page deals with the options concerning:

  • the working catalog
  • the component default height
  • the constraint use.

Default Catalog

Selected catalog file

Enter the path of the folder in which the catalog of components is stored, or use to locate the folder of interest.

By default, this option is left blank.

Component

 

Default height

When importing data, the .lib file contains the component footprint.

You need to set a default height for the component so if its height is null, the Default height will be used as the component pad height.

By default, the default height is 0.100.

With constraints

When you select this option, mechanical constraints will be generated according to the component status:

  • if the component is fixed: FixTogether constraint
  • if the component is placed: Offset constraint.

By default this option is cleared.

With random color at import

Select the With random color at import check box to set a random color to the component representation when importing it from library files.

By default this option is selected.
