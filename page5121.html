Connecting Tool Paths

This task explains how to connect a tool path.

  1. The tool path and a dialog box are displayed.

  2. Select the points to connect using the Selection bar

  3. Select a connection mode using the Connection mode bar

  4.  Click OK to validate and revert to the main dialog box.
If you want to check the tool path, choose the operation that you used to create it and click Replay.
You will see that the gap in the tool path is now closed.
