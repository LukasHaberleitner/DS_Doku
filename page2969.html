Performing a Simulation at Inspection Points

This task shows you how to perform a forming simulation at inspection points.
Open the QuickSimulation01.CATProduct document.

  1. Click Producibility Inspection .

  2. Under Create file from, select Complete Product.
    You could also create the file from a selection of parts contained in the product.

  3. If cut-pieces exist, decide to take them into account or not.

  4. Select the body INSPECTION POINTS under FORMING-SIMULATION (make them visible).

  5. Click Apply.
    A quick forming simulation is launched to retrieve the actual fiber direction at the inspection points.

  6. Under Export Data, enter the path and name of the file to create.
    Select Open file after creation.
    Click Export Data.
    The inspection data is exported as a spreadsheet, ready to be reviewed.
