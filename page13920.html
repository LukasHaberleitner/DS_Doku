Enabling SmarTeam Global Refresh  

This section describes the procedure which the administrator must perform in order to configure the use of the Global Refresh operation with the SmarTeam database.
   

To enable the use of Global Refresh:

  1. In Script Maintenance, select CATIA Product from the Documents class. In the right pane, open the User Defined tab.
  2. In the Operation column, add a key: Global Refresh

  1. Double-click in the Function Name cell of the Global Refresh operation. The Script Browser window opens.
  2. In the File Name column, select GlobalRefreshST.bs.
    In the Functions column, select GlobalRefreshST.
  3. Click Compile.
  4. After clicking OK, the Global Refresh script is hooked to the database.

Note

Following this procedure, you may update all CATIA products stored with the CATIA Product class in the database.
To update CATIA products stored with other classes, e.g., CATIA Standard Product class, repeat the same steps for other CATIA Product files classes.
