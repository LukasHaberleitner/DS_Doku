Tree Manipulation

 
This tab deals with the options concerning:
  • scroll
  • automatic expand
  • zoom on tree

Scroll
 

Automatic scroll activation during DragDrop

If this option is selected along with the Activate the Visualization of Show/No Show option, you can scroll the tree automatically when dragging and dropping icons or objects onto it.
The tree is scrolled until you remove the mouse from the scroll area.

For more information on dragging and dropping, refer to Dragging and Dropping Icons and Objects in this guide.

By default, this option is activated.

Display geometry during tree scrolling/DragDrop

If this option is selected, you can see the geometry while scrolling the specification tree using the scroll bar displayed to the left.

Note that this option has no impact when scrolling the specification tree using the mouse scroll button since the geometry is displayed. And, when dragging and dropping icons with the Automatic scroll activation during DragDrop option activated, the geometry is displayed.

By default, this option is cleared.

Zoom on Tree
 
  If this option is selected, it means that if you click any branch in the tree, the geometry is dimmed and only the tree is active, enabling you to manipulate it on its own (zoom it up and down, etc.).

When deactivated, the only way to manipulate the graph is to click the reference axis in the bottom right corner of the document.

By default, this option is activated.
