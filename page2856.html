Creating a Component

This task explains how to create and manage components.
This feature is available for both the board and the manufacturing panel.
Open the Component.CATProduct document.
The window looks like this:

  1. Select Create a Component .

    The New Component dialog box opens.

  2. Enter comp1 in the Package number field.

  3. Optionally, select a type for the component: electrical or mechanical.

  4. Select Pad.1 in the specification tree. 
    It is the geometrical representation of the new component.

  5. Click OK to validate.

The component is created.
