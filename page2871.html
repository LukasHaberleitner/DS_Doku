Changing the Constraint Areas
Color Settings

This task shows you how to change the user settings. These are permanent settings.

  1. Select Tools > Options.
    The Options dialog box opens.

  2. Click Equipments & Systems in the left-hand box.

  3. Click Circuit Board Design:

    The Graphic tab is displayed.
    • You can set the color of your choice to the different areas.
      If you check the Update documents in session box, your change will be applied to the current document.

  4. Click OK to validate.
