Analyzing Patterns

You can analyze all the patterns in a part with a particular type or context.

Open an existing 3D shape with different types of patterns. 

  1. From the Analysis option, select Pattern Analysis . 

  1. In the Pattern Analysis dialog box, select the filters by type and by context.

    Filter Option Check Box
       
    By Type
    • Rectangular
    • Circular
    • User
    By Context
    • Solid
    • Volume
    • Surface
    Patterns Found displays the total number of patterns. 

  2. Click Analyze.

    The analysis is shown in the table. 
      You can change the type and context filters and click Analyze again to generate a new analysis. 

  3. Click Close.
