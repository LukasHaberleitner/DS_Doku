Robot Cable Simulation Toolbar

Licensing
  Depending on your licensing, you may not have access to the Robot Cable Simulation toolbar.

If you set the environment variable DLM_CABLE_SOLVER_NAME to either DNBFlexilutionCableSolverImpl or CATFLXCableSolverImpl, the solver specified by the environment variable DLM_CABLE_SOLVER_NAME is be used. If the environment variable is not enabled, the system uses the Flex Dynamic Cable Simulation (FDS) solver, provided the FDS license is available.
