Apply Modes

The Apply Modes functionality is available in various functions for managing the update mode of the active feature during its edition.
You can define the following options:
 
  • Dynamic: Feature update occurs dynamically when moving manipulators or editing parameter values by sliders in the dialog box.
    Note:
    The 'Dynamic' option is not available for for each command.
  • Preview: When geometry is modified via manipulator and left mouse button pressed, a preview will be displayed first. The original geometry remains unchanged. Feature update only occurs upon release of the left mouse button.
  • Static: Feature update only occurs upon release of manipulators.
  • None: Feature update only occurs upon selection of the Apply button.
