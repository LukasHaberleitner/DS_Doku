Locking the Current Compass Orientation

This task explains how to lock the current compass orientation.
Open the document Manipulators.CATProduct.

  1. Point to the compass and right-click to display the contextual menu.

  2. Select Lock Current Orientation.

  3. Drag the compass and drop it onto the object.

    When you drop the compass onto the object, this orientation is kept.
    Note that as you drag the compass over objects, the compass symbol remains oriented the same. For example, if the default orientation was current before dragging, the symbol remains like this while you drag:
    When Lock current orientation mode is selected, the compass orientation cannot be changed (either manually or automatically) whatever actions are done with the compass. In particular, the compass orientation is kept even if Use Local Axis System is selected.

  4. Right-click then clear Lock Current Orientation.
    The compass is reoriented.
