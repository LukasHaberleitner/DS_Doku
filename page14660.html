Managing Structural Objects in Geometrical Sets

This task explains how to manage the structural objects in geometrical sets without switching to Generative Shape Design workbench.
Open a product containing superplates, plates, stiffeners, openings, slots, and other structural objects.

  1. With the document open, select the structural objects to be grouped.

  2. Select Insert > Geometrical Tools > Geometrical Set... .

    The Insert Geometrical Set dialog box appears displaying the features which will be grouped in the newly created geometrical set. For the pre-selected elements, the name of the part body appears in the Father box.

  3. In the Name box, enter the name for geometrical set.

  4. If required, in the Father list, select the body where the new geometrical set is to be inserted.

  5. Click OK.
    The geometrical set is created and the selected structural objects are grouped under the geometrical set.
You can drag and drop the newly created structural objects inside an existing geometrical set. For more information about geometrical sets, refer to Generative Shape Design User's Guide: Managing Geometrical Sets.
