Defining a Termination

This task explains how to define a termination on a connector.
The termination is indissociable from the electrical component and corresponds to a contact crimped into a cavity.
Open any document containing a connector where you want to place a termination.

  1. Click Define Termination .

  2. Select the electrical device where you want the termination to be defined.

    The Termination Definition dialog box opens:

  3. Change/enter a value in the IdNumber field.
     Note that the IdNumber must be unique.

  4. Optionally select a Representation, for example a pocket, a pad, a face...

  5. Click OK to validate.

    The specification tree is updated.
A termination is allowed on connectors.
