Getting Started

The following short tutorial provides an introduction to the Equipment Arrangement product.  It is intended to give you a feel for the product's capabilities in a few step-by-step scenarios, which are listed below.

Entering the Workbench
Placing Equipment
Changing the Current Axis
Set Correct Working Units and Grid
Saving Documents
Updating Documents
Step Input Box & Construction Elements

These tasks can be completed in about 15 minutes.
Certain functions will not work without setting up directory paths and options. The system administrator should refer to the tasks under Understanding Project Resource Management as well as platform dependent sample data in Finding sample data on various platforms.

The task Setting Up the Application (in the Customizing section) describes the various steps you have to take, and the order in which you have to do them, to set up Equipment Arrangement.
