Mono-Shuttle Fitting Simulation

Recording a Simulation
Select Insert > Simulation... from the menu bar. Move the shuttle as often as necessary, clicking Insert to record shots, and then click OK to confirm your operation.
Recording a Simulation Automatically
Select Insert > Simulation... from the menu bar, then select the Automatic insert check box in the Edit Simulation dialog box. When done, click OK and start recording your simulation.
Using the Smooth Command
Select the simulation in the specification tree and click Smooth . Click OK.
Animating a Viewpoint
Replay your simulation with the Animate viewpoint check box selected. The simulation is replayed with the viewpoint changes recorded in the simulation.
Generating a Replay
Select Tools > Simulation > Generate Replay.  Select the track or the simulation from which you want to generate a replay. Click OK to create a replay object.
Replaying a Compiled Simulation
Select Tools >  Simulation >  Replay or double-click the Replay in the specification tree. Replay the recorded simulation using the VCR buttons. 

In the V5 Infrastructure User's Guide, see also Recording Interactions in Video Format, which provides information on how to generate an AVI file with the standard tools.
