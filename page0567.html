Managing the Default Parameters

This section explains and illustrates how to use or modify various kinds of features.

Using Aerospace Sheetmetal Design assumes that you are in a CATPart document.
Edit the parameters: select the Parameters tab and define the element thickness and bend radius values.
Compute the bend allowance: select the Bend Allowance tab and define the allowance value (K factor).
Define the compensations and runout: select the Joggles tab and define the compensations and the runout for the joggle.
  Please refer to the Customizing chapter to define the Sheet Standards Files.
