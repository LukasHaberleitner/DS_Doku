Creating Cableways

This section describes ways in which you can create cableways. A cableway is a space reservation through which cables can pass.

When routing cables with ENOVIA V5, the following cableways are supported: hangers and lofts, raceway runs, conduit runs.

For non-ENOVIA V5 cable routing, the following cableways are supported: hangers and lofts, path reservation.

You must place hangers before you can create a lofted pathway. This section contains a more detailed explanation for creating lofted pathways, compared to the task in the Getting Started section.
