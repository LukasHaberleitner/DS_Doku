Copy/Paste Behavior: Data Sheet

Accessing this behavior This behavior can be accessed by clicking the Copy/Paste Behavior icon () in the Reuse toolbar.
Describing this behavior This behavior is designed to copy a feature and to paste it into a document. For more information about the Copy/Paste commands, see Part Design User's Guide and Shape Design & Styling User's Guide.
More information Note that the output feature of this behavior is a PastedFeature.
Using this Behavior Interface

 

Name: Name attributed to the behavior.
Comment: Comment attributed to the behavior.
Copy Feature to copy: Indicate the name of the geometric feature to copy by clicking it in the Behaviors tree.
Paste

 

 Destination: Indicate the destination of the instantiation. It can be an object or a body.
