Visualizing 2D Annotations

This task shows you how to visualize the 2D annotations in a document.
The following task can be performed using the cache system option off only. See Annotations and Cache System.
Open the Review_Product_01.CATProduct document.

  1. Click the 2D Visualization option:

A new window is displayed with the annotations created using CATIA V4 FD&T function presented in a table.

  1. Select Window  >  Tile vertically from the menu bar to visualize the 2D and 3D windows.

  1. Select in the 3D window a dimension in the expanded specification tree.

  1. Zoom in in the 2D window.
 

The 2D Visualization command works exactly the same way as the 3D Visualization command. When an annotation is selected in the model the 2D table reframes on the corresponding annotation.
 

In a nutshell, there is a complete associativity between the 2D table, the specification tree and the 3D model.
