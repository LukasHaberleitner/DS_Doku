Creating a Network
Now that you have a conveyor system, create a conveyor network within the system.
1. Double-click the Conveyor System in the specification  tree to make it active.
2. Select the Logical Line button .
3. Select Conveyor Network as the type.
4. Click OK. 

The Conveyor Network is created.
5. In the specification tree, click on the plus sign (+) next to the Conveyor System to see the newly created conveyor network.
