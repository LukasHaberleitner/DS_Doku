Modifying a Catalog

This task explains the necessary steps a librarian should follow whenever he wishes to modify CATIA catalogs saved in SmarTeam.

Whatever modifications required (deletion or modification of a pointed document for instance), proceed as follows:

  • Modify the Documents
  • Release the CATIA V5 catalog inside SmarTeam
  • Copy the Catalog in a Shared Directory

Modifying the Documents

  1. Go to Tools/Option/Infrastructure/Catalog Editor.

  2. Select the "Instantiate only resolved family component" option under Resolve Family Component.

  3. Set the folder path to the default working folder (SmarTeam work directory).

  4. Check out the catalog containing the documents to be modified.

  5. Check out the documents you wish to modify.

  6. Modify the documents in your CATIA session.

  7. Save the documents in the SmarTeam database.

Releasing the CATIA V5 catalog inside SmarTeam

  1. Release the catalog.
    The Release operation automatically releases all documents. The catalog and all its children are now saved and released in SmarTeam.

Copying the Catalog in a Shared Directory

  1. Set the shared directory folder as the default shared directory folder.

  2. Search for your catalog using the Find command.

  3. Right-click and select Copy File.

  4. Propagate operation on all standard parts.
