Using This Guide

This book describes how to use the Raceway and Conduit Design product. Before you read it, you should be familiar with basic Version 5 concepts such as document windows, standard tool bars, and view tool bars.

To get the most out of this guide, start with the tutorial in the Getting Started section.

The remaining sections of the book describe in detail the procedures for using all features of the product. The procedures are divided into basic, advanced, and customization sections.
