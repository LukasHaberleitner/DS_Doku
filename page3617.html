Upgrading Geometry in Views   

This task shows you how to upgrade sketched geometry in views when opened in different release levels. This will let you benefit from the latest technological enhancements brought to sketched geometry in V5. While saving you the need to transfer geometry in new views as was previously the case.
Open a drawing or a 2D layout containing views with sketched geometry. As you cannot upgrade geometry in locked views, make sure that the views whose geometry you want to upgrade are not locked. You can unlock a view through its properties.
Note: Sketched geometry means geometry that was drawn directly in 2D. Features other than sketched geometry will not be upgraded, such as geometry generated from 3D, annotations, dimensions, dress-up elements and so on.

  1. Select Tools > Options > Mechanical Design > Drafting > Administration tab. In the Drawing Management area, clear Prevent view geometry upgrade.
    This activates the Geometry Upgrade contextual command that is available for drawings, layouts, sheets and views.

  2. Perform one of the following actions:

    • To upgrade sketched geometry (including background geometry) for all views of all sheets (including detail sheets) in a drawing or layout, right-click the drawing or layout in the specification tree, and select Drawing object > Geometry Upgrade in the case of a drawing or Layout object > Geometry Upgrade in the case of a layout.
    • To upgrade sketched geometry (including background geometry) for all views (including detail views) in a drawing or layout sheet, right-click the sheet in the specification tree and select Sheet.n object > Geometry Upgrade.
    • To upgrade sketched geometry for a selection of views (including detail views) in a drawing or layout, right-click the views in the specification tree or on the sheet and select Selected objects > Geometry Upgrade.
    • To upgrade sketched geometry for a single view in a drawing or layout, right-click the view in the specification tree or on the sheet and select X view object > Geometry Upgrade.

    Sketched geometry is upgraded in the selected drawing, layout, sheets or views. Dress-up, annotations and dimensions that are linked to the upgraded geometry are updated.
  • In a 2D Layout for 3D Design context, a manual update of 3D geometry or of some 3D geometry features (such as 3D outputs or use-edges) is sometimes necessary. In such a case, the views are set as being not up-to-date and Update is automatically activated in the Tools toolbar.
  • Occasionally, following an upgrade, some 2D elements may be moved, deleted or their orientation reversed. In this case, annotations, dimensions or dress-up element linked to these 2D elements will also be impacted.
  • The version number of upgraded views is incremented so as to mark the document as modified.
