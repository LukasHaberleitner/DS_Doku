Satellites

  

 
This tab deals with one category of options: Distributed Rendering.
Distributed rendering enables to decrease rendering time since several processors are used to generate the image.
This category concerns the definition of the remote computers ("satellites") used in network to compute a rendered image.
 
This tab is available only when using Photo Studio Optimizer.
  For detailed information on how to use this tab, refer to "Using Distributed Rendering" in the Version 5 - Photo Studio Optimizer User's Guide.
