Isolating Projections and Intersections

This task shows how to isolate the elements resulting from the use of the Project 3D Elements or Intersect 3D Elements operations.
Open the Intersection_Canonic.CATPart document and create an intersection as explained in Intersecting 3D Elements with the Sketch Plane.

  1. Select any yellow element obtained from the projection or the intersection.

  2. Select Insert > Operation > 3D Geometry > Isolate command from the menu bar or right-click Mark.x > Isolate.

  3. The elements are no longer linked to the initial geometry, which means that you can edit them the way you wish.
