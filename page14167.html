Creating a Tubing Assembly

This task explains you how to create a tubing assembly from a part type product.
Open a .CATProduct containing the tubing parts.

  1. Click Build Tubing Part in the Build Create toolbar.
    The     Create Part dialog box appears.

  2. Click Display Class Browser to select the component type.
    The Class Browser dialog box appears.

  3. Select Tubing Assembly.

  4. Select the existing product.
    The name of selected product appears in File Name box.

  5. Select the Assembly option: Standard or Unique.

  6. Click Apply.
    The three options are available for edition:

    • Define Properties
    • Design Table
    • Define Connectors

  7. Click Define Properties   to add the attributes at the assembly level.
    The Properties dialog box appears.
    You can modify the following attributes:

    • End Style, End Style 2: for compatibility check

    • End Style 3, End Style 4: for multi-branching assemblies

    • Nominal Size, Nominal Size 2: for compatibility check

    • Nominal Size 3, Nominal Size 4: for multi-branching assemblies

    • Standard: for compatibility check

    • Schedule

    • Rating

    • Part Name

    • Weight Dry

    • Weight Wet

  8. Click Apply.

  9. Click Design Table to create a design table and associate it to the tubing assembly.
    For more information about design tables, see Infrastructure User's Guide: Using Knowledgeware Capabilities: Relations: Design Tables.

  10. Click Define Connectors to manage the publication name of the assembly.
    The Manage Logical Publications dialog box appear.

  11. Select the required tubing part in the geometry area.
    The Publish dialog box appears displaying the status of connectors whether it is linked or unlinked. You can decide whether to publish or unpublish the connector.

  12. Click Close in the Publish dialog box.
    The information is displayed in the Manage Logical Publications dialog box and a Publication node is added to the specification tree. The tubing assembly is created.

  13. In the ENOVIA V5 VPM Navigator toolbar, click Save in ENOVIA V5 VPM  to save the tubing assembly in the database.
    The Save in ENOVIA V5 VPM dialog box appears.
    For more information about different options in the dialog box, see ENOVIA VPM Navigator User's Guide: VPM Configured Product Design: Saving Objects.

  14. Click OK.
    The tubing assembly is saved in the database.
  • When you open a tubing assembly, containing unique tube bendable, from the catalog and click Build Tubing Part , you can select the unique tube bendable and modify the properties of tube.
  • When you click Build Tubing Part and select the tube bendable from a tubing assembly, only the Define Properties option is available for edition.
