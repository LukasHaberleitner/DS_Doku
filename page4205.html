Using Dual Arm CMMs

These procedures describe how to create processes for dual arm CMMs.

There are three execution modes when multiple arms are present. These modes are:

  • Sequential (which uses Arm 1 or Arm 2 activities)
  • Simultaneous
  • Synchronous

The main node level of the operation activity is considered to be in sequential mode. This means that the simulation (and execution of the DMIS program at the CMM) is done in a "top-to-bottom" way. In the sequential mode, only one arm is moving at any one moment in time. Simultaneous mode means that multiple arms are moving at the same time. Synchronous mode means that multiple arms are moving at the same time AND arriving at specific positions and measurements at the same time. This mode is usually when two arms are measuring symmetric features on two opposite sides of a symmetric part.

Once dual arm activities have been created, all non-dual arm activities must be created as children of the dual arm activities.
