SmarTeamCIX Setting Repository
Quick Reference

SmarTeamCIX

Role: Handling of SMARTEAM (CWI) Data Exchange Import and Export parameters

Detailed role: Handling of SMARTEAM (CWI) Data Exchange Import and Export parameters

The SmarTeamCIX repository has the following characteristics:

  • Primary Tab Page: /General/Compatibility/SMARTEAM CWI
  • Repository File: SmarTeamCIX
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

SmarTeamCIX setting repository attributes:

  • AutoCheckOut
    Role: Display check out/New Release dialog box on first modification
    • Type: boolean
    • Default value: true
  • AutoConnect
    Role: Automatically launch connection on startup
    • Type: boolean
    • Default value: false
  • DisconnectAtExit
    Role: Automatically disconnect from SMARTEAM when exiting CATIA
    • Type: boolean
    • Default value: true
  • EMComponent
    Role: Internal components of CATProduct document
    • Type: boolean
    • Default value: true
  • EMSheet
    Role: Sheet of CATDrawing document
    • Type: boolean
    • Default value: true
  • ServerName
    Role: Server URL
    • Type: string
    • Default value:
  • ServerNameModified
    Role: Modified Server URL
    • Type: integer
    • Default value: 0
  • TreeIcons
    Role: SmarTeam simplified status
    • Type: integer
    • Default value: 1
    • Values:
      0 None
      1 SmarTeam simplified status
      2 SmarTeam detailed status
  • UIShow
    Role: Show menu and toolbar
    • Type: boolean
    • Default value: false
