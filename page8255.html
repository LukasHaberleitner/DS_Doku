Access the Human Posture Workbench 

This task describes how to access the Human Posture Editor by using the Manikin Workbench Access toolbar.
Your manikins must already have been created via the Human Builder workbench.

The Manikin Workbench Access toolbar

 
 

  1. Select the required function from the Manikin Workbench Access toolbar from the Human Builder workbench.

  2. Select Human Posture and the command prompt asks you for Select a segment or a line of sight..., select the required segment.

  3. The software changes to the Human Posture Analysis workbench.


     

  4. Or if you already have an active product containing a manikin, select the manikin or double-click a segment to change to the workbench.
