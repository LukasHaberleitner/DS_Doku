General

 

This page deal with options concerning:

  • Chaining Edges Design Change
  • Colors
  • Analyses
  • Process Behavior
  • Global options

Chaining Edges Design Change
  By default, Use chaining edges geometrical reroute is not selected.

Colors

Sets the default colors of Die Face Design. Click a color patch to select another color.

Analyses

Select or clear a check box to display or hide corresponding information.
Preventing the display of information improves performances.
By default, all check boxes are selected.
 
 

Process Behavior

Enter the value used to extrapolate cutting lines. This value initializes a parameter with the same name. When modified, all the cutting lines of type "outer cut tools" are extrapolated on both sides.
There is no check to see whether the extrapolation intersects the part to stamp.

You can also activate the automatic flange detection for the process generator wizard.
  By default, the extrapolation length is set to 0 mm, and the automatic detection is not selected.
 

Realistic Trim Line

Enter the path

  • of the directory where Abaqus is installed (folder that contains the win_b64 folder)
  • and the path of the catalog containing the materials.

Both inputs are mandatory to run a material base computation for the realistic trim line.
  By default, the fields are empty.
 

Global

Select this check box to display more guidance in

  • Create Press Direction
  • Create Cam Direction
  • Process Method Plan
  • Process Step.
  By default, the check box is not selected.
