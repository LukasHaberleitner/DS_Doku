Recomputing the Virtual Stacking from the Grid

This task shows you how to recompute the virtual stacking when the grid has been modified.
Either work on the part where you have defined cells or open the VirtualStackingGA01.CATPart document.

  1. Make sure Entity level is set to Sequence, and View mode is set to Cells.

  2. Click Recomputation from Grid in the Tools Palette.
    Recomputation from Grid:
    • Imports the grid modification in the virtual stacking,
    • Starts Compare with Current State,
    • De-activates Lock symmetry,
    • The following commands can be activated:
      • Check ply rules,
      • Single row edition,
      • Cell valuation,
      • Insert blank row,
      • Cut Row,
      • Select Row from Cell.
