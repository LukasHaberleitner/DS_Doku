Reviewing Failure Modes

This task shows you how to review the possible failure modes on an annotation and notify the possible failures while manufacturing.
 
Failure Failure modes review is provided in the 3D Functional Tolerancing and Annotation in the descending order, according to the severity (Very high to least important).

When several failure modes are defined with the same severity, the failure mode descriptions are merged.

Review is possible for the data, generated under downward compatibility and only plain text information is displayed for the review.
