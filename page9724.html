WorkerActivity (Object)

The object that represents an Worker activity.

Following activity types qualify as worker activities:
  • MoveToPostureActivity
  • WalkActivity (Forward/Backward/SideStep)
  • AutoWalkActivity
  • PickActivity
  • PlaceActivity
  • HumanActivityGroup
  • HumanCallTask
  • CollisionFreeWalk (Forward and Backward)
    • Property Index
    HumanTask Returns the parent HumanTask
    WorkerResource Returns the Worker-Resource associated with this worker-activity.
