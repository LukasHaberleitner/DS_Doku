Font Properties

This reference describes the Font properties concerning:
  • Text.
  • Flagnote.
  • Datum.
  • Target.
  • Geometrical tolerance.
  • Surface Texture.
  • Dimension.
  • Coordinates Dimension.
  • Note Object Attribute.
Some properties may not be relevant depending of the selected object.
 
  This page deals with the properties concerning:
  • The General.
  • The Attributes.
  • The Character.
  • The Preview.
 

General
 
 

Font
Defines the font name.
By default, Monospac821 BT.
 

Style
Defines the font style.
By default, Regular.
 

Size
Defines the font size.
By default, 3.500 mm.
 

Underline
Defines the underline effect, this property is not compatible with Overline property:
  • None.
  • Underline.
By default, none.
 

Color
Defines the font color.
By default, white. See the screen capture.
 

Attributes
 
 

Strikethrough
Defines whether the strikethrough effect is applied or not.
By default, this option is not selected.
 

Overline
Defines whether the overline effect is applied or not.
This property is not compatible with Underline property.
By default, this option is not selected.
 

Superscript
Defines whether the superscript effect is applied or not.
By default, this option is not selected.
 

Subscript
Defines whether the subscript effect is applied or not.
By default, this option is not selected.
 

Character
 
 

Character
Defines the character ratio.
By default, 100%.
 

Slant
Defines the character slant according to the font capabilities.
By default, 0.000 deg.
 

Spacing
Defines the character spacing.
By default, 20%.
 

Pitch
Defines the character pitch according to the font capabilities.
By default, Fixed.
 

Kerning
Defines the character should be kerned according to the font capabilities.
See Customizing Fonts for Displaying Geometry Area Texts in the Infrastructure documentation.
By default, this option is not selected.
 

Preview
 
Previews the selected properties and the font type.
