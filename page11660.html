Using DELMIA D5 Data

This functionality is an initial enabler for users with already created machine libraries consisting of D5 Virtual NC machines. Two scripts convert these D5 libraries into V5 catalogs and create work system components (WCS) to use with the PPR Hub.

Accessing the scripts

To access the scripts, go to the install directory for your machine.  Then, relative to the install directory, go to ../Startup/Manufacturing/Samples.

  • To create V5 catalogs:

    • DNBCreateCatalog.exe

  • To create WSC from these catalogs:

    • DNBCreateWSC.exe
For more information about using V5 catalogs, see Component Catalog Editor User's Guide.
