Copying Chapters and Keywords

This section explains how to copy catalog chapters as well as keywords from one chapter to another.
 

If you try to copy objects other than keywords or chapter, the error message "It is not allowed to paste" will appear.
 
You have opened the ALL_FASTENERS.catalog, which has the following structure:

Copying keywords

  1. Select the target chapter (i.e. the chapter to which you want to paste keywords).

  2. Right-click the source chapter (i.e. the chapter from which you want to copy keywords) then select Copy keywords....
    The Copy keywords dialog box appears. The source chapter (Chapter 6) and target chapter (ALL FASTENERS) are displayed along with the list of keywords you can copy.

  3. Choose the keywords to be copied from the list then click OK to validate.
The All button lets you select the whole list. You can also use multi-selection.

The keywords are pasted with their properties to the selected location.

 

Copying chapters

  1. Select the chapter to be copied. In our example, we copy Subchapter2.

  1. Select the Copy contextual command or the Edit > Copy command.

  2. Select the chapter under which you want to paste the copied chapter (Chapter.6 in our example).

  3. Select the Paste contextual command or the Edit > Paste command.
    The chapter is copied with its keywords (Name, Diameter, Width and Height) but not its description:

 

When copying a chapter, you cannot copy its description unless the copied chapter contains a subchapter.
