MotionBasis (Enumeration)

enum MotionBasis {
  MOTION_ABSOLUTE,
  MOTION_PERCENT,
  MOTION_TIME
}
Types of motion for the motion profile in robot controller.
Values:
MOTION_ABSOLUTE
Absolute motion control
MOTION_PERCENT
Percentage of speed used for motion
MOTION_TIME
Specify the motion by duration using TIME
