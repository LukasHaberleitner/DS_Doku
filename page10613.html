How to launch the V4 To V5 Migration Batch

This task shows you how to launch the V4 To V5 Migration Batch. Launching V4 to V5 batch can only happen via the PLMV4ToV5Migration Batch. Trying to start the CATV4ToV5Migration batch will prompt the user with the following message as shown below.

You can launch the V4 To V5 Batch by using the Batch Monitor. For more information about it, please refer to Running Batches Using the Batch Monitor, in CATIA - Infrastructure user's Guide. You will learn how to run the V4 To V5 Batch directly from the Batch Monitor, from the command line and in remote mode.
For more information about the Migration Batch settings, please refer to Migration Batch, in CATIA - Infrastructure User's Guide.
 

Using the PLMV4ToV5Migration Batch via the Batch Monitor (from Version 5 Session)

  1. Start a Version 5 Session.

  2. Select the Tools > Utility commands. The Batch Monitor window opens. A list of Version 5 batches along with their description will be displayed in the Type column.

  1. Double-click PLMV4ToV5Migration: this tool allows you to convert CATIA Version 4 models into CATIA Version 5 models and have access to databases as well.

  2. Click the MigrateV4ToV5 line and the Migration Mode window is displayed:
For more information about using this batch, please refer to the succeeding chapters.
