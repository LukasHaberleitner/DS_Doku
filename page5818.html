How is the Wrapping Representation Calculated?

You generate a wrapping from existing geometric elements by:

  • computing a set of points regularly spaced on the envelope of the original geometry (distance between points = grain parameter)
  • offsetting these points to obtain a inflated result which better includes the original geometry (offset value = grain * offset ratio)
  • building triangles from the set of offset points.

When the cubic check box is selected, the result is a voxel (cubic) representation of the envelope. The voxel size equals the grain parameter. The offset ratio is not used.
