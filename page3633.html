Creating an Associated Text

This task shows you how to create a text that will be associated to an existing element. This text will remain associated with this element. You can set text properties either before or after you create the text.
Open the Brackets_views03.CATDrawing document. Create two diameter dimensions, for example.

  1. Click the Text icon in the Annotations toolbar.

  2. Select the element to which you want to associate a text.
    For example, we will use a dimension. In this particular case, you can associate the text to the dimension line by clicking it.

     

    The text frame is displayed as well as the Text Editor dialog box.

  3. Enter the text in the Text Editor dialog box or directly on the drawing. For example, enter "diameter".
    Press Shift+Enter to add new line in the Text Editor if needed and press Enter to validate the text.


    The text is created below the dimension value and associative position and orientation links are created automatically. The text is associated with the position and the orientation of selected sub-part of the dimension.

  4. Click in the free space or click the Select icon to end the text creation.

  5. If needed, select the dimension and move it  to the desired location.

     

    The text remains associated with the position and the selected sub-part of the dimension.

     

     

    • If you select the dimension value, the text is created below the dimension value with the Anchor Point positioned at the Top Center, and the Anchor Mode at Global, by default.
      W      hen you select a dimension line, the text is created below the dimension line with a 1 mm offset.

    • If you select the dimension line, the text is created below the dimension line, with the Anchor Point positioned at the Top Center, and the Anchor Mode at Global by default.

      When you select a dimension value, the text is created below the dimension value with an offset equal to "Value > Vertical Offset" + 1 mm. If the "Value > Vertical Offset" is superior to 0; by default the offset is 1 mm.

    • When you select one of the extension lines, the text is created vertically above the extension line, with the Anchor Point positioned at the Top Center and the Anchor Mode at Global , by default.
      However, note that the text is positioned according to the length of the extension line during the text creation. It may not remain at the middle of the extension line if its length changes.


      In all the above cases, a rigid associative positional link is created which prevents unintended move of the text.

      For more information about annotations, refer to Before You Begin.
