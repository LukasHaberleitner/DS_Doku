Glossary

D
DXF AutoCAD File Exchange format. DXF files usually correspond to ASCII format. Binary DXF files are also supported at import.
DWG AutoCAD File format (binary format ). Both DXF and DWG Files have the same content but under a different format.

I
IGES The Initial Graphic Exchange Specification (IGES) format, is a neutral format to transfer data between heterogeneous CAD systems.

S
STEP STEP AP203 / AP214 formats (Standard for the Exchange of Product model data)
