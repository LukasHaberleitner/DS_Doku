Setting a Technological Package as Current

 

You can change the current Technological Package in order that the following DMU Reviews and applicative data be associated to the Technological Package of your choice.
 

  1. Right-click the Technological Package you wish to set as current, select the Technological Package object and Set as current from the contextual menu. 
    The designated Technological Package will now be current, as indicated by the underlining of the Technological Package name in the specification tree.
