Creating a Document from an Existing One Using New From...

In the CATIA session you can create a new document based on an existing SmarTeam database document. To do so, you need to use New From.... This task shows you how to do so.

 

  1. Select SmarTeam > New From...
    The Open dialog box is displayed, showing you your leading project.

    If the document to be used as a template is located in another project, select the All Projects button on top of the window, and navigate to the project of interest. Another way of accessing the document is to perform a search operation. For more about searches, see the SmarTeam documentation.

  2. Select the requested document.

  3. Click OK to confirm the creation operation.
    A new document is created from the document you selected in SmarTeam. In our example, the original document was a CATProduct document.
