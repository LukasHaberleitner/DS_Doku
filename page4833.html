Creating Multiple Holes

You can create several holes at once and on multiple supports.
Create a wall and draw a sketch, points on a surface, or points on a plane.
 

Create Multiple Holes
You can create several holes at once.

  1. Click MultiHole .

  2. From the list, select the hole type.

  3. Select the limit type and define a limit and offset when required.

  4. Define the following parameters:

    • Main diameter
    • Main depth
    • Bottom type
    • Angle

  5. Define the direction of the hole.
    • By default, the instances are created perpendicular to the support. You can define a direction but it is common to all the instances. Therefore, it might be not compliant with some instances.
    • To reverse the direction, click Reverse.

  6. Click OK. 
     

    To create a hole on an overlapping element or a bend with a null radius, either choose the top skin of the element, or unfold the 3D shape.

   
 

Create Multiple Holes without Deformation
You can create several holes without any deformation.
Create a wall with a cylindrical bend or a flange .

  1. Click MultiHole .

  2. To position the hole, select the surface.
    • The hole points must be associative with the selected surface.
    • The hole direction must be normal to selected surface.
    • The holes must not cut separately more than one face.

  3. Optional: In the Deformation tab, select the No Deformation check box.

    When the No Deformation check box is cleared:

    • You can create the hole only on a planar or cylindrical, and single support surface.
    • You cannot create a hole on a double curvature hopper.
