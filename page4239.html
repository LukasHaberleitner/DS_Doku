Loading Parts

This procedure describes how to load or insert a part onto a blank PPR tree.
To follow along with the procedure, use the following sample parts and resources:
  • SAMPLE_PART.CATProduct
  • FIXTURE.CATProduct
Products and resources appear in product structure documents.  The files containing resources have a .CATProduct extension.  Users may create resource parts (using Resource Builder) or load V5-created resources from the catalog browser.

  1. Select the SAMPLE_PART.CATProduct link above.

    The product opens in a Resource Builder window.

  2. Select Window > Cascade, so that both the Resource Builder window (.CATProduct) and an inspection process window are visible.

  3. In the product window, select the Sample Part node, and drag it into the process window.  Drop it under the Product List node in the process window.

    SAMPLE PART.CATProduct appears in the PPR tree beneath the product list and a 3D graphical representation of the part appears on the screen.  

  4. Repeat Steps 1-3 with the following substitutions:

    • Select FIXTURE.CATProduct.
    • Drag the FIXTURE.CATProduct into the Resource List node of the process window.
    The fixture FIXTURE will appear in the PPR tree under the Resource List node.
The part and fixture are in the right position as the fixture has been designed in the context of the part (that is, they are in the same reference axis).

Once the SAMPLE PART and FIXTURE are loaded, you are ready to load and position the CMM and probe from the catalog browser and load and position a tool assembly.
