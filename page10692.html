Wireframe and Surface Elements

This task shows you how the V4 Entity is copied As Result into CATIA V5.
Open the document WIREFRAME.model in CATIA V5.
It contains a number of Wireframe elements, such as points, lines, curves, circles, and so forth, as well as Surface elements (net surface, for example).

  1. Visualize the Wireframe and Surface elements. The V4 Model is in the No SHOW space:

  2. Apply a Check As Spec on the .model. The Check Validity dialog box appears:

  3. Click OK in the Check Validity dialog box.

  4. Select the *SET entity and choose the Copy contextual menu.

  5. Click the New icon and select Part from the New dialog box, to open a .CATPart document.

  6. Click OK.

  7. Select the Part1 from the specification tree and choose the Paste Special contextual menu.

  8. The Paste Special dialog box is displayed. Choose the paste AS_RESULT option and click OK.

All the Wireframe and Surface elements are copied into the .CATPart document.

  1. Update the document, either using the Update icon, or the Edit > Update menu. The copied elements cannot be edited, however they can now be used as input for CATIA V5 elements.
