An Overview of Nonlinear Structural Analysis and Thermal Analysis

Nonlinear Structural Analysis and Thermal Analysis are modeling and graphical user interface extensions to CATIA V5 that provide access to advanced simulation capabilities within the CATIA V5 environment. They each add a workbench to the CATIA V5 environment: the Nonlinear Structural Analysis workbench and the Thermal Analysis workbench, respectively. In these workbenches you can reuse familiar CATIA features, such as contact connections, and enhance them with general analysis capabilities, such as highly accurate interface friction. In Nonlinear Structural Analysis and Thermal Analysis you create a .CATAnalysis document from which nonlinear simulations can be performed. Abaqus 2018 technology embedded in the Nonlinear Structural Analysis and Thermal Analysis products performs the simulations and generates an output database. Finally, you can view the results in the Nonlinear Structural Analysis and Thermal Analysis workbenches.

Nonlinear Structural Analysis and Thermal Analysis make extensive use of the CAD-CAE associativity concept. Associativity means that any part modifications you make outside the Nonlinear Structural Analysis and Thermal Analysis workbenches are reflected automatically when you perform tasks within the Nonlinear Structural Analysis and Thermal Analysis workbenches. In particular, any parametric changes to the parts are reflected, so there is no need to update the part specifications within the workbenches.
