Converting a Wireframe to 3D Spline

This task shows you how to convert a wireframe to a 3D Spline feature.

  1. Click Convert to 3D Spline in the Selection Tools  toolbar.

  2. Select a wireframe or create one from the contextual menu.
    For example, create the boundary of the surface.
    A 3D Spline feature is created, as close as possible to the input.

 

  1. If relevant, edit the minimum distance between the points the spline is going through.

A Spline feature is created in the specification tree. It contains the spline points.
