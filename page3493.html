Reviewing Area, Length and Center of Gravity

This task shows you how to review area, length and center of gravity status.
Open a document with a stacking.
 

  1. In the Columns Display dialog box, select the corresponding columns (Area, Length or Center of Gravity).
    • The attributes are computed on the surface, set on all entity levels and displayed in the current units and decimals preferences.
    • If the Composites part contains shells not migrated to Automatic Ply Shells, the attributes are set to NA.
    • If the Composites part contains updated shells, the attibutes values are displayed.
    • If the Composites part contains automatic shells that have not been updated, the attributes are computed as you start Stacking Management.
      An information is displayed.
    • In selection and review mode, they are not computed if they have not been set before in standard mode.
    • Cores attributes (except Length) are always computed inside Stacking Management (not streamed).
    • They are supported by Export Current Table.
