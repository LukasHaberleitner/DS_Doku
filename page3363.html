Detecting Clashes 

Detect Interferences: Click the Clash icon, define the type in the Check Clash dialog box, then select the product and click Apply.
Read Clash Command Results: Run a check for interferences and read the global results in the Check Clash dialog box and Preview window.
View Results in a Dedicated Window: Run a check for interferences then click the Results window icon.
Export Clash Results: To save results in text and XML format, click the Export As icon.
