Creating Addenda Surfaces

This task shows you how to create addenda surfaces, as top or wall surfaces, to extend the stamped part.
Top surfaces extend the stamped part, and are continuous either directly with the part or with the clean contour surfaces.
They are referred to as Reference Surface.
Open AddendumSurface01.CATPart from the samples directory.

  1. Click Addenda Surface in the Addendum Design toolbar.

  2. Select an axis system (stamping direction) and the surface to extend.

    When you select the part surface, its boundary is analyzed, and tangency discontinuities are displayed.

  3. Select or create the first profile to create the top surface.
    Use the contextual toolbar on the selection to create or edit profiles.
    See Creating Profiles for more information.

  4. Select or create the second profile to create the top surface.
    • It must be of the same type as the first one (Select two profiles or two vertices).
    • You cannot select an element already selected as first profile or vertex and vice-versa.
    • If you have selected two vertices, you can create or edit top surfaces, with or without fillets, from the contextual toolbar.
    • If you have selected profiles, you can create or edit profiles or addenda surfaces from the contextual toolbar.
    • Profiles can be rotated along the Y-direction, in the UW plane.
    • The Z-direction of a profile can be aligned with the Z-direction of the input axis system.

  5. Create two more profiles.

  6. Select the first two profiles, and select Create an addenda surface from the contextual toolbar.

  7. Select the other two profiles.
    Create an addenda surface.
    From its contextual toolbar, select Top Surface With One Fillet.

  8. Select the first addendum surface, and select Top Surface with Two Fillets.

An Addenda Surface Set is created.

Profiles are created on the selected section, one on each tangent discontinuity.
A multi-section surface is then created between the profiles.
Whenever possible, one surface is created between two profiles.
Il not possibe, several surfaces are created.
