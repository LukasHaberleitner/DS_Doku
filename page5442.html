Valuating Cells

This task shows you how you can add or remove the orientation value of a cell.
Either work on the part where you have defined cells or open the VirtualStackingGA01.CATPart document.

  1. Click Cell valuation in the Tools Palette and select a cell with no orientation value in the 2D viewer,
    or right-click a cell in the 2D viewer and select Cell Valuation from the contextual menu,
    or double-click a cell in the 2D viewer.
    Multi-selection is available if you start Cell valuation from its icon, not if you start it from the contextual menu or by double-click.
    The orientation value of the cell is set to that of the row on which you have selected the cell.

  2. Repeat that step, but with a cell with an orientation.
    The orientation of the cell is removed.
