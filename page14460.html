Querying/Updating Images

This task explains how to query/update an image of a component placed in another document, or query/update a line route image linked to on/off-sheet connectors.
When you place a component image in one or more schematic diagrams and modify the master as explained in Placing Images in Multiple Diagrams, you can update the image with changes made to the master.

You cannot make any changes to the image attributes. However, changes to the master can be displayed in the image.

You can query/update the image when any of the following types of changes are made to the master:

  • Renaming using the Rename dialog box.
  • Modifying certain attributes/properties using the Properties dialog box.
  • Replacing a component using the Replace Component command, or replacing a cable with the Replace Cable Reference command.
1. To query/update an image, open the document containing the master, and the document containing the image.

Make the document containing the image the active document.

Click the Query/Update Images button . The Query/Update Images dialog box displays.
You must open the document containing the master before querying. If not, the Update Status column indicates Master Document Not Found.

If the name of the master is changed, Master Object Not Found displays in the Update Status column.
2.

You can update the status by selecting the image from the list in the Name column, and clicking Update Selection.

The Possible Masters dialog box displays.

Select the renamed master name from the list, and click OK.

The Query/Update Images dialog box redisplays. The Name column displays the name change, and the Update Status column indicates Up to Date.

Click Close and the image is renamed with the name of the master.
When you select a line route image linked to on/off-sheet connectors from the Name column, the line image in your document is highlighted.
3. If the change to the master is made through:
  • the Properties dialog box (certain properties of the master were modified), the Update Status column in the Updated Images dialog box indicates Out of Date.
  • the Replace Component command (the master component was replaced) or the Replace Cable Reference command (the master cable was replaced), the Update Status column in the Query/Update Images dialog box indicates Incompatible Master Object.

Select the image to be updated from those listed in the Name column, and click Update Selection.

The Update Status column now indicates Up to Date.

Click Close and the image is updated with the changes made to the master.
