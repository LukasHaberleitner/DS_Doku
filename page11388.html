Inserting Head Changes

This task shows how to add head changes in the program.
A machine with at least one head must be assigned to the Part Operation. See Machine Editor for more information.

Either the program or a program entity must be current in the specification tree.
1. Select Head Change in the Auxiliary Operations toolbar.

The corresponding dialog box is displayed.
2. Select the Properties tab to specify the characteristics of the head change.

The Interchangeable Head List proposes all the available heads on the machine that is assigned to the Part Operation.
  3. Select the Syntax tab .

  • Select the Initialize from PP words table check box to consult the head change syntax defined in the PP table that is referenced by the Part Operation.
  • Otherwise, enter a PP instruction for your head change. This user-defined syntax has no link with the PP table and its validity is not checked by the program.
4. Click OK to create the head change in the program.
