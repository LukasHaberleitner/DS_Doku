Getting Started

This tutorial will guide you step-by-step through your first DMU Navigator session, allowing you to get acquainted with the product.
You will need a DMU Navigator V5 session and should be familiar with basic concepts such as document windows, standard and view toolbars. See the Infrastructure User's Guide.
You should also be familiar with the basic concepts of Product Structure. See the Product Structure User's Guide.
You should be able to complete this tutorial in about 15 minutes.
At any time, you can reference the Workbench Description for more information concerning icon toolbars and menu bars.
