Managing Annotation Connections

  Manage annotation connection offers to create, delete, modify or rename geometrical elements or user surfaces of an existing annotation: 
Use the Scope: select the Geometry Connection Management command and select an annotation.
  Reconnect the Broken Link: select the Geometry Connection Management command and select a broken link.
Add Geometry: select the Geometry Connection Management command and select an annotation.
Add Component: select the Geometry Connection Management command and select an annotation.
