Creating the Side Walls

This task shows you how to add other walls to the sheet metal part.

  1. Click the Wall on Edge icon .
    The Wall On Edge Definition dialog box opens.

  2. Select the left edge.

  3. Enter 50mm in the Value field.
    The application previews the wall.

By default, the material side is such that it ensures a continuity with the reference profile. If needed, invert it using the Reverse side button, or clicking the arrow.

  1. Click the Reverse Position button to Invert the sketch profile.

  2. Click OK.
    The wall is created and the Wall On Edge.1 feature is displayed in the specification tree:

  1. Select the right edge.

  2. Click the Wall on Edge icon  again.
    The Wall On Edge Definition dialog box opens with the parameters previously selected.

  3.  Invert the sketch profile and click OK to validate.

  1. Click the Wall on Edge icon  again.

  2. Select the front edge.
    The Wall Definition dialog box opens with the parameters previously selected.

  3. Enter 30mm in the Value field and 10mm in the Limit1 and Limit2 fields, then invert the sketch profile.

  4. Press OK to validate.
