Uploading Multiple Programs Simultaneously

This procedure describes how to upload multiple programs simultaneously.
Open the Stand_Alone_Task.CATProcess file.

  1. Select Tools > Options > Resource Detailing > Offline Programming tab, and set Fanuc to be the Selected Uploader and Downloader.

  2. Click Create Robot Program  in the Offline Programming toolbar.

  3. Download all the robot tasks for the robot R2000iA-210F.1.

  4. Save the downloaded programs.

  5. Click Import Robot Program  in the Offline Programming toolbar.

  6. Select the robot task of R2000iA-210F.1 in the PPR tree.
    A File Selection dialog box appears.

  7. In the File Selection dialog box, upload all the robot tasks simultaneously.

  8. Select Fanuc.jar as upload translator in the Select Translator dialog box. The Uploading Method dialog box opens.

  9. Select Robot Base Coordinates and click OK.

    An uploading progress bar displays the status of upload.

    Finally the Upload Information dialog box opens. This has information about the status of upload, unparsed statements, logical errors etc.

  10. Note that all the robot tasks are imported to V5 robot tasks.
