Defining a Contact  

This task explains how to add electrical behavior to a standard part to turn it into a contact.
A contact is an electrical component used within a termination or cavity, or between bundle segments.
You can add electrical behavior to an instance or a reference.
Refer to Defining an Equipment.
The BasicTasks.CATProduct document is still open.

  1. Click Define Contact .
    You are prompted to select a part or a product.

  2. Select the part or product which is to become a contact either in the specification tree or in the geometry area.

    The Contact Definition dialog box opens:

  3. Enter ContactElec in the Part Number field.

  4. Click OK to validate.

    The specification tree is updated.

    The selected object is now an electrical contact as the electrical behavior has been added.
    Note that a termination is automatically created when defining the contact.
A contact allows bundle connection points, one cavity connection point and one termination.
