PERT Chart

This page describes the options that determine the appearance of the PERT chart viewer and the other viewers based on the PERT chart (e.g., the AST editor).

The options include:

  • Color
  • Type of Link
  • Inside a PERT Node...
  • Grid
  • Open PERT in ...
  • Description

Color
Select colors for Viewer Background, Node Background, and Links and Labels.
The default colors are shown above.

Type of Link
Select the desired appearance of links:
  • Straight (shown with the Grid set to View Grid)
  • Orthogonal (shown with the Grid set to Hide Grid)

     
The default is Straight.

Inside a PERT Node...
When you open PERT chart, you can choose to see the activities represented as:
  • only an icon
  • both an icon and a node label. The full name of the activity is displayed in the PERT chart.
     
  • Both an icon and a node label with names that are not truncated.
The default is View Icon and Node Label.

Grid
When you open PERT chart, you can choose to view or hide the grid. The Type of Link section illustrates the two options.
The default is View Grid.

Open PERT in...
When you open PERT chart, you can choose to open it in a single window or to open each level in the PERT chart in its own window.
The default is Multiple Windows.

Description

To distinguish between activities in the PERT chart, activities that have a user-modified description in the activity properties are marked in the PERT chart. If you select this option, it applies to not only the PERT chart but to similar 2D views to which you have licenses, including the:

  • PERT chart
  • Precedent Viewer
  • Product Flow Viewer

To mark activities with a description, select the Mark Activities PERTNode if description exists check box. The mark appears as a red triangle in the top right corner of the activity.

The description itself appears as a tool tip when you mouseover the activity, regardless of whether the check box is selected.
The default is that the check box is not marked; i.e., PERTNodes are not marked as having descriptions by default.
