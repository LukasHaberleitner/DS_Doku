Inserting Images and Editing Background

Click Insert > Picture in the menubar to insert an image, for example a company logo, into a title block. For more information see the Interactive Drafting User's Guide. The task is Basic Tasks > Images > Inserting Images.    

Click Edit > Sheet Background in the menubar to edit the background view. Diagrams commands are not available when you are in this view. To return to the working view click Edit > Working Views in the menubar.  For more information see the Interactive Drafting User's Guide. The task is Basic Tasks > Sheets > Creating a Frame and Title Block.
