Initializing and Saving ENOVIA Processes

This procedure has two parts:

  • How to initialize the connection between V5 and your ENOVIA system 

  • How to save a process document to ENOVIA.

In order to save a file to ENOVIA, it must have first been opened in ENOVIA.

Initializing the ENOVIA/V5 connection

1.

From the menu bar, select File>Init ENOVIA Connection. The plug-in and the adapter will be activated.

This procedure need only be performed once.

Saving a process document in ENOVIA

1.

From the menu bar, select File>Save in ENOVIA.
