Creating a Vectors Field from 2 Cloud of Points Files

  This task shows you how to create a vectors field from two cloud of points files, either in txt or STL ASCII formats.

 

  1. Click Vectors Field from 2 Cloud of Points Files   in the Realistic Shape Optimizer toolbar.

  2. Select a source file.

  3. Select a target file.

  4. Go to the Filter tab.

  5. Select the Keyword check box and enter a value.
    The files with the Keyword are imported.
    ASCI STL files having the standard keyword vertex are read by default, even if the check box is cleared.

  6. Click Apply to preview the result could of vectors...
 

Both files must contain the same number of points.
The points are specified by three coordinates, either in .txt files.

A Vectors Field from 2 cloud of point files feature is created.
It is not editabe as a feature but can be modified as a cloud of points in Digitized Shape Editor.
It can be used as input in other Realistic Shape Optimizer commands such as Vectors Field Transform, Vectors Field Filter, 3D Deformation Law or Digitized Morphing.
