Customizing

Before you start your first working session, you can customize the way you work to suit your habits.
This type of customization is stored in permanent setting files: these settings will not be lost if you end your session.

  1. Select the Tools  > Options command.
    The Options dialog box displays.

  2. Choose the Ergonomics Design and Analysis category in the left-hand box.

  3. Select the Human Activity Analysis workbench.
    The Ergonomic Analysis tab displays.


     

  4. This tab lets you define the parameters for the RULA (Rapid Upper Limb Assessment) Analysis.

  5. Set the options in this tab according to your needs.

  6. Select OK when done.

  7. For the customization of Reports see Human Builder workbench
