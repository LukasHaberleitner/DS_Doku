Using This Guide

This book describes how to use the Equipment Arrangement product. Before you read it, you should be familiar with basic concepts such as document windows, standard tool bars, and view tool bars.

To get the most out of this guide, you should start with the tutorial in the Getting Started section.

The remaining sections of the book describe in detail the procedures for using all of the features of the Equipment Arrangement product. The procedures are divided into basic, advanced, and customization sections.
