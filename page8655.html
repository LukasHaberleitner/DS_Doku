Starting ENOVIA V5 VPM from CATIA V4

/enovia command in CATIA V4 enables to start ENOVIA V5 VPM.

  1. In CATIA V4, key in /enovia in the SEL FUNCTION field.
The following window is displayed only if ENOVIA application is not already running. Otherwise, see How to perform the interoperability with a running ENOVIA application.
 

  1. Click Start ENOVIA to launch ENOVIA V5 VPM.
