Modifying Seams

This task describes the modifcation of seams outside of our command.
1.

Nominal Case

The user launches Seams command. Then he removes or adds cutting element(s) and automatically the system
handles the deletion or creation of split objects.
 

 

 2.

Modifications outside of Seams command


In this example, the user edits the sketch used for creating seams in the Transverse Blk and removes one line of the sketch.

This removal implies that the number of split Plates will change. As you are not into the specific Seam command, the error panel telling you to launch Seams command is displayed.

Finally, the user will have to go to the Seams command  to reconciliate the result.
