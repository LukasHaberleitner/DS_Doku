Copying Graphical Properties

This task shows you how to copy the graphical properties of a text to other existing texts.
Note that the operating mode described here applies to datum, datum targets and geometrical tolerances too.
Open the Annotations_Part_04.CATPart document.

  1. Select the text which graphical properties which are to be modified.

  2. Click Copy Object Format  in the Style toolbar.

  1. Select the text to be used as the graphic reference for the texts you selected.

    The graphical properties assigned to the text used as the reference are now copied onto the selected text to be modified.
    See Text Graphical Properties.
The leader color and anchor point are take into account.
 

For semantic tolerances, the graphical properties are copied without tolerance values.
