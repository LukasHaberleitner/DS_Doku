What's New?

Enhanced Functionality

Adding Elements to a Selection
You can use the Add Inclusive Mode option in the tools palette to add elements to already selected elements.
You can add elements, without removing any element from the current selection.
For more information, see Selection of Elements.
Sweep Drawing
In the Revolve  or Extrude  command, you can draw a curve to generate the surface.
You can draw a profile directly and change the number of arcs of the profile.
For more information, see Creating a Surface Using Revolve, Creating a Surface Using Extrude.
Aligning on a Smooth Curve
In the Alignment command with On Body selected, you can use the Smooth Selection option to perform an alignment on a smooth curve created from selected vertices.
You can generate the support automatically by smoothing the selected vertices.
For more information, see Aligning Vertices.
Merging Vertices Automatically
You can merge close vertices automatically. During the manipulation, a merge icon appears when a merge is possible. The icon appears if you have selected the Automatic Merge option in the tools palette.
For more information, see Modifying Surfaces.
Defining a Plane in the Mesh Tools Command    
In the Mesh Tools command, you can use the Cut Mesh by Plane option to define the plane by pressing Alt and selecting two points outside the surface.
For more information, see Modifying the Mesh.
Creating Parallel and Symmetric Cuts
In the Face Cutting command, you can use the Parallel cut option to add a cut which is parallel to the reference edge. You can also use the Symmetric cut option to add a cut to the side next to the cutout side.
Benefits: The two options are available for all the cut modes.
For more information, see Cutting Faces.
