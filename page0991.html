Product Function Optimization  

  1. Select Tools > Options. The Options dialog box is displayed.

  2. Expand the Knowledgeware option and click the Product Function Optimization  tab. The following tab is displayed:

    The Invention Machine tools tab is displayed containing the following set of options:

    • Problem presentation in tree
    • Concept presentation in tree
    • Knowledge Base Access
    • File Locations
    • Additional

Problem presentation in tree

The Problem presentation in tree section relates to the listing of the problems in the Problem Manager.

 

All Problems
Choosing All problems causes the problems detected by the FunctionalSystem and the user defined ones to appear in the list.

User problems only
 If you do not want to see the ones detected by the FunctionalSystem, select User problems only.
By default, this option is set to All problems.

Concept presentation in tree

All concepts
Enables you to display all concepts in the tree.

By default, this option is set to All concepts.

User concepts only
Enables you to display user concepts only in the tree.

Knowledge Base Access

The Knowledge Base Access section determines how you access the information in the Knowledge and Innovation Server (KnIS).

Start Web Browser in CATIA
Opens the default Web Browser and displays the effects/examples from KnIS in it.

By default, this option is unchecked.

Start ENOVIA portal
Opens the effects/examples in the ENOVIA portal.

By default, this option is checked.

Absolute addressing
Enables you to use the absolute addressing.

By default, this option is checked.

Relative addressing
Enables you to use the relative addressing.

By default, this option is unchecked.

File Locations

IM-Prediction Database
Enables the user to indicate the path of the IM-Predication database.

By default, this field is empty.

IM-Principles Database
Enables the user to indicate the path of the IM-Principles Database.

By default, this field is empty.

Additional

Show warnings
By default, this option is checked.

  1. Change these options according to your needs.

  2. Click OK when done to validate your settings.
