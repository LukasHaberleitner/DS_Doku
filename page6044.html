Deleting a Sheet

This task will show you how to delete a sheet. When a CATDrawing document is opened, one sheet is necessarily displayed.
For the purpose of this scenario create more than one sheet.

  1. Select the sheet from the specification tree. For example, Sheet 2.
     

  2. Right-click the selected sheet and display the contextual menu.

  3. Select the Delete option from the contextual menu. A dialog box is displayed.

  4. Click OK to confirm the deletion.

    Sheet 2 is deleted.
Deleting a sheet is irreversible.
