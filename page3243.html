Specialized Tesselation

This command allows to create a temporary tesselation for an improved shaded display.
Note: Specialized Tesselation is supported by the Parallel Command Session command.
 

  1. Click Specialized Tesselation .
    The Specialized Tesselation dialog box opens.
  You can define the following option:
  • Fixed 3D accuracy: Specifies an arbitrary tesselation tolerance from 0,001 mm (micron tesselation) to 1.0 mm (max.).
    Tesselation tolerance 0.001 mm
    The shaded display shows a
    smooth course of the boundaries.    		 Tesselation tolerance 1.0 mm
    The shaded display of analyses shows a strong
    aliasing at the boundaries of analysis stripes.
