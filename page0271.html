Understanding CATIA V5

Abaqus for CATIA V5 users should be familiar with basic CATIA V5 concepts such as document windows, specification trees, etc. In addition, it is expected that you are familiar with the Part Design, Assembly Design, and Advanced Meshing Tools workbenches in CATIA V5. Prior to using this manual, we recommend that you review the following guides in the CATIA V5 documentation collection:

  • Infrastructure User's Guide

  • Part Design User's Guide

  • Assembly Design User's Guide

  • CATIA V5 Advanced Meshing Tools User's Guide

  • Generative Structural Analysis User's Guide (many of the tools in the Abaqus for CATIA V5 workbenches are documented in the Generative Structural Analysis User's Guide)
