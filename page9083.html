ABQRigidBodyConstraint (Object)

Represents an Abaqus rigid body constraint (ABQRigidBodyConstraint) object.
Property Index
MoveToCenterOfMass Sets or returns the move to center of mass flag.
NumPinSupports Gets the number of pin point supports for the rigid body.
NumTieSupports Gets the number of tie point supports for the rigid body.
Method Index
AddSupportFromBody Creates a new support for the body and adds it to the description of the Analysis Entity.
