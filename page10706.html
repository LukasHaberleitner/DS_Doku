Color Tables

This task shows you how the V4 Color Table entity is copied As Result and As Spec into CATIA V5.
Open the document COLORED_RSUR.model in CATIA V5.

  1. Copy / Paste AS RESULT the model into a CATIA V5 CATPart...

Then, update the new CATIA V5 document:

The solid elements seem to have lost their specific colors. Let's check this by following the steps 2 and 3 detailed hereafter.

  1. Select any solid element as shown below:

  1. Select Properties from the contextual menu then click the Graphic tab. As the Properties dialog box opens, the whole .model document is highlighted and the Fill Color displayed corresponds to the color of the .model viewed as a whole and not to the selected element which was initially pink:

The Copy / Paste AS RESULT copies the overall color and not the color (whether standard or user-defined) applied individually to each element.

  1. Copy / Paste AS SPEC the .model document into a CATIA V5 CATPart...

...then select any colored element, the pink sphere for instance:

  1. Select Properties from the contextual menu then click the graphic tab:

The Fill Color displayed matches the one of the selected element since the Copy / Paste AS RESULT lets you copy the color applied individually to each V4 model element.
