Creating an Forearm/Hand Model

This task describes how to create a forearm model with fully-articulated hand.

  1. From the Insert menu, select New Manikin.

     

    OR

    Select Manikin Creation in the Manikin Tools toolbar.

  2. The New manikin dialog box appears.


  3. Select the Father product and the Manikin name.  Accept the defaults in the Gender, Population, and Percentile fields.

    For Model: Select Right Forearm or Left Forearm.


  4. Select the OK button.  The selected forearm/hand model appears.

Once the forearm/hand model is created, position it with the compass in the same manner that you position a manikin. It is also possible to:

  • use the Posture Editor on the forearm

  • use the Forward Kinematics command on the hand, fingers, and thumb

  • paste local postures from a catalog

Some functions may not be used with the forearm/hand model. These are: the Vision function , the Inverse Kinematics functions  , the Reach Mode    the Constraint function, and all commands in the Human Activities workbench.
