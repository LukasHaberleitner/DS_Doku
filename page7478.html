Selecting the Geometries for Annotation Creation

This topic explains how you can easily select the geometries for annotation creation using tolerancing advisor.
The new propagation option All the elements of closed contour is added in the tolerancing advisor. This propagation option is applicable for surface edges and solid edges. It has different behavior and conditions for border edges and common (non-border) edges.

1. Surface border edge:
If you select the border edge (edge having only one adjacent face), “All the elements of closed contour” propagation option will be available and enabled in the tolerancing advisor. If you select this option, all the edges of the closed contour will be selected provided that all the edges forming a closed contour must be a border edge. The total number of elements of that closed contour should not exceed 20.

2. Common (non-border) edges:
If you select the common edge (edge having more than one adjacent faces), “All the elements of closed contour” propagation option will be available and enabled in the tolerancing advisor. If the user selects this option, all the edges forming a closed contour will be selected provided that all the edges forming a closed contour must have a tangent continuity.
 
For the option “All the in between edges with tangent continuity” the behavior is enhanced in below two ways:

1. Propagation of selection even with tangent discontinuity:
In existing behavior, if the user selects two edges to decide a range and clicks on this option then the in-between edges will be propagated only if there is tangent continuity between all the elements otherwise the propagation will not be done.
Now this function provides the capability that even though there is tangent discontinuity present within the given range, the propagation will be done until the tangent discontinuity exist. Those propagated edges will remain selected.
Here user can select the next tangent discontinued edge and relaunch the “All the in between edges with tangent continuity” option to proceed further.

2. Deciding the direction of propagation:
When the user selects the two edges to decide the range of propagation, there are two possible directions for propagation as depicted below.As of now, there is no control for the user to decide the direction of propagation and the user has to accept the default direction which could be either of two. This enhancement gives the control to the user to decide the direction of propagation.
While deciding the range itself, the user has to do one more selection. In other words, the user has to do the selection in sequence as depicted above. The first selection and third selection decided the range of propagation while the second selection decides the direction of propagation. In this case, the propagation will be done in “Direction 2”. Here the first and second selections has to be adjacent edges.

Note: If border edge is selected and “All the elements of closed contour” propagation option is clicked then the propagation will be done only if the total number of elements confining the close contour should not exceed the 20.

New propagation option  “All the elements of closed contour” will be displayed in propagation of selection frame when common edge or border edge is selected.
