Saving Graph Session

About Saving Graph Session

Save Graph Session is used to save the layout of a graph in Relationship Navigator. Opening a saved session saves time by eliminating the need to expand a project node from the parent object every time you start your session. When saving your session, you are able to begin working in the node tree where you previously ended.

When Saving Graph Session

  • When you make changes in a previously saved graph session, you should save your work under a new name to avoid having duplicate session names for your saved work.
  • If nodes in the graph have been removed, or added after you have saved your graph session, saved session graph will display the latest changes next time you open it.

How to Save Graph Session

You can do the following for saving a graph session:

  1. Open an object in Relationship Navigator and expand various relationships.

  2. Once you have completed your work and are ready to save the graph, click theSave Graph Session button at the bottom of the Relation Navigator window. The Load Session window is displayed.

  3. Enter a name for your graph in the Please enter a name for the session field and click the OK button. The Graph session is saved in the My Sessions folder by default. If you do not already have a My Sessions folder, Save Graph Session will create one for you.

  4. To reopen your saved session, select Add to Navigator from the shortcut menu in the Favorites window.
