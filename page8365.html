Determining the Current Nationality of an Manikin

There are two ways to determine the current nationality of a manikin:
 

  • In the Human Builder workbench, through the Properties dialog box

  • In the Human Measurements Editor workbench, through the Population command

Human Builder workbench > Properties dialog box

  1. Access the manikin Properties dialog box. To do this, select the manikin, then select Edit -> Properties from the main menu bar. You can also right-click on the manikin in the PPR tree and select Properties from the contextual menu.
     
  2. In the Properties dialog box, select the Anthropometry tab.


     
  3. The current nationality of the manikin is displayed on the first line in the Population: field.

Human Measurements Editor workbench  > Population command

  1. Select Population from the Anthropometry Editor toolbar . 
  2. The Population dialog box displays the available populations; the manikin's current nationality is highlighted on the list.
