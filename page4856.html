Adding Hyperlinks to the Simulation

This procedure describes how to create, or change a hyperlink activity within a process.

 

 This procedure describes how to add a hyperlink activity to the process; to add hyperlink activities to the products or resources, see the Process and Resource Definition User's Guide : Advanced Tasks: Editing Product Documents within the Process: Adding Hyperlinks to an Activity or Going to a Hyperlink.

Depending on how you have your simulation options set, remember that you may not see hyperlinks during a simulation or process verification.  When you add a hyperlink activity to a simulation and the simulation options are set to Pause, the simulation automatically pauses, so that users can access the information in the hyperlink.  To restart the simulation, users select a command from the Process Simulation sub-toolbar (e.g., Play Forwards).  Similarly, when Process Verification or Auto Synchronization are in animation mode, the simulation stops so that viewers can read hyperlinked data.

If the simulation options are set to Continue, you can add a pause activity to allow users to access the hyperlink data.

  • See Tools > Options > DELMIA Infrastructure > Simulation for an explanation of how to control the default settings regarding hyperlink activities.

Create a Hyperlink Activity

  1. On the PPR tree, select the activity after which you wish to create the hyperlink activity.

  2. Click Hyperlink Activity from the Simulation Activity Creation toolbar.

    The Edit Hyperlink Operation dialog box appears.

  3. Either enter the path or use the Browse button to select the link you want.

  4. Click OK.

    The hyperlink activity appears on the PPR tree.  Its label contains the hyperlink path.

Edit a Hyperlink Activity
 

  1. On the PPR tree, double-click on the hyperlink activity you wish to edit.

    The Edit Hyperlink Operation dialog box appears.  The current hyperlink activity's URL appears in the dialog box.

  2. Either enter the path or use the Browse button to select the link you want.

  3. Click OK.

    On the PPR tree, the hyperlink activity's label changes to contain the new hyperlink path.
When you set the simulation options so that the hyperlink activity's mode is Pause, the pause occurs at the end of the hyperlink activity.  In most cases, the hyperlink activity has a duration of 0 (i.e., the start time equals the end time), so the practical effects of when the pause starts are not noticeable.  However, if the hyperlink activity occurs in parallel with other activities, the concurrent activities will begin before the hyperlink activity halts the simulation.
