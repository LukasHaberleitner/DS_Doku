Uploading or Importing Daihen Robot Programs

This procedure describes how to upload or import a Daihen robot program.
Open the DaihenUpload.CATProcess file.
Uploading reads a Daihen robot program file and creates the corresponding activities in the PPR tree. The following steps take place:
  • The Daihen robot program is read and an XML file is generated. The commands in the program are mapped to elements in the XML file. The generated XML file is independent of the robot language.
  • The XML file is read and the required activities, actions, sequences or other items are generated as specified.

  1. Click Import Robot Program  in the Offline Programming toolbar.

  2. Select robot AV06.1 in the PPR tree.
    A File Selection dialog box appears.

  3. In the File Selection dialog box, navigate to the desired program file, e.g., P003.ASC and select Daihen.jar.
    The Uploading Method dialog box opens.
     

  4. Select Robot Base Coordinates and click OK.

  5. Repeat the procedure for AV06.2 by selecting program P001.ASC. Program P002.ASC gets called from P001.ASC and is also uploaded and a separate task created.

  6. Switch to Workcell Sequencing workbench.

    Please remember to open the DaihenUpload.CATProcess file after the switch is performed.

  7. Click Multiple Resource Simulation in the Simulation toolbar.
    The Multi Resource Simulation dialog box appears.

  8. Choose the device tasks shown above and click OK.
    The Process Simulation toolbar appears.

  9. Click Play in the Process Simulation toolbar to simulate the process.
