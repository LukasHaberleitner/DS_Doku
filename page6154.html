Removing Elements

This task shows how to remove a vertex, and edge or a triangle from a mesh using the Mesh Edition toolbar.

This command is one shot on single-click and multi shots on double-click.
It does not create any new mesh.

To exit a command:

  • Use the Esc key on the keyboard,
  • Or click the icon again,
  • Or click another icon.

For an easier work, we recommend that you switch to Wireframe view mode:
Open the MeshEdition1.CATPart model from the samples directory.

Picking an element removes it at once.
 

  1. Click Remove element . A Tools Palette is displayed:

    Select the type of elements you want to remove (point, edge, triangle).

  2. Pick the element you want to remove. It is removed at once.
