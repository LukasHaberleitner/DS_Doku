Inserting a Part

This task shows you how to insert a part. 
This procedure is used for inserting a part at a location on a waveguide, for instance.

1.

 With the run displayed, click the Place Waveguide Part button . The Place Waveguide Part dialog box displays. Click the Class Browser button and navigate to the type of part you want to place.  See Placing a Part on a Run.

2. Select the part you want to place and click where you want to place it. The part will be placed.  In the image below a flange gasket has been inserted.

3. You can adjust the location of the part after it has been placed.  If not already 'on', click the Toggle Manipulation Handle Mode button and select the part you want to move, in this case the flange gasket and mating parts. Click the dot and drag the part along the run to the desired location.  Click the Update button when finished.
