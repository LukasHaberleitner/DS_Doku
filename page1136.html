Structure Design

This page deals with the following category of options in the General tab: Color and Manage Piece Parts.

Color

 Use the Shape and/or Plate color chooser to define the default color for shapes and plates respectively.

By default, colors are as shown above.

Manage Piece Parts

Execution log directory

Specify the location where the execution log directory is stored.

By default, the PPG execution logs are saved in the <CATTemp folder>\PPGExec\<Source part> directory. You can specify a different parent folder by clicking Browse icon.

Generate execution logs

Select the check box to generate the execution log at the specified location. This log is used to manage the piece parts when some changes are done in the source data and features. The execution log naming convention is: <Source part>_<yyyymmdd>_<hhmmss>.txt.

By default, this check box is cleared.
