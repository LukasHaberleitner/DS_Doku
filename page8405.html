Creating a Manikin

If you do not have an active product with a manikin, follow these steps to create a manikin.

If you are in the Human Builder Workbench, and already have an active product containing a manikin, proceed to step 6.

  1. Go to the Start menu (see below).

  2. From the Start menu, select Ergonomics Design & Analysis > Human Builder to create a new manikin.


  3. The workbench appears. Select the inserts a new manikin

  4. Under the Manikin tab, select the father product,  select the Product1 in the 3D geometry window. Select the defaults or enter a name for the manikin, set the gender, and the percentile.

  5. Under the Optional tab, set the population, model-type, referential, and choose whether or not to set the referential to a compass location.



      

  6. Select the required icon from the Manikin Workbench Access toolbar from the Human Builder workbench.

  7. Select the Opens the Human Posture Analysis workbench and the command prompt asks you for Select a segment or a line of sight..., select the required segment.

  8. The software changes to the Human Posture Analysis workbench.

  9. Or if you already have an active product containing a manikin, select the manikin or double-click a segment to change to the workbench.
