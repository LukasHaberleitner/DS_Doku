Aggregating Mold Areas

This task shows you how to aggregate surfaces of an Geometrical set to improve performances. 

  1. Still in the same CATProduct, click Aggregate Mold Area in the Pulling Direction toolbar.

  2. The dialog box is displayed.
    Select the Geometrical set where you want to aggregate surfaces in the list proposed.

    The list of the surfaces contained in this Geometrical set and that are not in the NoShow is displayed.

  3. If you want to create a join datum, select that check box. Click OK.
    One surface or one join is created for each connex domain and the initial surfaces are deleted.
