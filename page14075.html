Unfolding Surfaces

This task shows you how to unfold a surface as another surface or a trim line.
Open Unfold01.CATPart from the samples directory.
A SIMULIA installation is required to access the whole functionality.
SIMULIA installation must be R2021x FD06 or later, with QUF specific license.
   

  1. Click Unfold in the Process Part Separation  toolbar.
    If the check box Display detailed text in dialog in Tools/Options/Mechanical Design/Die Face Design is cleared, the dialog box looks like this:

    If the check box Display detailed text in dialog in Tools/Options/Mechanical Design/Die Face Design is selected, the dialog box looks like this:

  2. Select an axis system (stamping direction).
    The unfold orientation and angle are computed from the Z-Axis.
    The mean line or the folding line from axis is projected on its XY plane.
    The XY plane is used to compute a support surface, when none is selected.

  3. Select the Surface to Unfold.

  4. Select a Fixed Area.
    It is a surface or 3D curve that intersects the surface to unfold.
    The actual fixed area is specified by all the edges of the surface
    to unfold that lie entirely on the fixed area.
    These edges form the folding line.

  5. Optional: Select a Support Surface.
    The unfold is projected
    • On the support surface, if selected
    • On an extrude of the folding line in the reference XY plane otherwise.

  6. Select the type of unfold to create:
    • Unfold as Planar Surface: Unfolds the selected surface as a planar surface, possiblbly on a support.
    • Unfold as Trim Line: Unfolds the selected surface as a trim line, possibly on a support.
    • Partial Unfold: Unfolds the selected surface by keeping the inital geometry and recomputing a fillet.

  7. For a realistic trim line computation, define the material with
    • The material name.
      Click the Material icon to display all availabe materials.
      Materials are defined using the SIMULIA format, in a Material Catalog file specified under Realistic Trim Line in the Tools > Options.
    • The material thickness
    • The thickness orientation in regard with the surface. It defines where the matter is found from the surface to unfold. It can be Top Surface or Middle Surface or Bottom Surface. The direction of the matter is illustrated by an arrow.

    Click the orientation arrow to invert it.

  8. When a SIMULIA installation exists, select the Activate option under Realistic Trim Line Computation to enable it.
    Note that the realistic trim line computation does not support Middle Surface as Thickness Orientation, nor the Partial unfold mode.

  9. Under Advanced, click Open

  10. Under Geometric Unfold Options:
    • Either select the Automatic K-Factor check box or clear it and enter a K-factor value.
      By default, the Automatic K-Factor check box is selected, for an automatic computation.
    • Select the Reverse direction check box, if required.
    • Select a Plane Orientation:
      • Mean Line: The folding line is a line computed between the first and the last points of the folding line.
      • Mean Line From Axis: The folding line is the computed mean line projected on the reference XY plane.
      • Folding Line: The folding line is the intersection between the fixed area and the surface to unfold.
      • Folding Line From Axis: The folding line is the intersection between the fixed area and the surface to unfold,
        projected on the reference XY plane.
    • Faces to ignore: The selected faces are considered as a plugged holes when computing the unfold.
      Example of unfold with no faces to ignore

      unfolds as

  11. Under Realistic Unfold Options:
    • Select the Enable plastic material properties check box.
      Plastic properties mut be defined in the material pointed by the feature.
      If the simulation does not converge on the plastic properties, but did on the elastic properties, a warning is displayed.
    • Select the Enable non converged results check box.
      It is selected by default. The result is kept if the convergence of the simulation reaches the 32nd iteration.
    • Enter the Convergence tolerance to control the display of the result areas where the convergence is not achieved, based on their last displacement.
      • Areas above the Convergence tolerance  are displayed in red.
      • Areas below the Convergence tolerance  are displayed in green.
    • Select the Improve trim line fidelity check box.
      It is selected by default.
      It is available if you have selected Unfold as trim line and activated the Realistic Trim Line Computation.
      It activates the computation of a second type of result.
      Use the list to display the required type of result (No update required).
    • Select a Mesh quality.
      • Default
      • Coarse. Reduces the computation time, but provides approximate results.

  12. Under Partial Unfold/Automatic Support Surface Computation Options
    • Choose a fillet option
      • No Fillet. This option is not available when unfolding as spec.
      • Manual Fillet: Defines the fillet using local radiuses on the folding line.
        A list of default local radii computed from the current automatic options is proposed.
        Select a radius in the list or in the graphic area and modify its value in the dialog box.
        Alternatively, pick a vertex, select Add Target Radius   from the contextual toolbar and enter its value.
        Drag a local radius to move it.
      • Automatic Fillet: Computes the fillet radius automatically from the bend angle
        • with the specified maximum chord where the angle between the tangent
          of the surface to unfold and the Z-Direction is minimum
        • with the minimum chord where the angle is maximum.
          In this case, the radius for the other angles is computed by a linear interpolation.
      • Extrapolate: Creates the automatic support surface by extrapolating the fixed element.
    • Enter the other parameters values
      • Angle between the unfolded surface and the Z-Direction of the selected axis system.
      • Minimum Fillet Chord
      • Maximum Fillet Chord
      • Selected fillet radius
      • Leader Fillet Profile: The leader fillet profile is the one used to compute the rotation of the surface.
        It is the only fillet profile kept as is.
        Other target fillet profiles are replaced by splines, that reach the rotated surface.
        If you do not select a leader fillet profile, the target fillet radius closest to the mean radius is used.

  13. Click OK.
    The unfold is created under an Unfold Set.
