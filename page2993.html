Geometric Elements and Geometric Properties

This table show how V5 Drafting geometric elements and geometric properties are interpreted in CADAM.  The CADAM Drafting product and the Interactive Drafting product support the same basic set of geometry. All basic geometric elements are transferred with mathematically accuracy between the two systems. The geometric properties (e.g., line styles) are not identically defined between the two products with the current releases. Therefore, minor differences in visual appearance may occur between the two systems.

A curve (spline, ellipse) may have style or appearance differences when transferred to the target CATIA application. This is because of underlying differences in the CADAM Drafting and CATIA documents. The translator will attempt to achieve the closest possible appearance while maintaining the correct functional intent of the curve - for example, bay point, curve length.
