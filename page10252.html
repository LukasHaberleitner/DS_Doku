CatFilletEdgePropagation (Enumeration)

enum CatFilletEdgePropagation {
  catMinimalFilletEdgePropagation,
  catTangencyFilletEdgePropagation
}
Fillet edge propagation.
The fillet can apply to the selected edge only or to all contiguous edges that are tangent to the selected edge.
Values:
catMinimalFilletEdgePropagation
The fillet is applied only to the selected edge and propagated to the first natural relimitation
catTangencyFilletEdgePropagation
The fillet is propagated to all the tangent contiguous edges
