Routing Branches on Floating Point

This tasks shows you how to update a 3D point or an explicit point that links geometrically 2 branches together.
This point is called: Floating Point. Its coordinates will be recomputed using the route of the 2 selected branches.
Open FLOATING_POINT.CATProduct document.
The product contains a floating point (3D point) and two branches connected through the floating point.
The floating point allows changes in the branch route to obtain a more realistic view.

  1. Open FLOATING_POINT.CATProduct.

  2. Click Floating Point .
    You are prompted to select branch extremity that links to two or more branches.

  3. Select the floating point and the two branches connected connected through this floating point.

    The Select Mode dialog box is displayed:

    There are two ways of repositioning the floating point: using Ratio or Length.

  4. Select Ratio and click OK.
    The floating point is repositioned to generate a realistic shape of the Harness Branches.

  5. If you select Length, the length of the original curve is chosen.

    • The floating point can be linked to more than two  branches. Once the point is modified, the connected branches will also be updated.
    • Both selected branches must be geometrically linked.
    • A floating point cannot be repositioned on the basis of branches which have distant links together.                                      
    • The Floating Point command is not supported on branches built in constrained length mode and on external curves.
