Using Tag Commands

NC Machine Tool Simulation users can specify particular locations on their machines or other geometrical data by using one of four tag commands.  These tags are accessible by selecting View > Toolbars > Tags.  Tags are always assigned to a group, which is assigned to the resource list.  To create a tag, users first create a tag group, then tags within the group.  The location of tags can be altered by projecting tags or by transforming them.  Procedures describing how to create and modify tags appears below.  Although this information uses examples containing robots, the procedures apply to machining.
