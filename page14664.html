Accessing the Browser

This task lists per Structure Functional Design command the editors that call the reference plane browser.
The reference plane browser is accessible only from contextual menu.
Here is a sample on plate dialog:

 

 1. Plate Command
Contextual menu accessible from:
  • Support editor.
  • Limits editor.
2. Stiffener Command
Contextual menu accessible from:
  • Support editor.
  • Limits Start editor.
  • Limits End editor.
3. Stiffener on Free Edge command
Contextual menu accessible from:
  • Limits Start editor.
  • Limits End editor.
4. Beam command
Contextual menu accessible from, in reference planes mode:
  • Transversal editor.
  • Longitudinal editor.

Contextual menu accessible from, in Guide curve and Point to point mode:

  • Limits Start editor.
  • Limits End editor.
5. Split Plate command
Contextual menu accessible from:
  • Splitting Object editor.
6. Split Stiffener command
Contextual menu accessible from:
  • Splitting Object editor.
