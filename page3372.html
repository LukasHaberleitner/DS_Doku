Annotation Planes

  Using a View/Annotation Plane
Creating a Front View: select the Insert  >  Views  >  Front View command and a planar element.
Creating a Section View/Annotation Plane: select the Insert  >  Views  >  Section View/Annotation View command and a planar element.
Creating a Section Cut View/Annotation Plane: select the Insert  >  Views  >  Section Cut View/Annotation Plane command and a planar element.
