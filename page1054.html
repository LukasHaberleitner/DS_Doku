Libraries 

This page describes the options concerning adding and removing process libraries.

  • List of default libraries
  • Generate Process Identifier

List of default libraries

 

Add libraries

To add a process library to the list of default libraries, click the Add libraries... button to display the Default libraries file selection dialog box.

Use the file selection dialog to navigate to the location where the libraries of interest are stored. After a library has been selected, click the Open button to add the full file path to the selected library to the list of default libraries.

Remove selected libraries
To remove a library from the list of default libraries, select the library to be removed and click the Remove selected libraries... button. The selected library is removed from the list.

The default value is that the check box is not selected.

Generate Process Identifier

 
With the help of a Process ID (Process Identifier), processes can be uniquely identified. This function is optional. The assigned Process ID must be unique, and can only be assigned once in a process.

The default value is that the Ensure unique ID among siblings check box is selected.
