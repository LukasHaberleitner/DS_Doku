Part Analysis Documents

You create the analysis documents used in an analysis assembly when you add a mesh and properties to part geometry or add properties to an orphan mesh. These .CATAnalysis files may be used to analyze the design of individual parts or may be created exclusively to use in assemblies. Creating the analysis documents on the parts allows multiple designers and analysts to work independently on the parts until it is necessary to analyze the complete assembly. The goal is to reuse the analysis documents so that progress on the assembly does not require meshes and properties to be redefined.

The following information from the part analysis documents can be reused in an analysis assembly:

  • General analysis connections and other connections stored in the Properties container

  • Groups

  • Material properties

  • Meshes

  • 1D, 2D, and 3D Properties

  • Springs and dashpots

  • Virtual parts

Additional information contained in the part analysis documents will be visible in the specification tree for the assembly under the associated parts, but it cannot be used in the analysis assembly.

When you make changes to the parts used in an analysis assembly, you should use the Synchronize option in the Analysis Assembly 2D Viewer to update the assembled analysis definition with the latest changes in the part information. For more information, see Analysis Assembly 2D Viewer in the CATIA V5 Generative Assembly Structural Analysis User's Guide.
