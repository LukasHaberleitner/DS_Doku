Installing Abaqus for CATIA V5 (AFC)

The following instructions guide you through the steps to install Abaqus for CATIA V5 (AFC). The procedure for installing AFC is similar to the procedure for installing additional products after you have installed CATIA V5. For more information, see “Installing Additional Products” in the CATIA Infrastructure Installation Guide.

To install AFC:

  1. Ensure that you have Administrator privileges.

  2. Close all Windows programs.

  3. Install the Abaqus for CATIA V5 component according to the installation procedure provided with the media.

To run Abaqus for CATIA V5, you must also have a license for either Abaqus for CATIA Complete (ACC) or for the CATIA Generative Part Structural Analysis (GPS) simulation product. If you do not have the GPS license, follow the procedure described in Configuring Licensing for Abaqus for CATIA V5 Prerequisite (ACP) and Abaqus for CATIA V5 to define the license server for ACC.
