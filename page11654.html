Power Machining Parameters
Select the Dynamic pattern explicitly; in other operations (Roughing, Cavity Roughing, Power Machining, Spiral Milling, Multi Axis Spiral Milling, Lathe Rough Turning and Groove Turning) every open pocket uses the Dynamic pattern. This pattern is optimized regarding the machining time but the computation time increases hugely with small radial engagement. Now the computation time has been reduced to make it acceptable for these use cases.
