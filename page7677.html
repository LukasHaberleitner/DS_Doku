Using the Smooth Command

This task shows you how to smooth the path created by Path Finder in the previous task.

  1. Click Smooth  in the DMU Check toolbar.

    The Select dialog box appears.

  2. In the Select dialog box, select the required track, e.g., Track.4.

    The Smooth dialog box appears.

  3. Click OK.

    This is what you obtain:
    The smooth result (which is a track operator) is identified in the specification tree under Track.2.
    The parameters changed are identified, too.
You can edit the parameters double-clicking them.

The Edit Parameter dialog box appears; alter its values, and click OK.

  You can also display the track properties using the contextual menu.
