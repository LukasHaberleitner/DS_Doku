Transfer a Fabrication

This task shows you how to transfer a fabrication to another document.
If your design is stored in ENOVIA you should NOT use this option for transferring fabrications. Instead you should use the Transfer Elements command.

1.

 In the specifications tree, make active the document to which you want to transfer the fabrication, and click the Create Fabrication button. The Create Fabrications dialog box appears and displays the list of fabrications for the current product. 
2. Select the fabrication you want to transfer and then click the Transfer button. The Transfer dialog box displays.
3. Check one of the options - the first option will result in a new document being created under the active document, while the second option transfers the fabrication to the document you activated earlier.
4. Click OK. The fabrication is transferred.
