Using Doc-chooser

This task explains how to use Part Supply from the doc-chooser.

  1. Select one of the following actions:
    Open: File > Open
    Insert: Insert > Components > Existing Component...  or Right-click the part > Components > Existing Component...
    Replace: Insert > Components > Replace Component... or Right-click the part > Components > Replace Component...

    Doc-chooser dialog box is displayed.

  2. Click Part Supply to open the part supply panel.

  3. Browse through the directory and select the required component.

    Details of the component are displayed.

  4. Click OK to confirm the component selection.

    Part Supply Download Folder dialog box appears.

  5. Select a folder to extract the downloaded component and click OK.

    When you select the Do not remind me next time check box, subsequent components are saved in the same location and the Part Supply Download Folder dialog box does not appear. If you want to change the location, select Tools > Options > General > Compatibility > 3DEXPERIENCE Platform and select the required folder.

When you use Replace Component from side panel, you cannot manage instances in the Impacts on Replace dialog box. To manage the impact on instances, use Replace Existing Component.
