Display

   
 

This page deals with the following option in the Display tab:

  • Tree
  • Elements Analysis Colors
 

Tree
 

Track history of view display

Select this check box to track history of view display in the current working session. Once they are displayed, views are identified in the specification tree with a light purple background.

Note: Already-displayed views will remain highlighted with the light purple background within the current working session, even if you deactivate the setting later on.
Views that are not displayed yet are identified in the specification tree using a transparent background.
Refer to 2D Layout for 3D Design User's Guide: View Management: Displaying Views for more information.

By default, this check box is selected.

  
 

Elements Analysis Colors
 

Callouts for non-perpendicular views

Select a color to represent the callouts for non-perpendicular views.

To create callout for non-perpendicular views, see Callout Creation.

By default, the color is green.
