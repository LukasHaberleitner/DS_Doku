Creating a Technological Object

This task explains how to create a technological object using the Business Process Knowledge Template workbench and how to add sub-types to this object.

  1. From the Start>Knowledgeware menu, access the Business Process Knowledge Template workbench. The FeatureDictionary window opens.

  2. Click the Create Technological Type icon (). The Create technological type dialog box opens.
    To associate an icon with a technological type, a user must create a bmp icon, size 22*22, name I_typename in icons\normal directory. For example, for a type named "MyType1", the icon must be named I_MyType1.bmp.

  3. Enter the name of the object, Manikin in this scenario and click OK.

Manikin is displayed under the Classes View node in the Feature Dictionary tree.

  1. Click the Manikin object in the Feature Dictionary tree and click the Create technological Type icon (). The Create technological type dialog box opens.

  2. Enter the name of the sub-type, Driver in this scenario and click OK to validate.

Driver is displayed under the Manikin object in the Feature Dictionary tree.

  1. Save your file, keep the file open and proceed to the next task: Adding an Attribute to a Technological Object.

Types names must not contain spaces and non-alphanumeric characters.
