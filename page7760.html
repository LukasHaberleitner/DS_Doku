Reordering Features within a Solid Functional Set or a Functional Set

This task shows you how to reorder features within a solid functional set or a functional set.
Open a CATPart document containing a Solid Functional Set or a Functional Set including three features at least.
 
To simplify the display of the tree you can reorganize the contents of a solid functional set or a functional set. To reorder them, proceed as follows:

  1. From the specification tree, right-click the Solid Functional (or the Functional Set) of interest and select Solid Functional Set.x object > Reorder Children. 

    The Reorder Children dialog box is displayed, showing the list of features of the set in the same order as displayed in the specification tree.

  2. Select the feature you want to move within the set.

  3. Click the arrows to move it up or down.

  4. Click OK when done.

    Whatever location you chose, the geometry is not affected by the change.
