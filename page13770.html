Getting Started

The following short tutorial provides an introduction to the Version 5 Systems Routing product.  It is intended to give you a feel for the product's capabilities in a few step-by-step scenarios, which are listed below.

Entering the Systems Routing Workbench
Creating a Conveyor System
Creating a Network
Routing a Run
Placing Conveyor Parts on the Run
Set Correct Working Units and Grid
Changing the Current Axis
Saving Documents
Updating Documents

 
These tasks can be completed in about 15 minutes.
