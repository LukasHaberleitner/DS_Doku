Scenario (V4/V5 SPACE): How to use Gap Healing

This task shows you how to use Gap Healing in Tools > Options > general > Compatibility: V4 / V5 SPACE.

There are three examples below that illustrate the possibilities of these settings.
Open SURFACE_NON_C0_WITH_CURVE.model. In all these examples, the option Keep Segmentation is not activated.
