Simulate Replication Package

While creating a replication package, you have the option to simulate the creation and view a html report of the objects of the replication package. This allows you to view all the objects that will be replicated to the external site.

Follow the steps to simulate a replication package:

  1. Repeat steps 1 - 3 from in the Create Replication Package chapter. Your Replication Package Creation dialog box should look similar to the one below.

  2. Click Simulate to view the html report of the objects of the replication package.

  3. You can review the report of the objects, which will be replicated. If you want to create the package click OK. If you do not want to create the package, you can click Cancel.
