Creating a Flange Tool

This task shows you how to quickly create a Flange Tool feature with its geometries.
Open FlangeToolP1.CATPart from the samples directory.

  1. Click Flange Tool  in the Process Part Separation toolbar.


  2. Select the Flange surface, that corresponds to the final geometry resulting from the definition of the flange.
    Be aware that the automatic creation of the flange tool on complex flanges is not supported.
    Supported flanges have a clear distinction between :
    • The flange surface, shown in grey below.
    • A filleted area (in orange) between the folding line (in pink) and the flange surface.

  3. Select the Stamped Surface or Unfolded surface from which the flange was folded.

  4. Select the Tool axis.

  5. Select a Type of flange
    • Form to form the surface to create the flange.
      The flange area is used as the bottom surface for the flange tool.
    • Bending to bend the surface along a given direction, leading to a planar flange area.

    Be aware that some configurations do not allow the computation of all the geometries of the tool.

    • A warning indicates the elements that are not correctly computed.
    • Correctly computed elements (punch, punch wall, 2D punch and 3D punch) are provided.

  6. If you have selected Form, enter the following Flange tool parameters values:
    • Wall height
    • Flange extrapolation.

  7. If you have selected Bending
    • If required, select the Use stamped surface or unfolded surface check box.
      By default, all parameters are available.
      Once you have selected Use stamped surface or unfolded surface, Bottom surface angle is no longer available.
    • Enter the following parameters values.
      • Flange tool parameters
        • Depth: Distance between the bottom surface and the intersection of the extrapolated flange with the stamped surface.
        • Width: Distance between the side of the tool directly in contact with the fllange surface and the other side of the tool.
        • Flange extrapolation: Length added on both sides of the flange to create the tool.
        • Bottom radius: Radius of the fillet between the bottom surface and the flange surface.
        • Bottom surface angle: Angle between the bottom surface and the flange surface.
      • Overbend parameters.
        Overbend may be required to process the elasticity of the material in the bending process.
        • Overbend direction
          • Tangent to flange area
          • Tool direction.
        • Overbend angle: Additional angle between the flange surface and the punch back surface.
          Deactivated if Overbend direction is the Tool direction.
        • Offset correction: Offset from the initial position of the punch back surface, to define its new position.
    Flange Tool creates the bottom surface as the translation of the split of the unfolded surface at the corresponding depth.

  8. If you have selected Manual, enter the following parameter values.
    • Flange tool parameters
      • Depth
      • Width
      • Flange extrapolation
    • Overbend parameters
      • Overbend direction
      • Overbend angle. Deactivated if Overbend direction is the Tool direction.
      • Offset correction.

    Flange Tool provides the 2D punch, the punch wall and the bent support surface.
    The bent support surface is computed from the Flangte tool parameters and the Overbend parameters.
    It can be used to manually design the bottom surface and the rest of the tool.

  9. To correct an erroneous computation
    • Select the Fixed Curve (folding line)
    • Select the Fillet/Flange separator.

A Flange Tool feature is created in the specification tree.
