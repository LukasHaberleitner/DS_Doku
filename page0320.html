Postprocessing for Damage Initiation Output

The Abaqus output variable CSMAXCRT is supported for use in postprocessing. It stores results for the maximum stress-based damage initiation criterion. To request results for this variable, you must modify the Abaqus input file to include the CSMAXCRT variable, submit a job using the modified file, and manually attach the results. Alternatively, you can add CSMAXCRT to the field output request by using the Additional variables option. For more information on field output requests, see Creating and Modifying Field Output Requests.

Select Max stress-based damage initiation from the Abaqus Image Generation dialog box to display an image of the stress-based damage initiation. For more information, see Generating a Results Image.
