Making the Position of a Text Associative 

This task shows you how to set a positional link between a text and another element.
This allows you to move several annotations in only one interaction.
  • Open the Annotations_Part_04.CATPart document.
  •  Ensure that the Create positional link for body of annotation with leader check box is selected, to create positional links for annotations with leader associated to geometric elements. To specify this option, select Tools > Options > Mechanical Design > Functional Tolerancing & Annotation > Annotation.

  1. Right-click the driven text (text itself, frame or leader) and select the Annotation Links > Create Positional Link contextual menu.

  2. Select the driving text (text itself, frame or leader).

    An information popup appears to warn you that the driven text is now at the elevation of the driving annotation. 
     
Driving and driven texts must belong to the same active view and associated geometrical element.
 

  1. Move the driving text: both texts are moving and their distance remains the same.

  2. Now, if you move the driving text you selected, only this annotation is moved.
To delete the associativity, right-click the driven text and select the Annotation Links > Delete Positional Link contextual menu.
 

  1. Right-click the driven text and select the Annotation Links > Query Object Links contextual menu.

    The Query Link Panel dialog box appears.
    It show that a positional link has been created between this annotation and Text.2 annotation.
