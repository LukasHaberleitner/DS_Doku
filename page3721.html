Equipment and Systems Parameters

This section provides information about generative view style parameters. These parameters are used in the drawing production function, which allows you to customize and create a 2-D drawing from a 3-D document. See the section Drawing Production in the user guide for the product you are using for more information, including the location of the sample file.      

Customizing these parameters affects the appearance of your drawings and is an administrative task. These parameters are applicable to the following disciplines/applications:

  • Piping
  • Equipment
  • HVAC
  • Tubing
  • Compartment and Access
  • Raceway & Conduit Design
  • Waveguide

These generative view style parameters are located in the corresponding nodes of the generative view style XML file (available via Tools > Standard > generativeparameters category > DefaultGenerative Style.xml file).

 

Parameter Value Description
ATS_1 ......ATS_40 An ATS name from the application ATS catalog. Text is created if this entry corresponds to an attribute on the 3-D object.
ATS_AutoPosition Yes/No Yes positions the text anchor point at the edge of the box surrounding the generated geometry. No positions the point at the approximate center of the geometry (center of the first segment for multi-segment pipes).
ATS_CharacterOffset An integer value The text is offset by the number (of characters) entered as the value.
CenterLines Yes/No Yes means a center line will display (available in objects with a "double" graphic representation).
CenterLinesExtensionLength A real value The length by which a center line will extend beyond a pipe.
CenterofGravitySymbol Yes/No When set to Yes, a symbol is displayed at a part's center of gravity (for equipment only).
Color From 0 to 255 for each The RGB value of the color of the generated part.
CornerBendPoint Yes/No If set to Yes, the intersection of adjacent pipe (or tube) segments will display a point.
EndStyleSymbol Yes/No Yes means an endstyle symbol will be displayed in the drawing for each piping part that has an endstyle.
FabricationBreakSymbol Yes/No Yes means a symbol will display at the ends of each spool.
GR_1 Representation names such as Single/Double/Envelope/ Layout The value to which GR_1 is set determines the type of graphic representation output to 2-D.
GR_2 Representation names such as Single/Double/Envelope/ Layout The value to which GR_2 is set determines the type of graphic representation output to 2-D.
GR_3 Representation names such as Single/Double/Envelope/ Layout The value to which GR_3 is set determines the type of graphic representation output to 2-D.
LineGapping Yes/No Yes means line gapping will display.
LineGappingValue A real value The length of a gap in a line at a point where it is crossed by another line, in millimeters.
LineThickness An integer value as defined in the Drafting application The value indicating the thickness of the line. You can define a different line thickness for each graphic representation within an application.
LineType An integer value as defined in the Drafting application The value indicating the line type of the generated part.
MaxNominalSize A string value or no entry, from the NominalSize table A part larger than the entered value will not be generated in the 2-D drawing. No entry means there is no limit.
MinNominalSize A string value or no entry, from the NominalSize table A part smaller than the entered value will not be generated in the 2-D drawing. No entry means there is no limit.
Visible Yes/No Yes means the part will be generated in the drawing. No means it will not be generated.
