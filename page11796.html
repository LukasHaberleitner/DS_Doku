Creating a Drilling Hole Table

 

This task shows you how to create a drilling hole table using the Drilling Hole Table command in the Drafting workbench:
  • A table is created in the current front view.
    It contains all the hole and thread technological results corresponding to the CATPart linked to the current front view,
    as well as the labels linking the holes in the front view to the table lines.
  • The result table can be exported to a dxf file.
  • The result is not associative with the CATPart document.
    If the part or its holes are modified, updating the drawing will not update the result.

Technological Results are systematically associated to holes created with Mold Tooling Design or Tooling Design.
As a consequence, Drilling Hole Table always processes those holes.
Be aware that holes created in other workbenches (e.g. Part Design) are not associated with Technological Results.
As a consequence, they are ignored by Drilling Hole Table.
To generate a drilling hole table from those holes, first generate the Technological Results for those holes
(in Part Design, Create Technological Results in the contextual menu of the body containing the holes).
  • Open TechnologicalResult02.CATProduct from the samples/TechnologicalResultB directory.
  • You need to have appropriate Mold Tooling and Drafting licenses.
  • Launch the command from a drawing sheet containing a front view.
  • This front view must be generated from a CATPart, not a CATProduct.
  • The axis system must be selected in the CATPart linked to the front view.

  1. In the Drafting workbench, go to View > Toolbars and make sure Tooling to Drawing is selected.
    The Tooling to Drawing toolbar looks like this

  2. Create a front view from the CATPart containing the technological results and an axis system selected in the same CATPart.

  3. Click Drilling Hole Table .
    The Drilling Hole Table dialog box is displayed.

  4. If necessary, select another axis system in the linked CATPart.
    All the hole positions will be recomputed in this axis system.

  5. Key in a title.
    Note that the CATPart name, the name of the axis system selected and the time of creation are automatically added.

  6. Define a prefix:

    • If Based on Param is not selected, all references will be constructed as Prexi + incremented index.
    • If Based on Param is selected,  use parameters to give different holes different references.
    • If Display Labels is selected, the reference is displayed as a label in the front view.
    • If Display Labels is not selected, no label is created.

  7. Select the items to display in the table by passing them from Not Shown Items to Shown Items and vice versa.
    You can also order them.
    • If Merge Thread into Hole is selected, the information about the hole and its thread are merged in one line.
    • If Merge Thread into Hole is not selected, the information are displayed in two lines.

  8. If necessary, sort the table contents.

  9. Click OK to generate the drilling hole table and exit the dialog box.
    The table is positioned at the origin of the front view but you can move it to another location.
