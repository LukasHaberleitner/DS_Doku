Closing Elements  

This task shows how to close circles, ellipses or splines using relimiting operation.

Create a three point arc.

 

  1. Click Close arc in the Operation toolbar (Relimitations sub-toolbar).

  2. Select one or more elements to be trimmed. For example, a three point arc.

    The arc is now closed.
In the case of a spline that was relimited by using Trim , the spline is set to its original limitation.
