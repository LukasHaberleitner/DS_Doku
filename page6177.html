Toolbars

 

This section describes the various icons available in the Digitized Shape Editor workbench.

The toolbars

  • Cloud Import and Cloud Export
  • Cloud Edition
  • Cloud Reposit
  • Mesh
  • Operations
  • Scan Creation
  • Mesh Edition
  • Curve Creation
  • Analysis
  • Transformations
  • Wireframe
are located on the right in the default set-up

The View Management tool bar is located at the bottom of the screen.
