Tool Path Display

Display Tool Path during Simulation

If this option is selected, the tool path is visible when running the Tool Path Simulation command.

By default, the option is not selected.

Show Point

If this option is selected (and the Display Tool Path during Simulation option is selected), the TCP points are displayed.  You can select the following display options for the points:

  • Color, which can be selected from the list

  • Type, which can selected from the list.

By default, the options are as shown.

Show Line

If this option is selected (and the Display Tool Path during Simulation option is selected), the tool path trace is displayed as a line.  You can select the following display options for the line:

  • Color, which can be selected from the list

  • Type, which can selected from the list.

  • Line Thickness, which can be selected from the list.

By default, the options are as shown.

Tool Path Modification

Modification Range
The Modification Range determines the range of modifications. Each tool path comprises discrete segments.  The number of segments set in the Modification Range is the number of segments available for modification when a user chooses Modify Tool Path.  Similarly, when users select Partial Simulation, the Modification Range determines how many segments of the tool path are shown.

By default, the Modification Range is set to 5.

Integrated Material Removal

Selecting the Enable material removal computation during machine simulation check box enables material removal to be computed during simulation. The Material Removal Updates OFF/ON, and Saving of stock commands are available in the Process Simulation toolbar only when this check box is selected.

Selecting this check box consumes more time and memory for simulation launch. If you do not require material removal simulation during machine simulation, your performance is improved by performing simulation without material removal updates. Video based on NC Code can be used to validate material removal without affecting performance.

By default, Display tool color on stock is enabled when you check  Enable material removal computation during machine simulation. When Enable material removal computation during machine simulation is not selected, Display tool color is not available.
