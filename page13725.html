Keeping Input Selection in Cutting Plane Analysis
You can now select the new Keep input element selection after Ok is pressed option available in Tools  > Options > Shape > Mechanical Surface Refinement > General > Cutting Plane Analysis Option. It keeps the input elements of the Cutting Plane Analysis selected after the validation so that you can reuse them to create a new analysis.
If you select the new option, you can assess the geometry with different parameters on the same set of input elements. This reduces mouse clicks and mouse miles required to reselect the input for a new analysis.
