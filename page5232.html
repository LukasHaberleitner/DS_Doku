Creating an Opening

This task shows you how to create an opening for an access object, such as a door, hatch, or other object for which you did not create an opening automatically when you placed it. The access part must have been defined with an opening associated with it for this command to work. Refer to Creating an Opening for an Access Part for more information.
1. Select the Create Opening button. You are prompted to select an access part.

As you move your mouse pointer in your document, any parts that are valid candidates for this command are highlighted.
2. Select an access object.

The opening is created and associated with the object you selected.
