Delete/Unbuild a Component

This task shows you how to "unbuild" or "deconstruct" a component.    
When you unbuild a component you to "untype" it - remove the subclass, connectors and flow path that define the component. All you are left with is the geometry. This allows you to reuse the geometry to build another component.
1. To unbuild a component right-click on it and select Delete in the drop down menu that shows.
2. A message will ask if you want to delete the reference component. Click OK. You will be left with the geometry only. Repeat the process to delete the geometry also.
