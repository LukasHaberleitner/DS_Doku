Managing Producibilities

The producibility is a feature associated to the plies, that defines the parameters representing the way the material is used in the downstream process to manufacture the plies.
You can run the producibility, which simulates the way in which fibers of a ply behave.
This creates a visualization of the draping of the material so that it is possible to detect manufacturability problems.
The producibility depends on the fiber strategy (hand lay-up, tape laying, fiber placement).
It allows you to analyze the fiber propagation on each ply.
