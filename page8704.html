Creating a Blend Surface

The task shows you how to create a blend surface.

  1. Select the flange surface in the specification tree.
 

  1. Activate the Disassemble icon in the Shape Management toolbar and click OK in the Disassemble dialog box. The associativity created with the previous task is eliminated.
 

  1. Select Surface.1 and click the Translate icon in the Shape Management toolbar.

  2. Choose in the Dialog box Coordinates as Vector Definition and change the values for x to -20 and for z to 20. Click Hide/Show initial element and OK.
 

  1. Now click the Blend Surface icon in the Surface Creation toolbar.
    The Blend Surface dialog box is displayed.
 

  1. Select one edge of each surface.
 

  1. Click Apply to create the blend surface.
