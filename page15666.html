Special Rules for Saving CATIA V5 Data into ENOVIA LCA

This task shows you special rules to be fulfilled in CATIA V5 before saving the Product in ENOVIA LCA.

Commands applied to special rules

The special rules applied to the following CATIA V5 commands are:

  • Replace Component,
  • Manage Representations,
  • Copy/Cut/Paste.
 

Special Rules

Here are the rules for these commands:

  • A work package or a file CAN be mapped into a work package,
  • A work package or a file  CAN be mapped into an exposed structure,
  • An exposed structure CAN be mapped into an exposed structure,
  • An exposed structure CANNOT be mapped into a work package or a file.
