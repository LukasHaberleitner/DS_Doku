Licensing
  Depending on your configuration and licenses, all the functionalities of NC Manufacturing Verification can be used to analyze output of material removal simulation (using the Video command) based on ISO code.

For example, if material removal simulation (Video) has been run using ISO Code, then every NC Manufacturing Verification command (for example, Analysis of pick point information or Run Summary Generation) provides information about the ISO line.
