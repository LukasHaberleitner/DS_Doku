Inserting in a New Geometrical Set

This task shows how to insert the result of an operation in a new geometrical set.
Only available with the following commands:
  • Break Curves
  • Break Surfaces
  • Disassemble
  • Copy Geometric Parameters
  • Apply Dress-Up

Open the FreeStyle_Part_40.CATPart document.

  1. Click Break .
    The FreeStyle Break dialog box appears.

  2. Select the Break Curves option in the Break dialog box:

  3. Click Insert in a new geometrical set  in Tools Dashboard toolbar.

  4. Select the surface to be cut.

  5. Select the cutting curves.

  6. Click OK in the Break dialog box.
    All the created surfaces resulting from the break are displayed in a new geometrical set in the specification tree.
    For further information about the Break command, please refer to the following chapters: Breaking Curves and Breaking Surfaces.
