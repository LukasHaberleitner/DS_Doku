Specific Use of Multi-slide Lathe Machine

Instead of using a default Horizontal or Vertical lathe machine, you are recommended to use a Multi-slide lathe machine restricted to one spindle and one turret.

This section provides information about this specific use of the Multi-slide lathe machine.

The main objective is to enable you to define a single tooling definition in a tool repository (database or catalog) that is independent of the selected machine.

As from V5R15, using the Lathe Machining product it is possible to select a Multi-slide lathe machine (restricted to one spindle and one turret) to allow:

  • Definition of Turning Tool Axis System independently from Spindle Part Axis System.

  • Definition of Turning Tool Axis System for tool visualization.
    The convention is the same as usual: the radial axis is the reference for Setup angle (0 deg).

  • Definition of Spindle Part Axis System for NC output purposes.
