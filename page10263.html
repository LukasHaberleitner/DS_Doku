CatPrismExtrusionDirection (Enumeration)

enum CatPrismExtrusionDirection {
  catNormalToSketchDirection,
  catNotNormalToSketchDirection
}
Prism extrusion type.
The prism shapes can be extruded using a default direction normal to the sketch plane. Otherwise, the extrusion direction should be specified.
Values:
catNormalToSketchDirection
The prism is extruded using the sketch normal direction
catNotNormalToSketchDirection
The prism is not extruded using the sketch normal direction. The actual extrusion direction should be specified.
