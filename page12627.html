Filleting an Edge

This task shows you how to use one of the fillet commands designed to fillet edges.

  1. Click Edge Fillet .
    The Edge Fillet Definition dialog box appears. It contains default values.

  2.  Select the edge to be filleted, that is, to be rounded.

  The icon now available after the Objects to fillet field lets you edit the list of the objects to be filleted. For more information about that capability, refer to Editing a List of Elements.

Clicking Preview lets you see what the default fillet would look like.
