Changing Manikin Display Attributes

This task describes how to set and edit manikin display attributes.

  1. Select the Display Attributes   and in the 3D geometry window or the specification tree, select a manikin.

  2. The Display dialog box has the following choices:

    • Ellipses

    • Resolution
    Vision

    • Cone type
    Others Center of gravity
