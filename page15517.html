Data Mapping

You will see here how to work with analysis data mapping files stored in ENOVIA.

To modify a data mapping file (.xls) stored in ENOVIA, you have to work with ENOVIA V5 VPM:

  • Search for the data mapping file (.xls file)

  • Check out the data mapping file

  • Modify the data mapping file

  • Check-in the data mapping file.

To find out more, refer to the ENOVIA / CATIA Interoperability User's Guide - Checking in an ENOVIA V5 VPM Document.
