Creating a Logical Set

This task shows you how to create a logical set.
A logical set is a mechanism for organizing and grouping. You can create logical sets under a logical assembly (such as a Raceway Assembly) and then create runs under them. See Creating a System Assembly for more information.

1. Click on the Logical Set button. The Logical Set Creation dialog box is displayed.

2. Click on the down arrow to designate your network as being of a certain type, i.e. work set.
3. Click OK.
