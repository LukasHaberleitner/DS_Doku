Adding Attributes to an Activity

Attributes provide the means of enriching the information provided in the library. Any attributes of a parent activity type are passed onto child activity types when the child activity types are created.

For example,  if the Time at Temp. attribute is defined for the Heat Treatment activity, then all Heat Treatment's child processes inherit that attribute.  In the example below, Quenching, as well as the other activity subtypes, inherits the attribute.

  1. To create a new attribute (e.g., Time at Temp), select the activity type on which the attribute has to be added (e.g., Heat Treatment).

  2. Click Add Attribute .

    The Add attribute dialog box appears:

  3. Enter the attribute name in the Attribute Name field.

  4. Choose the Attribute Type: Integers, Double, and String.

  5. Set the Attribute Value for the default attribute.

  6. Click the Add button.

  7. Create another attribute or click OK.

  8. Save the process library using File> Save As.
  • Process libraries have a file extension of .act.
  • You can set the associated operator for an activity or activities within a library before saving it.
