Numerical Display Properties

This reference describes the Numerical Display properties concerning:
  • Semantic geometrical tolerance.
 
  This page deals with the properties concerning:
  • The Numerical Display.
  • The Extension Line.
 

Numerical Display
 
 

Precision
  Defines the number of digit displayed after the numerical separator.
  By default, the precision is 0.01.
 

Separator
  Defines the symbol used as numerical separator.
  By default, the annotation set standard numerical separator, for example a comma for ISO standard.
 

Displays leading zeros
  Defines whether a "0" is displayed before the numerical separator when the value is less than 1 or not.
 
  • Selected:
  • Not selected:
  By default, this option is selected.
 

Displays trailing zeros
  Defines whether "0" are displayed to complete the number of digit displayed after the separator according to the precision or not.
 
  • Selected (default precision):
  • Not selected:
  By default, this option is not selected.
 

Extension Line
 
 

Length
  Defines the extension line length, positive or negative, between the leader and the geometrical tolerance frame.
  By default, 0 mm.
