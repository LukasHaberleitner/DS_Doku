Tolerances

Tolerance is the threshold below which a value is considered as null. Here are the tolerances for the different magnitudes::
  • G0 tolerance = factory resolution for length, e.g 0.001mm by default.
  • G1 tolerance = 0.01 degree.
  • G2 tolerance = 2 %.
  • G3 tolerance = 0.1 degree.
  • Tolerance for angle between osculator plane = 0.01 degree”.
