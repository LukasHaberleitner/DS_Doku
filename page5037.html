Managing Project Resources

Project resources are typically managed by system administrators. Typically this involves entering information such as folder paths for resources, like user dictionaries such as the feature dictionary, without which the application cannot function. What administrators enter will be visible in the Browse tab of the Project Resource Management dialog box (Tools > Project Management > Select/Browse), but it cannot be changed there.



Note: The following resources mentioned above are no longer used or are for internal use only:
DpmStrChamferAngleTable
DpmStrMBOMtoXMLLocation
StructureDrawingSymbolsCatalog

To make sure all the required and optional resources are correct in the PRM file, use the PRM Checker command under Tools  >  Project Management. Then click Apply in the dialog box that opens. The results are displayed in the same dialog box.

A sample project resource management file is provided and by default, the application will start with this file. The resources managed in this file are described below.

Drawing Templates

DpmStrDrwTemplateLocation: specifies the location of all drawing templates (CATDrawing and XSL files). Template CATDrawings containing the title block and specifying the drawing standard applied (such as ANSI or ISO) as well as the sheet format are provided. The name of each drawing template is defined in the corresponding workshop document style XML file.
The default location for this resource is:

..\OS\startup\sample\WDExtraction

Location for Saving Documents

DpmStrWDSaveLocation: specifies the location in which all extracted workshop documents are to be saved.
The default location for this resource is: C:\Temp

Drawing Symbol and Manufacturing Resource Catalogs

DpmStrGVSSymbolsCatalog: specifies a sample catalog containing GVS drawing symbols.
The default location for this catalog is:

..\OS\startup\EquipmentAndSystems\Manufacturing\DrawingSymbols\GenerativeViewStyle_2DSymbols.catalog

ProfileFabricationSketchSymbolsCatalog: specifies the catalog containing the drawing symbols for straight profile drawings.
The default location for this catalog is:

..\OS\startup\EquipmentAndSystems\Manufacturing\DrawingSymbols\ProfileFabricationSketch_2DSymbols.catalog

StructureResourcesCatalog: specifies a catalog containing descriptions (and links to parts) of manufacturing resources such as Pin Jigs.
The default location for this catalog is:

..\OS\startup\components\StructuralCatalogs\ManufacturingCatalogs\ManufacturingCatalog.catalog

Generative Parameters for Workshop Documents

DpmStrGenerativeViewStyle: This GVS resource file is the main repository for storing graphic properties of all standard or user-defined types and sub-types. All workshop documents will use this file for graphic properties. The default location for this resource is:

..\OS\resources\standard\generativeparameters\SAMPLES\DPMStructure_GVSSample.xml

All other properties related to the other various workshop documents are in different XML style files. These files are  located in:
..\OS\resources\standard\generativeparameters\SAMPLES\nameOfWDStyleFile
where nameOfWDStyleFile is the name of the workshop document XML style file.

DpmStrFabSketchPlateStyle: specifies the full path of the XML file defining the style used for plate workshop documents. This file contains general drawing options and generative view style parameters. The workshop document generated is a CATDrawing of scale 'Fit to paper'.
The name of the style XML file is FabSketchDocument_PlateStyle.xml

Note: If you want to generate plate workshop documents in DXF format, use the sample XML file located in ..\OS\startup\sample\WDExtraction. This file will generate plate workshop documents with a scale of 1:1.

DpmStrTemplateFabSketchPlateStyle: specifies the full path of the XML file defining the style used for template sketches. This file contains general drawing options only. There are no generative view style parameters for template sketches.
The name of the style XML file is FabSketchDocument_TemplateStyle.xml

DpmStrProfileStraightDrwStyle: specifies the full path of the XML file defining the style used for straight profile fabrication sketches. This file contains general drawing options and generative view style parameters.
The name of the style  XML file is FabSketchDocument_ProfileStyle.xml

DpmStrInverseBendingDataStyle: specifies the full path of the XML file defining the style used for inverse bending curve data. This file contains the name of the XSL stylesheet used to generate the XML file.
The name of the style XML file is FabSketchDocument_IBCStyle.xml

DpmStrPinJigDataStyle: specifies the full path of the XML file defining the style used for pin jig data. This file contains the name of the XSL stylesheet used to generate the XML file.
The name of the style XML file is FabSketchDocument_PJDStyle.xml

DpmStrPinJigDrgStyle: specifies the full path of the XML file defining the style used for pin jig resource sketches. This file contains general drawing options and generative view style parameters.
The name of the style XML file is FabSketchDocument_PinJigDrgStyle.xml

DpmStrGirthTapeDataStyle: specifies the full path of the XML file defining the style used for girth table data. This file contains the name of the XSL stylesheet used to generate the XML file.
The name of the workshop document XML file is FabSketchDocument_GTDStyle.xml

DpmStrGirthTapeDrgStyle: specifies the full path of the XML file defining the style used for girth table resource sketches. This file contains general drawing options and generative view style parameters.
The name of the style XML file is FabSketchDocument_GirthTapeDrgStyle.xml

DpmStrProfileXMLDataStyle: specifies the full path of the XML file defining the style used for profile data. This file contains the name of the XSL stylesheet used to generate the XML file.
The name of the style XML file is FabSketchDocument_ProfileXMLStyle.xml

Thickness Tables

DpmStrMarginSizeTable: specifies the design table containing possible margin thicknesses (positive or negative). This file is accessed when user clicks the Design table icon in Joining Activity's Added Material tab.
The default location for this resource is:

..\OS\startup\components\StructuralCatalogs\ThicknessTables\MarginListSample.txt

Edge Preparation

DpmStrEdgePrepCatalog: specifies the catalog containing definitions of edge preparation UDFs.
The default location for this resource is:

..\OS\startup\FeatureCatalogs\Edge_Preparation.catalog

DpmStrEdgePrepTable: specifies the design table used during automatic generation of edge preparations when building the activity list. This table contains the different edge preparations to be used for each weld (e.g., butt-butt between 20mm plates) and their values. Entries for the plates and profiles used in a given project must be defined in this file to support the automatic generation of edge preparations.

The default location for this resource is:

..\OS\startup\components\StructuralCatalogs\ManufacturingCatalogs\Edge_PreparationSample.txt

DpmStrEdgePrepValuesTable: specifies the design table containing all the possible values for the edge preparation UDFs. This file is used with the previous one (DpmStrEdgePrepTable).
The default location for this resource is:

..\OS\startup\components\StructuralCatalogs\ManufacturingCatalogs\Edge_Preparation_ValuesSample.txt
