Enabling Image Capture

This procedure describes how to enable your system to capture images. This procedure specifically describes how to set the CaptureImage attribute for the main process (because all viewpoint activity children created under this process will then be captured). Users, however, can follow these directions (while substituting a particular activity for the process node) to set the attribute for a sub-process.

  1. On the PPR tree, select the Process node.

  2. Use the right mouse button (for the context menu) and select Properties. The Properties dialog box appears.

  3. Select the Specific tab.


     

  4. Use the pulldown menu to select the Attribute Type "String."

  5. In the Attribute Name box, enter CaptureImage.

  6. Click Add. The CaptureImage attribute appears under Label.

  7. Select the attribute. The Selected attribute value box opens up.


     

  8. Enter True in the CaptureImage box.

  9. Click Save to save the state.

  10. Click OK to close the dialog box.
