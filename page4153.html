Interpolating Tags  

This procedure describes how to interpolate tag orientation to make robot motion smoother.  The command relies on two tags in a sequence having the correct orientation.  By selecting those two tags, the ones in between will have their orientation interpolated by the command.

  1. Click Interpolate Tags .

  2. Select a tag within the tag group.

    The Interpolate Targets dialog box appears.
    The tags are ordered.

  3. Select the tags whose orientation is the desired one and all the tags in between.

     For instance, in the example above, Tags 1 and 3 have the correct orientation and Tag2 requires interpolation.  Select Tags 1-3.

     

  4. Click the Interpolate button.

    The orientation of the tags changes to match that of the two outer ones.

  5. Click the Close button.
