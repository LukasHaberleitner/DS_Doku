Performing a Depth Analysis

This task shows you how perform a depth analysis of the part.
Open CleanContour01.CATPart from the samples directory.

  1. Click Depth Analysis     in the Analysis toolbar.
    If the check box Display detailed text in dialog in Tools/Options/Mechanical Design/Die Face Design is cleared,
    the dialog box looks like this:

    If the check box Display detailed text in dialog in Tools/Options/Mechanical Design/Die Face Design is selected,
    the dialog box looks like this:

  2. Select an axis system (stamping direction).
    The Z-Axis represents the cutting tool axis.

  3. Select the surface or 3D curve to analyze with respect to the Allowed Depth.

  4. Select more analysis options:
    • Compute Areas: Computes connected areas instead of displaying a number of faces.
    • Fly Analysis: Displays precise analysis values while hovering over the shape.
    • Color mode:
      • Smooth: Displays analysis results as a global color gradient applied to the selected shape.
        Allowed values and draft settings are not used.
      • Face/Edge: Displays analysis results as colors, applied face by face on the shapes.
      • Detailed: Displays analysis results as colors, applied precisely on the shapes.
    • View Mode to display the analysis:
      • Mesh
      • Vectors used to compute the draft angle.
      • Planes used to compute the draft angle.
      • Dots
      • Acute/Obtuse Dots.
    • Hide 3D Labels
    • Reframe on selection.

  5. Set the Mesh Behavior.
    • Maximum Size is the step of the mesh. Its default value is 0. Modify it to have a more regular mesh.
    • Lay Down Tolerance.
    Analyses in Die Face Design are done on meshed surfaces. Once the surfaces are meshed, they are shared by all analyses.
    However, Radius Analysis uses its own sag value to mesh the surfaces (minimum allowed sag/3). As a consequence, once you enter the Radius Analysis, existing shared meshes are modified, other analyses are updated.

  6. Check the completion status of key indicators for undercut and vertical areas.

  7. If necessary, modify the Allowed Depth.
    The current maximum depth is displayed.
    Review the areas in error.
    • Click one face in the area to review the area.
    • Pick a face and create an annotation on the area.

  8. Under Extract Geometry, convert the results of the displayed analysis into dead geometry.
    • Extract Areas stores the results as dead geometries and 3D labels.
    • Extract Direction extracts the Z-Axis on each analyzed point.
    • Extract Geometries as Points, Vectors or Boundaries
    • Extract Values as Tags or Points
    • Extract Points number using Full, Minima and Maxima or User Selection.

  9. Check the completion status of key indicator.

A Depth Analysis feature is created. It contains the parameters of the analysis.
