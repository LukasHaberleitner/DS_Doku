Editing Annotated Views Properties

This task explains how to edit annotated views properties. This new capability eases collaborative work as you can now add information such as user name, creation or modification dates.
Insert the following cgr files:

ATOMIZER.cgr
BODY_1_2.cgr
BODY_2_2.cgr
LOCK.cgr
NOZZLE_1_2.cgr
NOZZLE_2_2.cgr
REGULATION_COMMAND.cgr
REGULATOR.cgr
TRIGGER.cgr
VALVE.cgr

Create an Annotated View (see Creating Annotated Views).

  1. Right click the view you need to edit in the specification tree.

  2. Select the  Properties item from the contextual menu displayed. The Properties dialog appears.

  1. Enter the required information (creator name, comments, etc.).

  2. Click OK to confirm.
The next views you create will be assigned the creator name you've just defined.
