Updating Object Names

This task contains information on updating object names. You may want to use this command when you rename an object like a stiffener, and want to give the sub-objects (split stiffener) the same name. The command can also be used with objects like plate and stiffener on free edge.  
1. The image from the specifications tree below shows a stiffener (FStf_001) that consists of three split stiffeners (FStf_001.1,2,3).
  2. Rename the stiffener to StiffenerTest - the names of the split stiffeners do not change.
3. To change the names of the split stiffeners to the new stiffener name, right-click on one of the split objects (you can select more than one), select XXX_object in the drop down menu and, in the second drop down menu, select Update Object Name.

The object is renamed.

Note: If you use the Update Object Name command on the main object, in this example FStf_001, then it will revert to the name as defined by the object naming rules.
