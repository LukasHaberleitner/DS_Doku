Creating MultiHole

This task shows you how to create multiple holes in an object. You can also select the required hole standards and presets to create multiple holes.
 
To create a multiple holes in Part Design:
  • Open the Hole1.CATPart document.
  • Create a sketch or points on any face of the Hole1.CATPart.

  1. Click MultiHole .
    The MultiHole Definition dialog box is displayed.

  2. In the Locations, to one of the following:

    • Select the sketch or points created on the face of Hole1.CATPart.
    • Click the to access the sketcher and then sketch the profile you need.
    • From the contextual menu, select Create Sketch or Create Point to create the sketch or points you need.

    The selected sketch or points are shown in the Locations field. You can click to access the selected locations list.

  3. Specify the hole parameters; Extension, Type, and Definition.
    For more information about the hole parameters, see Creating Holes.
     
  4. Optional: Click Preview to see the result.
     
  5. Click Ok.
    The MultiHole.1 is created in the tree.
