Composites Design

This section deals with Composites Design customizing options.  These options can be reached via Tools/Options and are set out in one tab:

  • Composites Design that defines the settings for Ply Books.
  • Export ply data that defines how ply data are exported to DXF, IGES 2D or IGES 3D.
  • Laser Projection that defines export and preview options for laser projections.
