Update Digitized Morphing

When you create a Digitized Morphing feature, a link to the text file containing the definition vectors is stored. When this text file is modified, the Digitized Morphing becomes out-of-date.

Update Digitized Morphing finds the out-of-date Digitized Morphing features and updates them on request. 

 
Either use a personal model with a modified definition text file,
or create a Digitized Morphing feature as explained in Digitized Morphing
and modify the definition text file DeformationVectors01.txt as you like.

 

  1. Click Update Digitized Morphing .

  2. A dialog box is displayed with the name of the Digitized Morphing feature that is out-of-date:

    The list may contain one or several features, resulting from one or several text file.

  3. Click OK, the Digitized Morphing feature is updated.
All the features of the list are updated in one shot.
If there is no Digitized Morphing feature to update, a message is displayed.
