Adding Images to Activities   

This command allows for images to be captured and added to an activity in the Process document. Either an existing image file can be read, or a new image can be grabbed and saved from the Space Inventory Window.
To use it this feature, the Space inventory window must be opened and made active.

To grab the entire Space Inventory Window
 

  1. Click Grab And Attach Image  .

  2. Simultaneously, press the Control key and select the activity under which the image is to be added.

  3. A Save panel will pop up to ask you the name of the file and the directory where you want to store the newly created image.

  4. After saving the image and clicking the OK button, two new attributes will be created under the selected operation:

  • ImageDir (for the path of the new file)
  • ImageName (for the name of the file)

To grab a portion of the Space Inventory Window
 

  1. Click Grab And Attach Image  .

  2. To grab a portion of the space inventory window, press the left mouse button and drag the mouse to select the desired portion of the window.

  3. Select the operation under which the newly-created image will be added.

To add an existing image
 

  1. Click Grab And Attach Image  .

  2. Select the operation under which the image will be added.

  3. A Read dialog will appear, prompting you to select the desired image file.

  4. After saving the image and clicking the OK button, two new attributes will be created under the selected operation:

  • ImageDir (for the path of the new file)
  • ImageName (for the name of the file)
