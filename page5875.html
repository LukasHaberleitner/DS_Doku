Customizing

Before you start your first working session, you can set options needed to run the product or customize the way you work to suit your habits.

This type of customization is stored in permanent settings files: these settings will not be lost if you end your session.

  1. From the menu bar, select Tools > Options.

    The Options dialog box appears.

  2. On the navigation bar on left of the dialog box, select the Infrastructure category.

  3. If not already expanded, select the plus, and then select the DELMIA Infrastructure category.

    The tabs appear.
    These tabs enable you to define the following options:
    • Simulation
    • Analysis
    • Device Analysis
    • Robot Analysis
    • IO Analysis
    • Simulation Trace

  4. Set options in these tabs according to your needs.

  5. On the navigation bar on left of the dialog box, select the General category.

  6. If not already expanded, select the plus, and then select the Display category.

    The tabs appear.
    This enables you to control the Performances options.

  7. On the navigation bar on the left of the dialog box, select the Parameters and Measure category.

    The tabs appear.
    You may wish to alter the Measure Tools options.

  8. On the navigation bar on left of the dialog box, select the Digital Mockup category.

  9. If not already expanded, select the plus, and then select the DMU Space Analysis workbench.

    The tabs appear.
    This enables you to control the DMU Clash options.

  10. Click OK.
