Administration Tasks

In the Interactive Drafting workbench, administration tasks deals with the administration of standards. These tasks must be performed by an administrator.

Administrators can manage and customize standards such as ISO, JIS, ANSI, ASME, etc. or company standards. The Standards Editor lets administrators set the standards used for dress-up, dimensions, annotations, etc. as well as set the styles that will be used as defaults for element properties in the Interactive Drafting workbench.

 

The format of the standard file has been changed from V5 R9 onwards. If you were using a customized CATDrwStandard file on a release up to V5 R8, you need to upgrade the standard file to the new XML format.
