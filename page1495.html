Snapping the Viewpoint

This tasks explains how to adjust the viewpoint automatically after rotating it so that it matches the standard view it comes closest to.
Open the Search1.CATProduct document.

  1. Select View > Navigation Mode > Viewpoint Snapping.

  2. Manipulate the current viewpoint by rotating the object using the mouse (press and hold down the middle mouse button then press the left or right button and drag still holding down both buttons):

    The viewpoint is not snapped if you use the compass to manipulate the viewpoint.

  3. Release the mouse buttons.

    The viewpoint is adjusted and snapped to the nearest standard view (Top View in our example) is applied:

  4. To deactivate viewpoint snapping, clear Viewpoint Snapping in View > Navigation Mode.
