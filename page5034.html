Extracting Workshop Documents

Workshop Documents Extraction lets you automatically generate the manufacturing deliverables used as input for shipyard cutting, marking and forming machines. Different types of workshop document can be generated:
  • fabrication sketches
  • template sketches
  • resource sketches
  • XML data files (inverse bending curve data, pin jig data, and girth table data).

It is the administrator's job to assign types of workshop document to the various activities as well as to specify all drawing parameters, including the drawing standard and the generative view style (GVS) defining the appearance of the drawing. Refer to Administration Tasks for more information.

Here are some required settings you should check before extracting workshop documents:

  1. Check that all workshop document style XML files are correctly set (see Project Resources for more information).

    • Select Tools > Project Management > Select/Browse
    • In the Browse tab, select the DPMStructure application and check the resources.

    Sample files are located in:
    ..\OS\resources\standard\generativeparameters\SAMPLES\nameOfWDStyleFile

    Note that the SAMPLES folder shown in the path above must be created under ..\OS\resources\standard\ manually. If the resources are not set correctly, contact your administrator.

  2. Set the correct drafting options for generating sketches.

    • Select Tools > Options > Mechanical Design > Drafting.

    • Select the Administration tab and select the Prevent generative view style usage check box.

    • Select the General tab and clear the Display in the current view check box:

    • Click OK when done.
