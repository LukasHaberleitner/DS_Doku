Visualization Properties

This reference describes the Visualization properties concerning:
  • Captures.
 
  This page deals with the properties concerning:
  • The Capture Options.
 

Capture Options
 
 

Display in 3D
Creates a representation of the capture in the 3D.
By default, this option is not selected.
 

Filter
 

Manage the visibility of part instances, bodies and geometrical sets
  Defines whether the visibility status of part instances, bodies and geometrical sets is kept in a capture or not.
  By default, this option is not selected.
 

Clipping plane
Defines whether the capture clipping plane is to be displayed or not.
By default, this option is not selected.

Hidden Text
Creates a tooltip for the capture in the specification tree and the capture callout.
By default, this option is not selected.
 

Append External File Content
  Imports the hidden text content from a text file.
 

Reset Field
  Empties the hidden text content.
