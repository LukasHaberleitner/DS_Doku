Associativity

This page deals with the options to allow you to choose which command when created is either a Feature result or a Datum result.

Associativity
  • Local: You can select individually commands for which associativity on ICEM developed feature-based commands shall be activated.
  • Global: You can globally activate/deactivate the associativity capability for all commands.

ISD Commands
Defines the commands to be feature-based if 'Local' is selected.
