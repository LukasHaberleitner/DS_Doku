ICEM External I/O

This tab contains options for the import and Export of ICEM data bases (ICEM DB):
  • Geometry to be imported
  • Continuity optimization
  • Attributes to be imported
  • Additional Options
  • Geometry to be exported
  • Attributes to be exported
  • Additional Export Options

Geometry to be imported

In this area you can select the ICEM DB element types to be imported  and determine in which CATPart element types they shall be converted.

Continuity optimization of Edge Curves and Independent Curves
You can choose between different tolerance settings to optimize continuity in case of C2 discontinuities on independent curves or 2D edges.
  • No Optimatization: Curves will be split at non C2 continuity points. The geometry of the curve remains unchanged.
  • Automatic Optimatization: Curves will be approximated at non C2 continuity points with the default tolerance.
  • Tolerance: Curves will be approximated at non C2 continuity points with the given tolerance.

Attributes to be imported
You can import Colors, Layers, Lists, Selection Sets (), Lights, Materials, and Texts defined in the imported ICEM databases.

  • Lists are converted to Selection Sets.
  • Materials individually assigned to an element can be imported together with texture data. Click the Texture Paths button to open the Texture Paths dialog box where you can add and remove texture paths.
    Note: The import of material and texture data is only supported with ICEMDB files version 4.12 and higher. You can select the ICEM DB version in the Additional Export Options area.
    • Allow Material in Geometrical Set: Assigns a material to a geometrical set with which all materials belonging to the set are displayed. You can assign another material to child sets to display them with this material.
      If all two dimensional elements of a group have individual materials the most used material is attached to the corresponding geometrical set.
      No material will be attached to the group if it includes a two dimensional geometrical object with no individual material assigned.
      A material will be attached to a geometrical element only if it is different from the geometrical set's material.
  • Texts are imported as text with leader (annotation).
  • Views are imported as cameras.
    Note: The data exchange supports both parallel and central perspective views.
    View layouts (multiple views) and zoom factor are not imported.

Attributes to be exported
The selected attributes defined in the CATPart are saved in the ICEM DB.
  • Selection Sets are converted to Lists.
  • Materials can be exported together with texture data. Click the Texture Path button to open the file browser for selecting a texture path where the texture images shall be exported. The path is displayed in the text field. If the text field is empyt, no texture data are exported.
    Note: The export of material and texture data is only supported with ICEMDB files version 4.12 and higher. You can select the ICEM DB version in the Additional Export Options area.
  • Cameras are exported to views.
    Note: The data exchange supports both parallel and central perspective views.
    Multiple views and zoom factor are not exported.
  • Capture data are saved in an ICEM Surf list. The created *.icem file contains an additional ICEM Surf list per CATIA capture. The list name is composed of the capture name plus the prefix "C_". It contains all geometry objects filtered by the capture.

Additional Export Options
  • ICEM DB Version 4.12 | 4.11: You can select the ICEM DB version for the export.
    Materials and textures can only be exported to ICEM DB version 4.12. If 4.11 is selected, the Materials check box and Texture Path button in the 'Attributes to be exported' area are disabled and no material will be exported.
  • Export Hidden Elements: Also elements in 'Hidden' mode are exported into the ICEM DB.
  • Crop underlying Surface: Underlying surfaces can be cropped at face boundaries.
In the following cases the underlying surfaces will unconditionally be retracted to the face boundaries:
  • unlimited geometry
  • if the underlying surface is smaller than the face.
