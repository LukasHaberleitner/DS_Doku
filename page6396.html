Creating Tangency Constraints on Profiles  

You can create tangency constraints between eligible components of a selected geometry and over several selected elements in the profile.

  • Create a sketch consisting of at least three elements.

 

  1. Select one element of the whole profile.

  2. Click Auto Search on the shortcut toolbar.
    The auto search uses point propagation. The search result consists of parts of the profile which are in contact with each other without any ambiguity.

  3. Right-click on the profile and select Selected objects > Profile Tangency .

Tangency constraints are created for the selected profile.
