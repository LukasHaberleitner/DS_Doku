Using the Sub-Tree

In a complex process, the PPR tree can become large and complicated. You may want to view or use only a select few tasks, activities, or data within the larger tree.

In the manikin's tree, right-click on the node that you would like to use and select the Open Sub-Tree from the contextual menu.

The sub-tree window for the selected manikin node opens. Objects can be selected and manipulated in the sub-tree window in the same way as in the PPR tree.
