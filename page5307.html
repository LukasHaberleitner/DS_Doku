Creating an Associative Solid From Zones

This task shows you how to create a solid from the zones you defined for the Composites part.
  • You can use Zones Groups created with the For Solid From Zones Creation Only check box.
  • If the solid is not complete or accurate, you have the possibility to create drop-offs later on.
Open SolidFromZoneSA1.CATPart from the samples directory.

  1. Click Associative Solid From Zones in the Preliminary Design toolbar.
    The Solid from Zones dialog box is displayed.
     

  2. The option proposed by default is Top Surface.
    Change it to Solid option. The name proposed in Solid name is updated accordingly.

  3. Select Zones Group.1 as the Zones group on which you want to create the solid.

  4. Now decide whether you want to create transition zones or not:

    • If you leave the Solid without Transition-Zones check-box cleared, you will create a solid with transition zones.
      The iso-thickness areas will be generated, taking into account the materials of the laminate of the underlying zones.
      The ramps will also be generated, taking into account the definitions of zones as well as the definitions of ITP, if they exist.
    • If you check the Solid without Transition-Zones check-box, you will not create the transition zones.
      Only iso-thickness areas will be created. Ramps will not be created.

    We have left the Solid without Transition-Zones check-box cleared

  5. Click Apply to visualize the solid, and OK to validate and exit the action.
    The solid is created in the specification tree under Solid from Zones and in the 3D view.

  6. See also Updating the Solid if errors occur while creating the solid.
