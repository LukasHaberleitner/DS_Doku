Defining the Global Parameters of Analyses

This task shows you how to configure the global parameters of the instant analyses.

  1. Click Analysis Configuration    in the Analysis toolbar.

  2. Select a Method Process Plan.

  3. Click the button of the required analysis and enter the values in the dialog box.
    • The global parameters are created directly under the Process Method Plan, under the DFDProcessBody.
    • Relations are created between the global and local parameters of each analysis created in the process.
    • The result of the cut analysis are available in Vectors and Acute/Obtuse modes outside the command.
