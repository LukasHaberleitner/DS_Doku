Editing Control Points

This task explains how to:
  • Edit the current control point coordinates.
  • Freeze control points to lock their selection and manipulate them from a distance.
  • Create construction points representing specific locations of control point.
Open the FreeStyle_Part_38.CATPart document.

  1. Click the Control Points icon:
    The Control Points dialog box appears.

  2. Select Surface.1
    Control points and the mesh are displayed on the surface.

  3. Click the Normal to compass icon if not already selected:

  4. Click the Points only icon if not already selected:

  5. Pre-select the point as shown, the cross cursor is displayed, and select Edit position from the contextual menu.
    The Edit position dialog box appears.
    The Edit position command is applicable to a control point or a selection point of a mesh line.

  6. Set the Z coordinate to -50mm.
    The pre-selected control point is moved accordingly.

  7. Click OK in the Edit position dialog box.

  8. Pre-select the point as shown, the cross cursor is displayed.

  9. Move away the cursor from the point as shown.
    The arrow cursor is displayed and the control point manipulator is deselected.

  10. Pre-select the previous point again and select Freeze option from the contextual menu.

  11. Move away again the cursor from the point.
    • The Freeze option allows you to keep selected mesh lines or control points.
    • You have to clear this option before to make another selection.
    • Contextual menu is now available anywhere in the geometry window.

  12. Clear Freeze option from the contextual menu.

  13. Pre-select the point as shown and select Keep this point from the contextual menu.
    The point is created at the current control point position.
    And in the specification tree.

  14. Move the previous selected control point.
    The created point has kept the position.
    The Keep all Points command allows you to keep the current position of all the control points.

  15. Optional: Select the modification mode.
    • Dynamic : Lets you modify control points and mesh lines dynamically.
    • Static : Lets you modify control points and mesh lines statically.

    These options are available only with the ICEM license.
