Merging Cloud of Points

 

This  task shows how to merge clouds of points.
Open the Merge1.CATPart model the from the samples directory.
It consists of three clouds of points that have been imported separately.

  1. Click Merge Clouds in the Operations toolbar. The Clouds Union dialog box appears.

  2. Select the clouds you want to merge. The list in the dialog box is updated.

  3. To remove a cloud from the list of clouds to merge, make its name active in the list and click Remove.

  4. To replace a cloud in the list of clouds to merge, remove it, then select a new cloud.

  5. Once you have selected all clouds to be merged, click OK. A new cloud is created.
  • An element Clouds Union.x is created in the specification tree.
  • You could also import a set of clouds.
