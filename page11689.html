Accessing NC Machine Tool Simulation

This procedure describes how to access the NC Machine Tool Simulation workbench. Two methods are available:
  • from the Start menu
  • from NC Manufacturing workbenches, such as NC Manufacturing Review.
NC Machine Tool Simulation  provides validation and modification facilities. To use NC Machine Tool Simulation, processes with manufacturing activities must be created using any of the NC Manufacturing workbenches.

From Start Menu
 

  1. Open a V5 session.

  2. Select Start > Machining Simulation > NC Machine Tool Simulation.

 

From Another NC Manufacturing Workbench

 

  1. From the NC Output Management toolbar, click Start NC Machine Tool Simulation .

  2. When you are finished using the workbench's features, click Exit NC Machine Tool Simulation, so that you can return to your previous workbench.
