Analyzing Holes 

You can analyze all the holes in a part. You can also use specific filters for the analysis.

Open an existing 3D shape with different types of holes. 

  1. From the Analysis option, select Hole Analysis . 

  1. In the Hole Analysis dialog box, select the required type for the holes.

    • Simple 
    • Tapered 
    • Counter Bored 
    • Counter Sunk 
    • Counter Drilled

  2. To filter the holes by the diameter and depth range, do the following.

    1. Select the Main Diameter Range check box and specify the minimum and maximum diameter for the holes.
    2. Select the Main Depth Range check box and specify the minimum and maximum depth for the holes.
     
    Holes Found displays the total number of holes. 

  3. Click Analyze.

    The analysis is shown in the table.
    The table inludes the following columns:
    • Type
    • Name
    • Main Diameter
    • Thread/Geometric Description
    • Main Depth
    • Extension
      You can change the filters and click Analyze again to generate a new analysis. 

  4. Click Close.
