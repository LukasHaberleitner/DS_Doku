Working with Dimension Systems

You can insert a dimension into a dimension system, remove a dimension from a dimension system or transfer a dimension to another dimension system using dedicated contextual menu commands. You can do so by grouping, ungrouping, or regrouping the dimensions and their systems, in new or existing systems.
This task shows you how to:
  • Group Dimensions
  • Regroup Dimensions
  • Ungroup Dimensions
   
 

Group Dimensions
  You can group two or more dimensions or dimension systems to form a new dimension system. You can also group a mixed selection of one or more dimensions and one or more dimension systems, if they fit the criteria.
Create two or more dimensions, or dimension systems that satisfy the following conditions :
  • They are of the same type (linear or angular)
  • They are in the same view
  • They can form a chained, stacked, or cumulative dimension system

  1. Select the dimensions that need to be grouped using Ctrl key.

  2. Right-click any of the selected dimensions and select Selected objects > Group Dimensions.
    The dimensions are grouped into a suitable dimension system.
    Dimensions form dimension systems based on the following criteria:
     

    • If two or more dimensions can be connected end to end to form a chain, they are grouped into a chained dimension system.

    • If two or more non-cumulative dimensions have a common dimensional element amongst them, they are grouped into a stacked dimension system.

    • If two or more cumulative dimensions have a common dimensional element amongst them, they are grouped into a cumulative dimension system.
      Other important things:
    • You can also group two or more compatible dimension systems using Group Dimensions. In this case, all the dimensions of the selected systems are grouped to form a new dimension system, using the current style. Previous systems, are emptied and deleted.
    • If the selection contains only all dimensions of one system, then this command is not available.

    • If a dimension is already a part of an existing system, it is removed from this system and grouped with the currently selected dimensions or systems to form a new system.

    • If an existing system no more contains any elements because of a grouping operation, the system is deleted.

 

Regroup Dimensions
You can regroup dimensions with the addition of individual dimensions to an existing system or transfer of dimensions that are already a part of one dimension system to another dimension system.
Create two or more dimensions, or dimension systems that satisfy the following conditions. :
  • They are of the same type (linear or angular)
  • They are in the same view.

  1. Select the dimensions or the dimension systems that needs to be regrouped.
    Ensure that Dimension system selection mode is selected when you select the dimension system.

  2. Right-click and select Selected objects > Regroup Dimensions.
    Dimensions form dimension systems based on the following criteria:
     
    • If the system is of chained type, then any one of its dimensions should have exactly one dimensional element same as an element of the dimension to be added.

    • If the system is of cumulative or stacked type, then the dimension to be added must have one of its dimensional elements same as the common element of the system.

    • If the system is of cumulative type, then dimension to be added also must be of cumulative type.
      You can regroup the dimensions which are already a part of another dimension system. In this case, the dimensions are removed from their previous dimension system and then get added to the new one.

  3. Select one or more dimensions or dimension systems to be grouped into the selected dimension system.
    The selected dimension systems is regrouped.
 

Ungroup Dimensions
You can ungroup dimension systems with the removal of individual dimensions from an existing system and deletion of an entire dimension system.
Create a dimension system with two or more dimensions.

  1. To ungroup an entire dimension system:
    1. Select the dimension system.
    2. Right -click and select Dimension System.x object > Ungroup Dimension(s).
      The dimension system is ungrouped and deleted. Each dimension is now treated as a separate element.

  2. To ungroup a one or more dimensions from a dimension system:
    1. Clear the Dimension system selection mode selection.
    2. In the dimension system, select the dimension that needs to be ungrouped.
    3. Right-click and select Dimension.x object > Ungroup Dimension(s).
      The selected dimension is ungrouped. The rest of the dimension system is still as it is.
