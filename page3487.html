Editing Rows in the Stacking Management

This task shows you how to edit several rows in one shot in the Stacking Management.
Not available in DMU Composites Review.
Open a document with a stacking.
 

  1. Select or filter rows in the Stacking Management dialog box.
    In the Tools Palette, click Multiple Row Edition

    The Rows Edition dialog box opens.

    • If all the children of the selected entities have the same current attribute, it is displayed in the corresponding list.
    • Otherwise, the corresponding list is left blank.
    • Modifications on a current entity applies to all its children.

  2. Edit the properties and other values as required.
    Only non filtered attributes are available and can be edited.

  3. For a plies group, edit the Name, enter a Start Index, a Step and the Nb of Digits to rename its contents.
    Other properties are not editable.

  4. For plies or sequences, edit the Material, Orientation, Rosette and Draping, or rename them.

  5. For cores, you can edit the Material, Orientation, Rosette.
    Only the draping of variable cores can be modified.

  6. For cut-pieces, edit the Rosette or rename them.
    Other properties are not editable.
