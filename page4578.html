KWECheck  
Definition:

Expert Checks are features generated by the Knowledge Expert product. Checks are regrouped into rule sets. Rule sets belong to a rule base. When writing a script with checks you must comply with the Rule Base/Rule Set hierarchy. See the Knowledge Expert User's Guide for more information on the concepts behind the expert rules and checks.
