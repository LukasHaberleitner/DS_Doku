Circles, Ellipses, Nurbs

 

When using vector formats, discrepancies related to the tessellation level may appear between the display on screen and the printed documents.

The tessellation level for 2D geometric primitives is set in the 2D Accuracy option which lets you optimize the rendering speed or the display quality. This parameter is not taken into account when printing documents in vector format because it only affects the display. It is available when viewing or previewing documents, not when printing.
