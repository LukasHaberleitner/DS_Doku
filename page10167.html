CatPartSurfaceElementsLocation (Enumeration)

enum CatPartSurfaceElementsLocation {
  catPartBodyLocation,
  catXGSLocation
}
Wireframe and surface elements location parameter's possible values.
Role: This enum is used in the interface.
Values:
catPartBodyLocation
Wireframe and surface elements are created within a PartBody
catXGSLocation
Wireframe and surface elements are created within a G.S. or an O.G.S..
