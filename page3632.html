Finding And Replacing Text

This task explains first how to locate a string of characters and then how to replace it.

Strings can be found and replaced in the following elements:

  • Balloons
  • Datum features
  • Datum targets
  • Dimensions
  • Texts
  • Flag Note
  • Non-semantic Datum Feature
  • Non-semantic Datum Target
  • Non-semantic Dimension (except the value)
  • Weld
Open the IntDrafting_Text_Replace.CATDrawing document.

 
1. Select Edit > Find or Replace from the menu bar.
To directly access the Replace dialog box, press Ctrl+H. 

2. In the Find what box, enter a text to search.

3. Click Find Next.

The following message appears in the dialog box: Searching All Current Sheet Views. If you previously selected a given number of sheets or elements in the document, the message will be Searching All Current Elements.

The first instance is found and highlighted.
4. If needed, click Find Next to search for other instances.
The searched string is highlighted with the color specified for selection through Tools > Options > Display > Visualization > Colors > Selected elements.		  

6. To replace the searched text string with another, enter a replacement text and click Replace.

To replace all instances of the text, click Replace All.

You can also match case, find whole words only, or re-frame the window.

7. Select Close.
 

You can replace the normal text string from any linked text. For example, from the above image, ABC can be replaced.

You cannot replace the following:
  •  An attribute link (red color)
  • A string in a locked annotation
  • A semantic annotation
  • A coordination dimension
  • A non-semantic dimension value
