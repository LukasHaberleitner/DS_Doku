Creating the Join

You are going to create a join with a merging distance of 0.01 mm.

  1. Recall all the bodies except Self-Intersection_0.01.1 from the No Show, i.e. make all valid surfaces visible.

  2. Insert a new Geometrical Set and make it the Define In Work Object
    (no need to enter a name, it is automatically called Geometrical Set.2).

  3. Select the whole part and click Join in the Repair Topology toolbar.
    In the dialog box, enter 0.01mm in the Merging distance box and clear the Check connexity check box.

  4. Click OK. Join.1 is created under Geometrical Set.2.
