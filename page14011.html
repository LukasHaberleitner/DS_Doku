Configuring the Links Display

 

This page shows you how to configure the links of the documents in SmarTeam by means of the Tree Properties contextual command.

  1. Select your document in SmarTeam.

  2. Right-click Tree Properties.

  3. Click the Visual Setting tab.
    This tab enables you to adjust the background color, line color and size, font type and size as well as the zoom factor of the displayed tree.

  4. Click on Links.
    This dialog box appears.

  5. You just need to check the classes you wish to display.

  6. Click OK to validate and close the dialog box.
