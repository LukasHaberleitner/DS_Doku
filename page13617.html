Getting Started

This tutorial will guide you step-by-step through your first Space Analysis session, allowing you to get acquainted with the product.

You will need a DMU Navigator Version 5 session and should be familiar with basic concepts such as document windows, standard and view toolbars.

You should be able to complete this tutorial in about 15 minutes.
