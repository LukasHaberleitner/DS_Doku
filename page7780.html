Core Extraction

In this task, you are going to use the Core Extraction command to create a Core Extraction feature. The Core Extraction feature will be placed into a new Body, and will reference an existing Body (the source Body).

The Core Extraction feature produces geometry useful in defining a core plate for molding the source Body. The output is associative, meaning if the source Body changes, the Core Extraction geometry will automatically change.

The output geometry includes the Core volumes and Interior volumes from the source Body. Features can be excluded, and an optional Extraction Properties feature can be used to include Protected volumes. The output geometry behaves as an Added volume.
Open the CoreExtraction1.CATPart document.

 

 

  1. Click Core Extraction  icon .The Extraction Core dialog box is displayed. Select Body.1 as the Source Body.

  2. Click Extract behavior list and select Extraction Properties.1.

  3. Click OK to confirm.

  4. Right-click Body.1 and select Hide/Show from the contextual menu.

Full edge Fillets and intersection fillets are automatically extracted during a core extraction. The extracted fillets are fillets belonging to features related to the core extraction, i.e. cutouts.
