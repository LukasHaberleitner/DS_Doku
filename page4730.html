Product

Description

Describes the product feature in the Product Structure workbench.

Inheritance path

Standard  >  Feature  >  ProductPackage  > Product.

Attributes

DefaultRepSource

Defines the source file of the representation.

Definition

Defines the product definition.

DescriptionInst

Defines the instance description.

DescriptionRef

Defines the reference description.

HasARepresentation

Defines whether the product has a representation or not.

Nomenclature

Defines the product nomenclature.

PartNumber

Defines the product part number.

Revision

Defines the product revision.

Source

Defines the product source.

LocationX

The coordinate X of the instance location relative to its parent (retrieved from the position matrix). This is a read-only attribute, you cannot modify it.

LocationY

The coordinate Y of the instance location relative to its parent (retrieved from the position matrix). This is a read-only attribute, you cannot modify it.

LocationZ

The coordinate Z of the instance location relative to its parent (retrieved from the position matrix). This is a read-only attribute, you cannot modify it.

  
 

Part

Description

Describes the part feature that derives from the product feature in the Product Structure workbench. It represents the terminal node in the product structure specification tree.

Inheritance path

Standard  >  Feature  >  ProductPackage  > Product.

  
 

Assembly

Description

Describes the assembly feature that derives from the product feature in the Product Structure workbench. It represents an intermediary node in the product structure specification tree.

Inheritance path

Standard  >  Feature  >  ProductPackage  > Product.
