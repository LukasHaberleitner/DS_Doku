Disabling or Enabling 3D Annotations 

This task shows you how to disable or enable the annotations you created. 

  1. Click List Annotation Set  Switch On/Switch Off  in the Visualization toolbar.
    The Annotation Set Switch On/Off dialog box is displayed.

  2. Click Part1 then the Disable All button.

  1. Click OK in the dialog box.
    The annotations are disabled in the geometry.

The specification tree no longer displays them.

  1. Click List Annotation Set  Switch On/Switch Off again to restore the previous state.

  1. Click Part1 then the Enable All button.
Whenever you create a new feature (such as annotation, view, dimension, constructed geometry, or restricted area), and if the features are disabled using the Annotation Set Switch On/Switch Off option, then all the hidden features are visible again.
