XML Mapping Properties

Only available with the ELFINI Structural Analysis (EST) product.

An XML mapping property is a property that applies physical properties to finite elements using an XML mapping file. The mapping file is an XML (Extensible Markup Language) file that contains finite element identification and associated property characteristics.

Two commands are available to let you apply an XML mapping property to a support:

  1. First, you can export an XML mapping file by taking into account the characteristics values of properties you have already defined. The exported file respects a particular syntax and format.
    To know more, refer to Exporting XML Mapping File.

  2. Then you can use the exported XML mapping file to create a mapping property.
    To know more, refer to Create Mapping Property.

At any time you can refer to Mapping File Syntax to know more about the syntax rules and the content of the XML mapping file.
