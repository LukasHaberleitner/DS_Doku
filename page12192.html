Object Properties

 
  • An object is created by default with some property values. These properties are defined or re-defined within the braces following the object declaration (isa keyword).
 
  • Unless otherwise specified, the units are SI units.
 
  • When defining properties, the semicolon ( ;) is a terminator (see example below).

Example 

  1. Select Tools > Options > General > Parameters, select Knowledge Environment and select Load extended language libraries and All packages.
  2. Open the PktObjectPropertiesScript.CATGScript and click Generate.
For more information about the objects, see Generative Script Objects.
