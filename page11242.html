About Visualization in 2D Layout for 3D Design  

This section provides information on visualization in 2D Layout for 3D Design.

In a 2D layout sheet or view, elements are visualized according to a priority which is based on the element type. Visualization priorities are the same in the 2D and 3D windows.

The following topics are discussed:

  • Front Elements
  • Rear Elements

Front Elements

The following elements are visualized at the front of a 2D layout sheet or view, whether they are located in the view or in its 2D or 3D background:

  • 2D layout elements
  • Functional Tolerancing and Annotation (FTA) elements
  • 3D wireframe

Within this category, elements are visualized according to the following logical order (from front to back):

  • 2D layout annotations
  • Geometry and 3D annotations
  • Area fills

Rear Elements

The following elements are visualized at the back of a 2D layout sheet or view:

  • Solids 
  • Surfaces
  • Surface and solid edges 

Within this category, elements are visualized according to the following logical order (from front to back):

  • 2D layout annotations
  • Geometry and 3D annotations
  • Area fills
Within the 2D window, the grid may be visualized at the front, that is, above solids, surfaces and their edges.
