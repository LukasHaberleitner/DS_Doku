Creating Annotation Texts

This task shows you how to create two textual annotations related to the 3D geometry of the part.
If you wish to improve the highlight of the related geometry, see Highlighting of the Related Geometry for 3D Annotation. All the screen captures have been performed according to this option.

  1. Click the face as shown to define the surface and the location for the arrow head of the leader line.

  2. Click Text with Leader  in the Annotations toolbar.
    The Text Editor dialog box is now displayed.

     

    The Tools Palette is displayed according to the type of face selected depending on the canonicity. In this scenario the Tools Palette is not used. For more information, refer to Propagating Geometry Selection for Feature Creation.

  3. Enter Surface treatment in the dialog box.
    Press Shift+Enter to add new line in the Text Editor if needed.

  1. Click OK or press Enter to end the text creation.
    The text is displayed in the 3D space in an annotation plane.

The textual annotation identified as Text.1 is added to the specification tree.
Note also that the application creates a Front View as shown in the tree. This view is automatically created when any view has been created yet or any existing view cannot be re-used.

Any front view created corresponds to an annotation plane (called "Front view annotation plane" too in the workbench).
If no annotation plane still exists, one is created by default when specifying the first annotation.

  1. To create another text annotation, select the inner cylindrical face:

  2. Click Text with Leader  in the Annotations toolbar.

  1. Enter Surface 2 in the dialog box
    Press Shift+Enter to add new line in the Text Editor if needed and click OK or press Enter.
    You have created a second textual annotation in the same front view.

    You can right-click the text with leader and select Switch to 3D Leader to make the 2D leader a 3D leader. For more information, see 3D leader.

    The diamond at the arrow end of the leader becomes parallel to the screen. Based on the preferences specified you can modify the position arrow end of the leader.
