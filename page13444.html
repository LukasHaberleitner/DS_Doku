Creating Tangent Walls

This task shows you how to create a wall tangent to a planar part of another wall or of a flange. This capability will then allow this tangent wall to be seen when unfolding the part, even though there is no bend linking it to its tangent support, provided this support can also be unfolded.

Open the NEWTangentWall1.CATPart document.

  1. Select a face of an existing wall and click Sketch .

    Here we selected the flange's planar face.

  2. Select Profile and sketch the profile as shown below.

  3. Using the Constraint Defined in Dialog Box , set coincidence constraints between the edges where the support and sketch are to coincide.

  4. Click Exit workbench to return to the 3D world.

  5. Make sure the sketch is still active, then click Wall in the Walls toolbar. The Wall Definition dialog box is displayed.

    Note the arrow which indicates the orientation of the wall to be created.

  6. Make sure the Tangent to field is active, and select the wall to which the new wall has to be tangent. Here, you need to select the planar face of the flange.

    Note that the orientation automatically changes to conform to the material orientation already defined on the support wall.

  7. Click OK.

    The wall is created and a Wall.xxx feature is added in the specification tree.

  8. Click Fold/Unfold in the Views toolbar (Fold/Unfold sub-toolbar).

    The tangent wall is unfolded as a wall linked by a bend to another wall, would be.

Click icon of the Tangent to selection field to display the Tangency connections dialog box describing the tangent faces or features connections. This capability allows you to connect a wall to several faces. In order to define the multi-connected wall, you can select in Tangent to field: top faces, bottom faces, and sheet metal features.

  • You cannot select a lateral face for the Tangent to selection.

  • The selected elements in Tangent to field must be in the same lump.

  • If you want to create the wall on a wall that cannot be unfolded, the system displays a warning message and prevents you from creating the tangent wall.
 

Creating a Tangent Wall on Edge
Open the NEWTangentWall1.CATPart document.

  1. Click Wall On Edge and select an edge of the wall.

    The wall on edge is previewed.

  2. Clear the With Bend option.

  3. Enter 180 deg in the Angle field.

  4. Click OK.
For further information concerning walls on edges, refer to Creating walls from an edge.
