Sculpting With Surfaces

This task shows how to modify a portion of a polygonal mesh with a NURBS surface.
Open the SurfaceSculpt1.CATPart document.

  1. Click Surface Sculpt .

  2. Press the Ctrl key and select four curves that form the closed contour of the region to be sculpted.

  3. Click on the polygonal mesh inside the curves.

    Control points are displayed as well as the Control Points dialog box.

  4. Sculpt the polygonal mesh using the Control Points tool.

    For further information, refer to the Editing Surfaces Using Control Points chapter in the FreeStyle Shaper, Optimizer, & Profiler documentation

  5. Click OK in the Control Points dialog box.
