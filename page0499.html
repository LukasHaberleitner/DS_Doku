Nonlinear Structural Analysis Workbench Description

This section contains an index to the icons and menus that are specific to the Nonlinear Structural Analysis workbench. The menu items are located along the top border of your analysis document. The icons are grouped in toolbars, which are located along the right side of the window. The following toolbars are available:

  • Analysis Connections Toolbar

  • Analysis Control Toolbar

  • Analysis Steps Toolbar

  • Assembly Properties Toolbar

  • Constraints Toolbar

  • Image Toolbar

  • Model Manager Toolbar

  • Nonlinear and Thermal Properties Toolbar

  • Postprocessing Toolbar

  • Prescribed Conditions Toolbar

  • Virtual Parts Toolbar

The final section, Specification Tree discusses the structure of the Nonlinear Structural Analysis specification tree.
