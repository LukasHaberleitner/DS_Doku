Creating a Layup

You can create a layup that consists of laminae, or layers, of materials. Each lamina, or layer, is defined by a CATIA V5 user material, a thickness, and an orientation. If the user material is a two-dimensional orthotropic material, you can apply the layup to a two-dimensional property enhancement to model the layers of a composite material, such as carbon fiber. See Creating a 2D Property Enhancement for more information.

Creating a Layup: Defines a layup of materials.

This task shows you how to create a layup of materials.

  1. Click the Layup icon .

    The Layup dialog box opens, and a Layup object appears in the specification tree under a Nonlinear and Thermal Properties feature. The data table for the layup appears in the top of the dialog box.

  2. You can change the layup identifier by editing the Name field. This name will be used in the specification tree.

  3. For each lamina, or layer, select the material name from the specification tree and enter the thickness and orientation in the data table cells.

  4. To add or delete table rows, click Add or Delete below the data table.

  5. To import layup data from a file, click the Folder icon , and select a text file.

  6. Click OK in the Layup dialog box to create the layup.
