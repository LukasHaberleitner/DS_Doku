Loading User Context

The Load User Context command allows you to load the selected user context in DPM.

  1. Select Load User Context  in the Process Context Builder toolbar.
    The User Context dialog box is displayed.

    The User Context dialog box presents the following options:

    Load Context Representation

    Load Context Objects

    This command searches the context blob stored on the loaded process or one of its parents and extracts all the User Contexts present in blob and displays them in Load Context Representation and Load Context Objects drop-down menus. The name of contexts in drop-down menu is exactly same as stored in the context BLOB. If there are no pre-computed user contexts/context representations available, then these options are disabled.

  2. Click OK.
    User context data can take awhile to load; a progress bar is provided.
