DELMIA D5 

This page describes how to set the Filter options of DELMIA D5 Integration.

Root Libraries

The Root Libraries section of the DELMIA D5 page displays the currently registered PROJECT$LIB entries, and is is used to indicate the directory locations where D5 Workcells, Devices and Parts exist.

  • Locations can be added to the Root Libraries list by clicking the Browse button and selecting the path location via a File dialog.

  • Entries listed in the Root Libraries list can be sorted within the list by selecting an item and clicking the Up and Down buttons to move the selected item to the desired location. This is helpful in optimizing the search for referenced files during a workcell or device import, as such a search is performed through these paths in the order in which they appear in the list.

  • Items can be removed from the Root Libraries list by selecting the desired item and clicking the Remove button .

Config files

The Config files section allows for the specification of DELMIA D5 config files to be imported.

  • Config files can be added to the list by clicking the Browse button and selecting the config file location via a File dialog.

  • Entries listed in the DELMIA D5 Config File Paths list can be sorted within the list by selecting an item and clicking the Up and Down buttons to move the selected item to the desired location.
  • Items can be removed from the Config files list by selecting the desired item and clicking the Remove button .

  • Settings imported from the config file(s) will take precedence over the root library defined in the Root Libraries list. 

PDB Cache Settings

The PDB Cache Path Settings section allows for the specification of a cache directory:

  • The cache path may be entered directly, or selected via the Browse button .

  • If a config file is imported, the PDB cache definition read from the config file will be used in place of this definition.

DELMIA D5 data import

The DELMIA D5 data import section allows you to specify which D5 attributes are imported into DELMIA V5. When an item listed above is enabled, that attribute will be imported.

DELMIA V5 frame

The DELMIA V5 frame section allows you to indicate whether Coorsys, ToolFrames and BaseFrames are imported and displayed when DELMIA D5 workcells, devices, and parts are brought into DELMIA V5. To import and apply D5 visibility settings, select Retain.

DELMIA D5 entities are mapped in V5 in accordance with the following table:

D5 Entity V5 Equivalent
Coorsys Design frame
UTool Tool frame
Tags in workcell Manufacturing frame
Device base frame Base frame

Tool Frame import ignores multiple UTools coinciding with the mount frame.
