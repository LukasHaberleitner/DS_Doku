Generating an NC Output File

This task explains how to interactively generate NC code from the program you have just created.

  1. Save your program with File/Save as ... in the directory of your choice (here we have a directory called models)

  2. Call your program MyGettingStarted. Click Save.

  3. Select Manufacturing Program.1 in the specifications tree then select the Generate NC Code Interactively icon .

  4. Call your file SurfaceMachining (the aptsource suffix is automatic).
    Browse to the directory where you want to save it
    (here we have chosen to put it in the same directory as the CATPRocess but this is not obligatory).

  5. Click Execute to create the APT file.

The APT file can be read with any kind of editor.
