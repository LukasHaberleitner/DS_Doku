Creating a Tool/Tool Assembly from Scratch

This task shows how to create tooling (tool and tool assembly) from scratch by using the Tool change command directly on the Resource list node.
1. Select the Resource list node in the specification tree.
2. Select the desired icon in the Auxiliary Operations toolbar (Tool Change sub-toolbar).

The Tooling Creation dialog box appears for defining the tooling.
3. Select the Tool tab . A default tool is proposed.

You can specify the tool parameters using the same procedure described in Edit a Tool in the Resource List.
4. Select the Tool Assembly tab . The page is empty except for the Name field.

Enter a name for the new tool assembly.
The geometric and technological parameter fields are then available for edition.
In addition, the tool defined in the Tool tab is automatically associated to the new tool assembly.

You can specify the tool assembly parameters using the same procedure described in Edit a Tool Assembly in the Resource List.
5. Click OK to create the tooling in the Resource List.
