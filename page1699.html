Information Behavior: Data Sheet

Accessing this behavior This behavior can be accessed by clicking the Information icon ().
Describing this behavior This behavior is designed to specify a message that will display when it executes. It is recommended to use it to display an error message for example. 
Using this Behavior Interface

 

Name: Name attributed to the behavior.
Comment: Comment attributed to the behavior.
Error Message: String corresponding to the message that will be displayed. 

 

Using this behavior When executing a behavior, the expert user finds out that an object was not published. The expert user uses this behavior to display a warning message for the end-user to fix the problem.
