Topology

This page deals with the options to allow you to control the elements to be selected:
  • Only input elements which meet the tolerances defined here will be selected.
  • If the input elements are outside of these values the commands supporting these Tolerances should indicate within the graphics which selected input elements do not meet these values.

Topology Creation

G0 (Point):
Where required all commands should check if the selected input geometry are below the value defined here.
If not the command should not produce a result and indicate the User which elements are not below the value.
By default, the value is set to 0.005mm.

G1 (Tangency):
Where required all commands should check if the selected input geometry are below the value defined here.
If not, the command should not produce a result and indicate to the User which elements are not below the value.
By default, the value is set to 0.5deg.

G2 (Curvature):
Where required all commands should check if the selected input geometry are below the value defined here.
If not the command should not produce a result and indicate the User which elements are not below the value.
By default, the value is set to 5mm.

G3 (Torsion):
(In ISD only)
Where required all commands should check if the selected input geometry are below the value defined here.
If not, the command should not produce a result and indicate to the User which elements are not below the value.
By default, the value is set to 15deg.
