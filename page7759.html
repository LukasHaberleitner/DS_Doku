Removing a Functional Set

This task shows you how to remove a Functional Set.

Removing a Functional Set from a Solid Functional Set does not affect the geometry of your design.
Open a CATPart document containing a Functional Set with two features.

  1. From the specification tree, right-click the Functional Set you want to remove and select Functional Set.x object  > Remove Functional Set.

    The Functional Set is removed. The features it contained are located in the Solid Functional Set above. The geometry is unchanged.
