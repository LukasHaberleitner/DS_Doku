Length Rule

This task explains the length design rule.

As of R10 the length design rule included with this application is only used by a sample Knowledge Expert rule. The Knowledge Expert rule, which you can insert into your document, checks that you do not exceed the maximum recommended length of a bendable tube or pipe. (See Using & Modifying Knowledgeware Rules.) Values for the attributes Minimum Straight Length,  Turn Angle and Standard Length are not used at present for any design checks.

This is a standard-specific design rule and the associated table is located in the directory ...\intel_a\Startup\EquipmentAndSystems\XXXXX\Standards, where XXXXX is the application name, such as Tubing.
1. The image below shows the length rule table for the Tubing Design product. Tables for other applications will have different entries but operate in the manner described below.

Column headings MaterialCategory, MinNominalSize and MaxNominalSize contain the input data. If you are adding an entry then enter the relevant input data in these columns. The columns MinNominalSize and MaxNominalSize represent a range. 

The column MaxLength contains the output data. Enter the relevant output data in this field. Data in the columns MinStraightLength, TurnAngle and StandardLength is not used for any rule as of R10.
