Selecting Faces Up To Boundary

This task shows you how to select faces up to a boundary.
 

  1. Click Up To Boundary Face Selection in  the Selection Tools toolbar.

  2. Select a face to initialize the propagation.

  3. Define the boundary of the selection, by selecting either edges of a surface or a wireframe.
    The selection is displayed in the work area.

  4. Optional: Activate Result As Selection.
    Useful to create a Generative Shape Design join or modify the visual properties of faces.
    No feature is created, the result is a selection of faces.

A Up to Boundary feature is created (Unless Result As Selection is selected).
It contains the total area of the feature.
