Simulation Toolbar

The Simulation toolbar has one pop-up toolbar associated with Machine Path Simulation : Process Simulation.  The Process Simulation toolbar has a sub-toolbar, which contains two options for saving stock.

The hyperlinks below link to the procedures that describe how to use the commands in the Simulation toolbar and its associated pop-up and sub-toolbars.

Simulation

Restore Initial State: see Saving and Restoring Initial State in the 3D Simulation for Manufacturing User's Guide.

Machine Simulation: see Simulating NC Tool Paths and Milling Operations and Viewing a Simulation
The procedure Viewing a Simulation refers to a Simulation Controls pop-up toolbar.  For NC Tool Machine Simulation, this pop-up toolbar does not appear. One of its commands (Visualization Options) is on the Process Simulation pop-up toolbar; the remainder do not apply.
