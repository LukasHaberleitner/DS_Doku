Areafill standard  patterns

 Areafill pattern CADAM Drafting areafill patterns are based on repeating lines or repeating cells.  The current Interactive Drafting product supports patterns based on repeating lines only. The CADAM standard patterns based on repeating lines (hatch, grid, steel, dashes, copper, rubber, aluminum 1, aluminum 2, titanium, plywood, plank, tile) are transferred to a pattern simulating the CADAM pattern. CADAM standard patterns based on repeating cells (brick, soil, hex, herring) are transferred as bitmap image patterns.
The bitmap image patterns for these are CCDPatternMotifBrick.bmp, CCDPatternMotifSoil.bmp, CCDPatternMotifHex.bmp, and CCDPatternMotifHerring.bmp. These files are located in the directory specified by the variable CATGraphicPath.

User defined areafill patterns

 Areafill pattern User defined CADAM areafill patterns are transferred as bitmap image patterns.
The CCD concrete pattern is provided as CCDPatternMotifUserDef_concrete.bmp. Bitmap image patterns must be created for any CADAM user defined patterns that are required. For example, a pattern named CCDPatternMotifUserDef_XXX.bmp must be created for the CADAM user defined pattern XXX. These files must be placed in the directory specified by the variable CATGraphicPath.
