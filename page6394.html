Creating Multiple Output Features  

You can create multiple output features from all the detected 2D profiles in the Multi Output Profiles dialog box. You can also apply a specific color to the output features created together.

Create a sketch with multiple 2D profiles.

 

  1. Click Multi Profile Features in the Tools toolbar.
    The Multi Output Profiles dialog box lists all the possible 2D profiles which can be used to create the output features.

  2. Select the required 2D profiles using one of the following methods:
    • Press Ctrl and select the required 2D profiles from the Geometry list.
    • Select the Select all check box to select all the detected 2D profiles from the Geometry list.
    The selected 2D profiles are highlighted in the work area.

  3. Select the appropriate color from the Color palette.
    The selected color is applied to all output features.

  4. Click anywhere outside the palette and click OK.
    Several output features are added to the tree.
Once created, you can edit the output features individually, but you cannot edit them together.
