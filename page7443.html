Setting Graphical Properties as Default 

This task shows you how to set graphical properties as default properties.
Note that the operating mode described here is also valid for datum, datum targets and geometrical tolerances.
Open the Annotations_Part_04.CATPart document.

  1. Select the text.

     

  2. Right-click and select the Set as Default contextual option.
    The Set as Default command records the graphical properties regardless of the selected option in the Style toolbar.
    Default graphical properties before the first use of the command are application default properties.
    See Text Graphical Properties.

  1. Select the Original Properties option in the Style toolbar.

  2. Create a new text and enter the following text: Original.

  3. Press Shift + Enter to add new line and enter new text Properties.

    See Creating an Annotation Text.
    The text appears in the geometry and takes only the application default properties. 

  4. Select the User Default Properties option in the Style toolbar.

    This functionality is not available with parts created with version V5 R11 and later, nor with parts created with older versions and whose standard has been updated or changed in V5 R11 and later.
    These parts use the styles which are defined in the standard used by the drawing.
    Standards are managed by the administrator.

  5. Create a new text and enter the following text: User Default.

  6. Press Shift + Enter to add new line and enter new text Properties.

    The text appears in the geometry and takes the application default properties and the Text: Line Spacing property from the first text.
