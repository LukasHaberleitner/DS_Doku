Using the Detailed Geometry Wizard

This task shows you how to easily navigate through the structure generated by a concept to detail link.
Open a part with a structure generated by a concept to detail link.

  1. Click Detailed Geometry Wizard  in the Method Plan toolbar.

  2. Click a node in the tree displayed in the dialog box.
    It is the same tree as the one displayed in the process part by Process Generator Wizard.
    • Only the geometry of the selected tools or step is displayed.
    • The corresponding design geometrical set is activated.

  3. Select the tool to design and start a command.
    The generated feature is automatically placed in the relevant geometrical set.

  4. Select a tool or a step geometry node in the tree.
    • Under Select Tool Detailed Geometry, select the resulting detailed geometry.
      The resulting detailed geometry is copied in the Detailed Geometry set of the selected tool or step geometry.
    • Use it to create a Detail to Result Link to update the process replica.

  5. Select the Geometry to Propagate set.
    Under Select Geometry to Propagate, select the geometries to propagate to another detailed part.
    The Detail to Detail Link associated with the current Concept to Detail Link makes a copy with link of the geometries to propagate.
