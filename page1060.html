MBOM Viewer Options
 

MBOM Viewer as 3D Viewer

Select the checkbox to display 3D objects in the MBOM Viewer window.

The default value is that the check box is not selected.

PPR Tree Options
 
When you activate this option, only the Manufacturing Assemblies that are assigned to other Manufacturing Assemblies are displayed. You may wish to hide these duplicate entries to display the hierarchically organized structures of Manufacturing Assemblies more clearly, especially if you have large structures that are assigned many processes and operations.

In practice, Manufacturing Assemblies often recur several times in different processing stages of a given production process. The hierarchical structure of the Manufacturing Assemblies takes into account multiple usage. In the example below, all manufacturing Assemblies Assy1 and Assy2 are on the same hierarchical level in the structure. The Manufacturing Assembly Assy1 additionally occurs a level lower in the structure as the child of Manufacturing Assembly Assy2.

If you activate the Hide duplicate entries for manufacturing assemblies option, the duplicate entry displayed for Manufacturing Assembly Assy1 is hidden and the Assy1 is only displayed in the structure of Manufacturing Assembly Assy2.

This hide option is unrelated to the standard Hide/Show command that hides or shows the 3D representations of Manufacturing Assemblies.

The default value is that the check box is not selected.

Show Fasteners
 
To show fasteners in the MBOM Viewer, select the fastener type under Show Fasteners. You can select several fastener types at the same time.

The default value is that none of the fastener types are selected.

Show Assigned Fasteners Only
 

To show only fasteners that are linked to the process in the MBOM Viewer, select the check box Show Assigned Fasteners Only. In addition, you always must select at least one fastener type under Show Fasteners.

Enabled

Fasteners assigned to the processes are displayed in the tree view if they are assigned to any part. They are shown in the graphic view if they are shown in the tree.

Parts assigned to processes with the relation types Process First Processes Product and Process Processes Product are shown in the tree view and in the graphics. Fasteners linked to such a part are shown automatically in the graphics but not in the tree.

Graphics
Spot Weld-10-0-0 and Part1 are shown in the graphics since Part1 is shown in the tree. Also all fasteners linked to Part1 (Spot Weld-10-0-0 and Spot Weld-0-0-10) are shown in the graphics.

MBOM of Operation2
Tree View
Shows the part assigned to Operation2, but does not show the fastener assigned to Operation2 because this fastener (Spot Weld-0-10-0) is not linked to any part.
Graphics
Only Part2 is shown. No fasteners are shown because there are no fasteners in BOM tree and there are not fasteners linked to Part2.

Disabled

All Fasteners assigned to the processes under consideration are shown in the tree view. They are shown in the graphics if they are shown in the tree.

Parts assigned to processes according to the rules are shown in the tree view and in the graphics. As is the case when the option is enabled, fasteners linked to such a part are shown automatically in the graphics but not in the tree.

The results are the same those when the Show assigned fasteners only check box is selected.

MBOM of Operation2

Graphics

The default value is that none of the fastener types are selected.
