General

Before you start your first working session, you can customize the way you work to suit your habits.
This type of customization is stored in permanent setting files: these settings will not be lost if you end your session.

  1. Select the Tools > Options.
    The Options dialog box opens.

  2. Select the General category in the left-hand box.
    A number of tabs letting you customize general options appear in the right-hand box.

  3. Select one of these tabs to customize General options or select a sub-category in the left-hand box to access specific options.

    General
    Help
    Shareable Products
    Licensing
    Document
    Macros
    PCS
    Printers
    Search
    Server Manager
    Statistics
    Display
    Compatibility
    Parameters and Measure
    Devices and Virtual Reality

  4. Set options in the tabs according to your needs.

  5. Click OK when done.
