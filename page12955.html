Browsing for a Variant  

This task shows you how to browse for a variant. The Browse Variant command allows you to easily and quickly activate and deactivate some material variants.
Open the Tube.CATProduct document. Follow the steps mentioned in Managing a Variant.

  1. Click Browse Variant in the Scene Editor toolbar. The Browse Variants dialog is displayed:

  2. Click Previous Aspect Variant or Next Aspect Variant in order to scroll through the list of available variants. A preview of the applied variant is displayed in the geometry.
    You can also use the Left Arrow and Right Arrow keys to scroll through the list of available variants.

  3. Click More. The dialog box with all the available variants is displayed:

  4. Double-click a variant name. The Define Shortcut dialog box is displayed. It allows you to assign a shortcut to a variant.

    The same shortcut can be assigned to different variants in order to activate several variants by a single click.

  5. Assign a character for the variant and click OK.

  6. Click OK in the Browse Variants dialog box.
Variants can be browsed in a DMU Presentation when creating a DMU Review.
