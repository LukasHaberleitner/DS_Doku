Preparing an Assembly for Analysis

Use the Assembly Design workbench to create an assembly from the required parts. Each part should include an analysis document containing the mesh and properties. You can also add other assemblies to the analysis, provided that all of the analysis documents for these subassemblies are defined on the parts. Any analysis information defined on the product level for a subassembly cannot be added to the analysis assembly. Likewise, none of the features that appear under a part analysis case in the specification tree can be used in the assembly analysis case.

Only one analysis document per part can be used in the assembly. If a part has more than one analysis document, make sure that the associated document is the one that you intend to use for the assembly. Each analysis document can be used only once in the assembly. To create an assembly containing multiple instances of the same part, you must create copies of the part and associated analysis documents for use in the assembly.

This task shows you how to create an analysis assembly.

  1. Open the Assembly Design workbench.

  2. Right-click on the Product container in the specification tree, and select Components>Existing Components.

    The File Selection dialog box appears.

  3. Select a .CATAnalysis file to add it to the assembly.

    Nonlinear Structural Analysis or Thermal Analysis adds the part associated with the analysis to the specification tree.

  4. Repeat Step 2 and Step 3 as needed to add all the required analysis documents to the assembly.

    You can add other assemblies to the analysis as long as the analysis documents for the assemblies are defined on each part, not on the product (assembly).

  5. Add any other parts to the assembly.

    You must create a mesh and add materials and other properties to parts that do not already have them.

  6. Open the Nonlinear Structural Analysis or Thermal Analysis workbench.

  7. Add the steps, boundary conditions, interactions, loads, element assignments, and other information required to complete the assembly model for analysis.

  8. Run the analysis.

  9. Attach the results.

    When you attempt to attach the results to the assembly analysis, Nonlinear Structural Analysis or Thermal Analysis displays a message indicating that the results cannot be attached to an assembly of analysis document, they must be attached to an empty analysis document.

    Click Yes to import the output database to a new, empty document.
