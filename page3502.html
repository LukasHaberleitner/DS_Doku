Displaying Information on the Fly

This task shows you how to display information on the fly on Composites elements and to apply selection filters.
Available in Composites Engineering Design (CPE) and Composites Design for Manufacturing (CPM).
Open any CATPart containing a Composites design.

  1. Click On the Fly Information in the Analysis toolbar.
    Stacking is automatically put in pickable mode.

  2. Place the cursor over an element in the specification tree or in the 3D Viewer.

  On the Fly Information is part of the following toolbar that provides filters to display information and select entities:
  • On the Fly Information:
    Displays information about the element under the cursor and selects it.
  • On the Fly Information for Plies Groups:
    Forces the selection of (and displays information about) the corresponding plies group.
  • On the Fly Information for Sequences:
    Forces the selection of (and displays information about) the corresponding sequence.
  • On the Fly Information for Plies or Inserts:
    Forces the selection of (and displays information about) the corresponding ply or insert.
  • On the Fly Information for Cut Pieces:
    Forces the selection of (and displays information about) the corresponding cut-piece.
  • On the Fly Information for Limit Contours:
    Forces the selection of (and displays information about) the limit contour that generated the selected edge.
  • On the Fly Information for Limit Contours Groups:
    Forces the selection of (and displays information about) the limit contour group corresponding to the limit contour that generated the selected edge.

  • If one contour is shared by several objects, the list of those objects is displayed, with one object highlighted. Use the up and down arrow to go from one object to another.
  • On the Fly Information provides the following information for ETBS:
    • Name of the ETBS
    • Name of the 2 zones connected by the ETBS with the number of layers, thickness, material and orientation information.
  • On the Fly Information provides the following information for core samplings:
    • Name of the CS
    • Number of layers, thickness, material and orientation information.
    • If there are several materials, On the Fly Information displays the list of materials with the orientations. However, the order of this list does not reflect that of the materials in the core sampling.
    • If the core sampling is not up-to-date, the core sampling is no longer valid, until you modify it. In that case, On the Fly Information displays only the name of the CS, followed by (Not Accurate).
