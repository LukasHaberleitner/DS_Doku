Setting Context Options

This task shows you how to automatically re-mesh the surface after modifying local specifications in the Surface Mesher workshop.

Open the Sample15.CATAnalysis document from the samples directory.

  • Enter the Surface Mesher workshop.
    For more details, refer to Entering the Surface Mesher Workshop.

  1. Click Element Distribution .

  2. Select an edge.

    In this particular example, select the following edge:

  3. Set the desired parameters in the Edit Elements Distribution dialog box.

    In this particular example:

    • Select Uniform as Type option.
    • Enter 8 as Number of elements value.

  4. Click OK.

    The mesh visualization disappears from the two impacted faces.

  5. Launch the mesh execution.

    To do this, click Mesh the Part .

  6. Click Context Options in the Execution toolbar.

    The Context Options dialog box appears.

    • Auto re-mesh on modification: if you select this option, the mesh operation will be launch after each modification of local specifications.

  7. Select the Auto re-mesh on modification check box.

  8. Click OK.

  9. Click the Element Distribution icon .

  10. Select an other edge.

    In this particular example, select the following edge:

  11. Set the desired parameters in the Edit Elements Distribution dialog box.

    In this particular example:

    • Select Uniform as Type option.
    • Enter 8 as Number of elements value.

  12. Click OK.
