Storing Output from Abaqus for CATIA V5 Analyses with Model Data Analyses

The Computation and Results Data directory setting in the Tools>Options dialog box specifies where Abaqus analysis jobs are actually run and is the location where their output is stored. You can now automatically specify that computation and results data be stored in the same location as the model data that you use in your analysis. This enhancement can make it easier to track all the files associated with an analysis.
