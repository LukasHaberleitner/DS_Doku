Performing a Radius Analysis

This task shows you how to find radii that are too small to be machined, and add an annotation requesting they are modified.
Open CircleAnalysisP1.CATPart from the samples directory.

  1. Click Radius Analysis     in the Analysis toolbar.
    If the check box Display detailed text in dialog in Tools/Options/Mechanical Design/Die Face Design is cleared,
    the dialog box looks like this:
     
    If the check box Display detailed text in dialog in Tools/Options/Mechanical Design/Die Face Design is selected,
    the dialog box looks like this:

  2. Select the Analysis Direction (stamping direction).

  3. Select the Element to Analyze (surface).

  4. Select the Radius Analysis Mode.
    • Concave
    • Convex
    • All.

  5. Select the Color Mode.
    • Detailed
    • Face/Edge
    • Smooth.

  6. Select the View Mode.
    • Mesh
    • Vectors
    • Planes
    • Dots
    • Acute/Obtuse Dots.

  7. Enter an Allowed Radius value.
    The minimum radius found is displayed.
    Areas strictly under the Allowed Radius value are shown in red.
    • Click one face in the area to review the area.
    • Pick a face and create an annotation to request designers to modify the radius of the area.

  8. Enter a Triangle Angle Filter value.
    Meshes may contain thin triangles that are very long and almost flat. As a consequence, some areas may appear with a radius under the Allowed Radius.
    • Triangle Angle Filter is applied only when a radius is found under the Allowed Radius.
    • If a radius under the Allowed Radius is also under Triangle Angle Filter, it does not appear in the result of the analysis.

  9. Activate more options.
    • Compute Areas
    • Fly Analysis.

  10. Set the Mesh Behavior.
    • Maximum Size is the step of the mesh. Its default value is 0. Modify it to have a more regular mesh.
    • Maximum Deviation is the mesh sag.
      Analyses in Die Face Design are done on meshed surfaces. Once the surfaces are meshed, they are shared by all analyses.
      However, Radius Analysis uses its own sag value to mesh the surfaces (minimum allowed sag/3). As a consequence, once you enter the Radius Analysis, existing shared meshes are modified, other analyses are updated.

  11. Click Display Mesh to verify the regularity of the mesh.

  12. Under Extract Geometry, convert the results of the displayed analysis into dead geometry.
    • Extract Areas stores the results as dead geometries and 3D labels.
    • Extract Direction extracts the Z-Axis on each analyzed point.
    • Extract Geometries as Points, Vectors or Boundaries
    • Extract Values as Tags or Points
    • Extract Points number using Full, Minima and Maxima or User Selection.

    An Analysis Set is created with the analysis parameters.
