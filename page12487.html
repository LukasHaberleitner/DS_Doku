Using the Search Command

The Search command in DPM allows you to search for PPR objects in a process document based on activity types and attributes. Searches based on the following attributes are supported: 

  • Name
  • Activity
  • Description
  • Specified Cycle Time
  • Specified Beginning Time
  • Calculated Cycle Time

The following example illustrates the use of the Search command.
