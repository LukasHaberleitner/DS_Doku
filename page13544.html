Sweeping the Top Surface

This task will show you how to sweep the top surface of the part and how to change a tool.

  1. In the Manufacturing view, select the machining area called Top from the list of areas. 

  2. Click Sweeping .

  3. In the Strategy tab, give a Maxi. distance between pass value of 1mm.

  4. Go to the Geometry tab and select the Part autolimit check box

  5. Go to the Tool tab. Click ... and select tool T1 End Mill D 10.

  6. Click Tool Path Replay to compute the tool path.

  7. Click OK to close the operation dialog box.

Now you are going to machine the sides of the part.
