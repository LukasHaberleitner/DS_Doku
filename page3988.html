Editing Geometry Orders

This task explains how to edit the geometry orders: along U for curve and along U and V for surfaces:
You can edit the order value:
  • By selecting a label indicating the order number, in this case you can only select an allowed order value.
  • By selecting a mesh line along the desired direction, in this case you can use increase or decrease commands, or select an allowed order value.
  Allowed order values are defined by the geometry and the maximum allowed order value set in the Tools>Options>Shape>FreeStyle, General tab, Geometry section, Order option.
Open the FreeStyle_Part_38.CATPart document.

  1. Click the Control Points icon:
    The Control Points dialog box appears.
     

  2. Select Surface.1.
    Control points and the mesh are displayed on the surface. Label indicating the orders along U and V are displayed.

  3. Right-click the V order label Nv:7 and select 10 from the contextual menu.
    The mesh and the V order label have been modified.

  4. Click the Mesh only icon from the Options frame:

  5. Right-click a mesh line along U as shown and select Order>Increase from the contextual menu.
    The mesh and the U order label have been modified.
