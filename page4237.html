About Creating Tolerances on Inspection Data

To add tolerancing information to inspection data, go to Start>Mechanical Design>Product Functional Tolerancing and Annotation.

All FTA annotation creation commands work with inspection geometry except the following dimensions command:

  • Dimensions
  • Cumulated Dimensions
  • Stacked Dimensions
  • Coordinate Dimensions
  • Curvilinear Dimensions
  • Basic Dimensions

The user can create semantic dimensions by using the Tolerance Advisor capabilities.
For a complete explanation of how to create tolerances on V5 data, see Functional Dimensioning and Tolerancing User Guide.
