Querying a Zone

This task shows how to query a zone for its members, or query a member to see which zone it belongs to.

  1. With your document open, click the Select/Query Zone button .

    The Select/Query Zone dialog box will display.

  2. Check Local under Scope if you want to see all zones in the document.
    Check All if you want to see all zones available to you.

  3. Use the Filter and Sort functions if you need to.

  4. To see which zone the boundary is part of, check Zone ID under Selection type and click the boundary. The zone will highlight, as shown above.
    If you click the zone, all boundaries that are part of it will highlight.

  5. To see which objects are part of the zone, check Zone ID Members and click the zone.

    The objects in the specification tree are highlighted accordingly.

  6. Click OK when you are done.
