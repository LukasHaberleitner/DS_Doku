Initializing the Environment for Direct Mode

 

  The Customizing section titled External Formats describes how to set the conversion environment interactively. 
   
  • In order to specify the SAG to use for cgr creation, set 3D Accuracy to Fixed in the tab Tools/Options/General/Display/Performances. In effect, even if 3D Accuracy is set to Proportional, V5 will use the value of the Fixed 3D Accuracy.
