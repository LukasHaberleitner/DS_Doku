Creating the Nearest or the Farthest Entity
of a Multiple Element

This task shows you how to create the nearest or the farthest entity of an element that is made up from several sub-elements.

Open the Near1.CATPart document.

  1. Select Insert > Operations > Near/Far from the menu bar.

    The Near/ Far Definition dialog box appears.

  2. Select one of the below options:
    • Near: allows you to create an entity nearest to the reference element.
    • Far: allows you to create an entity farthest from the reference element.

  3. Select the element that is made up from several sub-elements.

  4. Select a reference element whose position is close to/ far from the sub-element that you want to create.

  5. Click OK to create the element.
    The element (identified as Near.xxx /Far.xxx) is added to the specification tree.
