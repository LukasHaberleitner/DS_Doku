Saving a Design Table in ENOVIA V5 VPM Using VPM Navigator  
This task is designed to show you how to save a design table and its external file in ENOVIA using the VPM Navigator and how to use the external file in another file (it is possible to share a Excel file with other design tables). This scenario is divided into the following steps:
  • In CATIA, your create the design table using a pre-existing file.
  • You save the file in ENOVIA.
  • You create another part in CATIA, save it and associate the design table external file stored in ENOVIA.
For more information about the VPM Navigator, see the VPM Navigator User's Guide.
Before performing your scenario:
  • Set the ENOVIA V5 VPM option to Allowed in the Tools>Options>General>Document tab.
  • Enable Display in specification tree > Relations in the Tools>Options>Infrastructure>Part Infrastructure>Display tab.

Create the Design Table

  1. Open the KwrDesignTable.CATPart file.

  2. Click the Design Table icon () in the Knowledge tool bar. The Creation of a Design Table window is displayed.

  3. Click the Create a design table from a pre-existing file option and click OK. The File Selection dialog box opens.

  4. Select the KwrDesignTable1.xls file and click Open. Click Yes when prompted to associate the columns of the table with the parameters.

  5. Click OK to apply the default configuration. DesignTable.1 is displayed below the Relations node.

Save the Files in ENOVIA

  1. Save your files in ENOVIA.

    • Choose the Start > ENOVIA V5 VPM > VPM Navigator menu item.

    • Click the Connect user to ENOVIA V5 icon ( ) in the ENOVIA V5 toolbar. The ENOVIA Logon dialog box is displayed.

    • Enter your user name and your password.

    • Select your "role" in the Role scrolling list and the workbook and click OK.

    • Click the Save in ENOVIA icon (). The Save in Enovia V5 dialog box is displayed. Click OK to save the files. Close the file in CATIA.

'The KwrDesignTable1.xls file and the KwrDesignTable.CATPart file are now both saved in ENOVIA.

Apply the Design Table File Saved in ENOVIA to Another Part

  1. Open the New_DesignTable.CATPart  file. Click the Save data in Enovia V5 icon (). The Save in Enovia V5 dialog box is displayed. Click OK to save the file.

  2. Click the Design Table icon () to associate a design table with the part.

  3. Click the Create a design table from a pre-existing file option and click OK. The File Selection window is displayed as well as the browser.

  4. Click Cancel in the File Selection window and click Enovia in the browser.

  5. The Search Conditions window is displayed.

  6. In the Revision Owner field, enter the creator's name and click OK. The results are displayed in the Search Result window.

  7. Select the KwrDesignTable1.xls file and click Open.

  8. Click Yes when prompted to associate the columns of the table with the parameters.

  9. Click OK to apply the default configuration. DesignTable.1 is displayed below the Relations node.
