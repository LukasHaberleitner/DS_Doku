Filleting Intersection Edges

This task shows you how to fillet the edges at the intersection of the features using Part Design Edge Fillet .
Open the EdgeFillet01.CATPart and EdgeFillet02.CATPart documents.

  1. Click the Edge Fillet in Dress-Up Features toolbar (Fillets sub-toolbar).

    The Edge Fillet Definition dialog box is displayed.

  2. From the Selection mode field, choose the Intersection option you need among the following options:

    • Intersection with Current Solid
    • Intersection with Selected Features
    • Intersection with Core
    • Intersection with Cavity
    		         

    Whatever option you choose, keep in mind that you fillet features, not bodies. In case you need to fillet bodies, first perform a Boolean operation as explained in the Part Design User's Guide, in Associating Bodies.
 

Intersection with Current Solid

Perform this scenario in the EdgeFillet01.CATPart document.

  1. Set Intersection with Current Solid.
    Intersection with Current Solid     fillets all the edges produced by the intersection of the faces of the selected objects and the rest of the faces of the functional body.

  2. Select Shellable Prism Cylindrical as the feature from the specification tree which intersects with the current solid to be filleted.
    Shellable Prism Cylindrical is displayed in the Object(s) to fillet field and the geometry area.

  3. Click OK.
    The edges of the intersection between Shellable Prism Cylindrical  and the current solid are filleted.

     

Intersection with Selected Features

Perform this scenario in the EdgeFillet01.CATPart document.

  1. Set Intersection with selected features.
    Intersection with selected features     fillets all the edges produced by the intersection of the faces from the objects selected and the faces of the features you explicitly select.

  2. Select Shellable Prism Cylindrical as the feature which intersection with the features you want to be filleted.
    Once Intersection with selected features is on, the Selected features field appear to let you select the features you want to interact with the object to fillet.

  3. Click the Selected features field and select Shellable Prism.2.

    Shellable Prism Cylindrical is displayed in the Object(s) to fillet field and the geometry area.

  4. Click OK.
    The edges corresponding to the intersection of Shellable Prism Cylindrical with Shellable Prism.2 are filleted.

  5. Double-click EdgeFillet.1 in the specification tree to edit the specifications: click the Selected Features field and select Shellable Prism.1.

  6. Click OK.

Intersection with Core

Perform this scenario in the EdgeFillet02.CATPart document.

  1. Set Intersection with core.
    Intersection with core     fillets all the intersection edges between the faces from the objects selected -displayed in Object(s) to fillet- and the faces from the Core (internal shape) of the solid.

  2. In the EdgeFillet02.CATPart document, select Added Prism.2 as the feature which intersection with the core is to be filleted. In this mode, a new multi selector appears to select the features which interact with the object to fillet.
    Added Prism.2  is displayed in the Object(s) to fillet field and the geometry area.

  3. Click OK.
    The edges corresponding to the intersection of Added Prism.2  with the core of the body are filleted.

Intersection with Cavity

Perform this scenario in the EdgeFillet02.CATPart document.

  1. Set Intersection with cavity.
    Intersection with cavity     fillets all the intersection edges between the faces from the objects selected -displayed in Object(s) to fillet- and the faces from the Cavity (external shape) of the solid.

  2. Select Added Prism.2 as the feature which intersection with the cavity is to be filleted. In this mode, a new multi selector appears to select the features which interact with the object to fillet.
    Added Prism.2  is displayed in the Object(s) to fillet field and the geometry area.

  3. Click OK.
    The edges corresponding to the intersection of Added Prism.2  with the cavity of the body are filleted.
You can apply relevant intersection fillets (with Core, Cavity or Core/Cavity) for the Added, Shellable, Internal, External, Core, Cutout, Protected, Push, Pull, Pocket, Boss, Rib, Rest, Grill and Reinforcement functional features in the Fillet tab.
