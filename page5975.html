Propagating View Modifications  

This task shows you how to propagate view modifications to other views.

The synchronize view definition command helps to remove inconsistencies that can occur in a drawing between two views, when one view is inherited from the other or due to the non-propagation of view / section plane modifications.
Open the GenDrafting_part.CATPart document. Tile your windows vertically to see both your part and the related drawing.

  1. Create a front view of the part.

  2. Create a section view on the front view.

  3. Create a detail view on the section view.

  4. Modify the section profile in the front view, so that the segment goes through the other hole.
    The section view is updated, but the detail callout is displayed in red.

  5. Right-click the detail view and select Detail X object > Synchronize View Definition to synchronize the detail view with respect to the front view, which is the parent view.
    In this case we can see that the detail view is synchronized and is no longer shown with red callout.
  • Auxiliary, section and detail views can be synchronized.
  • Clipping, broken, breakout and 3D clipping operators can also be synchronized.
  • Annotations are also synchronized, as the view in which they are present is synchronized. However, in some cases rerouting or recreation of the annotation might be necessary.
  • Synchronization of section views with associative section profile is not possible. The section profile is designed to follow the 3D design of a particular object. Thus the view cannot follow the profile as well as the parent view support plane modifications. When both definitions become incompatible, the section callout becomes red. In this case, if synchronization is applied, error message is displayed explaining why the operation cannot be carried out.
  • View from 3D does not automatically update its support plane when the corresponding 3D definition is changed. Until now the Modify Projection Plane command was being used to update the 2D projection of a 3D front view. From R18, the synchronization of a 2D projection of a 3D Front view/Section view will be done using the Synchronize View Definition command.
    The Modify Projection Plane command does not work for synchronization of a 2D projection of a 3D View.
 

Synchronizing Views
Open the 3DViews.CATPart document.
Select the Synchronize during update check box in Tools > Options > Mechanical Design > Drafting > View tab, View from 3D area.

  1. Create a new drawing in ISO standard. Tile the screen horizontally by selecting Window > Tile Horizontally from the menu bar.

  2. Create a view from 3D on the FrontView.2.

  3. Create projection views.

  4. Change the definition plane of the FrontView.2 in the CATPart.
    In the CATDrawing, the drawing along with the sheet and views turn not-up-to-date as the view projection plane is not synchronized. Update icon appears in the specification tree to indicate the not-up-to-date status.

  5. Click Update in the Update toolbar or use the contextual menu command Update Selection to synchronize the views generated from 3D (Functional Tolerancing and Annotation workbench and 2D Layout for 3D Design workbench).
    You can see that the view from 3D is updated and synchronized, in this case     Front View.2.

  6. To synchronize the child views select Synchronize View Definition from the contextual menu. In the drawing specification tree or geometry, right-click Left view > Left view object > Synchronize View Definition. A progress bar is displayed showing the status of view synchronization.
    You can see that the left view is synchronized. Similarly synchronize the bottom view.
  • If the check box Synchronize during update using Tools > Options > Mechanical Design > Drafting > View tab, View from 3D area is cleared, then even after updating the sheet and/or views will not be synchronized.
  • If the Synchronize during update option is cleared, you have to manually synchronize views from 3D using the Synchronize View Definition contextual menu command.
  • While using Synchronize View Definition contextual menu command on view from 3D a message is displayed asking you if you want to synchronize the view projection plane. In case of other generated views, this message is not displayed.
  • For section view created using View From 3D, if there is a callout associated with the parent 2D view, the section callout has to be updated when the 3D plane changes. To update the section callout, right-click the section view in 2D, select Synchronize View Definition contextual menu command and then update the current sheet. After this the section callout on the parent view is updated.
