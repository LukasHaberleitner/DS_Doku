Selecting Core Samples in Stacking Management

This task shows you how to select core samples from the specification tree, the work area or the 2D Viewer.
Selection of core samples is taken into account also at Cut-Piece level.
Open a document with a stacking.
 

  1. In the work area, select a core sample.
    All associated plies are highlighted in the specification tree.
    All associated plies are highlighted in the 2D Viewer.

  2. In the 2D Viewer, select a core samplie in the dedicated column.
    All associated plies contours are highlighted in the work area.
    All associated plies are highlighted in the specifidation tree.
    All associated plies are highlighted in the 2D Viewer.

  3. In the specification tree, select a core sample.
    All associated plies contours are highlighted in the work area.
    All associated plies are highlighted in the specifidation tree.
    All associated plies are highlighted in the 2D Viewer.
