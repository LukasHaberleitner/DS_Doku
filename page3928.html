Using the Stretch View Analysis Tool

This task shows how to use the stretch view tool prior while analyzing a FreeStyle element.

Stretching means distorting the space view, without affecting the element itself. It is especially useful to examine precisely the curvature of an element from different viewpoints and with specific angles.
This stretching is done according to a ratio between the Y and X axes (Y/X) taking the screen as a reference into the active window.

This command is only available with the FreeStyle Optimizer and P2 mode.

Open the FreeStyle_Part_17.CATPart document.

To make it more explicit, we choose to open two windows (Window > New Window) and to display them horizontally (Window > Tile Horizontally).

  1. Select a surface and click  to display the control points on the surface in both windows.
    The Control Points dialog box is displayed.

  2. Click Stretch View .

The control points are grayed out, meaning you cannot use them for geometry modifications, and the Stretch View dialog box is displayed:

  1. Click Reset.
    The default value is taken into account (1.00). The view no longer is stretched.

You could also stretch a specific area of the screen, using a trap.

The center of the trap (cross) becomes the center of the screen.
Using the Ctrl-key, the trap center will be located where you click first.

  1. Click Close, the views remains, and you can continue to deform the surface as you wish, using the control points.
    The stretch View icon () is highlighted to indicate that the stretch view is activated.
  • Whatever the stretch options you choose, you can continue to use any other FreeStyle Shaper or Optimizer capability.
    For example, you could apply the Curvature analysis.

  • Even though the stretch is performed in reference to your screen (X is the horizontal axis and Y the vertical one), you can still manipulate the geometric elements as usual. You can rotate, move elements and so forth.
 If the Display Stretch check box under Tools > Options > Shape > FreeStyle > General > Display stretched or not is selected, a 'Stretch' label is displayed in the right bottom corner of the work area to indicate that the stretched view is activated.
