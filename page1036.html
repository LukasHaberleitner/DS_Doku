Mold Tooling Design

This section deals with Mold Tooling Design customizing options. 
These options can be reached via Tools/Options and are set out in the following tabs:
  • Catalogs, that defines the storage location of catalogs,
  • Add/Remove, that allows the association of add/remove bodies with receiving bodies,
  • Component, that defines the components options,
  • Miscellaneous, that defines the behavior of miscellaneous components,
  • Split, that defines the type of split to apply
  • Viewer Mapping, that defines the lists used in the Tool Viewer.
