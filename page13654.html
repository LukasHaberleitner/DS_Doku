Modifying and Deleting Copings

This task shows you how to modify and delete copings.

To learn how to create copings, see Coping Plates and Shapes.
1. With your document open, determine the copings you want to modify or delete. In the given example, a vertical stiffener is trimmed by a shell plate and and a deck longitudinal stiffener. Coping.1 and coping.2 are created.
2. To modify a coping:

With the parent product active, right-click the object with the coping. Select _xxxobject and Modify/Delete Coping. The Coping dialog box is displayed, listing all the copings created for the selected operated element.
3. Select and right-click the coping you want to modify. Select the new Nibbling Strategy. The new strategy is displayed in the Coping dialog box. Click OK.
4. If the new coping strategy results in an error, Error is displayed in the Coping column. No new coping is created. Right-click and select Refresh Coping. NONE is displayed. Right-click and select a valid coping strategy or click Cancel to revert to the previous strategy.

You can repeat these steps to modify other existing copings in the Coping dialog box. You can also select all the copings, right-click and assign the same Nibbling Strategy to all the copings.
5. To delete a coping:

Select _xxxobject and Modify/Delete Coping. The Coping dialog box is displayed, listing all the copings created for the selected operated element. Select and right-click the coping you want to delete. Select Delete Coping. NONE is displayed in the Coping column.

     

Click OK. The selected coping is deleted from the design and the specification tree. In the given example, Coping.2 is deleted.

You can repeat these steps to delete other existing copings in the Coping dialog box. You can also select and delete all the copings at the same time.
