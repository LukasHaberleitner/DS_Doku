Accessing CATIA V4 Comment Pages

 

This task will explain how you can access V4 comment pages in visualization mode.
You must have first generated a cgr file using the CATDMUUtility with the option -apps com. See Running the CATDMUUtility Batch Process.
 

  1. Load the model.

  2. Select the model and in the menu bar, select Edit  >  Properties.
    The Properties panel appears.


     

  3. In the lower-right corner, click the More button.
    The Comment tab appears and the comments are displayed.
