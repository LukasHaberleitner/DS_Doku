Assigning the Output Alignment

This procedure describes how to change the output alignment for one or a group of features. During output, all the features will be exported according to the assigned alignments.

  1. From the Thickness toolbar, select the Assign Output Alignments .

    The Assignment Output Alignments for Features dialog box appears.

  2. Select the features you wish to assign a new alignment, and then select an alignment from the list.

  3. Click the Assign button.
