Creating Wireframe Construction Elements

This task shows you how create wireframe construction elements using the vertices of solids.

  1. Click Line .

    The Line Definition dialog box appears.

  2. Create a line by selecting a vertex on Pad 1 and the corresponding vertex on Pad 2.

  3. Click OK. Repeat the above two steps to create four lines as shown in the figure below.
