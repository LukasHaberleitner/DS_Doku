Mechanical Surface Refinement in a Nutshell

The Mechanical Surface Refinement app provides a set of CATIA ICEM main functionalities for Generative Shape Design users which may be used to re-design visible mechanical parts from style point of view.
Therefore, a Feature toolbar, an Advanced Analysis toolbar, and the Tools Dashboard provide selected functionality available with FreeStyle Shape Design, ICEM Shape Design Center and ICEM Shape Design Expert.

Before Reading this Guide

Before reading this guide, you should be familiar with basic Version 5 concepts such as document windows, standard and view toolbars. Therefore, we recommend that you read the Infrastructure User's Guide that describes generic capabilities common to all Mechanical Surface Refinement products. It also describes the general layout of V5 and the interoperability between workbenches.

Getting the Most Out of this Guide

The Basic Tasks section deals with handling all the product functions.

The Workbench Description section, which describes the Mechanical Surface Refinement workbench, will also certainly prove useful to find your way around the Mechanical Surface Refinement workbench.

Navigating in the Split View mode is recommended. This mode offers a framed layout allowing direct access from the table of contents to the information.

Accessing Sample Documents

To perform the scenarios, sample documents are provided all along this documentation. For more information about this, refer to Accessing Sample Documents in the Infrastructure User's Guide.
