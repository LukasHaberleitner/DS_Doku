Entering the 3D Functional Tolerancing and Annotation Workbench

This task shows you how to enter the workbench and open the document you need for performing this tutorial.

  1. Select the Start > Mechanical Design > Functional Tolerancing and Annotation command to launch the workbench.
    The 3D Functional Tolerancing and Annotation workbench is opened.
    The commands are available in the toolbar to the right of the application window.

Update Warnings

You will get update warnings in a panel for FTA features which become or remain invalid after update. The warning panel will list all the FTA features such as Annotations, Dimensions, Restricted areas or Constructed Geometries that are invalid.

This behavior will be applicable only for Part FTA features. This behavior is by default disabled and can be enabled by activating dedicated FTA setting from the preferences panel. You must activate below FTA setting from the V5 Options:

Display update errors for invalid FTA features in a warnings panel check box  at Tools -> Options -> Mechanical Design -> Functional Tolerancing & Annotation -> Display ->.
