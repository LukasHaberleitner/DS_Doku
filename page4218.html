Exporting a Log File  

This procedure describes how to export information about activities into a CSV file.

The log lists all the features and their parameters associated with all the activities. This file will be used by the reporting application to generate automatically the roadmap documents (documents that describe the user the work that has to be done). Only information about defined features will be exported.
This procedure assumes that you have created a process.

  1. Click Export a Log File .

    The system provides a Save window, which allows you to navigate to the directory in which you want the CSV file saved.

  2. Navigate to the correct directory, provide a file name, and select Save.
