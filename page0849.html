VPM Navigator

Before you start your first working session, you can customize the way you work to suit your habits.
This type of customization is stored in permanent setting files. These settings will therefore not be lost if you end your session.

  1. Select the Tools>Options... command.
    The Options dialog box opens.

  2. Select the ENOVIA V5 VPM category in the Options tree on the left.
    The ENOVIA V5 VPM Logon, ENOVIA V5 VPM Options and VPM Impact Graph Properties tabs appear:

    The ENOVIA V5 VPM Logon tab lets you define:

    • the server hosts

    • the roles

    • the server size limit.

    The ENOVIA V5 VPM Options tab lets you define:

    • the lock options

    • the save options.

    The VPM Impact Graph Properties tab lets you define:

    • the Publication option

    • the relation type

    • the multiline mode.

  3. Click the VPM Search subcategory underneath the ENOVIA V5 VPM category.
    The VPM Search Properties  tab appears:

    The VPM Search Properties tab lets you define:

    • the results per page

    • the contents of the search result window.

  4. Click the VPM Navigator subcategory underneath the ENOVIA V5 VPM category.
    The VPM Navigator Properties  tab appears:

    The VPM Navigator Properties tab lets you define:

    • the start options

    • the merge mode

    • the background color

    • Publish/Subscribe

    • the expansion of the PRC at open

    • stable ordering

    • versioning

    • synchronization

    • object deletion

  5. Set options in these tabs according to your needs.

  6. Click OK when done.
