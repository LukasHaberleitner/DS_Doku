RRSSettingAtt (Object)

Interface to handle parameters of IGRIP RRS Tools Options Tab page Role: This interface is implemented by a component which represents the controller of IGRIP RRS Tools Options parameter settings.
  • Methods to set value of each parameter
  • Methods to get value of each parameter
  • Methods to get information on each parameter
  • Methods to lock/unlock value of each parameter
