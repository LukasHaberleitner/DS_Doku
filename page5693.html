Navigating in Examine Mode

 Examine Mode is the default navigation mode. You can examine your document as you would from the outside by moving around the document's perimeter, or as you would from within, modifying your orientation (turn head) or moving closer to different objects (zoom in, zoom out).

Note: When in Beginner's fly mode, click the Examine mode icon in the View toolbar to return to the default navigation mode.

For more information, see Activating Viewing Tools Using the Mouse in the Infrastructure User's Guide.
