Manikin Simulation Toolbar

This toolbar contains several commands belonging to the DMU Fitting Simulator 2 (FIT) product. When opening the Human Builder workbench, this toolbar will not appear unless the stated product is properly installed and a valid license exists for this product.

The hyperlinks below link to the procedures that describe how to use the commands in the Manikin Simulation toolbar and its associated sub-toolbars. The Manikin Simulation toolbar contains the following sub-toolbars:
