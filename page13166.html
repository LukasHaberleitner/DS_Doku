Customizing

Access Preferences

Before you start your first working session, you can customize the way you work to suit your habits.
This type of customization deals with permanent setting customization: these settings will not be lost if you end your session.

  1. Select the Tools > Options menu item.

    The Options dialog box appears.

  1. Click the Shape category in the left-hand box.

  2. Click the General Shape Design workbench
    The General tab is displayed.
 
  • The General tab lets you define the tolerant modeling, axes visualization groups, and stacked analysis options.
  • The Work On Support tab lets you define the work on support and the work on support 3D options.
 

  1. Set options in these tabs according to your needs.

  2. Click OK when done.

Save Default Values

You can define default values in standard files either by directly editing the corresponding Generative_Shape_Design.xml file or by using the standard editor panel available by selecting Tools > Standards. For more information on these methods, see CATIA Infrastructure User's Guide: Customizing: Customizing Standards: Customizing Standards for Layers and Filters. In administration mode you can also use the Save As Default command in the Tools toolbar.

  1. In the Tools toolbar, click Save As Default .
    The state of the dedicated options of the current command is saved in the xml standard file when closing the command.

    The following values can be managed with this mechanism:
    • Extrude: Limit 1, Limit 2
    • Cylinder: Radius, Length 1, Length 2
    • Revolve: Angle 1, Angle 2
    • Sphere: Part/Whole sphere mode
    • Circle: Part/Whole circle mode
    • Offset: Smoothing modes, regularization, maximum deviation
    • Sweep: Fill twisted areas, Compute C0 vertices as twisted areas, Canonical shape detection for non-canonic result, Canonical shape detection
    • Extrapolate: Maximal deviation imposed
