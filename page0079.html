Configuring Licensing for Abaqus for CATIA V5 Prerequisite (ACP) and Abaqus for CATIA V5

Abaqus for CATIA V5 Prerequisite (ACP) licensing and Abaqus for CATIA V5 licensing both use the FLEXnet network license manager or the Dassault Systï¿½mes License Server (DSLS). If you enable licensing for ACP, the license manager uses the same configuration to enable licensing for Abaqus for CATIA V5.

Note:  When you start a CATIA V5 session with ACP and Abaqus for CATIA V5 licensing enabled, an Abaqus for CATIA V5 license token and an ACP license token are consumed.

To configure licensing of Abaqus for CATIA V5 Prerequisite (ACP) and Abaqus for CATIA V5:

    This task explains how to configure licensing of ACP and Abaqus for CATIA V5.

  1. Start CATIA V5.

  2. Select Tools>Options from the menu bar.

    The Options dialog box appears.

  3. Select the General category in the tree to the left, then click the Licensing tab.

  4. From the list of available configurations or products, toggle on

    • ACC - SIMULIA - Abaqus for CATIA V5 Complete

      Note:  If you have a license for the CATIA Generative Part Structural Analysis (GPS) simulation product, you do not need an ACC license.

    • AFC - SIMULIA - Abaqus for CATIA V5

  5. Select the Analysis & Simulation category in the tree to the left, then click the Nonlinear and Thermal Licensing tab.

  6. Enter the name of the License Server. This option provides the name of the host computer running the license server for Abaqus for CATIA V5 Prerequisite, Abaqus for CATIA V5, and Abaqus analysis products. This option must be set the first time you run the product. You must restart your session whenever you change the license server setting for the change to take effect.

    If a single server is used, this option should be set to @license_server_host, where license_server_host is the name of the computer running the server. If redundant servers are used, the option should be set to a colon-separated list of the names of the server hosts and their ports. For example, if the host names for a set of three servers are maple, pine, and oak, the following definition is appropriate:

     27000@maple:27000@pine:27000@oak

    If the host computer running the Abaqus analysis license server is at a different location from the computer running the Abaqus for CATIA V5 Prerequisite and Abaqus for CATIA V5 license server, you should enter the name of the Abaqus analysis license server and its port in the colon-separated list. See SIMULIA products licensing in the SIMULIA Licensing Guide for more information on license servers for Abaqus products.

  7. Toggle on Use Abaqus teaching license for job submission to submit analyses using a teaching license instead of a commercial license if both are available.

  8. Restart your CATIA V5 session to account for the new license settings. You do not need to restart if your only change was to toggle a teaching license on or off.
