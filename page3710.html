Setting General Generative View Style Parameters

 

  In the DefaultGenerativeStyle.xml file, the Drafting node is devoted to customizing generative view styles for views generated from any 3D workbench in general. It contains the following sub-nodes:
  • Generate parameters: Specify whether the elements should be projected in the view.
  • ViewDressup parameters: Define the style of the various parameters which deal with the dress-up of the view.

See also:

  • Before You Begin Setting Generative View Style Parameters for more information on the structure of generative view styles and on how to customize generative view style parameters.
  • Setting Workbench-Specific Generative View Style Parameters to learn about customizing generative view styles for views generated from given workbenches.
