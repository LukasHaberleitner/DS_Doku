Distance and Band Analysis

 

About distance & band analysis: Gives general information on the Edit Distance and Band Analysis command.

Measure minimum distances and distances along X, Y, and Z: Click Distance and Band Analysis, define the type in the Edit Distance and Band Analysis dialog box, then select products to measure and click Apply.

Run band analysis: Click the Distance and Band Analysis, measure the minimum distance then select Band Analysis in the Type box. Define the band and set the accuracy then click Apply.

Export & publish distance results: Click the Export As and select the desired format.
