Mechanical

This reference describes the Mechanical properties concerning:
  • Constraints.
 
  This page deals with the properties concerning:
  • The Update Status.
 

Update Status
 
  Defines the constraint update status.
 

Deactivated
  Defines whether the constraint is deactivated.
Deactivated constraints are identified by brackets in the geometry and the in specification tree.
  By default, this option is not selected.
 

To Update
  Displays whether the constraint need to be updated, the constraint does not reflect the latest changes to the assembly.
To be updated constraints are identified by the update mask icon in specification tree.
  Not editable.
 

Unresolved
  Displays whether the constraint is unresolved, the application detects problems.
Unresolved constraints are identified by the update mask icon in specification tree and the color defined in the settings.
  Not editable.
