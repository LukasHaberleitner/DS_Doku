Creating a Convert Surface

You can create complex volumes from complex surfaces created with an other workbench.

You can choose between three Convert Surface types:

  • Free: the feature result is non-connected and is similar to existing hopper feature result.

  • Connected: the feature result is connected with the existing sheet metal features.

  • Fixed Edges: you can define the edges from the surface to keep them distortion free.

This task shows you how to:

  • Create a Connected Surface.
  • Create a Free Surface.
  • Create a Convert Surface with Fixed Edges.

For each convert surface types, you must create a complex surface with an other workbench.
 

Create a Connected Surface

You can connect a complex surface to a sheet metal feature.

Create a complex surface to be connected to a sheet metal feature as shown below.

  1. Click Convert Surface in the Rolled Walls toolbar.
    The Convert Surface Definition dialog box appears.

  2. In the Convert surface type area, select Connected.

  3. In the Selection box, select the surface you want to connect.

    The red arrow indicates in which direction the material will be created.

  4. Click OK to validate.

    The connected surface is created with the same sheet metal parameters defined for the wall.

  5. Click Fold/Unfold .

    The distortion is taken into account when unfolding the volume.

    The Connected type is available only if a surface is connected to a sheet metal feature. If you start only with a surface, the Connected type is disabled.
 

Create a Free Surface

You can create a sheet metal feature from a complex surface created with another workbench.

Create a complex surface as shown below.

  1. Click Convert Surface .
    The Convert Surface Definition dialog box appears.

  2. In the Convert surface type area, select Free.
    New options appear on the dialog box.

  3. In the Selection box, select a surface.
    The reference wire and invariant point are automatically selected and displayed in the Reference wire and Invariant point boxes respectively. You can change them.

    The red arrow indicates the direction in which material will be created.

  4. To offset the convert surface, type the desired value in the Offset box.
    By default, there is no offset and the value is set to 0mm.
    The offset of the convert surface is previewed.

  5. Click OK to validate.
    The volume is created.

  6. Click Fold/Unfold .
    The volume is unfolded according to the Reference wire and the Invariant point you have defined in the Convert Surface Definition dialog box.

     
 

Create a Convert Surface with Fixed Edges

You can create a sheet metal feature from a complex surface by determining a fixed edge for the Unfold mode.

Create a complex surface as shown below.

  1. Click Convert Surface .
    The Convert Surface Definition dialog box appears.

  2. In the Convert surface type area, select Fixed Edges.
    New options appear on the dialog box.

  3. In the Selection box, select a surface.

  4. In the Fixed edges box, select the edge you want to be fix when unfolding.
    In our example, we have selected the edge at the bottom of the surface.

    The selected edge must be planar.

  5. Click OK to validate.
    The volume is created.

  6. Click Fold/Unfold .
    The volume is unfolding with respect to the fixed edge you chose.
