Applying Annotation to Several Geometrical Elements 

This task shows you how to apply an existing annotation to several geometrical elements of a part by making it the default annotation.
Open the Annotations_Part_03.CATPart document:
  • Improve the highlight of the related geometry, see Highlighting of the Related Geometry for 3D Annotation.

  1. Select the Insert > Annotations > Annotation Application Mode > Apply separately several times .

  1. Select the Geometric Tolerance.1 annotation as shown:

    The Apply separately dialog box appears.

    • Manual Selection
      Contains the list of geometrical elements relevant to the selected annotation and their validity.
      By selecting a geometrical element this one is automatically added to the list.
      • Remove: Removes selected items from the list.
    • Automatic Selection
      Allows you set as default annotation the selected annotation to a predefined group of elements.
      • All Faces: Sets the selected annotation as default annotation for all faces of the part.
      • Planar faces: Sets the selected annotation as default annotation for planar faces of the part only.
      • Cylindrical faces: Sets the selected annotation as default annotation for cylindrical faces of the part only.
      • Spherical faces: Sets the selected annotation as default annotation for spherical faces of the part only.
      • Non Canonical faces: Sets the selected annotation as default annotation for non canonical faces of the part only.
      • Fillet faces: Sets the selected annotation as default annotation for fillet faces of the part only.
        • Radius: Defines whether the fillet radius criteria is taken into account in the fillet faces selection.
        • Value: Defines the radius value which is taken into account in the fillet faces selection.
          Only fillet faces with this radius value are selected.

  1. Select Automatic Selection in the Apply separately dialog box, All faces option in the combo box and click OK.

    The annotation is now applied to all the faces of the part.
