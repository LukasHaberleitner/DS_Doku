The collection of analysis case making an Analysis.

Sub Item( iIndex, oPositionning, oPointed)
Returns analysis support object information using its index in the collection.
Parameters:
iIndex
The numeric index is the rank in the collection. The index of the first case in the collection is 1, and the index of the last is Count.
oPositionning
The positionning feature is exists.
oPointed
