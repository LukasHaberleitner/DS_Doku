State Management  

State Management Options

Use this option to show or hide states, conditions, and positions in the PPR tree. When selected, Start Condition, End Condition and the 3D Context for activity and states and position for products or resource are visible in the PPR tree.

The visibility in the Applications Folder remains unchanged by this option. If the Applications Folder is displayed, then positions on the first level are displayed regardless if the new option is activated or not.

  • Conditions – Conditions can be defined for activities and for resources with behavior. Conditions can be:

    • Begin Condition

    • End Condition

    • 3D Context

  • States – States can be define for products and resources. A state can contain other states or positions.

  • Positions - Positions can be defined for products and resources.

By default, the check box is selected.
