Types of workshop document that can be extracted are:
  • Fabrication sketches. A fabrication sketch is a CATDrawing file representing the sketch of a part (flat or curved plate, straight, bent or rolled profile) ready for manufacturing and is used as input for marking and cutting operations. It contains dimensions, annotations and some identification data.
    For curved plates, the fabrication sketch is also used for forming operations and shows, in addition, roll lines, template location lines and reference lines.
    Note: The system automatically detects whether the plate is flat or curved and generates the CATDrawing with the appropriate data.
    For profiles, the fabrication sketch is a not-to-scale CATDrawing and shows two views of the profile.
  • Nesting data. This data is contained in a DXF file.
  • Template sketches for plates and shapes. A template sketch is a CATDrawing file with a number of sheets, each sheet representing the shape of one template. In addition for plates, it contains the base plane trace, the template curve, the sight plane trace and annotations.
  • Not-to-scale profile drawings. A CATDrawing showing a symbolic view of the straight profile.
  • Inverse bending curve data. This data is contained in an XML file.
  • Plate distortion data. This data is contained in an XML file.

CATDrawings can then, if desired, be edited using the V5 Drafting workbench, for which the appropriate license is required.

The Structure tab in the Properties dialog box lists all workshop documents associated with the selected activity.

Understanding Definition Points

Points like AnchorPoint_Start , AnchorPoint_End and AnchorPoint_Top are defined in the body.

 

The start and end geometries are defined based on the axis shown in blue.

When the view is placed, the start axis is matched with AnchorPoint_Start. The end axis is matched with AnchorPoint_End. AnchorPoint_Top is defined in order to locate cutouts on stiffeners.

For cutouts, two dimensions are placed onto the drawing to locate them:

  • Horizontal distance from the start end of profile
  • Vertical distance from MoldedFlange1 (the face in contact with the Plate).      

The image shows how cutout distance is computed using AnchorPoint_Top point.

Notes:

VP_GO
Global orientation point. This point is used to locate the global orientation axis 2.
VP_UR
Upper right point 3.
VP_LL
Lower left point 4. VP_LL and VP_UR points are used to define the rectangle inside which the view is placed.
