Creating a Mechanism and Revolute Joints

This task shows you how to create a mechanism and revolute joints.

Open the rods.CATProduct document.

  1. Select the product in the specification tree, then select Edit > Representations > Design Mode.
    Now, expand the tree to show all the design components of the product.

  2. Click Revolute Joint in the Kinematics Joints toolbar.
    The Joint creation : Revolute dialog box appears.

  3. Click New Mechanism. The Mechanism Creation dialog box appears.
    you can now enter a name of your choice.

  4. Click Ok.
    In our example, keep the default name Mechanism.1.


     

    The Mechanism is identified in the specification tree.

  5. Select Line 1 in the geometry area. In our example select a cylinder as shown below.

    Zoom in if necessary using the View > Zoom In Out menu and drag (left mouse button) to zoom in progressively.

  6. Select Line 2 in the geometry area. Select a second cylinder.

  7. Select the planes as shown below.
     

  8. Click Ok.
    The revolute joint is created and the specification tree is updated.
Repeat the operation to create Revolute. 2, Revolute. 3
