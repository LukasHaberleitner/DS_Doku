Modifying a Sheet

This task will show you how to modify the standard, format, orientation and/or scale of a sheet. Doing this amounts to modifying the options you selected in the New Drawing dialog box when defining the sheet.
Create a sheet using the ISO standard, the A0 ISO format, and the Landscape orientation.

  1. Select File > Page Setup.

  2. In the Page Setup dialog box, select the ANSI standard, and the A ANSI format.

    You can update the current standards by clicking the Update button. This copies the most recent version of the standard file in the drawing, thus reflecting the latest changes an administrator may have performed in the standard file.

    If no new standard file is available, the Update button is disabled.

  3. Select the Portrait orientation, and click OK to validate and exit the Page Setup dialog box.

    The sheet is modified.
At this step, you can also insert a background view into the sheet you are currently modifying. For more information, refer to Managing a Background View.
The Page Setup dialog box also let you modify the sheet format and set it to the printer format. For more information, refer to Printing a Sheet.
