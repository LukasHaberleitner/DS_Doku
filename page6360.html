Creating Constraints with the Background Elements

You can create constraints on current sketch elements with background geometry with reference to its intersection or projection with the current sketch plane. When creating a constraint based on the background edge, a projection of the edge on the sketch plane is computed, by default. If an intersection exists, it is possible to select this intersection instead of projection and vice versa.
Create two sketches that intersect with each other. For example:
Create a spline on the yz plane and create a circle on the zx plane.

  1. Click Constraint in the Constraint toolbar (Constraint sub-toolbar).

  1. Select the center of the circle.

  1. Select the spline.
    An offset constraint is created. The constraint is displayed in the tree and the graphic area.
    In this case, you can constrain the current sketch element with the background geometry by creating a projection or an intersection constraint.

  2. Right-click the sketch element and perform any one of the following actions depending on the applied constraint:

    • Perform the following actions if the constraint is applied with respect to the projection of the external geometry with the current sketch plane.
      • Select Sketch.x/Edge.x - Intersection.
      The result of the projection of the background edge is displayed in green dotted curve in the graphic area.
      • Click in the graphical area to validate the constraint.
        The constraint is created. The selected sketch element is now constrained with respect to the intersection of the external geometry with the current sketch plane.
    • Perform the following actions if the constraint is being applied with respect to the intersection of the external geometry with the current sketch plane.
      • Select Sketch.x/Edge.x - Projection.
      The result of the intersection of the background edge is displayed in yellow point in the graphic area.
      • Click in the graphical area to validate the constraint.
        The constraint is created. The selected sketch element is now constrained with respect to the projection of the external geometry onto the current sketch plane.
    • If a projection or intersection operation is not valid for the selected sketch elements with the external geometry, then, the respective contextual menu option does not appear for that particular mode.
      • The constraint options for projection and intersection are available in the contextual menu only if you have selected at least one external element to constrain with the sketch elements.
    • If you want to constrain three elements together, you can select the required projection or intersection constraint for both external elements, whichever is applicable with the current set of geometry.

  3. Optional: Right-click the sketch element and select Sketch.x/Edge.x - Intersection or Sketch.x/Edge.x - Projection depending upon the available option.
    The constraint applied earlier (projection to intersection and vice versa) is swapped between the projected or intersected element and the current sketch.
