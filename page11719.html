Getting Started

This tutorial provides an overview of NC Machine Tool Builder functionalities.  These tutorials rely on samples located in two different directories.  The procedures provide the paths relative to the software installation's startup directory and the documentation's online directory.

The tasks show different ways to load and manipulate the data used to build machine tools.
