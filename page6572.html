Defining Harness Flattening Parameters

This command enables you to specify harness flattening parameters. Other commands are deactivated as long as the harness flattening parameters have not been defined.

Open a new product document.

  1. Click Harness Flattening Parameters .

    The Harness Flattening Parameters dialog box opens on the General tab.

    Five different tabs are available:

    • General
    • Flattening Orientation
    • Flattening Scaling
    • Synchronization
    • Link
    • Drawing

  2. Select the tab in which you want to define/change the parameters.

  3. Click OK to validate the entries made.
