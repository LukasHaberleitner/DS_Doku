Replacing the Process Part

This task shows you how to replace the process part created with the wizard.
Activate a process part created with the wizard.

  1. Click Process Part Replace   in the Design Change toolbar.

  2. Click Step 1: Part Replace.

    All the inputs of the process, such as input parts and in-press elements, are listed under Element to Replace.

    • Select an Element to Replace.
      It is highlighted in the work area.
    • Select the replacing element.
    • Repeat as required.

  3. After all the features have been replaced, click Edit Nesting or Position to control and edit the part positioning.
    Standard part positioning commands are proposed.

 

  1. Click Step 2: Update Part Definition
    • From the list, verify the status of all the wireframe elements that define the flanges.
    • Click one element to display it in the work area.
    • Click Update Elements.
      An update is done. BReps are automatically relinked, feature orientations are verified.
    • If the update fails, click Edit Element to edit the corresponding wireframe element.
    • If flanges have become useless after a design change, select Remove useless intermediate geometries to delete them.
      The selected curve and its associated unfolded view and process tool are removed.

  2. Click Step 3: Process Wizard
    • Define new flanges
    • Edit the trim concept
    • Plan new elements.

  3. Click Tools.
    • Perform a Geometrical Relink.
    • Perform a Detailed Process Part Replace.
