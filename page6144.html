Creating Clouds/Points

 

This task will show you how to create clouds of points from CATIA points or CATIA points from clouds of points.
The count of active points in a cloud to transform into CATIA points may not exceed 1000.
Open the CloudPoint.CATPart model from the samples directory.

  1. Click Clouds / Points   in the Operations toolbar.
    The Cloud/Points dialog box is displayed.

  2. Select Cloud Primitives.1.
    Multi-selection is available.
    is available to hide or show the element selected.

  3. Click Apply, then OK to validate and exit the dialog box.
    The points are created in a Geometrical Set:

  4. Select the Geometrical Set you have created and click Clouds / Points   in the Operations toolbar.
    The Cloud/Points dialog box is displayed and is populated with the points found in the Geometrical Set.

  5. Click Apply, then OK to validate and exit the dialog box.
    The cloud of points is created in a Geometrical Set:
