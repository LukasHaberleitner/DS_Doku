Adding Rosettes

This task shows you how to add rosettes and specify their transfer type.
Alternatively, use Power Copies.

See More about Materials and Laminates for more information.
Available in Composites Engineering Design (CPE), Composites Design for Manufacturing (CPM) and Composites Braiding (CPB).
Open an empty CATPart in either Composites Design or Grid Composites Design.

  1. Click Composites Parameters  in the Parameters toolbar.
    Go to the Rosettes tab.

  2. Click Add Rosette.
    The Rosette Definition dialog box appears

  3. Optional: Edit the proposed name.

  4. Select an Axis system.
    A contextual menu is available to create one.

  5. Specify the Transfer Type.
    See About Rosette Transfer for more information.

  6. For 0ï¿½ guided by curve and 90ï¿½ guided by curve, select the guide curve (neutral fiber curve).
    A contextual menu is available to create one.

  7. Click Preview Rosette Transfer.
    See Previewing the Rosette for more information.

  8. Specify the display size of the rosette.
    Each rosette can have its own display size.
    • With Constant pixel size, the rosette display is independent from the zoom. It has always the same size, given in pixels.
    • Wich Contant model size, the rosette display follows the zoom, like any geometry.

  9. Click OK to validate and revert to the Composites Parameters dialog box.
    The created rosettes are listed in the dialog box, with their name, the name of the axis system they use and the type of transfer.
    They are created under Composites Parameters, under the Rosettes node, as Rosette.X.
    The transfert type is aggregated under each rosette.

  10. Select a rosette in the list and click Edit Rosette to edit it, or Remove Rosette to delete it.
