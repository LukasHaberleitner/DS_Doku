Extracting Data from a Cloud of Points

 

This  task shows how to extract the visible points from a cloud of points to create a new cloud of points.
You will find two examples below:
  • Extracting Data from a Cloud of Points,
  • Extracting Data from a Mesh.

The name of the output cloud is Extract (name-of-original_cloud).x
Open ExtractData01.CATPart from the samples directory.

Extracting Data from a Cloud of Points

  1. Click Information and select Carmirror1.1

    to retrieve its statistics:

  2. Click Filter and perform a filtering on the cloud:

    Check the statistics of the cloud:

  3. Click Extract Data in the Operations toolbar or
    right-click Carmirror1.1 in the specification tree and
    select Extract Data in the contextual menu.

    Extract (Carmirror1.1) is created in the specification tree with the visible points of the Carmirror1.1.

  4. Close ExtractData01.CATPart without saving it.
 

Extracting Data from a Mesh

  1. Re-open ExtractData01.CATPart and create a mesh.

    Hide the cloud of points.
    Check the statistics of the mesh:

  2. Activate a portion of the mesh and check its statistics:

  3. Click Extract Data in the Operations toolbar or
    right-click Carmirror1.1 in the specification tree and
    select Extract Data in the contextual menu.
    Extract (Mesh Creation.1).1 is created in the specification tree, with the visible points of the original mesh.
