Selecting Plies in Stacking Management

This task shows you how to select plies from the specification tree, the work area or the 2D Viewer.
Open a document with a stacking.
 

  1. In the work area:
    • At Ply level, select a ply contour.
      The ply 3D contour is highlighted in the work area.
      The selected ply is highlighted in the specification tree.
      The selected ply is highlighted in the 2D Viewer.
    • At Sequence level, select a ply contour.
      All 3D contours of the sequence are highlighted in the work area.
      The parent sequence of the selected ply is highlighted in the specification tree.
      The parent sequence of the selected ply is highlighted in the 2D Viewer.
    • At Plies Group level, select a ply contour.
      All the 3D contours of the plies group are hightlighted in the work area.
      The plies group that contains the selected ply is highlighted in the specification tree.
      The plies group that contains the selected ply is highlighted in the 2D Viewer.
    • Press Ctrl key or draw a trap and select several ply contours.
      All selected plies are highlighted in the work area.
      All selected plies are highlighted in the specification tree.
      All selected plies are highlighted in the 2D Viewer.

  2. In the specification tree, select a ply.
    The ply 3D contour is highlighted in the work area.
    The selected ply is highlighted in the specification tree.
    The selected ply is highlighted in the 2D Viewer.
