Setting GVS Parameters for Contour Maps

 

 

Refer to the Generative Drafting User's Guide to learn how to manage and customize generative view styles using the Standards Editor.

Contour lines annotations can be edited as any drafting entities.
Contour maps standards can be defined in Tools > Standards, Category:generativeparameters and File:DefaultGenerativeStyle.xml, under the node Cloud:

MajorZLevelCurve and MinorZLevelCurve refer to contour lines.
MajorZLevelAnnotation and MinorZLevelAnnotation refer to the level annotation on the major and minor contour lines.

Parameter   Value Description
HideZLevelInsideConstruction   Yes/No Defines visibility of contour lines under construction.
ShowPeaks   Yes/No Defines visibility of peaks.
MajorZLevelCurve
 MinorZLevelCurve		 Display Yes/No Determines if major or minor contour lines are extracted in drafting
(default value is same as in 3D).
  Color Color Defines the color of major or minor contour lines
(default value is same as in 3D).
  Linetype Integer Defines the line type of major or minor contour lines
(default value is same as in 3D).
  Thickness Integer Defines the thickness of major or minor contour lines
(default value is same as in 3D).
MajorZLevelAnnotation
MinorZLevelAnnotation		 Display Yes/No Determines if annotations on major or minor contour lines are generated in drafting
(default value is same as in 3D).
  Font > Name Font Defines the annotations font name (default value is same as in 3D).
  Font > SizeScale Integer Defines the multiple factor of annotations font size
(default value is same as in 3D).
The font size shown in 3D multiplied by SizeScale gives the font size shown in the drawing.
  Font > HideScans Yes/No Defines visibility of contour lines under annotations.
  Font > Color Color Defines the annotations font color
(default value is same as in 3D).
  Font > Bold Yes/No Specifies whether the annotations  font is bold or not
(default value = No).
  Font > Italic Yes/No Specifies whether the annotations font is italic or not
(default value = No).
  Frame Yes/No Specifies whether a frame is drawn around the annotation
(default value = No).
