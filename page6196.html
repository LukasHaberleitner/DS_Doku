MULTICAx SolidWorks (S) Plug-in in a Nutshell

The MULTICAx SolidWorks (S) Plug-in allows users to import:
  • SolidWorks native data (part and assembly models).
    This import converts  SolidWorks part geometry into the CATIA Graphical Representation (CGR) or CATPart formats, which renders it readable by V5.  The MULTICAx S Plug-in imports SolidWorks assembly files into a V5 product structure document. Users can perform this import interactively or from the command line.
  • Parasolid formats (*.x_t and *.x_b): parts only.
  • VDA-FS data (*.vda): parts only.
