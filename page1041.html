Miscellaneous

   This tab lets you decide of the behavior of the Technological Results and of the Tooling Interface.

Technological Results Management
Requires a license for Prismatic Machining Preparation Assistant.
When this check box is selected, technological results are automatically created in the component added and propagated into the drilled components.
When this check box is cleared, no technological results are created.

Tooling Interface Management

  • Read-only access control:
    manages control access: select this check box to activate the detection of read-only Tooling Interfaces.
    Read-only status is accessed and existence of needed Tooling Interface file is checked to display warning messages when necessary.
    By default, this check box is not selected and no control is made.
  • Control type:
    defines the type of control: available when Read-only access control is selected.
    • User: when you select this option, feedbacks are displayed to help you make the right decision.
    • Automatic: when you select this option, the system takes the necessary decisions to make the process as transparent as possible.
      This is the default option when the Control type is active.
