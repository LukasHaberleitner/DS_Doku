Using the Keyboard

Key Command Action
Shift Activate,
Remove (Pick or Brush),
Split a Mesh or a Cloud 		Deselects selected elements
Shift Align using the compass Moves the compass without moving the cloud
Shift 3D Curve Activates/de-activates the Snap on elements option
Ctrl Scan on Cloud Previews the scan being created
Ctrl Curve from Scans Moves the split points
Ctrl 3D Curve Projects a point on its constraining element
