Assigning Versions to Manufacturing Assemblies 

Licensing
  This command requires data loaded from the Manufacturing Hub.  Consequently, this command is not available to you if you do not have a license for the Manufacturing Hub.

This task describes how to create versions of manufacturing assemblies. Versions of manufacturing assemblies may be saved as you modify an assembly. Only the manufacturing assemblies that have been changed can be assigned a version.

  • New versions of manufacturing assemblies can only be created if processes have been imported from the Manufacturing Hub.
  • For information on how to import a project, please refer to Loading a PPR from Project.
  • For information on how to create a manufacturing assembly, please refer to Creating a Manufacturing Assembly.

Manufacturing assemblies enable you to create a structure of product assemblies that is suitable for production and that may differ from the design parts list. If changes are made to manufacturing assemblies, you can make new versions of:

  • manufacturing assemblies that have been changed directly (in the image below, MA3 represents changes directly made to the manufacturing assembly).
  • all manufacturing assemblies that are at least one level higher in the hierarchy (in the image below, MA2 represents a manufacturing assembly is indirectly assigned a version via the hierarchy).

Starting the Version Assignment of Manufacturing Assemblies

With the aid of versions, design changes made in product assemblies may be handled, which are to take effect in production as of a certain date. The version assignment allows to define validity for Manufacturing Assemblies, e. g. to determine at what time a Manufacturing Assembly is to be scheduled for production.

1.

 To open the dialog, click this icon or select the menu item Tools > MBOMTools > Version Manufacturing Product.

2.

 The Create Version of Manufacturing Assembly dialog displays the Manufacturing Assemblies that have been changed:
  • The left window of the dialog lists all Manufacturing Assemblies you have changed directly.
  • The right window lists the Manufacturing Assemblies that are indirectly affected by changes and which are located at least one level higher in the hierarchy of Manufacturing Assemblies.
  • You will create new versions of Manufacturing Assemblies that have been changed directly as well as for Manufacturing Assemblies which are indirectly affected.

Create a new version by selecting the Manufacturing Assembly. You may create a version for individual or for several Manufacturing Assemblies:

  • Either select an individual Manufacturing Assembly with a mouse click or select all Manufacturing Assemblies displayed by choosing Select All.

  • Upon selecting, you may start the version assignment by clicking OK.

  • Confirm the message by clicking OK; the new versions with be saved in the Manufacturing Hub.
