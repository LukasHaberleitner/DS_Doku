Digital Process for Manufacturing

Before you start your first working session, you can customize the way you work to suit your habits.
This type of customization is stored in permanent setting files: these settings will not be lost if you end your session.

  1. Select Tools > Options.
    The Options dialog box opens.

  2. Select the Digital Process for Manufacturing category in the left-hand box.
    A number of tabs letting you customize solution options appear in the right-hand box.

  3. Select one of these tabs to customize Digital Process for Manufacturing options or select a sub-category in the left-hand box to access specific workbench options.

    Digital Process for Manufacturing
    DPM - Fastening Process Planner
    DPM - Assembly Process Planner
    Manufacturing System Definition
    DPM - Work Instructions
    Commands
    DPM - Shop Assistant
    DPM - Shop Order Release
    DPM - Structure Lofting

  4. Set options in the tabs according to your needs.

  5. Click OK when done.
