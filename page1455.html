Recording Interactions in Video Format

 

The capture in Pixel Mode or Vector Mode and the recording of videos in Document Window mode are based on the current viewer in the current active window. Therefore:

  • To capture several viewers in a single window, use the Screen Mode

  • To record interactions from several viewers in a single window, use the Area mode.
