HTSWalkMotionBasis (Enumeration)

enum HTSWalkMotionBasis {
  WALKMOTBASIS_DEFAULT,
  WALKMOTBASIS_USERSPEED,
  WALKMOTBASIS_USERTIME
}
Walk Motion Basis Types used in setting attributes of
, activities. Note: AutoWalk is always a time based activity - it supports WALKMOTBASIS_USERSPEED or WALKMOTBASIS_USERTIME type
Values:
WALKMOTBASIS_DEFAULT
Speed based activity
WALKMOTBASIS_USERSPEED
Time based activity - with user given speed
WALKMOTBASIS_USERSPEED
Time based activity - with user given time
