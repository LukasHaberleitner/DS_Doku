Choosing the Options for Dimensioning Using Tolerancing Advisor

This task shows you how to set the options you might use for tolerancing your part using tolerancing advisor.
Open the Tolerancing_Annotations_15.CATPart document.

  1. Select the Tools > Options command.
    The Options dialog box appears.

  1. Click Mechanical Design then Functional Tolerancing and Annotation in the left-hand column.

  1. Set ISO_3D as the standard in the Tolerancing tab.

  1. Set the Leader associativity to the geometry field to Perpendicular.

  2. Select the Tolerances tab, and set Separator as . in the Geometrical Tolerance field.

  3. Check that the Dimension following the cursor (CTRL toggles) in the Dimension tab and the Annotation following the cursor (CTRL toggles) in the Annotation tab are not selected.

  4. Click OK to validate and close the dialog box.
Note that this choice of the tolerancing standard must be expressed prior to specifying any tolerance.
