Getting Started

Before getting into the detailed instructions for using Cast and Forged Part Optimizer Version 5, the following tutorial aims at giving you a feel as to what you can do with the product. It provides a step-by-step exercise based on a realistic design scenario showing you how to use key functionalities for creating a complex part.

The main tasks covered in this section are:

Accessing the Cast and Forged Part Optimizer Capabilities
Using the Sew Surface Capability
Using the Advanced Draft Capability
Filleting While Keeping an Edge
