Using the Historical Graph

This task shows how to use the Historical Graph.
Open any .CATPart document containing elements.

  1. Select the element for which you want to display the historical graph.

  2. Click Show Historical Graph .

    The Historical Graph dialog box appears.
    The following icon commands are available:
    • Add graph
    • Remove graph
    • Reframe graph
    • Surface or Part graph representation
    • Parameters filter
    • Constraints filter

  3. Just close the dialog box to exit this mode.
    The position and size of the Historical Graph dialog box is saved for next instance. Every time you click the Show Historical Graph command, the Historical Graph dialog box opens in the previously changed position and size.
