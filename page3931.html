Filtering Trimmed and Untrimmed Curves and Surfaces

This task shows you how to filter trimmed and untrimmed curve and surface. You can further use this selection to perform any other command.

Open the FreeStyle_Part_49.CATProduct document.

  1. Click Trimmed element Filter in the Custom selection filters toolbar.

  2. Select the curve and surface using trap selection or mouse click or from specification tree. You can see that the curve or surface that is trimmed is selected. The not selectable icon is seen in case an untrimmed curve or surface is selected using mouse selection or from specification tree.
    In this case only the two trimmed surfaces and trimmed curve are selected.

  3. Click Trimmed element Filter to deselect it.

  4. Click No Trimmed element Filter .

  5. Again select the curve and surface. You can see that only the untrimmed surfaces and curve are selected.

  6. Click required command icon to continue working on the selected curve or surface.
You cannot select trimmed volume using the commands.
