Creating Profiles

This task shows you how to create profiles on the boundary of a stamping part, to compute the addendum surface later on.
Open AddendumSurface01.CATPart from the samples directory.

  1. Click Profiles in the Addendum Design toolbar.

  2. Select an axis system (stamping direction) and the surface.

  3. Select a vertex.
    The Compass snaps to the selection.
    A contextual toolbar appears.
    The dialog box lists are updated.
    Multi-selection of profiles is available.

  4. Drag the compass to set the length or orient the profile.

  5. Alternatively, in the contextual toolbar define the direction:
    • Switches between X-Axis and Y-Axis directions.
    • Aligns the selected profiles in the X-Axis direction.
    • Aligns the selected profiles in the Y-Axis direction.
    • Makes the profile perpendicular to the stamping direction.
    • Creates or removes a tangency constraint with the selected surface.

  6. Still in the contextual toolbar:
    • Deletes the selected profiles.
    • Aligns the selected profiles in the same direction as a selected edge.
    • Inverts the orientation of the selected profiles.
    • Creates an addendum surface between the selected profiles.
    • Creates a simple profile.
    • Creates a top fillet profile.
    • Creates a boss profile.
    • Creates an user profile from a sketch.
    • Reuses an already created profile.
      Select Reuse Profile  and select the sketch to replace to current profile.
      By default, it is copied without link.
      For another copy option, go to Advanced in the Profile Parameters tab and select one from the list (Copy user sketch with formula, Copy user sketch with link)

  7. Go to the Profile Parameters tab to enter more dimensions.

  8. Click OK.
    The profile is created under a Profile Set.
