3D Annotation Properties

This reference describes the 3D Annotation properties concerning:
  • Text.
  • Flagnote.
  • Datum.
  • Target.
  • Geometrical tolerance.
  • Surface Texture.
  • Dimension.
  • Coordinates Dimension.
  • Note Object Attribute.
 
  This page deals with the properties concerning:
  • The D&T Feature Name.
  • The Standard Representation.
  • The Numerical Size Tolerances.
 
 

D&T Feature Name
Displays the name of the dimension & tolerancing relative to the annotation.
Not editable.
 

Standard Representation
  Displays the annotation representation according to the standard.
Not editable.
 

Numerical Size Tolerances
  Displays the annotation tolerances. Only available for toleranced dimension..
