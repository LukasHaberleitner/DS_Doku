Check Features 

Select the Check icon to create a check, write its code, tests its syntax and apply it to your document.

A check is a set of statements intended to inform you if certain conditions are fulfilled or not. A check does not modify the document it is applied to.  A check is a feature. In the document specification tree, it is displayed as a relation that can be activated and deactivated. Like any feature, a check can be manipulated from its contextual menu.
