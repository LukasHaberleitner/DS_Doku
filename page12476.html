Locating Where a Process Library is Stored

There are two methods for finding the location of where a Process Library is stored. These methods are described below.

Using the Add Activity Command
Selecting the Add Activity command allows the user to see where the related Process Libraries are stored:

Using the Desk Command
Library file locations can also be determined by using the Desk command located in the File menu, as shown below:

Using the Desk command opens a window which shows the Process Libraries that are linked to the current Process Document. Highlighting a Process Library name and selecting Edit>Properties invokes a dialog which displays the path for the selected library:

The Properties dialog above can also be accessed by right-clicking on the Process Library name. This will display a context menu from which "Properties" may be selected.
