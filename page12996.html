Glossary

A

attribute Characteristics of an object such as thickness, material, throw orientation. Also referred to as properties.

B

beaming system A functional system defined by two intersecting planes and two limits.
block scheme Determines the basic block definition of the ship.

C

chock A plate or assembly of plates used to reinforce a profile welded on a plate to avoid buckling
connecting system Establishes functional connectivity between structural elements. Typical examples are plate to plate connections, plate to stiffener connections and stiffener to stiffener connections. Physically implemented downstream by means of joints and piece parts (chocks, transitions, brackets, etc.).
cutout The removal of a portion of a plating part, usually along one of its edges. Typical cutouts are rat holes and snipes. See also structural openings.

D

detail design block A section of the ship generated by applying a block scheme block to the functional design. A block is a 3-D entity.
detail design view The design view that documents a ship's structure in the 'as built' or 'as assembled' stage.

E

end cut A cutting detail. 

F

feature dictionary The document in which object classes are managed.
free-edge stiffener A structural shape used to stiffen the free edge of plating parts (i.e. part boundaries that do not connect to any other part). Typical examples are faces on free edges and coamings.
functional molded form A functional surface derived from a conceptual surface. You can define more than one functional molded form from a conceptual surface.

I

inserting system A special type of plating system.

M

molded convention A set of rules that give default orientations when placing structural elements, thereby ensuring structural continuity. These conventions are described in an XML file reference in the Project Resource Management (PRM) file.

P

paneling system The top level structure detail design object. This object contains a molded form and is created automatically when you create a plating system. It groups all plating and stiffening systems having the same molded form.
piece part Typically, chocks, transitions, brackets, etc.
plate A plate part is a part cut from flat raw material stock, which may later be rolled or bent to a non-planar shape.
plating system A functional system having a molded surface as support and defined by selecting limits, or limits and a free edge.
project resource Resources, like the feature dictionary, managed by system administrators in the Project Resource Management file.
property Characteristics of an object such as thickness, material, throw orientation. Also referred to as attributes.

S

section Describes the shape representation of a stiffening part.
slot A hole allowing penetration of another part.
standard part Connections and joints. Typical examples are brackets.
stiffener A structural shape used to stiffen plating parts. Shapes are usually welded to parts they stiffen. Cross-sections can be selected from a catalog.
stiffening system A functional system defined by a web support and limits.
structural openings An opening in a host part to reduce weight, provide accessibility or allow penetration of another part. Typical openings are lightening holes, access holes and penetration holes. 

T

throw orientation See molded convention.
trace Defines the path of a structural shape.

U

UDF User Defined Feature (UDF) is a set of features that are grouped in order to be used in a different context, and present the ability to be completely redefined.
