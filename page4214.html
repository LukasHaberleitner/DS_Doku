Activating an Alignment

This procedure describes how to activate an alignment.
To activate an alignment, you must have at least one alignment defined.

  1. Select the parent process or preceding activity for the Activate Alignment activity. 

  2. On the Inspection Activities toolbar, click Activate Alignment.

    The Activate Alignment dialog box appears.
    The alignment list that appears contains the alignments you have previously defined.

  3. Select the point coordinate system (PCS) you want to activate and then select the OK button.
If you create this activity before you insert a CMM into the resource list of the PPR tree, you will be asked to confirm that you wish to create the activity without a CMM assigned to it.  You can assign the CMM to all the activities within the process once you are ready to insert the CMM.
