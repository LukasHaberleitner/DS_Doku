3DEXPERIENCE platform

This tab deals with the configuration of the interoperability between CATIA V5 and 3DEXPERIENCE platform.

If you clear the Shows access to 3DEXPERIENCE platform in Start menu option, the 3DEXPERIENCE platform command is removed from the Start menu.
By default, the Shows access to 3DEXPERIENCE platform in Start menu option is selected.

The selected Integration Mode and its configuration options will only be applicable if, you have installed Connector for CATIAV5 server.
For more information refer Connector for CATIA V5 User's Guide: Using CATIA V5 Embedded Integration: Configuring and Activating the Embedded Integration Mode in ENOVIAV6/3DExperience documentation.
