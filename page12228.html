Part and Assembly Templates

See the Quick Reference topic for a comprehensive list of the interactions that can be carried out on document templates. See About Part and Assembly Templates for more information about these features.

 

Create a Document Template: Select Insert > Document Template Creation ... or click the Create Document Template icon () (if in the PKT workbench), select the elements making up the document template from the specification tree, define a name for the document template and its reference elements then choose an icon for identifying it.
