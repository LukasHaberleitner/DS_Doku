Getting Started

Before getting into detailed instructions for using Electrical Harness Installation, the following tutorial provides a step-by-step scenario demonstrating how to use key functionalities.
Before starting this scenario, you should be familiar with the basic commands common to all workbenches. These are described in the Infrastructure User's Guide.
The main tasks proposed in this section are:

Entering the Workbench
Setting Up the Options
Creating a Multi-Branchable Document
Creating Construction Points
Defining the Segment Parameters
Defining the Segment Route
Adding a Branch Point

All together, these tasks should take about 15 minutes to complete.
