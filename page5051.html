Working with Workshop Documents

Various types of workshop documents are used as inputs for cutting, marking and forming machines and can be generated automatically. The different types of documents that are supported include fabrication sketches, template sketches, resource sketches, and XML data files (for inverse bending curve data, pin jig data, and girth table data).

The sections which follow describe:

  • Defining the appearance of workshop documents
  • Examples of different document types, and extracting a single document of each type
  • Extracting all documents for a given process
  • Opening workshop documents associated with a selected product or operation
