Setting Current Position as Initial Position

 

This task will show you how to set the current position as the initial position for all or selected elements.
 
Open  Move.CATProduct document.

  1. Move some of the elements using the 3D Compass (see Moving Objects Using the 3D Compass in the Infrastructure User Guide).

  1. To set the current position as initial for all elements, in the DMU Move toolbar, click the Set Current Position as Initial icon .

    The current position becomes the initial position for all elements.

  1. To set the current position as initial for the selected elements, first select the elements, then, in the DMU Move toolbar, click the Set Current Position as Initial icon .

    The current position becomes the initial position for the selected elements.

  1. Now change the position of  some elements of the document (using Translate, Rotate, Position, Transform).

  2. Click the Reset Position icon .

    The position of all elements is reset to the new initial position.
In the above, the term initial position signifies the following:
  • for an existing document - the position of a product when the document was opened
  • for an inserted document - the position of a product when the document was inserted
  • for a new document - the position of a product when the document was created
