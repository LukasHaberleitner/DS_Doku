Annotation 

This page deals with the options concerning:
  • The Annotation Creation.
  • The Geometry Connection Management
  • The Leader associativity to the geometry.

Annotation Creation
Defines the annotation creation options:

Annotation following the mouse (Ctrl toggles)
Defines whether the annotation is positioned according to the cursor, following it dynamically during the creation process or not.
By default, this option is not selected.

Geometry Connection Management

Automatically propose reconnection of broken links
Defines whether the automatic reconnection of broken links is to be proposed or not.

By default, this option is selected.

End command after last reconnection
This option is available when the Automatically propose reconnection of broken links option is selected.

Select this option if you want to end the command after you select the last geometry corresponding to the last broken link to reconnect. Notice that in case a reconnection is skipped then the command does not end and lets you complete the repair and the (Automatic Reconnection of Broken Links) is removed from the title bar of the Connection Management dialog box.

By default, this option is selected.

Check automatically validity after last reconnection
This option is available when the Automatically propose reconnection of broken links option is selected.

Select this option if you want to check the validity automatically after you select the last geometry corresponding to the last broken link to reconnect. Notice that in case a reconnection is skipped then the validity is not checked and the (Automatic Reconnection of Broken Links) is removed from the title bar of the Connection Management dialog box.

By default, this option is not selected.

Give back control after last reconnection
This option is available when the Automatically propose reconnection of broken links option is selected.

Select this option if you do not want to perform an automatic action after you select the last geometry corresponding to the last broken link to reconnect. In this case, the (Automatic Reconnection of Broken Links) is removed from the title bar of the Connection Management dialog box.

By default, this option is not selected.

Leader associativity to the geometry
Defines the leader associativity options:
  • Free: specifies that leader annotations are freely positioned relative to their geometrical elements.
  • Perpendicular: specifies that leader annotations are positioned perpendicular to their geometrical elements.
By default, the Free option is selected.
