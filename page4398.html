Supported Process Relationship Types

The following section details the relationships between Process Engineer relation types and their DPM counterparts.

Supported Process Engineer relations in DPM

An object linked in Process Engineer to a process with one of the following relation will be loaded in DPM if the process is loaded.

Process Engineer Relation DPM Equivalent
Process Processes Product Items
Process Processes Features Items (For Fasteners and locators)
Process First Processes Product Items (first use)
Process Removes Product Items (removes)
Process Creates Product Outputs
Process Uses Resource Resources
Process Runs On Resource Resources (Process running on resource)

Process Engineer Relations that may be deleted from DPM (via Unassign commands)

All, except Process Runs on Resource. (Currently, the Process Runs on Resource may be deleted in a DPM session, but the deletion will not be saved back to the Manufacturing Hub.)

Relations that can be created from DPM

Process Engineer Relation DPM Equivalent
Process Processes Product Items
Process Processes Features Items (For Fasteners and locators)
Process First Processes Product Items (first use)
Process Creates Product Outputs
Process Uses Resource Resources

Note that certain relations can only be created if the proper license is available.
