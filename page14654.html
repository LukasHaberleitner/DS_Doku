Editing Material for Profiles

This task explains how you can edit the material properties of an existing profile. You can edit the material, grade and section combination that is defined in the specifications catalog. You can also edit the material and orientation properties of several profiles at the same time.

 

 

 1. With your document open in the appropriate workbench, double-click the object to open the dialog box, such as the Stiffener dialog box. (You can also open the dialog box by right clicking on the object, and selecting the object - xxx.object - and Definition in the drop down menus.) For shapes and plates, you have to double-click or right-click in the specification tree after making the product active.

Select the  Material & Orientation tab in the dialog box (e.g. Stiffener dialog box if you are working with stiffeners). The image below shows the options you can edit for material selection.

 

2.

 Select a material by clicking on the down arrow. When you select a material, the Grade and Section values do not change. If the previously selected values are no longer valid - they do not exist in the specifications catalog - the entries will display a symbol to show the entries are not correct (the symbol, called the Invalid symbol is shown below). In this example, the material was changed from steel to aluminum. The grade and section are not available in the catalog, therefore these fields display the Invalid symbol.
  3. Click the down arrow in the Grade field. The incorrect grade will display in the drop down list, along with valid grades. Select a valid grade. Once you select a valid grade, valid sections will be available for selection.
  4. Select a section in the Section field.
  5. Click OK when you have made your selections.
The material and grade you select for the Material and Grade fields above, should exist in both the specifications catalog and the material catalog. If the material and grade exist in the specifications catalog but NOT in the material catalog, you will get the following error messages:

In Structure Design: ERROR: The selected material, grade combination cannot be found in the material catalog. You will not be able to create the plate.

In Structure Functional Design: WARNING: The selected material, grade combination cannot be found in the material catalog. You will be able to create the plate, but will not be able to generate a piece part from your design.
 

Editing Material and Orientation for Several Profiles
1. You can edit the material and orientation properties of several profiles at the same time. Make the product active in the specification tree. Then, press Ctrl and select the objects (in the specification tree) you want to edit. Right-click and select Selected Objects and open the dialog box, such as the Definition dialog box for shapes. In the example given below, two shapes are selected.
  2. You cannot modify the geometry of the shapes. Click the Material & Orientation tab. Modify the Material, Grade, and Section of the objects, as required.

           

You can also modify the orientation properties, such as Anchor Point, Angle, and Flip the section orientation. You can also modify the orientation by using the manipulators in the design. The section manipulators appear only on the first object selected by you. For more information, see Using Section Manipulators.
  3. Only the values common to the selected objects are displayed. If the selected objects have different values for a property, the field appears blank or unset. In the example given below, the selected shapes have different sections, anchor points and angles.

                      

You can assign the same values to the objects. Click the down arrow next to the property and select a value in the list displayed.
