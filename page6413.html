Becoming Familiar with the Interface 

This page quickly makes you familiar with the basic functionalities available within the Sketcher workbench.

Before starting sketching, this task shows you how to:

  • Restore the toolbars positions
  • Use the Sketcher toolbars
  • Use the mouse buttons.

Restoring the Toolbars' Positions 

  1. Select Tools > Customize... from the menu bar.
    The Customize dialog box is displayed.

  1. Click Restore position in the Toolbars tab.
    A dialog box is displayed asking you to confirm the Restore operation.

  1. Click OK to validate.

  2. Click Close.
    All the Sketcher toolbars are now displayed at their original positions.

Using the Sketcher toolbars 

The Sketcher toolbars let you:

  • draw predefined profiles. For this, use the Profile toolbar commands.

  • set constraints. For this, use the Constraint toolbar commands.

  • perform operations on profiles. For this, use the Operation toolbar commands.

  • perform operations on both 2D and 3D geometry. For this, use the Tools toolbar commands.

Using the mouse buttons in order to...  

Select menus/commands/elements from the geometry area.

  1. Click the left mouse button.

  2. Select the desired element, for instance select a line from a rectangle.
 
 

 

Multi-select elements from the geometry area

  1. Click the left mouse button.

  2. Select a first element.

  3. Keep pressing the Ctrl key.

  4. Select one after the other the wanted elements.

or

  1. Press the left mouse button.

  2. Drag the cursor over the elements to be selected.

  3. Release the left mouse button.

The wanted elements are selected.
 

Drag elements in the geometry area

  1. Select an element from the geometry area.

  2. Keep pressing the left mouse button.

  3. Drag the mouse to move the selected element.

or

  1. Press the left mouse button.

  2. Drag the cursor over the elements to be selected.

  3. Move one of the rectangle line.

As you can notice the whole rectangle is moving.
 
 

Drag the whole geometry area

  1. Keep pressing the middle mouse button.

  2. Drag the cursor to move the whole geometry.

Re-center an indicated point

  1. Click a point in the geometry area using the middle mouse button.
    The indicated point is moved to the center of the window.

zoom in and out

  1. Keep pressing the middle mouse button.

  2. Press the right mouse button once.

  3. Drag the cursor up to zoom in.
 
 

or

  1. Drag the cursor down to zoom out.
 

Rotate Elements in the Geometry Area

  1. Keep pressing the middle mouse button.

  2. Keep pressing the right mouse button.

  3. Drag the cursor to rotate the geometry.

Now that you have seen how to use the Sketcher toolbars and how to use the mouse button, let's start creating:

  • sketches from new parts.
  • sketches from existing parts.
