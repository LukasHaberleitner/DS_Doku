Locating Reference/Resulting Views  

This task shows how to navigate easily between reference/resulting views in the case of section views, section cuts or auxiliary views:
  • You can locate the reference (i.e. parent) view of a resulting view.
  • You can locate the resulting (i.e. child) view of a reference view.

Open a .CATPart document. Double-click Sheet.1 in the specification tree to open the layout in the 2D window.
Create a section view, section cut or auxiliary view with callout.

Locating a Reference View
 

  1. Right-click the resulting view (auxiliary view, section cut or section view).

  2. Select Locate Reference View. The application window is reframed on the view where the view callout is defined.
 

Locating a Resulting View
 

  1. Right-click the view callout.

  2. Select Locate Resulting View. The application window is reframed on the resulting view.
