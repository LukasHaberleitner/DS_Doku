Selecting Colors for Processes in the Gantt Chart

The colors of processes that are linked with the resources can be individually selected and changed in the Resource Gantt Chart. You can use this setting exclusively in the Resource Gantt Chart and for DPE projects. Once the color settings have been changed in the Resource Gantt Chart, the processes are also displayed using those colors in the Process Gantt Chart.

All processes that are linked with a resource are displayed in the color that you select for displaying the processes of a resource.  As a result, processes that are linked with resources can be displayed in different colors for each different resource.

Selecting the Color for Processes in the Gantt Chart resource
 

  1. Click the Open a Resource centric Gantt Viewer .

  2. Right-click on a resource, and select Color in Gantt View.

  3. From the Color in Gantt dialog box, select the color for the processes that are linked with resources.
    The processes are displayed in the selected color.

  4. Select the color and click OK.

 
 

Limitation
  These settings do not apply to the Process-centric Resource Gantt Chart. Processes are displayed in the same manner as before in this Gantt Chart.
