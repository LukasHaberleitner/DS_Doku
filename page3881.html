Editing View/Annotation Plane Properties

This task shows you how to rename an annotation plane and hide its frame.
Open the Common_Tolerancing_Annotations_01 CATPart document.

  1. Right-click the Front View.1 annotation plane and select the Properties contextual command.

  2. Select the View tab in the Properties dialog box which appears.

    To rename the annotation plane, enter the new name in the Name field.
    Modify the Ratio value to apply a zoom ratio to all the annotations contained in the view.
    Check/uncheck the Display view frame option to show/hide the view/annotation plane frame.

  3. Click OK to confirm and close the dialog box.
