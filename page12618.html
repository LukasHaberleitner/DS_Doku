Customizing

A certain number of settings is available to let you customize your Part Design workbench.

The customization you perform is stored in permanent setting files, meaning that these settings are not lost when you end your session.

To access them, proceed as follows:

  1. Select Tools > Options.
    The Options dialog box displays.

  2. From the Infrastructure category, select the Part Infrastructure sub-category in the left-hand box.
    The General, Display and Part Document tabs appear.

    The General tab provides options dealing with:

    • External References
    • Update
    • Delete Operation
    • Replace

    The Display tab provides options dealing with:

    • Display in Specification Tree
    • Display in Geometry Area
    • Checking Operation When Renaming

    The Part Document tab provides options dealing with:

    • When Creating Parts
    • Hybrid Design
    • Color on Import Management

    Any modifications made to the Part Infrastructure settings can now be recorded in a Visual Basic file using Tools>Macro>Start Recording.  See Recording, Running and Editing Macros in the Infrastructure User's Guide for details about running macros. Additionally, you can launch a dedicated Visual Basic macro in order to set parameters for Part infrastructure.

  1. To access settings about annotations, select the 3D Annotations Infrastructure category.

Click the tabs of interest.

  • Tolerancing
  • Display
  • Manipulators
  • Annotation
  • View/Annotation Plane

  1. To access settings about thread / tap, select Tools > Options > Mechanical Design > Part Design category.

    The Part Design tab provides following check boxes for thread/tap and technological results.

    • Thread/Tap:
      • Display symbolic representation:

        Shows a texture representing the thread/tap on the face of the part.
        By default, this check box is cleared.
      • Main diameter used as reference for standard management: Enables the main diameter mode in the part.
        By default, this check box is selected.
    • Features color:
      • Inherit colors from surface or reference object

        Inherits the input surface colors in the surface based features and input reference object colors in the pattern and mirror features.
        Clear this check box to reset the color properties to default solid colors for these features.
        By default, this check box is selected.

        This option is available for Split , Close Surface , and Sew Surface commands.
        You can interactively switch between the two options while working on these commands.
    • Technological Results:
      • Display Technological Result data as Knowledge Parameters: Retrieves the technological result data as knowledge parameters.
        By default, this check box is cleared.
    • Insertion solid feature:
      • Insertion in PartBody if surface tool in work object: Insert the solid feature in a PartBody if the surface tool is defined as work object.
        By default, this check box is selected.

  2. Change these options according to your needs.

  3. Click OK when done to validate your settings.
