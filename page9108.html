CATAnalysisSetType (Enumeration)

enum CATAnalysisSetType {
  catAnalysisSetIn,
  catAnalysisSetOut,
  catAnalysisSetNeutral
}
Update type for analysis sets.
Role: CATAnalysisSetType defines the Category for Set behavior for update.
Values:
catAnalysisSetIn
The set will be an input for update.
catAnalysisSetOut
The set will be an output for update.
catAnalysisSetNeutral
The set will be have no impact for update.
