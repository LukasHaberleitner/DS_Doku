Automatic Compass

Snap to placed element

If this option is checked, you take advantage of the compass:
when you place an element, the compass snaps to this element allowing you to modify the orientation and location.
By default this option is checked.

Enable Sourcing Access
If this option is selected, the Collaborative Enterprise Sourcing (CES) environment is enabled.
By default this option is cleared.
