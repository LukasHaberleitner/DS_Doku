Creating a Detail to Detail Link

This task shows you how to export the process concept geometry to a target part.
Open the part from Creating a Concept to Detail Link.

  1. Click Link Detail to Detail  in the Structure Definition toolbar.

  2. Select the target detailed part.
    The geometry is copied to this part to initialize the detailed design.

  3. Select the reference link.
    This is a concept to detail feature, that contains the elements to propagate.

A link feature is created in the target detailed part.

  • Its name is Link to<Reference Feature Name>
  • It is created under a geometrical set, with the same name as the reference link.

It creates the following data model under the created geometrical set.

  • Propagate Geometry set: Copy of each feature located in the To Propagate set of the reference link.
  • Detailed Geometry set: Copy of each feature located in the Detailed Design set of the reference link. If the reference link is linked to a process step, the Detailed Design set is the one located in the Step Geometry set.
