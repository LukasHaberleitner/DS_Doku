Extension Line Properties

This reference describes the Extension Line properties concerning:
  • Dimensions.
 
  This page deals with the properties concerning:
  • The General.
  • The Extremities.
  • The Funnel.
 

General
 
 

Color
  Defines the extension line color.
  By default, the color is black. See the screen capture.
 

Thickness
  Defines the extension line thickness.
  By default index, 1..
 

Displays first extension line
  Defines whether the first extension line is displayed or not.
  By default, this option is selected.
 

Displays second extension line
  Defines whether the second extension line is displayed or not.
  By default, this option is selected.
 

Slant
  Defines the slant angle for the extension line. This angle is set between 90 degrees and -90 degrees excluded. This functionality works only on linear dimension line and the line linking extension line anchor points (blanking excluded) has to be parallel to the dimension line.
  By default, 0.000 deg.
 

Extremities
 
 

Overrun
  Defines the extension lines overrun.
  By default, 2.000 mm for both extension lines.
 

Blanking
  Defines the extension lines blanking.
  By default, 2.000 mm for both extension lines.
 

Funnel
 
 

Funnel
  Defines whether a funnel is applied to the extension line or not.
  By default, this option is not selected.
 

Height
  Defines the funnel height.
  By default, 10.000 mm.
 

Angle
  Defines the funnel angle.
  By default, 45.000 deg.
 

Width
  Defines the funnel width.
  By default, 5.000 mm.
 

Funnel mode
  Defines the funnel mode:
  • External.
  • Internal.
  By default, External.
 

Funnel side
  Defines the funnel side:
  • Left or Bottom.
  • Right or Top.
  • Both Sides.
  By default, Both Sides.
