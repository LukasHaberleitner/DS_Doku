Basic Draft

This task shows you how a basic draft is migrated into CATIA V5 using the option CATIA_SPEC.
Open the document SheerDraft.model in CATIA V5.

  1. Apply a Check AS SPEC on *SET1.

  1. Copy and Paste Special *SOL1 into another CATPart document using the CATIA_SPEC option and update your document to compute the geometry.

You have access to the model's geometry and specifications. CATIA V5 has created a draft feature.

Note that the "Square mode" is the default draft form:

Note:

In the following example, the solid contains four drafted faces. Two faces have been drafted using the same angle value, the other two faces with distinct angle values. After migrating this data into CATIA V5, we obtain three drafts. Both drafts using the same angle value have been converted into one Draft feature (Draft.2).

If you wish to perform the migration, open DraftMultiAngle2.model.
