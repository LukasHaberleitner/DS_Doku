Propagate Execution Behavior: Data Sheet

 

Accessing this behavior This behavior can be accessed by clicking the Propagate Execution Behavior icon ().
Describing this behavior This behavior is designed to trigger the execution of a given object: If the object is a V5 object, the update will be performed. If it is a behavior, the behavior will be executed.
Using this Behavior Interface

 

Name: Name attributed to the behavior.
Comment:  Comment associated with the behavior.
Operate Target: Select  the object that you want to be executed in the Behaviors tree.
Action Delayed: 
  • If unchecked, the behavior executes immediately.
  • If checked, the selected behavior will not execute before the current behavior executes to the end.
