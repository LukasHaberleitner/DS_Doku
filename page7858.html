Importing the Mesh

This task shows how to import one or more existing meshes from a Nastan bulk (.dat) document.

To know more about the CATIA V5 element type, refer to the Finite Element Reference Guide.

The default unit for the length value is fixed in the Options dialog box (General > Parameters and  Measure > Units tab).
For more details, refer to the Infrastructure User's Guide.

  1. Select Start > Analysis > Advanced Meshing Tools.

    The New Analysis Case dialog box appears.

  2. Click OK.

  3. Click Import Mesh in the Import/Export toolbar.

    The Import mesh dialog box appears.

  4. Select the Sample11.dat document from the samples directory.

    Multi-selection is available in the Import mesh dialog box.

  5. Click Open.
