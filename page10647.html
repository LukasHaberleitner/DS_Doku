Applying Filters to CATIA Version 4 Library Families

This task shows you how to apply a filter to a CATIA Version 4 Library family in CATIA Version 5.
Have a Version 4 library and its contents open as shown below.
 

  1. In the dialog box above, display all the objects contained within one or families, SCREWS and NUTS for example, by clicking on the plus sign opposite the family name, select the family name and right-click.

  2. Select Filter... in the menu that appears:

  3. The dialog box below is displayed:

  4. Complete the fields, then click Apply to redisplay the whole family structure.

  5. Click Reset to display the whole family structure as it was prior to the filter application.
