What's New?

Enhanced Functionality

Inherit Colors from a Surface or a Reference Object (V5-6R2024SP2)
The Inherit colors from surface or reference object check box is added in the Features color > Tools > Options > Mechanical Design > Part Design.
 When selected, inherits the input surface colors in the surface based features and input reference object colors in the pattern and mirror features.

For more information, see Customizing.
Colored Selected Faces in Union Trim Preview (V5-6R2024SP2)
The selected faces to remove and faces to keep are displayed in magenta and cyan colors respectively during the trimming bodies operation.
You can easily identify the selected faces when using the Union Trim command.

For more information, see Trimming Bodies.
Display Thread/Geometric Description and Extension Type in Hole Analysis (V5-6R2024SP2)
The Thread/Geometric Description and Extension columns are shown in the Hole Analysis dialog box. These columns provide the information regarding the thread description and the hole depth.

For more information, see Analyzing Holes.
Creating a Point on the Center of Gravity (V5-6R2024SP2)
You can create a point on the center of gravity of a volume or a surface using the Center of gravity point type in the Point Definition dialog box.

For more information, see Creating Points.
Creating a Remove Face Feature Using Limiting Elements (V5-6R2024GA)
 
While removing faces using limiting elements, you can limit the remove face feature at a clicked point or at a selected point on an edge fillet.

Select the limiting element on the fillet face by clicking on the edge or a predefined point on the edge. A blue disc is displayed at this point to indicate the position of the limiting element.
A handle is also displayed to indicate the direction of the limit. You can click the handle to invert the direction of the limit. You can also select multiple points on the edge of a fillet face as limiting elements.
 
For more information, see Creating Remove Face Features.
Two New Attributes Added in Thick Surface Definition Dialog Box (V5-6R2024GA)
 
You can define the Max deviation and Tangency objective attributes to apply constraints for lateral face smoothing.

Max deviation: Maximum distance between the lateral faces and the exact 90ï¿½ surface.
Tangency objective: Target angle for the lateral edges smoothing.

You can define these new attributes only if the Disable tangency continuity check box is cleared.

For more information, see Creating Thick Surfaces.
Previewing Results for Thickness and Replace Face Features (V5-6R2024GA)
 
The Preview option is added in the Thickness.x and Replace Face.x dialog box.

You can use the Preview option to check the effect of your adjustments to the feature without having to close the dialog box.

For more information, see Creating Thicknesses and Creating Replace Face Features.
Arbitrary Selection Order for Isolating Geometric Elements (V5-6R2024GA)
 
You can now isolate the elements by selecting them in any order. The new algorithm correctly arranges the elements for the isolation.

For more information, see Isolating Geometric Elements.
Creating MultiHole (V5-6R2024GA)
 
The MultiHole  command is now added in the Sketch-Based Features section. You can create multiple holes in a single operation on different supports. the available positioning objects are point on surface, point on plane, and sketch.

For more information, see Creating MultiHole.
