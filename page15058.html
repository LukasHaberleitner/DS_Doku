Project Resource Management

Users need to manage their resources in a way that is different from what they have been used to. Previously, system administrators could click Tools - Options in the menu bar, navigate to a tab and field, and enter information. Information such as directory paths for various resources without which the application cannot function - line list catalogs, user dictionaries, catalogs, etc.

The same information is still needed, but the way of entering it is different. Administrators can no longer enter it using the Tools - Options dialog boxes. Instead, they need to enter the information in one file, which is in XML format. What they enter will be visible in the Options dialog boxes, but it cannot be changed there.
