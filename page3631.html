Creating a Surface Texture

This task shows you how to create a surface texture.

Open the SurfaceTexture.CATDrawing document.
  1. Click the Surface Texture from the Annotations toolbar.

  2. Select an attachment point of the surface texture.
    The surface texture position and orientation is associative to this point.

  3. In the Surface Texture dialog box, enter the values in the required fields.
     
    • You can click the red manipulator to invert the direction of the symbol.
    • You can manage the availability of the manipulator through Tools > Options > Mechanical Design > Drafting > Manipulators.
  4. Click OK.
    The surface texture is created.

  5. Optional: To modify the surface texture position by dragging it to the required location.

     
    • An extension line may be displayed between the surface texture and the element to which it is attached (providing this element is linear), depending on where you drag the surface texture.
    • The profile parameter list does not contain engineering symbols. These are simple text strings that assist in inserting the parameters quickly.
      These parameters always have DS ISO 1 font even if the font of surface texture is different. However, if the font of the surface texture feature is changed after inserting the parameters in it, the font of the parameters is also changed and they are visualized incorrectly.
      This parameter list is available only for specific boxes.

About Surface Texture

  • By default, there is a 1 millimeter space between the geometry and the extension line. There is also a 1 millimeter space between the end of the extension line and the surface texture. These values cannot be customized.
  • The default parameters of the surface texture are 1 for thickness and solid for line type. These values cannot be customized.
    For more information, see Surface Texture Parameters and Styles.
  • You can modify the surface texture at any time. To do so, double-click the surface texture and enter the required modifications in the Surface Texture dialog box.
  • If you have selected the Use style values to create new objects option in Tools> Options> Mechanical Design> Drafting> Administration tab, the Surface Texture dialog box is pre-filled with custom style values (as defined in the Standards Editor). In this case, Properties toolbars and the Tools Palette are disabled during the creation of the surface texture.
    However, if you have not selected this option, the Surface Texture dialog box is pre-filled with the last entered values (if any). In this case, the Properties toolbar and the Tools Palette are active during the creation of the surface texture.
  • If you have selected the Use style values to create new objects option, you can reset the current style values in the Surface Texture Editor dialog box at any time using the Reset button.

Position and Orientation 

  • By default, the orientation of surface texture is determined according to the orientation of the line it is associated with.
    To modify the orientation, right-click the symbol and select Properties. In the Text tab, under Orientation, select the Invert check box.
  • When this is not already the case, you can link the surface texture position and the orientation to another element, see Making an Existing Annotation Associative.
