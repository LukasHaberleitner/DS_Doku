Entering the Workbench

This task shows you how to enter the Structure Functional Design workbench. Structure Functional Design consists of two workbenches - Structure Functional System Design and Structure Functional Object Design. You must first create a system, and make the CATPart under it active, before you can correctly enter the Object Design workbench. This is shown below.
1. To enter the Structure Functional System Design workbench, from the main menu bar click Start, select Equipment & Systems and then Structure Discipline  >  Structure Functional System Design. The workbench opens.
2. Create a system - refer to Creating a System. You can correctly open the Structure Functional Object Design workbench once you have created a system. To open the workbench, select the CATPart that is created under the system in the specifications tree and go Start  >  Equipment & Systems  >  Structure Functional Object Design. In the image below the CATPart you must select is highlighted.

Once you make your selection in the menu bar, the Structure Functional Object Design workbench opens.
