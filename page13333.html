Editing GVS File

This task explains how to edit xml based GVS file by editing the parameters, adding or removing the nodes and preview the changes in the GVS editor.
  • Ensure that the Project Resource Management resources, drawing catalog, and feature dictionary are assigned in Project Resource Management.
  • Store an xml GVS file locally on your system.
  • Import and open a GVS model from the database. A sample DefaultModelGVS.CATProduct is located at ...\startup\EquipmentAndSystems\Drafting\DefaultModelGVS.
  • Clear the Prevent generative view style usage check box in Tools > Options > Mechanical > Drafting > Administration tab.
 

Modify the GVS Parameters 

  1. With the GVS model open, click GVS Editor in the Detailing toolbar.
    The     GVS Editor dialog box appears.

  2. Select File > Open to select a GVS xml file.
    The GVS file is selected and the tree structure is displayed in the dialog box. 

  3. Click a node in the tree.
    The properties are displayed in the Parameters tab.

  4. Change the values of the generative view style parameters.

  5. Select the Preview tab to preview the changes.
    The parameters are modified and changes appear in the Preview tab.

  6. Select File > Save to save the changes done in the GVS file.
    The modifications are saved in the GVS file.

  7. Click to close the dialog box.
    The parameters are changed and the GVS file is modified.
 

Add and Edit Extensions

  1. Right-click the element you want to add an extension, and select Add Extension.
    The ExtensionTreeDlg dialog box appears.

  2. Click SetCategory to select the category.
    The ClassBrowser dialog box appears. It displays the extensions valid for the selected element.

  3. Select the required extension and click OK.
    In the ExtensionTreeDlg dialog box, the extension and its nodes are displayed.

  4. Select the nodes you want to keep for the new extension and click OK.

    If the extension already exists, a warning message appears suggesting to check the extension in the Edit mode. 

  5. In the GVS Editor dialog box, right-click the element to which you want to add the extension and select Refresh.
    The extension is added to the structure element.

  6. To edit an extension, right-click the required extension and select Edit Extension.
    The ExtensionTreeDlg dialog box appears displaying all the nodes of the extension.

  7. Select the nodes you want to keep and click OK.

  8. Right-click the edited extension in the GVS Editor and select Refresh.
    The nodes are updated.

Remove Extensions and Nodes

  1. To delete a node, right-click the required node and select Remove Node.
    A message appears informing that all modification for the selected node will be lost and asking if you want to continue with removing the node.

  2. Click Yes to continue with removing the node.
    The node is removed from the extension. 

  3. To remove the extension from the structure element, right-click the extension and select Remove Extension.
    A message appears informing that all modification for the selected extension will be lost and asking if you want to continue with removing the extension. 

  4. Click Yes to continue with removing the extension.
    The extension is removed.

    • You cannot perform an undo after removing a node or an extension.
    • Previews of certain elements like beam, stiffener on free edge, seam and planning break are not generated.
