Getting Contextual Help

This task explains how to get contextual help on the current command.

  • Select Help > "ProductLine"  V5 Help, or press F1 (where "ProductLine" stands for the brand you have installed, for instance CATIA).
    For example, if you are using the Edge Fillet command, the online help library is displayed in HTML format in a Web browser window, at the topic explaining how to use the Edge Fillet command:
    There is always one current active command (by default, the Select command) whose icon is highlighted. This is the command whose help is displayed when F1 is pressed. However some commands can be executed without changing the current command (for example Tools > Image > Capture...). Contextual help gives the priority to the last of these commands that has been started and is still running rather than to the current active command, otherwise it would not be possible to get help for such commands.
