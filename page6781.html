Accessing Data Through a Catalog

This task explains how to instantiate electrical devices in a CATProduct document using a catalog.
The devices to be imported are:
  • An item of equipment: EquipmentDA4
  • Two single connectors: SingleConnectorDF56L and SingleConnectorDF56R

  1. Click Catalog Browser to open CatalogOfDevices.catalog.

    The dialog box opens:
    If necessary, click Open to browse another catalog and select the CatalogOfDevices.catalog from the samples folder. The full path is: .../online/elbug_C2/samples/CatalogOfDevices.catalog.

  2. Double-click Electrical Devices: the folder content is displayed.

  3. Double-click Equipments: the folder content is displayed.

  4. Select EquipmentDA4, drag and drop it onto the Product1 in the specification tree.

    The equipment is instantiated: it is composed of three cavities.

  5. In the Electrical Devices folder, double-click Single Connectors

  6. Select SingleConnectorDF56L, drag and drop it onto the Product1 in the specification tree.

  7. Select SingleConnectorDF56R, drag and drop it onto the Product1 in the specification tree.

    The result looks like this:

  8. Close the Catalog Browser.
Note that when you insert a device into a product, you can take advantage of CATIA - Assembly capabilities to constrain the device within the digital mock-up, profiting therefore by the associativity. For more information, refer to Using Assembly Constraints
