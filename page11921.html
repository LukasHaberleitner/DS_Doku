Creating Actions Groups  

This task explains how to group functional actions. 

  1. Open the PfdInterior.CATSystem file. The following System is displayed. (Click the graphic below to enlarge it.)

  1. Click the Functional Actions Group icon () in the toolbar.

  2. Create an arrow going from the Driver object to the Clutch pedal object. An arrow named ActionsGroup.1 is displayed.

  3. Double-click the ActionsGroup.1 arrow: the Edit Group of actions is displayed.

  1. Change the name of the Groups of actions in the Name field (Press in this example) and select the press actions in the system. Click OK and save your file.
