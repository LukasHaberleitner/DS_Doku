Moving Constrained Components Using the Compass 

This task consists of manipulating the assembly to check if the components react the way we want, i.e. according to the constraints we set in the previous task.
1.  Select the red patch at the center of the compass and drag it onto CRIC_SCREW. For details about how to use the compass, please refer to Infrastructure User's Guide.
As the compass is snapped to the component, you can manipulate the component.

2. Now, if you press and hold down the Shift key, and select v/z axis on the compass, then drag and drop the component up and down, you can see that three components are moving.

3.  Repeat the operation as many times as you wish.

The assembly reacts correctly. CRIC_FRAME does not move because it is fixed. The other three components can move.
4.  Release the left mouse button before releasing the Shift key.
5.  Drag the compass away from the selected object and drop it.
