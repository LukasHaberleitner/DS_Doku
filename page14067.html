Managing Standard Files

This task shows you how to modify, add, or remove sections in a Die Face Design Standard file.
Standard files are stored under ...\resources\standard\DieFaceDesign

  1. Select the Tools > Standards.
    The Standard Definition dialog box is displayed.

  2. Under Category, select Die Face Design.

  3. Select a standard file from the list.

  4. Click an item on the left to display and edit its value on the right.

  5. Click OK to validate and exit.

  6. Alternatively, go to ...\resources\standard\DieFaceDesign and edit the files with a text editor.
