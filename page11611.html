Creating Machining Axial Features

This task shows you how to create machinable axial features based on the selected filter criteria like diameter values, depth values, etc.
You can create the required axial feature using your selected filter criteria and then assign them to operation in sequence. You can also modify the filter criteria and create the next set of axial features and assign these newly created axial features to operations.
You can make use of the features created in the Globally Create All Prismatic Machinable Features task.
1. Select the Global Feature Recognition icon . The Global Feature Recognition (GFR) dialog box appears.
The Axial Filter tab is activated only when Hole is selected in the Basic tab.
  2. Select Axial Filter tab.
3.

Select Apply Axial Filter check box, to enable filtering of axial features.

The following are the available options under Axial Filter Parameters:

  • Diameter: This applies filter on the hole diameter value and create only those axial features that satisfies the diameter criteria.
    • Equal: Specify hole diameter absolute value.
      If you enter a value 2mm, then all the holes whose diameter is equal to 2mm are recognized by GFR.
    • Range: Specify hole diameter range in From and To up-down menus.
      Any hole of diameter falling in between this range applies be considered for feature recognition.
  • Depth: This applies filter on the hole depth value and create only those axial features that satisfies the depth criteria.
    • Equal: Specify hole depth absolute value.
    • Range: Specify hole depth range in From and To up-down menus.
      Note: The actual diameter of the hole and actual depth of the hole is considered while applying the filter based on diameter or depth. Counter bore diameter/depth, counter sunk diameter/depth, and counter drill diameter/depth are not considered for filtering the axial features.
  • Extension: This applies filter on hole extension.
    • Blind
    • Through
  • Hole Type: This applies filter on hole type. Select different types of holes from the drop-down menu.
    • Simple Hole
    • Tapered Hole
    • Counter Bore Hole
    • Counter Sunk Hole
    • Counter Drill Hole
  • Hole Threaded: The thread related parameters like Thread mode, Thread Diameter, and Thread Depth are available for selection only when the main thread parameter Hole Threaded is selected.
  • Thread Mode: If the hole is threaded then it applies filter on thread diameter of the hole. Thread mode have Left and Right as options to select.
  • Thread Dia: This applies filter on the thread diameter of hole.
    • Equal: Specify thread diameter absolute value.
    • Range: Specify thread diameter range in From and To up-down menus.
  • Thread Depth: This applies filter on the thread depth of hole.
    • Equal: Specify thread depth absolute value.
    • Range: Specify thread depth range in From and To up-down menus. 

Only those axial features applies be created that satisfies all the given filter criteria. Even a single criteria failing applies result in that particular axial feature not being recognized. All the other axial features satisfying the criteria applies be recognized.

Local Feature recognition does not support the capability to recognize machinable axial feature using filter information.

The design intent of the hole applies not be followed while axial filtering of the hole features.  
  The above mentioned settings are stored only to the extent of the parameter being selected. The value of the parameter provided by you is not stored. Next time when the GFR dialog box is opened, the previously selected parameter in the Axial Filter tab page applies be selected but the value applies be set to 0 units.
