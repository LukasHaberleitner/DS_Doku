Converting Standard into Construction Elements

You can convert standard elements into construction elements and vice versa.
Construction elements are internal to, and only visualized by the sketch. They are used as positioning references and are displayed in gray.
This task shows how to convert standard elements into construction elements and vice versa.
Ensure that Construction/Standard Element is not selected and sketch a simple profile.
Open the Construction_Standard.CATPart document.

  1. Select the line (standard type) you wish to convert into a construction line.

  1. Click Construction/Standard Element from the Sketch tools toolbar.
    The line you previously selected appears dashed to show it is a new type of line.


         
Construction elements are internal to, and only visualized by the sketch. They are used as positioning references and cannot be visualized in the 3D area.

  1. Click Construction/Standard Element again.
    The construction line is converted into a standard line.
To convert any geometrical element into a construction element use either of the following ways apart from using the Sketch tools toolbar:
  • Double-click the geometrical element to display its edition dialog box and select the Construction element check box.
  • Right-click the geometrical element and select the Construction Element option.
Conversely, to convert a construction element into a geometrical element, use either of the following ways apart from using the Sketch tools toolbar:
  • Double-click the geometrical element to display its edition dialog box and clear the Construction element check box.
  • Right-click the geometrical element and select the Geometrical Element option.
Both these options are available in all the profile commands contextual menu and edition dialog boxes.
Double-clicking on the line displays the Line Definition dialog box in which you can un-check the Construction element option if you want to convert the construction line into a standard line. For more information, refer to Modifying Element Coordinates.

  • In certain cases, construction elements are automatically created (e.g. when offsetting canonical elements, or when creating lines normal to a curve). If you subsequently delete the constraint or one of the elements, the construction element will be automatically removed.
  • Construction lines are not taken into account when entering another workbench.
  • Applying Construction/Standard Element on axes has no effect.
