Accessing Make

This task explains how to use Make from the 3DEXPERIENCE side panel.

  1. Select Start> 3DEXPERIENCE platform and enter Username and Password to log in into 3DEXPERIENCE server.

  2. Click Me > As a Business Model > Make.

    3DEXPERIENCE Marketplace Make opens.

  3. Now click “Add part to your request”. After the part has been added, you can send a quotation to the respective service providers. When you finalize the request, you can choose the format you want to export the model to the service providers. The available export formats are CATPart (Native), STEP, and STL. For more information on Marketplace Make, see 3D Modeling | xDesign | xDesign Basics | Action Bar | Marketplace Section in 3DEXPERIENCE User Assistance.

     Note:  If the Make service has been launched from a cloud 3DCompass, the user will have an access to the same Cloud 3DCompass in Make service and will be able to navigate back to app switcher. A close button is available for the user to go back to full 3DCompass view. If the Make service has been launched from an On Premise 3DCompass, the user’s cloud login is requested and Make service is displayed without any 3DCompass access. A close button is available for the user to go back to full On Premises 3DCompass view.
