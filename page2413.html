SmarTeamCAI Setting Repository
Quick Reference

SmarTeamCAI

Role: Handling of SMARTEAM (CAI) Data Exchange Import and Export parameters

Detailed role: Handling of SMARTEAM (CAI) Data Exchange Import and Export parameters

The SmarTeamCAI repository has the following characteristics:

  • Primary Tab Page: /General/Compatibility/SMARTEAM CAI
  • Repository File: SmarTeamCAI
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

SmarTeamCAI setting repository attributes:

  • AutoCheckOut
    Role: Display Check-Out/New-Release dialog box on first modification
    • Type: boolean
    • Default value: true
  • AutoConnect
    Role: Automatically launch connection on startup
    • Type: boolean
    • Default value: false
  • CheckOutNonLatest
    Role: Allow Check-Out/New-Release operation on non-latest revision
    • Type: boolean
    • Default value: false
  • ChooseProject
    Role: Choose SmarTeam project at first save
    • Type: boolean
    • Default value: false
  • LoadContext
    Role: Load the contextual assembly
    • Type: boolean
    • Default value: true
  • QuantityBasedOnActivity
    Role: Compute SmarTeam quality based on instances' activity
    • Type: boolean
    • Default value: false
  • UIActivated
    Role: Show menu
    • Type: boolean
    • Default value: true
