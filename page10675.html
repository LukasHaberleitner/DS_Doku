Customizing

Before you start your first working session, you can customize to suit your needs.
This type of customization is stored in permanent setting files: these settings will not be lost if you end your session.

  1. Select the Tools > Options command. The Options dialog box opens.

  2. Select the General category in the left-hand box.

  3. Click the Compatibility workbench. The tabs of the V4 Integration settings appear.
 
  In this workbench, the following V4 Integration tabs are available:
  • V4 Data Reading,
  • Saving as V4 Data,
  • V4/V5 DRAW,
  • V4/V5 SPACE,
  • V4/V5 SPEC,
  • V4/V5 TUBING,
  • Migration Batch.
   
For the customization of V4 / V5 SPACE settings, the following scenarios are available:
  • Scenario (V4/V5 SPACE): How to use Gap Healing,
  • Scenario (V4/V5 SPACE): How to use Curvature Improvement (Maximum Deformation),
  • Scenario (V4/V5 SPACE): How to manage Surface and Curve Sub-elements (Segmentation),
  • Scenario (V4/V5 SPACE): How to manage V4 3D Text Migration.
 

  1. Set options in these tabs according to your needs.

  2. Click OK when done.
