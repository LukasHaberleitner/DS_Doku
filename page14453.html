Querying Connector Attributes

This task explains how to select a component and query connector attributes. The following connector information can be viewed for the selected component:
  • Connector Name
  • Flow Capability
  • Associated Size Attributes

See Defining Connectors on a Component to add connectors to a component.

Dynamic connectors are explained in Defining Dynamic Connectors.
1. With your document open, click the Query Connector Info button .
2. Select the component with the connectors you want to query. The Query Connector Information dialog box displays.

The connectors are listed under the Cntr Name column. You can view the Flow Capability and Associated Size Attribute for each connector.

Click Close after querying the information.
  You can add connectors to the component with the Build Connector command. See Defining Connectors on a Component.
