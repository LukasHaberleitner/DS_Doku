About Finite Element Models

The Structure Design application provides a command to automatically generate a CATIA mesh model from a Structure design model. This model contains appropriate mesh constraints and properties.
The objective of this command is to make the Finite Element Model (FEM) creation much quicker compared to a manual approach.

The created FEM model can be treated within the Advanced Meshing Tools workbench. For more information, refer to the Advance Meshing Tools User Guide.
  For each Structure Design object to mesh, an intermediate object is created within a common CATPart (the proxy CATPart).

In the Proxy CATPart, necessary features to constrain each mesh part are computed and stored under a geometrical set. MeshParts are created with pertinent constraints and properties.
