GenericObjFrameProfile (Object)

Interface to manage Generic Object frame Profile of Robot controller.

Role: This interface provides methods to get/set data related to Object Profile.
Method Index
GetController Retrieves controller owning the profile.
GetName Gets name of the Object frame Profile.
GetObjectFrame Retrieve the underlying x,y,z, roll, pitch, yaw of the Object Frame.
SetName Set name of the Object frame Profile.
SetObjectFrame Set the underlying x,y,z, roll, pitch, yaw of the Object Frame.
