Managing URLs

You can add hyperlinks to your document and then use them to jump to a variety of locations, for example to a marketing presentation, a Microsoft Excel spreadsheet or an HTML page on the intranet.
This task explains how to add hyperlinks.
You should prepare a document that you want to see displayed via a hyperlink.

  1. Click the object you want to represent the hyperlink.

  2. Select the Insert >  Add Hyperlinks command.
    The Manage Hyperlink dialog box appears.

 

  1. Enter a name identifying your hyperlink.

  2. Click Browse... and select the file in the Link to File dialog box then click Open.

  3. Click OK in the Manage Hyperlink dialog box.
    You can now test the link you added.

  4. Click the object to which you just added the hyperlink.

  1. Click the Go to Hyperlinks icon in the DMU Review Navigation toolbar.
    The linked file is displayed.
