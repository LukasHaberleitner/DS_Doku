Editing Arrow Properties

This task explains how to access and modify the arrow properties.
Create a arrow.

  1. Select the arrow annotation.

  2. Select Edit > Properties.
    You can also right-click this annotation and select Properties.

  1. Click the Arrow tab.
  Apply scale on extremities:
Select the check box to apply the scale of the view to the extremities of the selected arrow.
If an arrow annotation is created outside a view, the scale of the sheet is applied to the extremities of the selected arrow.
