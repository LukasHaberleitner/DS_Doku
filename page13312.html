Estimating Weight and COG

This task shows how to compute a Weight Estimate and a COG on a Functional Volume set, and on a Functional Volume.

The computation is available through a contextual menu on the Functional Volume set and on a Functional Volume.
Object Contextual Menu
Functional Volume Set
Functional Volume

An accurate result can only be computed if the Functional Volume has been applied through the Planning Break Preparation command on all Functional Systems intersecting the Functional Volume.
  NOTE: You can use Knowledgeware to extract the weight and COG of a Functional Volume. For more information, see List of Attributes.

 

 1. Select the Compute Weight and CoG contextual menu. The system computes the data.

NOTE: To access the contextual menu, the Structure Functional Object Design workbench must be active.
. 2. Select the Properties contextual menu on a Functional Volume set and on a Functional Volume, to read the results.
