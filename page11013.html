Saving a Product Structure Containing a Rule in ENOVIA V5 VPM

This task explains how to save a product containing a rule in ENOVIA V5 VPM. In this scenario, you are going to:
  • Create a product in ENOVIA V5 VPM and send it to CATIA.
  • Add 2 components to the product as well as a rule that drives the 2 added components. Then you are going to store the product in ENOVIA V5 VPM.
  • Make sure you have selected ENOVIAV5 in the View > Toolbars  menu.

  1. Create a Product in ENOVIA V5 VPM. (In this scenario the product is named Advisor).

    Note that you can close ENOVIA V5 VPM after creating the Product.

Send the Product to CATIA

  1. To send the product to CATIA:

    • In CATIA, click the Connect icon ( ) in the ENOVIA Logon toolbar. The Log On to dialog box is displayed.
    • Type your User Name, and your password.
    • Select your Host from the Host scrolling list.
    • Select your role from the Choose a Role scrolling list.
    • Select a Workbook from the Choose a Workbook scrolling list and click OK.
    • Click the Search ENOVIA icon (). The VPM Search dialog box is displayed.
    • From the Objects scrolling list, select Product.
    • Enter the Owner's ID.
    • Click OK when done. The Product ID is displayed in the Result window.
    • Double-click the product. The VPM Navigator view is displayed.
    • Right-click the root product (Advisor) and select Open. The Open Modes dialog box is displayed. Leave the default settings and click OK. The CATIA view is displayed.

Add Components to the Product

  1. Click the root product and select Insert > Existing Component.... Select the KwrSyringeContainer.CATPart document. The syringe container is inserted into the product.

  2. Click the root product and select Insert > Existing Component.... Select the KwrSyringePiston.CATPart document. The syringe container is inserted into the product.

Add a Rule to the Product

  1. From the Start > Knowledgeware menu, access the Knowledge Advisor workbench and click the Rule icon () to create a rule.

  2. Enter the rule body below using the parameters available in the Members of All column in the Rule Editor, and click OK when done.

    SyringeContainer\PartBody\Sketch.2\Radius.2\Radius = SyringePiston\PartBody\Sketch.2\Radius.2\Radius +5mm
    SyringeContainer\PartBody\Sketch.1\Radius.1\Radius = SyringePiston\PartBody\Sketch.1\Radius.1\Radius

The rule is applied to the document.

  1. From the Start > Mechanical Design menu, access the Assembly Design workbench and click the Update All icon () to update the document.

  2. Right-click the technological package, and select Properties. Rename the technological package: Advisor_TP in this scenario.

Save Data in ENOVIA V5 VPM

  1. Click the Save in ENOVIA icon (). The Save in ENOVIA dialog box is displayed. Click OK. Your document is saved in ENOVIA. Close the product in CATIA.

  2. Send again your product to ENOVIA.

    • Close all windows in CATIA.
    • Click the ENOVIA Search  icon (). The VPM Search dialog box is displayed.
    • From the Objects scrolling list, select Product.
    • Enter the Owner's ID.
    • Click OK when done. The Product ID is displayed in the Result window.
    • Double-click the product. The VPM Navigator view is displayed. Open the assembly
