KWA Language Behavior: Data Sheet

Accessing this behavior This behavior can be accessed by clicking the Knowledge Language Behavior icon () in the Knowledge toolbar.
Describing this behavior This behavior is designed to write Knowledge Advisor scripts that will be executed at run time in the User workbench. The library of functions available in the Knowledge Advisor editors is also available in this behavior. The attributes of the technological type are accessed as the type is defined as an input of this code.

The Knowledge code written in this behavior can directly handle:

  • Parameters of behaviors belonging to the same main behavior.
  • Parameters of the technological type.

These parameters can be referenced in the code by selecting them in the Knowledge wizard or in the behaviors tree.

Using this Behavior Interface

 

Name:  Name attributed to the behavior.
Comment:  Comment attributed to the behavior.
Edit button Lets you access the editor.
