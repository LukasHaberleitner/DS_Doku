Reordering Catalog Entities 

This task shows you how to reorder the elements your catalog is made of, i.e. families, chapters, keywords, etc.) in order to classify them the way you want.
  To do so, three contextual commands are provided:
  • Reorder sub-chapters... to reorder children (sub-chapters, families or part families),
  • Reorder keywords... to reorder keywords,
  • Reorder components... to reorder catalog components (references to external documents, features, V4 or V5 documents).

Note that according to the element you activated, some commands may be grayed. For instance, you will not be able to select the Reorder sub-chapters... command if you activated a family in the specification tree.

This scenario takes the Reorder sub-chapters... command as an example but the steps to follow are identical when using the other two reorder commands.

  1. Open the ALL_FASTENERS.catalog catalog.

The ALL_FASTENERS chapter, the only chapter contained in the ALL_FASTENERS.catalog appears in a catalog navigator window.

  1. Double-click ALL_Fasteners. The entities contained in the chapter selected then appear in the left-hand part of the catalog navigator as shown:

  2. Double-click the element you want to reorder the components, for instance ALL FASTENERS.

  3. Right-click then select xxx object > Reorder sub-chapters... command to reorder the families contained in the ALL FASTENERS chapter. The reorder dialog box opens:

  4. Select the element to move.

  5. Click the up arrow to move the element up in the list.

  6. Click the down arrow to move it down.

  7. Click the button to sort the list in ascending order.

  8. Click the button to sort the list in descending order.

  9. Click OK to validate and close the dialog box.
When reordering keywords or components, the display is automatically updated in the corresponding tab (i.e. Keywords or Preview).
 
Note: You can sort the elements in alphabetical order by clicking or buttons.
