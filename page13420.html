In V5 R13, when trying to create a new part in the Sheetmetal Design workbench, a message appeared, suggesting to use the Generative Sheetmetal Design workbench to do so.
Since V5 R14, you can no longer create new sheet metal parts in the Sheet Metal Design workbench. To create new sheet metal parts, you must use the Generative Sheet Metal Design workbench. However, you can still use the Sheet Metal Design workbench to edit and modify any existing sheet metal part previously created within this workbench.
