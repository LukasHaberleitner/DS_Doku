Setting the Options

This task shows you how to set the application options required to perform the getting started.

  1. Set the Visualization Mode as the default mode for your sessions: in the Tools  >  Options command, from the Infrastructure category, select the Product Structure sub-category then the Cache Management tab and check the option Work with the cache system.
 
 

  1. Save the annotation data in the cgr representation: in the Tools  >  Options command, from the Infrastructure category, select the Product Structure sub-category then the Cgr Management tab and check the option Save 3D Annotation representation in cgr.
 
 

  1. Improve the highlight of the geometry selection: in the Tools  >  Options command, select the General category, select the Display sub-category then the Navigation tab and check the option Highlight faces and edges.
