Getting Started

Before getting into the detailed instructions for using CATIA Version 4 Interoperability, the following tutorial aims at giving you a feel as to what you can do with the product. It provides a step-by-step scenario showing you how to use key functionalities.

The main tasks described in this section are:

Opening a CATIA Version 4 Model in CATIA Version 5
Copying CATIA Version 4 Model Data to CATIA Version 5
Saving CATIA Version 5 CATPart Documents as CATIA Version 4 Models
