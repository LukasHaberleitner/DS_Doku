Repairing the Part

You are now going to repair the part:

  1. Send all the bodies except Embedded cases.1 in the No Show.

  2. Select each surface in the Embedded cases.1  body to find which one is in surplus.

  3. Surface.45 is in surplus. Delete it.

  4. Send Embedded cases.1 in the No Show and recall Multiple connection cases.1.
    An edge is shared by three surfaces.

  5. Change the color of the surfaces in Multiple connection cases.1 to blue

  6. Recall Geometrical Set.1 from the No Show.

  7. Right-click Geometrical Set.1 and select Define In Work Object.

  8. Click Split in the Operations toolbar and split  Surface.134 from Multiple connection cases.1 by Surface.7 from Geometrical Set.1. Split.1 is created in Geometrical Set.1

  9. Now split Surface.138 from Multiple connection cases.1 by Surface.7 from Geometrical Set.1.
    Split.2     is created in Geometrical Set.1.

    Surface.134     and Surface.138 have been sent to the No Show and replaced with Split.1 and Split.2.
