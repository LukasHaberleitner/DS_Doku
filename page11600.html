Machining Operations Toolbar

 

This section describes the various icons available through the Multi-Axis Surface Machining workbench. Their disposition varies with your configuration.

The Multi-Axis Surface Machining icons are available in the following toolbars when installed with 3-Axis Surface Machining:

  • Sweeping Operations,
  • ZLevel Operations,
  • Contour-Driven Operations,
  • Spiral Milling Operations,
  • Profile Contouring Operations.
 

Sweeping Operations Toolbar

Multi-Axis Sweeping
See the Basic tasks that illustrate the following tool axis modes for this operation:
  • Lead and Tilt 
  • Fixed
  • Thru a Point
  • Normal to Line
  • 4-Axis Lead/Lag
  • Optimized Lead.
 

 
 

ZLevel Operations Toolbar

Multi-Axis Tube Machining
See Multi-Axis Tube Machining Operation.

 
 

Contour-Driven Operations Toolbar

Isoparametric Machining
See the Basic tasks that illustrate the following Tool Axis modes for this operation:
  • Lead and Tilt
  • 4-Axis Lead
  • Interpolation.
Multi-Axis Contour Driven
See the Basic tasks that illustrate the following guiding strategies for this operation:
  • Between contours
  • Parallel contours
  • Spine contours.

Profile Contouring Operations Toolbar

Multi-Axis Curve Machining
See the Basic tasks that illustrate the following machining modes for this operation:
  • Contact
  • Between 2 curves with Tip / Interpolation machining  
  • Between a curve and part with Side / Interpolation machining.
