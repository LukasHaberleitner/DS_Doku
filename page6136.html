Creating Scans

This task shows you how to create scans by picking points on the cloud.
Open the Cloud1.CATPart from the samples directory.

  1. Click Scan on Cloud in the Scan Creation toolbar and select a cloud.
    This first pick selects the cloud and defines the first point of the scan.

  2. Pick points on the cloud to create the scan.

  3. Double-click to exit the action. A Scan_on_Cloud.x element is created in the specification tree.

 
  • Undo and Redo are available for each pick.
  • One single scan can be created over several clouds.
  • If you press the Ctrl key wile moving the cursor on the cloud,
    the creation of the scan is displayed interactively.
