Compatibility

This page deals with Compatibility options in the following tabs:
  • the V4 Data Reading tab lets you customize compatibility settings
  • the Saving as V4 Data tab lets you customize compatibility settings, and more particularly for the V5 to V4 Migration (Save As Model).
  • the V4/V5 DRAW tab lets you customize given options when using interoperability from V4 to V5.
  • the V4/V5 SPACE tab lets you customize given options when using interoperability from V4 to V5.
  • the V4/V5 SPEC tab lets you customize given options when using interoperability from V4 to V5.
  • the V4/V5 TUBING tab lets you customize given options when using interoperability from V4 to V5.
  • the Migration Batch tab lets you customize Migration Batch settings.
  • the 3D XML tab lets you customize 3D XML settings.
  • the DELMIA D5 tab enables you to set the options for DELMIA D5 Integration.
  • the DXF/DWG tab lets you set DXF/DWG options.
  • the Electrical Migration tab lets you set electrical migration options.
  • the ENOVIA V5 VPM tab lets you customize compatibility settings in CATIA V5 in order to work in a CATIA/ENOVIA LCA interoperability context.
  • the External Formats tab lets you customize the import settings.
  • the Graphics Formats tab lets you set options for using graphics formats.
  • the IGES tab lets you customize IGES settings.
  • the IGES 2D tab lets you customize IGES 2D settings.
  • the SMARTEAM (CWI) tab lets you customize SMARTEAM CATIA Web Integration settings.
  • the STEP tab lets you customize STEP settings.
  • the VRML tab lets you customize VRML visualization settings.
  • the ICEM External I/O tab lets you customize options for the import and Export of ICEM databases.
  • the 3DEXPERIENCE platform tab lets you connect to 3DEXPERIENCE platform.
