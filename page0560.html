Setting Standard Parameters  

Aerospace Sheetmetal Design parameters are located in the Standard  >  Aerospace Sheetmetal Design  >  Characteristic Curves node of the standard XML file  (available via Tools  >  Standard  >  General category  >  AerospaceSheetMetalDesign.xml file):

  • Graphic Properties
  • Visibility
    • Flange Features
    • Stamping Features

    Visibility parameters apply to folded and unfolded 2D views (Generative Drafting workbench).
    Users can manage the visibility of characteristic curves in 3D views through settings. Refer to Display Settings in the Customizing chapter for more information.

Refer to Before You Begin Setting Standard Parameters for more information on how to set standard parameters.

 

Graphic Properties
 

Characteristic Curves

Parameter Name

 Value

Description

BTL Base/ BTL Support/ IML/ OML/ Second OML/ CLB		 Color Color Defines the color of the characteristic curves (BTL Base/ BTL Support/ IML/ OML/ Second OML/ CLB).
Linetype Integer Specifies the linetype of the characteristic curves (BTL Base/ BTL Support/ IML/ OML/ Second OML/ CLB).
Thickness Integer Specifies the line thickness of the characteristic curves (BTL Base/ BTL Support/ IML/ OML/ Second OML/ CLB).
Trimmed Yes/No Default value is No. Set this parameter Yes to display characteristic curves on unfold view of drawings from parts that are trimmed by the unfolded view of the sheetmetal solid.
Broken Display Yes/No Default value is No. Set this parameter Yes to display two parts of the characteristic curves on unfold view of drawings, by breaking the curves for a given length from the start and end vertices of the wire. This parameter will be taken into account for a given characteristic curve only if its Trimmed parameter is also set to Yes.
Breaking Length in mm Specifies the distance to split the corresponding characteristic curve if the Broken Display parameter set to Yes.
Clearance Yes/No Default value is No. This parameter will be taken into account for a given characteristic curve only if its Trimmed parameter is also set to Yes.
Clearance Length in mm Specifies the distance to split the corresponding characteristic curve if the Clearance parameter set to Yes.
