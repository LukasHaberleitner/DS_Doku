Editing the Pad

You may like to make the pad thicker. This task shows you how to edit the pad and how to color the part. 

  1. Double-click Pad.1.
    You can do it in the specification tree if you wish.

  2. In the Pad Definition dialog box that appears, enter 90 mm as the new length value.

  3. Click OK.
    The part is modified accordingly.

  4. Now select Part Body.

  5. Select Edit > Properties and click the Graphic tab to change the color of your part.

  6. Set the color of your choice in the Color combo box and click OK.
    To have details about how to change graphic properties, see Infrastructure User's Guide Version 5.
