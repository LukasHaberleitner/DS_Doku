About Creating Joints

Defining Machine Kinematics Using V5 Data and Using Rigid Joints provides information about the algorithm for creating three kinds of joints:
  • rigid
  • prismatic
  • revolute

The complete list of available joint types appears below.
You can create 11 joint types as shown in the following table:
Revolute Roll Curve 
Prismatic Slide Curve
Cylindrical    Point Curve 
Spherical  Point Surface
Planar Universal
Rigid   

In addition to these joint types, there is a method of creating joints using frames that relies on Joint from Axis .
