V4 / V5 TUBING

This tab deals with the following option:

Migration Parameter:

By default, this option is not selected.

When option is not selected, for every physical Part a function is created in Spec-Tree, usually it is not necessary and it can be deleted manually. If you manually delete the Functions and save the data, the functions will not be visible anymore.

When the option is selected, the unwanted tubing functions are not generated or created at all.
