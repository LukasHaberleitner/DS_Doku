Working with a Part and a Skeleton  

 

This task describes the methodology for working with a Part and a Skeleton.

Updating the Part and the Skeleton in the Database

Your objective is to import the file-based modifications into the database.

  1. Apply the Overwrite rule to the Part.

  2. Apply the Overwrite rule to the Part.

  3. Save the documents in the database.


Updating the Part and the Unmodified skeleton in the Database

  1. Apply the Overwrite rule to the Part.

  2. Apply the Reload rule to the Part.

  3. Save the documents in the database.

Updating the Part and the Modified skeleton in the Database

  1. Apply the Overwrite rule to the Part.

  2. Apply the Reload rule to the Part.

  3. Save the documents in the database (or Save As New version).
