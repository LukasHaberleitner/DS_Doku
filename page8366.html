Changing the Nationality of an Manikin

 There are two ways to change the nationality of a manikin:
 
  • In the Human Builder workbench through the Properties dialog box

  • In the Human Measurements Editor workbench through the Population command

Human Builder workbench > Properties dialog box
  1. Access the manikin Properties dialog box. To do this, select the manikin, then select Edit > Properties from the main menu.
     
  2. Use the pull-down menu to select the new Population of the manikin.

     
     
  3. Select Apply to complete the operation.

Failure to press the Apply button will cause the operation to remain incomplete; the new population will not take effect.

Human Measurements Editor workbench  > Population command
 
  1. Select Population from the Anthropometry Editor toolbar. 
  2. Select the new population in the Population dialog box.

     
     
  3. Select OK. The manikin is updated with the new population.
