Shop Fabrication Design Rule

This task explains the shop fabrication design rule.

The shop fabrication design rule included with this application is used by the welding rule to obtain a value, as explained below.

This is a specification-related design rule and the associated table is located in the directory ...\intel_a\Startup\EquipmentAndSystems\XXXXX\Specification where XXXXX is the application name, such as Tubing.

1. The image below shows the shop fabrication design table for the Tubing Design product. Tables for other applications will have different entries but function in the manner described below.

The columns in the table contain the input values. Only one nominal size value is input - the columns MinNominalSize and MaxNominalSize constitute a range, inclusive of the values in the two columns. Thus, an input of 2 inches is a match for this table, 3.5 inches is not.

When the application can match the values in this table the value SHOP is the output. For sample purposes, the meaning of the value SHOP is that the parts should be welded in a shop.

If the application does not find a match in this table then the value FIELD is the output. For sample table purposes the meaning is that the parts can be welded in the field.
