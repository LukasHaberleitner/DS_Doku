Stamping

This section explains and illustrates how to create and use various kinds of stamps.

  • Stamps must be created on walls, or walls on edge, except for the stiffness rib that is to be created on a bend. 
  • If a stamp is created over the limit between several supports, such as walls, bends, and so forth, this stamp is not visible on the unfolded view.
  • When unfolding a part, only the largest imprint of the stamp is retained on the stamped wall.
  • Stamps cannot be created on an unfolded part.
