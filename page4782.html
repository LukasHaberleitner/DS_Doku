Setting GVS Parameters for Mesh Drawings

Refer to the Generative Drafting User's Guide to learn how to manage and customize generative view styles using the Standards Editor.

GVS parameters for mesh drawing can be defined in Tools > Standards, Category:generativeparameters and File:DefaultGenerativeStyle.xml, under the node Cloud:

Parameter    

 

Value

 Description
MaxNbOfPoints   Integer Defines the maximum supported cloud size (in number of points)
Point Display Yes/No Determines if points are extracted in drafting
(default value is No)
  MaxNbOfPoints Integer Defines the maximum number of extracted points
(default value is 1.000)
Scan Display Yes/No Determines if scans are extracted in drafting
(default value is Yes)
  MaxNbOfSegments Integer Defines the maximum number of segments in a scan
(default value is -1: no limit)
Grid Display Yes/No Determines if grids are extracted in drafting
(default value is No)
MaxNbOfPoints Integer Defines the maximum number of points in a grid
(default value is -1: no limit)
Mesh Display Yes/No Determines if meshes are extracted in drafting
(default value is No)
MaxNbOfTriangles Integer Defines the maximum number of triangles in a mesh
(default value is -1: no limit)
