Adding a User-Defined Method Based on Optimization
Using Thoracic and Lumbar Angle

This task describes how to add a user-defined method based on optimization using thoracic and lumbar angle.

  1. In the User Defined Method panel of the Tools > Options > Human Builder > Vehicle Occupant Accommodation tab, click the Add button.
    The User-Defined Method Definition dialog box appears.

  2. In the Prediction Based On area, select Optimization.
    The Seat Back Angle check box appears. By default, it is not selected; if it is selected, clear it.

  3. In the Name field, type the name you want to assign to your new user-defined method.

  4. Click on the f(x) [Function] button near the parameter you want to modify.
    The Formula Editor the dialog box appears.

  5. Enter a formula in the Formula Editor  then click OK.
    The User-Defined Method Definition dialog box reappears with the value of the parameter updated, and the signature of the formula appears in the textbox near the edited parameter.

  6. Repeat the steps from inserting a function through to this step for all the parameters you want to modify.

  7. Click OK.
    The User Defined Method panel reappears with the added method appended to the end of the list.

  8. In Tools > Options, click OK  for the definition to persist in the CATSettings file.
