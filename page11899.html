Defining Maximum Machining Feedrate

This procedure describes how to define maximum machining feedrate for a part operation as well as for a specific machining operation.

From the samples directory, open the Processfinal.CATProcess file, or use your own data.

Define Maximum Feedrate for a Part Operation

 

  1. In the PPR tree, double-click on Part Operation.1.

    The Part Operation dialog box appears.

  2. Click Machine in the Part Operation dialog box.

    The Machine Editor dialog box appears.

  3. Input the desired value in the Max machining feedrate field and click OK.

    Errors (tool collision with the stock) are reported if the feedrate exceeds this value.
 

Define Maximum Feedrate for a Specific Machining Operation
 

  1. In the PPR tree, double-click a specific machining operation such as Tool Change.1.

    The machining operation (e.g., Tool Change.1) appears.

  2. In the dialog box, click the More button.
    The dialog box expands. The maximum machining feedrate can be set in the Feeds & Speeds tab.
    Errors (tool collision with the stock) will be reported if the feedrate exceeds this value.
