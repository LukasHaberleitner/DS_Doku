Dimensioning and Tolerancing a Slot 

This task shows you how to create tolerances in a series using the Tolerancing Advisor.

  1. Select the two parallel surfaces of the groove (use the Ctrl key to select the second surface).
    The Semantic Tolerancing Advisor dialog box is updated.
    The buttons and options available in the dialog box depend on your selection.
  2. Click the Distance Creation icon (Tab/Slot): .
    The Limit of Size Definition dialog box appears. Keep options as is.
  3. Click OK.
     The slot size specification is created.
  4. Click Add... in the Semantic Tolerancing Advisor dialog box.
    The Datum and Datum Systems dialog box appears.
  5. Select each datum and specify its box (primary, secondary, and tertiary)
  6. Click OK.
    The Semantic Tolerancing Advisor dialog box is updated.
  7. Select the slot size.
  8. From the Datum systems list, in the Semantic Tolerancing Advisor dialog box, select the required datum system (A|B in this case).
    The     Semantic Tolerancing Advisor dialog box is updated.
  9. Click the Position-with-DRF-Specification icon (Tab / slot): .
    The     Geometrical Tolerance dialog box appears.
  10. Enter the requried numerical value and select the required modifiers.
    The position tolerance is automatically grouped with the previously created size tolerance.
  11. Click OK.

  12. Click Semantic Datum (1 Plane) in the Semantic Tolerancing Advisor dialog box.
    The     Datum Definition dialog box appears.

  13. Under Datum/ Datum System Creation, select From this Datum feature.

  14. Click OK.

  15. Select another required element and click Semantic Datum (1 Plane) in the Semantic Tolerancing Advisor dialog box.
    The     Datum Definition dialog box appears.

  16. Select the option Create a DRF with the 3 last created datums.

  17. Click OK. You can see that datum C is created.

    The Datum Reference Frame A|B|C is seen in the specification tree also.
  18. Do not close the Semantic Tolerancing Advisor dialog box to perform the next task.
