CGM: Export

This task lets you quickly see how to export the data contained in a CATDrawing document into a CGM file.
Open the CATDrawing document to be exported into a CGM file.

  1. Select the File > Save As item.

    The Save As dialog box is displayed:

  2. Change the Save as type into CGM type.

  3. Enter the file name.

  4. Press Save.
