Creating Four-Points Patches

This task shows you how to create a four-points patch.
Open the  RoofStart.CATPart document. It contains windshields of the car, style curves limiting these (windshield boundaries) style curves, onto which the roof should be mapped, and curvature analyses applied onto each of these styling curves.
You can hide the Curvature Analysis for the moment.

  1. Check the Snap On Vertex icon from the FreeStyle Dashboard to snap onto the closest corner when moving the pointer close to the geometry.

  2. Click Unified Patch in the Surface Creation toolbar (Patches sub-toolbar).

  3. Click 4-Point Patch .

  4. Click the four style curve end-points (yellow  styling D curves) as limits to the roof patch.
    Because the auto detection capability has been activated, the end point is easily detected.
    The patch is created.
