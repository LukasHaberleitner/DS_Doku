BasicDevice (Object)

Interface representing a Basic Device.

Role: This interface is used to interact with devices that are available in the Device Building workbench. This includes devices created in both V5 and D5.
The following code snippet can be used to obtain a device in a CATProduct document. 
   Dim objDevice As BasicDevice
   set objDevice = CATIA.ActiveDocument.Product.GetTechnologicalObject("BasicDevice")
Method Index
GetDOFValues Get the DOF values for the device.
GetHomePositions Get list of home positions of the device.
SetHomePosition Set home position of the device.

Set home position of the device.
Parameters:
iName
This parameter contains name of home position .
idbTrans
This parameter contains distance data in Meters or angular data in degree radian.
