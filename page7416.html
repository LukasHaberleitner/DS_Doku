Grouping Annotations During Creation 

This task shows you how to group an annotation you are creating to an existing annotation.
Groups of annotations can gather as many annotations as you wish.
While creating an annotation, if you select any existing annotation, the new annotation is automatically added into the group of the selected one.
Open the Annotations_Part_04.CATPart document.

  1. Click Text with Leader in Annotations toolbar.

  1. Select "Surface Treatment" as the annotation to which you want to attach the annotation you are going to specify.

  2. Enter the text "Grouped Text" in the Text Editor dialog box and click OK.

    The two annotation text are grouped.
    If you need to edit the group properties, multi-select the annotations and use the Properties contextual command.
When creating an annotation by selecting a dimension, the annotation is associated with selected element in the dimension:
  • With the dimension value, the annotation is associative with the value and can be moved freely by manipulation.
  • With the dimension line, the annotation is associative with the line and can be moved along the line direction only.
