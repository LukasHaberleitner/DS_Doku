Dragging and Dropping Icons onto Objects

This task explains how to drag and drop icons onto objects to run commands, which is a quicker alternative to selecting commands or icons.
Open the document Manipulators.CATProduct.

  1. Activate "Manipulators2" under "Manipulators2(Manipulators2)" in the specification tree.

  2. Click Select to enter selection mode, if it is not already activated.

  3. Drag the Edge Fillet icon from the toolbar onto an edge.

    As soon as you point at the edge, it is highlighted and the pointer changes to this shape :

  4. Release the mouse button.

    This displays the Edge Fillet Definition dialog box, and you can proceed to create the fillet:

The drag and drop method of running commands is not available everywhere: refer to your workbench documentation for more details.
