Reordering Output Features  

You can reorder the output features contained under the Outputs node in the tree outside the Sketcher app.

  • Create a sketch with multiple 2D profiles.

  • Create more than one output features.

  • Exit the Sketcher app.

 

  1. Right-click the Outputs node in the tree and select the Outputs > Reorder Children.

  2. In the Reorder Children dialog box, select the output feature.
    To select the multiple output features, press Shift and select the output features.

  3. Click Up Arrow   to move up the selected output feature.
    The selected output feature is moved up to a new position.

  4. Click Down Arrow to move down the selected output feature.
    The selected output feature is moved down to a new position.

  5. Select one or multiple output features and click Move to destination  .

  6. Select the new position for the output feature in the output feature list.
    The selected output feature is moved to a new position.

  7. Click Ok to save the changes.
