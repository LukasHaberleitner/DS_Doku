Exporting Data (as IDF)

This task shows how to export data to an IDF file.

  1. Click the Export as IDF file button .

    The Export dialog box opens.

  2. Click the button.

    The idf File dialog box opens:

  3. Enter DemoExport in the File name field.

  4. Click Open.

  5. You are back in the Export dialog box. Select the Decimal Symbol to be used for the export.

    The Decimal Symbol makes the export totally independent of the language of the session.

  6. Click OK to perform the export.
    An Information message is displayed:

  7. Click OK to close the information window.
