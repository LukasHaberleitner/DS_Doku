Moving Along Edges

This task shows you how to translate vertices along the edges of the base mesh, i.e. the mesh lines.

Create a 3D shape containing a subdivision surface.

  1. Click Modification in the Modification toolbar.

  2. Click the surface.

  3. Click Mesh-lines in the Tools Palette.
    The vertex mode is activated.

  4. Select several vertices.

  5. Move them along the direction shown with the arrow.
    The nearest direction is selected. All vertices move at the same time, for the same length along their own edge.

    You can press and hold Alt and move vertices parallel to reference elements. This mode is only available if vertices belong to the same line on the same surface, and provided the Manipulation Law command is deselected.

  6. Release the left mouse button (and Alt, if pressed) when you are satisfied with the new position of the surface.
