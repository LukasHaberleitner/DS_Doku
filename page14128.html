Creating a Detailed Trim Line

This task shows you how to create a Detailed Trim Line from a Concept Trim Line. This reverts to creating the trajectory of the cutting tool from the line that cuts the part.
Open DetailedTrimLineP1.CATPart from the samples directory.

  1. Optional: Right-click a Geometrical Set to create the design inside the OP Detailed Part, and select Define in Work Object in the contextual menu. Detailed Trim Line uses the correct axis as Stamping direction or projection plane.

  2. Click Detailed Trim Line in the Trim Line  toolbar.


  3. Select the Concept trim line.
    It must be a mono-domain curve.

  4. Select either the Stamping direction or projection plane.
    The concept trim line is projected on the selected plane, or on the plane defined by the Stamping Direction.

  5. Select the Surface to cut.

  6. Select a Reference elements (Optional) to stabilize the selected extremity.

  7. Alternatively, select the Automatic reference element check box.
    Detailed Trim Line computes the best possible reference element to stabilize the design change in case of rotation or replacement.
    When you select a detailed trim line to comppute the second extremity, Reference element is automatically populated, and the Reverse direction check box is selected.

  8. Select an Extension Type.
    Graphic help is available for the associated parameters.
    • With notch: The Detailed Trim Line cuts through the Addendum Surface, with a notch, with possible extrapolation lengths.

      The required parameters are
      • Outline angle
      • Notch radius
      • Notch depth
      • Length between notches
      • Variable radius 1 and Variable radius 2
      • Concept TrimLine extrapolation length
      • Detailed TrimLine extrapolation length to cut the metal all along.
    • Without notch: The Detailed Trim Line cuts through the Addendum Surface, without a notch, with possible extrapolation lengths.

      The required parameters are
      • Outline angle
      • Corner radius
      • Distance to trim line end
      • Concept TrimLine extrapolation length
      • Detailed TrimLine extrapolation length to cut the metal all along.
    • Straight extrapolation: The Detailed Trim Line is a straight extrapolation of the Concept Trim Line.

      The required parameter is Detailed TrimLine extrapolation length.
    • Extrapolation tangent on parallel: The Detailed Trim Line is an extrapolation tangent to a parallel created at a distance to the concept Trim Line. The tangent extrapolation intersects the boundary of the last cut operation.

      The required parameters are
      • Parallel distance
      • Detailed TrimLine extrapolation length.

  9. Verify that the point indicated as the modified extremity is correct.
    If not, select another from the Select the extremity list.
    If you select a Reference element, the first extremity is the nearest to the reference element. The second extremity is the furthest.

  10. Verify the direction of the detailed trim line.
    Select the Reverse Direction check box if required.

A Detailed Trim Line feature is created. It contains the parameters used for the creation.

The Detailed Trim Line can be applied as the detailed geometry of the corresponding tool in the corresponding detailed OP.
The geometry is imported to the result part process, using Link Detail to Result.
