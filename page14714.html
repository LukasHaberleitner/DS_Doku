Creating Mesh Parts

One mesh part is created for each split object.

All the mesh parts will be shown in the specifications tree under Nodes and Elements. Two categories are created, 2D Surface and 1D Beam Element.

Plates
  A Surface mesh part is created for each split plate.
 
  Constraints for a given mesh part are aggregated under this mesh part in the specifications tree.

There are two categories of constraints: Projected Curve constraints and Projected Point constraints.
 

Stiffeners and Stiffeners on Free Edge
  A 1D mesh part is created for each split stiffener. The support of the mesh part is obtained from the trace of the split stiffener.

Because the plates are constrained with the trace of the stiffeners, the compatibility between the 2D mesh part (plate) and 1D mesh part (stiffener) is assured.

Beams and Pillars
  A 1D mesh part is created for each beam and pillar. The support of the mesh part is considered as the trace of the beam/pillar.

Each beam/pillar is constrained by points as a result of intersecting with other traces and/or with other plates.
