Setting Speed and Acceleration for Joints 

NC Machine Tool Builder allows the user to set the speed and acceleration of each joint.
Open the V4Axis_Machine.CATProduct file from the  online/mtbug_C2/samples/V5/4AxisMachine directory.  In addition:
  • add the new machine
  • add the machine kinematics definition
  • create mount parts and points
  • create travel limits

  1. On the specification tree, select the V4Axis_Machine.

  2. Click Speed and Acceleration Limits .

    The Maximum Speed/Acceleration dialog box is displayed.

  3. For each joint, type the value or use the arrows to set a maximum speed or acceleration.

  4. Click OK.
