Performing an Easy Query

This task shows you how to perform an easy query in the Mapping Groups Tree. Each Mapping Group Type can contain other Mapping Group Types.

The easy query allows you retrieve the most recent revision of an object from the database that satisfies the query criterion.

Mapping Groups

Multiple mapping groups may appear under one Mapping Group Type, but each mapping group is dedicated to a specific Reconciliator query/queries.

For example, under the Reconciliator Query Mapping Group Type, there are two mapping groups, Generic Query and Easy Query as shown below:

The Easy Query mapping group is used by the Reconciliator Query.

The Generic Query mapping group is used by queries basked on the Latest Available Revision and Interactive Latest Available Revision queries.

Similar changes have been done in the Reconciliator Query Result mapping group type.

Interactive Easy Query

The Interactive Easy Query can be accessed in the Reconciliator Window from the Query Criteria drop down list.

The Interactive Easy Query is available by default on CXR20 and above provided that the proper SmarTeam Integration Tools Setup operations are performed. See Prerequisites for details.
