Shelling the Part

Towards the end of the scenario, you will learn how to shell the part.

  1. Select the bottom face of the part.

  2. Click the Shell icon .
    The selected face turns purple and the Shell Definition dialog box appears.

     

  3. Enter 5mm as the inner thickness value.

  4. Click OK to shell the part. 
    You have defined a positive value, which means that you are going to obtain a thin part thickness.
For more information about shells, see Creating Shells.
