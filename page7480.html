Adding Global Attributes 

You can add global attributes in a text.

The global attributes are displayed as a string that is replaced by the actual value of that global attribute while printing or exporting.
Open a 3D part.

  1. While creating a text annotation, right-click in the Text Editor dialog box and select Insert Global Attribute.

     

  2. In the Insert Global Attribute dialog box, select a placeholder display from the following:
    • *: Stores the attribute in the form of an asterisk.
    • $$Attribute: Stores the attribute in the form of $$ followed by the name for that particular global attribute.

  3. Select an attribute from the following and click Apply:
    • User
    • Time
    • Date
    • Day
    • Month
    • Year

    To add several attributes, keep on selecting the required attributes and clicking Apply.
    Except the user attribute, the values of all the attributes are according to the local settings of the computer.
