Parabolic Quadrangle

Parabolic Quadrangle is a eight-nodes surface element based on the Reissner/Mindlin theory.

 

Type

surface element

Physical property

shell
membrane

Mesh connectivity

QD8

Number of nodes

8

Degrees of freedom
(per node)

6 (3 translations and 3 rotations)

Type of behavior

elastic
 

This element has four gauss points:

P1 (-1/ ; -1/)               P2 (1/ ; -1/)

P3 (1/ ; 1/)                  P4 (-1/ ; 1/)
