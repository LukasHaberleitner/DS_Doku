Parameter Tolerance  

This tab allows you to specify the default tolerance of Angle and Length type parameters. Angle and Length type parameters can be assigned a tolerance. To do this, edit the parameter, right-click the value field then select Add Tolerance... from the contextual menu. The default values which are displayed in the tolerance dialog box are those you have specified in this tab.

This page deals with the following categories of options:

  • Tolerance
  • Length
  • Angle

Tolerance

Tolerance
When checked, enables you to specify default values.

By default, this option is unchecked.

Length

Maximum tolerance
Enables you to assign a maximum tolerance.

By default, the default value is 0 mm.

Minimum tolerance
Enables you to assign a minimum tolerance.
By default, the default value is 0 mm.

Angle

Maximum tolerance
Enables you to assign a maximum tolerance.

By default, the default value is 0 mm.

Minimum tolerance
Enables you to assign a minimum tolerance.
By default, the default value is 0 mm.
