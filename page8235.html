Generating a Status Report

Right-click on the Manikin to get the contextual window.  Select Manikin1 Object > Generate Status Report...


In the Status Report you can select the items that you want to save in the report.  By default, all the items are checked.  You must specify the output file name, the directory and the file format for the report.
This command does not work when selecting a forearm model manikin, an error message will pop up.
