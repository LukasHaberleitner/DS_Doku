Saving Selected Product Data as a CATProduct  

Selected products in a CATProcess document can be saved as a CATProduct file. The respective representations and graphical properties of the selected products saved into the resulting CGR. You can choose to save Manufacturing Assemblies in addition to selected products.

  1. Select one or more products in the PPR tree. Selections can be made from both the product list and the resource list.

  2. Click Export as CATProduct .

    The Save in a new CATProduct panel appears, displaying the list of selected objects: 
     

  3. Click Preview to see a graphical preview of the selection.

    Note: If a representation or graphical property of the selected product exists only transiently, e.g. inside another command, the representation or graphical property is not saved. Anything that is not visible in the window is not saved.
     

  4. Select Close to dismiss the Selection Preview dialog box.

  5. Select OK Save in the Save in a new CATProduct dialog box. A new V5 product document is created with the product content, and loaded in Product Structure.

  6. Select File > Save As to save the new CATProduct.
