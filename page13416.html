Splitting Geometry

This functionality is only available with Sheet Metal Design.

This task shows how to create a split normal to a sheet metal part by means of a cutting surface.

Open the Split1.CATPart document.

  1. Click the Split icon . The Split dialog box appears.

 

  1. Select the Splitting Surface.

  2. Select the Reference Side of the element to split.

 
The red arrow shows the side to keep

  1. Click the Inverse Side button to keep the other side.

  1. Click Apply to preview the split: the side to be removed is highlighted in green. 

  1. Click OK to split the element.

The created element (identified as Split.xxx) is added to the specification tree.

You can create a split using the unfolded view of the part.
