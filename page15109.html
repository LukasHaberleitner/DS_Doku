Working with ENOVIAVPM

 

This task describes the methodology for working with ENOVIAVPM .
 

Updating an Assembly/Assemblies and all the Parts in the Database and Creating the Container in the Database

  1. Apply the Overwrite rule to the Product.

  2. Apply the Overwrite /Reload rule to the Parts.

  3. In the Actions toolbar, enter an applicative container name and click the Save Application Container icon .

  4. Save the documents in the database.
