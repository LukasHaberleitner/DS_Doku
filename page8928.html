Inverting the Normal of a Subdivision Surface

You can invert the normal of meshes or subdivision surfaces.
Create a 3D shape containing a subdivision surface.

  1. From the Styling Surfaces toolbar, click Reverse Normal .

    You can also access this command by pressing Alt+Shift+I.

  2. Select one or more subdivision surfaces.

    The Tools Palette is displayed:

    • Select Subdivision : Changes the surface selection.
      You can also access this command by pressing Ctrl+Shift+J.
    • Reverse Side : Inverts the normal for all subdivision surfaces.
      You can also access this command by pressing Ctrl+Shift+I.

    A normal arrow is displayed for each subdivision surface.

    Default normal Inverted normal

  3. Click Reverse Side or the arrow to reverse the normal.

    This command can be launched while working in another command.
  4. Now apply another Styling Surfaces command such as Extrusion .
