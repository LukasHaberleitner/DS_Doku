Using Document Windows

This task explains how to use document windows.

  • Select Window > New Window.

    A new window is opened containing the document you are editing.
    The remaining commands on the Window menu let you organize your windows horizontally or vertically with respect to each other, so that they do not overlap, (Window > Tile Horizontally and Window > Tile Vertically) or in a cascading arrangement in which they overlap each other (Window > Cascade).

    Note that you can switch from one document window to another by selecting the window name at the bottom of the Window menu.

 

About Minimizing and Maximizing Windows

In the top right corner of the window, click:

  • Minimize to minimize the size of the window. In some cases, this means to hide the window.
  • Maximize   to restore the window to it largest size. The restore process is based on the window's height (not on its width) and objects displayed in the window are zoomed accordingly.
  • Restore Down to make the window back to its previous size and location. This icon is only displayed when the window is at its largest size.
 

About Filenames

The size of filenames displayed at the bottom of the Window menu must not exceed 32 characters (without the extension) and the extension part after the dot (.) should not exceed 13 characters. Otherwise, filenames are truncated with ".." in between.
