Measuring a Slot Feature  

This procedure describes how to measure a slot feature.  A slot feature measure will appear as an activity of the process on the PPR tree.
To measure a slot feature, you must have a product loaded and a slot defined.

  1. Select the parent process or preceding activity for the measure slot activity. 

  2. On the Inspection Activities toolbar, click Measure Slot.

  3. Select the defined slot you wish to measure.

    The Measure Slot dialog box appears.

  4. Alter the information in the dialog box as desired.

    (A preview of the path appears as you alter the data).

  5. Once you have defined the measurement desired, select the OK button.

    The activity is added to the PPR tree.
    Note that creating a measure slot activity also creates a tag list resource (which consists of slot path groups) and slot path applications.
    The measured slot appears highlighted on the part geometry.
If you create this activity before you insert a CMM into the resource list of the PPR tree, you will be asked to confirm that you wish to create the activity without a CMM assigned to it.  You can assign the CMM to all the activities within the process once you are ready to insert the CMM.
  When the slot is defined as having thickness, then the offset is ignored while creating the measure path.
  When an outer slot is being measured, goto points will be added automatically to avoid a collision.
