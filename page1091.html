Documents

This page deals with the Documents tab options:

  • Drawing Documents

  • IPM Documents

  • When Creating IPM

  • IPM Instance Documents

Drawing Documents

Store extracted Drawings at the same location as the Process

This option lets you store all the extracted Drawings at the same location as the Process file containing the IPM they are extracted from.

Store Drawings at the PPR Hub Shared Location root

This option lets you to store all the extracted Drawings at the PPR Hub shared location root.

IPM Documents

Store IPMs at the same location as the Stock Part

This option lets you store all the IPMs at the same location as the Stock Part they come from.

IPM Documents in PPR Hub context

This option lets you store all the IPMs at the PPR Hub shared location root.

When Creating IPM

When Create IPM from option is selected, the Browse button is active and you can select the user-template CATPart document to be used for IPM generation.

By default, Create IPM from option is not selected and user-template location is empty.

IPM Instance Documents

IPM Instance Documents

This option lets you store all the IPM instances at the same location as the Stock Part they come from.

IPM Instance Documents in PPR Hub context

This option lets you store all the IPM instances at the PPR Hub shared location root.
