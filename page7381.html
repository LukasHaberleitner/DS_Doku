Creating Note Object Attributes 

  Note Object Attributes (NOA) lets you create a customized annotation from a text or a 2D component.
When extracting a Note Object Attribute to 2D, frames are not generated.
Note Object Attribute From a Text: click this icon, select a geometry then enter a text.
Note Object Attribute From a Ditto: click this icon, select a geometry then select a 2D component from a catalog.
  Store a Note Object Attribute into a Catalog: create a new catalog, select the Note Object Attribute.
