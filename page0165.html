Fields

Predefined fields are time-dependent, non-solution-dependent fields that exist over the spatial domain of the model. Abaqus for CATIA V5 supports the definition of initial temperature fields for thermal analysis.

Creating Initial Temperatures: Applies an initial temperature to a geometry selection.

You apply environmental actions, such as fields, to supports (geometrical features) on your model: points/vertices, curves/edges, surfaces/faces, or volumes/parts. You can either select the support and then set the field specifications or set the field specifications and then select the support. Table 10–3 summarizes the supports to which an initial temperature field can be applied.
