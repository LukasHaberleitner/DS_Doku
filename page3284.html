Altering PERT Chart Layout

This procedure shows how to alter the layout of the PERT chart.

  1. On the PPR tree, select a process.

  2. Click Open PERT Chart .

    The PERT Chart window appears:

  3. Click Layout PERT Chart in the PERT Tools toolbar.

    The Layout PERT Chart dialog box appears.
    The value for X provides the distance in mm between nodes placed horizontally; the value for Y provides the distance in mm between nodes placed vertically.

  4. Enter the desired spacing and click OK.

    The PERT chart reflects the change when the x value is entered:
