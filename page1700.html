Add Type Behavior: Data Sheet

Accessing this behavior This behavior can be accessed by clicking the Add Type Behavior icon ().
Describing this behavior This behavior is designed to associate technology with an object. The selected type is to be specified, as well as the .CATfct file in which the type is defined and the object to extend.
More information Note that the output feature of this behavior is the object being typed.
Using this Behavior Interface

Name:  Name attributed to the behavior.
Comment: Comment attributed to the behavior.
Type Name: Enter the name of the technological object.
...: Enables you to select a main behavior in the Exposed main behaviors browser. 
Object to Type: Select the object to type in the Classes View or in the Behaviors tab.
Using this behavior The expert user created a product and wants to associate the Vehicle type with this product.
