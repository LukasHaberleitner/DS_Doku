Infrastructure

Before you start your first working session, you can customize the way you work to suit your habits.
This type of customization is stored in permanent setting files: these settings will not be lost if you end your session.

  1. Select Tools > Options.
    The Options dialog box opens.

  2. Select the Infrastructure category in the left-hand box.

  3. Click the workbench to be customized.

    Product Structure
    Material Library
    Catalog Editor
    Photo Studio
    Real Time Rendering
    Part Infrastructure
    3D Annotations Infrastructure
    Collaboration Infrastructure

  4. Set options in the tabs according to your needs.

  5. Click OK when done.
