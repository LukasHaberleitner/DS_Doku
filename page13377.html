Simplification of Result Faces

Due to the enhancement in Wall and Wall on Edge commands, the result is improved by simplification of result faces. The Sheet Metal design workflows are now improved.
To check the below scenarios, you must use data created in CATIA Sheet Metal Design application.

1. Connected wall with simplification

1.Launch Wall command
2.Select valid sketch connected with previous Wall feature
3.Select face in ‘Connect to’ selector
4.Validate the command




2. Tangent Wall on Edge result with simplification

 1. Launch Wall on Edge command
2. Select input edge.
3. Uncheck ‘With bend’ check box
4. Change angle value to 180 deg
5. Validate the command




3. Wall on Edge with clearance result with simplification

1. Launch Wall on Edge command
2. Select input edge.
3. Select mono-clearance is clearance combo and enter 20 mm
4. Validate the command

4. Sketch based Wall on Edge result with simplification

1. Launch Wall on Edge command
2. Select input edge
3. Select valid sketch connected with previous Wall feature
4. Uncheck ‘With bend’ check box
5. Change angle value if needed so that support face and Wall on Edge result face are in same plane
6. Validate the command

 


5.Wall on Edge (multiple edges) result with simplification

1. Launch Wall on Edge command
2. Select multiple input edge
3. Select mono-clearance is clearance combo and enter 20 mm
4. Validate the command




6. Pattern of connected Wall or Sketch based Wall on Edge result with simplification

1. Launch Pattern command.
2. Select Wall or Wall on Edge having simplified geometry.
3. Create a valid pattern.
4. Validate the command and resultant topology.






7. Mirror of connected Wall or Sketch based Wall on Edge result with simplification

1. Launch Mirror command.
2. Select Wall or Wall on Edge having simplified geometry.
3. Create a valid mirror.
4. Validate the command and resultant topology.
  • Result faces won’t simplify for Wall or Wall on Edge created on non-planar support. For example, connected Wall created on Rolled Wall
  • Pattern or Mirror of Wall or Wall on Edge feature (for which simplification of result faces took place) in a view different from creation view of input feature’s view, is not allowed.
