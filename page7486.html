Customizing

  Before you start your first working session, you can customize the way you work to suit your needs.
This type of customization is stored in permanent setting files: these settings will not be lost if you end your session.

  1. Select the Tools > Options command. The Options dialog box opens.

  1. Select the Mechanical Design category in the left-hand box.

  1. Select the Functional Tolerancing and Annotation sub-category.

    Eight tabs are displayed:
    • The Tolerancing tab lets you set the tolerancing options.
    • The Display tab lets you define the annotation display options.
    • The Constructed Geometry tabs lets you set options for constructed geometry.
    • The Manipulators tab lets you set the manipulator options.
    • The Dimension tab lets you set the dimensioning options.
    • The Annotation tab lets you the annotation display options.
    • The Tolerances tab lets you set the other tolerancing options.
    • The View/Annotation Plane tab lets you define the view/annotation plane options.
    • The Administration tab lets you define the administration options.

  1. Select the Infrastructure category in the left-hand box.

  1. Select the Product Structure sub-category.

    Various tabs are displayed, the Cache Management and Cgr Management tabs also interfere with Functional Tolerancing and Annotation.
    • Cache Management
    • Cgr Management

  2. Select the Part Infrastructure sub-category.
    Various tabs are displayed, the Display tab also interfere with Functional Tolerancing and Annotation.
    • Display

  3. Select the General category in the left-hand box.

  4. Select the Display sub-category.
    One tab also interferes with Functional Tolerancing and Annotation.
    • Navigation

  5. Select the Compatibility sub-category.

    One tab also interferes with Functional Tolerancing and Annotation.
    • 3D XML
