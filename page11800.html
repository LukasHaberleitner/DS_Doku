Using your Catalog

This task shows you how to use one of your own catalogs. 

  1. Open the MoldUserComponents.CATProduct file in the samples/catalog directory.

  2. Click Add User Component  . This opens the catalog browser.
    Click the catalog icon to access the catalog browser.

  3. Then Click the folder icon to browse your directories to reach the UserCatalog you have linked with your catalog in the previous task.

  4. The user component catalog is displayed with the MyComponent chapter that you created.

You can now use the components in your catalog.
