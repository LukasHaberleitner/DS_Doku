Setting the Attenuation Factor for a Curve

This task shows you how to set the attenuation for the manipulator's displacement.
See General Settings for more information.
 

  1. Click Modification .

  2. Select the curve.

  3. Click Transformations .

    The default Attenuation   is set to no attenuation.
    You can click the icon to allow an attenuation .
