Connecting Segments at Branch Points

This task explains how to branch any type of bundle segments and how the bundle segment update is performed in case of route modification.
You can connect either branches or single bundle segments.

  1. Add several branch points to the branch.
    The branch is divided into as many bundle segments.

    The geometry looks like this:

  2. Click Multi-Branchable Document or Bundle Segment to create new bundle segments.

    The Electrical Harness Assembly workbench switches to the Electrical Harness Installation workbench.
    The dialog box opens:

  3. Change the parameter value of your choice, if need be.

  4. Click Route Definition. Refer to Routing Bundle Segments.

    After you have created two multi-branchable documents, the result looks like this:

  5. Using the compass, change a connector location.

    You now need to update the geometry.

  6. Click Update All .

The result looks as follows:
  • The main branch is recomputed as well as the branch point coordinates.
  • The branched bundle segments have also been recomputed.
