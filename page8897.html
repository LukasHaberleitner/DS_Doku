Creating a Stripped Surface

You can create a stripped surface from sections drawn on a support and assemble them to efficiently rebuild the subdivision surface.
Create a 3D shape which acts as a support surface.

  1. From the Creation toolbar, click Strip Drawing .

    The Tools Palette is displayed:

    Options Description Shortcuts
     Added Surface Selection Adds surfaces to the selection. Ctrl+Shift+J
    Add Strip Switches from creation mode to modification mode. Ctrl+Shift+M
    Erase Erases sections of the stripped surface in creation mode and surfaces or edges in modification mode. Ctrl+Shift+D
    Constant Dimension Defines the section size using a slider. Ctrl+Shift+C
    Automatic Sections Defines an automatic orientation of the sections. Ctrl+Shift+P
    Apply Applies the command. Enter
     

    Working Zone Activation/Deactivation : If this option is selected, only the elements selected under working zone are considered for manipulation.
    This is option is available only when a working zone is defined using the Working Zone Definition command.
    For more information, see Defining the Working Zone.

  2. Click to define a start point and then click to define an end point.

  3. Click Constant Dimension .
    A slider is displayed on the right. Drag the manipulator to adjust the section size.
    When both sizes of the section are different, the manipulated size is the biggest one, and the proportion is kept.

  4. Click Automatic Sections .
    Sections are oriented approximately parallel to each other.

  5. Click Apply .
    A new subdivision surface is created.

  6. Click Add Strip .
    The Tools Palette for modification mode is displayed.

    Options Description Shortcuts
    Projection View Computes the first projection that is found along the view direction.
    Toggles between Projection View and Projection Auto .    		 Ctrl+Shift+U
    Free Edge Mode Shows only the free edges. Ctrl+Shift+H


  7. Modify the strips using the pointer.

  8. In the modification mode, hover over near an edge of the strip and click in the work area to extrude the edge.
