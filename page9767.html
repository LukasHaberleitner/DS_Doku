CalibOffsets (Object)

Interface to set Robot base/Tool offsets based on Calibration.

Role: This interface provides methods to get/set calibration offsets related to Robot base and Robot tool position.
Method Index
ApplyCorrectedBaseOffset Applies corrected base offset.
ApplyCorrectedToolOffset Applies corrected tool offset.
ApplyNominalBaseOffset Applies the nominal base offset.
ApplyNominalToolOffset Applies nominal tool offset.
