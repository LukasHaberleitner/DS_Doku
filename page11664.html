Using NC Code for Machine Simulation

The procedures in this section describe how to simulate, create video replays, and analyze collisions using NC code.  This section describes how to run machine-only simulations; for information on how to simultaneously simulate machine and material removal, see Simulating Machine Motion and Integrated Material Removal. The actions described all apply to the manufacturing program and its operations.  Simulations or video replays created using commands other than those described in this section are referred to as tool-path-based simulations.

To simulate NC Code, a Controller Emulator (CE) -- valid for the NC machine being used -- must be assigned to the NC machine. Sample CEs, along with sample NC Machine models, are available in the startup/Manufacturing directory.
