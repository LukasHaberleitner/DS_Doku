Visualizing and Selecting Protected, Internal and Core Features

  While working on a part containing protected, internal and core features, the application provides you better visualization and selection options.

For this you need to right-click Solid Functional Set.x in the specification tree and select Solid Functional Set.x object > Behavior visualization. The Behavior visualization dialog box is displayed.

The following topics are discussed based on the options in the above dialog box:

  • Visualizing Protected, Internal and Core Features
  • Visualizing Features Depending on Material Availability
  • Selecting a Part of the Feature

Visualizing Protected, Internal and Core Features

You can select the appropriate options of visualization for the Protected, Internal and Core features. Selecting a Part of the Feature

You can select a part of the protected or internal geometry (e.g. edges, faces etc), and apply functional modifier on them.

For example, you can select the edge of the internal feature and apply functional fillet to it.
