Converting Tagged Profiles

This task shows you how to convert the tagged profiles and shapes into sub-assemblies like two plates, a plate and a flat bar, two flat bars, single plate or bent plate.
The profiles from Structure Functional Design and Structure Detailed Design cannot be directly converted into structural assemblies. Those profiles need to be tagged during creation. When these tagged profiles are generated using the Structure Piece Part Generator command, the attributes are transferred along with the generated piece in Structure Design workbench.
Open a structure containing tagged profiles of various sections. For more information about tagged profiles, refer to Tagging a Profile.

  1. In the Structure Design workbench, create the piece parts of the structural objects using the Piece Part Generator command.
    The piece parts are created.

  2. Click Convert Profile in Physical Plates and Shapes toolbar.
    The Convert Profile dialog box appears.

  3. Click Get Tagged Profiles.
    The dialog box displays all the tagged profiles.
    Their respective conversion mode based on assigned conversion attribute value is computed and displayed in the Conversion Mode column.

  4. Select one or more rows and click Remove if you want to remove the conversion of any profiles.

  5. Click OK in the Convert Profile dialog box.
    The Conversion Result dialog box appears displaying the conversion status of shapes.

  6. Click OK.
    The shapes are converted. The newly created structural parts are added in the specification tree and the converted profiles are renamed.

    • The newly created structural parts can be edited in SR1.
    • After conversion, the end cut information on the profiles are lost.
