Updating Analysis Connections

This task shows you how to update all analysis connections belonging to a connection manager set.

Only available with the Generative Assembly Structural Analysis (GAS) product.

  1. Right-click either the Analysis Connection Manager set to update all analysis connections or an Analysis Connections set to update all the analysis connections belonging to this set.

  2. Select Update All Analysis Connections .

    All the analysis connections belonging to the Analysis Connection Manager set are updated: the symbol disappears.
