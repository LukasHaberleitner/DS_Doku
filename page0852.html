VPM Navigator

This tasks explains how to define VPM Navigator settings.

  1. Select the Tools > Options... command.

  2. Click the ENOVIA V5 VPM category then select the VPM Navigator subcategory to display the VPM Navigator Properties tab:

Start Options

  1. In the Start Options section, click the Logon at start box if you want to log on immediately to VPM Navigator whenever you open a CATIA session.
    This means that you skip selecting the Connect icon . The logon box appears automatically.
    During Warm Start, Logon at Start option is deactivated. The login panel which appears is triggered by Warm Start itself.

Merge Mode

  1. In the Merge Mode section, click the Merge Mode box if you want to be able to work with a tree that combines certain aspects of both the VPM Navigator tree and the CATIA V5 Product Structure tree.
    A message appears telling you that the merge mode will be active the next time you open VPM Navigator.

  2. Click OK.

    Once you have selected the Merge Mode option, you can in fact see a Product Structure in merge mode in the Product tree without restarting VPM Navigator.
    However, you are strongly advised to restart VPM Navigator, otherwise this could result in data being shown only partially in merge mode.

Background Color

  1. In the Background Color section, click the combo to select a color other than the light blue default background.

  2. To access more colors, or to create your own colors, click the More Colors... option at the bottom of the list to access the color palette.
    For more information, see the appropriate section in "Displaying and Editing Graphic Properties" in the CATIA V5 Infrastructure Guide.

Publish/Subscribe

  1. In the Publish/Subscribe section, click the Enable Publish/Subscribe daemon box to activate the Publish/Subscribe functionality.
    Once you have done this, you can change the default interval of 3 minutes in the Timer interval box. This interval reflects the maximum time you have to wait between the event occurrence in the database and the interactive notification in CATIA. (The higher the value the better CATIA performance is.)

    Once activated, the Publish/Subscribe daemon will continue to poll the server at the specified interval regardless of what you are otherwise doing. You must therefore disconnect if you wish to stop the server.

    Bear in mind the following:

    • Performance may be adversely affected due to frequent polling if you set the timer to an interval lower than 15 minutes. Another way to avoid this is to use Select the Get Published Events command (see "Retrieving Published Events" in the "Publish/Subscribe" section of the VPM Navigator User Guide).

    • If the server timeout is greater than the selected timer interval, then no timeout will occur because the daemon will continue to ping the server and the server timeout will never be reached.

    • If the server timeout is less than the selected timer interval, then a timeout could occur.

Expand PRC on Open

  1. The Expand the PRC automatically box is selected by default. This means that when you open a PRC it is automatically expanded to one level.

    To deselect this option, just uncheck the box.

Stable Order

  1. Select the Order the VPM Navigator graph nodes box. The result of this is to order graph nodes by type i.e. PRC, GCO, instance, etc. and by name. Each of these ordered groups will be in alphabetical order.

    To deselect this option, just uncheck the box.

Versioning

  1. In the Versioning section, click the Authorize versioning depending on maturity status of Parts box if you want to enforce the maturity rules i.e. make sure that only those Parts that are in a frozen status can be versioned.
    By default this option is not selected.

  2. Select Authorize revisioning depending on maturity status of Documents check box to verify the maturity rules before creating a new version of the document.
    By default, this option is not selected.
     
    Note: If this option is not selected, you can create a new revision of the document even if the maturity rules are not fulfilled.

    For more information on Versioning, see "Versioning and Revisioning: Dos and Don'ts" in the "Versioning" section of the VPM Navigator User Guide.

Synchronization

  1. In the Synchronization section, there are four options to choose from to customize synchronization. You can select one of the following three options:

    • Propagate synchronization to all displayed instances which means all expanded instances are included in the synchronization operation.

    • Always prompt for propagation to all displayed instances (the default option) which means you will be systematically prompted to specify if you want to synchronize or not all expanded instances other than those selected.

    • Never propagate to other instances which means that only the instances you expressly select will be synchronized.

  2. You

    3. can also click Display Action/Modification Chooser if you want to be able to select a modification when you synchronize.

    For more information, see "Synchronizing" in the VPM Navigator User Guide.

Delete Objects

  1. In the Delete Object section, click the Authorize object deletion with impacted objects box if you want to allow the deletion of the instance and dependency links of Part versions or document revisions.
    By default this option is not selected.
    To deselect this option, just uncheck the box.
    For more information, see "Deleting Objects" in the VPM Navigator User Guide.

Save

  1. In the Save section, click Display Action/Modification Chooser if you want to be able to select a modification when you save.

Configuration

  1. Click the Display Specifications on Configuration Filter Definitions box to allow the display of configuration filter definitions specifications.
     

Expand in VPM Navigator

  1. Click the Expand the PRC with Hovering Capability to show attached Document Information box to activate the functionality that will allow you to see attached document information by hovering over an object.

Lock/Unlock Availability

  1. Click Only Easy Lock/Unlock to have only Easy Lock/unlock available, click Only Standard for Standard lock/unlock to be available and click Both for both the Easy Lock/unlock and Standard lock/unlock to be available.

Filter

  1. When you select Exclude Product Instances from Zone, Part and Graphical Bounding Box Filtering check box under Filter:
     Product instance is excluded from expanding further when you execute any volume(zone/part/graphical bounding box) filtering, even though the product instance is satisfying the boundaries defined for volume filtering criteria.

    Note: Since behavior of existing CATSetting is changed, you must delete the existing VPMNavigator.CATSetting before selecting this check box. A new CATSetting is created when you select Exclude Product Instances from Zone, Part and Graphical Bounding Box Filtering check box.

Refresh

  1. Click Do not apply expand on Product Instance node under Refresh to perform refresh only on the collapsed Product Instance node.

Graphical Properties in Preview

  1. Click the button for Apply Show/No Show on Instance after expand in the Expand in VPM Navigator section in VPM Navigator Properties tab to automatically apply the Hide/Show command and graphic properties like color, transparency as it is in ENOVIA V5 on part instance preview.
