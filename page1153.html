Action Libraries

This box lists action libraries available to users of Resource Detailing robotics products.  When you press the Add button, you get a navigation window that enables you select additional action libraries.  When you select a library and press the Remove button, you remove the library.

The initial setting is ~startup/DNBIgpSpotActionLib.act.

Actions View in the PPR Tree

The discrete actions that make up an operation, as defined in an action library, may appear as a full list or they may appear collapsed within the operation.

The initial setting is to have the actions collapsed in the PPR tree.
