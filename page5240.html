Creating an Opening for an Access Part

This task shows you how to create an opening for an access part. 
Creating an opening is similar to creating a graphic representation for a part. You must define and construct the part and one or more graphic representations of that part before creating the opening.
1. Click on the Build Access Part button to display the Create Part dialog box.
2. Click the Open Existing Part button , and navigate to the directory containing the part for which you want to create an opening.
3. Select the part.
4. Click on the Manage Graphic Representations button . The Manage Graphic Representations dialog box displays.

5. Select Opening, and toggle the no under the Defined column to yes. Do the same for the Activated column. This defines the opening for the access part and places it in the specifications tree.
6. Click the Expand representations checkbox so you can see all of the graphic representations in the specifications tree.
7. Click OK.

The Create Part dialog box displays. Click OK.
8. In the specifications tree, activate the opening you defined in the previous steps.

This opens Part Design.
9. Create the opening by doing the following:
  1. Click the Sketcher command.
  2. Select the yz plane.
  3. Expand the specifications tree so you can select the layout graphic of the part for which you are creating the opening.



    This creates the necessary link between the opening and the door.
     
  4. Click the Offset command, Select Sketch.1, as shown above, and drag your mouse pointer over to the door to create the opening.
10. With Sketch.1 still active, right-click and open the Properties dialog box:

Under Feature Name, enter a name for the opening.  The word "Opening" must appear somewhere in the name.

Click OK.
11. Exit the Sketcher, and select KRB_Door_99 to return to Compartment and Access.
12. Click on the Build Access Part button to display the Create Part dialog box.
13. Activate the buttons in the Create Part dialog box by clicking on the door.
14. Click on the Manage Graphic Representations button . The Manage Graphic Representations dialog box displays.
15. Select Opening, and toggle the yes under the Activated column to no. Click OK to exit the Manage Graphic Representations dialog and again to exit the Create Part dialog box.

You are now ready to add this part and its associated opening to your parts catalog.
