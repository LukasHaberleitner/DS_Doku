PPR Navigator 

This tab appears only if you have a DELMIA license.

This page describes the option that affects the behavior of the PPR Navigator. 

Lifecycle Options

The label effectivity criteria you create for an alternative resource can be applied to either:

  • the selected resource only

  • the selected resource and all of its related components (child resources, behaviors and their child activities, and output Manufacturing Assemblies).

Selecting the check box applies the effectivity criteria to all the components associated with the alternative resource.
The default value is that label effectivity is applied to all components associated with alternative.
