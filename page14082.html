Defining the Blank Material Properties

This task shows you how to define blank material properties: Split an existing blank geometry into several sections, and associate material properties to each section.
Open the StampingDirection01.CATPart from the samples directory.

  1. Click Blank Material Properties  in the Process Part Separation toolbar.

  2. Select
    • A reference direction, used to compute the rolling angle.
    • The blank to process, as a surface or a boundary.
    • The cutting elements that split the blank into several sections.
      If the blank type is constant, there is only one element, hence no need of cutting elements.

  3. Select the type of blank from the list under Blank Sections.
    • Constant
    • Tailor Welded
    • Tailor Rolled

    The sections are displayed in the table below.

  4. Select one line in the table and set the parameters values.
    • Thickness
    • Material
    • Rolling Angle, that is the angle between the X-Axis of the Reference Direction and the rolling directon of the blank section.
    If the bank type is set to Constant, all sections have the same parameter values, modifying one modifies all.

  5. Define a Thickness Direction.

A Blank Partition feature is created.
It contains a node for each Blank Section that in turn contains the parameters of the section.
