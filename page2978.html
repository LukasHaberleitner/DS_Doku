Opening a CADAM Drafting Model in V5

This task explains how to open a CADAM Drafting model in CATIA Version 5.
1. Click the Open icon  or select the File->Open... command.
2. In the File Selection dialog box, navigate to the appropriate folder.
 
You can filter the list of files to display only CADAM Drafting models by defining Files of type: cdd (*.cdd)
3. Select the model.
