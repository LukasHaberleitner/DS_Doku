Displaying Alternate Paths

This task shows you how to display alternate routes available to you when you are routing. See Routing a Run for more information about routing.
When you are creating a run you have the option of previewing alternate routes between the From and To points, including the shortest distance. You can use the Display Alternate Path button in the Run dialog box, or you can use the Shift key to display the paths. Both are explained here.

Normally the Display Alternate Path button is not available. It becomes available when you select the To location while creating a run, if the To selection is one of the following:

  • Part connector
  • Run segment
  • Run node
  • 3D point

The button is not available if your To selection is:

  • In space
  • A hole connector

If you want to display alternate paths when routing to a hole connector then use the Shift key, following the procedure explained in Step 4 below.

1. Select the beginning (From) and end (To) points of your route. Click once - if you double-click you will not be given a preview of alternate routes. In the image below the user has clicked once on the beginning point and ending point of his run, showing the path from start to end.

The minimum length value entered in the Min. Length field of the Run dialog box will be applied. No segment will be shorter than the minimum length value entered in that field.
  2. The Display Alternate Path button becomes available after the second (To) point of the run is selected (if it is a part connector). Clicking the button displays a field in the Run dialog box that shows the number of possible paths.

  3. Keep clicking the Display Alternate Path button to display the alternate paths. Once you have a route you want to accept, click OK or the Create the Run button and the run will be created.
  4. Using the Shift key to display alternate paths requires a slightly different procedure. Select the beginning point of the run, and then place the mouse pointer on the destination connector - do not click on it. With the mouse pointer on the connector, press the Shift key to display alternate routes. In the procedure below the Display Alternate Path button will not be available because the user is routing through a sleeve, which has a hole connector. In this case pressing the Shift key will display alternate routes.
