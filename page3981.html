Using Control Points Cross Diffusion Laws

When using the Control Points capability, you need to define cross diffusion laws. See also Using Control Points Diffusion Laws.
A cross diffusion law defines the type of deformation that is to be applied to the selected control points and mesh lines which are not directly manipulated:
  • Cross diffusion laws:
    • Cross Constant Law: deforms all the selected control points which are not directly manipulated identically.
    • Cross Linear Law: deforms all the selected control points which are not directly manipulated with a linear law.
    • Cross Concave Law: deforms all the selected control points which are not directly manipulated with a linear concave.
      This option is P2-only.
    • Cross Convex Law: deforms all the selected control points which are not directly manipulated with a linear convex.
      This option is P2-only.
    • Cross Bell Law: deforms all the selected control points which are not directly manipulated with a bell law.
      This option is P2-only.

    • Cross Constant Law gets applied as default law, when none of the above laws is selected.
    • The cross diffusion laws are available only if Mesh Only or Points and Mesh ,and any of the diffusion laws are selected.

  • To link/unlink the diffusion laws with the cross diffusion laws:
    • Link Laws: diffusion laws with the cross diffusion laws are linked, the selected cross diffusion law selects automatically the equivalent diffusion law.
    • Unlink Laws: diffusion laws with the cross diffusion laws are unlinked.

Cross Constant Law

Follow steps 1 to 7. Carry out the steps as given below.

  1. Click the Cross Constant icon:

  2. Drag the mesh line close to the cursor in the image below.
    • The deformation according to the diffusion law is applied to the selected control points of the selected mesh line.
    • The deformation according to the cross diffusion law is applied to the selected control points out of the selected mesh line.
    In the image below, all points lying on the selected mesh line and the other selected control points undergo the same translation.

  3. Click OK to validate the modifications and close the dialog box.

Cross Linear Law

Follow steps 1 to 7. Carry out the steps as given below.

  1. Click the Cross Linear Law icon:
    • The deformation according to the diffusion law is applied to the selected control points of the selected mesh line.
    • The deformation according to the cross diffusion law is applied to the selected control points out of the selected mesh line.
    The deformation progresses along a straight line between the most distant selected points out of the selected mesh line.

  2. Click OK to validate the modifications and close the dialog box.

Note: After application of one diffusion law on the selected control points, it is possible to apply multiple diffusion laws one after another on the same set of control points.
To apply multiple diffusion laws one after another on the same set of control points, clear the Delay Law Application check box under Tools > Options > Shape > FreeStyle > Control Points Diffusion/Cross Diffusion Law.
