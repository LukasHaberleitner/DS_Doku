Placing Annotation Symbols

This task explains how to place an annotation break symbol on a route.
To place an annotation symbol on a route in your 2-D document:
1. Click the Place Annotation Symbol button .

The Catalog Browser displays.
2. Select Annotation Symbols on the Catalog Browser, and select a specific annotation symbol.

Place the annotation symbol on the route.

You can place multiple annotation symbols on the route.
3. You can move an annotation break symbol along a route using its slide manipulator ().

Select the symbol and the slide manipulator displays.

Drag the slide manipulator along the route to move the annotation break symbol.
