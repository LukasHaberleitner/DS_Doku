Using the What's This? Command

 

This task explains how to use the What's This? command to get help on toolbar icons.

  1. Click What's This? in the Standard toolbar or select Help > What's This?.

    The cursor takes on the shape of the What's This? icon.

  2. Point to an icon, then click the icon.

    This displays a long help message in a box:

In some cases, What's This? gives you help on dialog boxes. To do so, just click the icon in the top right corner of the dialog box then point to the desired area to display the long help message.
