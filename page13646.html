Importing SDNF

This task explains the procedure for Importing SDNF.
SDNF stands for Steel Detailing Neutral File.

This command provides the capability to import the data from the neutral file format called as SDNF.. SDNF supports the exchange of a steel structure data between two applications.

1. With your document open, click the SDNF Import button. The SDNF Import dialog box displays.

Fill in data as explained below.

Enter the location of SDNF file to be imported, you may type in the editor box or click on browse to specify the location.

Options

Use Angle: You can select this option if the data to be imported is in Angle format.

You can select either Start Point to End Point or End Point to Start Point option, to get required orientation.

Swap Left and Right Side Anchor Points: You can swap the anchor point of the shape to be imported, by selecting option Swap Left and Right Slides.
  2. Click OK after specifying all necessary details from SDNF Import dialog box.
  3.

In case, the SDNF Import data is inaccurate or insufficient, a warning message is displayed specifying the path to the stored log file. You can refer the error log created to get details.
