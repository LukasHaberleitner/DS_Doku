What's New?

Enhanced Functionality

Colored Selected Faces in Union Trim Preview (V5-6R2024SP2)
The selected faces to remove and faces to keep are displayed in magenta and cyan colors respectively during the trimming bodies operation.
You can easily identify the selected faces when using the Union Trim command.

For more information, see Trimming Volumes.
Creating a Point on the Center of Gravity (V5-6R2024SP2)
You can create a point on the center of gravity of a volume or a surface using the Center of gravity point type in the Point Definition dialog box.

For more information, see Creating Points.
Zoom All Infinite Elements (V5-6R2024SP2)
The Zoom All Infinite Elements command is added in the Tools toolbar. Deselect this command to zoom in and zoom out the features with infinite geometry; for example, infinite planes.

For more information, see Tools Toolbar.
Displaying Analysis Results (V5-6R2024SP1)
You can now display the individual results of the Connect Checker Analysis on each comb's spike by hovering over the comb. With MinInfo and MaxInfo, you can display the local and global minimum and maximum values. A contextual menu is available to create points at these positions.

For more information, see Checking Connections Between Surfaces.
Save Position and Size of Historical Graph Dialog Box (V5-6R2024SP1)
The position and size of the Historical Graph dialog box is saved for the next instance.
Every time you click the Show Historical Graph command, the Historical Graph dialog box opens in the previously changed position and size.

For more information, see Using the Historical Graph.
Quick Select - Retain View in 3D After OK (V5-6R2024SP1)
The Retain View After OK option is added in the Quick Select dialog box. Selecting this option and clicking OK shows only the selected elements in the tree and in the 3D area.



For more information, see Quick Selection of Geometry.
Displaying Trim Information for Filled Surfaces (V5-6R2024GA) 

During the creation of fill surfaces, you can select Check if the fill creation required a trim to display a warning message whenever surfaces had to be trimmed to fit into the boundary edges.



For more information, see Creating Fill Surfaces.

Arbitrary Selection Order for Isolating Geometric Elements (V5-6R2024GA)
 
You can now isolate the elements by selecting them in any order. The new algorithm correctly arranges the elements for the isolation.

For more information, see Isolating Geometric Elements.

Displaying Continuity Tolerance for Smoothed Curves (V5-6R2024GA)
 
When you smooth out a curve, the continuity information displayed near the curve in the 3D area considers the specified tolerances.
The status of the curves to smooth now updates in the 3D area according to the current curve smooth parameters.

For more information, see Smoothing Curves.

Extracting Sub-Elements (V5-6R2024GA)
 
You can select and remove multiple elements from the list in the Multiple Extract Definition dialog box by holding Shift or Ctrl key.
With this enhancement, you can remove multiple elements at once instead of removing them one at a time.

For more information, see Extracting Sub-Elements.

Creating Extract Features for Multiple Sub-Elements or for All Domains (V5-6R2024GA)
 
You can use the keep sub-elements using Extract option to manage the result of a multidomain feature. This option lets you select multiple subelements and create an extract for each of them.

For more information, see Managing Multi-Result Operations.

Generate a Flat Surface While Creating Automatic Midsurfaces (V5-6R2024GA)
 
The Generate a flat surface check box is added in the Midsurface Definition dialog box.
You can create a flat midsurface without step jumps.

For more information, see Creating Mid Surfaces.
