Reading STL files

This task shows you how to open STL files into your Surface Machining session.

  1. Click Inserts an STL file .

  2. Select the Destination of the STL file in the specification tree.

  3. Click ... and navigate to the samples directory. Select the Basic.stl file.

  4. Click Open in the File Selection box then OK in the STL File Import box.

    You can now machine the part that you have just opened.
