Progress Indicator

  When performing certain operations in CATIA - Web Integration, a Progress Indicator window appears. These operations are:
  • New From...
  • Save to SmarTeam
  • Lifecycle operations
    • Copy
    • Check out
    • Check in
    • New Release
    • Release
  • Open
  • Refresh
