Migrating Thread Standards Files to XML

You can migrate thread standards files from standard formats to .xml format. Using the migration utility, you can perform the following operations:
  • Select a file or a folder to be migrated.
  • Select the output destination to save the migrated files.
  • Migrate the selected .txt, .xls, or .xlsx files to .xml format.
In a .txt thread standard file, the thread values must be followed by units in mm or inch.

  1. Select Tools > Utility.
    The Batch Monitor dialog box appears.

  2. From the Utilities tab, double-click MigrateThreadStandardToXML in the Type column. 
    The MigrateThreadStandardToXML dialog box appears.

  3. In the Object Selection area, click Add File(s).
    The File Selection dialog box appears in which you can browse to select one or more files you want to migrate.

  4. To remove the added files, select the files and click Remove File(s).

  5. In the Destination Selection area, click Browse to enter the path in the Destination Directory box. This is the destination where the .xml files are migrated.

  6. Optional: Click Save to save the current options for migrating the .xml files.
    This file can be used later on for another batch migration process without specifying the options again.

  7. Click Run to start the migration operation.
    • The migrated files are created in the specified directory.
    • The reports are generated in the default temporary folder of your system.

  8. Optional: Open the report file to see a log of the migration activity.
    If the migration fails, reasons of failure are indicated in this report.
