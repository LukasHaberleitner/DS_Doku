Transferring a Diagram

PID or HVAC Diagrams documents can be transferred to other computers or networks. However, there is one factor that you must consider.

Each line ID you create has a file associated with it. These files are stored in a directory specified by you. When you transfer a diagram, you must either make sure that this directory can be accessed from the new site, or you must copy the files also and specify the new directory path.

See Understanding Project Resource Management for more information on using resources.
