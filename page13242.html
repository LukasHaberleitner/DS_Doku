Generating a Reconciliation Report

This task shows you how to generate a report showing the rules you have applied or not to the documents you want to reconcile.

  1. Click Report .
    The report is displayed in a viewer. For each object, it sums up its characteristics (Name, PDM environment, storage mode, reconciliation rule, PDMID) and comparison status.
    The layout of the report can be customized to fit your needs. For more information, see Customizing Report Layouts.

  2. If need be, print the report.
