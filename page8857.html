Applying Modifications on a Subdivision Surface

This task shows you how to validate commands automatically or manually and manage the update of features when the pointer is released.

Open a 3D shape representation with subdivision surface.

  1. From the Modification toolbar, click Modification .
    The Tools Palette appears.

  2. Modify the shape using any modification command.
    The selection of elements is previewed on the surface and the Apply (Enter) command becomes available once a modification can be validated.

  3. Click Apply (Enter) to apply the modifications.
    The Apply (Enter) command is now unavailable , as there is no new modification.
    Note:
       If you escape the command without validating it, an warning dialog box appears. Click Yes to apply the modifications or click No to exit the command without applying the modifications.

  4. Modify the shape again.

  5. To switch from the manual mode to the automatic mode, click Apply (Enter) twice, first to validate the current modification and second to access the automatic mode.

    The visualization of the Apply (Enter) icon is modified to . Modifications are now automatically validated.

    Note:
       You can perform an undo or a redo operation when you have validated a command. All the modifications are grouped and replaced by the Apply single state. However, when you switch to the manual mode, you can undo or redo each modification individually.
