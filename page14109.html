Creating a Boundary Partition

This task shows you how to create boundary sections to stabilize inputs for the detailed design.
Before you begin, see information in Creating an Addendum Design.
Open AddendumSurface01.CATPart from the samples directory.

  1. Click Boundary Partition  in the Addendum Design toolbar.

  2. Select the Reference Direction used to compute the wall of the part and the die.

  3. Select the boundary or the surface to process.
    If you select a surface, the boundary is computed from that surface.

  4. Select the elements to cut the boundary.
    They can be a sketch or points.

  5. Specify the type of the output curve:
    • Prehem
    • Neutral
    • Variable Offset Curve

  6. Enter the Parameters values.
    • Flange Radius to compute the edge fillet between the surface extension and the wall of the part and die.
    • Max Flange Length of the flange extension.
    • Flange Angle Die to compute the wall of the die.
    • Flange Angle Punch to compute the wall of the punch.

  7. Click Apply.
    The Boundary Sections are listed with their Type.
    The type can be Offset or Transition. If the type is Offset, the value is listed under Flange Extension.

  8. Select one Boundary Section and
    • Change its type from the list below.
    • For Offset, edit the value under Flange Extension.

A Partition Boundary node is created in the specification tree. It contains

  • The parameters used for the partition.
  • The Boundary Sections with their type.
