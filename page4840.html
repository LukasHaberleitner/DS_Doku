Managing PowerCopies

Create PowerCopies: Select the Insert  > Knowledge Templates  >  PowerCopy Creation command, select the elements making up the PowerCopy from the specification tree, define a name for the PowerCopy and its reference elements then choose an icon for identifying it.
Instantiate PowerCopies: Select the Insert  >  Instantiate From Document command, select the document or catalog containing the  powercopy, complete the Inputs within the dialog box selecting adequate elements in the geometric area.
Save PowerCopies into a Catalog: Select the PowerCopy from the specification tree, select the Insert  >  Knowledge Templates  >  PowerCopy Save In Catalog... command, enter the catalog name and click Open.
