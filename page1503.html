Shading with Material

 

This tasks explains how to display materials that have been applied onto objects.
  Refer to "Applying Materials" in the Version 5 - Real Time Rendering User's Guide for detailed information on how to use materials.

  • Select View > Render Style > Shading with Material  or click Shading with Material in the View toolbar.
 

Note that you can also select View > Render Style > Customize View Parameters then click Material.

Setting a shaded part to Shading with Material has the following effect:
 

Shading only

Shading + Material
(Roughcast material has been applied)
