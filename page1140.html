Geometrical Set Creation

 

This page deal with options concerning the creation of a specific geometrical set by some commands to store their output.

Many commands of Die Face Design create a specific geometrical set to store the result features.
Clear the corresponding check boxes to prevent the creation of the specific geometrical set: The result features are created under the current geometrical set.

Select All selects or clears all check boxes in one shot.
  By default, all check boxes are selected.
