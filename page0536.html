Activating/Deactivating Images

This task shows how to modify the current visualization by activating or deactivating images.

Open the sample26.CATAnalysis document from the samples directory.

Activating Images
 

You can activate images using the Activate/Deactivate contextual menu.

  1. Right-click the image you want to activate.

    In this particular example, right-click Von Mises Stress (nodal value).1 in the specification tree.

  2. Select Activate/Deactivate , or select Force Activate if the image is not within the current analysis case.

    The image is now activated:

You can also activate an image by editing it. In this case, the Image Edition dialog box appears. To know more, refer to Editing Images.

 

Deactivating Images

 

You can deactivate images using the Activate/Deactivate contextual menu.

  1. Right-click Von Mises Stress (nodal value).1 in the specification tree.

  2. Select Activate/Deactivate .
