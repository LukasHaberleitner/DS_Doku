Using Knowledge Assembly Templates

This task shows you how to create a catalog assembly that you can use repeatedly in your design.

Main steps are:

  • Creating an assembly using Electrical Harness Assembly commands
  • Creating an assembly template
  • Storing the template in a catalog
  • Instantiating the assembly.

Catalog assemblies are document templates.
A PKT license is required.

  1. Create your harness assembly as normal using Electrical Harness Assembly commands.

  2. Create an assembly template.

    1. Select Insert > Document Template Creation...

      The Document Template Definition dialog box appears.

    2. In the Documents tab, set devices to Same document Action status (Switch between New Document and Same Document). In our example, the plug.
      This means you will use the existing catalog reference for these devices.

      Any devices except adaptive supports must be set to Same document Action status.

    3. Select the Inputs tab to see the point used when instantiating the template.

    4. Click OK when done.

    5. Save the document.

  3. Store the assembly template in a catalog.

    1. Open the catalog in which you want to store the template.

      The Catalog Editor workbench opens showing the contents of the catalog.

    2. Activate the family to which you want to add the template.

    3. Click Add Component to associate the appropriate assembly template.

      The Description Definition dialog box appears.

    4. Click Select external feature, then select the document template in the specification tree of the document containing your template.

    5. Click OK in the Description Definition dialog box.

      The harness assembly template is added to the catalog.

    6. Save the catalog when done.

  4. Instantiate the assembly.

    1. Switch to the Electrical Harness Assembly workbench.

    2. Select Catalog Browser .

    3. In the Catalog Browser, double-click the assembly template you created.

      The Insert Object dialog box appears.

    4. In your design document, select the point at which to position your assembly template, then click OK.

    5. Repeat to instantiate the assembly as many times as desired.

      Note: All points are instantiated at the same location.

    6. Using the Electrical Assembly Design workbench, connect plugs to cavities to complete your design.
