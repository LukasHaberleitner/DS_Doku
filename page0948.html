ICEM Shape Design AeroExpert

This page deals with ICEM Shape Design AeroExpert options in the following tab:
  • The General tab lets you set the general options.
  • The Approximation tab lets you set default parameters for approximation options.
  • The Associativity tab lets you set the associativity options.
  • The Manipulators tab lets you set the control point options.
  • The Tolerance tab lets you set tolerances.
  • The Topology tab lets you control the elements to be selected.
