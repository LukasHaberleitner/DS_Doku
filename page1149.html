Robot Task Display  


This page deals with the Robot Task Display options for Resource Detailing robotics products.

  • Cartesian Target
  • Joint Target
  • Line
  • Set/Wait Line
  • Other Options

Cartesian Target

The color and symbol options control the appearance of tag, weld, and Cartesian targets.  Selecting Show X, Y, Z Values or Show Y, P, R Values means the values are displayed when you move a mouse over the target.

The initial color and symbol settings are shown above; the default setting is not to show the values.

Joint Target

The color and symbol options control the appearance of joint and home targets.  Selecting Show Joint Values or Show Auxiliary Axes Joint Values means the values are displayed when you move a mouse over the target.

The initial color and symbol settings are shown above; the default setting is not to show the values.

Line

The color, line, and weight settings determine the appearance of the lines appearing on the geometry that connect the operations.  You can choose to show or not show the line.

The initial color, line, and weight settings are shown above; the default setting not is to show the line.

Set/Wait Line

The color, line, and weight settings determine the appearance of lines appearing on the geometry that show the IO connections.

The initial color, line, and weight settings are shown above.

Other Options

  • Show Actions displays as icons the actions in the geometry.
  • Selecting the Show Target Name check box makes the target name appear when you move the mouse over the target.
  • Selecting the Display VN or Operation Name check box shows the name of the operation or the via node (VN) number of the operation.

By default, none of the options are selected.

The task display can be activated using the contextual menu on the task.
