Display Information About Routables

This task shows you how to display information about routables. It works only when you are performing an action with a routable, such as placing a part, branching a run or creating an offset route. To display more detailed information see Generating Detail Information.
1. Go to Tools - Options, Equipment & Systems. Select the Display tab and check the  Analysis Mode box.
2. Place the pointer over the run. If a segment is highlighted you will see the length displayed. If a node is highlighted you will see the angle between two adjacent segments, the coordinates of the node and the bend radius if there is one.
3. Click the button again if you do not want to see the information.
