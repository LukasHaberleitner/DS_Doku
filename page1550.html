Using 3DEXPERIENCE Platform Panel

This task explains how to use Part Supply from the 3DEXPERIENCE side panel.

  1. Select Start> 3DEXPERIENCE platform and enter Username and Password to log in into 3DEXPERIENCE server.

  2. Select 3DEXPERIENCE Marketplace and click Part Supply in 3D Compass.

  3. Browse through the directory and select the required component.

    Detailed view of the component is displayed. Following options are available:
    Add to bookmark : Bookmarks the component.
    Compare : Compares the component with other components.
    Open : Opens the component in CATIA V5.
    Insert : Inserts the component at the insertion point.
    Replace : Replaces the selected component in the product structure.

    When you select Open, download starts.
    When you select Insert, you must select insertion point.
    When you select Replace, you must select a component for replacement.

    After completing the download, Part Supply Download Folder dialog box appears.

  4. Select a folder to extract the downloaded component and click OK to open, insert or replace the component.

    When you select the Do not remind me next time check box, subsequent components are saved in the same location and the Part Supply Download Folder dialog box does not appear. If you want to change the location, select Tools > Options > General > Compatibility > 3DEXPERIENCE Platform and select the required folder.
