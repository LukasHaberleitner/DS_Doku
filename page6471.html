Creating an Electrical Bundle  

This task shows you how to create an electrical bundle.
An electrical bundle is a document containing the wires routed through geometrical bundles. An electrical bundle is sometimes also referred to simply as a bundle.
Open a document containing a geometrical bundle composed of electrical devices and bundle segments.
  If you work in visualization mode, since routing is possible with this mode, you need to update your document at opening to load the publication.

  1. Click Update .

    An update warning opens:

  2. Click OK to validate.

  1. Select New Bundle .

    The New Bundle dialog box opens:

  2. Change the electrical bundle name if needed.

  3. Select a geometrical bundle in the specification tree that you want to be connected to the electrical bundle.

  4. Click OK to validate.

    The electrical bundle is automatically created under the active product.
    It is added to the specification tree.
An electrical bundle is associated to a geometrical bundle by the wires it contains.

If the geometry is not loaded, check the Product Structure settings.
An alternative to display the geometry is to choose the Representations  >  Activate Terminal Node item. Right-click Product1 to use the contextual menu or select Edit  >  Representations.
