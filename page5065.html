Editing and Deleting Scallops

This task explains you how to edit and delete the scallops created at the corners of the plate.
Open a document with scallops placed already. For more information about creating scallop, refer to Creating Scallops.
 

Edit Scallops

  1. Click Scallop in the Detailing Tools toolbar.

    The Scallop dialog box appears.

  2. Select the plate whose scallop you want to edit.

    All the scallops on the plate are listed in the dialog box.

  3. To edit the scallop, select one or more rows, then right-click and select Define Scallop.

    The Scallop Parameters dialog box appears.

  4. Edit the dimensions:

    1. In the Contour Parameters area, edit the value by double-clicking it.
    2. In the Position Parameters area, edit the angle value by double-clicking it.
    3. Click OK to return to the Scallop dialog box.

    The scallop dimensions are changed.

  5. Click OK in the Scallop dialog box.
    The scallop is edited.
    You can also edit the dimensions of the scallop by double-clicking the parameters under the ScallopContour node in the specification tree. The Edit Parameter dialog box appears.

 

Delete Scallops

  1. Click Scallop .

    The Scallop dialog box appears.

  2. Select the plate whose scallop you want to delete.
    All the scallops on the plate are listed in the dialog box.

  3. To delete the scallop, select one or more rows, then right-click and select Delete Scallop.

  4. Click OK in the Scallop dialog box.
    The scallop is deleted.
