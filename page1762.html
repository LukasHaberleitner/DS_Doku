Supported Objects by the Typing Commands  

The goal of a typing command is to dynamically add a type (also called technological object) on an existing object. The supported  objects by the typing commands are listed in the table below:

Products
  • Product References
  • Product Instances
Mechanical Bodies
  • PartBody
  • Body Geometrical Set
  • Ordered Geometrical Set
Mechanical Features It applies to any type of Mechanical Feature. For example, a pad, a hole, a fillet, a point, a line...
