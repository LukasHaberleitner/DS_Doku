Resolved Sections Catalog: Overview

When you create a section, it is created as a parametric part, with an associated design table. In order to create a usable resolved sections catalog, and place sections in it, you need to take the following steps. These are explained in this section.
  • Resolve the parametric parts, using the procedure described in Generating Resolved Parts from Parametric Parts.
  • Create a catalog, using the procedure described in Creating a Specifications Catalog. The catalog is created using a standard facility provided with the V5 Product line.
  • Add sections to the catalog, a process that is incorporated in the task Creating a Specifications Catalog.
  • Make sure the catalog is correctly pointed to in your project resource management file.
