Logical Activities Creation and Process Library Creation Toolbars

The hyperlinks below link to the procedures that describe how to use the commands in the Logical Activities Creation toolbar and the Process Library Creation toolbar. The  Logical Activities Creation toolbar is available in both the workbench and in the Derivation Viewer for defining process libraries.  In the Derivation Viewer, it is called the Process Library Logical toolbar. The Logical Activities Creation toolbar is available in the Derivation Viewer only.
