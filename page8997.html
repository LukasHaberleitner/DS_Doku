AbqMsTargetMethod (Enumeration)

enum AbqMsTargetMethod {
  ABQ_BELOW_MIN,
  ABQ_UNIFORM,
  ABQ_SET_EQUAL
}
Mass Scaling Target Method.
Values:
ABQ_BELOW_MIN
Scale if below minimum target.
ABQ_UNIFORM
Scale uniformly to satisfy target.
ABQ_SET_EQUAL
Scale nonuniformly to equal target.
