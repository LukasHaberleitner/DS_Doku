Opening a User Dictionary

This task explains how to make a reference to an existing .CATfct file.

  1. From the Start>Knowledgeware menu, access the Business Process Knowledge Template workbench. The Feature Dictionary window opens.

  2. Click the Open User Dictionary icon (). The Open User  Dictionary dialog box opens:

  3. Navigate to the directory where your .CATfct files are stored. The default is .. intel_a\resources\graphic.

  4. Enter the name of the .CATfct file and click Open. The .CATfct file opens. Its types are now displayed under the Classes View node.
