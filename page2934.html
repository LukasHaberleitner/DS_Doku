Customizing Component Catalog Editor

This task explains how to customize your Catalog settings.
This type of customization is stored in permanent setting files: these settings will not be lost if you end your session.
 

  1. Select the Tools  >  Options menu. The Options dialog box is displayed.

  2. Choose the Infrastructure category in the left-hand box.

  3. Click the Catalog Editor workbench.

    The options for Catalog Editor settings appear, organized in tab pages.

    The Catalogs tab enables you to define options for resolved family components.
 

  1. Set options in these tabs according to your needs.

  2. Click OK in the Options dialog box to confirm the chosen options.
