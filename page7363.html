Administration Tasks

In the Functional Tolerancing & Annotation workbench, administration tasks deal with the administration of standards.

Administrators can manage and customize standards such as ISO_3D, ISO_3D_DS, JIS_3D, JIS_3D_DS, ASME_3D, etc. or company standards.
The Standards Editor lets the administrators set the standards used for dress-up, dimensions, annotations, etc. as well as set the styles that will be used as defaults for element properties in the Functional Tolerancing & Annotation.

  • These tasks must be performed by an administrator.
  • Some standard definition do not affect Functional Tolerancing & Annotation.
