View Generation Modes

This task explains the following modes of view generation in Structure applications:
  • FTA View
  • Advanced Front View
  • Logical View
  • Shell Expansion View
  The table below shows a comparison between these modes:
 
Feature FTA View Front View/Advanced Front View Logical View Shell Expansion View
Creates 3D annotations, 3D Dimensions Applicative views in Structure Logical drawings only in Structure Expanded view of shell in transverse direction
