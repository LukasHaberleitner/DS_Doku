Characteristic Curves

Reset with Standard Settings

Click this button to reset settings on this tab according to the parameters defined in the standards. For more information on standards, refer to the Administration Tasks chapter in the Aerospace Sheet Metal Design User's Guide.

Clear all Check Boxes

Removes the check mark from all check boxes on this tab.
 

Surfacic Flange Features/ Swept Flange Features

 

Brake formed
 
  Specify whether the BTL Base Feature, BTL Support, IML, OML, Second OML, and CLB should be seen in folded and unfolded 3D views of brake formed surfacic flanges, by selecting respectively the Folded view and Unfolded view check boxes.

The default value for these check boxes is specified in the standards.
 

Hydro-pressed
 
  Specify whether the BTL Base Feature, BTL Support, IML, OML, Second OML, and CLB should be seen in folded and unfolded 3D views of hydro-pressed surfacic flanges, by selecting respectively the Folded view and Unfolded view check boxes.

The default value for these check boxes is specified in the standards.
