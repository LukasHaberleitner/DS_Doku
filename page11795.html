Displaying Technological Results

 

This task shows you how to display technological results and how to export them in a .csv file.
  • Open TechnologicalResult02.CATProduct from the samples/TechnologicalResultB directory.
  • Note that the export to a .csv file is available only if you have a license for Prismatic Machining Preparation Assistant (MMA or MPA).
  • The command takes into account the units defined in Tools > Options.

  1. Go to the Tools > Options > Mechanical Design > Mold Tooling Design  > Miscellaneous tab and check that the Technological Results creation check box is selected.

  2. Select Analyze > Display Technological Results.
    The Display Technological Results dialog box is displayed. It is resizable.

  3. Select he body where the Technological Results have been created.
    The dialog box is populated with information about the Technological Results found in Part1.

    The origin of holes (not of the drill holes) and axis direction of the holes are displayed in the 3D viewer.

  4. By default, the absolute axis system of the part selected is used.
    You can select another axis. All position and direction values are then recomputed in this new axis system.

  5. Scroll to the bottom of the table in the dialog box.
    There is a Thread associated to Hole.70. The name of this Thread is given in the Thread column.
    If you select the line in the dialog box, the corresponding elements are highlighted in the 3D viewer.

     

  6. Click a column header to sort the technological results on that criterion.

  7. To export those information to a .csv file, enter the path and name of this file in the Output Path field, and click Output.

    Note that the Technological Results are not sorted in the Ouptput.
