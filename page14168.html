Generating Resolved Tubing Assembly

This task explains you how to resolve the tubing assembly and add the resolved tubing assembly to the catalog.
  • You must be connected to ENOVIA database.
  • Tubing assembly must be saved in the database.
  • A template must be available in the repository.

  1. Click Generate Resolved Tubing Assembly  in the Build Create toolbar.
    The     Generate Resolved Tubing Assembly dialog box appears.

  2. Click the Browse button to select the template.
    The selected file appears in the Template List box.

  3. Click the Browse button to select the location at which the resolved assembly is stored.
    The selected path appears in the Report Folder box.

  4. Click OK.
    The tubing assembly is resolved.

  5. In the ENOVIA V5 VPM Navigator toolbar, click ENOVIA Search to find the resolved tubing assembly.

  6. In the Search Results window, select the resolved tubing assembly, right-click and select Open.
    The Open Modes dialog box appears.

  7. Select the Catalog Editor option and click OK.
    You are switched to the Catalog Editor workbench.

  8. Click the Create/Modify Catalog in the Specification Catalog toolbar.
    The Create/Modify dialog box appears.

  9. Browse for the resolved assembly file in the repository.
    The selected file appears in the Parts List box.

  10. Select the required options for creating catalog.
    For more information, see Catalogs.

  11. Click OK.
    All the tubing assemblies are saved in the catalog under the respective part families.
    The tubing assemblies are fully resolved. The reference status for tubing assemblies appears as Fully resolved for standard assemblies and Parametric for unique assemblies.
