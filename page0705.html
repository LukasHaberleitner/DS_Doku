Features in Manufacturing Hub
This has the option to choose whether the position of features should be loaded to V5 with respect to the world coordinate frame or relative to the part. This can be set using the following.

Position is used for visualization of feature in 3D viewer. In V5 position information is displayed in Properties dialog box.

Maximum offset to edit manufacturing position

When users edit manufacturing locations, they can set constraints on how far the manufacturing locations can be set from their design location.
  • Allowing maximum offset along the X, Y and Z directions
  • Allowing maximum Tilt in the Z Axis.

By default, the constraints are set to 100.00 mm along the X, Y and Z directions, and to allow a 180.00 degree tilt from the Z axis.

Manufacturing Position Management

When selected, the option creates a unique fastener in the following manner:  the first time the position of a fastener is modified within a process, a new manufacturing fastener is created and associated with the process. The second (or any subsequent) modification of the position in the same process updates the new manufacturing fastener. If this manufacturing fastener is assigned to a new process and the user modifies the position and saves it to the manufacturing hub, a new manufacturing fastener is created that differs from the previous one.

When unchecked, any change in the position in V5 to the fastener is propagated to all processes to which fastener is assigned. A unique manufacturing fastener is maintained.

By default, option is selected.
