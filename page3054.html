Creating a Bounding Box

This task explains how to create an associative bounding box around a shape, that lets you modify either the shape or the axis system.
You must have a molded part in the workbench.
Go to Tools > Options > Infrastructure > Part Infrastructure > Display and select Parameters under Display in Specification Tree

  1. Open FilterCover.CATPart in the samples/import directory.

  2. Click Bounding Box in the Models toolbar.
    The Create Bounding Box dialog box is displayed.

  3. Select a shape and an axis system.
    A bounding box is displayed around the shape.
    The dialog box is updated.

  • The shape must be connex, with no undercut.
  • Only direct unit axis systems are supported.
 

  1. Select the type of the bounding box: Box or Cylinder.
    By default, the type is set to Box.

  2. If necessary, use the manipulators of the bounding box or the spinners of the dialog box to define offset planes.
    The Approximated Bounding Box Size is updated automatically.
    The displayed lengths may vary depending on the graphic parameters.
  • Negative offset planes are supported.
  • When the bounding box is a cylinder, X Length and Y Length correspond to the diameter of the cylinder.
 

  1. Click OK to validate and exit the dialog box.
    The bounding box is created:

    • A pad is created in the Bounding Box.1 body,
    • The standard features and parameters are created (respectively in Bounding Box Elements.1 and Parameters bodies) and are editable.
