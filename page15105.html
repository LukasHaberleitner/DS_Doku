Working with a Drawing and a Part  

 

This task describes the methodology for working with a Drawing and a Part.

Updating a Drawing and a Part in the Database

  1. Apply the Overwrite rule to the Drawing and to the Part.

  2. Click the Apply button to apply the reconciliation.

  3. Force part drawing links resolution by using the Edit > Links > Replace

  4. Click Update to update the drawing.

  5. Save the documents in the database.


Updating a Drawing Based on a Modified Part in the Database

Your objective is to keep the drawing modifications and update this drawing with the modified Part stored in the database.

  1. Apply the Overwrite rule to the Drawing.

  2. Apply the Reload rule to the Part.

  3. Click Apply, to implement the reconciliation.

  4. Click Update to update the drawing.

  5. Save the documents in the database.

Updating a Drawing in the Database Based on a Part Modified in the Database

Your objective is to save the (file-based) modification of the Part in the database and to update the drawing.

  1. Open the CATPart in CATIA session and launch Reconciliator Window dialog box.

  2. Apply the Overwrite rule to the Part.

  3. Click Apply, to implement the reconciliation.

  4. Save the CATPart document in the database.

  5. Open the drawing in session from the database.

  6. Click Update to update the drawing.

  7. Save the documents in the database.
