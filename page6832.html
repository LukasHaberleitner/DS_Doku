Renaming Pins on a Component

This task shows you how rename pins after adding them to a component.

See Defining Pins on a Component for an explanation on how to add pins to a component.

  1. With your document open, click the Rename Pin button and select the component.

    The Rename Pins dialog box displays.

  2. Select the pin to rename from the list. Click and type over the name. Click OK.

    The renamed pin displays in the specifications tree with the name you entered.
