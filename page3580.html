Modifying a Datum Feature

This task shows you how to modify a datum feature by editing it.
Open the Brackets_views03.CATDrawing document. Create a datum feature with a value of A.

  1. Double-click the datum feature you want to modify. The Datum Feature Modification dialog box is displayed.

  2. Modify the datum feature value. For example, enter B instead of A.

  3. Click OK. The datum feature is modified.

  4. Optionally drag the datum feature to move it.

 
Note that depending on the type of element to which the datum feature is attached, you may not be able to move the datum feature as wanted. For example, if the datum feature is attached to a dimension line, you will only be able to move the datum feature along the dimension line direction.
