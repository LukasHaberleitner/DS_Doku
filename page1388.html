Closing Documents

This task shows you how to close a document.

  1. Select File > Close.

  2. If changes have been made since the last save, the Close dialog box appears. 

    Specify whether the file is to be saved or not.

Closing a document does not always unload its associated documents (whether they are modified or not). In fact, if some documents are pointed to by documents edited in other windows, they stay in memory and keep their modified status.
In this case, when reopening the closed document, a warning message is displayed to indicate that the document has already been loaded and modified in the current session and that the already loaded document will be edited.
