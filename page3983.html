Smoothing Surfaces

When using the Control Points capability, you are able to smooth a surface according to a smoothing factor.
Open the FreeStyle_Part_31.CATPart document.

  1. Click the Control Points icon:
    The Control Points dialog box appears.
     

  2. Select Surface.1.

  3. Select the control point as shown.

  4. Click the Smooth icon:
    You can adjust the smoothing factor slider to fit your needs.
    The greater the smoothing factor value is the smoother the result will be.
    The surface has been smoothed according to the selected control point.

  5. Click OK to validate modifications and close the dialog box.
