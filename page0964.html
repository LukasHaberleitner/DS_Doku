V4 Integration

This page deals with V4 Integration options placed in the following tabs:

  • The V4 Data Reading enables you to define the options of V4 data reading (3D, 2D, PRJ File) in CATIA V5.

  • The Saving as V4 Data lets you define the V5/V4 migration options.

  • The V4/V5 DRAW lets you define the options for V4/V5 migration of DRAW Data.

  • The V4/V5 SPACE lets you define the options for the migration of models into CATIA V5: geometric deformation, detail/dittos and mock-up control.

  • The V4/V5 SPEC lets you define V4/V5 migration options in order to optimize update & reroute operations and get V5 results that are closer to CATIA V4 (for instance, V4 propagation of edges for Draft).

  • The V4/V5 TUBING  lets you define V4/V5 migration options in order to optimize tubing operations in Spec tree.

  • The Migration Batch lets you define the V4/V5 migration options in Batch mode.
