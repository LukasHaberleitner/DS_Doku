Renaming Images in the Album

This task explains how to rename images in the album. Do not forget that you can only modify the name, you cannot modify the class or the extension, for instance.

  1. Select Tools > Image > Album to display the album contents:

  2. Select the image to be renamed then right-click and select Rename.

    In our example, we are going to rename "Capture_002".
    When Rename is selected, the name of the selected image becomes editable to let you type the new image name:

  3. Key in the new name then press Enter to validate. The new image name is displayed in the list:
