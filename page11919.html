Linking a Script to a Component

This task explains how to associate a script generated by using  the Generative Script product with a component. For information on how to write a script, see the Product Knowledge Template User's Guide.

  1. Open the PfdSyringeAndFinger.CATSystem document. The following graphic is displayed. 

  1. Right-click the Container object in the geometry or in the specification tree, then select the Generative Script of Functional Object command from the contextual menu. The Edit the generative scripts of a functional elements dialog box is displayed.

  2. Click New, indicate the name of the new script and click OK. Double-click the name you have just indicated, the GenerativeScript editor opens.

  3. Enter or copy/paste the script provided in the PfdSyringeContainer.CATGScript sample in the editor, then click Generate. The document below is generated. Please note that you need a Product Knowledge Template license to generate a script.
