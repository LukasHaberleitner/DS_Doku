Specification Tree

 

Within the DMU Navigator workbench, you can generate a number of features that are identified in the specification tree by the following icons.

Icons displayed in the specification tree and specific to the DMU Navigator workbench identify:

Annotated views

Audio markers

Hyperlinks
3D annotations

Cameras

Groups

Enhanced Scenes

Scenes

Simulations

Replays
Measures made using the Measure Inertia command
Measures made using the Measure Item command
Measures made using the Measure Between command 

Distance and band analysis entries

For more information on the specification tree, see the Infrastructure User's Guide, Basic Tasks, Viewing the Specification Tree and Geometry Area.
