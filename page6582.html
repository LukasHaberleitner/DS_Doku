Partially Flattening the Harness

This task shows you how to partially flatten a harness by selecting a bundle segment.

The ElectricalFlattening.CATProduct document is still open.

  1. Click Flatten .

    The Flatten dialog box opens.

    The default values of this dialog box are the values you have already defined in the Harness Flattening Parameters dialog box.

  2. Select the Active plane. The default plane is the xy plane.

  3. The algorithm mode can be coarse or fine.
    Select Coarse for the purpose of this scenario.

  4. The angle mode can be standard or regular.
    Select Standard.

  5. In the geometry area or the specification tree, select a bundle segment.

    Where you select the bundle segment is important: the end nearest your selection becomes the fixed point.
    Two points are highlighted in the geometry area:

    • The Fixed Point identifying the start point for the partial flatten. No bundle segments upstream of this point are taken into account.

    • The Select Network identifying the end point for the partial flatten. Bundle segments downstream of this point can be taken into account by selecting them individually or by clicking Select All Branches in the Flatten dialog box.
      In the case of multi-branchable documents, the end points of each bundle segment are highlighted.
    Notes:
    • In order to avoid twisted and impossible configurations, each bundle segment profile is rotated such that, once flattened, the local V-axis of the profile sketch is aligned with the normal direction of the active plane.
    • When flattened, the external curve is migrated to a standard electrical curve. Both the length and bend radius (equal to the diameter) of the external curve are kept.

  6. Optional: Double-click Fixed Point label to invert start and end points.

  7. Click Select All Branches in the Flatten dialog box.

    All bundle segments beyond the Select Network point are taken into account and highlighted in the geometry area.

    Note: If you do not click Select All Branches, you can select a Bundle Segment only if the previous one has been selected.
     

  8. Click OK.
